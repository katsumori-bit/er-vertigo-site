>
> 1. **来院 10 分以内に 12 誘導 ECG**
> 2. **5 大致死的胸痛**（ACS / 大動脈解離 / 肺塞栓 / 緊張性気胸 / 食道破裂）を外す
> 3. **HEART / Wells・PERC** で前確率を絞る
> 4. **トロポニン・D-dimer・胸部 X 線・必要で造影 CT**
> 5. **診断がついたら疾患ページへ** → [[STEMI]] / [[ACS]] / [[大動脈解離]] / [[肺塞栓]] / [[気胸]] / [[食道破裂]]

> [!tip] 主訴が胸痛ではないとき
> - 呼吸困難が前景 → [[呼吸困難]]
> - 動悸・失神が前景 → [[意識消失]]
> - 心窩部痛が前景 → [[腹痛]]（下壁梗塞・膵炎・胆道系も）

## 0. 来院 10 分以内にやること

> [!danger] 来院 → 10 分以内
>
> 1. **バイタル**（血圧は両上肢で）
> 2. **モニター**（SpO2・ECG・自動血圧）
> 3. **末梢ルート**（できれば 18–20G 2 本）
> 4. **12 誘導 ECG**（AHA/ACC：10 分以内）
> 5. **採血**：hs-cTn、D-dimer、CBC、生化、凝固、血型・クロス、血ガス
> 6. **酸素**：SpO2 < 90% or 呼吸困難で投与（ルーチン高流量酸素は避ける）
> 7. **アスピリン 162–325 mg 咀嚼**：ACS が否定できなければ早期に（禁忌・出血リスク確認）

## 1. 致死的胸痛 5 つを外す

| 疾患 | 典型 | 初手 | 疾患ページ |
| --- | --- | --- | --- |
| **ACS（STEMI/NSTEMI/UA）** | 圧迫感、冷汗、放散痛、嘔気、労作誘発・休息軽快 | **12 誘導 ECG + トロポニン**、STEMI なら直ちに循環器コール | [[STEMI]] / [[ACS]] |
| **大動脈解離** | 突然の引き裂かれる胸背部痛、移動する痛み、左右上肢血圧差、新規 AR 雑音 | **造影 CT**（ADD-RS ≤1 + D-dimer 陰性で除外補助） | [[大動脈解離]] |
| **肺塞栓（PE）** | 突然の呼吸困難 + 頻脈、胸膜痛、片側下肢腫脹、長期臥床・術後・OC | **Wells/PERC → D-dimer → CTPA** | [[肺塞栓]] |
| **緊張性気胸** | 突然の片側胸痛 + 呼吸困難、片側呼吸音消失、頸静脈怒張、気管偏位 | **画像を待たず針減圧 → ドレナージ** | [[気胸]] |
| **食道破裂（Boerhaave）** | 強い嘔吐後の胸痛、皮下気腫、縦隔気腫 | **造影 CT or 食道造影**、外科コンサルト | [[食道破裂]] |

## 2. 問診

### OPQRST

- **O**nset：いつから、どんな状況で（労作中？安静時？嘔吐後？）
- **P**rovocation/Palliation：労作・体位・呼吸・食事
- **Q**uality：圧迫感、絞扼感、引き裂かれる、刺す、灼熱
- **R**egion/Radiation：左肩・顎・両腕・背部
- **S**everity：NRS
- **T**iming：持続時間、反復パターン

### 必ず聞く

- **発症様式**：突然 vs 徐々（突然なら解離・PE・気胸）
- **持続時間**：数秒（非心臓性多い）vs 持続（ACS・解離・PE）
- **随伴**：呼吸困難、冷汗、嘔気、失神、動悸、咳、血痰、発熱
- **危険因子**：年齢、性別、喫煙、HT、DM、脂質、家族歴、避妊薬、悪性腫瘍、長期臥床、手術、外傷、コカイン

> [!tip] ACS らしさの増減
> **らしさ UP**：放散痛（両腕 / 右肩 / 顎）・冷汗・嘔気嘔吐・労作誘発 - 休息軽快・ACS/PCI 歴
> **らしさ DOWN**：体位変換で増悪・触診で再現できる胸壁痛・数秒の刺す痛み
> （**ただし胸壁再現性があっても PE / 解離は除外できない**）

---

## 3. 身体所見

- **バイタル**：両上肢血圧（差 ≥ 20 mmHg → 解離示唆）
- **頸**：頸静脈怒張（PE / タンポナーデ / 緊張性気胸）
- **胸**：呼吸音左右差・減弱、皮下気腫
- **心音**：新規雑音（AR → 解離）、S3/S4、心膜摩擦音
- **腹部**：圧痛（下壁梗塞は心窩部痛）
- **下肢**：片側腫脹（DVT）

---

## 4. 12 誘導 ECG の見方

| 所見 | 考える疾患 |
| --- | --- |
| **ST 上昇**（隣接 2 誘導以上） | STEMI → [[STEMI]] |
| **ST 下降 / T 陰転** | NSTEMI / UA、左主幹部・三枝、後壁梗塞 |
| **後壁梗塞パターン**（V1–V3 R 増高 + ST 下降） | → [[後壁梗塞]]（V7–V9 再評価） |
| **de Winter T** | LAD 近位部閉塞 → [[de Winterパターン]] |
| **Wellens 征候** | LAD 高度狭窄 → [[Wellens症候群]] |
| **新規 LBBB** | STEMI 等価（Sgarbossa） |
| **右心負荷**（S1Q3T3、右軸、V1–V3 T 陰転） | PE |
| **電気的交互現象** | 心タンポナーデ |
| **広範 ST 上昇 + PR 下降** | 急性心膜炎 |

> [!check] ECG は 1 回で終わらせない
> - 初回非診断的・症状持続 → **15–30 分後に再検**
> - 右側胸部誘導（V3R, V4R）→ 右室梗塞、後壁誘導（V7–V9）→ 後壁梗塞

詳細 → [[入院時心電図]]

## 5. リスク評価

### HEART スコア（ED での ACS リスク層別化）

| 項目 | 0 点 | 1 点 | 2 点 |
| --- | --- | --- | --- |
| **H**istory | 非典型 | 中程度 | 典型的 |
| **E**CG | 正常 | 非特異的 | 有意な ST 偏位 |
| **A**ge | <45 | 45–64 | ≥65 |
| **R**isk factors | なし | 1–2 個 | ≥3 個 or 動脈硬化既往 |
| **T**roponin | 正常 | 1–3 倍 | >3 倍 |

- **0–3 点**：30 日 MACE < 2% → 早期帰宅検討（hs-cTn 0/3h プロトコル併用）
- **4–6 点**：観察入院 + 再検
- **≥7 点**：入院 + 早期侵襲的評価

### Wells / PERC（PE）

> [!check] PERC（全項目満たせば PE 検査不要）
> 年齢 < 50 / 心拍 < 100 / SpO2 ≥ 95% / 片側下肢腫脹なし / 喀血なし / 4 週以内手術・外傷なし / 既往 DVT・PE なし / エストロゲン使用なし

PERC 満たさない or Wells 中等度以上 → **D-dimer**（50 歳以上は age × 10 ng/mL）→ 陽性で **CTPA**

## 6. 採血と画像

### 採血

| 項目 | 意義 |
| --- | --- |
| **hs-cTn** | ACS 中心。**0/1h or 0/3h プロトコル**で rule-out |
| **D-dimer** | PE / 解離の rule-out |
| **BNP / NT-proBNP** | 心不全合併評価 |
| **CBC / 生化 / 凝固** | 腎機能、抗凝固評価 |
| **血型・クロス** | 解離・大量出血 |
| **血ガス** | 呼吸不全、A-aDO2 |

### 画像

- **胸部 X 線**：気胸、縦隔拡大（解離）、肺水腫、肺炎、皮下気腫
- **造影 CT（CTPA / 大動脈 CT）**：PE、解離、肺梗塞
- **心エコー**：壁運動異常（ACS）、右心拡大（PE）、心嚢液（タンポナーデ）、AR（解離）
- **POCUS**：ショック例で迅速評価

## 7. 初期対応（診断がつくまで）

> [!tip] 診断ごとの詳細治療は疾患ページへ
> 抗血小板・抗凝固レジメン、Stanford A/B 別治療、massive PE の線溶療法などは → **[[STEMI]] / [[ACS]] / [[大動脈解離]] / [[肺塞栓]]**

- **アスピリン 162–325 mg 咀嚼**：ACS 否定できないなら早期（禁忌・出血リスク確認）
- **酸素**：SpO2 < 90% のみ
- **硝酸薬（ニトログリセリン）**：持続痛・高血圧で。**右室梗塞・PDE5 阻害薬使用中は禁忌**
- **鎮痛**：硝酸薬で改善しない強い痛みには**モルヒネ**を限定使用（P2Y12 吸収遅延の懸念）
- **解離を疑ったら降圧**：**SBP 100–120 / HR ≤ 60 を目標**にランジオロール or エスモロール（詳細 → [[大動脈解離]]）
- **緊張性気胸を疑ったら画像を待たず針減圧** → [[気胸]]
- **STEMI → 直ちに循環器コール + 再灌流**（door-to-balloon ≤ 90 分、PCI 設備なければ転送、DIDO ≤ 30 分）→ [[STEMI]]
- **P2Y12 / ヘパリン / β 遮断薬 / スタチン** の用量と選択 → **[[STEMI]] / [[ACS]]** で解説

> [!warning] 初期対応での落とし穴
>
> - **胸壁圧痛で「筋骨格」と決めつける**（ACS・PE・解離は否定できない）
> - **初回 ECG 陰性で ACS 除外**（15–30 分後に再検、右側・後壁誘導）
> - **解離を疑わず抗血小板 + 抗凝固を走らせる**（大動脈解離で致命的）
> - **右室梗塞に硝酸薬 / ニトロ**（前負荷低下で血圧崩壊）

## 8. disposition

### 入院

- STEMI / NSTEMI / 高リスク UA
- 大動脈解離（A は手術、B は ICU 管理）
- PE（中等症以上は入院、低リスクは外来 DOAC も可）
- 緊張性気胸 / 食道破裂
- **HEART ≥ 4 / 持続痛 / 不安定**

### 帰宅検討

- **HEART 0–3 + hs-cTn 0/3h 陰性**
- **PERC 全項目陰性**
- 明らかな非心臓性原因が同定
- 再診基準を口頭 + 書面で説明

> [!warning] 帰宅前に伝える再診基準
> 痛み再発・悪化 / 呼吸困難 / 失神・冷汗・嘔気 / 背部への広がり / 片側下肢腫脹

---

## やりがちなミス

- 初回 ECG 陰性で ACS を除外する
- 胸壁圧痛があるので筋骨格痛と決めつける
- 解離を考えず抗血小板 + 抗凝固を流す
- 右室梗塞に硝酸薬を使う
- ルーチン高流量酸素投与
- HEART 0–3 なのに入院、HEART ≥ 7 なのに帰宅
