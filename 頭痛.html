頭痛

元: 臨床/ER/頭痛.md

このページの使い方
  1. まず危険な二次性頭痛かを5秒でふるい分ける
  2. 次にどのパターンの頭痛かを決める
  3. 必要な枝だけ開く
  4. 詳しい診察や各疾患はリンク先で確認する
最初に結論

以下のどれかがあれば、一次性頭痛と決め打ちしない

  • 雷鳴頭痛
  • 発熱 / 項部硬直 / 意識障害
  • 局所神経症状
  • 妊娠中 / 産褥期の新しい頭痛
  • 眼痛 + 視力低下 + 充血
  • 50歳以降の新規頭痛
  • 癌 / 免疫不全 / 抗凝固薬
  • 高血圧緊急症を疑う所見
  • 乳頭浮腫 / 体位で増悪 / 進行性悪化

最初の5秒

  • バイタルは安定しているか
  • 意識障害はないか
  • 雷鳴頭痛か
  • 発熱・項部硬直はないか
  • 麻痺・失語・複視・歩行障害・けいれんはないか
  • 妊娠中 / 産褥6週以内ではないか
  • 眼痛・視力低下・充血はないか
  • 50歳以上の新しい頭痛ではないか
  • 癌・免疫不全・抗凝固薬内服はないか
  • 今までと同じ頭痛か、それとも初めて / 最悪 / 明らかに違う頭痛か
頭痛で最初に見るもの
  • 発症様式：突然か、じわじわか
  • ピーク到達時間：一瞬〜1時間以内か
  • 初回か / いつもの頭痛か
  • 随伴症状：発熱、項部硬直、嘔吐、視覚症状、局所神経症状
  • 背景：妊娠、産褥、癌、免疫不全、抗凝固薬、最近の穿刺・外傷

まず分類する

  • 雷鳴頭痛
  • 発熱・髄膜刺激を伴う頭痛
  • 神経脱落症状 / 意識障害を伴う頭痛
  • 妊娠・産褥関連頭痛
  • 眼科救急を疑う頭痛
  • 50歳以上の新規頭痛
  • 頭蓋内圧異常を疑う頭痛
  • 典型的な一次性頭痛

対応フロー

1. 雷鳴頭痛

  • くも膜下出血を最優先
  • 他にも
    • 可逆性脳血管攣縮症候群
    • 頸動脈 / 椎骨動脈解離
    • 脳静脈洞血栓症
    • 下垂体卒中
    • 高血圧性脳症

      • を考える

この場でやること

  • 頭部単純CT
  • 神経診察
  • 血圧評価
  • 必要ならCTA or LPを追加

判断の実際

  • 神経学的にほぼ正常で、発症1時間以内にピークに達した新規重症頭痛では、オタワくも膜下出血ルールが高感度のふるい分けに使える
  • 発症6時間以内で、神経所見が正常、高性能CTで頭部単純CTが正常なら、ACEPは非外傷性くも膜下出血の除外に使えるとしている
  • ただし、6時間を超えている、またはなお疑いが残るなら、LPまたはCTAを追加する

雷鳴頭痛で見逃したくない所見

  • 項部硬直
  • 失神
  • 運動時発症
  • 神経症状
  • 血圧著明高値
  • 妊娠 / 産褥
  • 抗凝固薬

詳細 → くも膜下出血 / 脳静脈洞血栓症 / 頸動脈・椎骨動脈解離

2. 発熱・項部硬直・意識障害を伴う頭痛

  • 髄膜炎 / 脳炎を最優先
  • 迷っても、治療遅れが危険

この場でやること

  • 血液培養
  • 採血
  • LPを考える
  • ただし治療を遅らせない
  • 抗菌薬を早期開始
  • 細菌性髄膜炎が強く疑わしければデキサメタゾン併用を考える

重要

  • WHOは、急性細菌性髄膜炎が疑われたら経験的抗菌薬はできるだけ早く、理想的には1時間以内としている
  • LPは重要だが、検査待ちで治療を遅らせない
  • CT先行が必要なのは、WHOではGCS低下、局所神経徴候、脳神経障害、乳頭浮腫、新規けいれん、重度免疫不全などがある場合

詳細 → 髄膜炎・脳炎

3. 神経脱落症状 / 意識障害を伴う頭痛

  • 脳卒中、脳炎、脳腫瘍、脳膿瘍、高血圧性脳症、PRES を考える
  • 「頭痛だけ」に見えても、軽い失語・半盲・歩行失調・複視を落としやすい

この場でやること

  • 神経診察
  • 頭部画像
  • 必要なら血管画像
  • 血圧、血糖、けいれん既往、抗凝固薬を確認

画像の考え方

  • 神経症状を伴う頭痛や発熱 / 免疫不全 / 癌 / 50歳以上の新規頭痛などの危険所見があれば、ACRでも初期画像評価が適切とされる
  • 逆に、危険所見がない典型的な一次性頭痛では、ルーチン画像は通常不要

詳細 → 神経診察 / 脳卒中

4. 妊娠中 / 産褥期の頭痛

  • 妊娠関連頭痛は閾値を下げて二次性を探す
  • 特に注意
    • 子癇前症 / 子癇
    • 脳静脈洞血栓症
    • PRES
    • 可逆性脳血管攣縮症候群
    • 硬膜穿刺後頭痛

この場でやること

  • 血圧
  • 尿蛋白
  • 神経診察
  • 眼症状確認
  • 画像をためらわない

重要

  • ACRでは、妊娠中または周産期の新規頭痛 / パターン変化は画像評価を考える状況に入る
  • ACOGでは、産褥6週以内で頭痛 + 高血圧なら子癇前症を考える

詳細 → 妊娠・産褥の頭痛 / 脳静脈洞血栓症

5. 眼痛を伴う頭痛

  • 急性閉塞隅角緑内障をまず外さない
  • 眼科救急を疑う所見
    • 眼痛
    • 充血
    • 視力低下
    • 虹視
    • 悪心嘔吐
    • 片側優位

この場でやること

  • 視力
  • 瞳孔
  • 眼球運動
  • 角膜混濁の有無
  • 眼圧測定できれば確認
  • 眼科コンサルト

詳細 → 急性閉塞隅角緑内障

6. 50歳以上の新規頭痛

  • 巨細胞性動脈炎を必ず考える
  • 聞くこと
    • 側頭部痛
    • 顎跛行
    • 視覚症状
    • 頭皮圧痛
    • 発熱
    • PMR症状

この場でやること

  • ESR
  • CRP
  • 視覚症状があれば緊急対応
  • 強く疑えばステロイドを遅らせない

重要

  • GCAは50歳以上で起こる新規頭痛で疑うべき疾患で、診断は症状、側頭動脈生検、血管画像などで行う
  • 視力障害を残すことがあるため、疑いが高ければ迅速対応が必要

詳細 → 巨細胞性動脈炎

7. 頭蓋内圧異常を疑う頭痛

頭蓋内圧亢進を疑う

  • 朝に強い
  • 咳・いきみで悪化
  • 仰臥位で悪化
  • 嘔吐
  • 乳頭浮腫
  • 複視

頭蓋内圧低下を疑う

  • 立位で悪化、臥位で改善
  • 最近のLP、硬膜外、脊椎処置

この場でやること

  • 眼底
  • 神経診察
  • 画像
  • LP適応は慎重に判断

詳細 → 頭蓋内圧亢進 / 低髄液圧症候群

8. 典型的な一次性頭痛に見える場合

以下がそろえば一次性頭痛寄り

  • 以前と同じパターン
  • 神経所見なし
  • 発熱なし
  • 雷鳴頭痛ではない
  • 妊娠 / 産褥ではない
  • 50歳以降の新規発症ではない
  • 癌 / 免疫不全 / 抗凝固薬なし

よくあるもの

  • 片頭痛
  • 緊張型頭痛
  • 群発頭痛

救急での治療の考え方

  • 片頭痛ではオピオイドを第一選択にしない
  • AHS 2016では、急性片頭痛に対して
    • メトクロプラミド静注
    • プロクロルペラジン静注
    • スマトリプタン皮下注

      • が有力

  • デキサメタゾンは再燃予防に有用
  • モルヒネ / ヒドロモルフォンは第一選択として避ける方向

詳細 → 片頭痛 / 群発頭痛 / 頭痛の対症療法

危険所見

危険所見
  • 突然発症
  • 進行性悪化
  • 発熱
  • 項部硬直
  • 意識障害
  • 局所神経症状
  • けいれん
  • 乳頭浮腫
  • 妊娠 / 産褥
  • 50歳以上の新規発症
  • 癌
  • 免疫不全
  • 抗凝固薬
  • 外傷後
  • 体位で変化
  • 眼痛 / 視力低下

SNNOOP10は、二次性頭痛を拾うための危険所見リストとして広く使われている

最低限の診察

  • 意識レベル
  • 血圧
  • 体温
  • 項部硬直
  • 瞳孔
  • 眼位 / 眼球運動
  • 視力
  • 顔面左右差
  • 構音障害
  • 上下肢の麻痺
  • 感覚障害
  • 小脳失調
  • 歩行
  • 乳頭浮腫を可能なら確認

詳細 → 神経診察

検査の置き方

頭部単純CTを急ぐ

  • 雷鳴頭痛
  • 神経症状あり
  • 意識障害
  • 頭蓋内圧亢進疑い
  • 外傷後
  • 抗凝固薬
  • 発熱 + 中枢感染を否定しきれないがまず占拠性病変も気になる時

CTA / CTVを考える

  • SAH疑いでCT陰性だがまだ疑う
  • 解離疑い
  • CVST疑い
  • RCVS疑い

LPを考える

  • 髄膜炎 / 脳炎疑い
  • SAH疑いでCT後も未解決
  • ただし占拠性病変 / ヘルニアリスクには注意

救急外来での実際のまとめ

実戦用
  • 雷鳴頭痛 → SAH first
  • 発熱 / 項部硬直 → 髄膜炎 first
  • 神経症状 / 意識障害 → 脳卒中・脳炎・腫瘍 first
  • 妊娠 / 産褥 → 子癇前症 / CVST / PRES first
  • 眼痛 + 視力低下 → 緑内障 first
  • 50歳以上の新規頭痛 → GCA first
  • どれにも当てはまらず、いつもの頭痛なら一次性頭痛を考える
