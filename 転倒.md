#ER #救急 #転倒 #外傷 #頭部外傷 #骨折 #失神 #高齢者

> [!summary] このページの使い方
> 
> 1. **バイタル・意識・受傷機転** で緊急度を決める
> 2. **頭から足まで** 系統的に外傷を評価する
> 3. **頭部 CT が要るか** を判断する（抗凝固薬の有無が特に重要）
> 4. **なぜ転んだか** を考える（外傷だけ治して終わらせない）
> 5. **帰宅 or 入院** を決める

## 最初に決めること

- **意識は清明か**
- **頭を打ったか / 打った可能性があるか**
- **抗凝固薬 / 抗血小板薬を内服しているか**
- **どうやって転んだか**（受傷機転）— 立位からの転倒か、階段か、失神が先行したか
- **外傷（出血・変形・疼痛）はあるか**

> [!danger] すぐ動く
> 
> - **意識レベル低下（GCS ≤14）**
> - **瞳孔不同 / 新規の片麻痺 / けいれん**
> - **大量出血 / 止まらない出血**
> - **四肢の変形**（開放骨折・脱臼）
> - **頸部痛** → 頸椎保護
> - **ショック**
> - **失神が先行している** → 原因検索が必要

## 問診

- **いつ、どこで、どうやって転んだか**
- **目撃者はいるか**（失神の有無の判断に重要）
- **転ぶ前に症状はあったか**：失神・前失神・めまい・胸痛・動悸・脱力
- **頭を打ったか / わからないか**
- **受傷後の健忘はあるか**
- **内服薬**：特に **抗凝固薬・抗血小板薬・降圧薬・睡眠薬・血糖降下薬**
- **既往歴**：心疾患・不整脈・てんかん・骨粗鬆症・認知症

## 身体診察（頭から足まで）

### 頭部

- **頭皮の腫脹・裂創・圧痛**（後頭部は必ず見る）
- **Battle 徴候**（乳様突起の皮下出血 → 側頭骨骨折）
- **Raccoon eyes**（眼窩周囲の皮下出血 → 前頭蓋底骨折）
- **耳・鼻からの液体漏出**（髄液漏）
- **瞳孔**（左右差・対光反射）

### 顔面

- 変形・腫脹・圧痛・開口障害・歯牙損傷

### 頸椎

- **正中圧痛** → 頸椎損傷の可能性 → **固定して画像評価**
- 高齢者では軽微な受傷でも起こりうる — 痛みがあれば安易に否定しない

### 胸部・腹部

- **胸郭の圧痛**（肋骨骨折）
- **呼吸音の左右差**
- **腹部の圧痛・膨満**（抗凝固薬内服中は腹腔内出血も考慮）

### 骨盤・股関節

- **骨盤の圧迫痛** → 骨盤骨折
- **鼠径部・大転子部の圧痛** → 大腿骨近位部骨折
- **下肢の短縮・外旋** → **大腿骨近位部骨折を強く示唆**
- 股関節痛がある場合 → [[股関節痛]]（画像評価・整形外科コンサルトのフロー）

### 四肢

- **変形・腫脹・圧痛・可動域制限**
- **手関節の変形** → 橈骨遠位端骨折（手をついた場合）
- **足関節の腫脹** → 足関節骨折
- **荷重できるか** → 歩行を試せる状態なら確認

### 皮膚

- **擦過傷・皮下出血の分布** → 受傷機転の推定
- **頭部裂創** → 洗浄 + 縫合 or ステープラー
- **スキンテア** → 皮弁を戻して被覆

## 頭部 CT の判断

> [!danger] 抗凝固薬内服中の頭部打撲は迷ったら撮る
> 抗凝固薬内服者は**軽症頭部外傷でも頭蓋内出血のリスクが高い**。
> GCS 15 でも、頭部打撲（疑い含む）があれば **CT を撮る閾値は低くする**。

### 撮るべき場面

- **GCS < 15**
- **頭部打撲あり**（疑いを含む）
- **抗凝固薬 / 抗血小板薬を内服中**
- **65歳以上の頭部打撲**
- **受傷後の健忘**
- **嘔吐（2回以上）**
- **新規の神経所見**
- **受傷機転が不明**（目撃者なし → 頭部打撲を否定できない）

### 撮らない選択肢がありうる条件（Falls Decision Rule）

以下の**すべて**を満たす場合：

- 頭を打った形跡がない
- 転倒前後の健忘がない
- 神経学的診察で新規異常がない
- 抗凝固薬を内服していない

（感度 98.6%、NPV 99.8%）

### 抗凝固薬内服者の注意

- 初回 CT で出血あり → **数時間後に再撮影を考慮**（急速増大のリスク）
- 初回 CT 正常でも → **遅発性出血に注意**（24–72時間）
- **ワルファリン** → PT-INR 確認。出血あれば拮抗（ビタミンK ± PCC）
- **DOAC** → 施設の拮抗プロトコルに従う
- **出血あり** → **脳神経外科コンサルト**

## なぜ転んだか（原因検索）

**外傷の治療だけで終わらせない。** 特に高齢者では「なぜ転んだか」が重要。

まず **本人・目撃者の証言から失神を否定できるか** を確認する。
否定できない → **syncope および syncope mimic の検索**。

| 原因 | 確認すること |
|---|---|
| **つまずき / 滑り** | 環境要因。靴・段差・暗さ |
| **起立性低血圧** | 臥位→立位の血圧測定（≥20 mmHg 低下で陽性）。降圧薬・脱水 |
| **失神 / 前失神** | 目撃者の証言、前駆症状、ECG |
| **迷走神経反射** | 転倒時の状況（長時間立位、排尿後、食後、痛み刺激） |
| **不整脈** | ECG。徐脈・頻脈・pause |
| **低血糖** | 血糖測定。インスリン・SU 薬 |
| **てんかん** | 目撃者の証言、舌咬傷、尿失禁 |
| **薬剤性** | 睡眠薬・ベンゾ・降圧薬・オピオイド |
| **アルコール** | 飲酒歴、臭い、血中アルコール |

### 失神を疑うときの追加検索

**心電図で確認するもの**：

| 所見 | 疑う疾患 |
|---|---|
| VT / VF | 致死性不整脈 |
| AF（頻脈性） | 心原性塞栓・血行動態破綻 |
| III度 AVB / Mobitz II型 | 高度房室ブロック |
| SSS（洞不全症候群） | 徐脈性失神 |

**心エコーで確認するもの**（施行可能な場合）：

| 所見 | 疑う疾患 |
|---|---|
| low EF | 心不全による低灌流 |
| 壁運動異常 | ACS |
| HOCM（左室肥大 + SAM） | 肥大型心筋症による流出路閉塞 |
| 心嚢液 | 心タンポナーデ |
| 大動脈拡大 | 大動脈解離 |
| 腹部大動脈瘤 | AAA 破裂 |

**その他**：
- **PE** → 胸痛・SpO2 低下がないか
- **起立性低血圧** → 起立時血圧測定
- **迷走神経反射** → 転倒時の状況から判断

→ 何も見つからない場合 → **逆行性健忘** の可能性（頭部打撲で記憶が飛んでいるだけ）

> [!warning] 失神が先行している転倒
> 「転んだ」のではなく「倒れた」可能性がある。
> **目撃者がいない + 受傷機転が不明確** → 失神の可能性を考える。
> → **ECG は必ず取る**。心疾患の既往があれば循環器への相談も検討。
> — ESC 2018 失神ガイドライン

## 外傷評価

### ABCD 評価

| 項目 | 確認すること |
|---|---|
| **A（気道）** | 発声、口腔内異物、頸部腫脹、stridor |
| **B（呼吸）** | フレイルチェスト → 胸郭運動の左右差・胸部圧痛 / 緊張性気胸 → 気管偏位・頸静脈怒張・呼吸音減弱 |
| **C（循環）** | FAST → 大量血胸・腹腔内出血・心タンポナーデ |
| **D（意識）** | GCS、瞳孔、四肢の動き |

### 部位別の評価

- **頭部** → [[#頭部 CT の判断]] を参照
- **頸部** → 正中圧痛があれば頸椎損傷を考慮 → 固定
- **胸部・腹部** → 圧痛、呼吸音左右差、腹膜刺激症状
- **四肢** → 変形・腫脹・圧痛・可動域制限
- **股関節痛** → [[股関節痛]]（大腿骨骨折の評価フロー）

## 検査

必ずやるもの：
- **バイタルサイン**
- **画像**（受傷部位に応じて X 線 / CT）

状況に応じて追加：
- **血糖**（ふらつき・意識変容がある場合）
- **心電図**（失神・前失神・動悸が先行した場合、高齢者）
- **心エコー**（失神 + 心疾患の既往・リスクがある場合）
- **血液検査**（抗凝固薬内服中 → 凝固、高齢者 → 貧血・腎機能・電解質）
- **起立試験**（起立性低血圧を疑う場合）

## disposition

### 帰宅できる条件

- 意識清明、神経所見なし
- 頭部 CT が正常（撮影した場合）、または CT 不要と判断
- 骨折がない or 保存的に帰宅管理できる骨折
- 歩行できる / 日常生活が自力でできる
- 転倒原因が明らかで重篤でない
- 抗凝固薬内服中でも CT 正常 + 経過良好なら帰宅可（遅発性出血の説明が必要）

### 帰宅時の説明

- **頭を打った場合** → **頭部外傷の注意事項**（嘔吐・意識変容・強い頭痛が出たら再受診）を必ず説明
- **骨折なしでも痛みが続く場合** → **再受診して再評価**（不全骨折の可能性）
- **独居高齢者** → 帰宅後の見守りが確保できるか確認

### 入院 or 専門科コンサルト

- **頭蓋内出血** → **脳神経外科**
- **大腿骨近位部骨折** → **整形外科**（手術適応の判断）
- **頸椎損傷の疑い** → **整形外科 / 脊椎外科**
- **開放骨折 / 脱臼** → **整形外科**
- **失神の精査が必要** → 入院 or 循環器
- **歩行不能 / ADL が維持できない** → 入院
- **抗凝固薬内服中 + 頭部打撲** → 経過観察入院を検討

## 折りたたみ用メモ

### 見落としやすいもの

- **後頭部の打撲・裂創**（仰臥位のまま確認していない）
- **大腿骨近位部の不全骨折**（初回 X 線で見えない → 痛みが続けば CT / MRI）
- **頸椎損傷**（高齢者の軽微外傷でも起こりうる）
- **脊椎圧迫骨折**（背部痛として訴える）
- **失神が原因の転倒**（「つまずいた」で片付けている）
- **抗凝固薬の確認漏れ**

### やりがちなミス

- 「頭を打っていない」という本人の言葉だけで CT を省略する（目撃者がいなければ否定できない）
- 頭部だけ見て四肢・骨盤を診ない
- 外傷対応だけして転倒原因を考えない
- 高齢者の「歩ける」を過信して帰宅させる（翌日歩けなくなることがある）
- 抗凝固薬内服者の頭部外傷で帰宅時説明が不十分
