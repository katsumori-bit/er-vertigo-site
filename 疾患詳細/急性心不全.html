急性心不全

元: 臨床/ER/疾患詳細/急性心不全.md

#ER #救急 #呼吸困難 #急性心不全 #ADHF

このページの使い方
  1. 呼吸困難 から心不全を疑ってここに来た時点で、心電図・胸部X線・血液検査は済んでいる
  2. 心電図で 緊急の異常がないか を自分で確認する
  3. ルート・バルーン・CT・エコー準備 — 循環器が診療しやすい体制を整える
  4. 循環器ホットラインに連絡して引き継ぐ

前提

呼吸困難 のフローに沿って以下が済んでいる：

  • バイタル確認・モニター装着
  • 血液検査（血ガス・BNP・トロポニン・CRP・血算分画・生化）提出済み
  • 12誘導心電図 施行済み
  • 胸部X線 施行済み

まず心電図を読む

循環器に電話する前に、自分で拾わないといけない緊急所見がある。

心電図で見逃してはいけないもの
所見意味対応
ST 上昇STEMI → 心筋梗塞が心不全の原因 or 増悪因子緊急カテの判断 → 即循環器コール
wide QRS + 頻脈（規則的）VT（心室頻拍）不安定なら電気的除細動
完全房室ブロック（3度 AVB）高度徐脈 → 低心拍出経皮ペーシング準備 → 循環器
新規の高度伝導障害（交代性脚ブロック、二枝+Ⅰ度房室ブロック等）完全ブロックに移行するリスクモニター・循環器相談
AF rapid（HR >150）AF が心不全の引き金になっている循環器に伝える
著明な徐脈（HR <40）低心拍出アトロピン・ペーシング準備

これらがないことを確認する → 「心電図としては緊急対応は不要」と判断できる。

循環器が来るまでに整えること

薬剤投与（ラシックス・硝酸薬等）は循環器が来てから行う。 自分の仕事は循環器が来たらすぐ動ける体制を整えること。

ルート確保

  • 末梢静脈路を確保（循環器が薬剤をすぐ使えるように）
  • できれば太めの留置針（18G 以上が望ましい）

尿道バルーン留置

  • 尿量測定ができるように尿道カテーテルを留置
  • ラシックス投与後の尿量評価は循環器にとって重要な情報になる

CT

  • 胸部 CT を撮っておく
  • 循環器が来てから「CT 撮ってきて」となると時間をロスする
  • 肺塞栓の除外、大動脈解離の確認、肺野の評価にも有用

エコーの準備

  • エコー機器をベッドサイドに用意しておく
  • 循環器が来たらすぐ心エコーを当てられる状態にする

モニタリング

  • バイタルモニター継続
  • SpO2・呼吸数・血圧を定期的に記録
  • 呼吸仕事量の変化を観察
こうなったら循環器を待たずに動く
  • VT / VF → 電気的除細動
  • 完全房室ブロック + 不安定 → 経皮ペーシング
  • 低血圧 + 冷たい + 意識障害 → 心原性ショック → ICU・上級医すぐ
  • 呼吸が止まりそう → 挿管の準備

循環器ホットラインに伝えること

  • 年齢・性別・主訴
  • バイタル（特に血圧と SpO2）
  • 心電図の所見（STEMI? 不整脈? 伝導障害?）
  • BNP・トロポニンの結果（出ていれば）
  • 胸部X線の所見（肺水腫の程度）
  • 現在の状況（酸素投与量、ルート確保済み、バルーン留置済み、CT 撮影済み、エコー準備済み）
  • 既往歴（心不全の既往、弁膜症、虚血性心疾患、ペースメーカー等）
モルヒネは使わない

ESC 2021 で Class III（用いない）。急性肺水腫でも原則使わない。

知っておくと引き継ぎがスムーズな枠組み

重症度 → 詳細は 急性心不全の重症度評価

  • うっ血のみ（Warm-Wet）（最も多い、約70%）→ 利尿薬で対応
  • うっ血 + 呼吸不全 → NIV が必要
  • 低灌流あり（Cold-Wet）（冷たい・尿が出ない・意識障害）→ 即 ICU — 最も予後が悪い

増悪因子（CHAMPIT）

Coronary急性冠症候群
Hypertension高血圧緊急症
Arrhythmia不整脈（AF rapid、VT 等）
Mechanical急性弁膜症
PE肺塞栓
Infection感染（肺炎等）
Tamponade心タンポナーデ
特に心筋梗塞だけは自分で拾う

心筋梗塞が増悪因子であるときは対応が根本的に変わる（緊急カテ）。 だから心電図で ST 上昇がないかだけは必ず自分で確認する。

入院時オーダー・病棟指示の枠（ADHF）

救急後の入院オーダーは循環器主治医・HF チームが主導する。ここでは ESC 2021／HFA 科学声明で繰り返される項目のチェックリストとして整理する（薬剤の具体用量は施設プロトコル）。

モニタリング

  • バイタル・SpO2・尿量（バルーンカテで時間尿量）
  • 体重・出入量（うっ血管理）
  • 心電図モニター（不整脈・虚血）
  • 電解質・腎機能・BNP/NT-proBNP（日数に応じて）

うっ血（Warm-Wet が典型）

  • 利尿薬（ループ利尿薬）— 電解質・腎機能・血圧とセットで調整（ESC 2021）
  • 血管拡張薬：高血圧うっ血で血圧が許容なら検討（血圧低下に注意）

低灌流・心原性ショック寄り（Cold）

  • 強心薬・血管収縮薬— ICU での使用が多い
  • 機械的循環補助の評価は専門医（ESC 2021・ACVC 2022）

全例で意識すること

  • 誘因の評価（CHAMPIT）：感染・ACS・不整脈・PE 等
  • GDMT の再開・導入：退院前にエビデンスのある内服を可能な範囲で開始することが推奨される（ESC 2021 Class I）
  • 患者教育：体重・症状悪化時の対応、塩分、服薬

退院前（病棟）

  • うっ血の残存がないか再評価（残存うっ血は再入院リスク）
  • 口服薬の処方を確定（ACEI/ARB/ARNI、β遮断薬、MRA、SGLT2 阻害薬など、禁忌と血圧・腎機能で調整）

外来フォロー・退院後（ADHF）

  • 退院後早期訪問：ESC 2021 では、ADHF 入院後退院後 1–2 週以内の早期フォローをClass Iで推奨（うっ血の再評価、薬剤の漸増・不耐性の確認）
  • 多職種・HF クリニック：可能なら専門外来でGDMT の段階的増量（HFA 2023 科学声明でも退院前後の一貫した最適化を強調）
  • 自己管理：日々の体重、症状悪化時の行動プラン
