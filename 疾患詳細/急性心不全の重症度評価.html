#心不全 #重症度 #Nohria-Stevenson #Forrester #うっ血 #低灌流

このページの使い方

急性心不全の重症度を うっ血（wet/dry） と 低灌流（warm/cold） の2軸で評価する。 ベッドサイドの身体所見だけで分類でき、治療方針と予後の見通しに直結する。

基本の考え方

急性心不全の重症度は 2つの軸 で決まる：

  1. うっ血があるか（wet） — 肺や全身に水が溜まっているか
  2. 低灌流があるか（cold） — 心臓から十分な血液が送れているか

この2つの組み合わせで 4つのプロファイル に分かれ、それぞれ治療と予後が違う。

うっ血の所見（wet かどうか）

所見見方
起坐呼吸仰臥位で苦しくなる → 座ると楽になる
頸静脈怒張（JVP 上昇）45° 座位で内頸静脈が張っている
湿性ラ音聴診で両側の crackles
下腿浮腫両側の圧痕性浮腫
肝頸静脈逆流肝臓を圧迫すると頸静脈が膨らむ
胸部X線で肺うっ血血管陰影増強、胸水、Kerley B line
体重増加数日で 2–3 kg 以上増えている
実用的には

起坐呼吸 + 頸静脈怒張 + 浮腫 + ラ音 のうち複数あれば wet と判断してよい。

低灌流の所見（cold かどうか）

所見見方
四肢冷感手足が冷たい、しめっている
意識障害・傾眠ぼんやりしている、反応が鈍い
乏尿尿量が減っている（< 0.5 mL/kg/h）
脈圧低下（収縮期 − 拡張期）÷ 収縮期 < 25%
低Na血症Na ≤ 135 mEq/L
腎機能悪化Cre 上昇
乳酸上昇血ガスで lactate 上昇
低灌流 = 心原性ショックの入口

冷たい + 意識障害 + 尿が出ない → 心原性ショックとして ICU・上級医にすぐ連絡。

Nohria-Stevenson 分類（身体所見で分類）

臨床所見だけで評価できる。Swan-Ganz は不要。

うっ血なし（dry）うっ血あり（wet）
低灌流なし（warm）Profile A（Warm-Dry）Profile B（Warm-Wet）
低灌流あり（cold）Profile L（Cold-Dry）Profile C（Cold-Wet）

各プロファイルの意味と対応

Profile A（Warm-Dry）

  • うっ血も低灌流もない → 心不全は代償されている
  • 急性期の治療は軽くてすむことが多い
  • 予後は最も良い

Profile B（Warm-Wet） — 最も多い（約 70%）

  • うっ血はあるが灌流は保たれている
  • 利尿薬が主体（± 血管拡張薬）
  • 血圧が保たれていれば治療反応は良い

Profile C（Cold-Wet） — 最も危険

  • うっ血 + 低灌流 → 心原性ショックに近い
  • 利尿薬だけでは足りない → 強心薬・昇圧薬・機械的補助の検討
  • 院内死亡率が最も高い（約 12%、Warm-Dry の約 6 倍）
  • ICU 管理が必要

Profile L（Cold-Dry）

  • まれ。低灌流があるがうっ血がない
  • 輸液で前負荷を上げることが必要な場合がある
  • 終末期心不全でも見られる

Forrester 分類（観血的 — 参考）

Swan-Ganz カテーテルで 心係数（CI） と 肺動脈楔入圧（PCWP） を測定して分類する。救急で自分が使うことはないが、循環器との会話で出てくることがある。

SubsetCIPCWP臨床像死亡率
I≥ 2.2< 18うっ血なし・灌流保たれる2.2%
II≥ 2.2≥ 18うっ血あり・灌流保たれる（= Warm-Wet）10.1%
III< 2.2< 18うっ血なし・低灌流（= Cold-Dry）22.4%
IV< 2.2≥ 18うっ血 + 低灌流（= Cold-Wet）55.5%
  • CI = 心係数（L/min/m²）。基準値は 2.2
  • PCWP = 肺動脈楔入圧（mmHg）。基準値は 18

Killip 分類（急性心筋梗塞に伴う心不全）

ACS の文脈で使われる。

Class所見死亡率（目安）
I心不全の徴候なし6%
II軽度心不全（ラ音 < 50%、III音、頸静脈怒張）17%
III肺水腫38%
IV心原性ショック81%

実際の使い方まとめ

ベッドサイドで最低限やること
  1. 触る — 四肢は温かいか冷たいか（warm / cold）
  2. 聴く — 肺にラ音があるか（wet / dry）
  3. 見る — 頸静脈は張っているか、浮腫はあるか
  4. バイタル — 血圧（脈圧）、尿量、意識

→ Warm-Wet が大多数。Cold が入ったら重症 → 循環器に「低灌流っぽい」と伝える。

入院オーダー・フォローへの反映（プロファイル別）

Nohria-Stevenson 分類はオーダー記載時の「一行要約」にも使える。詳細な薬剤・用量は 急性心不全 と ESC 2021 に従い、主治医指示とする。

プロファイル入院管理の方向性（概念）
Warm-Wet（B）利尿中心、必要に応じて血管拡張。ICU は通常不要。退院前に GDMT 再開を優先
Cold-Wet（C）心原性ショックの領域 → ICU、強心・昇圧・機械補助の検討。尿量・乳酸・末梢灌流を密モニタ
Cold-Dry（L）低灌流だがうっ血が目立たない → 前負荷・原因（虚脱、過利尿、ブロック等）の精査。輸液は慎重に（心不全では過剰輸液が有害になりうる）
Warm-Dry（A）増悪は軽度〜代償域。外来調整・短期観察が主になることも

外来フォロー：プロファイルに関わらず、ADHF 入院後は 退院後 1–2 週の早期再診を推奨するのが ESC 2021 の流れ（急性心不全 の「外来フォロー」参照）。
