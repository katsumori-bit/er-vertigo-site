#便秘 #下剤 #病棟 #薬剤 #OIC #高齢者

> [!summary] このページの使い方
> 
> 1. **便秘のタイプ**（排便回数減少型 vs 排便困難型）を見極める
> 2. **浸透圧性下剤を基本**にし、不十分なら次の段を考える
> 3. **刺激性下剤は屯用**にとどめる
> 4. **背景**（腎機能・オピオイド・閉塞の有無）で禁忌を確認する

## 下剤選択の基本方針

便通異常症診療ガイドライン 2023（日本消化器病学会）および AGA/ACG 2023 共同ガイドラインに基づく段階的アプローチ。

| 段階 | 内容 |
| --- | --- |
| **第1段階** | 生活習慣改善（水分・食物繊維・運動） |
| **第2段階** | **浸透圧性下剤**（酸化マグネシウム、モビコール、ラクツロース） |
| **第3段階** | **上皮機能変容薬**（リナクロチド、ルビプロストン）or **胆汁酸トランスポーター阻害薬**（エロビキシバット） |
| **レスキュー** | **刺激性下剤**（センノシド、ピコスルファート）は屯用。**外用**（坐剤、浣腸）も屯用 |
| **OIC 専用** | **ナルデメジン**（従来下剤で不十分な場合） |

> [!warning] 最も大事な原則
> **刺激性下剤を定期処方にしない。** 耐性・習慣性・大腸メラノーシスのリスクがある。
> 異常時指示から定期に移行しがちなので注意する。

## 浸透圧性下剤（第一選択）

### 酸化マグネシウム

| 項目 | 内容 |
| --- | --- |
| **用量** | 990–2000 mg/日 分2–3 |
| **特徴** | 安価、使い慣れている、即効性あり |
| **注意** | **腎機能低下時は禁忌**（高 Mg 血症のリスク） |

> [!danger] 高マグネシウム血症
> PMDA は死亡例を含む重篤な高 Mg 血症を複数報告している（2015年注意喚起）。
> **リスク因子**：eGFR ≤43 mL/min、BUN ≥22 mg/dL、1日投与量 ≥1000 mg、高齢者、便塞栓。
> 長期投与時は**定期的に血清 Mg を測定**する。
> 嘔吐・徐脈・筋力低下・傾眠が出たら**即中止**。

### モビコール（マクロゴール 4000 / PEG）

| 項目 | 内容 |
| --- | --- |
| **用量** | LD 2包/日（1回量、60 mL の水に溶解）→ 最大 6包/日。HD 1包/日 → 最大 3包/日 |
| **増量** | 2日以上あけて、LD は2包ずつ、HD は1包ずつ |
| **特徴** | 耐性なし、習慣性なし、腎機能に左右されにくい |
| **注意** | 水に溶かす手間がある。腹部膨満・下痢 |

AGA/ACG 2023 では PEG を**強い推奨**としている。酸化マグネシウムが使えない腎機能低下患者では**第一選択**になる。

### ラクツロース

| 項目 | 内容 |
| --- | --- |
| **用量** | 24–48 mL/日 分2–3（慢性便秘症適応は 2018年〜） |
| **特徴** | 肝性脳症にも使用される。腎機能に影響しにくい |
| **注意** | 腹部膨満・ガス。糖尿病では血糖に注意 |

## 上皮機能変容薬（第二選択）

浸透圧性下剤で効果不十分な場合に追加 or 切り替え。

### リナクロチド（リンゼス）

| 項目 | 内容 |
| --- | --- |
| **用量** | **0.5 mg** 1日1回 食前。0.25 mg に減量可 |
| **作用** | GC-C 受容体作動 → 腸液分泌促進 + 大腸痛覚過敏改善 |
| **特徴** | **腹痛を伴う便秘**に向いている（便秘型 IBS にも適応） |
| **副作用** | 下痢（13%）。重度の下痢が出たら減量 or 中止 |

### ルビプロストン（アミティーザ）

| 項目 | 内容 |
| --- | --- |
| **用量** | **24 μg** 1日2回 朝夕食後 |
| **作用** | ClC-2 チャネル活性化 → 小腸への水分分泌促進 |
| **特徴** | 長期使用でも耐性がつきにくい |
| **副作用** | 悪心（23%）、下痢（30%）。若年女性で悪心が出やすい。**妊婦禁忌** |

---

## 胆汁酸トランスポーター阻害薬

### エロビキシバット（グーフィス）

| 項目 | 内容 |
| --- | --- |
| **用量** | **10 mg** 1日1回 食前。最大 15 mg |
| **作用** | 回腸の胆汁酸再吸収を阻害 → 大腸に胆汁酸が流入 → 水分分泌・蠕動促進 |
| **特徴** | 効果発現が比較的早い（約5時間で50%が排便） |
| **副作用** | 腹痛（19%）、下痢（16%）。3剤中で腹痛が最も出やすい |

---

## 刺激性下剤（レスキュー・屯用）

> [!warning] 定期処方にしない
> 耐性形成と大腸神経障害のリスクがある。
> 屯用 or 短期使用（4週間以内）が原則（AGA/ACG 2023：強い推奨）。

### センノシド（プルゼニド）

| 項目 | 内容 |
| --- | --- |
| **用量** | 12–24 mg 眠前（高度便秘時は 48 mg まで） |
| **特徴** | 6–12時間で効果発現。安価 |
| **注意** | **連用で耐性**。大腸メラノーシス。妊婦には慎重投与 |

### ピコスルファートナトリウム（ラキソベロン）

| 項目 | 内容 |
| --- | --- |
| **用量** | 10–15滴 眠前 |
| **特徴** | 液剤で**微調整しやすい**。病棟の異常時指示に向いている |
| **注意** | 刺激性なので連用は避ける |

---

## 外用薬（レスキュー）

### 新レシカルボン坐剤

| 項目 | 内容 |
| --- | --- |
| **用量** | 1個 挿肛 |
| **作用** | 炭酸ガス発生 → 直腸を刺激して排便反射を誘発 |
| **特徴** | 直腸に便塊がある場合に有効 |

### グリセリン浣腸

| 項目 | 内容 |
| --- | --- |
| **用量** | 60 mL |
| **注意** | 直腸穿孔のリスクがある（特に高齢者・直腸術後）。挿入は浅めに、無理に押し込まない |

## オピオイド誘発性便秘（OIC）

オピオイド使用中の便秘は**通常の便秘と機序が異なる**（腸管の μ受容体を介した蠕動抑制 + 分泌抑制）。

### 対応の順番

1. **予防**：オピオイド開始時に下剤を併用する（酸化マグネシウム or センノシド屯用）
2. **従来下剤で不十分** → **ナルデメジン（スインプロイク）** を追加

### ナルデメジン（スインプロイク）

| 項目 | 内容 |
| --- | --- |
| **用量** | **0.2 mg** 1日1回（食事に関係なく） |
| **作用** | 末梢性 μ受容体拮抗薬（PAMORA）。腸管のオピオイド作用を選択的にブロック |
| **特徴** | **鎮痛効果を減弱させない**（血液脳関門を通過しにくい）。オピオイドの用量調整不要 |
| **副作用** | 下痢、腹痛（軽度〜中等度が多い） |
| **禁忌** | **消化管閉塞が疑われる場合**（穿孔リスク）。閉塞を除外してから開始 |

AGA は従来下剤で改善しない OIC に対してナルデメジンを**強い推奨（エビデンスの質：高）**としている。

## 特殊な状況での選択

### 腎機能低下

- **酸化マグネシウムは避ける** → **モビコール or ラクツロース**
- 透析患者でも PEG は使用可能

### 高齢者

- 酸化マグネシウムは慎重（定期的な血清 Mg 測定が必要）
- **モビコール**が安全性で優れる
- 刺激性下剤の連用は避ける
- 直腸指診で**宿便の有無**を確認してから方針を決める

### 術後

- 麻痺性イレウスの時期は下剤を使わない
- 排ガスが確認されてから下剤を検討
- **閉塞が疑わしいときは下剤で押さない**

### 妊婦

- **ルビプロストン（アミティーザ）は禁忌**
- 酸化マグネシウム、ラクツロースは使用可能
- センノシドは慎重投与

### 担癌患者

- オピオイドによる OIC が多い → ナルデメジンを早めに検討
- 悪性腸閉塞を常に除外してから下剤を出す

## 入院時オーダー・病棟指示の枠（便秘・OIC）

便通異常症ガイドライン 2023・AGA/ACG 2023 の**段階的アプローチ**を、オーダー記載用に整理する。

### 初期評価（必須）

- **腸閉塞・腹膜炎の除外**（腹部診察・必要なら画像）— **閉塞が疑わしいときは下剤で押さない**
- **腎機能**（酸化マグネシウムの可否）、**オピオイド使用**（OIC ならナルデメジンを頭に置く）
- **直腸指診**：宿便・便塞栓の有無（高齢・長期卧床）

### 定期オーダーの例（病棟）

1. **第 1 選択**：**浸透圧性下剤**（酸化マグネシウム — 腎機能 OK 時、または PEG／ラクツロース）
2. **不十分なら**：上皮機能変容薬・エロビキシバット等を**追加**（本表の段階に従う）
3. **レスキュー**：**センノシド／ピコスルファートは屯用**（定期オーダーにしない — AGA/ACG 強い推奨）
4. **OIC**：従来下剤で不十分なら**ナルデメジン 0.2 mg/日**（**閉塞除外後**）

### モニタリング

- **排便日時・性状**の記録
- **酸化マグネシウム長期使用時**：**血清 Mg**（PMDA 安全性情報）
- **下痢・腹痛**で分泌促進薬・PAMORA を調整

---

## 外来フォロー・退院後

- **退院処方**：急性期に効いた段階を**そのまま持ち帰す**。刺激薬だけを長期定期にしない
- **再診**：用量調整は**2–4 週**を目安に（効果不十分・副作用評価）。生活習慣指導を継続
- **オピオイド継続例**：ナルデメジンは**鎮痛薬を変えずに**便秘のみ改善を目指す（AGA OIC ガイドライン）

---

## 実践的な選択フロー

```
便秘の訴え
  │
  ├─ 閉塞・腹膜刺激あり → 下剤で押さない。画像評価・外科相談
  │
  ├─ 腎機能低下あり → モビコール or ラクツロース
  │
  ├─ オピオイド使用中 → まず従来下剤 → 不十分ならナルデメジン追加
  │
  ├─ 腹痛を伴う便秘 → リナクロチド（リンゼス）を考慮
  │
  └─ 上記以外の一般的な便秘
       ├─ まず酸化マグネシウム or モビコール（定期）
       ├─ 不十分 → 上皮機能変容薬 or エロビキシバット追加
       └─ レスキュー → センノシド or ピコスルファート（屯用）
                       坐剤 or 浣腸（直腸に便塊あり）
```

- **便通異常症診療ガイドライン 2023**：日本消化器病学会.（**下剤の段階的使用、刺激性下剤は屯用が原則、推奨度と分類**）

- **AGA 2018（OIC）**：Crockett SD ほか. AGA Institute Clinical Guidelines Committee Guideline on the Medical Management of Opioid-Induced Constipation. *Gastroenterology.* 2019.（**ナルデメジンを強い推奨・高いエビデンス**）
- **PMDA 安全性情報**：酸化マグネシウムによる高マグネシウム血症について. 2015.（**死亡例を含む報告、リスク因子、血清 Mg 測定の推奨**）
