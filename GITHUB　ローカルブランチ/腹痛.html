  5. 診断がついたら疾患ページへ → 胆嚢炎・胆管炎 / 膵炎 / 虫垂炎 / 憩室炎 / 腸閉塞 / 腸間膜虚血 / AAA / 異所性妊娠 / 尿管結石

腹痛診療の目的は、最初から診断名を当てることではない。まずやることは、今すぐ死にうる腹痛を拾うこと、緊急手術・IVR・専門科コールが必要な腹痛を拾うこと、帰してはいけない腹痛を拾うこと、残った患者では部位・発症様式・随伴症状から鑑別を絞ることである。

急性腹痛は ED 受診の約 10%、150 以上の病因があるが上位 20 疾患で 70% をカバーする（AAFP 2023）。

0〜5分でやること

診断名を考えすぎず、何を見て・何を聞いて・何を押して・誰を呼ぶかの順で進める。

0. 見た瞬間（〜30秒）：重症感

まず患者を見て、ショック・敗血症・腹膜炎・血管イベントを頭に置く。

  • 顔色不良、冷汗、末梢冷感、mottling
  • 意識変容、苦悶様表情、呼吸が荒い
  • 体動困難、歩くと響く、ベッド上でじっとしている／転げ回っている
  • 吐血・下血・黒色便の訴えがあれば問診を伸ばさず体制を組む

これらが強いときは、問診を長く続けず、バイタル・ルート・採血・鎮痛・上級医・外科と並行で進める。

1. 〜1分：バイタルと腹膜刺激

血圧・脈拍・SpO2・体温・意識を確認しながら、腹部を軽く触れる。ここでの目的は確定診断ではなく、今すぐ外科／上級医を呼ぶ患者かを決めること。

以下のいずれかがあれば不安定な腹痛として扱う。

  • ショック：低血圧、頻脈、冷汗、末梢冷感、意識変容
  • 敗血症疑い：発熱または低体温、頻脈、頻呼吸、mottling
  • 腹膜炎疑い：板状硬、反跳痛、体動で響く腹痛
  • 血管イベント疑い：突然発症、発症時最強痛、痛みの割に腹部所見が乏しい
  • 消化管穿孔疑い：突然の強い腹痛＋腹膜刺激、free air を疑う状況
  • 絞扼・閉塞疑い：嘔吐、膨満、排ガス排便停止、開腹歴、ヘルニア

この時点で不安定なら、モニター・ルート・採血（血ガス・乳酸）・輸液・鎮痛・制吐・外科／上級医・造影 CTを同時並行で進める（詳細は下記「まず打つ手」）。

2. 〜3分：病歴はまず 6 項目だけ

最初から全部聞かない。まず次だけを取る。

  1. いつから？
  2. どこが痛い？（本人の指さし＝罹患部位とは限らない→全身を触る）
  3. 痛みの始まり方（下記 sudden / acute / gradually）
  4. 持続痛か、間欠痛か、発作性か
  5. 嘔吐・下痢はあるか（性状まで）
  6. 発熱はあるか（無熱でも油断しない）

この 6 つで緊急疾患の方向性をかなり絞れる。

3. 始まり方は必ず 3 つに分ける

「急に痛くなりましたか？」だけでは sudden と acute を区別できない。カルテに「急に」とだけ書かない。

発症様式意味疑うこと
sudden onsetある一瞬を境に最強痛血管疾患、穿孔、破裂、捻転など
acute onset数分〜十数分で最強痛膵炎、胆嚢炎、胆石発作など
gradually onset数十分〜数時間で増悪虫垂炎、憩室炎、腸炎、腸閉塞など
聞き方の例

「まったく痛くなかったのに、ある瞬間にスイッチが入ったように痛くなりましたか？」
「おかしいなと思ってから、数分〜10 分くらいかけてだんだん強くなりましたか？」

sudden なら、まず血管・穿孔・破裂・捻転の枠で動く。acute では心窩部なら穿孔と膵炎の枝が重要になる。

4. 持続痛か間欠痛か

「ずっと痛いですか？」だけでは不十分（間欠でもイエスになりやすい）。

聞き方の例

いま強い痛みがあるとき：「今より少し楽な時間はありましたか？」
いま落ち着いているとき：「今より強い痛みが波のように来る時間はありましたか？」

痛みの型考えること
持続痛炎症、穿孔、虚血、膵炎、胆嚢炎など
間欠痛腸閉塞、腸炎、虫垂炎初期など（完全に痛くない時間があるなら間欠として扱う）
発作性胆石・尿管結石、消化性潰瘍、abdominal angina など

5. 最初に 5 パターンへ振り分ける

A. 不安定・腹膜刺激あり → 外科腹症として動く（ルート・採血・血ガス・乳酸・輸液・鎮痛・抗菌薬を考慮・外科・造影 CT）。
B. sudden onset／発症時最強痛 → AAA、解離、腸間膜虚血、穿孔、異所性妊娠破裂、卵巣捻転、尿管結石、胆石発作など。痛みの割に所見が乏しければ腸間膜虚血を疑い CTA（「まず打つ手」参照）。
C. 嘔吐・膨満・排ガス排便停止 → 腸閉塞。開腹歴・ヘルニア・担癌・オピオイド・腸雑音・CT（transition、closed loop、虚血所見）。絞扼が否定できなければ外科を早める。
D. 発熱＋局在痛 → 右下：虫垂炎、左下：憩室炎、右季肋：胆嚢炎・胆管炎、心窩：膵炎・穿孔・ACS、下腹部：婦人科・尿路感染・PID。局在があるときは画像で原因を確認。
E. 下痢主体で全身状態良好 → 腸炎も考えるが決めつけない。血便・高熱・強い腹痛・高齢・免疫不全・脱水・腹膜刺激・少量粘液便のみなどがあれば単純胃腸炎として帰さない。

High risk patients

所見が軽くても、低い閾値で検査・画像・相談を考える。

  • 高齢、透析、Af、心血管疾患・PAD、抗凝固薬、免疫抑制、担癌、糖尿病
  • 妊娠可能年齢、開腹歴、オピオイド使用
特に危険度を上げるフレーズ

透析患者の腹痛　Af 患者の腹痛　高齢者の突然の腹痛　痛みの割に腹部所見が乏しい腹痛

カルテで先に拾う（到着前の5つ）

開腹歴・担癌・オピオイド・抗凝固薬・心房細動

  • 開腹歴 → 癒着性腸閉塞
  • 担癌 → 悪性腸閉塞、穿孔、感染、治療関連有害事象
  • オピオイド → 便秘、麻痺性イレウス
  • 抗凝固薬 → 出血、腹壁血腫
  • Af / 血管リスク → 腸間膜虚血
  • 胆石・尿管結石歴 → 再発
  • 婦人科既往 → 異所性妊娠、卵巣捻転、PID

最優先に外す急性腹症

  • 絞扼性腸閉塞・腸間膜虚血・消化管穿孔・AAA
  • 重症胆管炎 / 重症胆嚢炎・重症膵炎
  • 異所性妊娠・卵巣捻転・嵌頓 / 絞扼ヘルニア
迷ったらこの 4 つを先に外す

AAA・腸間膜虚血・消化管穿孔・異所性妊娠

まず打つ手

不安定・腹膜炎が疑われるときは問診を長く続けない。モニター・ルート・採血・輸液・鎮痛・外科／上級医・画像を並行で回す。

1. 不安定 / 腹膜刺激あり

  • ルート → 採血（乳酸含む）→ 血ガス
  • 輸液・鎮痛・制吐、敗血症像なら早期抗菌薬（源が絞れたら疾患ページで）
  • 外科コール + 造影 CT を急ぐ（AAA が強ければ POCUS 先行も可）

2. 局在あり、外科腹症を否定できない

  • 造影 CT を低い閾値で
  • 例外：右上腹部痛 → US 先行、妊娠可能年齢の骨盤痛 → US 先行、典型的尿管結石 → 単純 CT

3. 痛みの割に所見が乏しい

  • 腸間膜虚血を疑う。乳酸正常でも否定しない。CTA を考える

まず取るべき最小情報

入口の病歴 6 項目に対応するチェックリスト。取れたら枝を決めて画像・検査へ。

  1. いつから（発症早期来院は手術疾患を意識、数日からは余裕も高齢・認知・麻痺は例外）
  2. どこが：心窩 / 右上 / 右下 / 左下 / 下腹部 / びまん性・背部痛のみでも腹痛を聞く。本人の指さしだけで診察を終えない（虫垂などでズレる）
  3. 始まり方（0〜5分 §3）：sudden（一瞬で最強）／acute（数分〜十数分で最強）／gradually（数十分〜時間で増悪）—「急に」だけで書かない
  4. 持続 / 間欠 / 発作性：疝痛、移動痛。間欠は楽な瞬間の有無で確認（0〜5分 §4）
  5. 嘔吐・下痢：悪心のみ／食物残渣様／水様・腸液様など。性状・回数・血便
  6. 発熱：無熱でも穿孔・壊死・敗血症では低体温があり得る
  • 随伴の追加：便秘、胸痛、失神、排尿症状
  • 女性：最終月経、妊娠可能性、不正出血
  • 背景：高齢、手術歴、血管リスク、免疫不全、抗凝固薬、胆石歴、尿路結石歴

胆管炎を疑ったら — Charcot / Reynolds / TG18
  • Charcot の三徴：発熱・黄疸・右上腹部痛（揃わなくても胆管炎はある）
  • Reynolds の五徴：上記 + ショック + 意識障害 → 重症胆管炎として即動く（CT を待たず消化器コール + 抗菌薬開始）
  • TG18 基準・重症度・抗菌薬選択・ドレナージタイミング → 胆嚢炎・胆管炎 / 胆管結石
妊娠可能年齢の下腹部痛

妊娠反応を飛ばさない。hCG 値だけで異所性妊娠は除外できない（ACOG / ACEP）。

検査の考え方

血液・尿・心電図

  • 採血：CBC、電解質 / 腎機能、肝胆道系 + T-Bil、CRP、リパーゼ、乳酸（重症感・虚血疑い）、トロポニン（心窩部痛・高齢・冠リスク）
  • 尿定性、妊娠反応（妊娠可能年齢で原則全例）
  • ECG：心窩部痛、高齢、冠リスクあり

画像選択の基本

場面初手
非局在性 / びまん性 / 外科腹症が気になる造影 CT
右上腹部痛超音波
妊娠可能年齢の骨盤痛骨盤超音波
尿管結石が強く疑わしい単純 CT
高齢 + 低血圧 + 腹痛 / 背部痛AAA を POCUS or CT で除外

ACR は非局在性急性腹痛で造影 CT を通常適切、腹部単純 X 線は「管理方針を変えないことが多い」としている。

画像を読むときの順番

手術・IVR・緊急コンサルトに直結する所見を先に拾う：

  1. 穿孔：free air、腹水、限局炎症
  2. 血管：AAA、解離、SMA / SMV 閉塞・血栓
  3. 閉塞：拡張腸管、transition point、closed loop
  4. 絞扼 / 虚血：腸管壁造影不良（特異度 95%）、腸間膜浮腫、腹水、pneumatosis、portal venous gas、whirl sign
  5. 炎症：虫垂、憩室、胆嚢、膵周囲
  6. 骨盤：卵巣腫大、骨盤液体貯留

閉塞 + 腹膜刺激 / 絞扼徴候があれば非手術に固執しない（WSES SBO）。

初期対応（診断がつくまで）

診断ごとの詳細処方は疾患ページへ

抗菌薬レジメン・膵炎輸液・胆管炎ドレナージ等は 胆嚢炎・胆管炎 / 膵炎 / 虫垂炎 / 憩室炎 / 腸閉塞 で解説。

鎮痛（診断前に投与してよい、推奨度 A）

  • アセトアミノフェン IV（アセリオ）1000 mg を 15 分点滴
    • 体重 50 kg 未満は 15 mg/kg、投与間隔 4–6 h、1 日 4000 mg 上限
  • ブチルスコポラミン（ブスコパン）20 mg IV/IM：疝痛（胆石・尿管結石）に。持続痛には効果乏しい。緑内障・前立腺肥大・頻脈性不整脈・麻痺性イレウス禁忌
  • NSAIDs（フルルビプロフェン IV 50 mg、ジクロフェナク坐 25–50 mg）：尿管結石・胆石疝痛に有用。腎・脱水・潰瘍・出血リスクでは慎重。STEC 疑い・脱水の下痢では避ける（→ 下痢）
痛みを後回しにしない

早期鎮痛は診断精度を損なわない（急性腹症診療ガイドライン 2015、推奨度 A）。 アセリオ IV → 疝痛なら NSAIDs or ブスコパン → 効果不十分なら外科コンサルトを早める。

制吐

  • メトクロプラミド 10 mg IV or オンダンセトロン 4 mg IV
  • イレウス疑いでは制吐しつつも原因評価と減圧を遅らせない
  • 詳細 → 嘔気・嘔吐

輸液

  • 嘔吐・脱水・敗血症・膵炎・閉塞では早めに補正（細胞外液 500–1000 mL ボーラス）
  • 膵炎の輸液戦略（moderately aggressive、乳酸リンゲル） → 膵炎

抗菌薬（原則）

  • すべての腹痛に入れない。source を絞ってから
  • 入れる場面：胆管炎、胆嚢炎、穿孔疑い、腹腔内感染、壊死性腸管疑い、complicated appendicitis、敗血症
  • 具体的レジメン（市中 / 重症 / 院内、TG18 重症度別） → 胆嚢炎・胆管炎 / 虫垂炎 / 憩室炎 / 腹腔内感染の経験的治療（JAID/JSC 2023）
  • 原則：source control が本体。ドレナージ・手術が遅れないように

便秘と決めつけない

  • 便秘で済ませにくい：排ガス停止、嘔吐、膨満、持続痛、発熱、腹膜刺激、担癌、開腹歴、炎症反応高い
  • オピオイド誘発性便秘（OIC）：下剤の予防・早期介入が基本。ただし閉塞が疑わしいときは下剤で押さない
  • 下剤選択・OIC の管理 → 下剤の選択

disposition

帰しにくい

  • 不安定 / 腹膜刺激 / 原因不明の強い痛み
  • 高齢・免疫不全・担癌
  • コントロール不能な嘔吐 / 疼痛
  • 腸管虚血・穿孔・胆管炎・AAA・婦人科救急が否定できない
  • 画像で虚血 / 閉塞 / 穿孔 / 膿瘍

帰宅を考えやすい

  • バイタル安定・赤旗なし・経口可・痛み改善
  • 重大疾患の除外ができている
  • 再診基準を明確に伝えられる
帰宅時に必ず伝える
  • 痛み増悪 / 発熱 / 嘔吐持続
  • 血便 / 黒色便 / 失神 / 新たな腹膜刺激感
  • 再受診の目安（48 時間以内）
折りたたみ用メモ

外してはいけないもの

  • AAA / 腸間膜虚血 / 消化管穿孔 / 絞扼性腸閉塞
  • 異所性妊娠 / 卵巣捻転
  • 重症胆管炎（TG18 Grade III）

AAA は腹部拍動性腫瘤がなくても否定できない。腸間膜虚血は初期に腹部所見が乏しく乳酸上昇は遅れうる。異所性妊娠は hCG 値だけで除外しない。

やりがちなミス

  • 「便秘」で済ませる（開腹歴・担癌・排ガス停止があるのに）
  • 鎮痛を後回しにする（診断精度は損なわない）
  • 右上腹部痛でいきなり CT（初手は US）
  • 心窩部痛で ECG を取らない（ACS の見逃し）
  • 妊娠可能年齢で妊娠反応を飛ばす
  • 「乳酸が正常だから虚血ではない」と決めつける
  • 鎮痛で痛みが消えた＝安全と錯覚（外科腹症が隠れる）

帰宅前に最低限

  • 再診基準を具体的に伝える
  • 鎮痛の処方、食事・水分指導
  • フォロー先と時期の確認

出典（トリアージ・部位別診断・早期鎮痛）
