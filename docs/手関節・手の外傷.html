手関節・手の外傷

⚡ 即時アクション
  1. 危険サイン（橈骨動脈触知低下 / SpO2 末梢低下 / 正中・尺骨・橈骨神経障害 / コンパートメント徴候 / 開放骨折）→ 整形即コンサルト
  2. 必須 X-line：手関節 AP + 側面。舟状骨疑い（嗅ぎたばこ窩圧痛）は scaphoid view 追加
  3. 冷却 + 挙上 + 副木固定 + 鎮痛（NSAIDs / アセトアミノフェン）
  4. Mallet / Jersey / Bennett / ゲームキーパー母指は専門コール、血行障害は入院

🚨 Red flags（血流・神経・開放）

血流

  • 橈骨動脈・尺骨動脈触知の左右差、毛細血管再充満時間の延長
  • 末梢 SpO2 低下、患肢冷感・蒼白
  • 整復不能な変形 + 血流障害 → 緊急整復・整形コール

神経

  • 正中神経：母指対立、示指掌側のしびれ
  • 尺骨神経：小指外転・指内転、小指側のしびれ
  • 橈骨神経：手関節背屈（drop hand）、母指・示指間背側のしびれ

その他

  • 開放骨折（皮膚損傷 + 骨折線）→ 抗菌薬 + 破傷風 + 整形コール
  • 強い前腕緊満・激痛・受動指伸展で増悪 → コンパートメント症候群を疑う
1 つでもあれば即整形コンサルト

主な緊急鑑別：

  • 手のコンパートメント症候群 / 前腕コンパートメント症候群
  • 開放骨折 / 橈骨動脈損傷 / 手の神経損傷
  • 整復不能な橈骨遠位端骨折

🩺 最小診察（神経血管・圧痛部位）

その場で行う最小セット

  • 視診：変形・腫脹・皮膚損傷・色調
  • 触診：橈骨遠位端、嗅ぎたばこ窩、舟状骨結節、月状骨、第 1〜5 中手骨基部・骨幹・頭部
  • 可動域：手関節屈伸・橈尺屈、各指の MP / PIP / DIP 屈伸
  • 神経：正中・尺骨・橈骨の運動 + 知覚（左右差確認）
  • 血管：橈骨・尺骨動脈触知、毛細血管再充満、末梢 SpO2、皮膚色調
  • 腱：屈筋腱（FDP / FDS）、伸筋腱の独立運動
舟状骨スクリーニング

嗅ぎたばこ窩圧痛 + 舟状骨結節圧痛 + 母指軸圧痛のいずれかがあれば舟状骨骨折を強く疑う。初回 X-line 陰性でも sugar-tong 副木で固定し、10〜14 日後に再撮影または MRI / CT。

忘れがちな診察
  • FDP は PIP 固定下に DIP 屈曲を確認
  • FDS は対側の指を伸展位で固定し PIP 屈曲を確認
  • 母指 MP の尺側側副靱帯（ゲームキーパー母指）の不安定性

📷 X-line の置き方

部位基本撮影追加
手関節 AP + 側面 舟状骨疑い → scaphoid view（尺屈位 PA）
手・中手骨 AP + 斜位 + 側面 第 5 中手骨頚部疑い（boxer's fracture）
指 指単独で AP + 側面 関節内骨折は CT 追加
母指 AP + 側面 Bennett / Rolando は CMC 関節を含めて撮影
陰性でも油断しない
  • 舟状骨骨折は初回 X-line で 10〜30% が偽陰性 → 圧痛があれば固定 + 再評価
  • 月状骨周囲脱臼 / 月状骨脱臼は側面で見落としやすい（lunate の三角形配置に注意）
  • 橈骨遠位端骨折で尺骨茎状突起骨折を伴うことが多い

🦴 主要鑑別（骨折・脱臼）

病態手がかり進む先
橈骨遠位端骨折 FOOSH 後の手関節変形・圧痛、dinner fork 変形 橈骨遠位端骨折
舟状骨骨折 嗅ぎたばこ窩圧痛、初回 X-line 陰性あり 舟状骨骨折
中手骨・指骨外傷 手背腫脹、限局圧痛、回旋変形 手・指の骨折
月状骨周囲脱臼 高エネルギー外傷、手関節腫脹、正中神経症状 月状骨周囲脱臼
Bennett / Rolando 骨折 母指 CMC 関節の関節内骨折、軸圧痛 Bennett骨折 / Rolando骨折
boxer's fracture 第 5 中手骨頚部、握り拳で殴打、knuckle 消失 手・指の骨折

✋ 指の特殊外傷（専門コール）

Mallet finger（槌指）
  • DIP 伸展不能、屈曲位で固定
  • 骨片を伴わなければ DIP 完全伸展位での副木固定 6〜8 週間（PIP は自由）
  • 関節面 1/3 以上の骨片 / 掌側亜脱臼 → 整形コンサルト
  • 詳細 → Mallet finger
Jersey finger（FDP 断裂）
  • FDP の付着部断裂、DIP 単独屈曲不能
  • 環指が最多、ジャージを掴んだ瞬間に発生
  • 緊急手術適応（早期再建が望ましい）→ 整形即コール
  • 詳細 → Jersey finger
Bennett 骨折（母指 CMC 関節内骨折）
  • 母指 CMC 関節の関節内骨折 + 亜脱臼
  • 軸圧痛、母指基部の腫脹
  • 整形コンサルト（多くは観血的整復・内固定）
  • 詳細 → Bennett骨折
ゲームキーパー / スキーヤー母指
  • 母指 MP 関節尺側側副靱帯損傷
  • 母指外転・橈屈ストレスで疼痛 + 不安定性
  • Stener 病変では靱帯が母指内転筋腱膜に絡まり手術適応
  • 詳細 → ゲームキーパー母指 / Stener病変
PIP 関節脱臼・骨折脱臼
  • 背側脱臼が多い、整復後は伸展制限副木
  • 関節面 30% 以上の骨片 → 整形コンサルト
  • 詳細 → PIP関節脱臼

🩹 初期固定・鎮痛

固定の選び方

  • 橈骨遠位端骨折：sugar-tong 副木 / volar 副木
  • 舟状骨骨折疑い：母指を含めた thumb spica 副木
  • 中手骨頚部骨折：ulnar gutter 副木（MP 70°屈曲位）
  • 指骨骨折：buddy taping または aluminum 副木
  • Mallet finger：DIP 完全伸展位（Stack 副木）

鎮痛

  • 第一選択：アセトアミノフェン 内服、必要時 NSAIDs（出血傾向・腎機能に注意）
  • 整復前：オピオイド静注 + 局所麻酔（hematoma block）
  • 冷却 + 挙上を併用
固定後は必ず再評価

副木装着後に神経血管所見を再確認。指先の色・温度・知覚・運動を本人にも教え、悪化時は受診と説明する。

📤 disposition

帰宅を考えやすい

  • 転位の少ない閉鎖骨折で副木固定済み
  • 神経血管異常なし、疼痛コントロール可能
  • 整形外来受診（数日〜1 週間以内）が確実

入院 / 整形コンサルト

  • 開放骨折 / 血行障害 / コンパートメント疑い
  • 整復不能な脱臼・骨折
  • 月状骨周囲脱臼、Jersey finger
  • 多発外傷 / 高齢で疼痛コントロール困難
  • 夜間で整形対応不能 + 再評価必要例

⚠️ ピットフォール

  • 初回 X-line 陰性で舟状骨骨折を見逃す — 圧痛があれば固定 + 再評価
  • 月状骨周囲脱臼を AP のみで見落とす — 側面像で lunate の配置を確認
  • Mallet finger を「打撲」で帰す — DIP 単独伸展を確認
  • 母指 UCL 損傷を捻挫扱い — Stener 病変は手術適応
  • 固定後の神経血管チェックを忘れる — 装着前後で必ず比較
  • 指の回旋変形を見逃す — 屈曲位で爪面と中手骨の整列を確認
