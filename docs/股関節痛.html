⚠️ トリガー連動の注意（条件で意識）

特定の状況に出会ったら必ず思い出すべき注意点。フローの判断と並行して意識する。

緊急・命のトリガー

  • if大腿骨骨幹部骨折／不安定型骨盤輪骨折
    出血量 1〜1.5 L（骨幹部）。輸液・輸血準備、骨盤輪は骨盤バインダー＋IVR 検討、骨幹部は整形オンコール緊急
  • if開放骨折
    即時整形コンサルト。創部洗浄・抗菌薬・破傷風（→ 開放骨折）
  • if足背動脈触知不良／下腿の感覚運動障害
    血管・神経損傷を疑う。整形に緊急連絡＋血管外科
  • if小児の股関節痛＋発熱＋荷重不能
    化膿性関節炎を最優先で除外。Kocher の所見・関節穿刺・即時小児整形

見逃し回避トリガー

  • if転倒後の高齢者股関節痛＋Xp 陰性
    ベッドサイドに戻り、股関節内外旋痛と恥骨・坐骨・仙骨の圧痛を取り直す。occult fracture の頻度は 2〜10%
  • if股関節内外旋痛あり／Xp 陰性
    第一選択は MRI（感度ほぼ 100%）。日中なら MRI、夜間・休日なら CT で代用可能
  • if独歩で来院した高齢者の股関節痛
    歩行可能 ≠ 骨折否定。occult や安定型脆弱性骨盤骨折は歩ける症例が混じる。帰宅厳禁
  • if骨盤の骨折を 1 ヵ所見つけた
    他箇所も探す（恥骨↔坐骨↔仙骨）。Xp で恥坐骨があれば CT 追加で仙骨評価
  • if大腿骨頸部か転子部か迷う
    関節包の内 / 外で分ける（頸部＝包内、転子部＝包外）。コンサルトでは「大腿骨近位部骨折」とせず必ず分類して伝える
  • if大腿骨頸部骨折
    Garden 分類を添えてコンサルト（I/II vs III/IV で術式が変わる。整形医の最大のこだわり）
  • if術後の患肢が転倒で痛い
    インプラント周囲骨折を疑う。手術した病院へ転院相談（業界ルール）
  • if受傷機転がない高齢者の急な歩行困難
    大腿骨近位部骨折の受傷不明例 4.4%を想起。「病気」のように来院することがある
  • ifMRI も CT も陰性で股関節内外旋痛のみ
    内転筋の筋挫傷を考える（MRI 陰性骨折例の 6 割超に筋挫傷あり）。安定すれば保存・帰宅も可

ピットフォール

  • ⚠Xp 陰性 = 骨折なしと即断
    高齢者では2〜10% が occult。荷重不能 + 内外旋痛なら MRI / CT を回避しない
  • ⚠NSAIDs を高齢者にルーチン
    急性腎障害（NICE 2023）。第一選択はアセトアミノフェン、追加は大腿神経ブロック
  • ⚠コンサルトで「大腿骨近位部骨折」と曖昧に伝える
    整形業界では NG。頸部 or 転子部 + Garden 分類まで添える
  • ⚠仙骨骨折を Xp で除外する
    単純 X 線の感度は 7〜44%。恥骨・坐骨に骨折があれば CT を追加
  • ⚠骨粗鬆症未治療の既往骨折に介入しない
    大腿骨近位部骨折・脊椎圧迫骨折は再発リピーター。骨粗鬆症治療 を必ず検討
📷 撮影方向の備忘
  • 骨盤正面：骨盤上縁〜大腿骨中央／スケール挿入
  • 大腿骨頸部・転子部：両側股関節 正面 + 軸位（ラウエンシュタイン位は描出があいまいで要注意）
  • 大腿骨骨幹部：両側股関節 正面 + 側面（大腿骨全長、膝関節含む）
  • Occult 疑い：第一選択は MRI（T1 黒・STIR 白）、夜間で撮れない時は CT 多断面（前額・矢状・冠状の 3 断面、皮質途絶 2〜3 スライス連続を骨折線とみなす）
  • 脆弱性骨盤骨折：単純 CT で恥骨・坐骨・仙骨を必ず確認（造影は出血所見ある時のみ）
  • インプラント周囲：CT 多断面でステム周囲を読影（金属アーチファクト低減プロトコル）

🚨 命・赤旗

前提

  • 多発外傷を疑う場合は気道・呼吸・循環の評価が先（転倒 も参照）
  • FAST 陰性 / 開放骨折なし / 全身状態安定 を確認してから股関節評価へ進む
  • 高齢者の場合、転倒の原因疾患（失神・薬剤・脱水）の検索もセットで行う

Red flags（見つけたら即対応）

命に関わる赤旗
  • 大腿骨骨幹部骨折：閉鎖骨折でも出血量 1〜1.5 L。両側性なら容易に出血性ショック。直達牽引と整形オンコールの即連絡
  • 不安定型骨盤輪骨折：高エネルギーが疑われる場面で血行動態不安定。骨盤バインダー＋IVR 検討
  • 開放骨折：即整形、創洗浄、広域抗菌薬、破傷風 → 開放骨折
  • 血管・神経損傷：足背 / 後脛骨動脈触知不良、下腿感覚運動障害
  • 脱臼骨折：明らかな変形、皮膚緊張で皮膚壊死リスク → 整形に即連絡

「歩いて来た」に騙されない

独歩でも骨折は否定できない

大腿骨近位部の occult fracture や安定型の脆弱性骨盤骨折は、転位が軽度で歩ける症例が混じる。「歩けるからレントゲン正常で大丈夫」と帰宅させると後日の追加検査で骨折が見つかり、トラブルになりうる。痛がる高齢者は歩けても帰宅させないを原則にする。

小児の赤旗

  • 発熱 + 荷重不能 + 安静時疼痛 → 化膿性関節炎を最優先で除外
  • SCFE / Perthes は跛行と運動制限が中心、診断後は小児整形へ
  • 詳細な分岐は §2 へ

🧭 鑑別の枠組み

股関節痛は「どんな患者がどう怪我したか」で鑑別が大きく変わる。年齢と機転で 4 群に分け、それぞれの最頻 + 落とし穴を押さえる。

4 群フレーム

群典型像第一鑑別落とし穴
高齢・転倒 外旋短縮位 / 動かせない 大腿骨頸部 or 転子部骨折 occult / 脆弱性骨盤 / 内転筋挫傷
術後の患肢 人工骨頭・髄内釘の既往 インプラント周囲骨折 occult もある（大転子周辺）
若年・高エネ 交通外傷・高所転落 大腿骨骨幹部 / 骨盤輪 / 遠位端 多発外傷・出血性ショック
小児 跛行・荷重不能 SCFE / Perthes / 化膿性関節炎 発熱合併では感染最優先

整形外傷の診断戦略

病歴 → 鑑別 → 身体所見＆画像

内科疾患は「身体所見 → 鑑別 → 検査」の順だが、整形外傷は痛みで身体所見が取りづらい。年齢と受傷部位だけで鑑別を立て、身体所見と画像をケースバイケースで集めて確定する流れが実用的。明確骨折タイプ（外旋短縮位）なら身体所見抜きで即レントゲンも可。

高齢者の 3 大鑑別

  • 大腿骨近位部骨折（頸部 + 転子部）：転倒後股関節痛の 9 割以上。受傷機転がない例も 4.4% 存在
  • 脆弱性骨盤骨折：恥骨・坐骨・仙骨。低エネルギーで発症、血行動態は安定
  • 筋挫傷（主に内転筋）：身体所見は近位部骨折とそっくり、MRI でしか区別できない除外診断

小児の鑑別の組み立て

疾患年齢特徴初期対応
SCFE10〜15 歳・肥満児慢性〜亜急性の鼠径部痛、放散痛として膝痛で来院することも免荷＋小児整形へ即紹介
Perthes4〜10 歳跛行と運動制限、無腐性壊死小児整形へ紹介、画像精査
化膿性関節炎全年齢発熱・荷重不能・強い動作時痛、CRP 高値採血・関節穿刺・小児整形・抗菌薬
一過性滑膜炎3〜10 歳感冒後の自然軽快、解熱、CRP 低除外診断として位置付け、化膿性除外が先
小児は専門紹介を原則に

非専門医の確定診断にこだわらず、発熱の有無を仕分けて化膿性関節炎を最優先で除外。それ以外は小児整形へ早期に紹介する。

🦴 各疾患の詳細

① 大腿骨頸部骨折（関節包内骨折）

  • 関節包内骨折。骨頭への血行障害から無腐性壊死リスクがあり、術式選択に Garden 分類が直結する
  • Garden I/II（陥入型・転位なし）→ 内固定（cancellous screw など）
  • Garden III/IV（部分転位・完全転位）→ 人工骨頭置換
  • 初期対応は 48 時間以内（可能なら 24 時間以内）の手術が目標。日中なら即時整形、夜間・休日なら入院整形で翌朝コンサルト
  • 詳細：大腿骨近位部骨折

② 大腿骨転子部骨折（関節包外骨折）

  • 関節包外骨折。出血量が頸部より多くなる傾向（受傷後の血腫腫脹あり）
  • 術式は基本的に髄内釘（CHS / γ-nail など）。Garden 分類は使わない
  • 初期対応は頸部骨折と同じ。コンサルト時は「転子部骨折」と明示する
  • 詳細：大腿骨近位部骨折
頸部 vs 転子部の見分け方

骨折線が関節包の内側か外側かで分ける。Xp 正面で頸の中ほど = 頸部骨折、小転子〜大転子間 = 転子部骨折。コンサルトでは「大腿骨近位部骨折」とまとめず、必ず頸部 / 転子部 + Garden 分類（頸部のみ）を添える。整形医側のお作法。

③ 大腿骨骨幹部骨折

  • 若年の高エネルギー外傷が典型。高齢者でもインプラント術後の周囲骨折として出る
  • 閉鎖骨折でも出血量 1〜1.5 L。両側性で容易に出血性ショック
  • 初期対応は直達牽引（疼痛軽減＋骨折部安定化）と整形オンコール緊急連絡
  • 疼痛管理に大腿神経ブロックが極めて有効
  • 詳細：大腿骨骨幹部骨折

④ Occult fracture（不顕性骨折）

  • 初期 Xp で写らないが CT / MRI で見つかる骨折。大腿骨頸部・転子部の Xp 感度は 91.9〜98.1% で、2〜10% は occult
  • 転位が軽度なため歩行可能なケースもある。「歩けるから帰宅」が最大の落とし穴
  • 身体所見：股関節の他動内外旋痛が高感度・高特異度（経験則 90% 超）でコンサルトでも通じる共通言語
  • 第一選択は MRI（感度・特異度ほぼ 100%）。夜間・休日は CT で代用可（前額・矢状・冠状の 3 断面で皮質途絶 2〜3 スライス連続を骨折線とみなす）
  • 大腿骨頸部の occult は原則手術、転子部の occult も多くが手術。診断確定したら整形へ入院依頼
  • MRI 所見：T1 黒・STIR 白が新鮮骨折。慣れれば初学者でも読める
Occult を疑う閾値
  • 転倒 + 高齢 + 荷重不能（または痛がって歩きたがらない）
  • Xp で骨折線がはっきりしない
  • 股関節の他動内外旋で疼痛誘発
  • → MRI（または CT）追加。MRI/CT 不可なら入院安静で再撮影

⑤ インプラント周囲骨折（大腿骨ステム周囲骨折）

  • 大腿骨近位部骨折の術後患肢が転倒で痛い = 第一鑑別
  • インプラント自体は破損せず、デバイス近傍の骨が折れる。大腿骨ステム周囲が最多
  • 多くは Xp で骨折線がはっきりするが、大転子部の小骨片など occult もある（CT 追加）
  • 原則インプラントを入れた病院で再手術。他院手術なら術側病院へ転院依頼が業界ルール
  • 過去の術式・インプラント種類が再手術計画に直結するため
  • 初期対応は大腿骨近位部骨折と同じ（48h 以内手術目標）
土日のインプラント周囲骨折

土日は術側病院でも主治医不在のことが多い。週末は overnight 入院で術前準備し、週明けに転院依頼するのが現実的な落としどころ。

⑥ 脆弱性骨盤骨折

  • 高齢者の低エネルギー骨盤骨折（しりもち、ベッドからの転落など）
  • 主訴の 94.4% は股関節痛 / 腰痛 / 殿部痛で、骨盤痛と訴えるのは 4.8%。「股関節痛 = 大腿骨だけでなく骨盤も」を癖付ける
  • 血行動態は安定、IVR 不要、創外固定不要が原則（成人の高エネ骨盤骨折とは別物）
  • Xp 感度は 46.8〜50%、仙骨に至っては 7〜44%。Xp 陰性でも CT を追加
  • 骨盤の 1 ヵ所で骨折を見つけたら他箇所も探す（恥骨↔坐骨↔仙骨）
  • 身体所見：恥骨は仰臥位、坐骨は股関節屈曲位、仙骨は側臥位で圧痛が取れる
  • 分類：恥坐骨のみ = 安定型（荷重制限なし、痛み弱ければ通院可）／恥坐骨＋仙骨 = 部分安定型（4〜8 週荷重制限、リハビリ入院）
  • 詳細：脆弱性骨盤骨折

⑦ 筋挫傷（主に内転筋）

  • 転倒後股関節痛で内外旋痛があり Xp 陰性 → MRI で骨折なし、内転筋の STIR 白で診断
  • MRI で骨折を認めなかった患者の 61〜65% に筋挫傷あり。occult の鑑別と並ぶ
  • 安静度はフリーで保存加療。多くは 1 週間前後で自宅復帰
  • 痛みで生活困難なら入院（保存目的）。骨折ではないので非整形医が主治医でも可
  • 夜間で MRI 不可 → 暫定的に「occult 疑い」として経過観察入院、翌日 MRI で確定

⑧ その他

  • 大腿骨遠位端骨折：高齢の転倒、若年の高エネ。膝関節合併損傷を評価 → 大腿骨遠位端骨折
  • 骨盤輪骨折（不安定型）：高エネ外傷。血行動態評価＋骨盤バインダー＋IVR 検討
  • 股関節脱臼：高エネ・人工股関節術後。患肢短縮屈曲内旋位、整復は早期に

🩺 身体所見・画像

身体所見の最小セット

  • 外旋短縮位（ストレッチャー上の肢位で 1 秒判定）→ 大腿骨近位部骨折を強く示唆
  • 股関節の他動内外旋痛（occult を含む近位部骨折で陽性）。整形医とも共通言語
  • 骨の触診：恥骨（仰臥位）、坐骨（股関節屈曲位）、仙骨（側臥位）、大転子
  • 大転子部の皮下血腫（近位部骨折で 56%、骨折なしで 6%）
  • 足背 / 後脛骨動脈、下腿感覚運動（血管神経評価）
  • 同側膝関節（大腿骨遠位端・骨幹部の合併損傷）
「触れる骨は触る、触れない骨は他動運動で」

体表から触れる骨（恥骨・坐骨・仙骨・大転子）は圧痛で骨折判定可能。関節部の骨（大腿骨頸部）は触れないので、他動内外旋で疼痛誘発の有無を見る。整形医とのコミュニケーションでもこの所見が共通言語になる。

画像検査

まず撮るもの

  • 骨盤正面（骨盤上縁〜大腿骨中央、スケール挿入）
  • 追加撮影は技師と連携してその場で判断

骨折部位に応じた追加撮影

疑い疾患追加撮影
大腿骨頸部骨折両側股関節 正面 + 軸位（必要に応じて大腿骨中央まで）
大腿骨転子部骨折両側股関節 正面 + 軸位（必要に応じて股関節〜膝まで）
大腿骨骨幹部骨折両側股関節 正面 + 側面（大腿骨全長、膝関節含む）
脆弱性骨盤骨折骨盤正面（必要時 inlet / outlet）→ 単純 CT で恥骨・坐骨・仙骨
インプラント周囲術側股関節〜大腿骨全長（金属アーチファクト低減プロトコル CT も）

骨折線イメージのコツ

  • 大腿骨は正面 + 軸位で確認。正面で見えず軸位で見える骨折線もある
  • ラウエンシュタイン位は撮影が容易だが描出があいまい。可能なら軸位を採用
  • CT は3 断面（前額・矢状・冠状）を全スライス確認。皮質途絶 2〜3 スライス連続で骨折線とみなす
  • MRI は T1 黒・STIR 白＝新鮮骨折。筋挫傷では筋肉の STIR が白く光る

🔍 Occult fracture

Occult とは

初回 Xp で骨折線が見えないが、CT / MRI で見つかる骨折のこと。大腿骨頸部・転子部骨折の初期 Xp 感度は 91.9〜98.1%、すなわち 2〜10% が occult。「Xp で正常 → 帰宅」とすると数 % で見逃しになり、訴訟リスクすらある。

Occult を疑う条件

  • 高齢者（特に 75 歳以上）、転倒（または機転不明の歩行困難）
  • 荷重不能、または痛がって歩きたがらない
  • Xp で大腿骨近位部にはっきりした骨折線なし
  • 股関節の他動内外旋痛あり

第一選択：MRI

  • 感度・特異度ほぼ 100%（NICE 2023 ガイドライン推奨）
  • T1 黒・STIR 白で新鮮骨折を判定
  • 同時に筋挫傷も鑑別可能（筋肉が STIR で白く光る）
  • 日中で MRI が稼働していれば最初から MRIが費用対効果的

夜間・休日の代替：CT

  • MRI が撮れない時は CT で代用可能（ガイドライン外だが実用的）
  • 3 断面（前額・矢状・冠状）の全スライスを読む
  • 皮質途絶と骨折線が 2〜3 スライス連続の 2 つを骨折線とみなす
  • CT 陰性でも疑い濃厚なら入院安静で MRI 待ち
CT vs MRI の使い分け

研究では MRI の方が感度が高い（CT で 2%、MRI で 47% の検出率という報告あり）。ただし夜間・休日に MRI 不可で CT 可という現実的状況では CT で診断確定が合理的。CT で見つかれば追加 MRI は不要。

CT も MRI もできない時

  • 身体所見の再評価：股関節内外旋痛・大転子部の皮下血腫が判断材料
  • 陽性ならoccult 疑いで入院（帰宅させない）
  • 翌日に MRI / CT を追加し確定

Occult でも手術になる

大腿骨頸部 occult は原則手術

転位が軽いから保存、ではない。非荷重で保存していても結局手術になる例が多い。診断確定後は早期に整形コンサルト → 入院 → 48 時間以内手術が目標。転子部 occult も多くが手術となるため、初期対応は同じ。

コンサルトの一言

「高齢者の転倒後股関節痛です。Xp では骨折線がはっきりしませんが、股関節の内外旋で疼痛があり occult fracture を疑っています。一緒に方針決定をお願いします」 ── この型でほぼ通る。

📤 マネジメント・disposition

5 択分類（コンサルトのタイミング）

区分意味典型疾患
即時整形当直帯でも即連絡骨幹部骨折・開放骨折・脱臼骨折・血行動態不安定な骨盤輪
大院整形日中なら即、夜間は入院で翌朝頸部・転子部骨折、インプラント周囲骨折、occult fracture
入院整形入院 → 整形コンサルトは翌日脆弱性骨盤骨折、筋挫傷で歩けない例
帰宅保存外来通院恥骨単独の安定型脆弱性骨盤骨折で痛みが弱い、軽症筋挫傷
転院依頼術側病院 / 手術室空き先へ他院インプラント術後の周囲骨折、自院手術室飽和の頸部骨折

採血・術前準備

  • 血算、凝固（PT, APTT）、生化（Cr, BUN, 電解質）、血型、クロスマッチ
  • 手術が前提なので術前検査の意味合い
  • 抗凝固薬・抗血小板薬の有無を確認、休薬・拮抗を整形 / 麻酔と相談

疼痛管理

  • アセトアミノフェン 15 mg/kg 静注（最大 1 g）— 第一選択
  • 必要に応じてオピオイド（フェンタニル等）追加
  • NSAIDs は高齢者で避ける（急性腎障害リスク、NICE 2023）
  • 大腿神経ブロックが施行可能なら極めて有効（NICE 2023, AAOS 推奨） → 大腿神経ブロック
  • 骨幹部骨折は直達牽引そのものが疼痛軽減になる

手術タイミング（NICE 2023 / AAOS）

  • 大腿骨近位部骨折は受傷後 48 時間以内、可能なら 24 時間以内の手術が目標
  • 24 時間以内手術はそれ以降と比較して死亡率・合併症が低い
  • 遅延の最大の理由は手術室の空きがないこと → 転院も視野に入れる

disposition 早見

  • 骨幹部骨折 → 即時整形 → 直達牽引 → 入院
  • 頸部・転子部骨折 → 入院 → 整形 → 48h 以内手術
  • インプラント周囲骨折 → 自院手術なら入院、他院手術なら術側病院へ転院相談
  • 脆弱性骨盤骨折 → 多くは疼痛で生活困難 → 入院（部分安定型はリハビリ入院 4〜8 週）
  • occult 疑い → 入院（独歩でも帰宅させない）
  • 筋挫傷確定 → 安静度フリー、痛み弱ければ帰宅、強ければ入院（非整形医主治医も可）
  • 骨折なし＋荷重可能＋独歩可能＋同居者あり → 帰宅（疼痛管理 + 数日内再診）
独居高齢者の入院基準

骨折がなくても、痛みで日常生活が成り立たない（トイレ・食事・移動が独力で不可）独居高齢者は入院適応。介護サービス調整が間に合わず帰宅させると再受診・転倒のリスクが高い。脊椎圧迫骨折と同じ判断基準。

再発予防（骨粗鬆症）

  • 大腿骨近位部骨折・脊椎圧迫骨折は骨粗鬆症の代表的ハイリスク。骨密度測定なしで治療適応
  • 未治療ならビスホスホネート週 1 回（アレンドロン酸 35 mg or リセドロン酸 17.5 mg）を退院前 / 外来で開始検討
  • 低 vitamin D（25OH-D < 30 ng/mL）なら活性化ビタミン D 製剤を併用
  • 10 年制限・腎機能（eGFR < 30 で禁忌）・歯科スクリーニング（顎骨壊死リスク）を確認
  • 詳細：骨粗鬆症治療

家族・本人説明の 3 点

  • 大腿骨近位部骨折は原則手術であり、寝たきり予防のためにも 48 時間以内が目標
  • 占明・歩行リハビリを早期に開始しないと予後が悪くなる
  • 骨粗鬆症治療を術後に開始する場合がある（再発予防）

⚠️ 見落とし・ミス

診断の落とし穴

  • Xp 陰性 = 骨折なしと即断 → occult fracture（2〜10%）の見逃し
  • 歩行できる = 骨折否定と判断 → 独歩で来た occult や安定型脆弱性骨盤骨折を帰宅させて後日トラブル
  • 受傷機転がない = 骨折なしと判断 → 機転不明の大腿骨近位部骨折が 4.4% 存在
  • 仙骨骨折を Xp で除外 → 感度 7〜44%、CT 必須
  • 骨盤の骨折を 1 ヵ所で満足 → 他箇所も探す（恥骨↔坐骨↔仙骨）
  • 術後患肢の疼痛をインプラント周囲骨折と疑わない

マネジメントの落とし穴

  • コンサルトで「大腿骨近位部骨折」と曖昧に伝える → 整形業界では NG。頸部 / 転子部 + Garden 分類（頸部のみ）を必ず添える
  • 骨幹部骨折で輸液・輸血準備を遅らせる → 1〜1.5 L 出血を過小評価
  • NSAIDs を高齢者にルーチン使用 → 急性腎障害（NICE 2023 で非推奨）
  • 大腿神経ブロックを使わない → 強力な疼痛管理オプションを逃す
  • 他院インプラントの術後骨折を自院で抱え込む → 術側病院での再手術が原則。早期に転院相談
  • 手術室の空きがないを理由に手術を 48 時間以上先送り → 転院を視野に
  • 骨粗鬆症未治療の既往骨折を放置 → 再発リピーター。退院前 / 外来でビスホスホネート介入を検討

小児の落とし穴

  • SCFE が放散痛として膝痛で来院 → 股関節を必ず診察
  • 化膿性関節炎を一過性滑膜炎と誤診 → 発熱・CRP・荷重不能を厳格に評価
