股関節痛

⚡ 即時アクション
  1. 血流・神経・脱臼の有無を確認（足背動脈、下腿感覚運動、患肢短縮・外旋）
  2. 高齢者の単純 X 線で骨折が見えなくても荷重不能 → 不全骨折を疑い CT / MRI（NICE 2023）
  3. 大腿骨頸部骨折の disposition：手術は入院日 or 翌日（24–48 h 以内）が目標
  4. 小児の股関節痛は SCFE / Perthesを専門紹介、発熱合併では感染性関節炎を念頭

🚨 前提と Red flags

前提

  • FAST 陰性（腹腔内出血なし）
  • 開放骨折なし
  • 全身状態が安定していることを確認したうえで股関節の評価に進む
  • 全身の外傷評価は → 転倒

Red flags

  • 足背動脈触知不良 / 下腿の感覚脱失・運動障害 → 整形外科に緊急連絡
  • 明らかな短縮・外旋 → 大腿骨近位部骨折を強く示唆
  • 発熱 + 股関節痛 + 強い動作時痛 → 化膿性関節炎を疑う
  • 小児の股関節痛 + 跛行 + 発熱 → 感染性関節炎 / SCFE / Perthes
  • 骨幹部骨折 → 出血量 1–1.5 L の可能性、整形オンコール緊急
血管・神経障害があるとき

足背動脈触知不良 / 下腿の感覚脱失・運動障害 → 整形外科に緊急連絡

🩺 まず確認すること

  • 足背動脈の触知（末梢循環の確認）
  • 下腿の感覚・運動（神経損傷の有無）
  • 患肢の短縮・外旋（大腿骨近位部骨折を強く示唆）
  • 鼠径部・大転子部の圧痛
  • 荷重可否
  • 発熱・全身炎症所見（感染性関節炎の鑑別）

📷 画像検査

まず撮るもの

  • 骨盤正面（骨盤上縁〜大腿骨中央）
  • スケール挿入
  • 追加撮影はレントゲン室にて判断

骨折部位に応じた追加撮影

骨折部位追加撮影
大腿骨頸部骨折両側股関節 正面 + 軸位（必要に応じて大腿骨中央まで）
大腿骨転子部骨折両側股関節 正面 + 軸位（必要に応じて股関節〜膝まで）
大腿骨骨幹部骨折両側股関節 正面 + 側面（大腿骨全長、膝関節含む）
  • CT も同様の範囲で撮影

🦴 骨折があった場合

採血

  • 血算、凝固（PT, APTT）、生化（Cr, BUN, 電解質）、血型、クロスマッチ
  • 手術に向けた術前検査の意味合い

整形外科コンサルト

骨折部位対応
骨幹部骨折直達牽引 が必要 → 整形外科オンコール（当直帯でも即連絡）
頸部・転子部骨折入院 → 後日整形外科コンサルト
骨幹部骨折は緊急性が高い

骨幹部骨折は出血量が多く（1–1.5 L）、直達牽引による整復・固定が必要なため、整形外科への即時連絡が必要。

疼痛管理

  • アセトアミノフェン（15 mg/kg 静注、最大 1 g）— 第一選択
  • 必要に応じてオピオイド（フェンタニル等）を追加
  • NSAIDs は高齢者では避ける（急性腎障害リスク）— NICE 2023
  • 大腿神経ブロック → 施行可能なら非常に有効 — NICE 2023, AAOS

🔍 骨折が見えない場合

  • X線で骨折が見えなくても股関節痛 + 荷重不能 → 不全骨折（occult fracture）を疑う
  • → MRI（感度最高）が第一選択 — NICE 2023
  • MRI が 24 時間以内にできない / 禁忌 → CT — NICE 2023
  • 帰宅させる場合 → 荷重制限 + 数日以内の再評価 を指示
不全骨折の頻度

X線で骨折が見えない股関節痛のうち、MRI で 約 2–10% に不全骨折が見つかる。 高齢者 + 転倒 + 荷重不能 → 安易に「骨折なし」と判断しない。

📤 disposition

  • 骨幹部骨折 → 整形外科オンコール → 直達牽引 → 入院
  • 頸部・転子部骨折 → 入院 → 後日整形外科コンサルト（手術は入院日 or 翌日が目標 — NICE 2023）
  • 骨折なし + 荷重可能 → 状態に応じて帰宅（疼痛管理 + 再診指示）
  • 骨折なし + 荷重不能 → 入院 or MRI/CT 精査を手配

⚠️ ピットフォール

  • X 線陰性 = 骨折なしと即断する（高齢者の occult fracture を見逃す）
  • NSAIDs を高齢者にルーチンで使う（急性腎障害）
  • 骨幹部骨折で出血量を過小評価する（1–1.5 L 出血の可能性）
  • 小児の股関節痛で SCFE / Perthes / 化膿性関節炎を見落とす
  • 大腿神経ブロックを使わない（疼痛管理の有用な手段）
