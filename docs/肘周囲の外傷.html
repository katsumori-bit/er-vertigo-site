画像確定後の注意・見逃し回避・ピットフォール

診察段階の Red flag は身体診察パネルに統合済み（⚠ アクション）。ここでは画像で診断が確定してから意識すべき疾患固有の注意・鑑別の入口・ありがちな誤りをまとめる。

① 画像確定後の注意（疾患固有）

  • 画像肘関節脱臼骨折で整復不能
    手術室での観血的整復が必要。1 回ワントライ後、戻らなければ即整形。「脱臼骨折は緊急で整復してただの骨折にする」が全関節脱臼の大原則　→ 肘関節脱臼 詳細
  • 画像鉤状突起骨折（脱臼の 10-15% に合併）
    "カギヅメ" 前の小さな骨片で、整復前は気づかれにくく整復後の前後比較で初めてわかる。剥離骨折なら整復に影響しないので非整形外科医も整復可。紹介状に記載
  • 画像橈骨頭骨折 2 mm 以上 / Hotchkiss Type III 以上
    手術考慮。2 mm 未満は保存。骨折線は 2 mm 以内が 62%（微妙骨折タイプが標準）。fat pad sign 陽性のみでも橈骨頭骨折として対応　→ 橈骨頭骨折 詳細
  • 画像上腕骨顆上骨折 Gartland Type II/III
    即時整形コンサルト（整復・観血的整復・ピンニング）。Type I と II で迷ったら骨折線ありとして Type II 扱い。神経合併症 19〜49%、AIN 損傷が最多　→ 上腕骨顆上骨折 詳細
  • 画像小児で尺骨骨折を見たら
    全例で Monteggia 骨折を疑い橈骨頭脱臼を必ず探す。radiocapitellar line で小頭中央通過を確認（Bado Type I 70%／III 25%）。必ず正面・側面 2 方向で確認。PIN 損傷 10〜20%、即整形
  • 画像上腕骨外顆骨折 Jakob Type II/III、2 mm 以上
    即整形（多くは Salter-Harris Type IV、関節面〜成長板〜骨幹端まで達する）。紹介状は「右上腕骨外顆骨折 (Salter-Harris Type IV)」表記。迷ったらオーバートリアージ

② 見逃し回避トリガー

  • if外傷後肘痛の全例
    橈骨頭骨折を必ず鑑別に挙げる。成人肘外傷の半分以上を占めるのに最も見逃されやすい。画像で迷ったら橈骨頭を触れて回内外で疼痛があれば骨折扱い
  • ifFat pad sign 陽性（成人）
    75% に occult fracture、86.6% が橈骨頭骨折。骨折線が見えなくても橈骨頭骨折として対応、外固定＋三角巾＋紹介状に「橈骨頭骨折疑い」と記載
  • ifFat pad sign 陽性（小児）
    76% に骨折、53% が上腕骨顆上骨折。Anterior fat pad sign の感度 96.4%／陰性的中率 98.2%。fat pad だけでも顆上骨折として対応
  • if上腕骨顆上骨折で神経症状が来院時にない
    数日後に後発することが多い。「最初は神経所見なし → 翌日整形外科で発覚」のパターンに注意。神経症状の後発を両親へていねいに説明。断言できないなら即時整形にコンサルト（訴訟リスクも）
  • if4 歳以下の上肢痛で受傷不明
    肘内障を疑う。典型病歴（手を引っ張った）は50% のみ。残り半分は「転んで腕を痛める」「よくわからないが腕を痛めた」。最初にエコーで J サイン・PFPS を評価
  • if成人激痛＝レントゲン直行、独歩＝身体所見ベース
    激痛なら肘関節脱臼／肘頭骨折を最初に疑いレントゲン直行。独歩なら橈骨頭骨折として身体所見＋fat pad sign＋肘伸展試験で総合判断

③ ピットフォール

  • 注意橈骨頭骨折を「捻挫」で帰宅
    骨折線が微妙で fat pad sign のみ陽性ということが珍しくない。外傷後肘痛は全例で橈骨頭骨折を鑑別＋身体所見＋fat pad sign＋肘伸展試験を組み合わせる
  • 注意肘内障をレントゲン室で整復してしまう
    レントゲン側面の撮影姿勢（肘 90° 屈曲＋前腕回旋）が整復法そのもの。クリック音が確認できないと診断が迷宮入り。肘内障疑いは最初にエコーで J サイン評価
  • 注意上腕骨顆上骨折で「神経所見今は無し」と帰宅
    神経症状は数日後に後発することが多い。神経症状の継続は約 1 カ月、後遺症化すれば訴訟リスク。神経症状を断言できない上腕骨顆上骨折は即時整形。骨折線が明瞭な顆上骨折も即整形
  • 注意Monteggia 骨折の橈骨頭脱臼を見落とす（尺骨だけ見て満足）
    尺骨骨折を見たら全例で橈骨頭脱臼を探す。radiocapitellar line を引いて小頭中央通過を確認。正常な小児でも 15.6% が中央通らないので、尺骨骨折時に限って引くのがコツ。正面・側面 2 方向で確認
  • 注意上腕骨外顆骨折を保存予定にして再診なく転位拡大
    保存とした Jakob Type I・転位 2 mm 未満でも8.5〜10% が後に手術、介入後も約 10% に肘の屈曲・伸展障害。整形外科医のシーネ内でも外顆がずれる。「最軽症でも後日手術が 10%、合併症が 10%」と両親に誠意を込めて説明＋必ず再診
X 線適応＋撮影方向

適応の考え方

  • 明らかな外力＋肘痛・腫脹・変形・自動運動制限のいずれかがあれば撮影。成人激痛なら身体所見をとろうとせず最初からレントゲン直行（肘関節脱臼・肘頭骨折は明確骨折タイプ）
  • 独歩 walk in 受診なら橈骨頭骨折を想起して画像微妙前提で身体所見＋fat pad sign＋肘伸展試験を組み合わせる
  • 4 歳以下の牽引後 or 受傷不明の上肢痛は最初にエコー。レントゲンは後回し（撮影姿勢自体が整復になってしまう）

撮影方向

  • 成人：肘関節 2 方向（正面・側面）。側面像が決定的（脱臼・肘頭骨折・橈骨頭骨折・fat pad sign すべて側面で）
  • 小児：左右両側を撮ると比較しやすい。最初から健側もオーダーすることを推奨
  • 肘内障疑い：レントゲン姿勢が整復法と同じ → まずエコー。J サイン（橈骨頭−上腕骨小頭間に輪状靱帯が J 字状に挟まる、感度 64.9%／特異度 100%）と PFPS（感度 98%／特異度 70%）
  • Monteggia 骨折疑い：尺骨骨折を見たら正面・側面 2 方向で radiocapitellar line を必ず引く（片方しか異常が出ないことがある）
  • 外顆骨折疑い：小頭の外側転位は正面像が最も分かりやすい。Type I/II の微妙タイプは radiocapitellar line で評価

追加検査の閾値

  • CT 追加：橈骨頭骨折でHotchkiss / 改訂 Mason 分類確定のため／terrible triad（脱臼＋橈骨頭骨折＋鉤状突起骨折）疑い／顆上骨折の転位確定／脱臼骨折で観血的整復前評価
  • エコー：肘内障の初期評価（J サイン）＋整復後の再評価。橈骨頭骨折の描出にも報告あり（熟練要）
  • MRI：靱帯損傷・筋挫傷（橈骨頭骨折で約 30% に合併）は整形外来で評価。ER で必須ではない

整復後 X 線（神経血管再評価とセットで必須）

  • 必ず整復前後 2 枚を撮り紹介状に添付。脱臼整復後／顆上整復後／前後比較
  • 整復後に鉤状突起骨折の有無を再評価（脱臼の 10-15% に合併、整復後の前後比較で初めてわかる）
  • 整復前後で正中・尺骨・橈骨神経＋橈骨動脈触知を比較し、カルテに必ず記載（画像のみで帰宅しない）
  • 肘内障では整復後にエコーで J サイン消失を再確認。両親に客観的に証明可能
X 線の読み方（系統的アプローチ）

① 撮影セットの確認

  • 成人は 2 方向（正面・側面）が揃っているか。側面像が決定的（脱臼・肘頭骨折・橈骨頭骨折・fat pad sign すべて側面で）
  • 小児は健側を含めた両側を並べる（左右比較が必須）。撮影してなければ追加オーダー
  • 撮影体位の妥当性：回旋不良で関節間隙が不明瞭なら撮り直しを依頼

② 側面像 — fat pad sign を必ず確認

  • Posterior fat pad：上腕骨後面はもともと脂肪と骨が密接 → わずかでも盛り上がりがあれば陽性 = 異常
  • Anterior fat pad：もともと前面はわずかなスペースあり → 強く張り出していれば陽性
  • Sail sign：両方陽性で 2 つの pad がヨットの帆のように見える
  • 成人で fat pad 陽性 = 75% に occult fracture、86.6% が橈骨頭骨折。骨折線が見えなくても橈骨頭骨折として対応
  • 小児で fat pad 陽性 = 76% に骨折、53% が上腕骨顆上骨折。Anterior fat pad sign の感度 96.4%／陰性的中率 98.2%

③ Anterior humeral line — 小児上腕骨顆上骨折の検出

  • 側面像で左右比較。上腕骨前縁の延長線を引く
  • 正常：上腕骨小頭の中央 1/3 を通過
  • 異常：線が小頭の前方を通る＝小頭が後方転位（顆上骨折の 98% は伸展骨折）
  • 上腕骨顆上骨折でこれが唯一の異常所見ということもある（必須評価）
  • 暗記必須の 3 ポイントの 1 つ：① 前方成分の若木骨折線、② anterior humeral line、③ fat pad sign

④ Radiocapitellar line — Monteggia 骨折・上腕骨外顆骨折

  • 橈骨の中心線を引いた延長線。正常：小頭の中央を通過
  • 異常：小頭の上・外側を通過 → 橈骨頭脱臼あり = Monteggia 骨折
  • Bado Type I（70%、最多）：側面像で radiocapitellar line が小頭の上を通過
  • Bado Type III（25%）：正面像で radiocapitellar line が小頭の外側を通過
  • 必ず正面・側面 2 方向で確認（片方しか異常が出ないことがある）
  • 正常な小児の肘でも 15.6% で小頭中央を通らない → ルーチンで引くと迷う。小児尺骨骨折を見たときに限って引く
  • 上腕骨外顆骨折でも radiocapitellar line を活用：延長線が小頭から外側に外れる → 骨折。迷ったら左右比較

⑤ 肘伸展試験（成人・小児共通）

  • 回外したまま肘を伸展できるかをみる
  • 成人：感度 98.4%、陰性尤度比 0.03 → 除外に強い
  • 小児：感度 94.6%、陰性尤度比 0.11 → 除外に強い
  • 画像で骨折線なし＋fat pad sign 陰性＋肘伸展可能 → 固定なし・再診不要
  • 逆に肘伸展不可なら「骨折がないと整形外科医にお墨つきをもらうまで固定」

⑥ 見落としやすい所見・特殊サイン

  • 前方成分の若木骨折線：上腕骨顆上骨折の 98% は伸展骨折 → まず前面に亀裂、後方は保たれる。小児骨は柔らかく片側だけ亀裂・反対側はしなり
  • 小頭の外側転位：上腕骨外顆骨折。外顆の付着筋が骨片を引っ張り回転転位（正面像が最も分かりやすい）
  • 橈骨頭の微妙な骨折線：くびれや頭部に骨折線。2 mm 以内が 62%。写真を拡大して一所懸命探す
  • エコー J サイン：肘内障の確定診断（特異度 100%）
  • エコー PFPS：肘骨折除外（感度 98%）。J サインが見えないときの骨折除外に使用

⑦ CT を追加するタイミング

  • 橈骨頭骨折で Hotchkiss / 改訂 Mason 分類確定（2 mm 以上 / Type III 以上の手術判定）
  • Terrible triad 疑い（脱臼＋橈骨頭骨折＋鉤状突起骨折）
  • 上腕骨顆上骨折の転位確定（Gartland Type 確定）
  • 脱臼骨折で観血的整復前評価

⑧ 整復後の再読影（必須）

  • 脱臼整復後：必ず前後 2 枚を撮影。整復後の前後比較で鉤状突起骨折（10-15%）が初めてわかる
  • 顆上整復後：anterior humeral line が小頭中央に戻ったか、Type I 扱いで保存可能か再評価
  • 前後比較を並べて紹介状に添付（紹介状の "決め手"）
  • 同時に正中・尺骨・橈骨神経＋橈骨動脈触知を再評価し、画像 + 神経血管所見をセットで記載（画像のみで帰宅しない）
  • 肘内障では整復後にエコーで J サイン消失を確認
治療リファレンス：整復・固定・抗菌薬・処置

経静脈麻酔（ketofol）

  • 肘関節脱臼の整復はほぼ全例 ketofol（ケタミン＋プロポフォール）下で実施。他の経静脈麻酔と比べて合併症が少なく鎮静も十分
  • 若い筋肉質の患者が多く力技は困難。十分な筋弛緩と鎮静（顔をしかめていれば投薬不足）
  • 気道管理体制を整えてから開始。最終飲食時刻を必ず確認

肘関節脱臼整復術

  • ① レントゲンで肘関節脱臼を確認 → 経静脈麻酔の準備
  • ② Ketofol で十分に鎮静・脱力
  • ③ 術者は前腕を遠位へ牽引しつつ、整復できなければ徐々に腕を屈曲させる。あるいは肘 90° 屈曲位のまま前腕牽引
  • ④ 助手は術者と同じ力で上腕にカウンタートラクション（麻酔担当者と兼任で OK）
  • ⑤ 整復後、覚醒したら正中神経・尺骨神経・橈骨神経の損傷を評価。所見があれば紹介状に記載
  • ⑥ 肘関節外固定＋三角巾で固定し後日整形外科再診へ
  • 整復の本質：腕力ではなく「十分な筋弛緩 + 適切なカウンタートラクション」がすべて。失敗時は力を強めるのではなく、鎮静追加・カウンタートラクション補強
  • 1 回ワントライしてダメなら整形へ。脱臼骨折は緊急で整復してただの骨折にするのが大原則

肘内障整復（回内法 → 回外法）

  • 術者は患児の前に立ち、親指で橈骨頭をしっかりホールド
  • 肘 90° 屈曲位 → 愛護的に回内（患児の掌が術者の方を向く方向）
  • 回内法のほうが回外法より痛みが少なく、整復率も高い（回内 80〜95% vs 回外 69〜77%）
  • 回内でダメなら回外法（掌を上に向ける）でリトライ
  • クリック音（橈骨頭ホールドの親指で感じる）が成功サイン
  • 整復後にエコーでJ サイン消失を再確認 → 両親に客観的に証明可能
  • 手羽先法：J サインがハッキリしない／クリック音が術者しかわからない場合、両親や研修医に橈骨頭近くを触ってもらってクリック音を共有。「手羽先を捻って骨が外れる感覚があれば成功」
  • 整復確認できれば腕を動かすまで ER に残す必要なし → 帰宅 OK
  • 「痛みの記憶」で腕を動かさない患児もいる → 事前に両親へ説明。再発率 20% を必ず説明

肘関節外固定（全肘外傷に共通）

  • 適応：成人＝肘関節脱臼整復後／橈骨頭骨折／肘頭骨折　小児＝上腕骨顆上骨折／上腕骨外顆骨折／Monteggia 骨折（応急処置）
  • ① 採寸：小指の付け根 → 肘 → 腋窩の約 10 cm 手前までの長さ (A cm)
  • ② 切断：A cm の長さにカット（追加カットなし）
  • ③ 固定：手部尺側 → 肘頭 → 上腕まで一連で固定（ulnar gutter 〜 long arm posterior splint）
  • ④ 完成肢位：肘関節 90° 屈曲、前腕は正中位（小さく前にならえの姿勢）
  • 外固定後は必ず三角巾を併用。バストバンドは肘外傷では通常不要

三角巾の実技ポイント

  • ① 固定が完了するまで肘 90°・前腕水平をキープ（重要）。三角巾の中心の線を前腕に重ねる
  • ② 垂れ下がりを折り返し首の後ろで結ぶ。結んだ後、肘 90°・前腕水平を再チェック。三角巾の肘側の余りは一つ結びで折り込む
  • ③ 三角巾だけで完成。バストバンドは肘外傷では通常不要（必要なら Y 字の間から手を出せる位置で）
  • 独居高齢者：装着不能になりやすい。本人＋家族で実技確認、書面でも交付

開放骨折・抗菌薬／鎮痛

  • 開放骨折：セファゾリン 2 g IV（Gustilo III なら + ゲンタマイシン）／破傷風トキソイド／創部写真撮影／生食ガーゼ被覆して OR まで dressing を外さない／整形即コール
  • 鎮痛 第一選択：アセトアミノフェン 1,000 mg × 4 回／日（4 g/日まで）。高齢者・小児にも安全
  • NSAIDs（第二選択）：ロキソプロフェン 60 mg × 3 回／日。PPI 併用／腎機能 eGFR ≥ 60／消化管出血歴・心血管疾患なし／脱水なし
  • 処方日数：整形外来日まで（通常 5-7 日分）
  • 小児：アセトアミノフェン 10-15 mg/kg/回（最大 4 回/日）。NSAIDs は要注意
disposition の目安

30 秒で決める（5 択マネジメント）

※ 上肢外傷の即時整形コンサルト適応はわずか 3 つ + 小児肘の追加 1 つ：① 開放骨折、② 整復不能な脱臼、③ 血管・神経症状の合併、④ 小児肘では上腕骨顆上骨折 Type II/III／Monteggia／転位 2 mm 以上の外顆骨折。それ以外は「帰宅手術 or 帰宅保存」。

  • 即時整形整復不能脱臼／開放骨折／神経血管症状／上腕骨顆上骨折 Gartland II/III／Monteggia 骨折／外顆 2 mm 以上
    夜間・休日でも整形コール。気道・出血の安定化 → 洗浄・抗菌薬・破傷風 → 手術室。「断言できないなら呼ぶ」。整形外科医側も事前連絡を歓迎
  • 入院整形多発外傷／ADL 困難な高齢者／脱臼骨折で手術待機／コンパートメント疑い
    入院 + 翌朝コンサルト。挙上・厳重 5P 観察、ADL 支援と社会調整
  • 帰宅手術橈骨頭骨折 2 mm 以上 / Hotchkiss Type III 以上／肘頭骨折 2 mm 以上／成人 Monteggia（稀）
    待機手術（10 日以内に整形外来）。肘関節外固定 + 三角巾で帰宅、紹介状に分類・神経所見・転位の程度を明記
  • 帰宅保存橈骨頭骨折 2 mm 未満／肘頭骨折 2 mm 未満／肘内障整復後／顆上 Type I で神経所見なし／外顆 2 mm 未満
    肘関節外固定 + 三角巾、整形外来 1〜2 週以内予約。整復後 5P 観察・神経再評価を本人／家族に教える
  • 再診不要肘伸展試験陽性＋fat pad 陰性で骨折除外／肘内障整復確認後で症状なし
    疼痛増悪・腫脹増大・しびれ増悪で再受診を指示。「48-72h で改善なければ再受診」を必ず書面交付（occult fracture・後発神経症状の安全弁）

観察時間・フォロー間隔サマリー

  • 整復後 30 分 5P 観察：脱臼整復・固定直後
  • 整復前後の神経評価（正中・尺骨・橈骨）：脱臼例で必須。経過記録に明記
  • 上腕骨顆上骨折で帰宅した場合：神経症状の後発（数日後）に注意。両親に両指の OK サインを毎日確認するよう指導
  • 上腕骨外顆骨折 保存例：8.5〜10% が後に手術、10% が機能障害。短期再診を必ず
  • 肘内障：再発率 20%。再発時は同じ手技で整復可能
  • 整形外来 1〜2 週以内：保存治療例の標準フォロー
  • 外固定期間：橈骨頭・肘頭骨折 3〜4 週／脱臼整復後 2〜3 週／顆上・外顆は整形外科の判断

社会軸・特殊状況

  • 利き手・職業：投手・大工・音楽家は閾値を下げて整形紹介。手術 vs 保存の判断材料
  • 独居高齢者：三角巾装着困難。ADL（食事・更衣・排泄）に支障 → 入院・社会調整の検討
  • 小児・両親への説明：occult fracture・後発神経症状・10% 手術リスクなど「目に見えないもの」を心に届く言葉で。「割り箸が 2 つに折れるイメージかもしれないが、亀裂だけの場合もある」
  • 反復受診・機転不釣り合い：小児虐待のスクリーニング（不明機転の上肢外傷）

退室前ピットフォール（退室時固有）

  • 注意上腕骨顆上骨折 Type I で「神経所見なし」と帰宅して両親への説明が不十分
    神経症状の後発（数日後）を必ず両親に説明。断言できないなら即時整形。OK サインの確認方法を指導
  • 注意上腕骨外顆骨折を「軽症」で帰宅・再診なし
    保存予定でも10% が手術、10% が機能障害。整形外科医のシーネ内でも外顆がずれる。短期再診を必ず
  • 注意肘内障で「整復したので終わり」と説明
    再発率 20% を必ず説明。「痛みの記憶」で腕を動かさない患児がいることも事前説明
  • 注意三角巾の着脱方法を口頭だけで説明
    独居高齢者・小児では装着不能になる。本人 + 家族で実技確認、書面でも交付

主要な根拠・参考文献

  • 骨折ハンター（中外医学社）— 第 7 章「成人・肘関節痛ハンター」（肘関節脱臼・肘頭骨折・橈骨頭骨折）
    　ER の "hidden tip"：①外傷後肘痛は全例で橈骨頭骨折を鑑別 ②fat pad sign 陽性なら 75% に occult fracture、86.6% が橈骨頭骨折 ③肘伸展試験は感度 98.4%（成人）／94.6%（小児） ④画像微妙でも回内外時痛があれば橈骨頭骨折として扱う ⑤激痛＝レントゲン直行、独歩＝身体所見ベース
  • 骨折ハンター— 第 8 章「小児・肘関節痛ハンター①（上腕骨顆上骨折）」
    　暗記必須 3 ポイント：①前方成分の若木骨折線、②anterior humeral line（小頭中央 1/3 通過）、③fat pad sign（小児 76% に骨折、53% が顆上骨折）。Gartland Type II/III は即整形。AIN 損傷で teardrop sign、神経症状は後発あり
  • 骨折ハンター— 第 9 章「小児・肘関節痛ハンター②（肘内障）」
    　1-4 歳の牽引後／受傷不明 → 最初にエコー。J サイン（感度 64.9%／特異度 100%）と PFPS（感度 98%／特異度 70%）。整復は回内法（80-95%）→ 回外法（69-77%）。手羽先法でクリック音を共有。再発率 20%
  • 骨折ハンター— 第 10 章「小児・肘関節痛ハンター③（Monteggia 骨折・上腕骨外顆骨折・小児骨端線骨折）」
    　Monteggia：尺骨骨折＋橈骨頭脱臼、Bado Type I 70%／III 25%、radiocapitellar line で評価、PIN 損傷 10-20%（下垂指）。外顆骨折：小児肘骨折の 15%、多くは Salter-Harris Type IV、2 mm 以上で手術、保存でも 10% 手術・10% 機能障害
  • 骨折ハンター— Part II「肘関節脱臼整復術」「肘関節外固定」「三角巾±バストバンド固定」
    　全肘外傷の固定は肘関節 90° 屈曲位の外固定（シーネ）＋三角巾に集約。ketofol で十分な筋弛緩、術者前腕牽引＋助手カウンタートラクション、1 回ワントライで戻らなければ整形
  • 他科コンサルト実践マニュアル — 整形外科コンサルト文の標準書式
  • AIN vs PIN 整理表：AIN = 顆上骨折＋正中神経の枝＋DIP/IP 屈曲障害（teardrop sign）／PIN = Monteggia 骨折＋橈骨神経の枝＋MP 伸展障害（下垂指）。両方とも感覚障害なし
