📞 来院前に確認すべきこと

ホットラインから患者到着まで

救急隊からの第一報で 「失神 / けいれん / 意識障害」のいずれかを振り分け、到着前に蘇生室・物品・関連科を整える。

📞 救急隊ホットラインで聞く（10 項目）
  • 年齢・性別・主訴、現場の状況（場所・目撃者）
  • GCS（普段 → 現在）、バイタル（HR・BP 左右差・RR・SpO2・BT）
  • 気道：発声可能か、いびき様呼吸、嘔吐の有無
  • onset（急性 vs 亜急性 vs 慢性、発症時刻 / 最終 well-known time）
  • けいれん目撃の有無、舌咬傷、尿失禁、完全回復したか
  • 外傷の有無（頭部外傷・転倒）
  • 既往・内服（特に抗凝固薬・血糖降下薬・向精神薬・免疫抑制薬）、アレルギー
  • 最終食事、曝露（CO・薬物・アルコール）
  • 家族連絡先、Key person、ADL（自宅 / 施設）
  • ETA（到着予定時刻）、搬入経路
🚦 振り分け：失神 / けいれん / 意識障害
  • 失神的エピソード + 意識は戻っている → 意識消失 ページへ
  • けいれんが主訴（持続中 or 連続）→ てんかん重積対応
  • どちらでもない / 意識が戻らない → 本ページで進める

判断つかなければ意識障害として動き始めてよい（後で軌道修正可能）。

🛏 蘇生室・物品準備
  • 蘇生室を確保、モニター（心電図・SpO2・NIBP・BT）装着準備
  • 気道：O2 リザーバマスク、吸引、挿管セット（喉頭鏡 / ビデオ喉頭鏡 / EtCO2）、ボスミン
  • ルート：18G ×2、生食、加温輸液（低体温時）
  • ベッドサイド検査：血糖測定器、血ガスシリンジ、採血スピッツ一式（CBC / 生化 / 凝固 / 培養）、心電図
  • 薬剤：50% ブドウ糖、チアミン、ナロキソン、ロラゼパム or ジアゼパム、ミダゾラム
  • 体温計（深部温・低体温対応）、加温毛布
🏥 関連科への heads up（必要に応じて）
  • 頭部 CT 室を空けておく（放射線科にコール）
  • 脳外科 / 神経内科：脳卒中疑い、けいれん、神経所見あり
  • 麻酔科：挿管リスク高い場合（GCS ≤ 8、嘔吐、顎位不良）
  • ICU / 救命科：ショック、敗血症、多臓器障害疑い
  • 感染症科：髄膜炎・脳炎疑い（発熱 + 意識障害 + 項部硬直）
  • 透析科：透析患者・電解質異常疑い

ABCD + バイタル（到着直後 / Primary Survey）

🚨 危険サイン — 見つけたら即対応
  • A：気道が保てない（GCS ≤ 8 / いびき様呼吸 / 嘔吐）→ 気道確保・挿管準備
  • C：ショック（低血圧・頻脈・末梢冷感）→ 輸液・昇圧、原因鑑別（出血 / 敗血症 / 閉塞性 → §6 dispo の「全身」へ）
  • D：瞳孔不同 + 意識低下 → 脳ヘルニアを疑う → 頭部 CT を最優先（§3 来院時 ④ CT へ即移送）
  • けいれん持続（> 5 分 or 連続）→ けいれん重積として治療を ABCD と並行で先行
  • V：高度低体温（< 30°C）→ 加温（受動 → 能動）、不整脈モニタ強化、循環安定まで CPR 中止せず

🩸 来院時にやること

来院後 5〜10 分のオーダー順

ABCD（§2）と並行で進める。① 血糖（指先） → ② 血ガス → ③ 採血セット → ④ 頭部 CT → ⑤ Coma Cocktail の流れで動く。

① 意識障害 = まず血糖（指先）

最も頻度が高く、最も早く治せる原因。指先血糖を真っ先に。

  • 低血糖（< 70 mg/dL） → B1 先行 → 50% ブドウ糖 40 mL IV（→ ⑤ Coma Cocktail）
  • 高血糖（> 400 + 脱水 + アシドーシス） → DKA / HHS → DKA・HHS
② 血ガス（VBG / ABG）

5 分で結果が出る。VBG で十分なことが多い（ABG は重症呼吸不全・酸素化評価が必要なときに）。

チェック項目（一気に拾う）

  • pH / PaCO2 / HCO3 / lactate — 代謝性アシドーシス（Lac↑ → ショック・腸管虚血・敗血症）、CO2 ナルコーシス
  • Na / K / Cl / iCa / glucose / Hb — 電解質異常・血糖の即時確認
  • COHb — CO 中毒（パルスオキシメータでは見えない！）
  • MetHb（疑わしい曝露歴があれば）
  • Anion gap：MUDPILES（Methanol / Uremia / DKA / Paraldehyde / Iron-INH / Lactate / Ethylene glycol / Salicylate）

意識障害で 説明のつかない代謝性アシドーシス を見たら septic abdomen（腸管虚血・敗血症）も鑑別に。

④ 頭部 CT のオーダー

意識障害では実質ほぼ全例 CT。先に CT 室に予約しておく（→ §1 来院前で heads up 済みなら即移送）。

CT を撮る適応（明示）

  • 新規の局所神経症状（片麻痺・失語・瞳孔不同）
  • 頭部外傷疑い（転倒・出血・打撲痕）
  • 抗凝固薬・抗血小板薬内服
  • 原因不明の急性意識障害
  • 項部硬直（SAH / 髄膜炎）
  • けいれん後の遷延意識障害
  • 高齢者（明らかな代謝性原因がない場合）

単純 vs 造影 vs MRI

モダリティ得意選ぶとき
単純 CT急性出血、SAH、大きな占拠性病変、ヘルニア第一選択。ほとんどはこれで足りる
造影 CT脳腫瘍、脳膿瘍、CVT（CTV）占拠性病変・感染・静脈洞血栓を疑うとき追加
MRI / DWI超急性期梗塞、脳幹病変、脳炎CT 陰性 + 持続意識障害、脳梗塞超急性期、CVT 確定（MRV）
CTA / MRA脳血管病変、解離、もやもやSAH 後の動脈瘤同定、血管炎疑い
CT 正常でも安心しない
  • 超急性期梗塞（24 h 以内）は CT 陰性が普通。疑えば MRI / DWI
  • CVT・PRES・自己免疫性脳炎・NCSE は CT 正常がデフォルト
  • extraluminal air・微細出血は見落とされる
  • 髄膜炎の髄液採取：占拠性病変 / 局所所見 / 意識低下 / 凝固異常があるなら CT 後に
⑤ 診断的治療（Coma Cocktail）
状況投与
低血糖 or 血糖不明 + アルコール / 低栄養チアミン 100 mg IV → 50% ブドウ糖 40 mL IV
縮瞳 + 呼吸抑制（オピオイド疑い）ナロキソン 0.4–2 mg IV（反復可）

低栄養・アルコール多飲者にブドウ糖先行で Wernicke 脳症を誘発しうる。迷ったら B1 を先。

🔍 意識障害 s/o の鑑別ステップ【①〜⑤】

網羅的 mnemonic より先に、順番を決める

AIUEOTIPS は抜けを作らない網羅表。時系列に何から潰すかは別の話。 下の ①〜⑤ を順に走らせる。

① ABC 異常の有無
  • 心筋梗塞・大動脈解離・massive PE による脳血流低下
  • qSOFA（意識変容・RR ≥ 22・sBP ≤ 100）で敗血症を拾う
  • ショック系列（出血・心原性・分布性・閉塞性）→ ショック対応
  • 徐脈 + 高血圧 = Cushing 反応（頭蓋内圧亢進）
② 血ガス・血液値の異常

最速で血ガス 1 本。以下をまとめて拾う：

  • CO2 ナルコーシス（pH 低下 + PaCO2 上昇）、CO 中毒（CO-Hb ↑）
  • 低 Na（< 120 で意識障害）/ 高 Na
  • 高 Ca（悪性腫瘍）
  • 低血糖 / 高血糖
  • Mg 異常、代謝性アシドーシス（Lac ↑）

採血：CBC、BUN/Cr、電解質（Na/K/Ca/Mg/P）、肝胆道系、NH3、凝固、CRP、TSH（亜急性）、薬物スクリーニング（疑えば）

③ 意識障害下での神経学的所見

意識障害者でも取れる所見で中枢病変を拾う：

所見意義
眼位・共同偏視偏位方向の大脳半球病変／けいれん後は逆側
瞳孔：左右差 / pinpoint / 散大ヘルニア・橋出血（pinpoint）・抗コリン / 心停止後
対光反射・角膜反射脳幹機能
呼吸パターンCheyne-Stokes（両側大脳・間脳）／中枢性過呼吸（中脳橋）／失調性呼吸（延髄）
指にぎり（右手・左手）左右差で片麻痺検出（意識低下でも比較可）
疼痛刺激での四肢の動き除脳硬直（中脳・橋上部）／除皮質硬直（間脳以上）
項部硬直髄膜炎・SAH

→ 脳血管障害 / 細菌性髄膜炎 / SAH を拾う

④ 頭部 CT
  • 脳出血・SAH・脳ヘルニア・外傷性脳損傷
  • 超急性期の梗塞は CT 陰性でも否定しない（24 h 以内は所見に乏しい。疑うなら MRI）

CT を撮る適応（実質ほぼ全例だが明示）：新規巣症状、頭部外傷疑い、抗凝固薬内服、原因不明の急性意識障害、項部硬直、けいれん後

⑤ 追加検査（内分泌・代謝・特殊）
  • Wernicke 脳症（眼球運動障害・失調・意識変容の三徴、ビタミン B1 低値）
  • 甲状腺クリーゼ / 粘液水腫性昏睡（TSH、fT3/T4、体温）
  • 高アンモニア脳症（肝硬変・尿素サイクル・バルプロ酸）
  • 高浸透圧性昏睡（HHS：Osm、血糖、Na 補正）
  • 副腎クリーゼ（低 Na + 高 K + 低血糖 + 低血圧、ACTH/コルチゾール）
⑤でも決まらないとき

脳波 + MRI を考慮し、脳神経外科・神経内科にコンサルト。 特に 非けいれん性てんかん重積（NCSE）は原因不明意識障害の 8–20%（Leitinger 2019, Neurology）。

🧠 鑑別の補完・確認

使い分け（鑑別ステップ ①〜⑤ で詰まったら）
  • 鑑別確認リスト — DONT/VITAL/LABO/BRAIN/それ以外で網羅的に外す。最初に開く
  • AIUEOTIPS — mnemonic で抜けチェック
  • ABCDEs — ①〜⑤ と AIUEOTIPS でも説明できない時の見逃し疾患（自己免疫性脳炎・CVT・PRES・薬剤性脳症など）
📋 鑑別確認リスト（DONT / VITAL / LABO / BRAIN / それ以外）

除外できたら消していく。順に当てはめて穴を見つける。

DONT（可逆的・致死的、必ず外す）

  • 低酸素 / CO2 ナルコーシス / CO 中毒
  • 低血糖
  • ビタミン B1 欠乏（Wernicke）

VITAL（バイタル異常で絞る）

  • 低血圧 → ショックの鑑別（ショック対応）
  • 高血圧性脳症（除外診断）
  • 体温 ↑↓ → Sepsis、甲状腺機能低下、副腎不全、熱中症、低体温症、悪性症候群

LABO（血ガス・採血）

  • 電解質異常：Na↑↓、Ca↑、Mg↑↓
  • 肝性昏睡（NH3 ↑）
  • 尿毒症

BRAIN

  • 頭蓋内病変（→ 頭部 CT / MRI）
  • 脳炎・髄膜炎（→ 髄液、髄膜炎・脳炎）
  • てんかん（→ 脳波、NCSE を疑うとき）

それ以外

  • まれな疾患：内分泌、ポルフィリン、TTP
  • 薬剤 / アルコール含む
  • 精神科（除外診断）

📤 disposition

一般ルール（行き先の決め方）

  • 挿管 / ショック / 脳ヘルニア徴候 → ICU
  • 脳血管障害 → SCU / 脳外科 or 脳内科
  • 髄膜炎・脳炎 → ICU or 感染症科、神経内科（自己免疫性脳炎疑いなら）
  • 代謝異常（補正で改善） → 入院 + 原因精査
  • 中毒 → 経過観察入院（重症は ICU）
  • 低血糖（補正後に完全回復 + 原因明確） → 経過観察後に帰宅検討（インスリン・SU 薬誘発は遷延 → 入院）

原因別 dispo（要点 + 詳細リンク）

原因が絞れたら下のカテゴリから該当の sub-fold を開く。各原因に 補正・dispo・警告・詳細リンク を 4〜6 行で。

内分泌・代謝

低血糖

  • 補正：B1 100 mg IV → 50% ブドウ糖 40 mL IV（B1 を先に）
  • SU 薬・インスリン誘発 → 遷延 → 入院（10% ブドウ糖持続点滴、頻回血糖測定）
  • 食事欠食など原因明確 + 完全回復 → 経過観察 → 帰宅検討
  • 注意：補正後 1〜2 h で再低下するリスク → 再測定必須
  • 詳細 → 低血糖

DKA

  • 補正順：生食 1 L 急速 → 0.5 L/h、K 確認後にインスリン 0.1 U/kg/h
  • dispo：HCU / ICU（pH < 7.0、意識障害、K 異常 → ICU）
  • 注意：K の急変、脳浮腫（小児・若年で頻度高）、ケトン消失まで継続
  • 詳細：補液 / インスリン / dispo 判断 / 落とし穴 / 全体

HHS

  • 補正：生食 1.5〜2 L/h、Na 値で液体組成を選択
  • dispo：ICU（高浸透圧、意識障害、心血管リスク高）
  • 注意：補正速度厳守（< 10 mOsm/L/h）、急速補正で脳浮腫
  • 詳細：補液 / dispo 判断 / 全体（HHS は 対比表を確認）

Wernicke 脳症

  • B1 500 mg IV ×3/日 を最低 3 日（疑ったら投与、検査結果を待たない）
  • dispo：内科または神経内科に入院
  • 注意：低 Mg 併発で B1 反応悪い → Mg も補正
  • 詳細 → Wernicke 脳症

高 NH3（肝性脳症）

  • 治療：ラクツロース経口/経管、リファキシミン、誘因除去（感染・電解質・消化管出血・便秘）
  • dispo：内科入院（昏睡・誤嚥リスク → ICU）
  • 注意：意識レベルと NH3 値は相関しないことあり → 誘因検索を優先
  • 詳細 → 肝性脳症

副腎クリーゼ

  • 治療：ヒドロコルチゾン 100 mg IV ボーラス → 50 mg q6h、生食大量輸液
  • dispo：ICU（ステロイド先行 — ACTH/コルチゾール結果を待たない）
  • 注意：低 Na・低血糖・高 K の併発を同時に補正
  • 詳細 → 副腎不全クリーゼ

甲状腺クリーゼ / 粘液水腫昏睡

  • クリーゼ：β遮断薬・PTU・ヨード（PTU の 1 h 後）・ヒドロコルチゾン
  • 粘液水腫昏睡：T4 IV・ヒドロコルチゾン・受動加温（急速加温は禁忌）
  • dispo：両方とも ICU
  • 詳細 → 甲状腺関連危機
電解質異常

低 Na

  • 急性 (< 48 h) + 重症（< 120、けいれん・意識障害）→ 3% NaCl 100 mL ボーラス（必要時 1〜2 回反復）
  • dispo：ICU（補正速度厳守 — < 8 mEq/L/24 h、再低リスクなら 4〜6）
  • 慢性：緩徐補正、原因鑑別（薬剤・SIADH・喪失・水中毒）
  • 注意：急速補正で ODS（橋中心髄鞘崩壊）、過補正なら水分負荷で戻す
  • 詳細 → 電解質異常

高 Na

  • 補正：自由水（5% 糖、経口水分）で < 0.5 mEq/L/h、< 10〜12 mEq/L/24 h
  • dispo：軽症は一般病棟、重症（> 160 + 意識障害）は ICU
  • 注意：急速補正は脳浮腫リスクで禁忌
  • 詳細 → 電解質異常

高 Ca

  • 補正：生食 200〜300 mL/h、ループ利尿（脱水補正後）、ビスホスホネート、カルシトニン
  • dispo：入院（> 14 mg/dL or 症状あり）、原因検索（悪性腫瘍・PTH）
  • 注意：心電図（QT 短縮・徐脈）、チアジド禁忌
  • 詳細 → 電解質異常
中毒

オピオイド中毒

  • ナロキソン 0.4〜2 mg IV（縮瞳 + 呼吸抑制で疑う）反復可
  • dispo：経過観察入院（短時間作用：4〜6 h、長時間作用：12〜24 h 監視）
  • 注意：再眠（オピオイド > ナロキソンの半減期）、慢性使用者は禁断症状
  • 詳細 → オピオイド中毒

CO 中毒

  • 治療：100% O2（リザーバマスク）即開始、HBO 検討（意識障害・妊娠・心血管症状で適応）
  • dispo：入院（HBO 後でも 24 h 観察、遅発性脳症のリスク）
  • 注意：パルスオキシメータは正常を示す（CO-Hb と HbO2 を区別不能）
  • 詳細 → CO 中毒

ベンゾジアゼピン中毒

  • フルマゼニル 0.2 mg IV は要注意（けいれん既往・三環系併用・慢性使用で禁忌）
  • dispo：経過観察入院、気道確保、輸液
  • 注意：拮抗剤の半減期が短く、再鎮静のリスク
  • 詳細 → ベンゾジアゼピン中毒

薬剤性脳症

  • 起因薬：セフェピム / セフトリアキソン（type I–III）、バルプロ酸、リチウム
  • 対応：起因薬中止、腎機能低下なら透析考慮（リチウム・セフェピム）
  • 注意：脳波でミオクロニー波、回復に数日かかる
  • 詳細 → 薬剤性脳症
CNS（神経）

脳卒中（脳梗塞・脳出血・SAH）

  • 脳梗塞 4.5 h 以内：t-PA 適応評価、6〜24 h で血栓回収検討 → SCU
  • 脳出血：血圧管理（< 140 mmHg）、抗凝固リバース → SCU / 脳外科
  • SAH：即脳外科コール、再破裂予防の血圧管理（< 140〜160）
  • 詳細 → 脳卒中疑い

髄膜炎・脳炎

  • 経験的治療：CTRX 2 g q12h + VCM + デキサメタゾン 0.15 mg/kg q6h（細菌性）
  • ヘルペス疑い → アシクロビル 10 mg/kg q8h を追加
  • dispo：ICU / 感染症科、髄液採取は CT 後（占拠性病変除外）
  • 詳細 → 髄膜炎・脳炎

NCSE（非けいれん性てんかん重積）

  • 診断：脳波必須、確定したら抗てんかん薬投与
  • 治療：レベチラセタム 60 mg/kg IV、またはホスフェニトイン
  • dispo：ICU / HCU、神経内科コンサルト
  • 詳細 → 非けいれん性てんかん重積

自己免疫性脳炎（抗 NMDA など）

  • 治療：高用量ステロイド + IVIG + 血漿交換（早期に開始）
  • dispo：ICU、神経内科コンサルト、悪性腫瘍検索（卵巣奇形腫など）
  • 注意：けいれん・自律神経障害・不随意運動への対応
  • 詳細 → 自己免疫性脳炎
全身（敗血症・ショック）

敗血症

  • 1 h 以内 抗菌薬（培養取ってから）+ 30 mL/kg 晶質液
  • 目標：MAP ≥ 65 mmHg → ノルアドレナリン
  • dispo：ICU、Source control（ドレナージ・手術）
  • 詳細 → 敗血症

ショック関連

  • 出血性 → 輸血・止血、心原性 → カテコラミン・IABP、分布性（敗血症等）→ 抗菌薬・昇圧、閉塞性 → 解除（PE 血栓溶解、心タンポナーデ穿刺）
  • dispo：ICU、原因に応じた専門科コンサルト
  • 詳細 → ショック対応

📝 カルテテンプレ（S/O/A/P）

S/O/A/P で書く

S：主訴「意識障害」、普段 GCS → 現 GCS、ADL の推移、ROS アレルギー（薬剤・食物）、内服（服薬管理）、既往（入院・手術・定期受診・健診、HT/DL/DM/結核）、最終食事 生活歴：仕事（曝露 / 接触歴）、施設 / 独居、ADL（更衣・食事・移動・排泄・整容）、要支援要介護、Key person、喫煙（本/日×年、現在 / 禁煙済）、飲酒 家族歴：心筋梗塞・脳梗塞、悪性腫瘍

O：Primary Survey（A/B/C/D）、バイタル（HR、BP 左右差、RR、SpO2、BT）、身体所見（心音 S1→S2→S3/S4、murmur、呼吸音ラ音・左右差）、意識障害下での神経学的所見（眼位、指にぎり右手・左手、項部硬直）、採血、心電図、頭部 CT

A：#意識レベル低下 — 鑑別ステップ順記載 First impression から けいれん・失神は否定的として意識障害で鑑別開始 ① ABC 異常の有無（心筋梗塞・大動脈解離・qSOFA） ② 血ガス・血液値（ナルコーシス・Na・Ca・血糖） ③ 意識障害下の神経所見（脳血管病変・細菌性髄膜炎） ④ 頭部 CT（脳出血・ヘルニア） ⑤ 追加検査（Wernicke・甲状腺・粘液水腫・高 NH3・高浸透圧・副腎） ⑤' ここでも決まらなければ 脳波・MRI → 脳外科 / 神経内科 鑑別確認（DONT / VITAL / LABO / BRAIN / それ以外）、AIUEOTIPS、ABCDEs

P：B1 → ブドウ糖 の診断的治療、原因に応じた対応

⚠️ やりがちなミス

  • 血糖を後回しにする
  • B1 を入れずにブドウ糖先行（Wernicke 誘発）
  • CT 正常で安心（超急性期梗塞・CVT・PRES・NCSE は CT で見えにくい）
  • けいれん目撃なし = てんかんではない、と決めつける（NCSE）
  • ABCDEs を考えない（自己免疫性脳炎・CVT・薬剤性脳症を帰してしまう）
  • 精神疾患で結論（器質的原因を全部外してから）
