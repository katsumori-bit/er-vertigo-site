頭痛

⚡ 即時アクション
  1. SAH / 髄膜炎 / GCA / 緑内障を最初に外す（致死・失明）
  2. 雷鳴頭痛 → 頭部単純 CT 即施行（6h 以内なら高感度）
  3. SNNOOP10 red flags を 5 秒で確認（雷鳴・発熱・神経症状・妊娠/産褥・眼痛+視力低下・50歳以上新規・癌/免疫不全/抗凝固薬・乳頭浮腫/体位増悪・外傷/穿刺後）
  4. CT 陰性でも SAH 疑いが残れば LP または CTA（6h 超 / 神経所見あり）

発症様式・ピーク到達時間・初回か・随伴症状・背景（妊娠・癌・免疫不全・抗凝固薬・穿刺/外傷）

  • 発熱 / 項部硬直 / 意識障害
  • 局所神経症状 / けいれん
  • 妊娠中 / 産褥 6 週以内の新規頭痛

  • 癌 / 免疫不全 / 抗凝固薬
  • 乳頭浮腫 / 体位で増悪 / 進行性悪化
  • 外傷後・最近の穿刺操作

パターンfirstED でやること詳細

典型的な一次性頭痛片頭痛・緊張型・群発赤旗がなく以前と同じパターンなら対症療法へ頭痛の対症療法
雷鳴頭痛での CT/LP のタイミング（Ottawa SAH Rule / ACEP）
  • 発症 6 時間以内 + 神経所見正常 + 高性能 CT 陰性 → SAH は実質除外可
  • 6 時間超 or なお疑いが残る → LP または CTA を追加
髄膜炎で CT を先行する条件（WHO）

GCS 低下 / 局所神経徴候 / 脳神経障害 / 乳頭浮腫 / 新規けいれん / 重度免疫不全 該当しなければ CT を待たずに LP → 抗菌薬

🩺 最低限の診察

意識 / 血圧 / 体温 / 項部硬直 / 瞳孔・眼位・眼球運動 / 視力 / 顔面左右差 / 構音 / 上下肢麻痺 / 感覚 / 小脳失調 / 歩行 / 可能なら乳頭浮腫

詳細 → 神経診察

🔬 検査の置き方

検査適応
頭部単純 CT（急ぐ）雷鳴頭痛、神経症状、意識障害、ICP 亢進疑い、外傷後、抗凝固薬、発熱で占拠性病変を否定したい
CTA / CTVSAH 疑いで CT 陰性だが未解決、解離疑い、CVST 疑い、RCVS 疑い
LP髄膜炎 / 脳炎疑い、SAH 疑いで CT 陰性、占拠性病変・ヘルニアリスク評価の上で
ESR / CRP50 歳以上の新規頭痛（GCA）

💊 一次性頭痛の初期対応（赤旗がないとき）

  • 暗室安静・補液・制吐
  • オピオイド（モルヒネ・ペンタゾシン・トラマドール）は第一選択にしない
  • 具体的な薬剤・用量・帰宅処方は → 頭痛の対症療法

📋 救急外来での実戦サマリ

一行ずつ
  • 雷鳴 → SAH first
  • 発熱 / 項部硬直 → 髄膜炎 first
  • 神経症状 / 意識障害 → 脳卒中・脳炎 first
  • 妊娠 / 産褥 → 子癇前症 / CVST / PRES first
  • 眼痛 + 視力低下 → 緑内障 first
  • 50 歳以上の新規 → GCA first
  • 赤旗なし + いつもの頭痛 → 一次性頭痛 → 頭痛の対症療法

⚠️ やりがちなミス

  • 「いつもの片頭痛」と決めつけ神経所見を取らない
  • 雷鳴頭痛で CT 陰性 = SAH 除外と早合点（6h 超なら LP/CTA）
  • 髄膜炎疑いで CT 待ちして抗菌薬が遅れる
  • 50 歳以上の側頭部痛で ESR を測らず帰す（GCA で失明）
  • 妊娠・産褥の頭痛を片頭痛で帰す（子癇前症・CVST）
  • オピオイドで片頭痛を抑える（乱用性頭痛・慢性化）
