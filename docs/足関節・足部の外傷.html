⚠️ トリガー連動の注意（条件で意識）

特定の状況に出会ったら必ず思い出すべき注意点。フローの判断と並行して意識する。

緊急・四肢温存トリガー

  • if脱臼骨折・皮膚緊張・血流障害
    X 線確認前でも整復を検討。整形と相談
  • if開放創
    即時整形コンサルト。創部洗浄・抗菌薬・破傷風
  • if高エネ + 下腿緊満・受動伸展痛
    コンパートメント症候群を疑う。緊急減張
  • if高所転落
    両側踵骨 + 腰椎を必ず評価。脊椎骨折 10% 合併

見逃し回避トリガー

  • if内反捻挫 + 内果のみ痛い
    腓骨近位を触診。Maisonneuve 骨折を除外（足関節 X 線では写らない）
  • if中足骨骨折を見つけた
    斜位像でリスフラン関節の側方転位を確認（見逃し 20%）
  • if内反捻挫の全例
    第5中足骨基部の圧痛を裸足視診とセットで全例確認
  • if外果骨折を見つけた
    内果・後果の圧痛を再評価（二果・三果に格上げで方針変更）
  • if足を捻って外果痛
    ATFL損傷 vs 腓骨遠位端骨折は圧痛部位で鑑別（腓骨直接 = 骨折寄り）
  • if小学校高学年〜中学生の足関節外傷
    靱帯損傷より骨端線骨折を先に疑う（Tillaux／triplane）
  • if突然のポップ音／底屈力低下
    Thompson テストを必ず実施（非典型例 25% 見逃し）
  • if距骨 minor 骨折を疑う（外側突起／滑車／後方突起）
    レントゲン感度 24〜33%。CT を追加して評価
  • if捻挫だが疼痛・機能障害が遷延
    遠位脛腓靱帯損傷を疑う。スクイージングテスト・mortise の medial clear space

ピットフォール

  • ⚠Ottawa Ankle Rule で撮影省略
    日本では除外補助として使う。歩行可否で安心せず、所見があれば撮る
  • ⚠足関節 X 線で「異常なし」
    Maisonneuve は腓骨近位がフィルムアウト。下腿全長を追加
  • ⚠第5中足骨近位端の骨折線
    楔状骨関節面と連続して見えやすい。骨折線イメージを持って読影
  • ⚠微妙骨折タイプの踵骨骨折
    Böhler 角 < 20° で示唆。軸位像も必ず読む
  • ⚠「単独 ATFL 損傷」と診断確定
    遠位脛腓靱帯損傷の併発が多い。スクイージング陽性なら同時想起
📷 撮影方向の備忘
  • 足関節：AP + lateral + mortise（内旋15〜20°）
  • 足部：AP + lateral + 斜位（中足骨・リスフラン評価）
  • 踵骨：側面 + 軸位（Böhler 角測定）
  • Maisonneuve 疑い：下腿全長を追加
  • CT 追加：Pilon／距骨 minor 骨折／踵骨／リスフラン関節脱臼骨折
