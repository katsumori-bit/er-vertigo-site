足関節・足部の外傷

⚡ 即時アクション
  1. Ottawa Ankle / Foot Rules で X 線適応を判定（外果・内果後縁、舟状骨、第 5 中足骨基部、4 歩歩行不能）
  2. X 線は AP + lateral + mortise（足部疑いなら足部 3 方向）
  3. Maisonneuve は腓骨近位を必ず触る、Pilon 骨折は CT で関節面評価
  4. Lisfranc 損傷は見逃しやすい → 中足部圧痛・腫脹あれば荷重 X 線を検討

🚨 Red flags

  • 足背動脈・後脛骨動脈触知不良 / 足部の冷感・蒼白 → 血管損傷
  • 足趾の感覚脱失・運動障害
  • 明らかな変形・脱臼（足関節脱臼骨折・距骨脱臼など）
  • 開放創
  • 下腿コンパートメント緊満（高エネルギー外傷後）
脱臼骨折は速やかな整復

足関節脱臼骨折は皮膚緊満・血流障害をきたしやすい。整復は X 線確認前でも適応となることがある。整形外科と相談。

🩺 最小診察 + Ottawa Ankle / Foot Rules

その場で行う最小セット

  • 視診：腫脹、変形、皮膚色、開放創
  • 触診：外果・内果（特に後縁 6 cm）、舟状骨、第 5 中足骨基部、Achilles、腓骨近位（Maisonneuve）
  • 荷重可否（受傷直後と ER で 4 歩歩行可？）
  • 血流・神経：足背動脈、後脛骨動脈、足趾感覚運動
  • Achilles 腱損傷：Thompson テスト
Ottawa Ankle Rules（足関節）

以下のいずれかで足関節 X 線を撮る：

  • 外果先端 / 後縁 6 cm の圧痛
  • 内果先端 / 後縁 6 cm の圧痛
  • 受傷直後と ER で 4 歩歩行不能
Ottawa Foot Rules（足部）

以下のいずれかで足部 X 線を撮る：

  • 第 5 中足骨基部の圧痛
  • 舟状骨の圧痛
  • 受傷直後と ER で 4 歩歩行不能

すべて陰性なら骨折の可能性は極めて低い（感度 ほぼ 100%）。

📷 画像検査

第一選択

  • 足関節 AP + lateral + mortise（mortise view は内果・外果・天蓋の関係評価に必須）
  • 足部：AP + lateral + oblique
  • Maisonneuve 疑い → 下腿全長 / 腓骨近位を含めて撮影

追加画像

  • CT：Pilon 骨折 / 距骨骨折 / 踵骨骨折 / Lisfranc 損傷の関節面評価
  • 荷重 X 線：Lisfranc 損傷疑いで明らかな転位がない症例
  • MRI：靱帯損傷の精査、occult fracture、骨挫傷
Maisonneuve を見逃さない

足関節痛 / 内果遠位の圧痛だけでも、腓骨近位の骨折（Maisonneuve）を伴う回外外旋型損傷がある。必ず腓骨近位を触診し、所見があれば下腿全長 X 線。

🦴 主要鑑別

病態手がかり進む先
足関節果部骨折外果 / 内果圧痛、腫脹、荷重不能足関節果部骨折
Maisonneuve 骨折足関節痛 + 腓骨近位圧痛、内側裂隙開大Maisonneuve骨折
ATFL 損傷（捻挫）前外側の圧痛優位、骨圧痛軽い、Ottawa 陰性ATFL損傷
第 5 中足骨近位端骨折外側足部の限局痛、内反外傷後第5中足骨近位端骨折
Pilon 骨折高エネルギー、軸圧、関節面破壊 → CT 必須整形コンサルト
Lisfranc 損傷中足部圧痛 / 腫脹 / 足底斑状出血、転位は荷重で顕在化整形コンサルト + 荷重 X 線 / CT
距骨・踵骨骨折軸圧外傷（転落）、腫脹著明、CT で評価整形コンサルト
Achilles 腱断裂底屈力低下、Thompson 陽性、踵上方の陥凹整形コンサルト

📤 disposition

帰宅を考えやすい

  • Ottawa 陰性 / X 線で骨折なし、神経血管異常なし
  • シーネ or 装具固定、松葉杖、RICE 指示
  • 整形外来 1〜2 週で再診（症状持続なら追加画像）

整形コンサルト / 入院

  • 転位のある果部骨折、Pilon、距骨・踵骨骨折
  • Maisonneuve、Lisfranc 損傷
  • 脱臼骨折、開放創、血管・神経損傷
  • 明らかな足関節不安定性
  • Achilles 腱断裂

⚠️ ピットフォール

  • 足関節だけ撮って Maisonneuve を見逃す — 必ず腓骨近位を触る
  • Lisfranc を「捻挫」で帰す — 中足部圧痛 + 足底斑状出血は要 荷重 X 線 / CT
  • Pilon を単純 X 線だけで評価する — 関節面評価には CT 必須
  • Achilles 断裂を見落とす — Thompson テストを忘れない
  • 第 5 中足骨「Jones 骨折」と「裂離骨折」を区別しない — 治療方針が異なる
  • Ottawa Rules を高エネルギー外傷で過信する — 軸圧外傷では distal の所見がなくても踵骨・距骨骨折あり
