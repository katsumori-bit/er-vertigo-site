🚨 ABCD 評価

呼吸困難の最初の数十秒で見るのは 気道 / 呼吸 / 循環 / 意識。SpO2・呼吸仕事量・酸素方針もすべてここに含まれる。

A：気道（Airway）

  • stridor（吸気性喘鳴）／嗄声／舌・口唇浮腫 → 上気道閉塞・アナフィラキシーを疑う → アドレナリン 0.3–0.5 mg IMを遅らせない
  • いびき様呼吸 → 舌根沈下・意識障害合併 → 体位調整・気道確保
  • 喉頭浮腫サイン（喉のつかえ感・嚥下困難）→ 早期挿管考慮

B：呼吸（Breathing）

  • SpO2・呼吸数・呼吸仕事量を同時に見る（数値だけでなく見た目の苦しさ・会話の可否）
  • 補助筋使用・陥没呼吸・起坐呼吸・会話途切れ・チアノーゼ
  • 酸素方針：COPD 疑い SpO2 88–92%／それ以外 94–98%（過剰投与は CO2 貯留・吸収性無気肺）
B で即動く所見と治療
  • 片側呼吸音消失 + 頸静脈怒張 + ショック → 緊張性気胸 → 第 2 肋間鎖骨中線で穿刺脱気
  • ピンク泡沫痰 + 起坐呼吸 + 高血圧 → 急性肺水腫 → NIV（CPAP/BiPAP）+ 硝酸薬 + ループ利尿薬
  • silent chest（呼吸音ほぼ消失） → 重症喘息 → β2 刺激薬連続吸入 + ステロイド静注 + 早期挿管準備
  • SpO2 が酸素投与でも保てない / 呼吸疲弊 → 早期挿管

C：循環（Circulation）

  • BP・HR・脈触知・皮膚（冷汗・蒼白）
  • 頸静脈怒張：心不全・心タンポナーデ・緊張性気胸・PE（右心負荷）
  • 下腿浮腫・片側下腿腫脹：心不全 / DVT・PE
  • ショック所見 → ルート 2 本・モニタ・アドレナリン or 輸液

D：意識（Disability）

  • GCS、見当識、傾眠
  • CO2 ナルコーシス所見（傾眠・羽ばたき振戦・縮瞳）：原因は COPD・神経筋・薬物（オピオイド）→ 過剰酸素投与は撤退、原因に対して BiPAP / 拮抗薬
  • 低酸素性興奮（落ち着きがない・暴れる）は CO2 ナルコーシスの前段階
  • 低血糖を必ず除外
ABCD 維持なら、まず ECG で ACS を拾う

呼吸困難で介入余地が最大の疾患は心筋梗塞。再灌流で予後が一番変わる。ABCD が落ち着いた瞬間に ECG を回す。

疑う疾患聞くこと
心不全心不全の既往、心臓疾患（弁膜症・虚血性心疾患）、浮腫、起坐呼吸、夜間発作性呼吸困難

肺塞栓（PE）突然発症の呼吸困難・頻脈・胸痛、術後／長期臥床／担癌／長距離移動／妊娠・産褥／ピル使用、片側下腿腫脹・痛み、DVT 既往
気胸突然発症の片側胸痛＋呼吸困難、若年やせ型 or COPD・喫煙・気胸既往、外傷／中心静脈穿刺後／胸腔穿刺後
間質性肺炎急性増悪ILD の既往（IPF など）、亜急性経過（数日〜数週）、空咳、感冒様症状の先行

👂 身体診察

聴診だけでなく、呼吸仕事量・皮膚・心音・末梢の所見を組み合わせて方向性を固める。

呼吸仕事量

  • 補助筋使用（胸鎖乳突筋・斜角筋）、陥没呼吸（鎖骨上窩・肋間）、起坐呼吸
  • 会話困難（一文話せない＝重症）、表情・チアノーゼ

聴診

  • 湿性ラ音（crackles） → 心不全・肺炎・ILD
  • 喘鳴（wheeze、連続性ラ音） → 喘息・COPD（ただし心不全の cardiac wheeze もある）
  • 呼吸音減弱・片側消失 → 気胸・胸水・重症喘息（silent chest）
  • 局所性ラ音 → 肺炎
  • stridor（吸気性喘鳴） → 上気道閉塞・アナフィラキシー
  • fine crackles（velcro 様） → ILD・間質性肺炎急性増悪

皮膚・粘膜

  • じんま疹・口唇舌浮腫 → アナフィラキシー
  • チアノーゼ（中心性／末梢性）
  • 発汗・冷汗・蒼白 → ショック・ACS

心音・頸静脈・末梢

  • S3（奔馬調律）／新規雑音 → 心不全・弁膜症急性増悪
  • 頸静脈怒張 → 心不全・心タンポナーデ・緊張性気胸・PE（右心負荷）
  • 下腿浮腫（両側） → 心不全
  • 片側下腿腫脹・圧痛・Homans 徴候 → DVT/PE
  • 胸壁圧痛：気胸の片側胸壁、肋骨骨折の合併、外傷後

🩸 血液検査

最初にまとめて出すもの（全例）

  • 血液ガス（VBG ± ABG）：低酸素 vs 高 CO2 vs 代謝性アシドーシスの判定
  • CRP：感染の評価
  • 血算 + 分画：感染・貧血
  • 生化（腎機能・電解質）：全身評価

鑑別に応じて取捨選択するもの

検査出すべき場面不要な場面
BNP / NT-proBNP心不全 ↔ COPD で迷う／高齢者の呼吸困難／喘鳴の原因が不明明らかな若年喘息発作・既知 COPD の典型増悪
トロポニンACS が鑑別に入る／心不全の増悪因子として MI 評価／高齢者・冠危険因子あり若年喘息・気胸の典型例
D-dimerWells / Geneva スコアで PE 確率が中等度以下（除外目的）高確率群は image first（造影 CT）／高齢者・担癌・術後では偽陽性多くスコア優先
培養（血培・喀痰）発熱＋浸潤影／敗血症疑い明らかな心不全・喘息・気胸
「決めつけ」と「過剰検査」の中間を狙う

COPD に見えて心不全合併・心不全の裏に MI — こういうオーバーラップを拾うために必要なものは出す。一方、若年喘息に BNP・トロポニン・D-dimer をルーチンで出すのは過剰。問診と身体所見で 2〜3 疾患に絞ってから疾患特異的検査を追加するのが ER の標準。

📈 心電図

ABCD が落ち着いた瞬間に 最優先で回す。介入余地が最大の心筋梗塞を拾うため。

  • ST 変化（上昇 / 低下） → ACS（再灌流で予後が一番変わる）
  • 頻脈性不整脈（AF rapid、VT、SVT） → 心不全増悪因子・血行動態破綻
  • 右心負荷パターン（S1Q3T3、右軸偏位、V1-V3 陰性 T、新規右脚ブロック） → PE を示唆
  • 高度伝導障害（完全房室ブロック、Mobitz II、交代性脚ブロック）→ 徐脈性心不全
  • 低電位 → 心嚢液貯留・心タンポナーデ
  • 新規 LVH・電気軸変化 → 慢性心疾患の素地
「呼吸困難」で来た MI を逃さない

高齢者・糖尿病・女性は胸痛なしの ACS（dyspnea-equivalent）が典型。呼吸困難のみで来院しても 初期 ECG＋トロポニンを必ず。

🫁 エコー（POCUS）

肺エコー＋心エコーで 4 軸＋並走 3 の鑑別を一気に絞り込める。

肺エコー（BLUE protocol）

Lichtenstein 流：プロファイルで急性呼吸困難を分類
  • A プロファイル（A-line 優位 + lung sliding あり）→ 喘息／COPD
  • B プロファイル（両側 B-line 優位）→ 心原性肺水腫（感度・特異度ともに高い）
  • A/B または C プロファイル（局所コンソリデーション）→ 肺炎
  • A プロファイル + DVT 所見 → 肺塞栓を強く示唆
  • lung sliding 消失 + lung point → 気胸（CXR より高感度）

心エコー

  • EF：低下 → HFrEF／保たれていれば HFpEF・PE・他原因も考慮
  • IVC 径と呼吸性変動：拡張＋呼吸性変動なし → 右心系過負荷／虚脱 → 脱水・出血
  • RV strain（D-shape、RV ≧ LV、McConnell sign） → 急性 PEを示唆
  • 心嚢液貯留 → 心タンポナーデ
  • 新規弁膜症（重症 MR・AR）→ 急性肺水腫の原因

下肢静脈エコー（DVT スクリーン）

  • 大腿静脈・膝窩静脈の 2 点圧迫法で簡便に DVT を拾える
  • 陽性なら PE の可能性が大きく上がる

📷 放射線画像

胸部 X 線（CXR）

  • 肺水腫パターン（bat wing、Kerley B、胸水） → 心不全
  • 局所浸潤影 → 肺炎
  • 両側びまん性 GGO / 間質影 → ILD 急性増悪・肺水腫・ARDS
  • 片側透過性亢進 / 肺虚脱 → 気胸（少量気胸は見逃しあり、POCUS 併用）
  • 胸水、心拡大、縦隔拡大
  • PE は CXR ほぼ正常なことが多い（陰性でも除外不可）

胸部 CT（造影 / 単純）

  • 造影 CT：PE 確定診断（Wells/Geneva 高確率群）、大動脈解離、縦隔病変
  • 単純 CT（HRCT）：ILD 急性増悪、肺炎の精査、CXR で説明できない呼吸困難
  • 腎機能・造影剤アレルギーで判断
CXR 正常でも安心しない

PE は CXR・聴診ともに正常なことが多い。頻脈・低酸素・原因不明＋リスク（術後・臥床・担癌・妊娠）があれば 造影 CT を先行（D-dimer をスキップしてよい）。

⚠️ やりがちなミス

  • 「COPD だろう」で心不全合併を見落とす（高齢喫煙者は BNP + トロポニン併走）
  • 聴診と X 線が正常で安心する（PE は両方正常が多い）
  • ルーチン高流量酸素（COPD で CO2 ナルコーシス誘発）
  • 方向を決めてから検査（最初にまとめて出す方が遅れない）
  • 胸壁圧痛 = 筋骨格痛と決めつける（PE / 気胸も否定できない）
  • silent chest を「落ち着いた」と評価（重症喘息）
