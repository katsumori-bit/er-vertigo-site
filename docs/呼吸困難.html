呼吸困難

⚡ 即時アクション
  1. 緊張性気胸 / アナフィラキシー / 急性肺水腫 / PE / CO2 ナルコーシスを最初に外す
  2. SpO2 + 呼吸仕事量を見て酸素開始（COPD 疑い 88–92%、それ以外 94–98%）
  3. 方向決定の4 軸：心不全 / COPD / 喘息 / 誤嚥性肺炎（決めつけず BNP + トロポニン併走）
  4. 全例で 血ガス + BNP + トロポニン + ECG + 胸部 X 線をまとめて出す

🚨 鑑別の前に治療を先行する

この時点で即動く
  • ぐったり / 会話不能 / チアノーゼ
  • SpO2 が保てない / 呼吸疲弊
  • ショック所見（低血圧・冷汗・頻脈）
  • 片側呼吸音消失 → 緊張性気胸として減圧
  • 吸気性喘鳴 + 皮膚症状 → アナフィラキシーとしてアドレナリン IM
  • ピンク泡沫痰 + 起坐呼吸 + 高血圧 → 急性肺水腫として NIV

→ これらは鑑別の前に治療を先行する

🩺 いつでも・いかなる時でもやること

  • バイタルサインを全部見る：SpO2・RR・HR・BP・BT・意識レベル
  • 呼吸仕事量を評価する：補助筋使用、陥没呼吸、起坐呼吸、会話困難
  • カルテに記載する：数値だけでなく、見た目の苦しさ・会話の可否も記録
  • 低酸素があれば酸素開始（COPD 疑いなら SpO2 88–92%、それ以外は 94–98%）

🎯 問診で方向を決める

呼吸困難の鑑別は多いが、メインの4つで大部分をカバーできる。 明らかに違う主訴（外傷、アナフィラキシー、異物など）を除き、まずこの4つで考える。

疑う疾患聞くこと
心不全心不全の既往、心臓疾患（弁膜症・虚血性心疾患）、浮腫、起坐呼吸、夜間発作性呼吸困難
COPD喫煙歴、COPD の既往・診断歴、在宅酸素、前回の増悪
喘息喘息の既往、アレルギー歴、吸入薬の使用状況、発作パターン
誤嚥性肺炎（肺浸潤）発熱、咳・痰、悪寒、経過の長さ（詳細は → 誤嚥性肺炎）
一見 COPD でも心不全のことがある

喫煙歴のある高齢者で喘鳴 →「COPD だろう」と決めつけない。 BNP が高ければ心不全を合併している。 心筋梗塞が心不全の増悪因子になっていることもある。 だからトロポニンと BNP は最初から出す。

👂 聴診

  • 湿性ラ音（crackles） → 心不全・肺炎
  • 喘鳴(wheeze) → 喘息・COPD（ただし心不全の cardiac wheeze もある）
  • 呼吸音減弱・消失 → 気胸・胸水・重症喘息（silent chest）
  • 局所性ラ音 → 肺炎
  • stridor（吸気性喘鳴） → 上気道閉塞・アナフィラキシー

聴診所見と問診を合わせて、方向性を固める。

🩸 血液検査を提出する

聴診の後、以下をまとめて提出する。

  • 血液ガス（VBG ± ABG）：低酸素 vs 高 CO2 vs 代謝性アシドーシスの判定
  • BNP / NT-proBNP：心不全の評価
  • トロポニン：ACS の除外、心不全の増悪因子としての MI 評価
  • CRP：感染の評価
  • 血算 + 分画：感染・貧血
  • 生化（腎機能・電解質）：全身評価
なぜ全部出すのか

「これだろう」と決めてから追加で出すより、最初にまとめて出す方が診断の遅れを防げる。 COPD に見えて心不全、心不全の裏に心筋梗塞 — こういうオーバーラップは珍しくない。

📈 すぐに心電図と胸部 X 線

血液検査と並行してすぐ施行する。

心電図で見るもの

  • ST 変化 → ACS
  • 頻脈性不整脈（AF rapid、VT）
  • 右心負荷パターン → PE を示唆
  • 高度伝導障害（完全房室ブロック、Mobitz II、交代性脚ブロック など）

胸部 X 線で見るもの

  • 肺水腫パターン（bat wing、Kerley B、胸水） → 心不全
  • 局所浸潤影 → 肺炎
  • 両側びまん性 GGO / 間質影 → ILD 急性増悪・肺水腫・ARDS の鑑別へ
  • 片側透過性亢進 / 肺虚脱 → 気胸
  • 胸水

🔀 ここで分岐する

この段階で方向性が決まる → それぞれのリンクに進む。

方向性進むリンク
心不全を疑う→ 急性心不全
COPD 増悪を疑う→ COPD増悪
喘息発作を疑う→ 喘息増悪
誤嚥性肺炎（肺浸潤）を疑う→ 誤嚥性肺炎
ILD 既往 + 急性の両側浸潤→ 間質性肺炎急性増悪
4 つに当てはまらないとき

上記 4 つに合わない、または所見の辻褄が合わないときは以下を必ず考える。

  • 肺塞栓（PE）：突然発症、頻脈、胸痛、D-dimer 上昇、術後・臥床・DVT → 肺塞栓
  • 気胸：片側呼吸音低下、突然発症 → 気胸
  • アナフィラキシー：皮膚症状、喉頭浮腫 → アナフィラキシー
  • ACS：胸痛主体、ECG 変化 → ACS
  • 代謝性アシドーシス：DKA、腎不全、中毒 → 代謝性アシドーシス

特に PE は聴診も X 線も正常なことが多い。 呼吸困難の原因が見つからない、頻脈が説明できない、リスク因子がある → 必ず疑う。

⚠️ やりがちなミス

  • 「COPD だろう」で心不全合併を見落とす（高齢喫煙者は BNP + トロポニン併走）
  • 聴診と X 線が正常で安心する（PE は両方正常が多い）
  • ルーチン高流量酸素（COPD で CO2 ナルコーシス誘発）
  • 方向を決めてから検査（最初にまとめて出す方が遅れない）
  • 胸壁圧痛 = 筋骨格痛と決めつける（PE / 気胸も否定できない）
  • silent chest を「落ち着いた」と評価（重症喘息）
