🚨 初期評価（5P・バイタル・外傷歴）

下肢痛で ER に来た患者の最初の数分は、急性下肢虚血（ALI）の 5P 確認と外傷歴・発熱・片側性を切り分けることに使う。ALI は 4–6 時間で非可逆的組織変化に進むため、見逃すと下肢切断・致死につながる。

5P（ALI の特徴的所見）

  • Pain（疼痛）— 突然発症の強い疼痛
  • Pulselessness（脈拍消失）— 大腿／膝窩／後脛骨／足背の触知
  • Pallor（蒼白）— 冷感を伴う
  • Paresthesia（知覚鈍麻）— 神経虚血の進行
  • Paralysis（運動麻痺）— 末期サイン、不可逆寸前
5P 揃ったら手を止めて即コンサル
  • 診断つき次第、未分画ヘパリン静注（80 U/kg bolus → 18 U/kg/h、aPTT で調節）
  • 循環器内科・心臓血管外科に即コール（血栓吸引／Fogarty／血管内治療）
  • 感覚／運動所見を時系列で記録（不可逆性 III 群への進行を見逃さない）
  • 原因検索：Af（塞栓源）、最近の MI、外傷、医原性（CV 穿刺後）

初期バイタル・除外

  • バイタル（BP・HR・SpO2・体温）— 発熱があれば蜂窩織炎・敗血症
  • 外傷歴（受傷機転・転倒・打撲・荷重不可）— 該当なければ画像優先順位下げる
  • 左右差（片側か両側か）— 片側 ＝ ALI/DVT/蜂窩織炎/外傷／両側 ＝ CVI・心不全・腎不全
  • 関節炎所見（発赤・熱感・関節腫脹・歩行不能）— 痛風／偽痛風／敗血症性関節炎
  • 糖尿病背景の有無 — 神経障害で疼痛がマスクされ潰瘍進行を見落とす
「下肢痛 = 整形 / 神経内科」と決めつけない

整形・神経内科に振る前に、必ず脈触知と 5P を確認。腰椎症や糖尿病性神経障害に流すと ALI を逃す。LEAD は多血管疾患（PVD）の一部で、約 60% に他の動脈硬化性疾患（CAD/CVD）を併存（REACH Registry）。

🎯 問診で方向を決める

跛行・下肢痛の鑑別表

鑑別痛みの特徴・誘発・軽減
虚血性（LEAD）下腿の症状が多い。一定の歩行距離で誘発、休憩で数分で軽減。末梢冷感・潰瘍を伴う。内腸骨動脈病変は勃起障害を伴う。
神経性（腰部脊柱管狭窄症）中枢側（臀部〜下腿後面）が多い。日によって誘発距離が変わる。自転車や前屈で軽減（shopping cart sign）。立位／坐位でも疼痛持続することあり。
慢性静脈不全症（CVI）下腿全体、立位／坐位で残存、下肢挙上で改善。色素沈着・静脈瘤。
変形性膝関節症中年以降の女性、立位／歩行開始時にも痛む。膝関節の動作時痛。
急性下肢虚血（ALI）突然発症の強い痛み＋5P。Af・MI 後・血管炎・外傷の既往。
DVT片側下腿のだるさ・腫脹・圧痛。術後・長期臥床・担癌・経口避妊薬。
蜂窩織炎発赤・熱感・腫脹＋発熱。皮膚バリア破綻（外傷・浸軟）の前駆。
痛風／偽痛風夜間の急性発症、第 1 MTP（母趾基部）／膝に発赤・腫脹・激痛。

聞くべき項目

  • 発症様式：突然（ALI／DVT／痛風／骨折）vs 慢性進行（LEAD／CVI／OA）
  • 既往：糖尿病・高血圧・脂質異常・喫煙（LEAD リスク）／Af・MI（塞栓源）／担癌・術後・臥床（DVT）／膠原病・血管炎
  • 跛行距離（一定 vs 変動）と休憩で軽減するまでの時間
  • 外傷歴・受傷機転・荷重可否
  • 発熱・寒戦・皮膚バリア破綻歴（蜂窩織炎）
  • 勃起障害（内腸骨動脈病変＝Leriche 症候群を示唆）

👂 身体診察

靴と靴下を必ず脱がせる。可能なら左右比較で観察。趾間まで観察して潰瘍を見落とさない。

視診（左右比較）

  • 皮膚色調（蒼白・紫斑・チアノーゼ・発赤）
  • 毛量・爪形状の左右差（慢性虚血で脱毛・爪肥厚）
  • 潰瘍（趾間・踵・足底まで）— 部位で原因が分かれる
    • 足趾／趾間 → 虚血性
    • 足底（荷重部） → 糖尿病性神経障害
    • 下腿内側（脛骨内側） → 静脈性
  • 静脈瘤・色素沈着（CVI）

触診

  • 冷感・発汗・熱感（左右で温度差を必ず確認）
  • 脈触知 4 ヶ所：大腿動脈・膝窩動脈・足背動脈・後脛骨動脈
    • 触れなければドプラで確認（連続性・三相性／単相性）
    • 左右非対称・近位で消失する位置から閉塞高位を推定
  • 感染徴候（発赤・圧痛・波動）
  • 関節の発赤・熱感・腫脹・可動域制限（痛風／偽痛風／敗血症性関節炎）
  • 下腿筋緊満（コンパートメント症候群）

注意点

糖尿病背景は疼痛がマスクされる

糖尿病性神経障害により、無症候のまま潰瘍進行することがある。痛みが弱くても潰瘍／壊疽があれば LEAD・感染・骨髄炎を疑い、ABI/SPP と画像で評価。

LEAD は多血管疾患の一部

下肢の評価ついでに、胸部症状・神経学的所見も併せて確認（REACH Registry で約 60% に他の動脈硬化性疾患併存）。労作時胸部症状や心電図変化、透析例の血圧低下を再確認することで他病態の早期発見につながる。

🩸 血液検査

最初にまとめて出すもの（全例）

  • 血算 + 分画：感染・WBC↑（蜂窩織炎・敗血症性関節炎）
  • CRP：感染／炎症評価
  • 生化（腎機能・電解質・尿酸）：造影 CT 適応の判断、痛風で UA↑
  • 凝固（PT/APTT）：抗凝固開始前の baseline

鑑別に応じて取捨選択

検査出すべき場面不要／注意
CK・乳酸ALI 疑い／コンパートメント症候群／長時間虚血の筋崩壊評価横紋筋融解・腎機能合併も同時にチェック
D-dimerDVT が鑑別／Wells score 中等度以下で除外目的担癌・高齢・術後では偽陽性多くスコア優先／LEAD 慢性病変では意義薄い
トロポニン・BNPLEAD 既知の患者で胸部症状あり／多血管疾患の評価下肢痛単独では不要
培養（血培）発熱＋蜂窩織炎／敗血症疑い／壊死性筋膜炎疑い抗菌薬投与前に採取
関節液検査痛風（尿酸結晶）／偽痛風（CPPD 結晶）／敗血症性関節炎（培養・グラム染色）整形にコンサルト
HbA1c・脂質LEAD 確定後の二次予防評価急性期判断には不要
ALI では CK・乳酸を時系列で追う

虚血時間が長くなると CK・乳酸が上昇し、横紋筋融解・腎不全・高 K 血症を合併する。血行再建後は再灌流障害＋コンパートメント症候群に備えて CK・K・Cre を追跡。

📈 心電図

下肢痛主訴で ECG を回す目的は ALI の塞栓源を探すこと。Af（最頻）、最近の MI、左室内血栓のリスクを評価する。

  • 心房細動 / 心房粗動 → 心原性塞栓の最頻原因。CHADS2-VASc で抗凝固歴も確認
  • 陳旧性心筋梗塞（異常 Q 波） → 左室壁運動異常 → 左室内血栓のリスク
  • 新規 ST 変化 → 多血管疾患として ACS 併存の可能性
  • 右心負荷パターン（S1Q3T3、V1-V3 陰性 T、新規右脚ブロック）→ DVT 合併の PE
下肢痛で ECG を取る理由

ALI 患者の塞栓源評価と、LEAD 患者は ACS 併存リスクが高い（多血管疾患）ため。下肢の問題に集中して胸部症状を聞き忘れない。透析例では ALI に血圧低下が重なるため特に注意。

🫧 エコー（POCUS・ABI）

下肢痛の bedside 評価は 動脈（ABI/SPP・血流ドプラ）と静脈（CUS）を同時に当てると鑑別が一気に絞れる。

ABI（足関節上腕血圧比）

  • 正常 1.00〜1.40
  • ≤0.90 で主幹動脈の狭窄／閉塞（LEAD 確定）
  • 0.91〜0.99 はボーダー
  • ≥1.40 は中膜石灰化で偽高値（CKD・糖尿病で多い）→ SPP/TBI へ切替

SPP（皮膚灌流圧）

  • レーザー反射で毛細血管循環を評価
  • SPP ≥ 40 mmHg で治癒見込み
  • < 30 mmHg は治癒可能性低（CLTI 評価）

下肢血管エコー（動脈）

  • 血管径・石灰化・プラーク性状・瘤の評価
  • ドプラで狭窄度・血流評価（連続性・三相性／単相性／消失）
  • 術前評価＋術後フォロー

下肢静脈エコー（CUS）— DVT スクリーン

  • 大腿静脈・膝窩静脈の 2 点圧迫法で簡便に DVT を検出
  • 圧迫不能 → DVT 確定
  • 陽性なら PE のリスクも考慮（造影 CT で確認）

軟部エコー（蜂窩織炎・膿瘍・壊死性筋膜炎）

  • 蜂窩織炎はcobblestone appearance（皮下脂肪の浮腫）
  • 低エコー域＋液体貯留 → 膿瘍／ドレナージ要
  • 筋膜層の不整・ガス像 → 壊死性筋膜炎（外科的緊急）

📷 放射線画像

単純 X 線（XP）

  • 骨折（外傷あり・荷重不可）：股関節・大腿骨・下腿・足部
  • 軟部のガス像 → 壊死性筋膜炎・ガス壊疽（外科的緊急）
  • 骨膜反応・骨破壊 → 骨髄炎（CLTI 合併で重要）
  • 関節石灰化（CPPD）

下肢造影 CT（CTA）

  • ALI 疑い：閉塞部位・側副血行路・血栓の性状
  • 慢性 LEAD：石灰化を含む病変の性状・範囲（VR、MIP、dual energy CT 石灰化除去画像）
  • 動脈瘤・解離・外傷性血管損傷
  • 腎機能・造影剤アレルギーで判断（Cre 上昇例は SPP/MRA 検討）

MRI / MRA

  • 骨髄炎の精査（XP より高感度）
  • 造影禁忌の血管評価
  • 軟部腫瘤・血腫の鑑別

血管造影

  • 通常は治療目的（血管内治療と同時）に施行
  • 診断のみ目的の侵襲的造影は近年減少

🦴 外傷の見方

受傷機転と荷重可否

  • 機転（高エネルギー vs 低エネルギー）・受傷時間・受傷後の処置
  • 荷重不可 ＝ 骨折を強く疑う（高齢者の大腿骨頸部骨折は転倒後に荷重困難で発見）
  • 変形・短縮・回旋（大腿骨転子部・頸部骨折で外旋短縮）

必須の身体所見

  • 圧痛局在・腫脹・皮下出血
  • 関節可動域・荷重テスト
  • 遠位の脈触知＋神経所見（血管・神経損傷の合併）
  • 開放創の有無（開放骨折は抗菌薬＋破傷風＋整形緊急コール）
コンパートメント症候群を見逃さない

下腿骨折・長時間虚血・ALI の血行再建後・打撲血腫・熱傷で発生。不可逆になる前に減張切開。

  • 5P に似た所見＋下腿の異常な緊満
  • passive stretch pain（受動伸展で激痛、特異度高）
  • 腓骨神経麻痺（足背の知覚低下・足趾背屈不能）
  • コンパートメント内圧 ≥ 30 mmHg、または ΔP（拡張期 BP − 内圧）< 30 mmHg で診断
  • 整形外科に即コール、減張切開（fasciotomy）

高齢者の転倒

  • 大腿骨近位部骨折（頸部・転子部）の閾値を低く XP
  • 初診で見逃され MRI で診断されるケースあり（occult fracture）
  • 転倒の原因検索（失神・薬剤・環境）— 詳細は 転倒

軟部組織損傷

  • 捻挫（足関節 Ottawa rule で XP 適応判断）
  • アキレス腱断裂（Thompson テスト陽性）
  • 挫滅創・皮下血腫（広範ならコンパートメント症候群リスク）

「下肢痛」でやりがちなミス
  • 腰椎症と決めつけて ABI を測らない（神経性と虚血性は 合併もある）
  • 糖尿病だから疼痛が弱い ＝ 軽症と判断（神経障害で痛みがマスクされ潰瘍進行）
  • ABI ≥1.40 を「正常」と読む（中膜石灰化の偽高値、SPP/TBI へ切替）
  • ALI を「慢性 LEAD の急性増悪」と誤認して経過観察（4-6h で不可逆）
  • 蜂窩織炎の発赤範囲＞痛みの範囲を見て安心する（壊死性筋膜炎は痛みの範囲＞発赤）
  • DVT の Wells score を取らずに D-dimer 単独で除外する
