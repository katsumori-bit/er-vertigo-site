§1 病歴聴取

腹痛の鑑別はまず以下の 6 項目を聴取することから始める。これでかなり方向性がつく。

  1. いつから？
  2. どの部位が？
  3. 痛みの始まり方は？（Sudden / Acute / Gradually）
  4. 持続痛か？ 間欠痛か？
  5. 悪心嘔吐・下痢は伴うか？
  6. 発熱はあるか？
痛みの性状・強さ・放散痛などはどうする？

性状（鈍い／鋭い）・強さ（NRS）・放散痛・増悪緩解因子も聴取項目だが、まずは「腹痛の鑑別」に焦点を当てた 6 項目を取る。残りは身体診察や検査の流れの中で追加で拾う。

§1.1 発症からの時間「いつからか？」
  • 発症早期に来院 = 痛みが強い。重症化・緊急手術リスクの疾患を含む
  • 「この人手術かも？」と思うのは悪いセンスではない（同じ腸閉塞でも絞扼性は癒着性より早く来院する）
  • 逆に「数日前から」なら緊急手術の可能性は低くゆっくり考える余裕がある
「数日前から」の例外

高齢者・麻痺のある患者・統合失調症では、時間が経って重症化してから来院することがある。

§1.2 部位「どこが痛む？」
  • まず本人に痛む場所を説明してもらう。多くは罹患臓器の直上だが必ずしも一致しない
  • 典型例：急性虫垂炎で「胃が痛い」と訴え上腹部のみ診察 →「急性胃腸炎」と誤診。実際は右下腹部にしっかり圧痛あり
  • 体性痛と違い内臓痛は本人の位置把握が曖昧。「最も痛い場所」は身体所見で求める
  • 主訴が背部痛のみでも要注意：後腹膜臓器・胆道疾患のことがある。腹痛の有無はこちらから聞きにいく
若年男性の心窩部痛（窪田の経験談）

「若年男性によく見られる。痛みに弱いこの世代は初期の心窩部痛を強く感じるのだろうか？」 — 虫垂炎の初期心窩部痛をそのまま訴えてくる若年男性が多い。

§1.3 発症時の様子「痛みの始まり方」

発症様式（onset）を sudden / acute / gradually の 3 つに分ける。初期診断の最大のヤマ。

分類定義主な疑い
suddenある一瞬を境に最強痛血管疾患（出血／虚血）、消化管穿孔（特に上部）、各種捻転
acute数分〜十数分で最強痛急性膵炎、急性胆嚢炎、胆石発作
gradually数十分〜数時間で増強急性虫垂炎、急性憩室炎、腸炎、腸閉塞
同じ「強い上腹部痛＋汎発性腹膜炎」でも

sudden onset → 上部消化管穿孔。acute onset → 急性膵炎。発症様式の聞き分けで方針が変わる。

聞き方の例（sudden vs acute の区別）

「急に痛くなりましたか？」だけでは sudden と acute は区別できない（両方イエスになる）。カルテに「急に痛くなった」とだけ書かない。

  • sudden を引き出す：「まったく痛くなかったのに、ある瞬間にスイッチが入ったみたいに痛くなりましたか？」「それまで痛くなかったのがある瞬間を境にいきなりどーん！と痛みがきましたか？」
  • acute を引き出す：「あれ？ おかしい？ なんか変だ、お腹が痛い、と思っているうちにだんだん痛みが強くなって 10 分後くらいには痛くてたまらなくなりましたか？」
  • gradually の確認：「いつから？」に即答しなければ gradually の可能性。「半日前から？」「昨日の朝から？」と質問していく

sudden onset の鑑別（緊急処置を要する疾患を多く含む）— 「血管疾患」か「消化管穿孔」と考える。

  • 出血性（ショック合併で真っ先に疑う）：破裂性腹部大動脈瘤 / 破裂性腹腔動脈瘤 / 破裂性肝細胞癌 / 子宮外妊娠・卵巣出血
  • 虚血性（痛みの割に腹部所見が乏しい）：急性腸管虚血 / 各種捻転（卵巣捻転 など）
  • 消化管穿孔：上部は sudden が多い → 上部消化管穿孔。下部は sudden / acute / gradually どの形態もあり → 下部消化管穿孔
  • 腹腔外：心筋梗塞（上腹部痛）／大動脈解離（範囲が広い／移動する）→ 腹痛をきたす胸部疾患

§1.4 持続痛か？ 間欠痛か？ 発作性か？
  • 持続痛の聴取は易、間欠痛が問題。「ずっと痛い？」だけでは不十分（間欠でもイエスになる）
  • 間欠痛 = 消化管病変があり虚血／穿孔がない。典型は腸閉塞と腸炎。これ以外で典型的間欠痛を呈する疾患は稀
  • 虫垂炎は持続痛と誤判断されやすい：間欠の間隔が数時間と長くオンオフがはっきりしない。トライツからの距離 ∝ 痛みの間隔（近位空腸 → 数分／遠位回腸 → 数十分／虫垂 → 数時間）
  • 発作性：完全な無症状期間を挟む反復痛。典型は胆石・尿管結石・消化性潰瘍、稀に動脈硬化リスク患者の abdominal angina（SMA 血栓症 → 急性腸管虚血）
聞き方の例（間欠痛をあぶり出す）
  • 今強く痛がっているなら：「今より少し楽な時間がありましたか？」
  • 今楽そうなら：「今より強い痛みが波のように来る時間がありましたか？」
  • 「波がある？」だけだと持続でもイエス。痛みがおさまっている瞬間の有無を確認する
  • 追加：「強い痛みは何回？／何時間おき？／1 回どれくらい続く？／全然痛くない時間はあった？」

窪田の経験談：「虫垂炎の手術を待っていたら『痛くなくなった』という経験がある人も多いのではないか？」

§1.5 悪心・嘔吐、下痢

消化器症状の有無 ≠ 消化器疾患の有無。例外が多いので慎重に判断する。

  • 大腸憩室炎・繰り返す慢性の虫垂炎は消化器症状が乏しい
  • 結石（胆石・尿管結石）や臓器捻転は強い痛み＋悪心嘔吐があるが、下痢はない、嘔吐は持続しない
  • 水様大量下痢があるなら蠕動が保たれている＝汎発性腹膜炎なし（緊急手術になる可能性が下がる「ホッとする」情報）
嘔吐の性状意味
悪心のみ癒着性腸閉塞は否定的。腹部疾患の付随症状か、絞扼性腸閉塞早期の可能性
食物残渣様痛みに伴う、または胃出口部閉塞（小児：肥厚性幽門狭窄症／成人：幽門部胃癌）
水様（黄茶褐色）癒着性腸閉塞の典型。遠位閉塞ほど便汁様、近位閉塞ほど緑色が強い
胃液様（緑色）＋手術歴なしclosed loop obstruction＝絞扼性腸閉塞を念頭に → 小腸閉塞

下痢の性状：

  • 頻回大量水様 → 腸炎を考えてよい → 急性腸炎・感染性下痢症
  • 水様だが回数少ない／少量 → 非特異的、腸炎を想定しない方が無難
  • 白っぽい粘液 → 骨盤内炎症で直腸が刺激された粘液。周囲の炎症（虫垂炎・憩室炎・PID）を探す
  • 高齢で表現が曖昧：下血を「下痢」と言うことがある。実物観察を。hematochezia には特有の臭いがある
§1.6 発熱の有無
  • 発熱は感染症を考える契機。急性腹症は感染性炎症（虫垂炎・胆嚢炎）と非感染性（腸閉塞・穿孔初期）が混在する
  • 発熱なし＝安心ではない。穿孔・腸壊死から重篤敗血症に進むときに低体温を呈することがある（→ 身体所見でチェック）
§1.7 その他の病歴
  • 年齢・性別：NOMI（→ 急性腸管虚血）は若年では稀、下部消化管穿孔は高齢に多い、女性内性器疾患は女性のみ
  • 食歴：「思い当たる食べ物は？」→ 生もの／外食（弁当含む）／集団発生を過去 1 週間以内で確認。アニサキスを疑うなら食材の産地まで踏み込む
  • 旅行歴／月経歴
  • 増悪緩解因子：「歩くと響く」 = 腹膜刺激を示唆
  • 痛みの質：鈍痛は内臓痛を示唆（内臓神経は体性神経より鋭敏でない）
  • 放散痛の有無と場所
病歴は身体所見・検査と並行で

急性腹痛は早期介入を要する疾患を含む。聴取項目をすべて取ってから身体所見は現実的でない。詳細病歴は身体所見・検査と並行で追加していく。

§1.8 High risk patients の抽出

病歴聴取の中で「この人やばいな」という gut feeling を客観的項目に落とし込む。

カテゴリ項目
背景因子（一般）65 歳以上／免疫抑制状態（疾患・薬剤）／アルコール多飲／重篤併存（DM・癌・IBD）／妊娠早期
背景因子（特に注意）心血管疾患（PAD 含む）／腎不全（特に透析）／心房細動・冠動脈疾患・弁膜症
痛みの性状Sudden onset／発症時最強痛（→ 血管疾患示唆）／腹痛発生後の嘔吐（→ self-limited でない）／2 日以内の持続痛
特に危険なフレーズ

「透析の人がおなかが痛い」はそれだけで恐れおののいた方がよいフレーズ（窪田）。Tomino 2014 では透析手術例の予後は著しく不良と報告されている。

「2 日以内の持続痛」の意味

腸管虚血や下部消化管穿孔は適切処置なしで 24 時間を過ぎると重篤化する。数日以上同じ痛みが持続している場合はこれらの可能性は低いと考えてよい（例外はもちろんある）。

§2 身体所見

評価項目は バイタル・視診・聴診・打診・触診 の 5 つに、直腸診・意識レベル・High risk 抽出を加える。

§2.1 バイタルサインの確認
  • 「腹痛＋ショック」はほぼ例外なく緊急の外科処置を要する。原因は出血か敗血症のいずれか
  • 数値以外のショック所見も拾う：末梢冷感・冷や汗 → 出血性ショック／mottling（網状斑） → 敗血症性ショック
  • 体温を忘れない：高齢者の敗血症は若年より低体温になりやすい（Cooper 1994）
§2.2 視診
  • 主要観察項目：手術創／腹部膨満／皮膚色調（黄疸・mottling）／刺青／腹壁・鼠径・大腿ヘルニア
  • 鼠径ヘルニア・大腿ヘルニア（高齢痩せ女性に多く嵌頓しやすい）はパンツを下ろして鼠径部を見ないと見逃す → 嵌頓ヘルニア
  • 膨満は初診では判断難しい。一見して膨満とわかる場合のほとんどは大腸の拡張。小腸病変（腸閉塞）の膨満は初回視診で捉えにくい
  • 膨満すると皮膚が緊張してしわがなくつやが出る。腹部コンパートメント症候群ではテカリさえ出る
Mottling（網状斑）とは

皮膚色調が紫の網状になり一見「色が悪い」と感じる状態。高度末梢循環不全＋アシドーシスを示唆。急性腹症では腸管壊死に伴う敗血症で出る。多臓器不全を伴うことが多く、敗血症性ショックの予後不良因子（Postelnicu 2017）。

§2.3 聴診

腸雑音を 4 段階で評価する。

段階呼称典型場面
① 亢進Hyper active腸炎、機械的腸閉塞
② 正常Norm active—
③ 低下Hypo active虫垂炎、膵炎、麻痺性イレウス
④ 消失Silent汎発性腹膜炎（2 分以上消失で示唆）、腸壊死
腸雑音の覚え方（窪田）

腸が自分自身が調子悪いとき（腸炎・腸閉塞）は "騒ぐ"（正常〜亢進）、他臓器が調子悪いとき（虫垂炎・膵炎）は "黙る"（低下〜消失）。
注意：腸管虚血の発症早期は腸雑音が低下しない。消失していればすでに壊死を示唆する。

音の高さで病態を読む — 弦楽器の例え
  • 弦の長さ＝病的腸管の長さ（長い：低音／短い：高音）
  • 弦の強さ＝腸管内圧（低：低音／高：高音）
  • 弦の太さ＝腸管径（太：低音／細：高音）

絞扼性腸閉塞の closed loop は短く・内圧高く・径細い → 高音 = いわゆる金属音。
癒着性腸閉塞は病的腸管が長く内圧低く径太い → より低い音。
腸炎は閉塞がない（弦の端のしめが緩い） → さらに低い音にしかならない。

§2.4 打診

内容物の推定。腹部全体を丹念に。

音意味典型
① Tympanic（鼓音）よく響く高い音、ガス充満正常胃・盲腸・拡張大腸
② Hyperresonance鼓音と半濁音の間拡張小腸
③ Resonance（半濁音）響き悪いが濁音ではないガスのない正常臓器
④ Dull（濁音）響かない硬い音腫瘍／膿瘍／腸液充満腸管(closed loop)
  • 広範に tympanic → びまん性の腸管拡張（通常は大腸ガス充満）
  • 限局濁音＋圧痛一致 → 膿瘍などの炎症性病変
  • 限局濁音＋圧痛なし → 腫瘍性病変など
§2.5 触診

触診は light → moderate → deep の 3 段階で力加減を変える。light は腹壁が 1 cm も沈まない程度（板状硬は判別可、圧痛は不可）。moderate がメイン。deep は moderate で所見がないとき想定臓器上で試みる。双手法も併用。

評価項目は 5 つ：

項目意味
① 硬さ軟 / 硬 / 板状硬（→ 上部消化管穿孔を強く示唆）
② 圧痛有無・部位・局在性・再現性
③ 反跳痛壁側腹膜の刺激症状（腹膜炎とは限らない）
④ 腫瘤・膨隆急性炎症・腫れた臓器・嵌頓ヘルニア
⑤ 温度虚血部の温度差（窪田の独自所見）
① 硬さ — 板状硬と guarding の整理
  • 板状硬（board like rigidity）：圧痛も反跳痛もわからないほどカチカチ。出現するのはほとんどが 上部消化管穿孔に限られる。胃酸・胆汁の化学性反応
  • 同じ消化管穿孔でも下部は「押したら硬い」所見にとどまる（腸内細菌による感染性腹膜炎 → 下部消化管穿孔）
  • 「軟」を判断するには正常をたくさん経験する必要がある。「あれ、普通とちょっと違うかも？」と思えるかどうか
  • rigidity = involuntary guarding（腹膜炎を示唆）／voluntary guarding＝触診に反応して腹部を硬くする状態（非穿孔性虫垂炎などで腹膜炎は示唆しない）
② 圧痛 — 局在と再現性
  • 局在する場合は固定臓器の病変、局在しない場合は小腸の広範病変
  • 局在するなら数 cm ずつずらしてゆっくり深く押し範囲を特定する
  • 非穿孔性虫垂炎の本当に痛い部位は指 1 本程度の幅しかなく、虫垂の走行が触診だけでわかることもある
  • 腸炎では範囲特定し難く再現性も乏しい
  • 痛いか痛くないかは聞くより本人の反射的反応（表情変化・眉間のしわ・体動）の方が信頼性が高い
③ 反跳痛 — 限定的な特異度
  • 反跳痛＝壁側腹膜の刺激症状。必ずしも腹膜炎を意味しない。虫垂炎・胆嚢炎・重症腸炎で漿膜面まで炎症が達すれば出る
  • 憩室炎では反跳痛があるのが通常
  • どこもかしこも認めれば汎発性腹膜炎
  • 特異度は低い（0.48〜0.60）。tapping（指腹で腹壁を弾く）や pain over speed bump（来院車中で振動が痛かったか）の方がよいとする向きもある
  • 所見の取り方：指 2、3 本でゆっくり圧痛最強点を押し、反動をつけずに一気に離す。陽性陰性は痛いと言うかどうかではなく体の反応で取る

窪田：「単一の身体所見の感度特異度はどれもたかが知れている。だからいろいろな所見をとっているのだ。所詮大きな差がない手法をどうしようか迷う前に、いろいろな方法で所見をとり、腹膜刺激症状のある患者さんの診察経験を積むことが大事」

Heel test（踵をついたときに響くか）は感度高い。歩いてきた人なら病歴聴取の時点で取れる。

④ 腫瘤、⑤ 温度（と稀な触診所見）
  • 腫瘤 = 腫瘍とは限らない：急性炎症組織は硬く触れる（蜂窩織性虫垂炎・膿瘍）／腫れた臓器そのもの（胆嚢炎・痩せた人の腎盂腎炎）／嵌頓ヘルニアは硬い皮下腫瘤として触れる
  • 腫瘍そのものは強い圧痛を伴わないのが特徴
  • 小さなヘルニア（大腿ヘルニアなど）は視診で分からなくても触ればわかることもある
  • 温度：絞扼性腸閉塞や上腸間膜動脈閉塞では虚血部分の腹壁と周囲とで温度差を感じることがある（教科書外、窪田が多用する所見）
  • 稀：触った部分が皮下出血になる（敗血症 → DIC）／皮下気腫（後腹膜経由で達する 下部消化管穿孔）
§2.6 直腸診
  • 近年は施行されないケースが増えている（有用性が限られるとの意見＋医師の不慣れ＋患者の不慣れ）
  • 女性に対して男性医師が行う際は女性職員の立会い必須
  • やり方：示指を挿入し腫瘤・硬便をチェック → 6 時方向(仙骨側、通常無痛)を押して比較基準を作る → 他方向の圧痛を確かめる
  • 9 時方向の圧痛 → 小骨盤に下垂した虫垂（小児・スリム体型）
  • 最も有用なのは PID と虫垂炎の鑑別：cervical motion pain（直腸越しに子宮頸部を動かして生じる痛み）は PID の重要かつ感度の高い所見

窪田：「無意味に全例行う必要はないと思うが、必要時にも省くような方向には向いて欲しくない」

§2.7 意識レベルのチェック
  • 広範囲腸管壊死から敗血症へ進むと例外なく意識障害・呼吸障害・代謝性アシドーシスが進行
  • Sepsis-3（2016）の SOFA スコアにもaltered mentationが項目として記載
  • 虫垂炎・胆嚢炎などポピュラーな疾患でも敗血症移行で軽度意識障害は珍しくない
  • 受け答え可能で済まさず GCS で正しく評価。E or V が 1 点マイナスなら敗血症の可能性を考える
§2.8 High risk patients の抽出（身体所見側）

診察にいたずらに時間を要さぬよう、以下の単独所見があれば赤旗とする。

  • ショックの存在
  • mottling
  • 腹部の高度膨満（皮膚に光沢）
  • rigidity（involuntary guarding）
光沢を伴う膨満腹部＋膨満部が冷たい

窪田：「このような腹部を触診して膨満部が周囲よりも『冷たい』と感じたならば広範囲腸管虚血（壊死）を疑う」 → 急性腸管虚血

§3 初期診断と鑑別疾患

H&P（病歴 + 身体所見）を終えた段階で必ず初期診断と鑑別疾患をあげる。検査をする前に。トレーニングで最も重要な過程。

§3.1 初期診断について
  • 一切の検査をする前に初期診断と鑑別疾患をあげなくてはならない。これが診察である
  • CT は確かに高精度だが、これは「読影」であって「（臨床）診断」ではない
  • 最大の欠点は「臨床症状と画像所見が解離した場合」：人間の症状（痛い）よりも物（画像が異常なし）を優先してしまう。治すべきは患者の症状であって検査異常ではない
  • H&P のみで診断すべき重要疾患は ① 急性虫垂炎、② 消化管穿孔（上部／下部）。典型例は H&P で診断できる（確認の画像はその後）
  • 病歴と所見から虫垂炎・穿孔と診断したら、他疾患であることが判明しない限り安易に初期診断を覆さない
否定材料にならない検査結果
  • 「白血球が上昇していないから」 → 虫垂炎を否定できない
  • 「CRP が上昇していないから」 → 同上
  • 「CT で腫大した虫垂が見えないから」 → 同上
  • 「CT でフリーエアが見えないから」 → 消化管穿孔を否定できない
free air と extraluminal air の違い

用語としての free air（腹腔内遊離ガス）はフリースペースにあるガス。仰臥位 CT なら肝表面など前腹壁の壁側腹膜直下に存在する。しかし消化管穿孔で漏れた腸管ガスは必ずしも free air にならない — 周囲臓器でパッキングされたり腸間膜内穿孔で保たれた場合は病変周囲にトラップされる（= extraluminal air）。
微細な extraluminal air は腸管内と紛らわしく、初診で見過ごされることは珍しくない。外傷 CT では実際の腸管損傷の 50% 程度しか腹腔内ガスを指摘できない（Hefny 2015）。

初診時に鑑別すべき疾患（部位別）

部位鑑別
上腹部痛上部消化管穿孔 / 急性膵炎 / 胆石疾患（胆嚢炎・胆管炎） / 急性虫垂炎
下腹部痛急性虫垂炎 / 下部消化管穿孔 / 子宮外妊娠 / 卵巣嚢腫茎捻転
全般痛腸閉塞（含 絞扼・軸捻転・ヘルニア嵌頓） / 急性腸管虚血

窪田：「腹痛の部位と疾患の相関を記した一覧図はあまり有用でない。図を見比べて診断する人はいないだろう。それより解剖図・正常 CT・手術中所見で臓器の正確な位置を学ぶ方がはるかに有用」

「胃腸炎」「便秘」と診断しない
  • 「胃腸炎」という診断名は一切使用しない。胃炎は通常感染性ではなく、腸炎の主座は小腸 or 大腸。胃と腸の両方が同時に主座になることはまずない。「胃腸炎」とつけると同時にハズレ → 虫垂炎 など他疾患の見逃しが増える
  • 「便秘」も初期診断に使わない。これは病名でなく症状。「便秘」とされた小児が穿孔性虫垂炎、高齢者が下部消化管穿孔だったということを繰り返し経験する
  • 「胃・十二指腸潰瘍／急性胃炎／急性腸炎」は除外診断。緊急疾患を除外せず安易につけない

窪田：「便秘と言いたくなるところを 100 回我慢して 1 回の下部消化管穿孔をみつけることができればその人は名医ではないか」

constipation vs obstipation

英語では constipation（便秘：便が出にくい慢性状態）と obstipation（便もガスも通過しない急性状態）を使い分ける。痛みが発生するのは obstipation。fecal impaction（硬便による直腸閉塞 → 便性大腸閉塞）は急性大腸閉塞であって慢性便秘そのものの症状ではない。直腸膨大部に便が溜まっているだけの高齢者はいくらでもいる。

初期診断が最も難しい疾患

急性腸管虚血（特に NOMI）— 高齢者に多く、訴えが乏しく、初期は痛みの割に腹部所見が乏しいのが特徴。「胃腸炎」と初期診断され hospital delay（早期受診したのに診断が遅れる）となる代表例。
心房細動／透析患者の「高齢者のよくわからない腹痛」では常に念頭におく。

§3.2 鑑別疾患について
  • 鑑別は H&P を終えた後ではなく、同時並行で出ては消えるもの。Google の検索ボックスに用語を入れるたびに候補が変化するイメージ
  • 「下腹部痛」と聞いた段階で疾患が浮かび、本人に会うとそのうちいくつかが消えて別なものが浮上、診察を進めるうちに片手で数えられるくらいに絞られる
  • こうやって進めると陰性所見も拾いやすい（重要疾患の否定に大事）
  • 残った鑑別は1 個でもよい。「最低 N 個あげよ」は不要。診察所見に矛盾する鑑別を延々羅列したカルテを見て「すごい」とは誰も思わない
"よこしまな心" で診察する（窪田流）

「初学者は H&P 終了後に『え〜とー』と考え始める。きっとまっさらな心で患者の話を聞き、まっさらな気持ちで手順通り身体所見を取ったのだろう。でもこんなの診察とはいわない。診察はもっと "よこしま" な心で行う。こういう主訴ならアレじゃないか？ 前はあの疾患だったぞ、ここにこんな所見があるとはさてはアレか？ — 自分の思いつき（実はコレが臨床能力）に素直に従うのだ」

そして思いついた疾患を示唆する所見と示唆しない所見を上げてゆく。示唆する所見が多いとき、ここで初めて客観性が生じる（診断スコアはこの定式化）。

"A Little Book of Doctor's Rules"："An acute surgical abdomen is when a good surgeon says it is an acute surgical abdomen. There is no other test for it."

§3.3 ROS（Review of systems）
  • focused examination（主訴・症状から疑う部位に集中した診察）の対極
  • ROS はまっさらな気持ちで、手順通り順番にシステムごとに淡々とチェック
  • そのなかで「あっ！」という発見がときにあり、それが今回の疾患に直結することもある — 臨床って面白いと感じる瞬間

§4 確定診断のための検査

多くの腹部疾患は血液・尿は補足的で、画像検査で確定診断になる。常に 初期診断と鑑別疾患を確認する最低限の検査を行う。

検査の前に答えを用意せよ

「なぜ血液検査をしたのか？」「なぜ胸部立位 X 線を撮ったのか？」「なぜ腹部 CT を撮ったのか？」と聞かれて、「虫垂炎を疑い白血球上昇を見るため」「上部消化管穿孔を疑いフリーエアを確かめるため」「急性膵炎を疑い膵臓の状態を確認するため」 のように即答できなければ検査の意味がない。

とくに CT は被曝する。妊婦でなくとも若い女性や小児では安易に撮らない。

定型的な検査（あえて挙げるなら）：

  • 血液検査：血算・生化学（AST/ALT、ALP/γ-GT、T/D-Bil、s-Amy）
  • 尿検査：定性・沈渣
  • 腹部単純 X 線：立位・臥位
  • 腹部超音波
  • 40 歳以上の心窩部痛・高齢者は心電図（急性心筋梗塞を見落とさない）
  • 妊娠可能女性は尿で妊娠反応（妊娠関連疾患の有無確認＋ X 線前の確認）
§4.1 血液検査
  • 多くの腹部疾患では特異的所見がない。WBC・CRP の上昇はばらつきが多く、重症化で下がることもあり診断的価値は低い
  • 比較的特異性のある項目：
    • 胆道系酵素 → 胆管疾患（急性胆管炎・総胆管結石症）
    • アミラーゼ／リパーゼ → 急性膵炎（輸入脚症候群でも二次性に上昇）
  • 誤解されやすい点：胆嚢疾患（胆石症・急性胆嚢炎）では通常トランスアミナーゼ／ビリルビンは変動しない
§4.2 尿検査
  • 最も頻用は妊娠反応検査
  • 尿管結石の血尿（含 潜血）、膵炎のアミラーゼなど疾患特異的な指標もあるが、意義は補足的
§4.3 単純 X 線
  • 近年は救急室での腹部 X 線撮影は省略傾向。CT が簡便・短時間・高解像度で代替できるため
  • 初診時に意義があるのは：
    • 繰り返す腸閉塞（過去像と比較できる） → 小腸閉塞
    • 若年健康者の上部消化管穿孔 → 上部消化管穿孔（フリーエア検出）
  • フリーエア検出能：5 mL あれば指摘可、条件付き（5〜10 分立位後の立位胸部 X 線）なら 1〜2 mL のエアでも指摘可（Billittier 1996）
  • 上部消化管穿孔で撮るのは胸部 X 線。腹部 X 線対象は結果的に腸閉塞疑いのみ
窪田の経験：1 枚で 1 発診断 — gasless film

腸管ガスがほとんど見えない "gasless film"（胃泡と少量のガスは容赦）。比較的頻度の高い 2 つ：

X 線像病態典型ストーリー
右下腹部痛＋ガスレス右側大腸閉塞（回盲弁機能正常の上行結腸癌など）泥状便で閉塞、その先のガス排出済、回盲弁が小腸ガスを止める → 小腸大腸ともガスレス
食後嘔吐・体重減少＋ガスレスGastric outlet obstruction（幽門部胃癌など）胃の先にガスが通過しない

その他：全大腸型大腸炎（潰瘍性大腸炎など）— 全大腸の壁肥厚＋大腸ガス消失でほぼガスレス。

癌年齢の患者がガスレスを呈したら、右下腹部痛なら上行結腸癌、食後嘔吐＋体重減少なら幽門部胃癌と診断できる。既知の悪性腫瘍があれば「閉塞した」「増悪した」の病状変化把握にも使える。

§4.4 超音波検査
  • 検査と診察の中間：術者によって結果が変わり、リアルタイムでしか評価できない
  • 特性：局所解像度に優れる（ミリ単位の細かい病変）。一方エア・骨が介在すると見えない、深部も苦手
  • プローベ：固形臓器・深部はコンベックス（3.5〜5 MHz）／腸管・浅い部位はリニア（7〜10 MHz）
  • 超音波が CT より優位な場面：
    • 穿孔性虫垂炎（CT で見えない場合）
    • 小児・痩せ身の女性
    • 結腸憩室炎
    • CT に写らないコレステロール結石
  • 欠点：肥満者・腸管ガス多量で精度低下、立体的構造は頭で組み立てる必要、死角あり
  • 非特異的腹痛の精査でCT 前に超音波を行うと診断率向上＋被曝低減できる前向き研究あり（Laméris 2009、Jang 2014）
§4.5 腹部 CT
  • 禁忌がない限り造影 CT が原則。腹部内臓は造影しないと評価が難しく、出血や梗塞も単純では分かり難い
  • 腎毒性に留意。検査の利益とリスクを秤にかける
  • 腸閉塞・虫垂炎など多くは造影 CT のみで評価可能 → 小児・若年者では単純 CT カットも考慮可
  • 単純＋造影どちらもあった方がよい：総胆管結石、急性期血栓
  • 単純のみで十分：尿管結石
「とりあえず CT」はいただけない

窪田：「H&P でまったく鑑別がついてないままに、全然わからないから CT で調べてみようというケースを散見する。いくら感度特異度が高いといっても 100% ではない。あくまで検査であるので、その目的を明確にすることが大事」

検査の目的は疑っている疾患の存在確認 or 否定したい疾患の不在確認であって、「とりあえず CT」ではない。

「ウォーリーを探せ」— 鑑別がないと CT は読めない

窪田の症例：子宮筋腫手術歴のある高齢女性が腹痛で来院。初診時 CT でびまん性小腸拡張あり → 腸閉塞と診断され保存治療開始。実際は下部消化管穿孔で、5 日目に手術。「拡張腸管」と思った部分は穿孔した腸液が周囲小腸にパッキングされた像だった。

情報を整理すると：

  • 来院 1 週間前から体調悪く食欲低下
  • 嘔吐なし
  • 強い自発痛なし、下腹部に限局した圧痛

典型的な癒着性腸閉塞像とまったく合致しない。担当医は鑑別が浮かばないままゼロベースで CT に診断を託した → びまん性小腸拡張＋手術歴 = 腸閉塞 という短絡。消化管穿孔はまったく頭になかった。

「『ウォーリーを探せ』でウォーリーが見つけられるのは顔をあらかじめ知っているからであって、漫然と眺めた後に『この顔の人どこにいた？』と聞かれても誰も答えられない。人は頭のなかにないことはアウトプットできない」

下腹部痛で鑑別が浮かばないなら、① 急性虫垂炎、② 下部消化管穿孔、③ 子宮外妊娠／卵巣捻転を念頭に置いてから CT を見る。そうすれば extraluminal air を必死に探すし、「これ腸管？」「腸間膜内穿孔？」のように見え方が変わる。

§4.6 MRI
  • 急性腹痛のメジャー検査ではない
  • 妊婦の腹部検査（特に虫垂炎を疑った際の超音波と CT の間）として推奨されつつある（Long 2011）
  • 多用は胆道系評価（MRCP）：胆嚢摘出術前の胆嚢管位置・変異チェック、総胆管結石の有無確認 — 診断より治療方針決定の意義
  • 急性胆嚢炎：胆嚢壁肥厚は超音波が勝り、総胆管結石は MRCP が勝る（Park 1998）

§5 確定診断がつかないとき

確定診断は単一の疾患名でつけるのが原則（虫垂炎／癒着性腸閉塞／細菌性腸炎 など）。曖昧な「イレウス／胃腸炎／便秘」を診断名にしない。検査を終えても診断がつかない場合の方針は① 帰宅／② 入院経過観察／③ 緊急手術・診断的腹腔鏡 の 3 つしかない。

まず先にあげた鑑別すべき疾患（上腹部・下腹部・全般痛それぞれ）に該当しないか、High risk patient として抽出されていないかをもう一度確認する。該当があれば対象疾患の H&P を取り直し、未施行の検査をオーダーする。

§5.1 帰宅して経過観察

下部消化管穿孔と腸壊死以外のすべてが否定的、かつ以下を満たす場合のみ：

  • 重篤な基礎疾患がない
  • 強い自発痛がない
  • 筋性防御／反跳痛がない
  • 圧痛なし（あっても軽度）
  • 経口摂取が可能（嘔吐がない）
帰宅説明の言い方（窪田の例）

右下腹部を押さえて：「正直なところ今の時点では診断がはっきりしていません。症状が強くないのでいったん帰って様子を見ますが、痛みが強くなったり、吐き気が強くなったり、ここを押したときに強い痛みを感じるようであればまた来てください — 虫垂炎の可能性があります」

「虫垂炎」と疑っている疾患名をはっきり伝える。この一言で「見逃された」と言われなくなる。

帰宅した場合は外来でフォローすること（自身のスキルアップにもつながる：どんな場合に帰して大丈夫だったかを結果から学ぶ）。

§5.2 入院して経過観察

筋性防御・反跳痛なしだが痛みが強い／経口摂取不能の場合は入院して再評価する。経過観察＝放置ではない。中途半端な治療行為をしないのが鉄則。

① 疾患に対する治療を不用意に始めない

  • 診断がついていないのだから治療は行いようがない
  • 「熱がある／CRP が上昇している／感染症の可能性」だけで抗菌薬を始めるのは厳禁。培養を提出した上で対象疾患と起因菌を明確にしてから
  • 例外：敗血症は 1 時間以内の抗菌薬開始が強く推奨される（Gaieski 2010）。ただし敗血症と判断したならそもそも経過観察の対象外

② 症状に対する治療（対症療法）も注意

  • 痛み止めで一時的に良くなっても解決していない。治療はまだ行っていない
  • 「痛み時 ○○」の自動指示で看護師が延々と繰り返した結果、回診時にまったくよくなっていない…ということが起きうる
  • 痛み止めの使用そのものは診断を遅らせない（Brewster 2000、Thomas 2003）。Cope の「診断のついていない腹痛には痛み止めを使うな」は今は当てはまらない
  • ただし何度も痛み止めを必要とする = 治療介入を要する重篤な状態（要するに手術）と認識を改めるべき
  • ブチルスコポラミン（ブスコパン）は鎮痙作用のみで直接の鎮痛効果はない。腸管攣縮が原因のときはよいが、原因不明の腹痛には鎮痛成分を持つ薬剤を優先。胆石発作・尿管結石でもブスコパンの優位性は示されていない（Kumar 2004、Holdgate 2005）
経過観察で予想される 3 つの結末
  1. （診断がつかなかったが）症状軽快し self-limited な疾患だったと解釈
  2. 時間経過で所見が分かりやすくなり診断がつく
  3. 症状増悪して（診断はつかないものの）治療介入（開腹手術など）せざるを得なくなる

この間に必要なのは NPO（経口摂取禁）＋ 補液＋観察のみ。

§5.3 緊急手術／診断的腹腔鏡（外科コンサルト）
  • 身体所見で High risk（ショック／mottling／腹部高度膨満／筋性防御）と認識されたら、診断がついていなくても外科コンサルトの理由になる（de Dombal 1994）
  • ショック・敗血症併発なら検査結果を待っている余裕はない
  • このような状態は手術自体が診断的意義をもつ。画像診断は必須ではない。CT でも異常があるのは明らかだが、どこが異常かわからないことも珍しくない
  • 結果的に多いのは消化管穿孔か腸管壊死（絞扼性腸閉塞含む）
  • 診断的腹腔鏡のオプション：高齢者の NOMI、若年女性の PID vs 虫垂炎鑑別など、CT で診断できず肉眼所見で診断できる対象がある
"使えないかもしれない指導医リスト"（窪田）

パンペリ＋敗血症性ショックは mortality が高く、治療タイミングを逸すれば救命困難。コンサルトを受けた外科医が…

  • 「診断は何？」「CT 撮ってないの？」「ワイセは上がってるの？」を必要以上に聞いてくる → 「使えないかもしれない指導医リスト」に載せてよい
  • 「CT できたら呼んで」と診にも来ず電話で返事 → 「やばいかもしれない指導医リスト」にノミネート

「外科医にどやされるようなことがあってもコンサルトしたあなたが絶対的に正しい！」

§6 経過観察と再評価

入院後の経過観察は「入院させて放っておく」ではない。状況に応じたフォローアップ計画をつくる。診断がついても当たっているとは限らない（指導医が診たところで研修医より精度は上がらないという報告もある — Acute Abdominal Pain Study Group 2016）。まっさらな気持ちでもう一度レヴューする周到さが必要。

§6.1 確定診断がついている場合

① 保存治療が奏功すると予想される疾患（→ 退院計画）

  • 本来は外来治療対象だが入院を選択したケース。急性腸炎／合併症のない大腸憩室炎など
  • 入院後すぐに退院基準（criteria）を作る。例：「憩室炎：自発痛がなくなり、食事再開しても痛みの増悪がなければ退院」

② 保存治療適応だが増悪可能性あり（→ フォローアップ計画）

  • 急性膵炎／虚血性大腸炎などが該当。重症化を想定したフォロー計画を立てる
  • 急性膵炎は重症化で死亡率 30%、虚血性大腸炎が腸管壊死に移行すれば救命困難 — hospital delay だけは避けたい
  • 診察頻度（1 日 1 回？ 4 回？）と増悪の指標（indicator）をリストアップ
例：虚血性大腸炎のフォローアップ計画
  • 現在の状態：痛みは自制内、腹膜刺激症状なし
  • 増悪の指標：痛みの増悪または腹膜刺激症状（反跳痛・タッピング痛*）の出現
  • 診察頻度：1 日 2 回（腸管は約 12 時間の虚血で不可逆的な壊死となるため、最低 12 時間に 1 回）
  • チェック項目：自発痛の増悪、反跳痛・タッピング痛の有無（筋性防御は問わない — 全層変化が及んだ時点で腹膜刺激症状が出ると解釈）

*タッピング痛＝指先で腹壁を軽く叩いた時の痛み。文献では pain with percussion / pain on percussion。

③ 手術適応もあるが保存治療を選択した場合（→ 手術計画）

  • 穿孔性十二指腸潰瘍（保存）／急性胆嚢炎（保存）など
  • 増悪時に手術に踏み切る条件をあらかじめ細かく設定する。これがないと hospital delay になりやすい
  • 胆嚢炎は TG18 などガイドラインに従う。上部消化管穿孔のように明確基準のない疾患は症例ごとに設定
例：穿孔性十二指腸潰瘍に対する手術計画

現在の状態：

  • バイタル安定／尿量減少なし
  • 上腹部に限局した反跳痛と voluntary guarding
  • 自発痛は痛み止めでコントロール

手術基準（以下のいずれか 1 つでも抵触したら）：

  • バイタルサインに変化
  • 尿量減少（< 5 mL/kg/hr）
  • 下腹部に痛み／反跳痛が広がる、または involuntary guarding が出現
  • 痛み止めが経時的に増量

条件を決めることでスタッフ全員に注意が喚起され、変化を早めに見つけられる効果が期待できる。

この方針の最大のピットフォール

そもそも診断が間違っている可能性（Crofts 1989）。「上部穿孔だから保存治療しているよ」と申し送られて鵜呑みにしていると危険 — 実は下部穿孔かもしれない。

§6.2 疾患の局在性はわかったが診断に至っていない場合
  • 虫垂炎疑い、上部消化管穿孔疑いだが診断に至らないケース
  • 予想疾患をリストアップ → self-limited なものは消去し、surgical な疾患のみを対象に
  • 中途半端な治療（抗菌薬の処方など）はせず、絶飲食＋補液＋必ず再評価
  • 翌日になって増悪し確定診断もつかないなら、想定対象疾患を念頭に手術を検討
第三者がカルテを読んだらどう思うか？

一通り診察が終わって「今日は手術せず様子をみよう」と決めたあと、もう一度診療録を眺めてみる。

  • 診察した人のなかに 1 人でも「右下腹部に反跳痛あり」と記載していないか？
  • 手術しないと決めた理由は本当に症状で決めたのか？ それとも「画像でハッキリしなかった（CT でよくわからなかった）」ことが理由になっていないか？

窪田：「結果的に時間がたってから手術となった hospital delay のケースで、初診の診療録に典型的な病歴・身体所見が書いてあったのに、画像診断できなかったがために手術に踏み切れなかった、ということをときどき経験する」

§6.3 全然わからない場合

① びまん性の腸管拡張を認める場合

  • 腹部異常膨満や X 線でびまん性の腸管拡張があるなら、必ず疾患が隠れていると考える
  • 麻痺性イレウスとして小腸がびまん性に拡張しているのに「小腸イレウス」と書かれたからといって減圧だけで様子見をしてはならない
  • 方向性がつくまでは思考停止せず、診断のための努力を継続。増悪なら診断のための手術（審査腹腔鏡）もある
  • 麻痺性イレウスを呈した時に想定すべき病態は 腸管虚血（壊死） と 下部消化管穿孔 の 2 つ。High risk patient ならこの 2 つを否定するまで念頭から外さない

② 痛みはあるが、有意な所見を認めない場合

  • 自発痛だけ強く、身体所見は乏しい。高齢でも基礎疾患でもない。画像も異常がはっきりしない
  • 当日は積極的に痛み止めを使い再評価
  • 注意：腸管虚血の早期はこの群に入る

§7 各種アプローチ法

5 つの切り口（緊急性／罹患部位／痛みの性状／随伴症状／女性）から鑑別を組み立てる。

§7.1 緊急性からのアプローチ

腹痛にも優先順位がある：

緊急性の優先順位

ショック > 敗血症（septic abdomen）> 汎発性腹膜炎（のみ）> 強い自発痛（のみ）

「痛いよー」と叫ぶのは優先順位が低い。重症度が上がるほど意識障害が進行し元気に「痛いよー」と言えなくなる。

① ショックの場合 — 出血／敗血症／脱水に分ける

パターン典型疾患キモ
① 出血性（発症早期のショック）血管疾患（破裂性 AAA・破裂性腹腔動脈瘤）／肝脾（肝細胞癌破裂・脾破裂）／婦人科（子宮外妊娠・卵巣出血）エコーで腹水確認、悩めば穿刺。迅速止血（手術 or IVR）がすべて
② 敗血症性（時間経過長め）a) 原疾患→腹膜炎→敗血症：広範腸壊死／下部消化管穿孔。
b) 原疾患→敗血症（腹膜炎経由しない）：急性胆管炎／重症急性胆嚢炎／一部の穿孔性虫垂炎		a は死亡率高く全身管理＋集中治療。b は原疾患への治療が奏効しやすい
③ 脱水性（internal fluid shift）急性膵炎／腸閉塞／重症腸炎／上部消化管穿孔（数日経過）外から見えない脱水。上部穿孔のショックを敗血症と決めつけない（高浸透圧消化液で著しい脱水になる）

② 敗血症（Septic abdomen）

  • 所見：ショック／呼吸不全／軽度黄疸（胆管炎以外で）／低体温／mottling／意識障害
  • 広範腸壊死や腹部全体に便汁が広がる下部消化管穿孔は catastrophic abdomen
  • 代謝性アシドーシスを伴う。敗血症で他に説明のつかないアシドーシスを見たら septic abdomen を疑う
  • 術前画像診断が困難なことも多く、その場合は腹水穿刺（US ガイド下）で色調・臭い・培養を取る
腹水の色調・臭い早見表（窪田）

色調：

色調示唆
黄色混濁（turbid ascitis）虫垂炎の可能性。穿孔も壊死もない虫垂炎でも珍しくない（濁りはフィブリン析出。膿瘍の creamy pus とは別物）
淡血性（serosangenous）腸管虚血（絞扼性腸閉塞初期など）
血性（bloody）鮮血 → 出血。黒褐色 → 時間経った腸壊死（壊死なら悪臭）
胆汁性純胆汁 → 胆汁瘻。緑色 or 黄色混濁の腸液 → 上部消化管穿孔または近位空腸穿孔
便性（feces ascitis）下痢便そのもの → 大腸または回腸の穿孔

臭い：

臭い示唆
嫌気性臭（anaerobic smell）多くは B. fragilis（一度嗅げば覚える）
酸性臭（acidity smell）消化液・腸液の酸えた臭い → 上部消化管穿孔
便臭（feces smell）下部消化管穿孔
無臭（not smelly）単一菌増殖（多くは E. coli・Klebsiella）→ 消化管穿孔は否定的。穿孔性虫垂炎や（高齢女性なら）破裂性子宮留嚢腫を想起

腹水の生化学（アミラーゼ・ビリルビン）が同時期の血清値を上回れば腸液混在の可能性大。グラム染色で poly microbial なら腸内細菌由来＝消化管穿孔を高率に示唆。大腸壊死では腹水がほとんど出ないこともあるので注意。

③ 汎発性腹膜炎（Panperitonitis）— 3 ヴァリエーション

パンペリ＝開腹手術と認識（一部例外あり）。

タイプ典型疾患所見の特徴
① 板状硬（Board like rigidity）上部消化管穿孔カチンコチン、反跳痛を確かめる余裕すらない、強い自発痛（化学性腹膜炎）
② 反跳痛が著明下部消化管穿孔ゴム風船様の弾力ある膨満、どこもかしこも反跳痛、自発痛は反跳痛の強さに比してやや弱い（細菌性腹膜炎）
③ Focus が明らか穿孔性虫垂炎／穿孔性胆嚢炎圧痛・反跳痛が一点で最強、離れるほど弱まる

例外（汎発性腹膜炎を呈するが手術が治療にならない）：急性膵炎／原発性細菌性腹膜炎／重症急性腸炎（炎症が漿膜に達した場合）

§7.2 罹患部位からのアプローチ
  • 最も汎用される実際的アプローチ。「手の何センチ下にどの臓器がどう重なっているか」を目をつむってイメージできる解剖知識が前提
  • トレーニング：何も見ずに腹部臓器の位置関係をスケッチ／開腹を観察／互いにエコー／他人の CT もとりあえず見る
  • 初期診断の部位別鑑別（上腹部／下腹部／全般痛 → §3.1 参照）を骨格に
痛い場所と罹患臓器が解離する 3 パターン
  • ① 自発痛の位置との解離（放散痛）：虫垂炎の上腹部痛、胆嚢炎・膵炎の背部痛、尿管結石の尿管走行に沿う痛み、小児精巣捻転の下腹部痛。放散部位に圧痛はない — 全身を診察するのが必須
  • ② 罹患部位が想像より広がっている：左下腹部痛だが虫垂炎（痩せ身で虫垂が長く左下腹部まで届く）、右下腹部痛だが膵炎（炎症性浸出液が腸間膜根部から右骨盤腔に達する）。圧痛の最強部・中心部を丹念に探す
  • ③ 臓器が想定位置にない：先天的（内臓逆位／腸回転異常／左側胆嚢／馬蹄腎・片腎）／後天的（摘出済 — 胃全摘で消化性潰瘍にならない、胆嚢なしで胆嚢炎なし、膵切除後に膵炎は稀）／高度食道裂孔ヘルニアで胃が胸腔内／高齢者で肝が高位（典型的な Murphy が出ない）／高齢女性で椎体圧迫骨折により肋骨弓が骨盤腔に入り込み胆嚢炎を右下腹部痛・虫垂炎を右上腹部痛として表現
§7.3 痛みの性状からのアプローチ

「痛みをなんとかしてくれー」と訴えられる疾患は大きく 2 通り：

タイプ定義典型疾患本質
① 七転八倒型もんどり打って動き回る、じっとしていられない結石性疾患（尿管結石・胆石発作）腹膜刺激症状なし → 動いても痛みが増えない（内臓痛）
② 苦悶型動くと痛みが増えるのでじっとしている、冷や汗をかいて耐える、ときどき体位変更上部消化管穿孔（座位）／急性膵炎（側臥位 chest-knee）腹膜刺激症状あり → 重篤であることが多い
七転八倒型は安心できる？

七転八倒型は腹膜刺激症状がないため一見ホッとできる。ただし安心は痛みが治まってから。臓器虚血（含む捻転）でも七転八倒型を呈する — 心房細動・卵巣腫瘍などのリスクファクターを念頭におく。

痛みがマスクされる／和らぐ場合（要注意）

痛みがマスクされる基礎疾患（自分から訴えない）：麻痺（脊損後・脳血管後）／高齢者／統合失調症／悪液質／糖尿病／妊娠

痛みが和らぐパターン（治っていない）：

  • 穿孔性虫垂炎：穿孔の直前が最も痛い、穿孔直後に自発痛は少し和らぐ
  • 梗塞（上腸間膜動脈閉塞など）：虚血の時期は痛く、壊死してしまえば痛みはなくなる

窪田：「絞扼性腸閉塞で手術に向かうとき、苦悶の表情で痛がっているなら『まだ間に合うぞ（腸は生きている、腸切除しないでいけそう）』と外科医は考えている。逆に手術場に向かうエレベーターで『だいぶ楽になってきました』と言われたら『あっ、もう間に合わないかな（腸切除必要かな）？』と考えてしまう。痛みは重要サインだがときに逆を意味する」

§7.4 随伴症状からのアプローチ

大きく区分するときは 消化器症状（悪心嘔吐／下痢）と発熱の有無に着目。

① 消化器症状 — 本管 vs 側管モデル

悪心・嘔吐：本管／側管／虫垂で分ける

消化管を 本管（胃・小腸・大腸）と 側管（胆管・胆嚢・膵臓・憩室）に分けて考える。

  • 本管は悪心が強い。口側ほどさらに強い
  • 本管とも側管ともいえない虫垂は吐き気はあるが長続きしない
  • 側管は悪心が弱い（強い痛みに伴う吐き気はある）
  • 後天的にできた側管（憩室）は悪心ほぼなし

下痢の解釈：

  • 大量水様 → 急性腸炎
  • 頻回だが少量 → 骨盤内炎症（膿瘍など）の存在
  • 下痢どころか軟便もない → 急性腸炎を初期診断から一旦除外（可能性は残るが鑑別から外す方が得策）

② 発熱 — 窪田流アプローチ

  • Common disease（虫垂炎・胆嚢炎）は発熱の有無を重視しない
  • 明らかな発熱あり → 感染性疾患から鑑別
  • 平熱 → 非感染性疾患から鑑別（腫瘍など）
  • 低体温＝敗血症（septic abdomen）
  • 悪寒戦慄（shaking chill）＝外科的感染症が敗血症化した状態。緊急ドレナージを要する可能性。典型は急性胆管炎と閉塞性腎盂腎炎

③ 黄疸

  • 診断的には胆管炎を考える
  • いかなる疾患でも敗血症に至ると軽度黄疸を伴う（Sepsis-3 の臓器障害スコアにビリルビン値が含まれる） → 重症度評価としても用いる

④ 腹部以外の痛み

部位示唆
背部痛・腰痛後腹膜病変（大動脈含む） → 破裂性 AAA／膵炎
右肩甲骨部胆石／胆嚢炎
左肩痛脾臓／膵臓／左横隔膜
右肩痛肝臓／右横隔膜
下肢痛血管疾患（大動脈解離など）／閉鎖孔ヘルニア
§7.5 女性へのアプローチ

すべからく男女平等の世の中だが、腹痛については女性は別視する（婦人科疾患・妊娠の存在）。詳細は原著第 5 章 ①（婦人科疾患）参照。

① いつ婦人科疾患を考えるか

  • 消化器症状（悪心嘔吐・下痢）に乏しい
  • 発熱を伴わない強い腹痛、持続痛
  • 例外：卵巣捻転は嘔吐を伴うことも多い、PID は発熱を伴う

② 年齢別

  • 初潮〜20 代前半：卵巣茎捻転／子宮外妊娠／PID／卵巣出血
  • 20 代後半以降：上記＋子宮内膜症
  • 中年：子宮筋腫が加わる
  • 高齢者：子宮留膿腫
  • 卵巣茎捻転は全年齢で起こるが、奇形腫由来は若年に多い
問診のタイミングと聞き方

聞くべき項目（若い女性の下腹部痛で問診なしは診察したことにならない）：

  • 妊娠歴：回数・流産歴（人工含む）・人工授精
  • 月経歴：初潮・閉経、周期・期間・量、LMP（last menstrual period）と PMP（previous menstrual period）— LMP は不正確なことがあるので 1 つ前まで聞く
  • 性行為：最終性交日・性交時痛・出血、パートナー（誰か・人数）

聞き方の原則：

  • 話の初めには聞かない（関係性ができてから）
  • 他人や家族（特に母親）の前で聞かない
  • 同伴者がいる場合は必要な問診を済ませてから席を外させ、診察中・診察後に
  • 妊娠歴・月経歴 → 「女性の腹痛では女性器が原因のこともあるので…」と前置いて性交歴へ
  • 「この医者はしっかり診てくれている」と信頼を作ってから。「お前にそんなこといってもどうせわからないだろう」と思われると不正確な情報しか得られない

③ 妊娠反応 — 必ず取る

  • 妊娠を疑ったら当然、疑わなくても妊孕性のある患者では妊娠反応陰性を確認してから X 線検査
  • 「本人が絶対に妊娠していないと言った」は言い訳にならない。初診で patient-doctor relationship が確立していない段階の答えは信用に値しない
妊娠反応を断られないための言い方

「妊娠可能年齢の女性が腹痛のために X 線検査をするときは、事前に尿検査で妊娠反応を確かめることが医学的に定められています。これはあなた自身の妊娠の可能性とは関係がありません。各個人に可能性があるかどうかを聞くよりも、尿検査で確かめた方が全体として正確であることがわかっているからです。」

それでも断られたらカルテに「協力が得られなかった旨」を記載。

④ 妊婦への対応 — 3 大原則

妊婦の検査・治療 3 大原則（窪田）
  1. 必ず産科医と協議して行う(一般医単独で動かない)
  2. 妊娠・胎児には常に一定の自然リスクがある。薬剤・検査・治療が問題となるのはその自然リスクを上回る場合のみ。完全に危険性のない／100% 危険な薬剤・検査・治療は存在しない
  3. 母体の健康＝胎児の健康。母の疾患を治療せず胎児だけ健康はあり得ない
放射線被曝の閾値（ACR-SPR 2013／Williams 2010）
  • 0.05 Gy（5 rad）以下では、形態異常・発達障害・成長障害・流産（死産）の発生率は自然発生率を上回らない（ACR-SPR 2013）
  • 救急室レベルで問題になるのは腹部 CT。骨盤部を含む CT は2 回撮影で 0.05 Gy を超える（単純＋造影は 2 回撮影と意識）。撮影せざるを得ないなら 1 回のみにする努力
  • 小児癌発生率：0.05 Gy 以下なら自然発生率 1/3,000 が 1/1,500〜2,000 に上がる程度（一般成人 CT の発癌率上昇度と特段に違わない）

胎児被曝量の目安（Williams 2010）：腹部 CT 10 スライス 0.0024〜0.026 Gy／胸部 CT 0.001〜0.0045 Gy／頭部 CT < 0.0005 Gy／骨盤 CT 0.0073〜0.046 Gy／腹部 X 線 0.001〜0.003 Gy／胸部 X 線 2 view < 0.0001 Gy。

窪田：「『腹痛？ とりあえず CT 撮っていて、後で診に行くから！』などという対応をしている人が困るだけだ。普段の診療がしっかりしていれば『妊娠中だから』という特殊性はほとんどない」

妊婦への薬剤投与 — FDA カテゴリーは撤廃

FDA の妊娠中薬剤カテゴリー（A・B・C・D・X）は2015 年に撤廃された。理由は、過剰に単純化された分類が医師・患者にとって正しくない選択（危険度が高いから使用しない）につながるとの危惧。

放射線と同じく、使用するメリット・デメリットと使用しないメリット・デメリットを天秤にかけて決定する。基礎疾患の常用薬は必要だから常用しているのであって、妊娠だからといって中断できるならそもそも最初から不要だったということになる。

「『腹痛があって熱があるからとりあえず抗生剤いっとくか』という思考過程をしている人だけが『妊娠』という言葉にとたんにナーバスになる」
