血便・下血

⚡ 即時アクション
  1. ショックなら上部消化管出血も疑う（大腸出血の 10–15% は上部由来、大量上部出血で鮮血便）
  2. NG チューブ or EGD で上部除外（BUN/Cr 比 > 30 も上部示唆）
  3. 活動性出血 → 造影 CT（CTA）で extravasation 評価、安定 → 大腸内視鏡（緊急性は低い）
  4. 抗血栓薬の確認 + 中止可否（致死的出血なら拮抗、致死的でなければ一時中止）

🚨 最初に決めること

  • 循環が安定しているか（血圧・脈拍・意識）
  • いま出血が続いているか
  • 上からの出血（上部消化管）ではないか
  • 抗凝固薬 / 抗血小板薬を飲んでいるか
すぐ動く
  • ショック（低血圧・頻脈・冷汗・意識変容）
  • 大量の鮮血便が持続
  • Hb が著明に低い
  • 抗凝固薬内服中 + 止まらない出血

📝 問診・身体所見

問診

  • 便：色（鮮血・暗赤色・黒色）、性状、回数、量、排便時刻
  • 吐物：色・性状（褐色、黒、赤、コーヒー残渣様）、回数、量、時刻
  • 随伴症状：意識消失、ふらつき、胸痛、腹痛、黒色便、鼻血
  • 内服薬：NSAIDs、ステロイド、抗血小板薬、抗凝固薬、鉄剤
  • 既往歴：消化性潰瘍、憩室出血、肝疾患、腎疾患、心疾患
  • 最終経口摂取

身体所見

  • バイタル：意識、脈拍、血圧、体温、SpO2、呼吸回数
  • 眼瞼結膜：貧血の有無
  • 腹部：膨満、圧痛、肝硬変の所見（腹水・腹壁静脈怒張・クモ状血管腫）
  • 直腸診：便の色を確認しカルテに記載

💉 初期対応

  1. 末梢ルート確保：20G（重症なら 2本）
  2. モニター装着（SpO2・心電図・自動血圧）
  3. 採血

  1. 細胞外液（ラクテック等）を急速輸液 → 収縮期血圧 100 mmHg を目標
    • ショックが続くなら → 別ルート追加 + 蛋白製剤・輸血を考慮
  2. 輸血の閾値：Hb 7 g/dL（心血管疾患ありなら 8 g/dL）— ACG 2023

  4. 心電図 + 胸部（+ 腹部）X線

上部消化管出血を除外する

黒色便・タール便・コーヒー残渣様嘔吐 があれば上部を考える。 大量の上部消化管出血でも鮮血便として出ることがある。 迷えば BUN/Cr 比 > 30 → 上部を示唆。 上部が疑わしければ → 吐血

🩸 出血が続いている場合

造影 CT（CT angiography）

  • 活動性出血の局在同定に最も有用（感度約 90%）— ACG 2023
  • extravasation（造影剤の血管外漏出） の有無を確認
  • extravasation あり → 消化器内科に相談（内視鏡的止血の可否を判断してもらう）
  • 内視鏡で対応困難と消化器内科が判断 → 消化器内科から IVR に依頼
  • IVR でも止血できない → 外科コンサルト
ACG 2023 の推奨

ガイドライン上は CTA で extravasation 陽性 → 経カテーテル塞栓術（IVR）が第一選択（90分以内の施行で成功率約 98%）。CT で漏出が見えるレベルの出血では血液で視野が確保できず内視鏡的止血が困難なことが多い、というのが根拠。ただし治療方針の判断は消化器内科が行う。

📞 消化器内科コンサルトのタイミング

日勤帯

初期対応後、速やかに 消化器内科外来に相談。

当直帯

以下のいずれかを満たせば → オンコールに相談。 それ以外は 入院の上、翌日の消化器外来責任医師に直接相談（コンサルのみでは対応が遅れるため不可）。

オンコールを呼ぶ基準
  1. 輸液してもショック持続（血圧 80 以下 / ショック指数 ≥ 1）
  2. 造影 CT で extravasation 陽性
  3. 入院後も鮮血の下血が続く
  4. 重篤な疾患を疑う場合や対応に迷う場合

📊 出血が止まっている / 安定している場合

リスク評価（Oakland スコア）

  • Oakland スコア ≤ 8 → 低リスク → 外来フォローでの帰宅を検討 — ACG 2023
  • スコア高値 → 入院

大腸内視鏡のタイミング

  • ACG 2023：安定した下部消化管出血では 緊急大腸内視鏡（24時間以内）は必須ではない
  • 14日以内の大腸内視鏡 でも臨床的アウトカム（再出血率等）は変わらない
  • → 消化器内科と相談してタイミングを決める

📚 よくある出血源

疾患手がかり
憩室出血突然の無痛性大量血便。高齢者に多い。自然止血率 70–80%
虚血性腸炎腹痛 → 血性下痢。左側結腸に多い。高齢者・動脈硬化リスク
痔核排便時の鮮血付着・滴下。若年でも。直腸診で確認
大腸ポリープ / 大腸癌少量の血便が持続。体重減少・貧血
炎症性腸疾患血性下痢 + 腹痛。若年。慢性経過
血管異形成高齢者、大動脈弁狭窄症との関連。少量〜中等量の反復出血

抗凝固薬の対応

  • 生命を脅かす出血 → 拮抗を検討（ワルファリン：ビタミン K + PCC、DOAC：施設プロトコル）
  • 軽症〜中等症 → 一時中止 + 消化器と相談して再開時期を決める
  • ACG 2023：致死的出血では抗凝固薬の拮抗を推奨

📤 disposition

  • ICU：ショック / 大量出血 / IVR・手術が必要
  • 入院：Hb 低い、出血持続、抗凝固薬、高齢、合併症多い
  • 帰宅検討：Oakland ≤ 8、出血止まっている、Hb 保たれている、フォロー確保
