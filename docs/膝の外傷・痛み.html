膝の外傷・痛み

⚡ 即時アクション
  1. 赤旗を外す：膝関節脱臼（自然整復含む）/ 膝窩動脈損傷 / 開放骨折 / 下腿コンパートメント症候群 / 伸展不能
  2. 脱衣して両膝を視診：打撲痕・変形・開放創・大量関節血症の有無を確認（衣類で骨折線・裂創を見逃さない）
  3. X 線は AP + 側面。膝蓋骨を疑えば skyline（軸位）、脛骨高原を疑えば oblique + CT の閾値を下げる
  4. 3 つの骨を順に評価：① 膝蓋骨（骨折・脱臼） → ② 大腿骨遠位端 → ③ 脛骨高原 → ④ 軟部組織（ACL/MCL/半月板）
  5. 5 択分類で行先を決める：即時整形 / 入院整形 / 帰宅手術 / 帰宅保存 / 経過観察（多くはニーブレース＋松葉杖で帰宅可）

🚨 Red flags（命・四肢温存に直結）

1 つでもあれば緊急対応
  • 足背・後脛骨動脈触知不良 / 足色蒼白・冷感 → 膝関節脱臼・膝窩動脈損傷
  • 明らかな膝の変形・大量関節血症 + 高エネルギー外傷 → 膝関節脱臼（自然整復していることがある）
  • 下腿の緊満・受動伸展痛・激痛 → コンパートメント症候群（脛骨骨幹部・高原骨折に合併）
  • 下腿の感覚脱失・足趾運動障害 → 腓骨神経損傷（脛骨高原・腓骨頭近傍骨折）
  • 開放創・骨突出・大量出血 → 開放骨折（大腿骨遠位端骨折の 10% は開放）
  • 伸展不能（active SLR ができない） → 伸筋装置断裂（膝蓋腱・大腿四頭筋腱）または膝蓋骨横骨折

① 膝関節脱臼 — 自然整復していても血管評価

膝窩動脈損傷を見逃さない

膝関節脱臼は 受診時には自然整復している ことが少なくない。「明らかな脱臼」を待たない。高エネルギー外傷 + 大量関節血症 + 強い不安定性を見たら脱臼として扱う。

  • 足背・後脛骨動脈の左右差、CRT、皮膚色・温度を必ず確認
  • ABI < 0.9 / 左右差あり → CT angiography / 血管外科
  • 脈拍触知良好でも遅発性内膜損傷があり得る → 24 時間の血管経過観察を計画
  • 整復後も上記評価を必ず再確認、整形 + 必要に応じ血管外科コンサルト

② コンパートメント症候群 — 脛骨骨幹部・高原骨折で必ず想起

脛骨骨幹部骨折 や 脛骨高原骨折 は下腿コンパートメント症候群の代表的トリガー。「6P」（Pain out of proportion、Paresthesia、Pallor、Paralysis、Pulselessness、Poikilothermia）の Pain と受動伸展痛がもっとも早く現れる。

  • 受傷直後だけでなく 数時間後の遅発性増悪に注意 → 入院後も再評価必須
  • ギプス・シーネ後の急激な疼痛増悪は「外して再評価」が原則
  • 診断は臨床所見が第一。コンパートメント内圧測定は補助
  • 疑った時点で 緊急筋膜切開を整形と相談

③ 開放骨折

  • 裂創 + 骨折を「単なる擦過傷」で済ませない（衣類で隠れていることが多い）
  • 受傷後 1 時間以内の抗菌薬投与が予後を改善：第一世代セフェム ± アミノグリコシド／重度汚染ならペニシリン追加
  • 破傷風トキソイド ± 免疫グロブリンの適応確認
  • 洗浄・被覆して即時整形コンサルト → 詳細は 開放骨折

④ 伸展不能

仰臥位で踵を浮かせる active straight leg raise（active SLR）ができない場合、伸筋装置の連続性が破綻している。

  • 膝蓋骨高位 → 膝蓋腱断裂
  • 膝蓋骨低位 → 大腿四頭筋腱断裂
  • 膝蓋骨水平骨折で離開大 → 横骨折（手術適応）
  • いずれも整形コンサルト、ニーブレース伸展位固定で帰宅 or 入院

🔀 鑑別の枠組み（3 骨 + 軟部組織）

外傷性膝痛は「3 骨」で考える

膝の骨性外傷は ① 膝蓋骨 ② 大腿骨遠位端 ③ 脛骨高原（＋ 脛骨骨幹部） の 3 骨を毎回点検する。X 線撮影前に骨折線のイメージを持っておくと検出率が上がる。

エネルギーで絞る
  • 高エネルギー外傷（交通外傷・転落・スポーツ激突）：大腿骨遠位端、脛骨高原、脛骨骨幹部、膝関節脱臼、開放骨折、コンパートメント
  • 低エネルギー外傷（高齢者の転倒・躓き）：大腿骨遠位端骨折・脛骨高原骨折の occult fracture も起こり得る（高齢者の膝痛では受傷機転にかかわらず鑑別に挙げる）
  • 膝前面の直接打撲：膝蓋骨骨折
  • 膝の捻り・ピボット：膝蓋骨脱臼、ACL 損傷、半月板損傷
  • 膝外側からの直接外力（タックル）：MCL 損傷

骨性 vs 軟部組織の振り分け

所見骨性外傷を示唆軟部組織損傷を示唆
受傷後の歩行受傷直後から不能直後は歩けるが翌日悪化することも

🦴 主要疾患詳細

① 大腿骨遠位端骨折

診断と骨折線イメージ
  • 受傷機転は 2 峰性：若年高エネ（バイク・ダッシュボード）と高齢者の低エネ転倒。80% は高齢者
  • 関節外骨折は転位が強く読影しやすいが、関節内骨折は微細で見逃しやすい
  • 骨折線イメージ：正面像で Y 字（または T 字）、側面像では後方に骨折線が入る
  • 関節内骨折は CT がほぼ必須。読影に迷ったら躊躇せず CT
  • occult fracture は CT でも読み取れず、MRI で初めて診断されることもある（T1 で低信号、脂肪抑制 T2 で高信号）
マネジメント
  • 原則手術。閉鎖骨折でも準緊急手術が多い
  • 診断後は 即時整形 でコンサルト。受傷直後から手術プラン検討を希望されることが多い
  • 転位強ければ直達牽引で待機 → これも即時整形
  • 10% は開放骨折。脱衣して創を必ず確認。CT で骨折周囲の air は開放骨折所見
  • 大腿骨遠位インプラント周囲骨折は他院術後なら転院前提でマネジメント
  • 詳細：大腿骨遠位端骨折

② 脛骨高原骨折

Schatzker 分類と骨折線イメージ
  • 脛骨関節面（高原）の骨折。ほぼ全例 外側（± 内側）。外側の骨折線を徹底的に探す
  • 形状は 陥没（正面像で関節面の沈み込み） or 縦割れ（正面 + 側面で骨折線）
  • 受傷機転 3 パターン：① 歩行者の bumper injury、② 高所転落の軸圧、③ 高齢者の単純転倒（低エネでも起こる）
  • Schatzker IV〜VI は高エネ（全体の約 40%）、II/III は高齢者の微細陥没
  • レントゲンで迷ったら CT 必須。手術プランニングにも CT は不可欠
  • occult fracture は CT 陰性 + MRI 陽性で診断されることもある
  • 合併損傷が高率：手術例の半月板損傷 91% / 靱帯損傷 77%、保存例でも半月板 80% / 靱帯 40%
マネジメント
  • 関節面陥没 5 mm 以上は手術。保存でも荷重厳禁 → 原則入院加療
  • 転位強い・日中受診なら 即時整形、夜間休日で転位少なければ 入院整形
  • コンパートメント症候群 を 24 時間モニター
  • 合併損傷の MRI 評価は整形外科に任せる
  • 詳細：脛骨高原骨折

③ 脛骨骨幹部骨折

  • スキー・交通外傷など高エネが多い。開放骨折頻度が高い（脛骨は皮下に近い）
  • 診断後は 即時整形。閉鎖でも髄内釘の準緊急手術が多い
  • コンパートメント症候群のもっとも代表的なトリガー。受傷後 24〜48 時間は緊満・受動伸展痛・感覚異常を頻回チェック
  • 衣類で裂創が隠れる。必ず脱衣して全周性に観察
  • 詳細：脛骨骨幹部骨折

④ 膝蓋骨骨折

診断のコツ
  • 受傷機転はほぼ全例 膝前面の直接打撲。「転倒して膝の正面をついた」が典型
  • 膝蓋骨は皮下に触れる → 圧痛の有無で予測可能。必ず脱衣して打撲痕を確認
  • 膝前面強打 or 膝蓋骨圧痛があれば skyline view（軸位像）を追加
  • 骨折タイプ：横骨折 50〜80%、縦骨折 12〜17%、残りが粉砕
  • 横骨折は 側面像、縦骨折は skyline view でこそ拾える
  • 3 方向で診断つかなければ CT / MRI も可（実際にはレントゲンで決まる）
マネジメント
  • 膝伸展位では膝蓋骨に荷重がかからない（自分で膝を伸ばし膝蓋骨を左右に動かしてみる） → 転位 < 3 mm なら伸展位固定で 4〜6 週、保存可能
  • 転位 ≧ 3 mm（横骨折に多い） → 手術（テンションバンド）。ただし待機手術が多い
  • 初診時は転位の有無にかかわらず ニーブレース伸展位固定 + 松葉杖で 帰宅保存 or 帰宅手術
  • 伸展不能（active SLR 不能）は離開強い横骨折・伸筋装置断裂を疑い整形コンサルト
  • 二分膝蓋骨との鑑別：圧痛なし・仮骨あり・40〜45% 両側性 → 反対側膝も比較
  • 詳細：膝蓋骨骨折・脱臼

⑤ 膝蓋骨脱臼

  • 10 万人あたり 20 人。若いスポーツマンと中年〜壮年女性に多い
  • 受傷機転：膝伸展位での強い外反。患者は機転を表現しにくく、疼痛で病歴も曖昧になりがち
  • 80% 以上が外側脱臼（大腿四頭筋が膝蓋骨を外側上方に引く）
  • レントゲンで「膝蓋骨の位置が変」と気づければ診断できる。骨折線がないのに激痛なら膝蓋骨の側方シフトを冷静に確認
  • 整復は容易：軽度屈曲位を少しずつ伸展しながら、外側にある膝蓋骨を内側へ押し戻す。麻酔・鎮静は通常不要
  • 合併損傷：膝蓋骨骨折・骨軟骨骨折 5%、半月板・靱帯損傷 12% → 整復後に再評価 + sunrise / MRI 検討
  • 整復後はニーブレース伸展位固定 + 松葉杖で 帰宅保存 or 帰宅手術、翌日整形外来で再評価
  • 15% で再発。スポーツ選手は早期手術志向もあるが治療方針は整形に委ねる

⑥ ACL 損傷

診断
  • 受傷機転：非接触型が 2/3。ジャンプ着地・切り返しで膝が中に入って外反強制
  • ポップ音は 90% で聴取。病歴の有力な手がかり
  • 急性大量関節血症の代表。非骨折性関節血腫の 72% は ACL 損傷
  • 身体所見：
テスト感度特異度
前方引き出し試験62%67%
Lachman テスト84%100%
lever テスト98%100%
包括的診察82%94%
  • Lachman は膝 20° 屈曲、片手で大腿、もう片手で下腿を引く。エンドポイント不明瞭なら陽性。患者の大腿下に自分の膝を入れると力のない術者でもとれる
  • lever テスト：拳をふくらはぎ下に入れ大腿を押す。踵が上がらなければ ACL 損傷を疑う
  • 必ず健側と比較
  • エコーで関節内血腫の確認は感度 88% / 特異度 98% と MRI に近い
マネジメント
  • MRI 読影は非整形外科医には荷が重い（部分断裂・偽陽性あり）。暫定診断 + 整形コンサルトまでが ER の役割
  • ニーブレース固定 + 松葉杖で 帰宅保存 or 帰宅手術。院内整形なら MRI 予約してあげると親切
  • 「最終的に手術となることが多い」旨を説明
  • 詳細：膝靱帯・半月板損傷

⑦ MCL 損傷

  • ACL 損傷と並んで 膝の最多軟部損傷。受傷機転は接触型（横からのタックルなど外反強制）
  • 身体所見：内側圧痛 + 外反ストレステスト。左右差・動揺性で評価
  • 重症度：Ⅰ度（不全断裂・安定）/ Ⅱ度（不全断裂・不安定）/ Ⅲ度（完全断裂）
  • 合併 ACL 損傷の頻度：Ⅰ度 20% / Ⅱ度 52% / Ⅲ度 78% → ACL も必ず評価
  • 関節外靱帯のためエコーで評価可能（健側と比較し、膨化・低エコー・線維 line 不明瞭）
  • 修復能力高く保存治療が多い → 帰宅保存（時に帰宅手術）、ニーブレース + 松葉杖
  • 詳細：膝靱帯・半月板損傷

⑧ 半月板損傷

  • プライマリケアから整形紹介の 膝疾患の 25%
  • 内側半月板損傷が外側の 5 倍（MCL・関節包への付着で可動性が低い）
  • 2 峰性：若年スポーツの捻り、高齢者の変性 + 低エネ外傷
  • キーワード：「違和感」「キャッチング」「ロッキング」
  • 身体所見だけでは確定困難（McMurray 感度 53%）。総合評価が必要
  • MRI は感度・特異度高いが、無痛性半月板損傷の偽陽性に注意（70 歳以上 56%、変形性膝関節症 68〜90% で）
  • 非整形外科医は 暫定診断 + ニーブレース + 松葉杖で帰宅、整形外来で総合評価。緊急手術はない
  • 詳細：膝靱帯・半月板損傷

⑨ 膝関節脱臼

  • 高エネ外傷。受診時は 自然整復 していることが多い
  • 大量関節血症 + 多方向の不安定性 + 高エネ機転 = 脱臼として扱う
  • 膝窩動脈損傷の合併：脈触知良好でも内膜損傷で遅発性閉塞があり得る
  • 整形緊急 + 血管評価（足背・後脛骨動脈触診、ABI、必要に応じて CTA）
  • 整復後も 24 時間血管モニタリング

🩺 身体所見・画像

その場で行う最小診察

  • 視診（脱衣して両膝を裸足で）：打撲痕、変形、皮膚色、開放創、膝蓋骨の位置（外側シフトに注意）
  • 触診：膝蓋骨、腓骨頭、関節裂隙（内側・外側）、大腿骨遠位、脛骨近位、ハムストリング・腓腹筋付着部
  • 関節血症の有無（patellar tap / ballottement）。可能なら穿刺で脂肪滴 + 出血量を評価
  • 可動域：屈曲 / 伸展。active straight leg raise で伸筋装置の連続性を必ず確認
  • 不安定性テスト：Lachman、前方引き出し、内外反ストレス、McMurray
  • 血流・神経：足背・後脛骨動脈、CRT、下腿感覚、足趾運動（腓骨神経 = 背屈、脛骨神経 = 底屈）

主要な徒手検査

テスト方法陽性の意味
Lachman膝 20° 屈曲、大腿固定 + 下腿前方引き出しエンドポイント不明 / 前方シフト → ACL 損傷
lever拳をふくらはぎ下に入れ大腿を押す踵が浮かない → ACL 損傷
前方引き出し膝 90° 屈曲で下腿を引く感度低い。Lachman を優先
外反ストレス膝 30° 屈曲で外反内側痛・動揺性 → MCL 損傷
内反ストレス膝 30° 屈曲で内反外側痛・動揺性 → LCL（稀）
McMurray膝 90° 屈曲で内 / 外旋しながら伸展クリック・疼痛 → 半月板損傷（感度 53%）
active SLR仰臥位で踵を浮かせる不能 → 伸筋装置断裂・膝蓋骨横骨折離開
patellar apprehension膝伸展で膝蓋骨を外側に押す不安・抵抗 → 膝蓋骨脱臼 / 不安定症

Ottawa Knee Rule

5 項目のいずれかで X 線適応
  • ① 55 歳以上
  • ② 腓骨頭の圧痛
  • ③ 膝蓋骨単独の圧痛（他に骨圧痛なし）
  • ④ 膝の 90° 屈曲不能
  • ⑤ 受傷直後と ER で 4 歩歩行不能（跛行可でもよい）

感度 平均 98.5%。すべて陰性なら骨折はかなり少ない。ただし日本では撮らないという運用は一般的でなく、「帰宅前に骨折の可能性を下げる」補助として用いる。occult fracture や 55 歳以上には適用外であることに注意。

ピッツバーグ膝ルール：① 50 歳以上 / 12 歳未満、② 4 歩歩行不能。項目少なく感度は 86〜100% でやや劣る。

X 線：撮影方向

  • 第一選択は AP + 側面
  • 膝前面強打 or 膝蓋骨圧痛 → skyline view（軸位像）追加。縦骨折・骨軟骨骨折・膝蓋骨脱臼整復後の評価に必須
  • 脛骨高原を疑う → oblique view 追加 / 評価不十分なら CT へ
  • 大腿骨遠位端骨折を疑うが大腿骨やや長め → 大腿骨側を含める
  • 注意：転位の強い大腿骨・脛骨骨折で skyline は無理に撮らない（疼痛・転位悪化のリスク）

CT / MRI / 関節穿刺の使い分け

検査適応狙い
CTX 線で読影に迷う / 関節面の評価 / 高エネ / 大腿骨遠位端・脛骨高原・複雑骨折骨折線の確認、術前計画、occult fracture の一部
MRICT 陰性で荷重不能 / 関節血症遷延 / 軟部組織の暫定診断occult fracture（T1 低、脂肪抑制 T2 高）、靱帯・半月板
CT angio / ABI膝関節脱臼 / 自然整復後 / 脈の左右差膝窩動脈損傷
エコー関節血腫の有無、MCL・膝蓋腱・大腿四頭筋腱ベッドサイド評価。ACL は見えない
関節穿刺CT 陰性・MRI 不可で行先を決めたい時脂肪滴 → 骨折、出血量で ACL vs MCL/半月板
CT の閾値を下げる

大腿骨遠位端骨折と脛骨高原骨折は、レントゲンで迷ったら CT。診断確定だけでなく、整形外科の手術プランニングにも有用。「レントゲン + CT がそろったコンサルト」は喜ばれる。

骨性外傷の対応マトリクス（要点）

  • 膝前面強打 or 膝蓋骨圧痛 → skyline 追加 → 膝蓋骨骨折なら 帰宅保存 or 帰宅手術
  • 膝蓋骨に所見なし + 高エネ or 高齢者転倒 → 大腿骨遠位端 + 脛骨高原を骨折線イメージで読影 → 迷えば CT → 陽性なら 即時整形
  • CT 陰性で荷重不能・関節血症 + → MRI で occult fracture を除外（不可なら関節穿刺の脂肪滴で代用）
  • 関節血症の脂肪滴 + → occult fracture として固定 + 整形コンサルト

軟部組織損傷の対応マトリクス（要点）

  • 非接触ジャンプ + ポップ音 + 急性関節血症 → ACL 損傷暫定 → Lachman / lever → ニーブレース + 松葉杖、帰宅手術 or 帰宅保存
  • 膝外側からの外反強制 + 内側圧痛 → MCL 損傷暫定 → 外反ストレステスト → 多くは 帰宅保存
  • 捻り + 「違和感・引っかかり・ロッキング」 → 半月板損傷暫定 → 帰宅手術 or 帰宅保存、整形で総合評価
  • ロッキングで膝が動かない → 整形にすぐ相談（緊急手術ではないが観察短期化）

固定 + 装具 + 鎮痛

  • ニーブレース伸展位固定が膝外傷の基本（膝蓋骨骨折・脱臼、ACL/MCL/半月板の暫定診断、occult fracture 疑い）
  • 松葉杖は 非荷重〜部分荷重で処方。歩き方の練習を ER で行う
  • RICE：Rest、Ice、Compression、Elevation
  • 鎮痛：アセトアミノフェンを基本、必要に応じ短期 NSAIDs（腎機能・高齢で慎重）、強い疼痛にはオピオイド短期
  • 大腿骨遠位端骨折の高エネ・高齢で重度疼痛 → 大腿神経ブロック も検討

📤 disposition

帰宅させてよい条件

  • レントゲン / Ottawa Knee Rule で骨折除外できる、または転位なし膝蓋骨骨折・整復後膝蓋骨脱臼・MCL 単独・暫定半月板損傷
  • 神経血管異常なし、コンパートメント所見なし
  • 松葉杖で部分荷重歩行可（ニーブレース固定下で）
  • 家族・本人が帰宅後の悪化サインを理解し、再診できる体制がある

整形コンサルト / 入院

  • 骨折あり（大腿骨遠位端、脛骨高原、脛骨骨幹部、転位ある膝蓋骨骨折、occult fracture 含む）
  • 膝関節脱臼 / 自然整復後 / 血管損傷疑い
  • 伸筋装置断裂、開放創
  • 大量関節血症 + 多方向不安定性、荷重不能、ロッキング
  • コンパートメント症候群を疑った時点で即時整形

帰宅説明（説明文ひな型）

家族・本人へ
  • ニーブレースは 原則 24 時間装着、入浴時のみ外す（医師指示に従う）
  • 松葉杖で 非荷重 or 患側に体重をかけない歩行（医師指示）
  • RICE：安静・冷却（1 回 15〜20 分、初日は数時間ごと）・圧迫・挙上（心臓より高く）
  • 痛み止めはアセトアミノフェンが基本。NSAIDs は腎機能・胃腸症状・既往と相談
  • 整形外来は数日以内に必ず受診
  • 以下があればすぐ救急再診：
    • 下腿が硬く張ってきて、安静にしても痛みが増す
    • 足趾の感覚が鈍る、動かしにくい
    • 足色が蒼白・紫色になる、冷たい
    • 固定が外れた / 患部の変形が変わった
    • 発熱・創部からの膿（開放創がある場合）

フォロー

  • 非整形外科医の役割は「骨折の有無 ± 軟部組織暫定診断 + 固定 + 整形につなぐ」まで。MRI 読影は整形に任せる
  • 院内整形なら MRI を予約、院外なら紹介状にエピソード・身体所見・画像所見・暫定診断・固定方法・処方を書く
  • 整形外来に行くまでに痛みが急増した、歩行が悪化した場合は再診

⚠️ 見落とし・ミス

膝外傷で頻出するピットフォール
  • 「自然整復した膝関節脱臼」を見逃す — 「明らかな脱臼」は ER に来ない。高エネ + 大量関節血症 + 多方向不安定性は脱臼として血管評価
  • 足背動脈触れるから安心 — 内膜損傷では遅発性閉塞があり、整復後も 24 時間モニター
  • 脛骨高原骨折の関節面陥没を Xp 単独で見逃す — 高齢者の単純転倒でも起こる。荷重不能 + 関節血症が続けば CT
  • 大腿骨遠位端骨折の occult fracture — Y 字（正面）と後方（側面）の骨折線イメージを持って読影。CT 閾値を下げる。CT 陰性でも MRI で出ることがある
  • 膝前面強打の患者で skyline view を撮らない — 縦骨折は正面・側面で見えない。必ず脱衣して打撲痕・圧痛を確認
  • 膝蓋骨脱臼の整復後に骨軟骨骨折を見逃す — 整復後の sunrise / 必要なら MRI を追加
  • 下腿コンパートメント症候群の進行を見逃す — 受傷直後正常でも 24〜48 時間後に増悪。固定後の急増痛は「外して再評価」
  • 開放骨折を裂創扱いで見逃す — 衣類・血液で隠れる。脱衣して全周性に視診。CT で骨折周囲 air は開放
  • active SLR を確認しない — 伸筋装置断裂・膝蓋骨横骨折離開を見逃す
  • Ottawa Knee Rule を慢性痛・55 歳以上に適用 — 急性外傷・55 歳未満で「骨折を下げる」補助。occult fracture も拾えない
  • MCL 単独で安心して ACL を診ない — 重症度が上がるほど ACL 合併率は急増（Ⅲ度で 78%）
  • MRI 所見だけで ACL / 半月板を確定診断 — 部分断裂・偽陽性多い。整形外科の総合判断が必要
  • 二分膝蓋骨を骨折と誤診 — 圧痛なし・仮骨あり・両側性 40〜45%。反対側膝の比較で鑑別
  • 大腿骨インプラント周囲骨折を「骨折」とだけ書いて終わる — 「インプラント入れた病院で治療する」が業界ルール、転院前提でマネジメント
  • 関節穿刺をためらう — 画像で行先が決まらない時、脂肪滴 + 出血量で 3 群に分けられる便利な手技

X 線の読影チェックリスト

骨折線イメージで毎回確認
  • ① 膝蓋骨：横骨折（側面）・縦骨折（skyline）・位置（外側シフト＝脱臼）
  • ② 大腿骨遠位端：正面で Y 字、側面で後方に骨折線
  • ③ 脛骨高原：外側の陥没（正面）・縦割れ（正面 + 側面）。関節面の段差
  • ④ 関節血症 / 脂肪滴：lateral で suprapatellar pouch を確認
  • ⑤ 軟部組織：気腫（開放骨折）・腫脹
  • ⑥ 腓骨頭：高エネで剥離骨折 → 後外側 corner / LCL を疑う
