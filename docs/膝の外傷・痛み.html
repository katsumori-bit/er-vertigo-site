膝の外傷・痛み

⚡ 即時アクション
  1. 血流・神経・コンパートメントを最初に確認（足背動脈、下腿感覚運動、腫脹・緊満）
  2. Ottawa Knee Rules：55歳以上 / 腓骨頭圧痛 / 膝蓋骨単独圧痛 / 90度屈曲不能 / 受傷直後と ER で 4 歩歩行不能 → X 線
  3. X 線は AP + lateral、脱臼疑いなら sunrise（skyline）追加
  4. 膝蓋骨脱臼は速やかに整復、後外側 corner 損傷や膝関節脱臼は必ず MRI / 血管評価

🚨 Red flags（致死・四肢温存に直結）

1 つでもあれば緊急対応

  • 足背動脈触知不良 / 足色蒼白・冷感 → 膝関節脱臼や膝窩動脈損傷を疑う
  • 下腿コンパートメント緊満・激痛・受動伸展痛 → コンパートメント症候群
  • 下腿の感覚脱失・足趾運動障害 → 腓骨神経損傷など
  • 明らかな関節変形・大量関節血症 → 脱臼・骨折を強く疑う
  • 開放創・大量出血
  • 受傷後の高エネルギー外傷（交通外傷、転落）
膝関節脱臼は血管評価必須

膝関節脱臼は自然整復していることがある。明らかな不安定性・大量関節血症を伴う高エネルギー外傷では、整復後でも膝窩動脈損傷を念頭に ABI 測定 / CT angio / 血管外科コンサルトを検討。

🩺 最小診察 + Ottawa Knee Rules

その場で行う最小セット

  • 視診：腫脹、変形、皮膚色、開放創
  • 触診：膝蓋骨、腓骨頭、関節裂隙、大腿骨遠位、脛骨近位
  • 関節血症の有無（Ballottement）
  • 可動域：屈曲 / 伸展、伸展不能（伸筋装置損傷）
  • 不安定性テスト：Lachman / 前後引き出し / 内外反 / McMurray（必要に応じて疼痛緩和後）
  • 血流・神経：足背動脈、後脛骨動脈、下腿感覚運動
Ottawa Knee Rules（外傷時の X 線適応）
  • 55 歳以上
  • 腓骨頭の圧痛
  • 膝蓋骨単独の圧痛（他に骨圧痛なし）
  • 膝の 90 度屈曲不能
  • 受傷直後と ER で 4 歩歩行不能（跛行可でもよい）

1 つでも該当 → X 線。すべて陰性なら骨折の可能性は極めて低い（感度ほぼ 100%）。

📷 画像検査

第一選択

  • 膝関節 AP + lateral
  • 膝蓋骨脱臼疑い / 膝蓋骨骨折疑い → sunrise（skyline / merchant）追加
  • 脛骨高原骨折疑い → oblique view追加 / 評価不十分なら CT

追加画像

  • CT：脛骨高原骨折の関節面評価、複雑骨折、術前計画
  • MRI：靱帯・半月板・後外側 corner 損傷、骨挫傷、occult fracture、膝関節脱臼後の評価
  • CT angio / ABI：膝関節脱臼で血管損傷を疑うとき
X 線陰性でも骨折はある

高齢者の脛骨高原骨折、膝蓋骨水平骨折、occult fracture は単純 X 線で見逃しやすい。荷重不能や強い関節血症が続くなら CT / MRI を追加。

🦴 主要鑑別

病態手がかり進む先
脛骨高原骨折転倒・側方衝撃後の荷重困難、関節血症、内外反不安定性脛骨高原骨折
膝蓋骨骨折・脱臼膝前面打撲、伸展障害、膝蓋骨高位 / 触知異常膝蓋骨骨折・脱臼
靱帯・半月板損傷X線で骨折なし、不安定性、ロッキング膝靱帯・半月板損傷
膝関節脱臼高エネルギー、明らかな変形 / 不安定性、血管損傷リスク整形緊急 + 血管評価
伸筋装置断裂（膝蓋腱・大腿四頭筋腱）伸展不能、膝蓋骨上方 / 下方転位整形コンサルト

関節血症の意味

  • 急性大量関節血症（受傷後数時間以内）：ACL 断裂・脛骨高原骨折・膝蓋骨脱臼が多い
  • 脂肪滴を伴う関節血症：関節内骨折を強く示唆

📤 disposition

帰宅を考えやすい

  • X 線 / Ottawa Knee Rules で骨折なし、神経血管異常なし、荷重可能
  • シーネ / 装具固定、松葉杖、安静・冷却・挙上指示
  • 整形外来 1〜2 週で再診

整形コンサルト / 入院

  • 骨折あり（脛骨高原・膝蓋骨など）
  • 膝関節脱臼 / 整復後 / 血管損傷疑い
  • 伸筋装置断裂、開放創
  • 大量関節血症 + 強い不安定性、荷重不能

疼痛管理

  • アセトアミノフェンを基本に、必要に応じてオピオイド短期使用
  • NSAIDs は腎機能 / 高齢で慎重
  • シーネ固定 + 冷却 + 挙上で疼痛が軽減することが多い

⚠️ ピットフォール

  • 「自然整復した膝関節脱臼」を見逃す — 大量関節血症 + 不安定性 + 高エネルギーは血管評価
  • X 線陰性で安心する — 脛骨高原 occult fracture や後外側 corner 損傷
  • 膝蓋骨脱臼の整復後に骨軟骨骨折を見逃す — 整復後も sunrise / MRI
  • 下腿コンパートメントを忘れる — 脛骨高原骨折・骨幹部骨折は要注意
  • Ottawa Knee Rules を骨折以外に適用する — 急性外傷が前提、慢性痛には使わない
