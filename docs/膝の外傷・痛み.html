身体診察
画像確定後の注意・見逃し回避・ピットフォール

診察段階の Red flag は身体診察パネルに統合済み（⚠ アクション）。ここでは画像で診断が確定してから意識すべき疾患固有の注意・鑑別の入口・ありがちな誤りをまとめる。

① 画像確定後の注意（疾患固有）

  • 緊急膝関節脱臼（自然整復含む）／多方向不安定（≥3 靱帯）＋大量関節血症
    膝窩動脈損傷の合併 18-25%。脈触知不良・ABI < 0.9・左右差で即 CTA・血管外科併診・即時整形。整復後脈触知良好でも遅発性内膜損傷あり、24 時間血管モニタリング。整復は牽引＋押し戻し、整復不能・開放・血管神経合併は即手術室
  • 緊急急性化膿性膝関節炎（発熱＋荷重不能＋熱感）
    関節穿刺で WBC ≥ 50,000/mm³（PMN ≥ 75%）／グラム染色・培養／結晶（合併あり）。即時整形コンサルト＋関節洗浄ドレナージ。抗菌薬は培養前血培 2 セット → 穿刺後にエンピリック（ブドウ球菌カバーが主／院内では MRSA カバー追加）
  • 画像大腿骨遠位端骨折 / 脛骨高原骨折（転位 or 関節面陥没）
    原則入院・手術プランニング。脛骨高原はCT で Schatzker 分類確定（Type IV-VI は高エネ）。閉鎖骨折でも多くが手術。即時整形コンサルト　→ 大腿骨遠位端骨折 詳細　→ 脛骨高原骨折 詳細
  • 画像脛骨骨幹部骨折
    開放骨折頻度高い・コンパートメント症候群リスク高い。皮下脂肪が薄く皮膚直下に骨。下腿緊満・受動伸展痛・鎮痛抵抗性疼痛で組織内圧測定　→ 脛骨骨幹部骨折 詳細
  • 画像膝蓋骨脱臼整復後
    骨軟骨骨折 5%・靱帯/半月板損傷 12% を整復後に再評価。sunrise view（軸位像）± MRI。ニーブレース伸展位＋松葉杖、整形外来へ　→ 膝蓋骨骨折・脱臼 詳細
  • 画像レントゲン・CT 陰性＋大量関節血症で行先が決まらない
    関節穿刺で 3 群に分ける（仲田流）：
    ① 脂肪滴 + → 大腿骨遠位端 / 脛骨高原 occult fracture を疑い入院整形（要 MRI）
    ② 脂肪滴 − / 出血 ≥ 10cc → ACL 損傷を疑い帰宅手術 or 帰宅保存（ニーブレース＋松葉杖）
    ③ 脂肪滴 − / 出血 < 10cc → MCL 損傷 / 半月板損傷を疑い帰宅手術 or 帰宅保存　→ 膝靱帯・半月板損傷 詳細

② 見逃し回避トリガー

  • if高齢者の単純転倒・荷重不能 + 関節血症
    大腿骨遠位端・脛骨高原の occult fractureを疑う。レントゲンで迷えばCT、CT 陰性でもMRIで出ることあり（T1 低・脂肪抑制 T2 高）。占有率高い見逃し
  • if膝前面強打／膝蓋骨圧痛
    skyline view（軸位像）を必ず追加。縦骨折は AP・側面で見えない（縦 12-17%）。横骨折は側面像で評価
  • if非骨折性の急性大量関節血症
    非骨折性なら ACL 損傷を 72%（急性関節血腫全体では ACL 45-52% / 半月板 21-27% / occult fx 13%）で示唆。Lachman / lever テスト＋関節穿刺で暫定診断。整形外来へ
  • ifMCL 損傷を診断した
    ACL 合併を必ず評価（Ⅰ度 20% / Ⅱ度 52% / Ⅲ度 78%）。Lachman も合わせて施行
  • ifキャッチング・ロッキング・違和感の病歴
    半月板損傷を暫定診断＋整形外来。McMurray 感度 53% で身体所見だけでは除外不可。MRI 評価は整形に委ねる
  • ifSegond 骨折（脛骨外側顆の小骨片）
    ACL 損傷の 95% に合併する撕脱骨折。レントゲン正面で見つかれば ACL 損傷として対応
  • iffat pad sign（膝の脂肪パッド変化）
    側面像で suprapatellar bursa が膨満（lipohemarthrosis）→ 関節内骨折を疑う。脂肪と血の境界線 = 脂肪/液体レベル
  • if結晶誘発関節炎（痛風・偽痛風 CPPD）
    関節穿刺で結晶確認（尿酸 = 針状・負複屈折／CPPD = 棒状・弱正複屈折）。化膿性除外が最優先（合併あり）。鎮痛は NSAIDs・コルヒチン・関節内ステロイド（化膿性除外後）。腎機能・既往で薬剤選択

③ ピットフォール

  • 注意Ottawa Knee Rule 全陰性で安心
    急性外傷・55 歳未満で「骨折確率を下げる」補助。慢性痛・55 歳以上・受傷後 7 日以上経過には適用外。occult fracture は別ルートで拾う
  • 注意半月板損傷を「ただの捻挫」で帰宅
    McMurray 感度 53%。違和感・キャッチング・ロッキングの病歴で暫定診断 + 整形外来手配。「軟部組織損傷の疑い」は紹介状必須
  • 注意足背動脈触れるから血管 OK と判断
    膝関節脱臼後の遅発性閉塞あり。整復後も 24 時間モニター。ABI 測定をルーチン化
  • 注意MRI 陽性所見だけで確定診断
    部分断裂・偽陽性多い（70 歳以上 56%、変形性膝関節症 68-90%）。整形外科の総合判断に委ねる。ER で MRI 即時診断は不要
  • 注意転位の強い大腿骨・脛骨骨折で skyline 撮影
    疼痛・転位悪化のリスク。skyline は膝蓋骨骨折を疑った場合のみ。膝外傷でルーチン化しない
  • 注意裂創を「単なる擦過傷」扱い
    衣類・血液で隠れる。必ず脱衣して全周性に視診。CT で骨折周囲 air は開放骨折
  • 注意二分膝蓋骨を膝蓋骨骨折と誤診
    圧痛なし・仮骨あり・両側性 40-45%。反対側膝で比較。新鮮骨折は圧痛＋仮骨なし
  • 注意結晶確認したから化膿性ではないと判断
    結晶誘発関節炎と化膿性は合併する。グラム染色・培養・WBC は必ず提出。発熱・敗血症兆候があれば化膿性を捨てない
  • 注意大腿骨インプラント周囲骨折を一般病院で対応
    「インプラント入れた病院で治療」が原則。転院前提でマネジメント（CT で骨折線・インプラント情報を整える）
X 線適応（Ottawa knee）＋撮影方向

Ottawa Knee Rule（撮影適応）

  • 急性膝外傷で以下のいずれかに該当すれば X 線（系統的レビュー 感度 98.5%）：
    ① 55 歳以上　② 腓骨頭の単独圧痛　③ 膝蓋骨の単独圧痛　④ 90° 屈曲不能　⑤ 受傷直後＋ER で 4 歩歩行不能
  • 適用外：18 歳未満（小児）／受傷後 7 日以上経過／意識障害・酩酊／知覚障害／多発外傷
  • 激痛・変形・開放創：ルール無視で直行 X 線。脱臼／高エネ／開放骨折は明確診断目的

撮影方向（標準・追加）

  • 第一選択：膝関節 AP + 側面。側面像で fat pad / lipohemarthrosis を必ず確認
  • 膝前面強打 / 膝蓋骨圧痛：skyline（軸位）追加（縦骨折・骨軟骨骨折・脱臼整復後の評価）。転位の強い大腿骨・脛骨骨折ではルーチン化しない
  • 脛骨高原疑い：oblique 追加。評価不十分ならCT（Schatzker 分類確定・手術プランニング）
  • 大腿骨遠位端疑い：大腿骨側を含めて撮影。Y 字骨折線をイメージ
  • 膝関節脱臼整復後：必ず整復前後 2 枚を撮影し紹介状添付

追加検査の閾値

  • CT：レントゲン読影困難／関節面評価（脛骨高原・膝蓋骨）／高エネ／手術プランニング／terrible triad 様疑い
  • MRI：CT 陰性で荷重不能・関節血症遷延 / 軟部組織暫定（occult fracture：T1 低・脂肪抑制 T2 高）。ER で必須ではない、靱帯・半月板評価は整形外来
  • ABI / CTA：膝関節脱臼／自然整復後／脈の左右差／ABI < 0.9
  • エコー：関節血腫・膝蓋腱・大腿四頭筋腱・MCL（ACL は見えない）。膝関節穿刺ガイド
  • 関節穿刺：①画像で行先が決まらない急性血腫（脂肪滴＋出血量で 3 群化）、②化膿性 / 結晶誘発関節炎の確定診断

小児・特殊状況

  • 小児：Ottawa knee 18 歳未満は適応外（成長板評価が前提）。健側も撮ると比較しやすい
  • 人工膝置換後：インプラント周囲骨折を意識。元の病院での対応が原則
  • 抗凝固薬内服：低エネ転倒でも閾値下げて撮影
X 線の読み方（系統的アプローチ）

① 撮影セットの確認

  • AP + 側面が揃っているか。撮影体位の妥当性（回旋・関節面が見えるか）
  • 膝前面強打 / 膝蓋骨圧痛例で skyline を追加したか
  • 脛骨高原疑いで oblique を追加したか

② 側面像 — lipohemarthrosis / fat pad を必ず確認

  • lipohemarthrosis（脂肪/液体 fluid level）：水平像で suprapatellar bursa に 2 層（上＝脂肪、下＝血）→ 関節内骨折を強く示唆。X 線が陰性でも occult fracture を疑う
  • suprapatellar bursa の膨満：関節血症・滑液貯留。慢性なら OA・滑液包炎、急性なら関節内骨折 / ACL
  • fat pad sign：膝蓋骨下脂肪体（Hoffa）の偏位・腫脹。微妙な関節内出血の指標
  • 大腿骨膝蓋骨間（patellofemoral joint）：膝蓋骨高位（patella alta）＝膝蓋腱断裂、低位＝大腿四頭筋腱断裂を意識

③ 膝蓋骨評価（AP・側面・skyline）

  • 横骨折（50-80%）：側面像で透亮帯。離開していれば伸筋装置断裂
  • 縦骨折（12-17%）：AP・側面で見えない。skyline で初めて見つかる
  • 二分膝蓋骨：仮骨あり・両側性 40-45%。圧痛なしで反対側比較
  • 膝蓋骨脱臼後：内側膝蓋上稜の骨軟骨骨折（5%）を skyline で評価

④ 大腿骨遠位端評価（AP・側面・CT）

  • Y 字骨折線をイメージ：AO 分類で関節外（A）／部分関節内（B）／完全関節内（C, Y / T 字）を意識
  • occult fracture：高齢者の単純転倒・荷重不能で骨折線が見えない場合あり。CT・MRI で診断
  • インプラント周囲骨折：元の病院での対応原則。CT で骨折線・インプラント情報を整理

⑤ 脛骨高原評価（AP・側面・oblique・CT）

  • Schatzker 分類イメージ：I 外側縦割れ／II 外側縦割れ＋陥没／III 外側純陥没／IV 内側／V 両側／VI 関節外延長
  • 外側陥没：高齢者の低エネ転倒で多い。AP で関節面の段差を確認
  • CT 必須：レントゲンで迷えば躊躇しない。手術プランニングにも必須
  • 半月板損傷 80-91%・靱帯損傷 40-77% 合併。整形外科に必ず情報共有

⑥ 関節脱臼・靱帯損傷の所見

  • 膝関節脱臼：脛骨と大腿骨の重なりが完全に外れる。前方／後方／内側／外側／回旋脱臼の 5 型。自然整復例は変形なく見落とし注意
  • Segond 骨折：脛骨外側顆の小撕脱骨片（AP で確認）→ ACL 損傷の 95% に合併
  • reverse Segond：内側顆の撕脱 → PCL 損傷の指標
  • arcuate sign：腓骨頭の撕脱骨片 → posterolateral corner 損傷の指標

⑦ 整復後の再読影（必須）

  • 膝関節脱臼整復後：必ず前後 2 枚を撮影。関節面の再対合・残存骨折・関節内 air（開放骨折）を確認
  • 膝蓋骨脱臼整復後：sunrise view ± MRI で骨軟骨骨折・MPFL 損傷を評価
  • 同時に脈・神経・5P を再評価し画像 + 所見をセットで紹介状に記載
治療リファレンス：整復・固定・関節穿刺・抗菌薬・処置

膝関節脱臼の初期対応

  • ① 脈触知・ABI・神経（腓骨・脛骨）をまず評価。レントゲンで脱臼確認
  • ② 鎮静下に長軸牽引＋押し戻し整復（鎮静は ketofol 等）。1 ワントライで戻らなければ即整形・手術室
  • ③ 整復後すぐに脈・5P・ABI 再評価。ニーブレース 0° 伸展位で固定
  • ④ 整復後脈良好でも CTA（遅発性内膜損傷あり）。24 時間血管モニタリング
  • ⑤ 開放／血管神経合併／整復不能は即時整形・手術室

膝蓋骨脱臼の整復

  • 80% 外側脱臼。患者を仰臥位・膝伸展位でリラックスさせ、内側へ押し戻すと容易に整復
  • 整復後に sunrise view（軸位像）／関節穿刺で骨軟骨骨折・血腫を評価
  • ニーブレース 0° 伸展位 + 松葉杖で帰宅、整形外来 1-2 週以内
  • 骨軟骨骨折 5%・MPFL 損傷・再脱臼率 15-44% を説明

膝関節穿刺の方法

  • 体位：仰臥位、膝下に丸めたタオルで軽度屈曲（15-20°）
  • 穿刺部位：膝蓋骨外側上縁から 1 cm 上方・1 cm 外側（superolateral approach、最も成功率高い）
  • 消毒・局所麻酔・18-20G 針で suprapatellar bursa に向けて刺入
  • 採取液の評価：色調（黄／血性／白濁）／脂肪滴（油膜・lipid layer）／量（cc）／粘度
    提出：WBC・グラム染色・培養・結晶（偏光顕微鏡）。血培 2 セットも併行
  • 骨折の脂肪滴判定（仲田流）：①脂肪滴 + → occult fracture、②脂肪滴 − / 出血 ≥10cc → ACL、③脂肪滴 − / 出血 <10cc → MCL・半月板
  • 化膿性判定：WBC ≥ 50,000/mm³（PMN ≥ 75%）・グラム染色陽性で化膿性を強く示唆

化膿性膝関節炎の抗菌薬

  • 培養前に血培 2 セット → 関節穿刺で関節液培養 → 抗菌薬。原則は穿刺・培養を取ってから
  • エンピリック：市中＝セファゾリン 2 g IV q8h。MRSA リスク（医療曝露・人工膝・透析・MRSA 既往）＝バンコマイシン追加
  • Gram 陰性菌リスク（高齢・免疫不全・尿路感染既往）＝セフトリアキソン 2 g IV q24hを考慮
  • 淋菌疑い（若年・性活動）＝セフトリアキソン 1 g IV q24h
  • 即時整形コンサルト＋関節洗浄ドレナージ（保存では治癒困難）

痛風・偽痛風の鎮痛

  • 化膿性除外が先（関節穿刺で結晶＋WBC・培養）
  • 第一選択 NSAIDs：ナプロキセン 500 mg × 2 / ロキソプロフェン 60 mg × 3。腎機能・消化管出血歴・心血管疾患で禁忌チェック
  • コルヒチン：発症 24 時間以内なら 1 mg → 1 時間後 0.5 mg → 12 時間後から 0.5 mg × 2/日。腎・肝で減量
  • 関節内ステロイド（化膿性除外後）：トリアムシノロン 20-40 mg / メチルプレドニゾロン 40 mg
  • 経口プレドニゾロン：30-40 mg × 5 日（NSAIDs・コルヒチン禁忌例）

開放骨折・抗菌薬

  • セファゾリン 2 g IV（受傷 1 時間以内）。Gustilo III ＝ + ゲンタマイシン
  • 破傷風トキソイド（10 年以上未接種で）／創部写真撮影／生食ガーゼ被覆して OR まで dressing を外さない
  • 即整形コール

ニーブレース・松葉杖の指示

  • 適応：膝蓋骨骨折／膝蓋骨脱臼整復後／ACL・MCL・半月板損傷／脛骨高原 occult fracture／active SLR 不能例
  • 角度：原則 0°（完全伸展位）。膝蓋骨は伸展位で荷重がかからない
  • 松葉杖：膝への荷重を避ける必要がある場合。歩行訓練を ER で実施
  • 骨折タイプ・角度・荷重可否を紹介状に明記

鎮痛

  • 第一選択：アセトアミノフェン 1,000 mg × 4 / 日（4 g/日まで）。高齢者・小児にも安全
  • NSAIDs（第二選択）：ロキソプロフェン 60 mg × 3 / 日。PPI 併用／eGFR ≥ 60／消化管出血歴・心血管疾患なし／脱水なし
  • 大腿神経ブロック：大腿骨遠位端・膝蓋骨骨折の強い痛みに考慮　→ 大腿神経ブロック 詳細
  • 処方日数：整形外来日まで（通常 5-7 日分）
disposition の目安

30 秒で決める（5 択マネジメント）

※ 膝外傷の即時整形コンサルト適応は① 膝関節脱臼（自然整復含む）／② 開放骨折／③ 血管・神経合併／④ 化膿性膝関節炎／⑤ 整復不能な脱臼／⑥ 大腿骨遠位端・脛骨高原（手術プランニング）。それ以外は「入院手術 / 帰宅手術 / 帰宅保存」。

  • 即時整形膝関節脱臼（自然整復含む）／整復不能脱臼／開放骨折／血管神経症状／化膿性膝関節炎
    夜間・休日でも整形コール。膝関節脱臼は ABI・CTA・血管外科併診。化膿性は穿刺後即洗浄ドレナージ。「断言できないなら呼ぶ」。整形外科医も事前連絡を歓迎
  • 入院手術大腿骨遠位端骨折／脛骨高原骨折（偏位・関節面陥没）／脛骨骨幹部骨折／多発外傷／ADL 困難な高齢者
    入院 + 即日 or 翌朝コンサルト。挙上・厳重 5P 観察、社会調整。脛骨高原は CT 完備で整形に喜ばれる。直達牽引で手術待機もあり
  • 帰宅手術膝蓋骨骨折 / 脱臼整復後／ACL 損傷（脂肪滴 − / 出血 ≥10cc）／半月板損傷（手術適応）
    待機手術（10 日以内に整形外来）。ニーブレース 0° 伸展位 + 松葉杖で帰宅、紹介状に分類・関節穿刺所見・神経血流を明記
  • 帰宅保存Ottawa 陰性で骨折除外可・安定／MCL 損傷／半月板損傷（保存）／結晶誘発関節炎
    ニーブレース + 松葉杖、整形外来 1-2 週以内予約。痛風 / 偽痛風は NSAIDs・コルヒチン処方＋リウマチ / 整形外来へ
  • 再診不要Ottawa 陰性＋身体所見陰性＋荷重可能＋関節血症なし
    疼痛増悪・腫脹増大・しびれ・発熱で再受診を指示。「48-72h で改善なければ再受診」を必ず書面交付（occult fracture・遅発性血管障害の安全弁）

観察時間・フォロー間隔サマリー

  • 膝関節脱臼整復後：24 時間血管モニタリング（遅発性内膜損傷）。ABI 経時記録
  • 整復後 30 分 5P 観察：膝蓋骨脱臼・膝関節脱臼・固定直後すべて
  • 化膿性膝関節炎：洗浄後 24-48 時間で再評価。培養同定で抗菌薬 de-escalation
  • 大腿骨遠位端・脛骨高原：入院・荷重制限。occult fracture も同様の対応
  • 整形外来 1-2 週以内：保存治療例・関節穿刺で軸絞り済み例の標準フォロー
  • 外固定期間：膝蓋骨骨折・脱臼整復後 3-6 週 / 整形判断

他科コンサルト文の標準書式

  • 「他科コンサルト実践マニュアル」順：【目的】【疾患】【既往歴】【内服薬】【ADL】【現病歴】
  • 【目的】：膝関節脱臼整復後の血管評価／脛骨高原骨折の入院加療／化膿性膝関節炎の関節洗浄ドレナージ など、目的を 1 行で
  • 【疾患】：診断名＋分類（Schatzker / AO / Gustilo）＋転位の程度（mm）
  • 【既往歴】【内服薬】【ADL】：抗凝固薬・骨粗鬆症・人工膝・利き足・職業・スポーツを必ず
  • 【現病歴】：受傷機転＋エネルギー＋ER 初療＋関節穿刺所見＋整復前後の神経血流＋画像所見を時系列で

社会軸・特殊状況

  • 利き足・職業：アスリート・大工・農作業は閾値を下げて整形紹介。手術 vs 保存の判断材料
  • 独居高齢者：松葉杖歩行困難・ADL（食事・更衣・排泄）に支障 → 入院・社会調整の検討
  • 抗凝固薬内服：低エネ転倒でも閾値下げて撮影。血腫リスクを紹介状に
  • 人工膝：インプラント周囲骨折は元の病院での対応原則

退室前ピットフォール（退室時固有）

  • 注意膝関節脱臼整復後に脈触知良好で帰宅
    遅発性内膜損傷あり、24 時間モニター必須。ABI 経時測定・CTA を計画
  • 注意化膿性膝関節炎を「結晶誘発」だけで帰宅
    結晶 + WBC 高値は合併あり。培養・グラム染色を必ず提出、発熱・敗血症兆候があれば化膿性扱い
  • 注意occult fracture を「捻挫」で帰宅
    高齢者の荷重不能 + 関節血症はCT / MRI で大腿骨遠位端・脛骨高原を否定。脂肪滴 + は入院
  • 注意ニーブレース・松葉杖の使い方を口頭だけで説明
    独居高齢者では装着・歩行困難。本人 + 家族で実技確認、書面でも交付

主要な根拠・参考文献

  • 骨折ハンター（中外医学社）— 第 15 章「膝関節痛ハンター①（大腿骨遠位端骨折・脛骨高原骨折・膝蓋骨骨折）」
    　ER の "hidden tip"：①骨折線をイメージしてレントゲンを読む、②膝前面強打／膝蓋骨圧痛で skyline view を追加、③高エネ・高齢者は CT を躊躇しない、④大腿骨遠位端・脛骨高原は即時整形・入院・手術主体
  • 骨折ハンター— 第 16 章「膝関節痛ハンター②（ACL・MCL・半月板・関節穿刺）」
    　関節穿刺で 3 群化：①脂肪滴 + → occult fracture（入院整形）、②脂肪滴 − / 出血 ≥10cc → ACL（帰宅手術 / 帰宅保存）、③脂肪滴 − / 出血 <10cc → MCL・半月板（帰宅手術 / 帰宅保存）。MCL ＋ ACL 合併 Ⅰ20-Ⅲ78%。McMurray 感度 53%（除外不可）
  • Ottawa Knee Rule：Stiell IG, et al. JAMA. 1996;275:611-5. 系統的レビュー（Bachmann LM, Ann Intern Med. 2004）で平均感度 98.5%
  • Pittsburgh Knee Rule：Bauer SJ, et al. Acad Emerg Med. 1995. 感度 86-100%
  • Schatzker 分類（脛骨高原骨折）／AO 分類（大腿骨遠位端骨折）／Gustilo 分類（開放骨折）
  • Schenck 分類（膝関節脱臼）：KD-I〜V。膝窩動脈損傷 18-25%（特に KD-III/IV）
  • 急性化膿性膝関節炎：関節液 WBC ≥ 50,000/mm³ + PMN ≥ 75%／グラム染色・培養／頻度 ブドウ球菌＞連鎖球菌＞淋菌（若年）＞GNR（高齢）
