⚠️ トリガー連動の注意（条件で意識）

特定の状況に出会ったら必ず思い出すべき注意点。フローの判断と並行して意識する。

緊急・四肢温存トリガー

  • if高エネ + 大量関節血症 + 多方向不安定性
    膝関節脱臼（自然整復含む）として血管評価。ABI < 0.9 / 左右差 → CTA・血管外科
  • if膝関節脱臼整復後・脈触知良好
    遅発性内膜損傷あり。24 時間血管モニタリングを計画
  • if下腿緊満・受動伸展痛・激痛
    コンパートメント症候群を疑う。固定後の急増痛は外して再評価。緊急筋膜切開
  • if裂創 + 骨折／骨突出
    開放骨折。1 時間以内抗菌薬・破傷風・即時整形
  • ifactive SLR 不能
    伸筋装置断裂（膝蓋骨高位＝膝蓋腱／低位＝大腿四頭筋腱）または膝蓋骨横骨折離開。ニーブレース伸展位 + 整形コンサルト
  • if下腿感覚脱失・足趾運動障害
    腓骨神経損傷（脛骨高原・腓骨頭近傍骨折）を疑う

見逃し回避トリガー

  • if高齢者の単純転倒・荷重不能 + 関節血症
    脛骨高原の関節面陥没・大腿骨遠位端 occultを疑う。Xp で迷えば CT。CT 陰性でも MRI で出ることあり
  • if膝前面強打／膝蓋骨圧痛
    skyline view（軸位像）を必ず追加。縦骨折は AP・側面で見えない
  • if膝蓋骨脱臼を整復した
    骨軟骨骨折 5%・半月板/靱帯損傷 12% を整復後に再評価。sunrise view ± MRI
  • if非骨折性の急性大量関節血症
    ACL 損傷を 72% で示唆。Lachman / lever テストで暫定診断
  • ifMCL 損傷を診断した
    ACL 合併を必ず評価（Ⅰ度 20% / Ⅱ度 52% / Ⅲ度 78%）
  • if膝蓋骨に骨折線様の所見
    二分膝蓋骨と鑑別：圧痛なし・仮骨あり・両側性 40〜45%。反対側膝で比較
  • if大腿骨インプラント周囲骨折
    「インプラント入れた病院で治療」が原則。転院前提でマネジメント
  • if画像で行先が決まらない
    関節穿刺。脂肪滴 + → 関節内骨折／≧10cc → ACL／<10cc → MCL・半月板

ピットフォール

  • ⚠Ottawa Knee Rule 全陰性で安心
    急性外傷・55 歳未満で「骨折を下げる」補助。occult fracture や慢性痛・55 歳以上には適用外
  • ⚠半月板損傷を「ただの捻挫」
    McMurray 感度 53%。違和感・キャッチング・ロッキングの病歴で暫定診断 + 整形外来を必ず手配
  • ⚠足背動脈触れるから血管 OK
    膝関節脱臼後の遅発性閉塞あり。整復後も 24 時間モニター
  • ⚠MRI 陽性所見だけで確定診断
    部分断裂・偽陽性多い（70歳以上 56%、変形性膝関節症 68〜90%）。整形外科の総合判断に委ねる
  • ⚠転位の強い大腿骨・脛骨骨折で skyline 撮影
    疼痛・転位悪化のリスク。無理に撮らない
  • ⚠裂創を「単なる擦過傷」扱い
    衣類・血液で隠れる。必ず脱衣して全周性に視診。CT で骨折周囲 air は開放
📷 撮影方向・追加検査の備忘
  • 第一選択：膝関節 AP + 側面
  • 膝前面強打 / 膝蓋骨圧痛：skyline（軸位）追加（縦骨折・骨軟骨骨折・脱臼整復後）
  • 脛骨高原疑い：oblique 追加 → 評価不十分なら CT
  • 大腿骨遠位端疑い：大腿骨側を含めて撮影
  • CT：レントゲン読影困難／関節面評価／高エネ／手術プランニング
  • MRI：CT 陰性で荷重不能・関節血症遷延／軟部組織暫定（occult fracture：T1 低・脂肪抑制 T2 高）
  • ABI / CTA：膝関節脱臼／自然整復後／脈の左右差
  • エコー：関節血腫・MCL・膝蓋腱・大腿四頭筋腱（ACL は見えない）
  • 関節穿刺：画像で行先が決まらない時。脂肪滴 + 出血量で 3 群に分ける
  • Ottawa Knee Rule：①55 歳以上 ②腓骨頭圧痛 ③膝蓋骨単独圧痛 ④90° 屈曲不能 ⑤4 歩歩行不能 — 1 つでも陽性で X 線
