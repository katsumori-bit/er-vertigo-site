小児骨端線骨折

⚡ 即時アクション
  1. Salter-Harris 分類（Ⅰ-Ⅴ） ─ Ⅰ：骨端線のみ／Ⅱ：+骨幹端（最多 75%）／Ⅲ：関節内／Ⅳ：貫通／Ⅴ：圧挫
  2. X 線（健側との比較）2 方向撮影、Ⅰ・Ⅴ は X 線で見えにくく 圧痛 + 年齢で診断
  3. Ⅲ-Ⅴ は手術検討（関節面・成長板損傷の解剖学的整復が必要）
  4. 成長障害・変形のリスク ─ 全例整形外科紹介、6-12 ヶ月の長期 follow-up が必要

📞 この疾患が該当する主訴
  • 手関節・手の外傷 — 小児の手関節橈骨遠位骨端線損傷で多い
  • 膝の外傷・痛み — 大腿骨遠位・脛骨近位の骨端線損傷
  • 足関節・足部の外傷 — 脛骨遠位骨端線（Tillaux 骨折等）で多い
  • 転倒 — 運動中・転倒で発症

🦴 解剖

  • 小児の骨端と骨幹の間に 骨端線（成長軟骨板） がある
  • 靱帯 > 骨端線の強度 → 外力は靱帯を避けて骨端線に集中
  • 骨端線の損傷 → 成長障害（短縮・変形）

📊 Salter-Harris 分類

Type骨折線頻度成長障害リスク
I骨端線のみを通る（骨折なし、水平剥離）6%低
II骨端線 + 骨幹端75%（最多）低
III骨端線 + 骨端（関節内）8%中
IV骨幹端 + 骨端線 + 骨端10%高
V骨端線の圧挫1%最高（レントゲンで見えにくい）
覚え方（SALTR）
  • Slipped（Type I）：骨端線で滑る
  • Above（Type II）：骨端線の上（骨幹端）
  • Lower（Type III）：骨端線の下（骨端）
  • Through（Type IV）：骨幹端 + 骨端線 + 骨端を貫く
  • Rammed（Type V）：圧挫（ram = 突き刺す）

⚡ 受傷機転

  • 小児の運動中・転倒
  • 成長期の運動負荷（野球肘、ジャンパー膝など）

🩺 身体所見

  • 骨端線の直上に 限局した圧痛
  • 腫脹・変形は骨折の程度による
  • Type I はレントゲンで見えない → 圧痛 + 年齢で診断

🔬 画像

レントゲン

  • 患部 2 方向 + 健側の比較撮影
  • Type I：骨折線見えない、骨端の transposition or 健側との比較で判断
  • Type V：圧挫像、骨端線が狭小化

MRI / CT

  • 確定困難例、手術計画で使用

🏥 マネジメント

全例で整形外科紹介

  • 成長障害評価のため 長期 follow 必須（受傷後 6–12 ヶ月は定期観察）

治療

Type治療
Iシーネ固定 3–6 週、転位あれば整復
IIシーネ固定、転位あれば整復（関節内は慎重）
III手術が多い（関節内のため解剖学的整復必要）
IV手術（骨幹端 + 骨端線 + 骨端を解剖学的に整復）
V早期診断困難、手術適応は慎重

👨‍⚕️ 非整形外科医の役割

  1. 小児の外傷は骨端線骨折を常に疑う
  2. 両側撮影（健側との比較）
  3. 画像陰性でも 圧痛あれば Type I として固定
  4. 全例 整形外科外来紹介、成長障害の長期評価を

⚠️ 合併症

  • 成長障害（短縮・変形）
  • 関節不整合（Type III–IV）
  • 早期骨端線閉鎖（Type V）
  • 外傷性関節症（関節内骨折）

💬 親への説明

  • 「骨端線は成長に大切な部分、骨折すると後で成長に影響することがあります」
  • 「定期的なレントゲンでの確認が必要です」
  • 長期 follow の重要性を強調
