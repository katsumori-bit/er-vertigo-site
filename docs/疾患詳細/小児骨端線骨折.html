小児骨端線骨折

同時並行で評価する 3 軸
典型像
  • 骨端線損傷は小児骨折の 18%。靭帯 > 骨端線の強度
  • 14–15 歳頃に骨端線が閉鎖して成人型に
  • 骨端線損傷で四肢長不一致のリスク
  • 所見：骨端線直上の限局圧痛、若木骨折は片側変形のみ
  • FOOSH・足関節捻り・MP 関節の過度撓屈尺屈＋回旋など
血管

小児上腕骨顆上骨折は上腕動脈損傷に注意。Volkmann 拘縮の前駆病態となる。

  • 橈骨動脈触知（左右比較）、毛細血管再充満時間
  • 触れない／弱い → 上級医・整形へ即時相談
  • 非整形外科医がきついギプスを巻かない（コンパートメント誘発）
  • 手関節はsugar tong splintを選択
神経

上腕骨顆上骨折は正中神経・前骨間神経麻痺、外顆骨折は尺骨神経麻痺の可能性。

  • 上肢：OK サイン（前骨間神経）、母指外転（正中）、小指外転（尺骨）
  • 下肢：足関節背屈・第一趾背屈、足背外側感覚（腓骨神経）
  • 記録を残す（後の責任分界点）
  • 麻痺があれば固定肢位の見直し ＋ 整形へ即時相談
画像の見方（Salter-Harris は骨折線イメージ）

解剖

  • 小児の長管骨は端の骨端線（成長軟骨板）で伸びる
  • 靭帯 > 骨端線の強度のため、外力は靭帯を避けて骨端線に集中
  • 14–15 歳頃に骨端線が閉鎖して成人と同じ外傷になる
  • 骨端線が損傷して骨が伸びなくなると、四肢長不一致（変形）を起こすことがある

Salter-Harris は「分類」でなく「骨折線イメージ」

非整形外科医にとって Salter-Harris は分類ではなく骨折線イメージとして使う。頻度の高い骨折は Type が決まっているため、事前にイメージを持っておくことで読影できる。逆に分類を知らないと微細な骨折線を見落とす。

Type骨折線頻度成長障害リスク
I骨端線のみ（水平剥離）6%低（保存）
II骨端線 ＋ 骨幹端75%（最多）低（多くは保存）
III骨端線 ＋ 骨端（関節内）8%中（手術が多い）
IV骨幹端 ＋ 骨端線 ＋ 骨端を貫通10%高（解剖学的整復 → 手術）
V骨端線の圧挫1%最高（X 線で見えにくい）

頻度の高い 4 つはイメージを丸暗記

骨折典型 Type
上腕骨外顆骨折Type IV（小児肘・FOOSH）
橈骨遠位端骨折Type I・II（II で転位 15° 以内なら保存）
基節骨基部骨折（小指好発）Type II（25° 以上 or 1.5 mm 以上の離開で手術検討）
脛骨遠位端骨折（Tillaux 骨折）Type III。小児関節内骨折の 5–10%。骨端線が部分的に閉じる時期に起こる

X 線（健側比較が原則）

  • 患部 2 方向（正面・側面）＋ 健側を必ず撮影して比較
  • 左右差が骨折線の発見に直結（特に Type I・II・V）
  • 肘の正常基準線：anterior humeral line（上腕骨顆上骨折）、radiocapitellar line（上腕骨外顆骨折・Monteggia）— 延長線が小頭の中心を通らなければ異常
  • 手関節：橈骨遠位端の背側にのみ骨折線が入る若木骨折パターンを意識

身体所見の特徴

  • 骨端線の直上に限局した圧痛
  • 小児は若木骨折（greenstick）で片側だけ骨折線が入って反対側は屈曲変形のみのこともある
  • Type I は骨折線が見えないことが多く、圧痛 ＋ 年齢 ＋ 健側比較で診断
  • Type V は圧挫により骨端線がわずかに狭小化するのみ。後日 follow で診断確定することもある
  • 小児外傷で「画像陰性」だからといって帰すのは危険。圧痛があれば Type I として固定 ＋ 整形外科紹介
治療方針（転位 ＋ 関節内で手術判断）

転位の有無で disposition を決める

  • 転位が小さい → 帰宅保存（外固定 ＋ 三角巾）→ 後日整形外科外来
  • 転位 2 mm 以上 or 関節内 → 即時整形（手術検討）
  • 迷ったらオーバートリアージ。子どもの怪我は日中受診が多く相談しやすい

Type 別の治療

Type治療
Iシーネ固定 3–6 週、転位あれば整復
IIシーネ固定（橈骨遠位端は転位 15° 以内なら整復不要、15° 以上で整復）。基節骨基部骨折は 25° 以上 or 1.5 mm 以上の離開で手術検討
III関節内のため手術が多い（Tillaux 骨折は日中：即時整形／夜間：帰宅手術）
IV手術（上腕骨外顆骨折は典型例。転位 2 mm 以上で手術適応）
V早期診断困難、手術適応は専門医判断

長期 follow（必須）

  • 成長障害評価のため全例整形外科紹介、6–12 か月の定期観察
  • 合併症：成長障害（短縮・変形）／関節不整合（III–IV）／早期骨端線閉鎖（V）／外傷性関節症

帰宅時の鎮痛薬

  • イブプロフェン 10 mg/kg/day を 3 分割投与（最大 600 mg/day）が小児骨折の第一選択（アセトアミノフェンと差はないが副作用が少なく満足度高い）

非整形外科医の役割（やりがちなミスを避ける）

  1. 小児外傷は年齢別の頻度から鑑別を挙げる（小学校高学年〜中学 1 年の足関節外傷では靭帯損傷より骨端線骨折を疑う）
  2. 骨折線イメージを持って X 線を読む
  3. 健側比較撮影を必ず行う
  4. 画像陰性でも圧痛あれば Type I として固定 ＋ 整形外科紹介
  5. 小児にきついギプスは禁（Volkmann 拘縮）。手関節は sugar tong splint を選択
  6. 全例で長期 follow（6–12 か月）を依頼

親への説明

  • 「骨端線は成長に大切な部分、骨折すると後で成長に影響することがあります」
  • 「定期的なレントゲンでの確認が必要です」
