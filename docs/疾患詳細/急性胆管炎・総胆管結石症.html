急性胆管炎・総胆管結石症（Acute Cholangitis & Choledocholithiasis）

#ER #救急 #腹痛 #胆管炎 #敗血症 #ERCP #TG18

急性胆管炎は急性胆嚢炎としばしば混同されるが、まったく異なる病態で治療方針も異なる。胆嚢炎の臨床像は腹痛、胆管炎の臨床像は敗血症。厳格に区別する。

病態

① 急性胆管炎

  • 閉塞の原因で最多は総胆管結石（乳頭部嵌頓）。他に悪性腫瘍（胆管癌・膵頭部癌）／胆道再建術後／十二指腸憩室炎（Lemmel 症候群、存在に異論あり）
  • 嵌頓直後は感染症ではない（非感染性）。閉塞部位が腸管に近いため容易に感染症へ移行し、感染すれば胆管 → 肝内 → 血中へ細菌が移行 → すぐ敗血症
  • 敗血症の絶対数は腎盂腎炎より少ないが、敗血症化する確率は腎盂腎炎より高い
  • 本邦ではAOSC（acute obstructive suppurative cholangitis）という用語があるが、欧文では ASC として記載される程度で和製の感あり（胆汁が膿そのものになる重篤感染を意味する）

② 総胆管結石症

  • 結石があるだけでは無症状
  • 小さな結石：胆嚢由来（胆嚢管を通過するサイズ）＝若年者に多い。乳頭部に短時間嵌頓 → 心窩部痛発作のみで感染に至らず
  • 大きめの結石：総胆管内で生成／増成＝高齢者に多い。嵌頓が長引き胆管炎へ移行しやすい
  • 同サイズの結石が胆嚢内に複数あれば発作を繰り返す

初期治療は内視鏡的に行われるのが一般的で、初診で外科コンサルトする疾患ではなくなっている。

病歴

① 急性胆管炎

  • 発症は半日〜2-3 日。主症状は腹痛／発熱／黄疸 ＝ Charcot's triad（シャルコー 3 徴）。3 つ揃うのは 50〜75%
  • 腹痛（出現率 70%）：胆嚢炎より程度はマイルド。心窩部違和感や胸部不快感程度のこともあり、「発熱」だけが主訴のこともある → こちらから腹痛を聞き出す。胆石膵炎合併を考えて左上腹部・背部痛も確認
  • 発熱（出現率 90%）：高熱が多い。悪寒戦慄（shaking chill）を伴うのが特徴 — 20〜30 分ガタガタ震え → 高熱、しばらくおさまる → 再び悪寒戦慄、を反復
  • 黄疸（出現率 70%）：結石性は軽度（4〜5 mg/dL 程度）でアジア人の肌では分かりにくい。本人の自覚は皮膚色より「尿の濃染」のことが多い → 「発症後の尿の色が普段より濃かったか」を聞く
Reynolds の 5 徴（重症化サイン）
Charcot 3 徴 ＋ ショック（30%） ＋ 意識障害（10〜20%）。揃うと mortality > 50%。救急室では「敗血症」としてまず扱われ、25% は胆管炎とすぐに診断できない。

② 総胆管結石症

  • 胆石発作と同様だが痛みはマイルドのことが多く、心窩部違和感・不快感程度のこともある
  • 胆石発作と違って嵌頓した結石は症状軽快時には乳頭部を通過している → 1 つの石で痛みが起きるのは 1 回だけ

身体所見

① 急性胆管炎

敗血症としての所見と胆管炎としての所見に分けて取る。

  • 敗血症の所見：高熱（90%、高齢者で無熱もあり）／意識レベル低下（10〜20%）／低血圧（30%）。発症早期から出現することもある
  • 胆管炎の所見：右上腹部圧痛（65%）／黄疸（80%）
胆嚢炎と胆管炎の圧痛の取り方の違い
肝臓は大部分が右肋骨に覆われている。総胆管自体は膵・十二指腸・肝に覆われ直接の圧痛は強くないが、末梢の肝内胆管は肝表面に広がるため、右下位肋骨の叩打で響く痛みとして認識できる。
胆嚢炎は痛みが胆嚢にフォーカスするのに対し、胆管炎は肝表面全体が痛いのが特徴。
  • 黄疸は眼球結膜で 2〜3 mg/dL、皮膚で 5〜10 mg/dL 程度で認識できる。急性期は高度上昇していないことが多く、見落として血液検査でびっくりする経験はみんなある。軽度黄疸（〜3 mg/dL）は敗血症だけでも生じるので判断に迷うことがある

② 総胆管結石症

有症状時は急性胆管炎の「胆管炎による所見」のみと考えてよい。黄疸はないことが多い（嵌頓時間が短いため）。

初期診断と鑑別

3 徴のうち 1 つが欠けたとき。

① 発熱がない

  • 「黄疸＋腹痛」 → 総胆管結石症（有症状）／胆道閉塞の器質的疾患（膵頭部癌・胆管癌）／急性肝炎（アルコール／薬剤性、ウイルス性は通常無痛）
  • 重症で低体温になっている場合や、発熱の間欠期に来院した場合に「発熱なし」と評価することがある

② 黄疸がない

  • 「発熱＋腹痛」 → 右季肋部圧痛なら胆嚢炎・結腸憩室炎・肝膿瘍・上行虫垂炎・横隔膜下膿瘍・Fitz-Hugh-Curtis 症候群（女性で PID の肝周囲波及）
  • 心窩部痛なら急性膵炎・急性虫垂炎も加わる（虫垂炎は自発痛が心窩部でも圧痛は右下腹部）

③ 腹痛がない

  • 「発熱＋黄疸」 → 敗血症全般で見られ鑑別が広い

検査

血液

  • 胆嚢炎と違って生化学が意義を持つ
  • ビリルビン上昇 90%、ALP 上昇 80%。AST/ALT も軽度上昇のことが多い
  • ビリルビンは直接型優位（D-Bil > T-Bil）。間接型優位なら黄疸の原因を再考
  • 白血球上昇 80%、ただし敗血症では逆に減少することも
  • 膵酵素は必ずチェック：上昇していれば胆石性膵炎の合併を示唆。膵酵素正常でも膵炎のことあり、上昇＝膵炎ではない
  • 血液培養 2 セットを採取（陽性率 50%）

画像

超音波

  • 観察項目は総胆管と肝内胆管のサイズ。総胆管 ≥ 7 mm、肝内胆管 ≥ 4 mm で拡張。二次分枝より末梢が見えれば拡張
  • 感度 50〜90% で閉塞期間に依存。長期狭窄（悪性腫瘍）→ 高度拡張で容易、急性結石性 → 拡張軽度で慣れないと指摘困難
  • 偽陽性の落とし穴：胆嚢摘出後 → 代償で総胆管 10 mm 程度は異常でない／高齢者の乳頭括約筋不全 → 慢性的に拡張（近位主体・末梢肝内胆管は拡張せず）
  • 嵌頓結石の画出は乳頭部付近が見にくく必ずしも容易でない（胃・横行結腸ガス）

腹部 CT — 結石を見るなら単純、胆管を見るなら造影

  • 結石の同定は単純 CT が優れる（コレステロール結石は見えない、ただし総胆管結石は色素結石が多いため同定可なことが多い）
  • 肝内胆管は造影 CT が見やすい → 結局「単純＋造影」が必要
  • 軽度拡張は超音波同様、門脈と並行する肝内胆管二次分枝以遠が描出されれば拡張

確定診断（Tokyo Guideline 2018）

TG18 — 急性胆管炎の診断基準
  • A. 全身炎症：A-1 発熱／戦慄、A-2 炎症反応（検査値）
  • B. 胆汁うっ滞：B-1 黄疸、B-2 肝機能異常（検査値）
  • C. 画像：C-1 胆道拡張、C-2 etiology（狭窄・結石・ステント等）
疑診：A 1 項目 + B または C 1 項目
確診：A 1 項目 + B 1 項目 + C 1 項目

治療

① 治療方針

  • 初期抗菌薬の奏効率は約 80%。緊急手術になることはほとんどなく、保存治療＋必要に応じ内視鏡的ドレナージが基本
  • 抗菌薬を開始しただけで安心しない。最初の 24 時間以内に奏効するかを判断
緊急ドレナージを考慮すべき所見
  • 強い腹痛が持続する
  • 適切な補液にもかかわらず低血圧
  • 39℃以上の高熱
  • 意識障害（GCS ≤ 14）

② ドレナージのオプション

  • 第一選択：内視鏡的ドレナージ（ENBD／ERBD）。non-operative method が好ましい
  • 胃手術後（Roux-Y 再建）ではBE-ERCP（ダブルバルーン内視鏡）／EUS-BD（内視鏡超音波下ドレナージ）が選択肢だが、施設・術者の習熟度に依存（EUS-BD はガイドライン上「確立されていない」）
  • 第二選択：PTCD（経皮経肝胆道ドレナージ）。急性発症で胆管拡張が軽度な場合は穿刺困難になりやすい
  • 3 管合流部開存＋胆嚢拡張があれば PTGBD で減圧を図ることも可能
  • 経皮的いずれも不能なら緊急手術（重症例には太い T-tube ドレナージ）

③ 抗菌薬

  • 血液培養 2 セット採取後すぐ開始
  • monotherapy なら PIPC/TAZ 4.5 g IV q6h（または ABPC/SBT 3 g IV q6h、重症感染では増量 12 g/日）
  • かつての ABPC + GM はほぼ使わない

④ 保存治療奏効後

  • 胆道評価で総胆管結石が残っていなければ腹腔鏡下胆嚢摘出術を施行
  • 残っていれば（i）ERCP±EST で総胆管結石除去 → 腹腔鏡下胆嚢摘出術、または（ii）腹腔鏡下総胆管切石術。施設・症例により選択
  • 退院せず入院中に完結（早期再発、入院期間・コスト面）
  • 「画像で胆嚢内結石なし」でも、有症状胆石症（症状が出た）の診断で胆嚢摘出が適応 — 「画像に石があるかどうか」で手術適応を決めるのではない
胆道評価のモダリティ
US（体表）／DIC-CT／MRCP／ERCP／EUS／IOC（術中胆道造影）。MRCP と EUS の感度・特異度は同等。
高確率で結石があると考えれば ERCP（治療兼ねて）、結石はすでに通過したと考えるなら MRCP・EUS・DIC-CT から低侵襲に選ぶ。IOC で術中確認できるので術前確認は必須でない。

特殊な胆管炎

① Lemmel 症候群

十二指腸乳頭部付近の憩室による胆管圧迫が原因とされる。因果関係は証明されておらず、存在自体に異論あり。「憩室＋胆管炎」だけでこの診断に直結させない。

② Mirizzi 症候群

  • Type I：胆嚢頸部・胆嚢管の嵌頓結石の影響が CBD に及び胆管炎を併発（一般的な典型例）
  • Type II：胆嚢頸部・胆嚢管と総胆管に瘻孔
  • Type III：嵌頓結石で肝管狭窄
  • Type IV：無石性胆嚢炎で肝管狭窄
  • 治療は胆嚢摘出術が第一選択

③ 胆道再建後・乳頭機能不全に伴う胆管炎

膵頭十二指腸切除後・膵胆管合流異常に対する胆道再建後、小児なら総胆管嚢腫術後・胆道閉鎖症の Kasai／Suruga 後など。逆行性胆管炎で EST 後や乳頭機能不全でも同様。重症化せず敗血症には至らないことが多い。

④ 胆道狭窄に対するステント留置中の胆管炎

ドレナージのために留置したチューブが閉塞して胆管炎が発症。ERBD（細径ステント・体外導出なし）は閉塞を早期認知できず重篤な敗血症に進展する危険。
