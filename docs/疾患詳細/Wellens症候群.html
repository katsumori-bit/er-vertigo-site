Wellens症候群

⚡ これを見たら：LAD 近位狭窄の予兆
  1. LAD 近位狭窄の予兆と認識 → 介入しないと短期間で前壁梗塞に移行
  2. V2-3 deep T 反転（Type B, 75%）or biphasic T（Type A, 25%）がポイント
  3. 胸痛発作後の無症状期に出現 → 「胸痛が消えたから安全」は逆
  4. 緊急カテ（入院 + 同日循環器コンサルト）、ストレステスト禁忌（広範前壁梗塞リスク）

📞 この疾患が該当する主訴
  • 胸痛 — 胸痛発作後の無症状期に出現するのが典型
  • 入院時心電図 — V2-3 の T 波異常を見落とさない
  • 失神（意識消失） — 前壁梗塞移行時の致死性不整脈

  • Wellens症候群

📐 定義（de Zwaan / Wellens 1982）

  • V2–V3 を中心とする前胸部 のT波異常
    • Type A：二相性T（約25%）
    • Type B：深い対称性T陰性（約75%）
  • QRS は正常 のことが多い
  • Q 波は目立たない、R 波進行は保たれる
  • トロポニンは陰性〜軽微
  • 無症状 / 無痛期に記録されるのが典型

🩺 臨床的意味

  • LAD近位部の 高度狭窄 を示唆
  • 介入しないと 短期間で前壁梗塞に移行 しやすい
  • 運動負荷試験は禁忌（MI誘発のリスク）

⚡ アクション

  • 入院 + 同日循環器コンサルト
  • 症状が戻ったら12誘導を 再検（T波陰性が一時的に消えることがある）
  • 早期侵襲戦略の検討は循環器
  • 負荷試験は自分でオーダーしない
見落としの典型

「胸痛は消えたから安全」は 逆。Wellensは 無痛期に出る のがポイント。

🔍 鑑別

  • 持続する虚血の T波陰性（症状あり、動態変化）
  • 心筋炎、肺塞栓（右側誘導中心）
  • 脳出血などでの広範T陰性
  • de Winterパターン（T 波は「高く尖る」方向、時相が違う）
