アキレス腱損傷

同時並行で評価する 3 軸
典型像
  • 30–50 代男性、スポーツ復帰時（テニス・バドミントン・バスケ）
  • ダッシュ・ジャンプ・着地で「後ろから蹴られた」感覚、バチッという音
  • 初期診療で 25% が見逃し。スポーツ以外の非典型例（脱力のみ）が 19–40% → 受傷機転がはっきりしなくても疑う
  • 踵骨上 2–6 cm の凹み（アキレス腱の陥凹）、圧痛
  • Thompson test 陽性：腓腹筋を握って足底屈なし（感度 96%、特異度 93%） — 非典型例にもルーチン実施
  • 足関節底屈は弱いながら可能（他の屈筋が代償） → 「動くから大丈夫」と判断しない
  • 鑑別：腓腹筋挫傷（肉離れ） — 同じ受傷機転で“筋”に起きるか“腱”に起きるかの違い、両方併発もあり
  • フルオロキノロン・ステロイド使用歴はリスク
血管

アキレス腱損傷は 神経血管とは通常関係薄い。確認の流儀として一度だけ。

  • 足背動脈・後脛骨動脈を触知（左右比較）
  • 陳旧例で皮下血腫が大きければ局所循環評価
  • 循環障害を起こす機転ではないので、所見があれば別疾患を疑う
神経

アキレス腱損傷は 神経損傷を伴わない。確認の流儀として 1 分だけ。

  • 腓腹神経（外側踵部の感覚）— 手術アプローチ時に注意される
  • 脛骨神経（足底感覚・足趾屈曲）
  • 麻痺所見があれば別の重大疾患を疑う
画像の見方（US / MRI が主役）

診察での確定（画像より先に）

  • Thompson test：腹臥位で下腿ふくらはぎを握って圧迫 → 足底屈すれば腱断裂なし、しなければ断裂
    • 感度 96%、特異度 93%
  • 触診で gap（踵骨上 3–5 cm の陥凹）。痛みなし可
  • 「底屈できる」で断裂否定は危険：他の底屈筋（長母指屈筋・後脛骨筋など）が代償するため、底屈力は残る

レントゲン

  • 骨性の撕脱骨折がないか確認（稀）
  • 腱断裂自体はレントゲン・CT では分からない

エコー（ER でできる）

  • 感度 80–90%
  • 連続性の途絶・血腫を確認

MRI

  • 診断精度最高、手術計画に有用
  • Grade III／部分断裂の鑑別

Thompson test 陽性なら ER では 臨床診断のみ・画像検査なしでも OK（確定診断は外来でエコー／MRI）。

治療方針（保存 vs 手術）

状況5 択マネジメントER の動き
低活動・高齢者帰宅保存尖足位ギプス・松葉杖

保存 vs 手術

項目保存手術
再断裂率10–15%2–5%
合併症なし創感染・神経（腓腹神経）損傷
機能回復やや劣る良好
適応高齢者・低活動若年・スポーツ
  • 近年は早期可動域訓練を併用すれば保存と手術の再断裂率差が縮まる
  • アスリートは手術、そうでなければ保存も選択肢

ER で打つ手

  1. Thompson test ＋ エコーで診断
  2. 足関節底屈位（尖足位、equinus position）ギプス／シーネ
  3. 松葉杖で 非荷重
  4. 整形外科紹介（帰宅手術 or 帰宅保存）
  5. 鎮痛はアセトアミノフェン中心、後脛骨神経ブロックも選択肢

注意点

  • 陳旧性断裂（2 週以上経過）は腱退縮で手術困難になる → 早期診断必須
  • Haglund 変形などの踵部痛と鑑別
  • フルオロキノロン・ステロイド服用中 → 主治医と相談し中止

帰宅指示

  • 歩行禁止、松葉杖使用
  • 尖足位を保ち、背屈しない（ギプス内で背屈動作しない）
  • 1 週以内に整形外科外来
