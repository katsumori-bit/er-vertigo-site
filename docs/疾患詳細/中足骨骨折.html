中足骨骨折

⚡ 即時アクション
  1. 裸足で両側視診（腫脹・変形・皮膚緊張）
  2. AP + lateral + 斜位 3 方向で撮影
  3. リスフラン関節脱臼骨折を必ず鑑別（第2中足骨基部の側方転位）
  4. 後方シーネ + 免荷（松葉杖）で帰宅、後日整形外来
📝受傷機転 🩺身体所見 🧪画像
AP+lat+斜位 🦴部位と頻度 💉マネジメント 🏠帰宅時説明 📚出典・関連
概要：第2〜4 中足骨は保存型、第5近位端と Lisfranc は分けて考える
中足骨骨折 = 第2〜4 を中心に保存対応、隣接本も探す
  • つま先で強く着地・直達外力・足背打撲が典型機転
  • 多発骨折が多い：1 本見つけたら隣も必ず確認（外果と同様）
  • 頸部・基部に多く、骨幹部は少ない
  • 転位なし単独 → 後方シーネ + 免荷で帰宅、後日整形
  • 第2基部の側方転位 → リスフラン関節脱臼骨折へ
  • 第5近位端は別疾患 → 第5中足骨近位端骨折
📞 この疾患が該当する主訴
  • 足関節・足部の外傷 — 足部外傷の入口主訴

📝 受傷機転

  • つま先で強く着地（ジャンプ着地・段差踏み外し）
  • 重い物の直達外力（足背に物が落下）
  • 高エネ外傷で足背を打撲（交通外傷など）
  • 足部の捻りで骨幹部らせん骨折となることもある

🩺 身体所見

  • 裸足で両側視診（腫脹・皮下出血・変形）
  • 各中足骨を 頭部から基部まで圧痛走査
  • 軸圧痛（指先を長軸方向に押す）
  • 第2中足骨基部の圧痛はリスフラン関節脱臼骨折を疑う
  • 足背の皮膚緊張・コンパートメント徴候の有無
  • 足背動脈・後脛骨動脈・知覚チェック

🧪 画像

レントゲン 3 方向

  • AP + lateral + 斜位を必ず撮る
  • lateral では中足骨同士が重なって読みづらい → 斜位が必須
  • 1 本見つけたら 隣接中足骨も詳細に確認（多発が多い）
  • 第2中足骨基部と内側楔状骨の位置関係（リスフラン評価）

CT

  • リスフラン関節脱臼骨折が疑われる場合
  • 関節内骨折・粉砕型・手術計画

🦴 部位と頻度

部位頻度典型例
頸部（neck）多いつま先着地、直達外力
基部（base）多い足部捻り、直達外力
骨幹部（shaft）少ない強い直達外力・らせん骨折
  • 第2〜4 中足骨は 保存対応の典型骨折
  • 第5中足骨近位端骨折・Jones 骨折は別ページで管理 → 第5中足骨近位端骨折
  • 第2中足骨基部の側方転位はリスフラン損傷を疑う → リスフラン関節脱臼骨折

💉 マネジメント

保存（多くの症例）

  • 転位なし or 軽度の単独骨折
  • 後方シーネ固定（足関節 90° 中間位）
  • 松葉杖で免荷
  • 1 週以内に整形外科外来

整形へ即コンサル

  • 開放骨折
  • 著明な転位・短縮・回旋
  • 複数本の骨幹部骨折（足部アライメント崩壊）
  • リスフラン関節脱臼骨折の合併
  • コンパートメント症候群を疑う所見

🏠 帰宅時の説明

  • 松葉杖で 免荷、荷重開始は整形外科医の指示
  • 1 週以内に整形外科外来を受診
  • 挙上・冷却で腫脹軽減
  • 腫脹・疼痛・しびれ増強・指先の色不良で再受診
