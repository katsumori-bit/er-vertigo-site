前庭神経炎

⚡ 即時アクション
  1. 持続性めまい（24h 以上）+ 一方向性水平眼振 + 歩行不安定＝ AVS の代表
  2. 聴力は正常。難聴を伴うなら AICA 梗塞 も鑑別
  3. HINTS で末梢パターン（HI 陽性 + 一方向性水平 + 偏視なし）を確認
  4. 判断困難 → 中枢として扱い MRI/MRA（後悔最小化）

vs 中枢性 🔬検査 💊治療 📤disposition ⚠️ピットフォール
📞 この疾患が該当する主訴
  • めまい — AVS / 前庭性ノードから到達

🎯 典型像

  • 急性発症の持続性めまい（数時間〜数日続く、24時間以上）
  • 悪心・嘔吐を強く伴う
  • 一方向性水平回旋性眼振（健側向き）
  • 歩行不安定あるが、起立は可能（中枢性ほど重度ではない）
  • 聴力は保たれる（重要な末梢示唆）
  • 神経脱落症状なし
  • 先行する上気道感染やウイルス感染歴があることが多い（ウイルス性が想定）
経過

急性期は数日〜2週間で漸減し、中枢性代償により多くは数週間で日常活動に復帰可能。前庭機能は完全には戻らないことも。

中間例の判断

判断困難 → 中枢扱いで MRI/MRA。HINTS を 1 つでも疑う要素があれば末梢で片付けない。

🔬 検査

ベッドサイド

  • HINTS（AVS 適応）
  • 聴力（指こすり・ささやき）
  • 神経診察
  • 歩行・体幹失調評価

画像

  • 典型的な末梢パターン（HINTS 末梢、神経所見なし、聴力正常）→ 画像不要
  • 中枢示唆 / 判断困難 / 高リスク背景（高齢 + 血管リスク、抗凝固薬等）→ MRI/MRA
  • CT は後方循環虚血の検出感度が低いため、ルーチン使用しない

採血

  • 血糖・電解質・脱水評価（嘔吐強い時）

💊 治療

急性期

  • 対症療法：制吐薬（メトクロプラミド、プロクロルペラジン等）
  • 前庭抑制薬（メクリジン、ジフェンヒドラミン等）：短期のみ。長期使用は中枢性代償を妨げる
  • 嘔吐強ければ点滴補液

ステロイド

  • 発症 3 日以内に短期投与で前庭機能回復を促進する報告あり（プレドニゾロン）
  • 全例必須ではなく、リスク・ベネフィットを検討
  • 糖尿病・感染リスク等の禁忌に注意

抗ウイルス薬

  • 抗ウイルス薬の有効性は確立されておらず、ルーチン投与は推奨されない

前庭リハビリ

  • 急性期を過ぎたら前庭リハビリ（早期離床、視覚運動エクササイズ）が回復を促進

📤 disposition

帰宅可能な条件

  • HINTS 末梢パターン明確、神経所見なし、聴力正常
  • 経口摂取可、嘔吐コントロール可
  • 独居でなくフォロー可能
  • 歩行は不安定でも自力可能

入院 / 経過観察

  • 嘔吐強く経口摂取困難
  • 歩行不能・体幹失調強い
  • HINTS 判断困難 / 中枢示唆あり → MRI/MRA 後の評価まで入院
  • 独居・フォロー困難
  • 高齢 + 血管リスク強い

⚠️ ピットフォール

  • 「持続性めまい = 前庭神経炎」と決めつける — 後方循環梗塞も AVS で来る
  • HINTS なしで末梢診断 — AVS では HINTS 必須
  • 難聴があるのに前庭神経炎と診断 — AICA 梗塞鑑別。難聴 = 末梢ではない
  • 前庭抑制薬を長期処方 — 中枢性代償を遅らせる
  • 歩行不能を軽視 — 体幹失調強ければ中枢性疑い
  • 判断困難で末梢扱い — 中枢扱いで MRI が原則
