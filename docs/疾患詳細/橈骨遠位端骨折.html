橈骨遠位端骨折

同時並行で評価する 3 軸
典型像
  • 機転は FOOSH 一択（転倒して手をついた）
  • 高齢女性の脆弱性骨折（骨粗鬆症介入の入口）
  • 若年者は高エネルギー外傷
  • dinner fork（silver fork）変形
  • 橈骨遠位 3 所見（橈側圧痛・腫脹・回内外制限）すべてなしで感度 99%
  • 上肢骨折で最多
血管

急性虚血は稀。著明腫脹・整復後の動脈触知低下は骨筋膜室症候群を疑う。

  • 橈骨動脈触知（手首掌側）
  • 毛細血管再充満（CRT）< 2 秒
  • 5P（pain / pallor / pulseless / paresthesia / paralysis）
  • 左右差を必ず比較
神経

正中神経障害 5–10% 合併（手根管内圧迫）。整復後も持続するなら整形即コール。

  • 母指〜環指橈側掌側感覚（正中神経）
  • 母指対立・短母指外転筋の筋力
  • Tinel sign（手根管部）
  • 尺骨・橈骨神経領域もスクリーニング
画像の見方
  • 手関節 Xp PA / lateral（標準 2 方向）
  • 必ず測定する 3 パラメータ：
    • radial inclination（橈骨傾斜角）正常 22°
    • radial height（橈骨長）正常 11 mm
    • volar tilt（掌側傾斜角）正常 11° 掌側
  • 関節面段差（≥ 2 mm で手術適応）
  • ulnar variance（短縮の指標）
  • 尺骨茎状突起骨折合併 51–66%（見落とし注意）
  • 嗅ぎたばこ窩圧痛あれば舟状骨評価（scaphoid view 追加）
  • 関節内骨折・段差不明瞭例は CT 追加
  • 整復後・固定後は必ず再撮影で整復位を確認
分類（型）
型転位方向変形・特徴
Colles 骨折 遠位骨片が背側 dinner fork（silver fork）変形。最多
Smith 骨折 遠位骨片が掌側 garden spade 変形（逆 Colles）。不安定・手術率高
Barton 骨折 関節面を含む骨折脱臼 掌側 or 背側。手術適応
Chauffeur 骨折 橈骨茎状突起単独 裂離様

AO 分類：A = 関節外、B = 部分関節内、C = 完全関節内（B・C は整形コンサルト寄り）

治療方針（5 択マネジメント）
振り分け該当所見ER の動き
即時整形 開放骨折／整復不能／血管神経合併（持続正中神経障害・CRT 遅延） 整形即コール、創処置、抗菌薬・破傷風
入院整形 dominant 損傷の高齢者・多発外傷で ADL 維持困難 入院オーダー、シーネ仮固定、患肢挙上
帰宅手術 転位の強い関節内骨折／Smith／Barton／関節面段差 ≥ 2 mm／整復後再転位 整復＋シーネ ＋ 後日手術枠で整形外来
帰宅保存 転位の少ない関節外骨折／徒手整復で良好整復 シーネ ＋ 1 週後整形外来 + Xp
再診不要 該当なし（全例整形フォロー） —

ER で打つ手

  • hematoma block（1% リドカイン 5–10 mL を骨折部に注入）
  • 徒手整復：牽引 ＋ 掌屈 ＋ 尺屈（Colles）。整復後即時 Xp で再評価
  • 固定肢位：
    • Colles（背側転位）→ 軽度掌屈 + 尺屈で volar splint or sugar tong
    • Smith（掌側転位）→ 軽度背屈で固定（再転位予防）
  • MCP は free（指拘縮予防）、母指 IP も露出
  • 三角巾装着、患肢挙上、鎮痛（アセトアミノフェン / NSAIDs）
  • 固定法詳細（volar splint・sugar tong の手技、肢位、AAFP/AO 出典）は 上肢骨折の固定/橈骨遠位端骨折.html を参照

帰宅指示

  • 指の屈伸を積極的に（浮腫予防＋拘縮予防）
  • 挙上（心臓より上）
  • 1 週後整形外来 + Xp（再転位確認）
  • 5P（pain / pallor / pulseless / paresthesia / paralysis）出現で即受診
  • 骨癒合の目安：4–6 週
  • 高齢者：骨密度評価・骨粗鬆症治療介入を整形に申し送り

ピットフォール

  • 尺骨茎状突起骨折見落とし（51–66% 合併）
  • 舟状骨骨折合併見落とし（snuffbox 圧痛で必ず評価）
  • 正中神経障害の整復後持続を見逃す → 整形即コール
  • MCP を巻き込んだ固定で指拘縮
  • 高齢者の骨密度評価介入機会を逃す（脆弱性骨折の入口）
