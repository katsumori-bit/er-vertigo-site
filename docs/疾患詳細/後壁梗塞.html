後壁梗塞

⚡ これを見たら：V1-3 の ST 低下 + R 高 + T 上向き
  1. V1-3 で ST 低下 + R 高 + T 上向き（mirror image）で疑う → 「NSTEMI 扱い」で待機させない
  2. V7-9 を必ず追加で ST 上昇（≥0.5 mm）を確認 → 取ってから呼ぶのではなく呼びながら取る
  3. 緊急カテ（STEMI 等価）→ 循環器オンコール、再灌流経路
  4. 右室梗塞合併（下壁 + 後壁）も念頭、硝酸薬は慎重（前負荷低下で血圧破綻）

📞 この疾患が該当する主訴
  • 胸痛 — 下壁/側壁との合併が多い、典型主訴
  • 入院時心電図 — V1-3 ST 低下を「虚血性変化」で済ませない
  • 失神（意識消失） — 右室梗塞合併での徐脈・低血圧

📊 典型の心電図像

  • V1–V4 を最大とする水平 / 下行性 ST低下
  • V1–V2 の R 波優位（R/S 比 >1）
  • V1–V2 の高く直立したT波
  • 下壁 / 側壁梗塞を合併することが多い（単独後壁は約5%）
迷ったら

V1–V4 の ST低下が最大 なら、V5–V6 主体の虚血ではなく 後壁 を疑う。

📐 追加でやる12誘導

V7–V9 を取る：

  • V7：左後腋窩線、V6と同じ高さ
  • V8：左肩甲骨中線、V6と同じ高さ
  • V9：左傍脊柱、V6と同じ高さ

V7–V9 で ≥0.5 mm（男性 <40歳では ≥1 mm）のST上昇 → 後壁STEMIとして扱う。

⚡ アクション

即動く
  • STEMI と同じトリアージ：循環器オンコール、再灌流経路
  • V7–V9 を取る（取ってから呼ぶのではなく、呼びながら取る）
  • 下壁ST上昇を伴えば右室誘導 V4R も追加

🔍 鑑別

  • 前壁の「普通の」NSTE-ACS 系虚血（ST低下の分布が V5–V6 側に寄る）
  • 右脚ブロックの V1–V2 の二次性変化
  • 右室肥大（R 優位＋右軸偏位など）

⚠️ ピットフォール

  • ST低下だけ見て「NSTEMI扱い」で待機させる
  • V7–V9 を取らない
  • 高いR波を「正常変異」で片付ける
