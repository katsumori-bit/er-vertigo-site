視神経管骨折／外傷性視神経症

🚨 受傷直後からの急性視力低下＋RAPD は視神経管骨折／外傷性視神経症（TON）の主徴。
画像所見を待たず、来院後すぐに視力・対光反射・RAPD を記録し、眼科・脳外を即時コール。早期介入が望ましいとされるが、ステロイド・減圧術ともエビデンスは限定的で専門医の個別判断に委ねる。
典型像（疑うきっかけ）
  • 機転：眉間〜眼窩外上方への打撃、高エネ外傷、Le Fort II/III／ZMC／眼窩尖部骨折で合併
  • 外見上の眼球損傷が乏しいことも多い（特に小児の "white-eyed" 様）→ 意識して評価しないと見落とす
  • 主徴：受傷直後の急性視力低下、RAPD（Marcus Gunn 瞳孔）
  • 視野欠損（色覚低下も初期所見）
  • 進行性視力低下＝視神経鞘血腫や浮腫の二次性圧迫

→ 詳細所見は 同時並行 2 軸 と コンサル前確認 へ

同時並行で評価する 2 軸（治療方針を分ける所見）
視機能の評価と経時記録（治療判断の核）

画像より身体所見が判断材料。30 分毎に記録し進行性かを判断。

  • 視力：Snellen / 指数弁 / 手動弁 / 光覚弁 / 失明 — 経時記録
  • 対光反射・RAPD（swinging flashlight test）：陽性なら片側視神経損傷を示唆
  • 眼底所見：初期は正常（disc 浮腫は数週後）
  • 視野（指数法・対面法）、色覚（赤色比較）
  • 進行性視力低下＝二次性圧迫（出血・浮腫） → 緊急減圧術検討
  • 固定なら保存療法可（経過観察＋ステロイド検討）
合併損傷（治療ルートが分かれる）

頭蓋底・頚動脈損傷を必ず評価。眼窩尖部・上眼窩裂症候群の鑑別も。

  • 眼球運動制限＋V1 知覚障害＋視力低下＝眼窩尖部症候群（II/III/IV/V1/VI 多発）
  • 視力保たれ、眼球運動麻痺＋V1 知覚障害のみ＝上眼窩裂症候群（眼窩尖部の手前）
  • 頭蓋底骨折・頭蓋内出血（→ 頭部外傷評価）
  • 外傷性内頚動脈解離：CT angio で評価検討
  • Le Fort II/III／ZMC 骨折と合併（→ Le Fort 骨折／頬骨上顎複合体骨折）
ER で打つ手（初期処置）
  • 視力・対光反射・RAPD を 30 分毎に記録（進行性かを判断）
  • 頭部挙上 30°、SBP ≥ 110、SpO₂ ≥ 94% を維持
  • 鼻かみ禁・くしゃみは口を開けて（眼窩内圧上昇回避）
  • 頭部 / 顔面 / 頸椎 / 眼窩薄スライス CT を 1 セットで撮影
  • 頭蓋内・気道・循環の安定が前提
  • 眼科コンサルトを即時。施設方針に従いステロイド準備
  • 頭蓋内圧亢進・頭蓋底骨折合併時は脳外と協議
  • 絶飲食、ライン確保、術前検査
眼科・脳外コンサルト前に確認すべきこと

視神経管骨折／TON を疑ったら、即時コンサルト。並行して以下を整理。緊急であることを冒頭で明示。

  • 視力（来院時 → 経時）：受傷側／健側、Snellen or 指数弁／光覚弁／失明。30 分毎の値
  • 対光反射・RAPD：左右差、RAPD の有無
  • 視野：両側／○側で耳側／鼻側半盲
  • 色覚：赤色比較で低下の有無
  • 眼球運動：制限の有無と方向（眼窩尖部症候群の鑑別）
  • 眼底：視神経乳頭浮腫の有無（初期は通常正常）
  • 受傷後の症状経過：受傷直後から低下／受傷後○時間後より進行
  • 他顔面所見：頬部しびれ、咬合異常、髄液漏
  • 画像所見：眼窩〜頭蓋底薄スライス CT で視神経管骨折線（部位）、骨片による狭窄、合併骨折（Le Fort／ZMC／眼窩尖部）
  • MRI：施行可否、視神経腫脹の有無
  • 頭部 CT：頭蓋内病変、頸椎：問題の有無
  • 受傷前視力：両眼良好／既往（白内障・緑内障など）
  • 背景情報：抗凝固薬・抗血小板薬・ステロイド常用・糖尿病・感染症
  • ER 処置の記録：頭部挙上、鼻かみ禁、絶飲食、ステロイド（製剤・用量・時刻）
  • 術前検査：採血・心電図・胸部 X 線
画像所見と治療方針

① 画像で見る項目

眼窩〜頭蓋底の薄スライス CT（0.5〜1 mm）軸位＋冠状位再構成が必須。

  • 視神経管骨折線を視神経管の内側壁・外側壁・天井・床で評価
  • 骨片による管狭窄の有無
  • 視神経管内出血／視神経の腫脹は MRI（T2／FLAIR／DWI）が有用（CT 陰性でも TON はあり得る）
  • 蝶形骨小翼・前頭蓋底・眼窩尖部の骨折を同時に評価
  • 頚動脈評価が必要なら CT angio（外傷性解離・嚢動脈瘤）
  • 注意：CT 陰性でも臨床的に TON は否定できない。画像所見と症状は乖離しうる

② TON の分類

  • 直接型：骨折・骨片の直接圧迫
  • 間接型：剪断・浮腫・血管攣縮

③ 治療方針（エビデンスは限定的）

外傷性視神経症の治療は議論が続いている領域。IONTS（International Optic Nerve Trauma Study, 1999）では 観察／ステロイド／手術の各群で予後に有意差なしと報告され、CRASH 試験（2004）以降は頭部外傷でのステロイドルーチン投与は推奨されない。早期介入が望ましいとされるが、「8 時間以内」を絶対視せず、専門医（眼科・脳外）の個別判断に委ねる。

選択肢内容留意点
経過観察 軽症・進行なし・RAPD のみ 保存療法でも自然改善あり（IONTS 観察群 32%）
高用量ステロイド療法 かつて NASCIS 様プロトコルが用いられた 現在はルーチン推奨されない（IONTS で有意差なし、CRASH 試験で頭部外傷へのステロイドは予後改善せず）。糖尿病・感染リスクで害となる可能性。専門医個別判断
視神経管減圧術 経鼻内視鏡的アプローチが主流。骨片による直接圧迫例で適応 進行性視力低下・骨片による明らかな圧迫が比較的良い適応。IONTS で観察と有意差なしの報告もあり、エビデンスは限定的
併用療法 ステロイド＋減圧術 無作為比較試験は限定的

※ MRI 施行前に眼球内金属異物・体内金属の禁忌確認を必ず。

※ 頸動脈海綿静脈洞瘻（CCF）の鑑別も忘れない：眼球突出＋拍動性雑音＋眼球運動障害。
