大腿骨近位部骨折

同時並行で評価する 3 軸
典型像
  • 高齢女性：転倒で外旋・短縮位
  • 「急に歩けない」内科受診も 4.4%（転倒なしでも鑑別へ）
  • 鼠径部・大転子部の圧痛、内外旋で疼痛
  • Xp 感度 91.9–98.1%、2–10% は occult
  • 術後の転倒はインプラント周囲骨折を疑う
血管

頸部骨折は大腿骨頭血流障害で偽関節化（関節包内）。骨幹部寄りは出血量に注意。

  • 転位の大きい頸部 → 骨頭壊死リスク（人工骨頭適応）
  • 転子部は血流豊富 → 内固定（SHS / γ nail）
  • 転子下・骨幹部寄りは大腿動脈分枝からの出血
  • 抗凝固薬の有無を確認（手術日程に影響）
神経

高エネルギー転子下では坐骨神経領域を確認。低エネルギー高齢者では神経合併は稀。

  • 足趾運動・足背の感覚（坐骨神経領域）
  • 大腿前面の感覚（大腿神経）
  • 譫妄・認知症で訴えが乏しい高齢者は記録を厚く
  • 麻痺所見は固定肢位を見直す根拠になる
画像の見方

まず撮るもの

  • 骨盤正面 + 股関節正面 + 軸位（ラウエンシュタイン位で代用する施設もある）

Garden 分類（頸部骨折のみ）

stage所見術式
I外反陥入型、不完全骨折内固定（multiple cancellous screw）
II完全骨折、転位なし内固定
III骨折面に部分的転位人工骨頭置換
IV完全な転位人工骨頭置換
  • 非整形外科医にとって Garden 分類は整形外科医とのコミュニケーションツール。「Garden III の頸部骨折です」と言えるとコンサルトがスムーズ。
  • 転子部骨折の分類は覚えなくてよい（整形医はこだわらない）。

Occult fracture（亡霊骨折）

  • 2–10% は Xp に映らない。「歩けたから帰宅」は訴訟リスク
  • Xp 陰性でも内外旋痛・大転子圧痛あれば疑う
  • MRI が GS（感度・特異度ほぼ 100%、T1 黒・STIR 白）
  • 夜間 MRI 不可なら CT で代用（前額・矢状・冠状の全スライス、2–3 連続 or 骨皮質途絶を骨折線とする）
  • 画像で否定できなければ歩行可でも帰宅させない、整形へ「内外旋痛あり、occult 疑い」

インプラント周囲骨折

  • 術後高齢者の転倒・患肢痛で必ず鑑別（頻度 0.1–18%）
  • インプラントは折れない → 末端付近（骨幹部）で折れる
  • Occult もあるので Xp 陰性でもCT 追加
  • 術側病院の旧画像とインプラント機種を整形に伝える

筋挫傷との鑑別（Xp 陰性）

  • Xp 陰性例の 61–65% に内転筋筋挫傷あり
  • 身体所見では区別不能 → MRI で STIR 白の場所（骨か筋か）
  • 痛みで歩けなければ入院
治療方針（頸部 / 転子部 × Garden × 5 択）
非整形医の初期対応はどちらも同じ。コンサルトでは必ず区別

「大腿骨近位部骨折」はカルテ用語。コンサルト時は頸部か転子部かを伝える（整形外科の業界ルール）。

区分部位治療
頸部骨折関節包内血流障害・偽関節化 → 転位あり人工骨頭、転位なし内固定
転子部骨折関節包外血流豊富 → 内固定（SHS / γ nail）
転子下骨折関節包外の遠位稀。まず上 2 つをマスター
状況5 択マネジメントER の動き
日中・平日即時整形（理想 24h、48h 以内手術）整形即コンサルト
夜間・休日入院整形 → 翌朝 cons.シーネ ＋ 大腿神経ブロック
自院手術室満床転院調整他院整形と連携
Occult / 筋挫傷疑い入院整形（MRI 待ち）「内外旋痛あり」で通る
インプラント周囲術側病院へ転送 or 入院機種・旧画像取り寄せ

ER で打つ手

  • 外旋・短縮位 + 80 歳女性なら身体所見省略で Xp 直行可
  • 大腿神経ブロックが第一選択 → 大腿神経ブロック（NICE 2023, AAOS 推奨）
  • アセトアミノフェン（アセリオ 15 mg/kg 静注）
  • NSAIDs は避ける（高齢者の腎・消化管）
  • VTE 予防、譫妄対策、尿カテ早期抜去・早期離床
  • 術後の骨粗鬆症治療開始（ビスホスホネート未処方なら導入機会）
  • 24h 以内手術は 48h 超より死亡率・合併症が少ない（NICE, AAOS）

関連ハブ・隣接疾患
  • 主訴入口：股関節痛 ／ 転倒
  • 隣接疾患：大腿骨骨幹部骨折 ／ 大腿骨遠位端骨折 ／ 脆弱性骨盤骨折

  • 合併症対応：DVT 予防、譫妄対策、骨粗鬆症治療 をフォローへ
