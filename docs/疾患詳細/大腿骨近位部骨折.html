#ER #救急 #骨折 #大腿骨近位部骨折 #大腿骨頸部骨折 #大腿骨転子部骨折 #occult #Garden分類 #高齢者

このページの使い方
  1. 鑑別（転倒後股関節痛）に挙げて 骨折線イメージを持ってから Xp を読影
  2. Xp で見える骨折 → 頸部 / 転子部の区別 + Garden 分類 → 整形コンサルト
  3. Xp で見えない → occult fracture を疑い、内外旋痛 → MRI（or CT）
  4. 目標は 受傷後 48 時間以内（可能なら 24 時間以内）の手術

疫学

  • 国内で 約 20 万例/年 のコモンディジーズ。
  • 骨粗鬆症を背景とした脆弱性骨折の第 2 位（第 1 位は脊椎圧迫骨折）。
  • 受傷機転は 転倒 82.0% / 交通事故 9.9% / 原因不明 4.4%。
  • 「なぜか急に歩けなくなった」で来院する内科受診例が 4.4% いる → 転んでいなくても鑑別に挙げる。

2 つの骨折を区別する

非整形外科医の初期対応はどちらも同じ

ただしコンサルトでは必ず区別する（整形外科の業界ルール）。 「大腿骨近位部骨折」という単語はカルテ用語で、コンサルト時には使わない。

区分骨折部位治療
大腿骨頸部骨折関節包 内血流障害・偽関節化しやすい → 転位あれば人工骨頭置換、転位なければ内固定
大腿骨転子部骨折関節包 外血流豊富で癒合しやすい → 内固定（SHS / γ nail など）
大腿骨転子下骨折関節包外のさらに遠位稀。まず上の 2 つをマスター

身体診察

  • 外旋位・短縮位：1 秒で判断できる。見たら身体所見は省略して Xp 直行可
  • 鼠径部・大転子部の圧痛
  • 股関節の内旋・外旋で疼痛（他動運動で誘発）→ occult fracture の最も実用的な所見
  • 大転子部の新鮮皮下血腫（感度 56%、特異度は骨折なしでも 6%）
  • 聴性打診：恥骨結合に聴診器、膝蓋骨を打診して左右差（感度 79–96% / 特異度 86–95%）— 整形外科医はマイナーと捉えがち
痛くて動かせない高齢者は身体所見を省略してよい

『80 歳女性 転倒』＋『外旋・短縮位』→ すぐに Xp 室へ直送。 『身体所見 → 画像』ではなく『鑑別 → 画像 & 身体所見』。

画像

まず撮るもの

  • 骨盤正面 + 股関節正面 + 軸位（ラウエンシュタイン位で代用する施設もある）
  • 詳細な範囲は 股関節痛

Garden 分類（頸部骨折のみ）

stage所見術式
I外反陥入型、不完全骨折内固定（multiple cancellous screw）
II完全骨折、転位なし内固定
III骨折面に部分的転位人工骨頭置換
IV完全な転位人工骨頭置換
  • 非整形外科医にとって Garden 分類は整形外科医とのコミュニケーションツールとして重要。「Garden III の大腿骨頸部骨折です」と言えるとコンサルトが圧倒的にスムーズ。
  • 転子部骨折の分類は覚えなくてよい（整形外科医は非整形医にこだわらない）。

Occult fracture（亡霊骨折）

Xp 陰性＝骨折なし ではない

大腿骨近位部骨折の Xp 感度は 91.9–98.1%。 2–10% は occult fracture となり、レントゲンに映らない。 歩行できる例もあるため、「歩けたし Xp 正常だから帰宅」は訴訟リスク。

疑い方 — 鑑別に挙げていれば気づける

  1. 高齢者 + 転倒 + 股関節痛 → 大腿骨近位部骨折を鑑別に挙げる
  2. Xp で骨折線が見えない → occult の可能性を意識
  3. 股関節の内外旋で疼痛 or 大転子部圧痛 → occult 濃厚

MRI（ゴールデンスタンダード）

  • 大腿骨近位部骨折に対して感度・特異度ほぼ 100%
  • T1 黒・STIR 白が新鮮骨折のサイン
  • 正常骨 = STIR 黒 / T1 白。骨浮腫で白黒が逆転する

MRI が撮れない時 — CT で代用

  • 夜間・休日で MRI 不可 → CT で評価可能
  • 前額断・矢状断・冠状断のすべてのスライスで確認
  • 2–3 スライス連続 or 骨皮質の途絶があれば骨折線として扱う
  • CT で骨折線が指摘できる率：
    • ガイドラインでは MRI 優位（CT 2% vs MRI 47% の報告あり）
    • 実臨床では CT も十分役立つ（夜間のマネジメント決定には有用）

MRI も CT もない時

  • 画像で否定できなければ歩行可でも帰宅させない
  • 股関節の内外旋痛を他動的に評価し、疑えば整形外科へ入院依頼
  • 「電話で整形にコンサルト：『内外旋で疼痛あり、occult fracture 疑い』」が通る殺し文句

インプラント周囲骨折（大腿骨ステム周囲骨折）

  • 大腿骨近位部骨折術後の転倒・患肢疼痛で必ず鑑別
  • 発生頻度 0.1–18%。術後高齢者の増加とともに急増
  • インプラントは折れない → インプラントの末端付近（骨幹部）で折れる
  • Occult も起こるので Xp 陰性でも CT 追加

原則

インプラントを入れた病院で再手術する

過去のインプラント情報・術式を知っている病院で手術するのが業界ルール。 自院で術をしていなければ転院調整が必要。 土日は bridge して週明けに転院依頼でよい。

筋挫傷（内転筋）

  • 転倒後股関節痛の Xp 陰性例の 61–65% で、MRI で筋挫傷所見あり
  • 内外旋で疼痛誘発 → 大腿骨近位部骨折と身体所見では区別できない
  • MRI で STIR 白が筋肉にある（骨にあれば occult 骨折）
  • 治療：保存加療、安静度フリー。痛みで歩けなければ入院

マネジメント

コンサルトのタイミング

状況対応
日中・平日即時整形（手術は 48 時間以内、理想は 24 時間以内）
夜間・休日入院整形 → 翌朝コンサルト、日中に手術
自院の手術室が埋まっている転院調整（他院の整形と連携）
  • 24 時間以内手術は 48 時間超手術より死亡率・合併症が少ない（NICE, AAOS）
  • 実臨床では平均 39 時間、2/3 が 24 時間超（最大の遅延理由は手術室不足 61%）

疼痛管理

  • アセトアミノフェン（アセリオ® 15 mg/kg 静注）— 第 1 選択
  • 大腿神経ブロック — 施行可能なら非常に有効（NICE 2023, AAOS）
  • NSAIDs は避ける（高齢者の腎障害）

入院中の管理

  • VTE 予防（術前・術後）
  • 譫妄対策（高齢者のハイリスク群）
  • 尿カテーテル早期抜去・早期離床
  • 術後の骨粗鬆症治療開始（ビスホスホネート未処方なら導入機会）

やりがちなミス

  • 「歩けたから骨折なし」と帰宅させる → Occult で帰宅はトラブル・訴訟リスク
  • コンサルト時に「大腿骨近位部骨折」と伝える → 整形医は「頸部か転子部か」を聞きたい
  • Garden 分類を添えない → コンサルトが遅くなる
  • インプラント術後の転倒で鑑別から漏らす
  • 夜間 MRI にこだわりすぎる → CT で代用可
  • NSAIDs を出してしまう（腎機能悪化）
