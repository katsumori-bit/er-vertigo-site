#骨折 #足関節 #果部 #ankle #ER

足関節果部骨折 = 内反捻挫で外果、外反で内果、両方あれば不安定
  • 内反捻挫が最多の受傷機転（足が内側に倒れる）
  • 外果骨折（Weber 分類 A/B/C）と 内果骨折・後果骨折
  • 単果 → 保存多、両果・三果 → 手術（帰宅手術）
  • Maisonneuve 骨折（腓骨近位端）を必ず鑑別

受傷機転

  • 内反捻挫（最多）：外果骨折
  • 外反ストレス：内果骨折 + 三角靱帯損傷
  • 軸圧 + 回旋：両果・三果骨折

身体所見

  • 外果・内果の圧痛（Ottawa の判断基準）
  • 足関節周囲の腫脹
  • 腓骨近位端の圧痛チェック（Maisonneuve の除外）
  • 荷重試験：受傷直後 + ER で 4 歩歩けるか

画像

レントゲン 3 方向

  • 正面・側面・mortise view（足関節を 15–20° 内旋）
  • 関節適合性（mortise）の評価
  • 関節裂隙の左右差：Talocrural angle と medial clear space

CT

  • 関節内骨折の詳細
  • 粉砕型や手術計画

Weber 分類（腓骨遠位端）

Type骨折位置安定性治療
A脛腓靱帯結合下（足関節裂隙より下）安定保存
B脛腓靱帯結合のレベル（らせん状骨折が多い）中間転位なければ保存、ありなら手術
C脛腓靱帯結合より上（Maisonneuve 含む）不安定手術

骨折パターンと治療

パターン安定性治療
単果（外果 or 内果）多くは安定保存 or 帰宅手術
両果骨折不安定帰宅手術（ORIF）
三果骨折（外果 + 内果 + 後果）不安定帰宅手術、後果 > 25% なら手術必須
Maisonneuve不安定帰宅手術

マネジメント

非整形外科医の役割

  1. Ottawa Ankle Rules でレントゲン適応判断
  2. 腓骨近位端圧痛チェック（Maisonneuve 除外）
  3. 転位なし・単果 → ギプスシーネ固定 + 非荷重
  4. 転位あり・両果以上 → シーネ固定 + 整形外科紹介（帰宅手術）
  5. 開放骨折・脱臼骨折 → 即時整形

固定

  • U-slab（後方シーネ + U 字シーネ）
  • 90° 中間位で固定
  • 松葉杖で 非荷重

整復が必要な場合

  • 骨折脱臼（転位大・皮膚緊張）
  • 受傷直後なら 鎮痛 + 牽引整復を ER で試みる
  • 整復後レントゲンで位置確認

帰宅時の説明

  • 松葉杖で非荷重、荷重開始は整形外科医の判断
  • 1 週以内に整形外科外来
  • 腫脹・疼痛・しびれ増強で再受診
