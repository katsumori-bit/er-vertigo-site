めまい持続性判定（AVS / EVS）

⚡ 即時アクション
  1. 持続性 (AVS) = 急性発症 + 24時間以上持続するめまい + 眼振 + 歩行不安定
  2. 反復性 (EVS) = 発作的に繰り返すめまい（発作間は無症状）

📚 AVS / EVS 分類の整理

TiTrATE 思想（Newman-Toker, 2015）では、めまいの「質」（回るか・ふわふわか）ではなく「時間と誘因」で症候群を分類する。これにより workflow（HINTS を使うか、Dix-Hallpike を使うか等）が決まる。

症候群定義典型代表疾患workflow
失神前型 眼前暗黒感・立位悪化・動悸など 不整脈、起立性低血圧、出血、心原性 意識消失フロー
AVS
（急性持続性）		 急性発症 + 24時間以上持続 + 眼振 + 歩行不安定 前庭神経炎、後方循環梗塞、AICA 梗塞 HINTS で前庭/中枢を判別
t-EVS
（誘発性）		 頭位変換などで誘発、数秒〜1分で消失、発作間は無症状 BPPV Dix-Hallpike → Epley 法
s-EVS
（自発反復）		 誘因なく自発的に発作、数分〜数時間、発作間は無症状 Ménière 病、前庭性片頭痛、TIA、不整脈 背景・反復頻度・随伴で個別評価
分類できれば workflow が決まる

「持続しているか？」だけで AVS（24h+）vs EVS（発作） に大別できる。EVS の中はさらに「誘因あり / なし」で t / s に分ける。

⏱ 持続時間の目安表

持続時間典型分類代表疾患
数秒〜1分t-EVS（誘発性）BPPV（頭位変換で誘発）
数分（5〜30分）s-EVSTIA、前庭性片頭痛 軽症
20分〜12時間s-EVSMénière 病、前庭性片頭痛
数時間〜24時間未満s-EVS の延長 or 早期 AVS境界例（s-EVS 重症 or AVS 初期）
24時間以上持続AVS（急性持続性）前庭神経炎、後方循環梗塞
慢性持続（>1ヶ月）慢性めまい慢性前庭機能不全、機能性、薬剤性

「持続している」の解釈

  • 診察時にもめまいが続いているのが AVS の典型
  • 来院時に軽快していても、24時間以上連続していたなら AVS の workflow
  • 診察時に症状なし、過去 1 日に発作 → 反復性（EVS）の可能性
境界例（24時間前後）の扱い

持続時間が 12〜24 時間で「AVS とも EVS とも言いにくい」場合、AVS として扱うのが安全（後方循環梗塞の見逃し回避）。

🗣 問診のコツ

2 つを分けて聞く

  1. 今もめまいは続いていますか？（現在の状態）
  2. 始まってから一度も止まらず続いていましたか？（経過）

2 つの答えで分類が決まる：

  • ① Yes + ② Yes（連続して持続）→ AVS
  • ① No + ② No（発作と発作の合間あり）→ EVS
  • ① No + ② Yes（連続だったが今は治まった）→ AVS の改善期（HINTS 困難、慎重に）

誘因の聞き方

  • 頭の向きを変えて起こる？ → t-EVS（BPPV）
  • 立った時・坐った時に起こる？ → 起立性低血圧（前失神型）
  • 何もなく突然始まる？ → AVS or s-EVS

反復性を疑うサイン

  • 「昨日も似たことが」「以前にもあった」
  • 耳鳴・難聴を伴う発作 → Ménière
  • 頭痛と一緒に → 前庭性片頭痛
  • 短時間（数分〜30分）で完全に治る → s-EVS / TIA

持続性 (AVS) を疑うサイン

  • 「ずっと続いている」「治らない」
  • 悪心・嘔吐が強い、来院時も眼振あり
  • 歩行・起立が困難

🔀 分類後の流れ

AVS（持続性）

  • → HINTS で前庭神経炎 vs 中枢性脳血管障害を判別
  • HINTS で末梢パターン → 前庭神経炎
  • HINTS で中枢パターン or 判断困難 → 後方循環梗塞 として MRI/MRA
  • 「歩けないなら入院」が原則

t-EVS（誘発性）

  • → Dix-Hallpike で BPPV を確認
  • 陽性 → Epley 法で治療、画像不要
  • 非典型的眼振 / 反応乏しい → 中枢性も鑑別

s-EVS（自発反復）

  • → 反復頻度・持続時間・随伴症状で個別評価
  • 耳症状あり：Ménière 病
  • 頭痛伴う：前庭性片頭痛
  • 血管リスク高 + 数分の発作：TIA を強く疑う
  • 動悸・失神前様：不整脈評価（ECG / ホルター）

🤔 判別困難例の扱い

BPPV と AVS の中間例

  • BPPV と診断したが症状が長引く → 中枢性の可能性あり
  • 頭位変換でも誘発されるが「自発的にも続く」 → AVS 寄り
  • 判別困難 = 中枢扱いで MRI/MRA（後悔最小化）

「24時間前後」の境界例

  • 20 時間続いて軽快、3 時間後に再発 → 単純に AVS or EVS で割り切れない
  • 来院時に軽快していても、連続持続歴があれば AVS の workflow
  • HINTS は「現在も眼振あり」が前提なので、軽快期は HINTS 不可 → 画像重視

高齢者の不明瞭な訴え

  • 持続時間が曖昧な場合、「今朝起きてからずっと？」「昨日の夜から？」など具体的な参照点で聞き直す
  • 家族・付き添いから時系列を確認
判別困難な時の鉄則

判別困難 = 中枢として扱う。後方循環梗塞を末梢で片付けると致命的。MRI/MRA でクリアにしてから帰宅判断。

⚠️ ピットフォール

  • 「今めまいない」だけで反復性と決める — 24時間以上持続歴があれば AVS
  • 短時間 = BPPV と決め打ち — TIA も数分、後方循環の小発作もある
  • 境界例（12〜24h）を EVS 扱い — AVS 寄りで安全側に
  • BPPV 後の持続感を「BPPV 後遺症」で済ます — 中枢性の可能性
  • 「いつから？」だけで判断 — 連続性（途中で止まったか）も必ず確認
  • 慢性のふらつきを AVS と扱う — AVS は急性発症が必須
