喘息増悪

⚡ 即時アクション
  1. SpO2 / PEFR / 努力呼吸で重症度評価（mild-moderate / severe / life-threatening の三層）
  2. SABA + 全身ステロイドを即時開始、重症は イプラトロピウム + Mg 静注追加
  3. life-threatening（PEFR < 33%、SpO2 < 92%、無声）→ ICU・挿管検討
  4. シフト性アシドーシスで「悪化したように見える改善」（PaCO2 正常〜上昇は疲弊サイン）

📞 この疾患が該当する主訴
  • 呼吸困難 — 急性発作・喘鳴の主訴

📋 前提

呼吸困難 のフローに沿って以下が済んでいる or 進行中：

  • バイタル・呼吸仕事量 確認済み
  • 血液検査 提出済み
  • 心電図・胸部X線 施行済み or 進行中

🩺 重症度を判定する

Life-threatening のサイン
  • drowsy / confused
  • silent chest / quiet chest
  • SpO2 < 92%
  • 会話困難
  • 低血圧 / 不整脈 / exhaustion

→ 先に治療を始める。上級医 / ICU にすぐ連絡。

GINA 2025 の急性増悪フローはこの三層構造で、会話・呼吸数・脈拍・SpO2・PEF を使う。

💊 治療を始める

重症そうなら迷わず全部入れる：

  • SABA：pMDI + spacer で 4–10 puff、20分ごとに3回まで反復
  • 酸素：目標 SpO2 93–95%
  • ipratropium（アトロベント）：severe 以上で追加（入院リスク低下のエビデンスあり）
  • 全身ステロイド：プレドニゾロン 40–50 mg 内服（嘔気あれば IV ヒドロコルチゾン 100–200 mg）、5–7日
  • 反応不良なら MgSO4 IV：2 g を 20分で点滴静注
迷ったら

重症そうなら：SABA + O2 + ipratropium + steroid ここまでを先に入れる。考えるのはその後でよい。

やらないこと

  • sedation は避ける（BTS/SIGN: "No sedatives of any kind"）
  • 抗菌薬を routine で入れない（感染が明らかなとき以外）

🔄 1 時間後に判定する

  • 息苦しさが改善したか
  • 会話が戻ったか
  • SpO2 が保てるか
  • PEF が上がったか（取れれば）
  • SABA をまだ頻回に欲しがるか
  • 疲れてきていないか

血ガスを見るとき

  • 見た目の割に悪い
  • 疲弊している
  • quiet chest
  • 「正常〜上昇した PaCO2」は危険サイン（BTS/SIGN） — 通常の喘息発作では過換気で CO2 は低下するはず。正常 or 上昇は疲弊を意味する。

📤 disposition

帰宅寄り

  • 症状改善
  • SABA 連発不要
  • SpO2 >94%（room air）
  • PEF 改善して 60–80% 超
  • 自宅資源あり

入院寄り

  • 反応が鈍い
  • PEF 低いまま
  • 酸素が要る
  • 再増悪しそう

上級医 / ICU すぐ

  • drowsy / confused
  • quiet chest
  • exhaustion
  • 低血圧 / 不整脈
  • 呼吸不全進行

🏥 入院時オーダー・病棟指示の枠（急性喘息）

GINA 2025・BTS/SIGN を踏まえた病棟指示の骨格。

モニタリング

  • SpO2（目標 93–95%）
  • PEF（可能なら毎日、個人最良値との比較）
  • バイタル・意識：静かな胸（quiet chest）、会話可否、疲弊

薬物

  • SABA 吸入（スペーサー併用）— 頻回投与はプロトコル化（例：1 時間ごと評価）
  • SAMA：重症例で追加（GINA）
  • 全身ステロイド：プレドニゾロン 40–50 mg/日相当、5–7 日（早期減量は再発リスク — BTS/SIGN）
  • 硫酸マグネシウム 2 g IV：重症で反応不良時（GINA）

酸素・呼吸

  • 酸素で SpO2 目標を維持
  • NIV：高炭酸血症性呼吸不全で考慮（施設方針・呼吸器）

やらないことの再掲

  • 鎮静薬のルーチン使用は避ける（BTS/SIGN）
  • 抗菌薬は感染が明らかな場合以外ルーチンに使わない

📅 外来フォロー・退院後（喘息）

  • 退院／帰宅後 2–7 日以内の外来再診（GINA：増悪後のフォロー）
  • 書面のアクションプラン（緑・黄・赤ゾーン、吸入の段階的使用）を必ず更新
  • 吸入ステロイド（ICS）の継続：増悪時に SABA のみに偏らないよう確認（GINA は ICS を基幹とする管理を推奨）
  • スモーキング・アレルゲンなど誘因の再確認

⚠️ 落とし穴

喘息ではないかもしれないとき

  • 片側呼吸音低下 → 気胸
  • 発熱・局在性 crackles → 誤嚥性肺炎
  • 胸痛・頻脈・DVT 所見 → 肺塞栓
  • 吸気性喘鳴・喉頭違和感 → アナフィラキシー / 上気道
  • 湿性ラ音・起坐呼吸 → 急性心不全（cardiac wheeze との鑑別）

GINA 2025 も代替診断として心不全、上気道機能異常、異物吸入、肺塞栓を明示している。

退院時に最低限

  • ICS を途切れさせない
  • 吸入手技確認
  • アクションプラン確認
  • 2–7日以内フォロー
  • 再増悪リスク確認
