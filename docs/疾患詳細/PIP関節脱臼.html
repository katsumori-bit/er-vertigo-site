PIP関節脱臼

⚡ 即時アクション
  1. 背側 / 掌側 / 側方を区別、背側が最多（過伸展ストレス、突き指）
  2. 整復後 X 線で関節適合性確認（指根部ブロック → 牽引 → 屈曲）
  3. volar plate 損傷の評価 — 基節骨掌側基部の撕脱骨片が 1/3 以上なら手術
  4. 屈曲拘縮の後遺症リスク、buddy taping + 早期 ROMで対策
📝受傷機転 ↔️脱臼の方向 🩺身体所見 🧪画像
整復前後 🛠️整復手技
背側脱臼 ✅整復後の評価 💉マネジメント 🔬Volar plate 損傷 🏠帰宅・出典
📞 この疾患が該当する主訴
  • 手関節・手の外傷 — 突き指後の PIP 変形・腫脹

📝 受傷機転

  • 突き指による過伸展ストレス（バスケ・バレー）
  • PIP 関節が背側に脱臼（中手指節側が掌側に転位）

↔️ 脱臼の方向

方向頻度病態
背側脱臼最多掌側板損傷、過伸展で発生
側方脱臼次いで多い側副靱帯断裂
掌側脱臼稀中央索断裂 → ボタン穴変形

🩺 身体所見

  • PIP 関節の変形 + 腫脹
  • 動かない / 激痛
  • 神経血管所見：指先の感覚・循環

🧪 画像（整復前後で必須）

  • 指 3 方向
  • 掌側板撕脱骨片（基節骨掌側基部）を確認
  • 骨片が関節面の 1/3 以上なら手術適応

🛠️ 整復手技（背側脱臼）

  1. 指根部ブロック（リドカインで指の根元から麻酔）
  2. 基節骨と中節骨を同じ軸上に並べる
  3. 中節骨をわずかに背屈 + 掌側へ牽引 → 屈曲
  4. コリッと整復される

✅ 整復後の評価

  1. 神経血管所見
  2. 整復後レントゲン
  3. 関節の安定性：軽く屈伸して不安定性がないか
  4. PIP 関節の側方不安定性（側副靱帯損傷）

💉 マネジメント

背側脱臼（多い）

  • 整復後アルミシーネで 10–30° 屈曲位固定 1 週間
  • その後 buddy taping（隣指とテーピング）で可動域訓練
  • 帰宅保存、1 週以内に整形外科外来

掌側脱臼（稀・要注意）

  • 中央索断裂 → 放置でボタン穴変形（boutonniere deformity）
  • PIP 関節伸展位で 6 週固定
  • 整形外科（手外科）紹介

骨折脱臼（掌側板撕脱 1/3 以上）

  • 帰宅手術として手外科紹介
  • 関節面の不整合 → PIP 関節の慢性不安定

🔬 掌側板損傷（Volar plate injury）

  • 背側脱臼で必発
  • 基節骨掌側基部の撕脱骨片
  • 骨片小 → 保存、骨片大 → 手術

🏠 帰宅時の説明・出典

帰宅時の説明

  • 1 週以内に整形外科受診（手外科が望ましい）
  • 数週間の可動域制限が残る
  • PIP 関節は拘縮しやすいので早期可動域訓練が重要

出典
