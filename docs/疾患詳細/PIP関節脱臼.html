PIP 関節脱臼

同時並行で評価する 3 軸
典型像
  • 球技・転倒で「突き指」 → PIP 関節背側に変形
  • 多くは 背側脱臼（中節骨が背側）
  • 掌側脱臼は稀（中央索損傷で boutonnière）
  • 腫脹・変形・自動運動不能
  • volar plate 損傷（裂離骨片）が高頻度
血管

血管損傷は通常合併しない。整復前後で固有指動脈・CRT を再評価。

  • 固有指動脈の拍動・色調・CRT（< 2 秒）
  • 整復前と整復後で必ず再評価
  • 左右指比較で問題なし
  • 強い腫脹は挙上・冷却
神経

神経損傷は通常 OK。固有指神経の感覚を整復前後で確認。

  • 各指の固有指神経の感覚（橈尺両側）
  • 整復前・整復後で記録
  • 正中・尺骨・橈骨神経領域 [問題なし]
  • 麻痺所見は記録
画像の見方
  • 整復前 Xp（AP / lateral / oblique）→ 整復 → 整復後 Xp（必ず 2 セット）
  • 整復前 Xp で確認：
    • 脱臼方向（背側 / 掌側 / 側方）
    • volar plate 裂離骨片（中節骨基部掌側）
    • 関節内骨折（中節骨基部の段差）
    • 関節面の1/3 を超える骨片 → 不安定脱臼骨折
  • 整復後 Xp で確認：
    • 関節整合性（concentric reduction）
    • 軽度屈曲位での再脱臼傾向（V sign）
    • 骨片の関節内嵌頓
  • 整復不能・関節嵌頓は画像撮影前にコンサルでも可
治療方針（脱臼方向 × 安定性）
パターン所見方針ER の動き
背側脱臼・単純 整復可、整復後安定 整復 → buddy taping or 軽度屈曲位副子 ER 整復 → 翌日整形外来
背側脱臼 ＋ 大骨片 関節面 ≥ 1/3、整復後不安定 手術（経皮ピンニング／extension block） 整形相談
整復不能 volar plate 嵌頓・骨片嵌頓 観血的整復 徒手整復のご相談 → 観血的整復
掌側脱臼 中央索損傷、boutonnière 変形 手術 or PIP 伸展位固定 6 週 整形相談
開放創／血管・神経障害 創あり／拍動・感覚低下 緊急対応 洗浄・破傷風・抗菌薬・即コンサル

ER で打つ手（背側脱臼の整復）

  • 指神経ブロック（リドカイン）で除痛
  • 軽度過伸展しつつ縦軸牽引、中節骨基部を遠位掌側へ押し戻す
  • 整復後：自動・他動可動と関節安定性を確認
  • 整復後 Xp で concentric reduction を確認
  • buddy taping または PIP 軽度屈曲位副子（伸展位は中央索断裂時のみ）
  • 2〜3 回試みて整復不能なら観血的整復を整形に相談
  • 整復前後で固有指神経・指動脈を必ず再評価

関連ハブ・隣接疾患
  • 主訴入口：手関節・手の外傷
  • 隣接疾患：Mallet / Jersey finger ／ 中手骨頸部骨折 ／ ゲームキーパー母指
  • 処置：指神経ブロック ／ buddy taping ／ アルミ副子
  • 合併症対応：boutonnière 変形は中央索断裂、伸展位 6 週固定
