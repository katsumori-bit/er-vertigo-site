肘関節脱臼

⚡ 即時アクション
  1. 血管神経評価（橈骨動脈・正中/橈骨/尺骨神経）を整復前に必ず
  2. AP + lat X 線で脱臼方向と骨折合併を確認
  3. 整復後の再評価必須（神経損傷確認）
  4. 骨折合併（橈骨頭、内側上顆、冠状突起）に注意
📊疫学・受傷機転 🩺診断 🔧整復手技 🔍整復後の評価 ⚠️合併症 🩹整復後の固定 📤disposition 🏠帰宅時の説明 📚出典
📞 この疾患が該当する主訴
  • 手関節・手の外傷 — 上肢外傷の入口
  • 転倒 — FOOSH（Fallen On OutStretched Hand）が典型

📊 疫学・受傷機転

  • 全脱臼の 20%（肩関節脱臼の次に多い）
  • 10–20 代男性、50% がスポーツ外傷
  • 受傷機転の多くが FOOSH（Fallen On OutStretched Hand）
  • 多くは後方脱臼、前方脱臼は稀
肘関節脱臼 = 全脱臼の 20%、肩関節脱臼に次いで多い
  • ほぼ全例が後方脱臼（10–20 代男性、スポーツで FOOSH）
  • 側面像で明確に診断可能 → 迷わず整復
  • 経静脈麻酔下で整復、その後に神経所見を評価
  • 脱臼骨折では緊急手術の可能性

🩺 診断

  • 激痛で身体所見は困難 → レントゲン検査へ直行
  • 肘の 側面像で一目瞭然（明確脱臼タイプ）
  • 脱臼骨折の有無を整復前に評価

🔧 整復手技

  1. 経静脈麻酔（ketofol など）で十分な鎮静・筋弛緩
  2. 第 1 助手が 上腕にカウンタートラクション
  3. 術者が 前腕を牽引しつつ屈曲
  4. うまくいかなければ屈曲位での整復を試す
力技は禁物

筋肉質の若年男性が多く、経静脈麻酔下でないと整復困難。1 人でやろうとせず、最初は経験者と。

🔍 整復後の評価

神経所見（全例で実施）

  • 正中神経：母指・示指掌側感覚、OK サイン（前骨間神経）
  • 尺骨神経：小指・環指掌側感覚、指のかぎ運動
  • 橈骨神経：手関節背屈、母指伸展
  • 異常があれば紹介状に記載

血管所見

  • 橈骨動脈触知
  • 末梢循環、Capillary Refill Time

レントゲン（整復前後）

  • 鉤状突起骨折（10–15% に合併）：非整形外科医でも整復トライ可
  • 関節面にかかる脱臼骨折：ワントライで整復できなければ整形外科へ依頼

⚠️ 合併症

合併症頻度・対応
鉤状突起骨折10–15%、通常は一般脱臼と同じ整復で OK
脱臼骨折（関節面）観血的整復 → 即時整形
正中・尺骨・橈骨神経損傷紹介状に記載、多くは自然回復
上腕動脈損傷（稀）緊急血管評価
靱帯損傷（MCL など）後日整形外来で評価

🩹 整復後の固定

  • 肘関節 90° 屈曲位
  • 上腕〜手関節までの 後方シーネ + 三角巾
  • 肘外傷すべてこの固定で OK

📤 Disposition

  • 単純脱臼 → 帰宅保存、整復前後のレントゲン + 紹介状で後日整形外科外来
  • 脱臼骨折 → 骨折の大きさにより判断、即時整形の場合あり

🏠 帰宅時の説明

  • 固定は 2–3 週間
  • しびれ・蒼白・強い疼痛 → 即受診
  • 整形外科外来で合併症（靱帯損傷）の再評価
