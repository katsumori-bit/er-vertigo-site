橈骨頭骨折

同時並行で評価する 3 軸
典型像
  • FOOSH（手をついて転倒）が最多
  • 所見：肘外側の圧痛＋腫脹、肘の 回内外で激痛
  • 能動的な肘屈伸・回内外の制限
  • 合併：肘関節脱臼／鉤状突起骨折＝terrible triad
  • 遠位橈尺関節（DRUJ）痛 ＝ Essex-Lopresti 損傷を疑う
血管

血管障害は通常まれだが、念のため触知記録を。

  • 橈骨動脈・尺骨動脈触知（左右比較）
  • CRT・色調・温度を確認
  • 腫脹強い・転位大 → コンパートメント 6P チェック
神経

神経障害は通常まれ。後骨間神経（PIN）麻痺に注意。

  • PIN（後骨間神経）：手指の MP 伸展（drop finger）
  • 正中神経・尺骨神経：通常問題なし、念のため確認
  • 所見記録（責任分界）
画像の見方
  • 肘 2 方向（正面・側面）＋ 必要に応じて橈骨頭斜位
  • 側面像で必ず確認：
    • fat pad sign（前方は sail sign、後方は陽性で関節内出血を示唆）→ 不顕性骨折を疑う
    • 橈骨頭の段差（2 mm 以上の関節面段差は要手術）
    • 骨片数、転位の方向
  • 正面像：橈骨頭の輪郭、合併骨折（鉤状突起・肘頭）
  • 転位／粉砕／terrible triad 疑い → CT（手術計画用）
  • 手関節痛があれば 手関節 Xp（Essex-Lopresti を見落とさない）
  • fat pad sign のみ陽性で骨折線不明 → 不顕性橈骨頭骨折として保存治療＋再診
治療方針（Mason 分類 × 5 択）
分類（Mason）所見マネジメントER の動き
Type I：転位なし 転位 < 2 mm、関節可動 OK 保存（早期可動） 三角巾＋早期 ROM、整形外来
Type II：転位＋部分 転位 ≥ 2 mm、骨片あり 整形コンサル（手術検討） シーネ＋整形相談、CT
Type III：粉砕 橈骨頭の粉砕骨折 手術（ORIF / 人工頭） CT、絶飲食、整形コンサル
Type IV：脱臼合併 肘脱臼＋橈骨頭骨折 緊急整復＋手術 整復、CT、即コンサル
Essex-Lopresti 橈骨頭骨折＋DRUJ 損傷 手術＋骨間膜評価 手関節 Xp、整形コンサル

ER で打つ手

  • Type I：三角巾固定のみで早期可動域訓練が原則（拘縮予防）
  • Type II 以上：後方シーネ（肘 90° 屈曲・前腕中間位）＋ 三角巾
  • 関節血腫の穿刺＋局所麻酔は疼痛軽減に有効（整形外科判断）
  • 転位大・粉砕・脱臼合併は CT、絶飲食、ライン確保
  • 鎮痛はアセトアミノフェン中心。神経ブロックは整形医到着前は避ける

関連ハブ・隣接疾患
  • 主訴入口：肘の外傷・痛み ／ 転倒
  • 隣接疾患：肘関節脱臼 ／ 肘頭骨折 ／ 上腕骨顆上骨折
  • 合併損傷：terrible triad、Essex-Lopresti、開放骨折
  • 保存例：早期可動が原則（拘縮予防）、翌日〜数日で整形外来
