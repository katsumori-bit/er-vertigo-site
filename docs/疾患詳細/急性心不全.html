🚨 安定化と 3 軸評価の枠組み

急性心不全の入口
  • 血行動態と呼吸状態の安定化が常に最優先。ショックならカテコラミン・補助循環（IABP / VA-ECMO（PCPS） / IMPELLA）を検討。
  • 安定化と同時並行で 3 軸（① 血行動態 ② 基礎心疾患 ③ 増悪因子）を評価する。
  • CS 分類で止まらない。「主病態は何か」を必ず言語化してから薬を選ぶ。
  • 治療が効かないときは見逃しを疑う。3 軸を最初から見直す。

ベッドサイド NS 判定の手順

  1. うっ血の有無を聞いて触る：起坐呼吸・夜間発作性呼吸困難（肺うっ血）／体重増加・下腿浮腫・頸静脈怒張（体うっ血）あり＝ Wet、なし＝ Dry。
  2. 組織低灌流の有無を見る：脈圧縮小・四肢冷感・意識レベル低下なし＝ Warm、あり＝ Cold（LOS：low output syndrome、いわゆる「ロス」）。
  3. 4 Profile（A / B / C / L）に振り分け、主病態を決める。
モルヒネは原則使わない

ESC 2021 で Class III（用いない）。急性肺水腫でも原則使わない。

🩺 第 1 軸：血行動態（NS 分類）

軸の取り方

  • 横軸（うっ血）：起坐呼吸・PND（肺うっ血）／体重増加・下腿浮腫（体うっ血）あり＝ Wet、なし＝ Dry。
  • 縦軸（組織低灌流）：脈圧縮小・四肢冷感・意識レベル低下なし＝ Warm、あり＝ Cold（LOS）。

4 プロファイル

B（最頻） Wet & Warm 起坐呼吸・湿性ラ音・末梢温 利尿薬 + 血管拡張薬 + NPPV で多くは改善（→ CS1／CS2）

ベッドサイドで参考にする所見

  • IVC 径：絶対値で判断しない（慢性 AF / MR / TR、COPD で普段から太い人がいる）。経時変化で見る。詳細は §5 検査。
  • 乳酸（lactate）：Cold の客観的指標。改善傾向を経時的に追う。
  • 脈圧（収縮期 - 拡張期）：縮小は心拍出低下のサイン。
効かないときは戻る

NS 分類どおりに治療しても改善しないときは見逃しを疑い、第 2 軸（CHAMPIT）と第 3 軸（FAILURE）を再評価する。

🚑 第 2 軸：基礎心疾患（CHAMPIT）

急性心不全ではまず ACS を除外する

30 分前まで会話できていた症例が急変することもある。問診・繰り返しの 12 誘導心電図・採血（トロポニン）・心エコーは必須。

心エコーの実践ポイント

  • visual EF：目の訓練を反復。
  • 局所壁運動：周囲の正常心筋に引きずられて動いて見える tethering 現象に注意。「壁運動異常なし」と言い切るのは難しい。
  • IVC 径：絶対値で評価しない。慢性 AF / MR / TR、COPD で普段から拡大している人がいる。経時的な変化で見る。
トビラの向こう（発展）
  • E/A・DcT で左房圧推定：① 正常型／② 弛緩障害型／③ 偽正常型／④ 拘束型。心不全増悪で ② → ③ → ④ に進み、改善で逆方向。治療効果判定にも使える。
  • 高齢者は弛緩障害型でも左房圧は正常のことが多い。

📊 CS 分類と主病態の連結

CS は「入口」、主病態が「治療」を決める
  • CS 分類は院前〜初期評価向けの簡易ツール。SBP で 5 群に分ける。
  • CS 1〜3 は NS 分類の主病態と対応する：CS1 ≒ 肺うっ血／CS2 ≒ 体うっ血／CS3 ≒ 組織低灌流。
  • 実臨床では CS1 と CS2 が混合することが普通。CS で止めず、必ず主病態・基礎疾患・増悪因子まで進める。

混合例の対処

CS1 ＋ CS2 が混合するときは、肺うっ血で苦しいなら血管拡張薬＋ NPPV を優先し、体うっ血の体重・浮腫にはループ利尿薬を併用。「血圧」と「浮腫」を別物として治療する。

💨 CS1 治療 — 肺うっ血（NPPV ＋ 血管拡張薬）

主病態

  • 起坐呼吸・労作時呼吸困難。後負荷上昇による肺うっ血が主病態（体液再分布型）。
  • SBP > 140 mmHg、急性発症が典型。

初期対応の流れ

  1. 酸素投与＋ニトロ舌下スプレー（ミオコール 0.3 mg、息止めて 1〜2 回噴霧）で開始。
  2. 反応乏しければ早期に NPPV：CPAP で開始、高 CO₂ や呼吸努力残存なら BiPAP へ。NPPV は「つなぎ」、長引かせず必要なら挿管。
  3. 静脈ルート確保後に血管拡張薬を持続静注。

血管拡張薬（持続静注）

薬剤開始量／調整
ニトログリセリン（ミリスロール）0.05〜0.1 γで開始、初期は 10 分おきに 0.1〜0.2 γ調整
ニトロプルシド（ニトプロ）0.1〜0.2 mg/kg/hr
カルペリチド（ハンプ）0.0125〜0.025 γで開始、血圧・尿量で調整
ニコランジル（シグマート）0.05〜0.2 mg/kg/hr
血圧低下に注意するシチュエーション
  • EF 低下例・重症 AS 併存例では血管拡張薬で容易にショックに陥る。少量から、頻回に血圧を見る。
  • 下腿浮腫や体重増加があれば利尿薬を併用（CS1 ＋ CS2 の混合）。
硝酸薬のエビデンス（内科専攻医）

最近の RCT で肺水腫への有効性は確立せず、利尿薬が Class I・硝酸薬は Class IIb。ただし RCT は緩徐発症や超高齢で偏りがあり、急性発症で SBP 160〜180 台の CS1 らしい例には硝酸薬の使用は妥当と考えてよい。

💧 CS2 治療 — 体うっ血（ループ利尿薬を 60 分以内に）

主病態

  • 主病態は水分貯留（体液貯留型）。下腿浮腫・体重増加・頸静脈怒張・腹水。
  • SBP 100〜140 mmHg、亜急性増悪が典型。

段階的治療

  1. 救急搬送から 60 分以内のループ利尿薬静注：フロセミド静注で開始。
  2. 1〜2 時間以内に反応尿（1〜2 mL/kg/hr）をチェック。なければ倍量で追加。
  3. それでも乏しければ：
    • フロセミド持続静注（ボーラス量の 1/6 時間量目処、例：80 mg ボーラス → 10 mg/hr）
    • カルペリチド（ハンプ）追加
    • トルバプタン 3.75 mg／サイアザイド併用
    • 無効なら体外限外濾過（ECUM）を視野に、本人・家族にあらかじめ IC を入れておく

電解質モニタ

  • ループ利尿薬 → 低 K・低 Mg・代謝性アルカローシス
  • トルバプタン → 高 Na（口渇・自由飲水を確保）
  • 採血を逐次フォロー（少なくとも 1 日 1 回、不安定例は数時間ごと）
ピットフォール
  • 反応尿が乏しいまま様子見＝最悪のパターン。早めに段階を進める。
  • 退院前のうっ血残存は再入院リスク。体重・頸静脈・浮腫を必ず再評価。

🥶 CS3 治療 — 組織低灌流・低心拍出

主病態

  • SBP < 100 mmHg、四肢冷感・脈圧縮小・意識レベル低下・乳酸上昇。
  • NS 分類の Cold（C / L）に対応。

使い分け

状況方針
血圧に余裕あり血管拡張薬で後負荷を下げる
血圧が保てない強心薬（ドブタミン・ミルリノン）を追加
ショック（心原性ショック）カテコラミン＋補助循環（IABP / VA-ECMO（PCPS） / IMPELLA）— 詳細は心原性ショック章へ
Cold-Wet（NS C）は最も予後が悪い

低血圧＋冷たい＋意識障害を見たら ICU・上級医を即時に呼ぶ。心エコーで弁膜症（急性 MR・急性 AR）・タンポナーデを必ず除外。

L（Dry & Cold）の落とし穴

  • うっ血がない Cold は脱水・出血・敗血症の合併も疑う。強心薬の前に原因検索。
  • RV 梗塞・PE・タンポナーデなど右心系の問題が混じることも（→ CS5）。

❤️‍🔥 CS4 治療 — ACS パス

CS4 ＝ ACS が原因の急性心不全
  • ST 上昇 / 新規左脚ブロック / 後壁梗塞所見を見たら、急性心不全の治療を保留してでも緊急再灌流を優先。
  • 循環器・カテラボに即コール。door-to-balloon 90 分。
  • IABP / IMPELLA / VA-ECMO の適応はカテラボと同時並行で判断。

関連

  • 胸痛 — ACS のフロー

循環器ホットラインに伝えること

  • 年齢・性別・主訴
  • バイタル（特に 血圧と SpO₂）
  • 心電図の所見（ST 上昇の誘導・新規 LBBB ／ 不整脈 ／ 伝導障害）
  • トロポニンの値・時間
  • 胸部 X 線の所見（肺水腫の程度）
  • 現在の状況（酸素投与量、ルート確保済み、バルーン留置済み、CT 撮影済み、エコー準備済み）
  • 既往歴（心不全・弁膜症・虚血性心疾患・ペースメーカ・抗血栓薬）

🫁 CS5 治療 — 右心不全

原因を 3 つに絞って探す

原因所見・対応
急性 PE突然発症の呼吸困難・低酸素・右室拡大／McConnell ／ D ダイマー上昇／CTPA で診断（→ 肺塞栓）
右室梗塞下壁梗塞＋右側胸部誘導 ST 上昇／硝酸薬・利尿薬で容易にショック／輸液負荷で対応
肺高血圧（慢性増悪）既存の COPD / ILD / PAH ／心エコーで右室拡大・D-shape／専門施設へ
右心不全に「左心の感覚」を持ち込まない

左心不全のつもりで利尿薬・硝酸薬を急いで使うと前負荷が抜けてショックになる。右室は前負荷依存。輸液で前負荷を確保しつつ、原因（PE なら抗凝固・血栓溶解、RV 梗塞なら再灌流）を治す。

🏥 入院時オーダー・病棟指示の枠（ADHF）

救急後の入院オーダーは循環器主治医・HF チームが主導する。ESC 2021／HFA 科学声明で繰り返される項目のチェックリストとして整理する（薬剤の具体用量は施設プロトコル）。

モニタリング

  • バイタル・SpO₂・尿量（バルーンカテで時間尿量）
  • 体重・出入量（うっ血管理）
  • 心電図モニター（不整脈・虚血）
  • 電解質・腎機能・BNP / NT-proBNP（日数に応じて）

うっ血（Wet & Warm が典型）

  • 利尿薬（ループ）— 電解質・腎機能・血圧とセットで調整（ESC 2021）
  • 血管拡張薬：高血圧うっ血で血圧が許容なら検討（血圧低下に注意）

低灌流・心原性ショック寄り（Cold）

  • 強心薬・血管収縮薬— ICU 管理が多い
  • 機械的循環補助の評価は専門医（ESC 2021・ACVC 2022）

全例で意識すること

  • 誘因の評価（CHAMPIT・FAILURE）：感染・ACS・不整脈・PE 等
  • GDMT の再開・導入：退院前にエビデンスのある内服を可能な範囲で開始（ESC 2021 Class I）
  • 患者教育：体重・症状悪化時の対応、塩分、服薬

退院前（病棟）

  • うっ血の残存がないか再評価（残存うっ血は再入院リスク）
  • 口服薬の処方を確定（ACEI / ARB / ARNI、β遮断薬、MRA、SGLT2 阻害薬など、禁忌と血圧・腎機能で調整）

🚪 外来フォロー・退院後（ADHF）

  • 退院後早期訪問：ESC 2021 では、ADHF 入院後退院後 1〜2 週以内の早期フォローを Class I で推奨（うっ血の再評価、薬剤の漸増・不耐性の確認）
  • 多職種・HF クリニック：可能なら専門外来でGDMT の段階的増量（HFA 2023 科学声明でも退院前後の一貫した最適化を強調）
  • 自己管理：日々の体重、症状悪化時の行動プラン

⚠️ やりがちなミス

  • CS 分類で止まる：CS は入口に過ぎない。「主病態」「基礎疾患」「増悪因子」を必ず言語化する。入院後何日も CS で止まっているプレゼンは病態把握不足のサイン。
  • ACS を除外せずに利尿薬：CS4 を CS1/2 として治療すると致命的。心電図と心エコーを繰り返す。
  • 重症 AS に利尿薬・血管拡張薬を多量投与 → 容易にショック。心エコーで AS を疑った時点でブレーキ。
  • 急性 AR に IABP 単独使用は禁忌。心臓外科コンサルト。
  • RV 梗塞・PE に硝酸薬／利尿薬を急いで使う → 前負荷が抜けてショック（→ CS5）。
  • IVC 径を絶対値で判断：慢性 AF / MR / TR、COPD で普段から太い人がいる。経時変化で見る。
  • tethering を見落として「壁運動異常なし」と言い切る。周囲の正常心筋に引きずられる動きを意識。
  • 反応尿が乏しいまま様子見：60 分以内のループ利尿薬と 1〜2 時間以内のチェックを徹底。
  • 増悪因子（FAILURE）を毎回問わない：怠薬・感染・不整脈は再発の引き金。
  • モルヒネを使う：ESC 2021 で Class III。原則使わない。
  • 退院前のうっ血残存を放置：再入院リスク。体重・頸静脈・浮腫を必ず確認。
  • GDMT 導入を退院後に先送り：ESC 2021 Class I。可能な限り退院前に開始。
