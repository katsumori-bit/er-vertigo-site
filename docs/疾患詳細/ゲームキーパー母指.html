ゲームキーパー母指（skier's thumb）

同時並行で評価する 3 軸
典型像
  • スキー（ストック）・転倒で母指が橈側に強制外転
  • 母指 MP 関節尺側の腫脹・圧痛
  • つまみ動作（key pinch）で疼痛・力低下
  • UCL（尺側側副靱帯）損傷／裂離骨折
  • 慢性型がゲームキーパー母指、急性型が skier's thumb
血管

血管損傷は通常合併しない。母指固有指動脈の触知/拍動感を左右で比較。

  • 橈骨動脈触知（手関節掌側）
  • 母指固有指動脈の拍動・色調・CRT
  • 強い腫脹は挙上・冷却
  • 創あれば洗浄・破傷風
神経

神経損傷は通常 OK。母指固有指神経の感覚を独立して確認。

  • 母指掌側の感覚（正中神経・固有指神経）
  • 母指対立・外転（短母指外転筋・正中神経）
  • 第1背側骨間筋筋力（尺骨神経）
  • 麻痺所見は記録
画像の見方
  • 母指 AP・lateral・oblique（3 方向）
  • Xp で必ず追うポイント：
    • 近位指節骨基部尺側の裂離骨片
    • 骨片の転位・回転（> 2 mm 転位は手術考慮）
    • MP 関節の亜脱臼・拡大
  • Stress view（橈側ストレス）：
    • 30°屈曲位で橈側開大 ≥ 30°、または健側比 ＋ 15°以上 → 完全断裂
    • 痛みで実施困難なら局所麻酔下で
    • 骨片あればまず Xp 評価で十分
  • 骨片陰性の靱帯損傷は MRI / 超音波で評価（Stener lesion 検出）
  • Stener lesion＝断裂した UCL が母指内転筋腱膜に乗り上げて自然修復不能 → 手術適応
治療方針（部分損傷 vs 完全断裂 / Stener）
パターン所見方針ER の動き
UCL 部分損傷 橈側開大 < 30°、明確な end point あり 保存（母指スパイカ 4〜6 週） シーネ ＋ 翌日整形
裂離骨折・小骨片・無転位 骨片 < 2 mm 転位 保存 シーネ ＋ 翌日整形
裂離骨折・大骨片／転位 関節面 ≥ 1/3、≥ 2 mm 転位、回転 手術（スクリュー／アンカー） 整形相談
UCL 完全断裂・Stener lesion 橈側開大 ≥ 30°／end point 消失／MRI で Stener 手術（直視下修復） 整形相談
開放創／血管・神経障害 創あり／拍動・感覚低下 緊急対応 洗浄・破傷風・抗菌薬・即コンサル

ER で打つ手

  • 母指スパイカシーネ（母指軽度屈曲位、IP 関節は露出）
  • 三角巾装着、患肢挙上指導
  • ストレステストは疼痛で困難なら無理せず → MRI / 超音波を整形外来で
  • 「Stener lesion なら手術」を本人にも説明
  • 鎮痛は NSAIDs／アセトアミノフェン
  • 翌日〜1 週以内に整形外科外来へ

  • 主訴入口：手関節・手の外傷
