肘頭骨折

同時並行で評価する 3 軸
典型像
  • 肘頭への直接打撲（転倒で肘を打った）
  • または上腕三頭筋の急激な牽引で間接受傷
  • 所見：肘頭部の腫脹＋圧痛、皮下出血
  • 能動的肘伸展不能（転位例で典型）
  • 触診で骨片の段差／陥凹
  • 肘頭は皮膚直下のため開放骨折化しやすい → 皮膚を必ず観察
血管

血管障害は通常まれ。ただし高エネ・転位大は循環確認を。

  • 橈骨動脈・尺骨動脈触知（左右比較）
  • CRT・色調・温度を確認
  • 転位大／開放例 → コンパートメント 6P チェック
神経

尺骨神経が肘頭直近を走る → 必ず評価。約 10% に併発。

  • 尺骨神経：小指外転、小指掌側知覚（最重要）
  • 正中神経・橈骨神経：通常問題なし、念のため確認
  • 尺骨神経麻痺あり → 整復前後で記録（責任分界）、緊急手術の可能性
画像の見方
  • 肘 2 方向（正面・側面）。側面像が診断のキー
  • 側面像で確認：
    • 肘頭の骨折線・転位の程度（2 mm 以上の関節面段差は要手術）
    • posterior fat pad sign（不顕性骨折のサイン）
    • 粉砕の有無（手術選択に影響）
  • 正面像：橈骨頭骨折・鉤状突起骨折・脱臼の合併に注意
  • 関節内粉砕・転位大 → CT（手術計画用）
  • 合併損傷チェック：肘関節脱臼、橈骨頭骨折
治療方針（Mayo 分類 × 5 択）
分類（Mayo）所見マネジメントER の動き
Type I：転位なし 転位 < 2 mm、伸展可能 保存（シーネ） シーネ＋三角巾、翌日整形外来
Type II：転位＋安定 転位 ≥ 2 mm、肘関節安定 観血的整復＋固定 シーネ＋整形コンサル
Type III：転位＋不安定 脱臼合併、不安定 緊急手術 整復＋シーネ、即コンサル
尺骨神経麻痺合併 尺骨神経領域の麻痺 整形コンサル 所見記録、整形に早期相談
開放骨折 創あり 緊急手術 洗浄・抗菌薬・破傷風

ER で打つ手

  • 後方シーネ固定（肘 60–90° 屈曲）＋ 三角巾。患肢挙上、クーリング
  • 能動的伸展可否を必ず記録（転位の程度を反映）
  • 尺骨神経所見を必ず記載（小指の感覚・運動）
  • 転位大・関節内粉砕は CT、絶飲食、ライン確保
  • 鎮痛はアセトアミノフェン中心。神経ブロックは整形医到着前は避ける

関連ハブ・隣接疾患
  • 主訴入口：肘の外傷・痛み ／ 転倒
  • 隣接疾患：肘関節脱臼 ／ 橈骨頭骨折 ／ 上腕骨顆上骨折
  • 合併症対応：開放骨折 ／ 尺骨神経麻痺フォロー
  • 保存例：シーネ → 翌日整形外来、可動域訓練は早期開始（拘縮予防）
