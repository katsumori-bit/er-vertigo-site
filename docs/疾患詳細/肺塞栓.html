⚡ 即時アクション
  1. ショック合併（hypotension・心停止寸前） → massive PE として即座に対応 → 全身性血栓溶解療法 or IVR / 外科的血栓除去を循環器・心臓血管外科にコール
  2. 突然発症の呼吸困難＋頻脈＋低酸素 → Wells / 改訂 Geneva スコアで前確率を評価 → 中・高確率はそのまま画像へ
  3. 高確率群は D-dimer をスキップして造影 CT 直行（D-dimer は低・中等度確率の除外目的でのみ有用、年齢調整カットオフを使う）
  4. リスク因子の確認：術後・長期臥床・担癌・妊娠/産褥・経口避妊薬／ホルモン補充・DVT 既往・遺伝性血栓性素因・長距離移動・肥満

📞 この疾患が該当する主訴
  • 呼吸困難 — 突然発症、頻脈・低酸素を伴う
  • 胸痛 — 胸膜痛として（梗塞性）／massive では中心性胸痛
  • 意識消失 — 失神は massive PE のサインになりうる（右心拍出低下）

📋 前提

このページの使い方
  1. 呼吸困難 や 胸痛 から PE を疑った時点で開く
  2. まず Wells / 改訂 Geneva スコアで前確率を決め、D-dimer の使い方を判断する
  3. ショックがあれば D-dimer に時間を使わず、bedside POCUS と造影 CT に直行
  4. 診断確定後の重症度分類（massive / submassive / low-risk）で治療強度を決める

呼吸困難 のフローに沿って以下が済んでいる想定：

  • バイタル・モニター装着、SpO2、呼吸数
  • 血液検査（血ガス・トロポニン・BNP・D-dimer の検討）
  • 12 誘導心電図（S1Q3T3、洞性頻脈、新規右脚ブロック、V1-V3 陰性 T などの右心負荷所見）
  • 胸部 X 線（多くは非特異的、Westermark 徴候や Hampton's hump はまれ）
PE は X 線も聴診も正常なことが多い

頻脈・低酸素の原因が他で説明できないとき、リスク因子の確認とスコア評価を機械的に通すのが最大の予防策。

PERC ルール（外来低リスクで D-dimer も省略可）

PERC で全否定できれば PE はかなり低い

外来・低リスク群（ゲシュタルトで <15%）で 8 項目が 全て陰性なら D-dimer も省略してよい：

  • 50 歳未満
  • HR < 100
  • SpO2 ≥ 95%（室内気）
  • 片側下腿腫脹なし
  • 喀血なし
  • 4 週以内の外傷・手術なし
  • DVT/PE 既往なし
  • エストロゲン使用なし
高齢・担癌・術後はスコアが当てにならない

D-dimer は偽陽性が多くなるので、こういう群ではスコアより画像の閾値を低くする。

🔬 検査の流れ

D-dimer

  • 低・中等度確率の除外に使う（高感度・低特異度）
  • 50 歳未満：500 ng/mL 未満で除外
  • 50 歳以上：年齢 × 10 ng/mL をカットオフに（年齢調整 D-dimer、ESC 2019）
  • 高確率群は測定する意味が薄い（陰性でも除外できない）→ 直接造影 CT
  • YEARS アルゴリズムなどでさらに省略する報告もあるが、迷ったら撮る

造影 CT 肺動脈撮影（CTPA） — gold standard

  • 感度・特異度ともに高く、肺動脈の血栓を直接描出
  • 右心負荷所見（RV/LV 比 ≥ 1.0、IVS 偏位）が予後評価にも使える
  • 禁忌：造影剤アレルギー、重度腎機能障害、妊娠（胎児被ばくは V/Q より少ないとも）

肺換気血流シンチ（V/Q スキャン）

  • 造影禁忌（腎不全・造影剤アレルギー）の代替
  • 胸部 X 線が正常な若年・妊婦で有用
  • 解釈は「正常／低確率／中等度／高確率」（PIOPED 基準）

下肢静脈エコー（DVT 検索）

  • 近位 DVT が見つかれば PE を強く支持（近位 DVT は治療方針が PE と同等）
  • 圧迫法（CUS）：大腿静脈・膝窩静脈の圧迫不能で診断

POCUS（bedside echo）

不安定患者の bedside で当てる
  • RV 拡大（RV/LV ≥ 1）、D-shape（左室短軸像で IVS が左室側に押される）
  • McConnell sign：RV 自由壁の壁運動低下＋心尖部の動きは保たれる（PE にやや特異的）
  • IVC 拡張・呼吸性変動消失 → 右心圧上昇
  • 下肢 DVT の同時評価で診断補強
  • RV 機能正常で massive PE は考えにくい（ショックの原因が他にあるサイン）

心電図

  • 最多は洞性頻脈
  • S1Q3T3（古典的だが感度低い）
  • 右軸偏位、V1-V3 陰性 T、新規右脚ブロック
  • 右心負荷パターンは予後不良のサイン

血液マーカー（重症度評価）

  • トロポニン：陽性 → 心筋傷害 → submassive 以上
  • BNP / NT-proBNP：上昇 → RV 機能障害示唆

PESI（Pulmonary Embolism Severity Index）

  • 年齢、性別、担癌、心不全既往、慢性肺疾患、HR ≥ 110、SBP < 100、呼吸数 ≥ 30、体温 < 36℃、意識変容、SpO2 < 90%
  • 5 段階（Class I-V）：30 日死亡率を予測

初期抗凝固（全例）

状況第一選択
血行動態安定・腎機能良好DOAC：エドキサバン／リバーロキサバン／アピキサバン（後二者は単剤開始可）

担癌低分子ヘパリン（LMWH）または DOAC（消化管癌では出血リスク考慮）
妊娠LMWH（DOAC・ワーファリンは禁忌）
HIT（ヘパリン起因性血小板減少）アルガトロバン など非ヘパリン抗凝固

High-risk（massive）— 血栓溶解 / IVR / 外科

  • 全身性血栓溶解療法：モンテプラーゼ（本邦保険適用）、tPA / アルテプラーゼ
  • 禁忌：頭蓋内出血既往、3 ヶ月以内の脳卒中、活動性出血、最近の頭部外傷・手術、大動脈解離など
  • 禁忌や治療抵抗例は カテーテル血栓除去 / 局所血栓溶解（IVR） または 外科的血栓除去を循環器・心臓血管外科にコール
  • 循環不安定なら ECMO（VA-ECMO）を bridging に検討

Intermediate-high（submassive）

  • 原則は抗凝固＋ICU/HCU モニタリング
  • 代償破綻（低血圧化）したら救助的血栓溶解
  • カテーテル治療の選択肢も近年増加（適応は施設・チームの判断）

Low-risk

  • DOAC 単剤で外来 / 短期入院管理が可能なケースも
  • HESTIA 基準・sPESI 0 で外来管理候補を選別

抗凝固期間

  • 誘発性 PE（一過性リスク）：3 ヶ月
  • 特発性／持続性リスク：3 ヶ月以上、出血リスクとのバランスで延長検討
  • 担癌・遺伝性血栓性素因（高リスク）：原則無期限
  • 長期継続では DOAC 減量レジメ（リバーロキサバン 10 mg、アピキサバン 2.5 mg×2）も選択肢

下大静脈フィルター（IVC filter）

  • 適応：抗凝固禁忌の急性近位 DVT/PE、抗凝固下での再発
  • 原則 retrievable（回収可能）を入れて、抗凝固再開後に抜去
  • 長期留置の有益性は限定的

DVT の同時治療

  • 近位 DVT は PE と同レジメ
  • 遠位（下腿）DVT 単独は症例ごとに判断（症状あり・伸展リスクあれば抗凝固）
  • 離床はバイタル安定後早期に（旧来の絶対安静は推奨されない）

🏥 入院・帰宅判断

入院（ICU/HCU レベル）

  • High-risk：必ず ICU、循環器・心臓血管外科に即連絡
  • Intermediate-high：ICU/HCU、血栓溶解の準備を整えた病棟
  • 抗凝固禁忌・出血合併・腎不全・担癌活動性

一般病棟入院

  • Intermediate-low / Low-risk で社会的サポートが弱い場合
  • 初回 PE で内服管理に不慣れ

外来管理候補（HESTIA / sPESI 0）

  • 血圧維持・SpO2 良好・RV 機能正常・トロポニン陰性
  • 出血リスク低い、腎機能保たれている、内服管理可能
  • 家族・近隣のサポートがあり、急変時に来院できる
  • 強い疼痛・酸素需要・重い併存疾患がない
退院時に必ず説明すること
  • 抗凝固薬の飲み忘れ厳禁、出血徴候（黒色便・血尿・歯肉出血・打撲斑）への注意
  • 呼吸困難・胸痛・失神・片側下腿腫脹で即受診（再発・進行のサイン）
  • 誘発因子の改善（術後・臥床は早期離床、長距離移動は飲水・足首運動）
  • 1-2 週後に外来フォロー、必要に応じて専門外来（循環器・血液）へ

⚠️ やりがちなミス

PE で見落としやすいパターン
  • 胸部 X 線・聴診が正常で安心する：PE は両方正常が普通。頻脈・低酸素・原因不明でリスクがあれば疑う
  • D-dimer 陰性で必ず除外できると誤解：高確率群は陰性でも除外不可。スコアと組み合わせる
  • 失神を「迷走神経反射」で片付ける：高齢・術後・担癌の失神は massive PE / RV 不全のサインのことがある
  • ショックを敗血症・脱水と決めつける：原因不明のショック＋頻脈＋低酸素は PE を必ず鑑別に
  • 担癌患者の D-dimer を信用する：偽陽性が多く、スコアと画像で判断
  • 抗凝固開始の遅れ：診断確定前でも、確率が高ければ画像待ちの間にヘパリンを開始してよい（ESC 2019）
  • massive PE で血栓溶解禁忌を理由に手をこまねく：禁忌なら IVR・外科・ECMO を即連絡
  • 下肢の左右差を診ない：DVT の身体所見（片側腫脹・圧痛）は重要
  • 妊婦に CT を躊躇しすぎる：診断遅れは母体死亡のリスク。被ばく相談を理由に診断を引き延ばさない
  • 外来低リスクで PERC を使わない：D-dimer の偽陽性で過剰検査につながる
