月状骨周囲脱臼・月状骨脱臼

同時並行で評価する 3 軸
典型像
  • 若年男性：高エネルギー FOOSH
  • バイク事故・高所墜落・スポーツ
  • 手関節背側の強い腫脹・可動制限
  • 月状骨直上の圧痛、Watson 試験陽性
  • 「手関節捻挫」と誤診される代表格
  • PA だけでなく必ず lateral を読む
血管

純粋な動脈損傷は稀。橈骨動脈・尺骨動脈の触知と毛細血管再充満を必ず確認し、コンパートメント症候群を見逃さない。

  • 橈骨動脈・尺骨動脈触知（手首掌側）
  • 指先 capillary refill、皮膚色・冷感
  • 前腕の強い緊満・受動伸展痛 → コンパートメント疑い
  • 月状骨自体の血流障害 → 長期 Kienböck 病リスク
神経

月状骨が掌側に転位すると正中神経を圧迫する。急性手根管症候群は約 25% で合併し、緊急 release の適応。

  • 母指〜環指橈側の感覚（正中神経）
  • 母指対立・外転（短母指外転筋）
  • 夜間痛・しびれ・知覚低下は急性手根管
  • 尺骨神経・橈骨神経領域も左右比較で確認
  • 神経症状は経時的に進行 → 反復評価と記録
画像の見方
  • 手関節 Xp PA・lateral 必須。PA だけだと見逃す。
  • PA で必ず追うポイント：
    • Gilula's arc（手根骨配列の3本のなめらかな弧）のbreak
    • scapholunate gap > 3 mm（Terry Thomas sign）
    • scaphoid ring sign（舟状骨の短縮・回旋）
    • 月状骨が三角形に見える（正常は四角）
  • Lateral で病型を区別：
    • 月状骨周囲脱臼：月状骨は橈骨と整列、有頭骨が背側に脱臼
    • 月状骨脱臼：月状骨自体が掌側に転位し回転（spilled teacup sign）、有頭骨は橈骨側へ落ち込む
    • 橈骨・月状骨・有頭骨が一直線に並ぶ「co-linear」が崩れていないか
  • 合併骨折を必ずチェック：
    • 舟状骨骨折（trans-scaphoid perilunate dislocation）
    • 橈骨茎状突起、有頭骨、三角骨骨折
  • 判断困難なら早期 CT。靱帯損傷評価には MRI。
治療方針（Mayfield stage × 病型）
Mayfield病態方針ER の動き
stage I 舟状月状骨間（SL）解離 手術（靱帯修復） シーネ ＋ 整形コンサル
stage II SL 解離 + 有頭月状骨間（CL）破綻 手術 シーネ ＋ 整形コンサル
stage III SL + CL + 月状三角骨間（LT）破綻＝月状骨周囲脱臼 緊急徒手整復 → 手術 Tavernier 法 ＋ 整形即コール
stage IV 月状骨が掌側に転位＝月状骨脱臼 緊急徒手整復 → 手術（急性手根管 release 併施） Tavernier 法 ＋ 整形即コール
trans-scaphoid 型 舟状骨骨折を伴う月状骨周囲脱臼 手術（ORIF + 靱帯修復） シーネ ＋ 整形コンサル

ER で打つ手

  • Tavernier 法による徒手整復：十分な鎮静下で、まず手関節を背屈位で長軸牽引 → 続いて掌屈し、術者の母指で月状骨を背側へ押し込んで整復
  • 整復後も不安定で再脱臼しやすい → 必ずシーネ固定
  • 整復成功 / 不成功にかかわらず全例手術適応。「整復できたから帰宅」は禁忌
  • シーネは sugar tong または手関節背側シーネ（軽度掌屈位）、患肢挙上
  • 急性手根管症候群（正中神経症状）あれば緊急 release 適応 → 整形即コール
  • 鎮痛は NSAIDs / アセトアミノフェン、必要に応じて静注オピオイド
  • 原則同日入院。夜間でも整形外科に即コール

ピットフォール

  • PA だけ見て見逃す（lateral を必ず読む）
  • 「手関節捻挫」と誤診して帰宅させる
  • 正中神経症状を初診時に評価せず見落とす
  • 徒手整復成功で「治った」と判断し外来扱いにする
  • 舟状骨骨折合併（trans-scaphoid）を見逃す

関連ハブ・隣接疾患
  • 主訴入口：手関節・手の外傷 ／ 転倒
  • 隣接疾患：舟状骨骨折（trans-scaphoid 型で合併） ／ 橈骨遠位端骨折 ／ 三角骨骨折 ／ 上肢末梢神経損傷（急性手根管症候群）
  • 合併症対応：急性手根管症候群 → 緊急 release ／ Kienböck 病（月状骨無腐性壊死）は長期合併症
  • 処置：Tavernier 法による徒手整復 ／ シーネ固定 ／ 三角巾
