強い腹痛を呈する内科的疾患（Medical Diseases with Severe Abdominal Pain）

⚡ 即時アクション
  1. 血管系を CT で確認：大動脈解離 / 急性腸間膜虚血 / 脾梗塞（造影 CT・CTA で除外）
  2. 内分泌・代謝を血液で除外：DKA / 副腎クリーゼ / 高 Ca 血症 / 急性ポルフィリン症（血糖・ケトン・コルチゾール・Ca・尿 PBG）
  3. 感染・軟部組織：Fournier 壊死性筋膜炎（陰部診察）、SBP（腹水穿刺）、急性胆管炎
  4. 外科紹介前に「内科的でも外科的でもない原因」を一通り吟味：negative laparotomy を減らす一方、本来手術疾患の見落としは絶対回避

⚠️ Negative laparotomy の前提

本項の対象は「一見して手術が必要に見える腹痛だが、実は内科治療がメイン」の疾患群、いわば negative laparotomy の etiology。これらを知ることで不要な開腹を減らせる一方、生半可な知識で本来手術適応の疾患を内科疾患と誤診することは negative laparotomy よりはるかに罪が重いことを肝に銘じる。

📞 この疾患が該当する主訴
  • 腹痛 — 内科的でも外科的でもない原因の鑑別フィルタ
  • 嘔気・嘔吐 — DKA・副腎不全・急性膵炎の症状
  • 発熱 — SBP・PID・腎盂腎炎・肺炎
  • 意識障害 — 副腎クリーゼ・DKA・敗血症

🫀 1. 上部消化管穿孔を疑わせる強い心窩部痛

強い心窩部痛＋上腹部反跳痛があるが、典型的な「板状硬」とは言えないとき。

  • 急性出血性びらん性胃炎（AGML）
  • 胃アニサキス症

いずれも圧痛部位で軽い反跳痛を伴うことがあり注意。

関連リンク

  • 上部消化管穿孔（鑑別の対象）
  • 急性潰瘍・急性胃炎（AGML）
  • アニサキス症

🫁 2. 急性胆嚢炎を疑わせる右季肋部痛

右季肋部に圧痛、右側胸部に叩打痛、画像で胆嚢壁が肥厚している。でも胆石はない……というとき。

強い圧痛を伴うもの

  • 急性アルコール性肝炎（急激に腫大して被膜が引き延ばされて痛む。急性ウイルス性肝炎は通常無痛だが、アルコール性は右季肋部痛を伴うことがある）
  • （痩せた女性の）右腎盂腎炎（前腹壁から腎までの距離が短いと腹痛として訴える）
  • 肝湾曲部の大腸憩室炎
  • 右肺炎・膿胸
  • 多発肝転移（急激に増大する際に痛みを伴うことがある）

痛みは強くないが「胆嚢炎のような画像」を呈するもの

  • 急性肝炎（ウイルス性／薬剤性／アルコール性）
  • SLE
  • 川崎病（gallbladder hydrops、通常手術不要）

これらは「急性胆嚢炎」ではない。罹患部位は胆嚢ではなく肝臓。肝実質と被膜の間には疎な結合織がないため、肝臓からの滲出液は肝被膜下に溜まらず、胆嚢周囲の浮腫となる。
画像で「壁の漿膜下浮腫がメインで内腔は緊満していない」なら胆嚢炎ではない。肝臓側はグリソン鞘に浮腫性肥厚を認める。

関連リンク

  • 胆石症・急性胆嚢炎（鑑別の対象）
  • 腹痛をきたす胸部疾患

🦠 3. 虫垂炎を思わせる右下腹部痛

強い圧痛・反跳痛を有するもの

  • （若年女性）PID
  • 盲腸憩室炎
  • 回盲部炎（特に Yersinia 腸炎 ＝ pseudoappendicitis）
  • 急性膵炎（炎症性浸出液が腸間膜根部に沿って回盲部へ伝わる）

内科治療メインではないが治療方針が変わるもの

  • 大腸閉塞：どこが閉塞してもラプラスの定理で盲腸が痛む

関連リンク

  • 急性虫垂炎（鑑別の対象）
  • 急性憩室炎（盲腸憩室炎）

🌐 4. 汎発性腹膜炎を呈する広範な腹痛

腹部全体的膨満＋強い自発痛＋「明らかな」反跳痛。このようなときでも内科治療が奏効する疾患がある。

  • 急性膵炎
  • 急性腸炎（細菌性腸炎、偽膜性腸炎などの重症型）
  • 特発性細菌性腹膜炎（SBP：spontaneous bacterial peritonitis）
  • 血管炎に伴う虚血性腸炎（特に SLE：lupus enteritis）
  • 急性副腎不全
急性副腎不全

強い腹痛を引き起こすことがある。とくにアジソン病や長期ステロイド服用者の休薬時に出現することがある。

関連リンク

  • 非感染性腸炎（lupus enteritis 等）
