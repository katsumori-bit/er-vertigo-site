肩鎖関節脱臼

⚡ 即時アクション
  1. Rockwood 分類（Ⅰ-Ⅵ）を意識：ER では Ⅰ-Ⅱ（保存）/ Ⅲ-Ⅴ（手術考慮） の二分でまず判断
  2. X 線：肩 AP + 必要に応じて stress view（垂下位 X 線、両側比較）。肩峰下端と鎖骨下端の距離 ≥ 13 mm で Type Ⅲ 以上
  3. 治療方針：Ⅰ-Ⅱ は保存（三角巾 ± バストバンド）／Ⅲ は議論あり（職業・趣味で判断）／Ⅳ-Ⅵ は手術
  4. 鎖骨遠位端骨折と鑑別：圧痛部位（鎖骨外側端 vs 鎖骨中央）と X 線で区別、固定具も異なる

📞 この疾患が該当する主訴
  • 肩の外傷・痛み — 鎖骨外側端の圧痛・ピアノキーサインで疑う
  • 転倒 — コンタクトスポーツや転倒で受傷、男性に多い

🦴 解剖

肩鎖関節脱臼 = レントゲン + 身体所見の合わせ技
  • コンタクトスポーツや転倒で受傷、男性に多い（男女比 5:1）
  • レントゲンの 特異度 90%、感度 41% → 画像だけでは除外不可
  • ピアノキーサイン + 肩鎖関節圧痛で疑う
  • 三角巾 ± バストバンド固定で帰宅保存／帰宅手術
  • 肩峰（肩甲骨）と鎖骨外側部の関節
  • 3 本の強固な靱帯（肩鎖靱帯・烏口鎖骨靱帯 2 本）で結合
  • 可動性はほぼなく、靱帯損傷の数と転位方向で分類

📊 Rockwood 分類（非整形外科医用の簡略版）

非整形外科医は Type I / II と III / V の 2 つに分けるだけで OK。

Type状態転位治療方針
I / II肩鎖関節は連結保たれるなし〜軽度保存
III 以上烏口鎖骨靱帯断裂、鎖骨が挙上13 mm 以上手術考慮
  • 判定は 肩関節正面像で肩峰下端と鎖骨下端の距離
  • < 13 mm → Type I / II
  • ≥ 13 mm → Type III 以上

🩺 身体所見

  • 肩鎖関節の圧痛
  • ピアノキーサイン：飛び上がった鎖骨外側端をポコポコ押せる → Type III 以上を示唆
  • 変形：鎖骨外側が皮下に盛り上がって見える（Type III 以上）

🔍 診断

  • レントゲンは感度 41% と低い → 身体所見と画像の合わせ技で診断
  • Type I / II では転位がなく画像正常 → 肩鎖関節の圧痛だけで疑い診断
  • 迷う場合はストレス撮影（重り付き撮影）が有用だが、ER では必須でない

🩹 マネジメント

固定は三角巾 ± バストバンド

  • クラビクルバンドは使わない（鎖骨骨折との違い）
  • Type I：約 1 週間固定
  • Type II：約 3 週間固定
  • Type III 以上：帰宅手術 または 帰宅保存、仕事・趣味に応じて整形外科で判断

紹介状のコツ

  • 「転位はないが、肩鎖関節に強い圧痛があり臨床的に肩鎖関節脱臼を疑いました」
  • 上肢運動に関わる趣味や職業を記載（手術適応の判断材料）

⚖️ 鎖骨骨折との違い

鎖骨骨折肩鎖関節脱臼
圧痛部位鎖骨中央 or 内側 1/3肩鎖関節（鎖骨外側端）
変形鎖骨中央の段差鎖骨外側の挙上
レントゲン明確骨折が多い転位なければ正常
固定クラビクルバンド可三角巾 ± バストバンドのみ
