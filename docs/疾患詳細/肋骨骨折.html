肋骨骨折

同時並行で評価する 3 軸
典型像
  • 受傷機転：転倒で脇腹強打／咳嗽／高齢者の軽微外傷
  • 所見：肋骨レベルの圧痛・叩打痛
  • 深呼吸・咳嗽・体動で疼痛が増悪
  • 骨片可動性（hand-on で軋音／不安定）
  • 奇異呼吸＝フレイルチェスト（連続 3 本以上 × 2 か所）
  • 高齢者では肺炎・誤嚥のリスクが急上昇
血管・臓器（高位／低位）

骨折部位で随伴損傷が変わる。本数だけでなくレベルを読み取る。

  • 第1-3 肋骨：高エネ示唆。鎖骨下動脈・大動脈・腕神経叢損傷を造影 CT で評価
  • 第4-9 肋骨：気胸・血胸・肺挫傷の合併に注意
  • 第10-12 肋骨：右＝肝損傷／左＝脾損傷／腎損傷を必ず検索（造影 CT）
  • 多発・両側骨折＋ショック → FAST／eFAST、輸血準備
神経・呼吸

疼痛と呼吸抑制が次の合併症を呼ぶ。鎮痛と呼吸ケアは一体。

  • 肋間神経痛は遷延しやすい（数週〜数か月）
  • 疼痛 → 浅呼吸 → 無気肺・肺炎の悪循環
  • SpO2、呼吸数、副雑音、胸郭運動の左右差を経時評価
  • 高齢者・基礎肺疾患では入院閾値を下げる
画像の見方
  • 胸部 X 線（PA + 側面）：気胸・血胸・縦隔・肺挫傷をスクリーニング
  • 肋骨像（斜位）：個々の骨折線評価。ただしX 線では肋骨骨折の感度は低い（骨片透過・重なり）
  • 所見の取り方
    • 第何肋骨の前方／側方／後方かをカルテに明記
    • 連続性、転位、骨片の数
    • 気胸：肺尖部の透過性、肺紋理消失ライン
    • 血胸：肋骨横隔膜角の鈍化（≧ 200-300 mL で見える）
  • CT を撮る閾値
    • 第1-3 肋骨骨折（高エネ示唆）
    • 第10-12 肋骨骨折（肝・脾・腎の同時評価）
    • 多発（4 本以上）／両側／フレイルチェスト疑い
    • X 線で気胸・血胸が疑わしい／除外したい
    • 高齢者・抗凝固薬使用・胸部痛遷延
  • X 線陰性でも臨床所見があれば臨床診断で先に進める。CT は合併損傷検索が主目的
  • 胸部エコー（eFAST）：気胸（lung sliding 消失）、心嚢液貯留
治療方針（本数・合併症で分岐）
分類所見マネジメントER の動き
単発〜3 本／合併症なし／非高齢 転位なし、SpO2 良好、胸郭運動対称 帰宅保存 鎮痛＋呼吸訓練指導、外来フォロー
4 本以上／高齢者／呼吸合併症 頻呼吸、SpO2 低下、無気肺、基礎肺疾患 入院加療 鎮痛・酸素・呼吸理学療法、誤嚥予防
フレイルチェスト 連続 3 本以上 × 2 か所、奇異呼吸 ICU 管理／人工呼吸 or 内固定検討 呼吸器外科コンサル、鎮痛硬膜外も検討
気胸・血胸合併 呼吸音減弱、X 線・CT で空気・血液 胸腔ドレナージ サイズ・症状で適応判断、輸血準備
第10-12 肋骨＋臓器損傷 肝・脾・腎損傷を造影 CT で確認 外科・泌尿器科コンサル 輸液・輸血、IVR（TAE）の準備

ER で打つ手

  • 鎮痛が最優先：アセトアミノフェン定期内服 ± NSAIDs（腎機能・出血リスクで判断）、強い疼痛にはトラマドール／オピオイド
  • 区域麻酔も検討：肋間神経ブロック／硬膜外鎮痛／傍脊椎ブロック（多発骨折・高齢者で有効）
  • 肋骨バンドはルーチンで使わない：呼吸抑制で無気肺・肺炎を誘発
  • 呼吸訓練：深呼吸・咳嗽促し、インセンティブスパイロメトリー
  • 気胸・血胸合併：胸腔ドレナージ（小径ドレーンで対応可なケースも）
  • 抗凝固薬内服中・高齢者は遅発性血胸に注意（数日後の増悪あり）
  • 退院時指導：呼吸困難・発熱・喀痰増加で再受診、肺炎予防

関連ハブ・隣接疾患
  • 主訴入口：体幹・背部の外傷 ／ 転倒
  • 隣接疾患：胸骨骨折 ／ 腰椎横突起骨折 ／ 脊椎圧迫骨折
  • 合併症対応：誤嚥性肺炎、肺炎予防、高齢者は 骨粗鬆症治療 をフォローへ
