胆石症・急性胆嚢炎（Biliary Colic & Acute Cholecystitis）

#ER #救急 #腹痛 #胆石 #胆嚢炎 #Murphy #TG18

大きな石なら胆嚢頸部に、小さな石なら胆嚢管に嵌まりこんで内圧が上昇して発症する。有症状胆石症（胆石発作）と軽症の急性胆嚢炎を初診のワンポイントで線引きするのは困難で、数時間で治まれば胆石発作、痛みが強く入院させるなら急性胆嚢炎と判断するのが実践的。

結石の成因と病態

  • 結石はコレステロール結石と色素結石に二分
  • 若年女性の多くはコレステロール結石。赤血球崩壊が促進する状態（遺伝性球状赤血球症、抗凝固薬・抗血小板薬で易出血性など）では色素結石が多い
  • 成因と症状・病態は相関しない。重要視しなくてよい

自然経過の 4 パターン

① 重症化するパターン

  1. 胆石が嵌頓して胆嚢が緊満 → 頸部で胆嚢血管が圧迫 → 静脈還流障害 → 壁浮腫。この時点では感染症ではなく「非感染性胆嚢炎」
  2. 胆汁うっ滞に腸内細菌が感染 → 「感染性胆嚢炎」。胆汁は酸性に傾き黄 → 緑/黒へ（ビリルビン → ビリベルジン）
  3. 感染で胆嚢内は事実上膿瘍。さらに内圧↑ → 動脈血流相対的不足 → 壊疽性胆嚢炎（粘膜壊死脱落、内溶液は暗赤色/黒色）
  4. 底部の壁全層壊死 → 菲薄化して胆嚢穿孔（oozing rupture）。底部の一部が緑色変色＋周囲胆汁、肉眼で穴は見えない

虫垂炎と違って胆嚢炎では汎発性腹膜炎は少ない。穿孔前から大網・結腸でシールされ、漏れた胆汁は周囲にとどまって胆嚢周囲膿瘍（peribladder abscess）になる。

② 停滞するパターン（胆嚢水腫）

感染が成立せずとどまり、嵌頓胆石が動かない／胆嚢管が炎症で内腔閉鎖したまま。胆嚢内胆汁は分泌・再吸収を繰り返し、最終的に透明白色の粘液（white bile）となる ＝ 胆嚢水腫。緊満して収縮できず、再有症状化しやすい。

③ 慢性胆嚢炎

軽い胆嚢炎を繰り返したり保存治癒した後で粘膜壊死・脱落を反復、壁線維化で硬化。極端なら萎縮胆嚢。正常胆嚢は軟らかいが慢性胆嚢炎は硬いため、肝臓側に凸面で肝を圧迫する。

④ 無石性胆嚢炎

胆嚢管閉塞の原因が胆石と断定できないとき。本当に石がない／胆泥（gel 状物質）／画像で捉えられない小結石。高齢者に多い。

病歴

① 胆石症

  • 食後 3 時間前後（胃内容が十二指腸へ移動するタイミング）に始まる心窩部〜右季肋部痛＋しばしば右背部痛
  • パターン：acute onset / continuous pain
  • 脂っこい食事の後の発症が多く、夕食後 → 夜中の当直帯受診が多い
  • 嘔吐は伴うこともあるがメインではない
  • 「石系」の特徴：腹膜刺激症状がないので七転八倒できる。発症と消失がモードを切り替えたように「すごく痛い ↔ ぜんぜん痛くない」と一気に変化する。鎮静薬使用後にすっきりした顔をしていたら、薬が効いたと思うより「胆石かな？」と疑う
  • 4F（Forty / Female / Fat / Food after = post-prandial）が古典。10〜20 代男性の胆石は典型でないので遺伝性球状赤血球症（HS）を考慮（AD 遺伝なので親の胆嚢／脾臓手術歴を聞く）

② 急性胆嚢炎

  • 発症様式は胆石発作と同じ。違いは「数時間で治まるか／遷延するか」。来院までに数日経っているなら最初から胆嚢炎と判断してよい
  • 重症化のリスクファクター：高齢者／糖尿病／易感染性（ステロイドユーザーなど）

身体所見

  • 腹部膨満は通常なく、腸雑音も重症でなければ聴取できる
  • 圧痛は心窩部〜右季肋部。壊疽性／穿孔性でなければ反跳痛・筋性防御はない
  • 背部痛は自覚していないこともあるので、右肩甲骨下角の下に手を当てて「ここも痛いですか？」と聞く
Murphy 徴候は「胆嚢の位置」を踏まえて取る
吸気時に肝・胆嚢が尾側に下りて肋骨下から触れやすくなる、を利用するのが Murphy 徴候。普段から腹部から容易に触れる位置に胆嚢があれば Murphy をチェックする意義は乏しい。
逆に右横隔膜が極端に挙上していて吸気時にも胆嚢が肋骨弓を越えてこないなら、右季肋部からの所見は陰性にしかならない → エコーで胆嚢の位置を確認しながら直上の肋骨を叩打すると所見が出る。極度の肥満では深さが届かないこともある。
内臓逆位や左側胆嚢もある。型通りでなく位置と深さを考慮して所見を取る。

軽度黄疸を「胆嚢炎だから」と片付けない — 胆嚢炎では普通は黄疸が出ない。出ているなら敗血症や Mirizzi のサインかもしれない。

初期診断と鑑別

典型例の早期診断は難しくない。鑑別は感染サインと黄疸の有無で変わる。高齢者では発熱だけが主訴のこともあり、痛みはこちらから探しに行く。

  • 痛みのみ：胃潰瘍／急性胃炎（出血性びらん）／十二指腸潰瘍
  • 感染サインあり：急性胆管炎／結腸憩室炎（肝湾曲部）／急性虫垂炎（虫垂尖端が右傍結腸溝を上行）
  • 黄疸あり：急性胆管炎／急性肝炎（アルコール性／薬剤性 — ウイルス性は通常無痛）

検査

血液

  • 胆石症でも急性胆嚢炎でも AST/ALT・ビリルビンは上昇しない（重症胆嚢炎でも炎症が局所にとどまる限り）。両者が上昇 → 胆嚢ではなく「胆管」のトラブルを示唆
急性胆嚢炎で黄疸をきたす 3 つのケース
  1. 敗血症に至っている（T-Bil < 3 程度の軽度）
  2. 穿孔性胆嚢炎（漏れた胆汁が腹膜から再吸収）
  3. Mirizzi 症候群（急性胆管炎を併発）
これに該当しない黄疸なら胆嚢炎以外の疾患を考える。

若年で多いコレステロール結石は CT で見えないため若年の胆石発作は超音波が適。CT は全体視で胆嚢炎の評価がしやすい。

画像所見の核心 — 「緊満しているか」

胆石／胆嚢炎の本質は胆嚢出口の閉鎖と内圧上昇。サイズは個人差があるので形（短軸像が正円か）が一番の指標。短軸像が正円形 → 緊満して胆石／胆嚢炎の可能性高、どの方向から見ても楕円なら緊満ではない。

胆嚢壁肥厚は炎症以外（胆嚢腺筋症、急性肝炎、SLE、川崎病など）でも生じる。「壁肥厚あり = 胆嚢炎」と短絡しない。

培養について

  • 敗血症サインがあれば血液培養提出に意義あり
  • 重症でPTGBD 時の初回胆汁培養は有益
  • 手術時の胆汁培養ルーチン提出を支持する意見はとくにない

確定診断（Tokyo Guideline 2018）

TG18 — 急性胆嚢炎の診断基準
  • A. 局所炎症所見：(1) Murphy 徴候、(2) RUQ の腫瘤・痛み・圧痛
  • B. 全身炎症所見：(1) 発熱、(2) CRP 上昇、(3) WBC 上昇
  • C. 画像所見：急性胆嚢炎に特徴的な画像所見
疑診：A 1 項目 + B 1 項目／確診：A 1 項目 + B 1 項目 + C

ただし高齢者・免疫抑制状態で炎症反応が強くないケース、胆嚢が右季肋部から触診しづらいケースはこの基準から漏れる。嵌頓結石が同定できなくても胆嚢炎の診断に必須ではない。

治療方針

① 胆石症の治療

  • 有症状胆石症は 2 年以内に 70% で再燃 → 第一選択は腹腔鏡下胆嚢摘出術
  • 耐術不能で小さな浮遊性コレステロール石なら利胆薬（ウルソデオキシコール酸）。該当例は少ない
  • 無石性胆嚢炎で薬物療法 or ドレナージで軽快した場合は胆嚢摘出術不要
  • 無症状胆石症への予防的胆嚢摘出は正当性なし

② 急性胆嚢炎の治療

3 つの治療法（および組み合わせ）：抗菌薬／PTGBD（経皮的胆嚢ドレナージ術）／胆嚢摘出術。

腹腔鏡下胆嚢摘出術のタイミング

  • かつては「炎症をおさめてから手術」だったが、現在は最初の入院時に手術が推奨（入院期間・コスト面で優れ、合併症・死亡率は変わらない）
  • 当初は「発症 72 時間以内」と提唱されたが、近年は絶対的な禁忌タイミングは消失。施設・術者の習熟度で個別判断

治療アルゴリズム（基礎状態による）

  • ASA Class I・II：早期腹腔鏡下胆嚢摘出術が第一選択
  • ASA Class III 以上：周術期死亡率 5〜27%、手術はよい適応でない → まず抗菌薬保存、反応悪ければ PTGBD。それでも改善しなければ手術考慮
  • 難症例で定型手術が安全でないと判断した場合は subtotal cholecystectomy（胆嚢管処理にこだわらず頸部近くで終了）が、合併症（出血・総胆管損傷）回避でメリット

抗菌薬

  • 胆嚢炎ははじめは感染症ではない。すべての胆嚢炎に抗菌薬が必要かは疑問
  • IDSA：感染を疑う所見（WBC > 12,500、体温 > 38.5℃、画像で炎症）または高齢／糖尿病／免疫不全で適応
  • 起因菌：E. coli 41%、Enterococcus 12%、Klebsiella 11%、Enterobacter 9%。ABPC/SBT を第一選択にする意見
  • 「胆汁移行性」：βラクタム系の多くは元々移行良好で薬剤差は大きくない。胆嚢炎では胆嚢内腔と胆管の交通が断たれているので胆汁中の抗菌薬は胆嚢内腔に届かない

痛み止め

  • NSAIDs は有効性を示す文献あり
  • 汎用されるブチルスコポラミンは「痛み止めの効果」を有していない（有効性を示す文献も乏しい）

特殊な胆嚢炎

① 急性胆嚢炎の重症型

  • 無石性胆嚢炎／壊疽性胆嚢炎／穿孔性胆嚢炎／胆嚢周囲膿瘍
  • 気腫性胆嚢炎（Emphysematous cholecystitis）：胆嚢壁内・内腔にエア。糖尿病男性に多い、Clostridium perfringens など。全胆嚢炎の 1%、死亡率 15〜25% ときわめて高い
  • Mirizzi 症候群：頸部・胆嚢管嵌頓石の炎症が総胆管を圧排して胆管狭窄 → 胆嚢炎＋胆管炎。総胆管切石は不要

② 慢性炎症によるもの

  • 胆嚢水腫（gallbladder hydrops）／萎縮胆嚢／陶器様胆嚢（porcelain gallbladder）（胆嚢癌合併率 2〜3%、type I は手術考慮）／黄色肉芽腫性胆嚢炎（胆嚢癌との鑑別が問題）

③ 臨床像は急性胆嚢炎だが病態は異なるもの

  • 胆嚢梗塞：胆嚢動脈の梗塞。胆嚢動脈と胆管動脈の支配境界（頸部）に明瞭な demarcation line。膠原病など血管炎が基礎にあるとき多い
  • 胆嚢捻転：胆嚢床がもともと狭い素地。茎が細いので腹腔鏡手術はやりやすいことも
  • 胆嚢出血：胆嚢内が凝血塊で充満。胆嚢動脈仮性動脈瘤の破裂による報告あり、IVR 可能施設では術前 TAE が有効

④ 画像で「胆嚢炎」と指摘されるが胆嚢炎ではないもの

肝炎・SLE・川崎病などで肝に炎症があると、漿液が肝被膜下ではなく胆嚢周囲に溜まる（胆嚢周囲には脂肪織がある）→ 壁肥厚に見える。圧痛が乏しく胆嚢腔の緊満が強くないなら胆嚢炎ではない。
