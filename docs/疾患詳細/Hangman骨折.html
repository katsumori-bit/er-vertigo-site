Hangman 骨折（C2 外傷性脊椎すべり症）

同時並行で評価する 3 軸
典型像
  • 高エネルギー外傷：過伸展 ＋ 軸圧
  • 交通事故（ダッシュボード／フロントガラス）
  • ダイビング・高所墜落
  • 後頸部上部の圧痛、頸部可動制限
  • 神経学的所見は意外と軽症のことが多い（脊柱管が広い）
椎骨動脈リスク

C2 椎弓根骨折は横突孔近傍で起こりうる。横突孔骨折＝BCVI（鈍的脳血管損傷）のスクリーニング適応。

  • 頸部 CTA で椎骨動脈評価
  • 横突孔骨折・転位の有無
  • 後方循環症状（めまい・複視・小脳症状）の問診
  • BCVI 陽性なら抗血栓療法を血管外科／脳外科と協議
  • 解離・閉塞 → 後方循環脳梗塞のリスク
Levine-Edwards 分類

転位量・角度形成・facet 脱臼で 4 型に分類。Type IIa は牽引禁がキモ。

  • Type I：転位 <3 mm、角度形成なし、安定
  • Type II：転位 ≥3 mm、屈曲不安定（椎間板損傷）
  • Type IIa：角度形成あり／転位わずか、牽引禁
  • Type III：両側 facet 脱臼併発、手術適応
  • 神経所見：ASIA・四肢 MMT／感覚／反射を必ず記録
画像の見方
  • 頸椎 CT（薄スライス ＋ 矢状/冠状再構成）が第一選択
  • CT で必ず追うポイント：
    • C2 両側椎弓根（pars interarticularis）の骨折線
    • 矢状断で C2 椎体の前方転位（mm）
    • 矢状断で C2–C3 角度形成（°）＝屈曲変形
    • 横突孔骨折の有無 → 椎骨動脈損傷リスク
    • C2–C3 椎間板損傷の所見（前縦靱帯／椎間板開大）
    • 合併：Jefferson 骨折（C1）／歯突起骨折（C2）／下位頸椎骨折
  • 頸部 CTA：横突孔骨折・転位 ≥1 mm／神経症状あり／高エネルギーで考慮（BCVI スクリーニング）
  • MRI：脊髄損傷・椎間板損傷・後方靭帯複合体損傷の評価
  • Xp 単純撮影は補助。CT で確定診断する
  • 分類の閾値（Levine-Edwards）：
    • 転位 <3 mm かつ角度形成なし → Type I
    • 転位 ≥3 mm → Type II
    • 角度形成あり（転位わずか） → Type IIa
    • facet 脱臼合併 → Type III
治療方針（Levine-Edwards 分類）
分類所見方針ER の動き
Type I 転位 <3 mm、角度形成なし、安定 硬性カラー or SOMI brace 6〜12 週（保存） 硬性カラー固定 ＋ 脊椎外科コンサル
Type II 転位 ≥3 mm、屈曲不安定 牽引で整復 → Halo 固定（or 手術） in-line stabilization ＋ 入院、脊椎外科コンサル
Type IIa 角度形成あり、屈曲不安定 牽引禁！ Halo 直接適用 or 手術 硬性カラー ＋ 即時脊椎外科コンサル
Type III 両側 facet 脱臼併発 手術適応（後方固定） 硬性カラー ＋ 緊急脊椎外科コンサル
椎骨動脈損傷併発 横突孔骨折 ＋ CTA で解離／閉塞 抗血栓療法（脳梗塞リスクと出血リスクを天秤） 血管外科／脳外科併診

ER で打つ手

  • 硬性カラー固定 ＋ in-line stabilizationを最優先
  • 気道：高位頸髄損傷で呼吸抑制 → 必要なら用手 in-line で挿管（jaw thrust 中心、頸部伸展回避）
  • 神経原性ショックの評価（徐脈 ＋ 低血圧）
  • Type II/IIa で角度形成例は牽引禁（角度形成が悪化する）
  • 頸部 CTA で BCVI 評価（横突孔骨折・神経症状）
  • 原則入院 → 脊椎外科で Halo or 手術判断
  • 鎮痛：アセトアミノフェン、必要なら麻薬性鎮痛（呼吸抑制注意）

関連ハブ・隣接疾患
  • 主訴入口：頸部外傷・頸部痛
  • 隣接疾患：Jefferson 骨折 ／ 歯突起骨折 ／ 椎骨動脈損傷（BCVI）
  • 処置：硬性カラー固定 ／ in-line stabilization ／ Halo 固定
  • ピットフォール：
    • Type IIa を Type II として牽引 → 角度形成悪化
    • 椎骨動脈損傷スクリーニング忘れ（横突孔骨折で必須）
    • 神経所見ありで Halo に固執 → 手術遅延
    • 「脊柱管が広いため神経症状軽微」と侮ると次の不安定化で完全麻痺
