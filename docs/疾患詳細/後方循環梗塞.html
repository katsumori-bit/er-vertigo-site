後方循環梗塞（脳血管障害めまい）

⚡ 即時アクション
  1. めまい単独で来院しうる致死的疾患。見逃しは予後を大きく左右
  2. 歩行不能 / 体幹失調 / 神経脱落 / 後頭部痛 / 抗凝固薬の 1 つでも → 中枢性疑い
  3. HINTS で 1 つでも中枢パターン or 判断困難 → 中枢扱い
  4. CT は感度低い。MRI（DWI）+ MRA を回す。早期陰性でも油断しない
  5. t-PA / 血栓回収療法の適応となる時間枠を意識

📞 この疾患が該当する主訴
  • めまい — AVS / 中枢性ノードから到達
  • 頭痛 — 後頭部痛・頸部痛合併時
  • 構音障害 — 脳幹梗塞の代表症状
  • 意識障害 — 進行例

🚨 Red flags（中枢性を疑え）

これがあれば末梢で片付けない

  • 歩行不能 / 起立不能・強い体幹失調
  • 構音障害・複視・嚥下障害・片麻痺・感覚障害
  • 垂直眼振 / 方向交代性眼振 / 固視で抑制されない眼振
  • 強い後頭部痛 / 頸部痛（解離・SAH も鑑別）
  • 新規難聴 + 中枢所見（AICA 梗塞）
  • 高齢 + 血管危険因子（HT / DM / DL / AF / 喫煙 / 既往）
  • 抗凝固薬内服（出血性卒中も）
  • 失神前状態 / 胸痛 / 動悸（心原性塞栓も）
1 つでもあれば必ず鑑別を広げる

除外すべき主な致死的疾患：後方循環梗塞（小脳 / 延髄 / 橋 / AICA）／椎骨動脈解離／くも膜下出血／急性脳出血／心原性失神 / 致死性不整脈／大動脈解離

🧠 部位別の典型像

小脳梗塞（PICA / SCA / AICA 領域）

  • めまい・体幹失調・嘔吐
  • 歩行不能・起立不能のことあり
  • めまい単独で来ることがある（特に PICA 領域）
  • 巨大梗塞では後頭蓋窩浮腫 → 脳幹圧迫 / 水頭症のリスク

脳幹梗塞（延髄・橋）

  • 多彩な脳神経症状＋四肢症状の組み合わせ
  • Wallenberg 症候群（外側延髄梗塞）：めまい / 嚥下障害 / Horner / 顔面・対側体幹温痛覚低下
  • 橋梗塞：複視・構音障害・片麻痺

AICA 梗塞

  • めまい + 急性難聴が特徴（内耳動脈領域）
  • 「難聴 = 末梢」と誤診されやすい
  • 同側の顔面麻痺・小脳症状を伴うことあり

椎骨動脈解離

  • 後頸部・後頭部痛＋めまい / 神経症状
  • 外傷歴・カイロプラクティック・スポーツ後
  • 抗凝固 / 抗血小板療法の判断には脳神経外科コンサルト

🩺 HINTS（中枢パターンの捉え方）

詳細は HINTS ページ を参照。AVS でのみ実施可。

中枢示唆（INFARCT パターン）

  • Impulse Normal（HI 陰性 = catch-up saccade なし）
  • Fast-phase Alternating（方向交代性眼振）
  • Refixation on Cover Test（Skew あり）
  • 1 つでも該当 → 中枢性として扱う
HINTS の限界

適応外（t-EVS、s-EVS、眼振なし例）には使えない。判断困難 / 手技習熟度低い場合は中枢扱いで MRI/MRAへ。

🔬 画像

頭部 CT

  • 後方循環虚血の検出感度は低い（特に発症 6h 以内）
  • 適応：出血疑い / 外傷 / 強い頭痛 / 局所神経症状
  • CT 異常なし ≠ 中枢性除外

MRI / MRA

  • DWI が後方循環梗塞検出の主軸
  • 早期 MRI でも 10-20% は陰性（特に発症 24-48h 以内の小梗塞）
  • 強い臨床疑いがあれば24-48 時間後の再撮影を検討
  • MRA で椎骨動脈狭窄・解離・脳底動脈病変を評価

その他

  • 頸動脈エコー / 経胸壁心エコー（塞栓源検索）
  • ホルター心電図（発作性心房細動検出）

💊 急性期治療

再灌流療法

  • t-PA（rt-PA）：発症 4.5h 以内、適応・除外基準を確認
  • 血栓回収療法（mechanical thrombectomy）：基底動脈閉塞などの大血管閉塞に対して時間枠は施設プロトコル次第（24h まで延長されることも）
  • 後方循環の血栓回収は前方循環よりエビデンス相対的に弱いが、適応症例では積極的に検討

抗血栓療法

  • 急性期：抗血小板薬 or 抗凝固薬（脳神経内科 / 外科の指示）
  • 椎骨動脈解離：解離治療の標準は抗血栓療法（抗凝固 vs 抗血小板の優劣はガイドライン参照）

支持療法

  • 血圧管理（過降圧避ける、適応に応じた目標値）
  • 嘔吐コントロール、補液
  • 後頭蓋窩浮腫リスク高い場合は脳神経外科コンサルト（減圧手術の閾値低めに）

📤 disposition

確定 / 強い疑い → 入院

  • SCU / 脳卒中ユニット入院
  • 脳神経内科 / 脳神経外科コンサルト
  • 後頭蓋窩巨大梗塞は集中管理（脳幹圧迫リスク）

HINTS 判断困難・MRI 早期陰性

  • 原則入院 / 経過観察。48h 後 MRI 再検
  • 歩行可能・神経所見なしなら外来フォローも条件付きで検討（独居・フォロー困難なら入院寄り）

⚠️ ピットフォール

  • 「回転性 = 末梢」決め打ち — 後方循環梗塞も回転性で来る
  • CT 正常で安心 — 後方循環虚血に弱い
  • MRI 早期陰性で除外 — 24-48h で陽性化することあり
  • 難聴を末梢証拠と扱う — AICA 梗塞でも難聴あり
  • 歩けないのに末梢診断 — 体幹失調は中枢の重要所見
  • HINTS のみで中枢除外 — 神経所見・経過・血管リスクも統合判断
  • 後頭部痛を軽視 — 椎骨動脈解離 / SAH の見逃し
