℞薬剤情報 ⊞表計算

上肢末梢神経損傷（正中・尺骨・橈骨・AIN・PIN）

同時並行で評価する 3 軸
典型像
  • 切創・刺創（ガラス・刃物）
  • 骨折・脱臼に伴う神経牽引／圧迫
  • 長時間圧迫（saturday night palsy）
  • 橈骨遠位端骨折 → AIN／正中
  • 橈骨頭骨折・Monteggia → PIN
  • 「捻挫」「打撲」で麻痺を見落としやすい
運動評価

5 神経すべてを系統的に MMT で評価し、左右差を記録。「橈骨神経麻痺」とだけ書かず、具体的な筋名と MMT を残す。

  • 正中：母指外転（APB）、対立
  • AIN：OK サイン（FPL / FDP 示指）
  • 尺骨：指外転内転（第 1 背側骨間筋）、Froment 徴候
  • 橈骨：手関節背屈
  • PIN：MP 伸展（EDC）、母指伸展（EPL）
感覚評価

触覚・痛覚・2 点識別覚を区域ごとに評価。AIN／PIN は運動枝のみで感覚なし（鑑別ポイント）。

  • 正中：母指〜環指橈側半分の掌側
  • 尺骨：小指全域、環指尺側半分
  • 橈骨：第 1 背側 web area（典型）
  • AIN / PIN：感覚なし（運動のみ）
  • Tinel 徴候：損傷部位を叩打 → 末梢方向への放散
神経マップ（5 神経の運動・感覚）

正中神経

  • 運動：母指外転（APB）、母指対立（OP）、示指・中指 FDS、橈側 2 lumbricals。AIN を介して FPL（母指 IP 屈曲）、FDP 示指（DIP 屈曲）、PQ
  • 感覚：母指〜環指橈側半分の掌側、示指〜環指橈側半分の指尖背側
  • 損傷部位：手関節（carpal tunnel／切創）／肘以遠（AIN 単独）／上腕（高位正中）
  • 評価：母指外転 MMT、母指対立、OK サイン（AIN）
  • 高位損傷では「祈祷の手（hand of benediction）」（屈曲しようとすると示指・中指が伸展位）

尺骨神経

  • 運動：内在筋（指外転内転、骨間筋、4-5 lumbricals、母指内転筋）、FDP の小指・環指、FCU
  • 感覚：小指全域、環指尺側半分（掌・背とも）
  • 損傷部位：肘（cubital tunnel／肘部管症候群）／Guyon 管（手関節）／上腕
  • 評価：指外転内転（特に第 1 背側骨間筋）、Froment 徴候（紙を母指−示指間で挟むと母指内転筋麻痺で IP 屈曲代償）、Wartenberg 徴候（小指外転位）
  • 慢性低位損傷では「鉤爪手（claw hand）」（環指・小指 MP 過伸展＋ PIP 屈曲）

橈骨神経

  • 運動：上腕三頭筋（高位）、手関節背屈（ECRL／ECRB）、PIN を介して指 MP 伸展（EDC）、母指伸展（EPL）、母指外転（APL）、ECU
  • 感覚：上腕後面、前腕背面、手背母指〜中指橈側（第 1 背側 web area が典型）
  • 損傷部位：上腕（saturday night palsy、上腕骨骨幹部骨折）／肘以遠（PIN 単独）／橈骨頭骨折
  • 評価：手関節背屈、母指伸展、第 1 背側 web 感覚
  • 高位損傷では「下垂手（wrist drop）」

前骨間神経（AIN、正中神経分枝）

  • 運動のみ（感覚なし）：FPL（母指 IP 屈曲）、FDP 示指・中指（DIP 屈曲）、PQ（円回内筋ではなく方形回内筋）
  • 評価：OK サイン（pinch test） — 母指と示指で OK を作れない、ピンチが涙状（tear drop sign）になる
  • 損傷：橈骨遠位端骨折・前腕骨折・打撲・絞扼（Kiloh-Nevin 症候群）
  • 感覚障害がないため「腱断裂」と誤診されやすい

後骨間神経（PIN、橈骨神経分枝）

  • 運動のみ（感覚なし）：MP 伸展（EDC）、母指伸展（EPL）、母指外転（APL）、ECU
  • 評価：手関節背屈は保たれる（ECRL／ECRB が橈骨神経本幹から分岐）／指 MP 伸展不可、母指伸展不可
  • 損傷：橈骨頭骨折・Monteggia 骨折・回外筋症候群（supinator syndrome）
  • 手関節背屈の橈側偏位（ECU 麻痺）も特徴

分類（Seddon / Sunderland）

  • neurapraxia（Sunderland I）：一時的伝導障害、軸索保たれる、数日〜数週で回復
  • axonotmesis（Sunderland II-IV）：軸索断裂、Wallerian 変性後 1 mm／日で再生
  • neurotmesis（Sunderland V）：神経完全断裂、自然回復なし、手術一次修復必要
治療方針（開放/閉鎖 × 損傷タイプ）
パターン所見方針ER の動き
開放創 + 神経損傷 切創・刺創部に対応する麻痺 緊急一次修復（24h 以内） 即整形・形成（手の外科）コール、洗浄・抗菌薬・破傷風
骨折・脱臼整復後の麻痺 整復前から麻痺、または整復後出現 6 週経過観察 → 改善なしで MRI/EMG/NCS シーネ・神経保護姿位、整形外来予約
長時間圧迫（saturday night palsy） 上腕骨幹部圧迫後の下垂手 保存（多くは neurapraxia） cock-up シーネ、整形外来
AIN／PIN 単独損傷（閉鎖性） 骨折・打撲後の OK サイン不可／指伸展不可 3〜6 ヶ月経過観察（自然回復多い） シーネ・整形外来
骨折／脱臼合併 橈骨遠位端・上腕骨・橈骨頭・Monteggia 骨折整復 ＋ 神経評価継続 整形コンサル・固定

ER で打つ手

  • 5 神経すべて系統的に評価（運動・感覚・Tinel）して MMT 付きでカルテ記載
  • 開放創 → 洗浄・止血・抗菌薬・破傷風 → 即整形 or 形成（手の外科）コール
  • 骨折・脱臼 → 整復前後で必ず神経評価し記録（責任分界点）
  • 神経保護姿位の副木：
    • 橈骨神経麻痺 → cock-up シーネ（手関節背屈位）
    • 尺骨神経麻痺 → MP 屈曲位（claw 予防）
    • 正中神経麻痺 → 母指外転位を保持
  • 鎮痛は NSAIDs／アセトアミノフェン（神経障害性疼痛にはプレガバリンも検討、外来で）
  • 「整復後の改善期待」は神経牽引性（軸索ストレッチ）が多いことを説明、6 週がメルクマール

ピットフォール

  • 5 神経を系統的に評価しない（特に AIN／PIN を忘れる）
  • 「橈骨神経麻痺」とだけ書いて運動・感覚・MMT を具体的に記載しない
  • 開放創を「縫合だけ」して神経評価を怠る → 後日「断裂を見逃した」事案に
  • 骨折整復後の感覚障害を全部「圧迫だろう」で経過観察にする
  • AIN／PIN は感覚障害がないため「腱断裂」と誤診
  • 整復前の神経所見を記載しないと、整復後の麻痺が「医原性」扱いになる
