神経診察（動作ベース）

このページの使い方
  • ER：頭痛・意識障害・めまいなどから飛んできたときのベッドサイド手順として使う。
  • 入院／外来：下記「入院・外来での位置づけ」を参照（脳卒中・TIA など疾患別のフォローは各疾患ノートが本体）。

ベッドサイド手順（チェックリスト）

  • 意識（呼びかけ反応）
  • 見当識（名前・場所・時間）
  • 瞳孔（左右差・対光反射）
  • 眼球運動（追視・複視）
  • 顔面運動（眉上げ・歯見せ）
  • 顔面感覚（左右差）
  • 構音（ろれつ）
  • 舌突出（偏位）
  • 上肢Barré（回内下垂）
  • 手回内回外（回内ドリフト）
  • 下肢挙上保持
  • 筋力（左右差）
  • 表在感覚（触覚・左右差）
  • 深部腱反射
  • Babinski
  • 指鼻試験
  • 踵膝試験
  • 歩行
  • Romberg

入院・外来での位置づけ

このリストは記録様式であり、オーダー内容そのものではない。入院後は疾患ごとのガイドラインに沿って指示を組み立て、神経所見は経時的に同じ項目で比較できるように記載する。

脳梗塞・TIA（例）

  • 入院初期：NIHSS（標準化スコア）を用いるのが AHA/ASA ガイドラインの流れ（本ページの細かい動作チェックは補助的）
  • オーダーの柱：再灌流療法の適応、抗血小板／抗凝固（病因に応じて）、血圧管理、血糖・体温、誤嚥評価、早期リハビリ（AHA/ASA 2019 急性期脳卒中ガイドライン）
  • 外来フォロー：二次予防（抗血小板、スタチン、血圧）、生活習慣、心房細動の有無 — AHA/ASA 2021 脳卒中・TIA 二次予防ガイドライン

てんかん・意識障害の経過観察

  • 同じ神経項目を時間帯で追う（瞳孔・片麻痺の進行など）
  • 具体的な薬剤オーダーは神経内科／脳外科のプロトコルに従う

末梢性めまいなど軽症で帰宅する例

  • 再診指示：中枢徴候が出たら受診（めまい の Red flags と整合）
