上腕骨外顆骨折

「最軽症でも後日手術が 10%、機能障害が 10%」 — 保存予定でもこの 2 つを両親に誠意を込めて伝えるのが ER 医の最大の仕事。整形外科医のシーネ内でも外顆はずれる。短期再診を必ず設定。
同時並行で評価する 3 軸
典型像
  • 小児肘骨折の 15%（上腕骨顆上骨折の次に多い）
  • 幼稚園年長〜小学生低学年（4〜10 歳）、FOOSH が大半
  • 外顆の付着筋が骨片を引っ張り、小頭は外側へ回転転位
  • 所見：肘外側の限局性圧痛・腫脹、可動時痛
  • 外見上は「顆上に比べて軽症」に見えるが、転位拡大の罠あり
血管

外顆骨折で血管損傷は稀だが、高度転位例では橈骨動脈触知を必ず確認。

  • 橈骨動脈を触知（左右比較）
  • CRT・色調・温度・SpO₂ の確認
  • 6P（Pain・Pallor・Paralysis・Paresthesia・Pressure・Pulselessness）
  • 異常があれば即整形・血管外科
神経

神経合併は顆上骨折より頻度は低いが、整復前後の所見記録は必須。

  • 正中神経：母指対立、示指掌側知覚
  • 橈骨神経：手関節背屈、第 1・2 指間背側知覚
  • 尺骨神経：小指外転、小指掌側知覚
  • 整復・固定前後の所見を必ず記録（医原性の責任分界）
画像の見方（Jakob 分類 × Salter-Harris IV）
  • 肘 2 方向（正面・側面）＋ 微妙な転位は健側比較
  • 正面像が最重要：小頭の外側転位は正面像が最も分かりやすい
  • Radiocapitellar line（橈骨中心線の延長線）：
    • 正常：小頭の中央を通過
    • 異常：延長線が小頭から外側に外れる → 外顆骨折（Type I/II の微妙骨折で有効）
    • 正常小児でも 15.6% で中央を外れる → ルーチンではなく骨折を疑った時に引く
    • 迷ったら左右比較
  • Jakob 分類：
    • Type I：完全離断なし（骨折線のみ）／転位なし
    • Type II：完全離断あり、転位なし
    • Type III：完全離断あり、転位あり（明確骨折タイプ）
  • Salter-Harris Type IV が大半：見た目は小頭のみ転位だが、実は上腕骨遠位端で関節面 → 成長板 → 骨幹端まで骨折線が達する
  • 骨端線損傷で骨が伸びなくなれば四肢長不一致のリスク → 紹介状は「Salter-Harris Type IV」表記で統一
  • CT は通常不要、整形側で術前計画にオーダーされることあり
治療方針（2 mm × Jakob ／ Salter-Harris IV）
分類所見マネジメントER の動き
Jakob Ⅰ（< 2 mm） 骨折線のみ、転位なし 帰宅保存 肘 90° 屈曲シーネ＋三角巾、短期再診必須（数日〜1 週）
Jakob Ⅱ（< 2 mm） 完全離断あり、転位なし 帰宅保存 or 整形判断 シーネ＋三角巾、整形外科外来へ早期紹介（迷ったらオーバートリアージ）
Jakob Ⅲ（≥ 2 mm） 完全離断＋転位 即時整形（手術） シーネ仮固定、絶飲食、即コンサルト
open / 神経血管症状 創あり／橈骨動脈異常／神経麻痺 緊急手術 抗菌薬・破傷風・絶飲食・即コンサルト

ER で打つ手（保存例）

  • 肘 90° 屈曲位の後方シーネ＋ 三角巾（成人脱臼・顆上骨折と同じ固定）
  • 挙上指示、指先は露出して循環確認
  • 鎮痛はアセトアミノフェン中心
  • 両親への説明（必須）：「最軽症でも 10% が後日手術、10% が機能障害」「シーネ内でもずれる」「だから短期再診」
  • 整形外科外来の日時を確定してから帰宅

ER で打つ手（即時整形例）

  • 絶飲食、ライン確保、採血・胸部 X 線・心電図（術前準備）
  • 仮固定でシーネ＋三角巾、過屈曲は避ける
  • 神経ブロックは整形医到着前は避ける（神経評価が困難になる）
  • 「右上腕骨外顆骨折（Salter-Harris Type IV）、転位 2 mm 以上ありそう」と電話で伝えると通る

関連ハブ・隣接疾患
  • 主訴入口：肘周囲の外傷 ／ 転倒
  • 隣接疾患：上腕骨顆上骨折（小児肘骨折最多） ／ Monteggia 骨折 ／ 肘内障 ／ 肘関節脱臼 ／ 橈骨頭骨折 ／ 肘頭骨折
  • 合併症：開放骨折 ／ コンパートメント症候群
  • 固定法詳細：肘周囲骨折の固定
  • 家族説明：「最軽症でも後日手術 10%・機能障害 10%」を必ず伝え、短期再診の日時を確定して帰宅
