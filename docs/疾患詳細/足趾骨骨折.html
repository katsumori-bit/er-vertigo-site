  3. 爪下血腫 50% 以上はドレナージ

足趾骨骨折 = 開放チェック + バディテープで保存
  • サンダル等で物にぶつけた・重い物の落下が典型機転
  • 必ず靴下を脱がせて開放骨折を確認（あれば即時整形）
  • 爪下血腫 50% 以上はドレナージ、爪床裂創は後日整形に処置依頼可

  • 固定は 位置にかかわらずバディテープ

📝 受傷機転 + 開放骨折チェック

受傷機転

  • サンダル等で家具・段差に強くぶつけた
  • 作業中に重い物が足趾に落下
  • 足を踏まれた・転倒で趾を捻った

開放骨折チェック（必須）

  • 靴下を必ず脱がせる（履いたままだと見逃す）
  • 皮膚裂創・爪床裂創を確認
  • 開放骨折なら即時整形コンサル（洗浄・抗菌薬）

🩺 身体所見

  • 腫脹・皮下出血・変形・回旋転位
  • 爪下血腫の範囲を評価（50% 以上はドレナージ適応）
  • 爪床裂創の有無
  • 第1趾 MTP / IP 関節の脱臼を確認（最多）
  • 知覚・末梢循環チェック
  • 伸筋腱損傷の評価は初診では困難 → 後日整形に委ねる

🧪 画像

  • AP + 斜位を撮影（lateral は他趾と重なる）
  • 骨折線・転位・脱臼の確認
  • 第1趾は MTP・IP 関節の脱臼を必ず評価

💉 マネジメント

固定：バディテープ

  • 位置にかかわらずバディテープ（隣接趾と一緒に巻く）
  • ガーゼを間に挟んで皮膚浸軟予防
  • 硬めの底の靴 or 松葉杖で歩行

爪下血腫

  • 爪面の 50% 以上に及ぶ血腫 → ドレナージ（電気焼灼・18G 針穿孔）
  • 50% 未満 or 無痛なら経過観察

爪床裂創

  • 時間に余裕があれば縫合・爪床整復
  • 時間がなければ後日整形に処置依頼可

第1趾の脱臼整復

  • web block で麻酔後、長軸牽引整復
  • 整復後にレントゲンで位置確認
  • 整復不能 → 整形コンサル

整形へ即コンサル

  • 開放骨折
  • 整復不能な脱臼
  • 関節内骨折で大きな転位
  • 多発・粉砕型

🏠 帰宅時の説明

  • バディテープは 2〜3 週間継続、汗・浸軟で適宜巻き直す
  • 挙上・冷却で腫脹軽減
  • 1〜2 週で整形外科外来を受診
  • 爪が抜ける可能性あり（自然脱落で経過観察可）
  • 疼痛増強・発赤・排膿で再受診
