中手骨頸部骨折

⚡ 即時アクション
  1. Boxer's fracture（第 5 中手骨）が典型 — 殴打・喧嘩で受傷した若年男性で knuckle 消失
  2. AP + lat + oblique X 線で側面像の掌側屈曲角を計測
  3. 回旋変形 + angulation 閾値（5 中手骨 > 30°、4 中手骨 > 15°、2-3 中手骨 > 10°）で整復
  4. 保存（buddy taping / ulnar gutter cast）vs ピンニング。皮膚損傷あれば fight bite を念頭に

📞 この疾患が該当する主訴
  • 手関節・手の外傷 — 殴打後の knuckle 消失・MCP 周囲腫脹
  • 転倒 — 転倒時にグーで衝撃を受けて受傷

📝 受傷機転

  • 殴打（壁・人）、転倒時に手をグーで打つ
  • 第 5 中手骨頸部が最多（ボクサー骨折）
  • 次いで第 4 中手骨頸部

🩺 身体所見

  • MCP 関節の腫脹（knuckle の消失）
  • 背側の圧痛 + 変形（掌側凸）
  • 回旋転位のチェック：屈曲させて指先が舟状骨結節に向かうか
    • 爪の向きが他指と揃わない = 回旋転位あり
  • 皮膚損傷の確認：殴打で人咬傷（fight bite）の可能性

🧪 画像

  • 手 3 方向（正面・側面・斜位）
  • 側面像で掌側屈曲角を計測
  • 関節内進展の有無を確認

📐 許容される転位（非整形外科医用）

中手骨許容される掌側屈曲角
第 2 中手骨10°
第 3 中手骨20°
第 4 中手骨30–40°
第 5 中手骨40–50°（60° まで許容する施設もあり）
  • 橈側ほど許容度が小さい（機能障害が出やすい）
  • 回旋転位は度数に関係なく手術適応

💉 マネジメント

保存の基準

  • 転位角度が許容範囲内
  • 回旋転位なし
  • 皮膚損傷なし

保存の方法

  • Ulnar gutter splint（尺側半分を覆うシーネ）
  • MCP 関節 70–90° 屈曲位、PIP/DIP 関節は伸展位
  • 3–4 週間固定

手術の基準（帰宅手術 or 即時整形）

  • 許容角度超え
  • 回旋転位あり
  • 関節内骨折
  • 開放骨折・人咬傷

⚠️ 人咬傷（Fight bite）の扱い

MCP 関節背側の創 = 人咬傷として扱う
  • 殴打で歯が皮膚を切る → 関節内に感染源
  • 起因菌：Eikenella corrodens、連鎖球菌、口腔内嫌気性菌
  • 開放骨折 + 関節内感染 = 緊急洗浄 + 抗菌薬
  • アモキシシリン／クラブラン酸（経口）または SBT/ABPC（静注）
  • 整形外科で即時洗浄

🏠 帰宅時の説明

  • 痛みが強い、しびれ、指が動かない → 即受診
  • 固定中は清潔に、指先の末梢循環を確認
  • 1 週以内に整形外科外来
