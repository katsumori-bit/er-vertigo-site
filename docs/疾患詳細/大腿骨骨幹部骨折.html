大腿骨骨幹部骨折

同時並行で評価する 3 軸
典型像
  • 若年男性、交通事故（バイク）・高所転落
  • 明らかな変形 + 短縮 + 疼痛
  • 多発外傷の一部であることが多い
  • 軽微外力なら病的骨折（転移性骨腫瘍）を疑う
  • 受傷後 24–72h は脂肪塞栓症候群に注意
血管

出血性ショックが最大リスク。片側 1000–1500 mL、両側で 3 L 出血しうる。

  • 血圧・脈・乳酸を継時、ABC 優先
  • 両側骨幹部骨折は大量輸血準備
  • 大腿動静脈の損傷を念頭（特に貫通・粉砕）
  • 足背・後脛骨動脈を必ず触知（左右比較）、ABI 低下なら CTA
神経

坐骨神経（後方走行）の損傷を念頭。下腿のコンパートメント症候群も大腿で起こりうる。

  • 足趾運動・足背の感覚（坐骨神経領域）
  • 下腿の緊満・受動背屈痛 → コンパートメント疑い
  • 麻痺所見は固定肢位を見直す根拠
  • 記録を残す（後の責任分界点）
画像の見方
  • 大腿 AP + lat（膝・股関節含む）— 隣接関節の合併骨折を見落とさない
  • 多発外傷ならFAST + 全身 CTを優先
  • 両側を疑う場合は両側 Xp
  • 軽微外力 → 病的骨折の鑑別：既往癌歴、皮質欠損、骨融解像 → 骨シンチ・全身 CT で原発巣検索
  • ステム周囲はインプラント末端の透亮像に注目（旧画像と比較）

ER で打つ手

  1. ABC 優先、ショックなら輸液・輸血（クロスマッチ）
  2. FAST + 全身 CT で他外傷検索
  3. Thomas splint / 牽引固定で疼痛・出血軽減
  4. 大腿神経ブロックでオピオイド依存を減らす（エコーガイド下、鼠径靱帯直下、大腿動脈外側、20 mL）→ 大腿神経ブロック
  5. 髄内釘（IM nail）固定が標準術式（プレートは小児）
  6. 多発外傷・全身不安定なら Damage Control Orthopedics（外固定で一時安定化 → 全身回復後に内固定）
  7. 24h 以内手術が脂肪塞栓リスク軽減
