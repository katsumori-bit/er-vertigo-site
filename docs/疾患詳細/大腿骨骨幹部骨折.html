大腿骨骨幹部骨折

⚡ 即時アクション
  1. 血管神経評価 + 出血性ショックに注意（片側 1.5 L 出血あり得る）
  2. AP + lat X 線、両側 X 線も検討（多発外傷で全身 CT）
  3. 髄内釘が標準（プレートは小児）
  4. 脂肪塞栓症候群を念頭（24-48 h、呼吸困難・点状出血・意識障害）

📞 この疾患が該当する主訴
  • 転倒 — 高エネルギー転倒・転落の典型疾患

📊 疫学・受傷機転

  • 若年男性、交通事故（バイク事故）、高所転落
  • 高齢者では低エネルギーでも発生（骨粗鬆症・骨折歴の影響）
  • 病的骨折（転移性骨腫瘍）の鑑別を常に考える

🩺 診断

  • 明らかな 変形 + 短縮 + 疼痛
  • レントゲン or 全身 CT で確定
  • 多発外傷なら FAST + 全身 CT

⚠️ 併発リスク

合併症頻度・機序
出血性ショック骨折部からの出血 1000–1500 mL/側、両側なら 3 L

出血性ショック + 脂肪塞栓
  • 両側大腿骨幹部骨折 = 大量輸血準備
  • 受傷後 24–72 時間の低酸素・意識障害は 脂肪塞栓症候群を疑え

🚑 ER での対応

  1. ABC 優先、ショックなら輸液・輸血
  2. FAST + 全身 CT で他外傷検索
  3. Thomas splint / 牽引固定で疼痛・出血軽減
  4. 大腿神経ブロックでオピオイドに依存しない鎮痛
  5. 即時整形 → 髄内釘固定（24 時間以内が予後良好）

💉 大腿神経ブロックの意義

  • 高齢・多発外傷で オピオイド依存を減らせる
  • エコーガイド下 + 20 mL 程度の局麻薬（1% リドカイン or 0.25% ブピバカイン）
  • 鼠径靱帯直下で、大腿動脈の外側に穿刺
  • 詳細は 大腿神経ブロック を参照

🦴 病的骨折の疑い

  • 軽微な外力で大腿骨骨幹部骨折 → 転移性骨腫瘍を疑う
  • 既往歴（癌）・原発巣検索
  • 骨シンチ・全身 CT

🏥 マネジメント原則

  • 髄内釘固定（IM nail）が第一選択
  • 多発外傷・不安定な場合は Damage Control Orthopedics（外固定 → 全身安定後に内固定）
  • 手術は 24 時間以内が推奨（脂肪塞栓リスク軽減）
