膝靱帯・半月板損傷

⚡ 即時アクション
  1. ACL / PCL / MCL / LCL / 半月板 を区別（pop 音 + 急速血腫は ACL、外反痛は MCL、ロッキングは半月板）
  2. Lachman / anterior drawer / pivot shift / McMurray で評価（Lachman が ACL に最も鋭敏）
  3. MRI で確定（整形外科外来で。ER では Ottawa Knee Rules で骨折除外＋ヒンジ装具固定）
  4. 若年 active は手術、高齢非 active は保存（年齢・活動性・不安定性で判断）

🔢 膝外傷の除外順序

#靱帯損傷 #半月板 #膝関節 #ER #スポーツ外傷

膝内障 = 膝の靱帯・半月板損傷を総称、ER では除外診断で
  • 関節内骨折・脱臼を除外した後に疑う
  • ACL 断裂：pop 音 + 急激な血腫
  • MCL 損傷：外反ストレスの痛み
  • 半月板損傷：膝の引っかかり・ロッキング
  • ER では身体所見 + 装具固定 + 整形外科紹介で十分
  1. 骨折（レントゲン・必要なら CT）
  2. 脱臼（膝蓋骨・膝関節）
  3. 靱帯・半月板損傷（臨床所見 + MRI）

🦵 ACL（前十字靱帯）損傷

受傷機転

  • 非接触型：ピボット動作・着地で膝を捻る（バスケ・サッカー）
  • 接触型：タックル
  • 女性に多い（男性の 2–8 倍）

身体所見

  • 受傷時に「パキッ」という pop 音
  • 急激な関節内血腫（1–2 時間でピーク）
  • Lachman test：30° 屈曲で脛骨前方引き出し → 陽性（感度 85%）
  • 前方引き出しテスト：90° 屈曲（感度 55%）
  • Pivot shift test：感度 24% だが特異度 98%

マネジメント

  • 血腫は関節穿刺で排液することもあるが、ER では不要
  • ヒンジ装具 + 松葉杖
  • MRI で靱帯・半月板評価（整形外科外来で）
  • 手術 vs 保存：年齢・活動性・不安定性で判断

↔️ MCL（内側側副靱帯）損傷

受傷機転

  • 膝外側からの 外反ストレス（ラグビー・アメフトのタックル）

身体所見

  • 内側関節裂隙の圧痛
  • 外反ストレステスト（30° 屈曲位）：不安定性 → 陽性

分類・治療

Grade所見治療
I部分断裂、不安定性なし装具で保存
II部分断裂、軽度不安定装具 4–6 週
III完全断裂、明らかな不安定保存 or ACL 合併例は手術
  • 保存治療が基本（血流豊富で修復良好）

↩️ LCL（外側側副靱帯）損傷

  • 膝内側からの内反ストレス（稀）
  • 後外側支持機構（PLC）合併で 神経血管損傷リスク
  • 疑えば MRI + 整形外科紹介

🚗 PCL（後十字靱帯）損傷

  • ダッシュボード損傷（膝屈曲位で脛骨が後方に押される）
  • 後方引き出しテスト 陽性
  • 単独断裂は保存でも良好、合併例は手術

🌙 半月板損傷

受傷機転

  • 膝を 屈曲 + 回旋
  • 年齢で病態が異なる：若年は外傷性、中年以降は退行性

身体所見

  • 関節裂隙の圧痛
  • McMurray test：屈曲 → 外旋 or 内旋で伸展、関節裂隙に clicking
  • Apley 圧迫テスト
  • ロッキング（bucket handle tear で膝が伸びない）

マネジメント

  • MRI で確定診断
  • ロッキング → 早期関節鏡
  • 退行性変化 → 保存優先、リハビリ

📐 Ottawa Knee Rules（レントゲン撮影基準）

  • 膝痛で以下のいずれかなら撮影：
  1. 55 歳以上
  2. 腓骨頭の圧痛
  3. 膝蓋骨単独の圧痛（他の圧痛なし）
  4. 膝の 90° 屈曲ができない
  5. 受傷直後 + ER で合計 4 歩歩けない

🏥 ER での対応

  1. Ottawa Knee Rules でレントゲン適応を判断
  2. 骨折・脱臼を除外
  3. 靱帯・半月板損傷を疑えば ヒンジ装具 + 松葉杖
  4. 帰宅保存 or 帰宅手術 として整形外科外来へ（1–2 週以内）
