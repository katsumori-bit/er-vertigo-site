急性腸炎・感染性下痢症（Acute Enteritis / Infectious Diarrhea）

#ER #救急 #腹痛 #下痢 #感染性腸炎 #pseudoappendicitis #除外診断

「胃腸炎」という診断名を捨てる
そんな病気はない。胃炎と腸炎は成因が異なり前者は非感染性、後者は感染性であることが多い。「腹痛があるけどたいしたことない」のときに使われがちなゴミ箱診断で、「原因がよくわからない」と言っているのとほぼ同義。胃なのか腸なのかで想定する疾患は全然違う。
本項は小腸・大腸が主座の疾患を対象とする。

病歴

要点：「食歴」と「旅行歴」。基本だが順守率は低い。

食歴で誤解されがちな 2 点

  • 「食あたりは食べた直後」は誤り：実際は食後数日後の発症のほうが多い → 1 週間前まで遡って聞く
  • 「同じ物を食べた人は何ともないから違う」は誤り：集団発生のほうが稀、同じものを食べても全員発症するわけではない → 患者は自分だけ食べたものを想定しがち

細菌性 vs ウイルス性

  • 一般に細菌性のほうが重症。下痢・血便・発熱が顕著
  • 口側の病変ほど嘔吐、肛門側ほど下痢
    • Norwalk（ノロ）：近位空腸が主座 → 嘔吐が顕著、下痢は遅れる
    • Rotavirus：小腸の広範囲 → 下痢が顕著（小児で有名）
    • 軽症ウイルス性の多くは回盲部主座
    • 病原性大腸菌は大腸主座
  • Yersinia 属は「腸間膜リンパ節炎」「終末回腸炎」を呈し虫垂炎との鑑別を要する（pseudoappendicitis）

重症例のキーとなる病歴

  • 下痢の回数と量／食事摂取の可否／血便／高熱

身体所見

急性腸炎が「上位にあがる」のは以下の 3 つすべてが揃うとき。

  • 圧痛部位が局在しない（炎症がびまん性）
  • 圧痛部位に再現性が乏しい（小腸は可動性あり）
  • 反跳痛がない（粘膜中心の炎症で漿膜に達していない）

細菌性で重症型は回盲部〜大腸が主座のため圧痛部位に再現性がある。虫垂炎とは圧痛範囲が異なるが、鑑別は慎重に。

初期診断と鑑別疾患

感染性腸炎の症状として腹痛は必須ではない。下痢／嘔吐／発熱が主であるため、「腹痛」へのアプローチとしての急性腸炎は除外診断以外あり得ない。

除外診断とは「何の疾患を除外したのかをカルテに明記する」こと。除外した疾患を即答できないならアプローチはやり直し — そのやり方では見逃しが減らない。

検査

① 便培養（Stool culture）

結果が返る頃には軽快していることが多いが、病原性が高い／長期化に備えて可能なかぎり提出。

② 便のグラム染色（便スメア）

  • 白血球が多い → 細菌性腸炎を示唆
  • 本来の目的は「gull wing（カモメの翼形）」の確認 → Campylobacter 腸炎を一発診断できる

③ 画像検査

  • 下痢メイン・典型的な急性腸炎なら不要
  • 腹痛がメインなら腸炎は除外診断 → 除外対象の疾患に必要な検査を行う

確定診断

  • 下痢が主で腹痛が軽度なら感染性下痢症の可能性が高い
  • 腹痛がメインの「急性腸炎」の確定診断方法はない → 鑑別の否定で診断（除外診断）
  • 回盲部主座が多く、急性虫垂炎を否定すべきケースが多い。Yersinia 腸炎は強い腹痛で虫垂炎として手術されることがあり「pseudoappendicitis」と呼ばれる
「下痢があるから腸炎で安心」とは限らない
骨盤内直腸付近の炎症性病変（膿瘍）が刺激性に下痢を生じる。代表は虫垂炎穿孔後の骨盤膿瘍、女性なら骨盤腹膜炎（PID）も。
よく聞くと大量水様下痢ではなく「しょっちゅうトイレに行くが少量ずつ」=テネスムス症状のことがある。

治療方針

① 対症療法

「効能書き」だけで薬を選ばない
ブチルスコポラミンは鎮痙薬で鎮痛成分はない。メトクロプラミドは中枢性制吐だが蠕動亢進薬でもある。痛みはあるが蠕動が弱い人にブチルスコポラミン、吐いているが蠕動亢進している人にメトクロプラミドが使われるのは医学とは言えない。両者併用に至っては亢進したいのか抑えたいのか不明。
鎮痛が欲しいなら鎮痛薬、嘔吐が水様ならNG ドレナージのほうが効くことも。病態に即した治療を。
  • 下痢・嘔吐主体なら脱水が病態の中心。脱水量を計算で推定して輸液計画を立てる。漫然と補液しない

② 抗菌薬

  • ウイルス性なら不要。細菌性でも有効性を示す文献は限定的。多くはself-limited（ほっとけば治る）
  • S. aureus などの毒素型は infection ではなく poisoningで、機序的に抗菌薬は奏効しにくい

使用基準（例）

  • Traveler's diarrhea で 38.5℃以上の発熱または敗血症を疑う症状
  • 免疫抑制状態で頻回の血便・発熱・腹痛・テネスムス
  • 推奨薬：フルオロキノロン
EHEC（O157 など）には抗菌薬を使わない
EHEC では抗菌薬使用が HUS への移行および死亡率を上昇させる可能性があり推奨されない。
