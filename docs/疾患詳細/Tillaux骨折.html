Tillaux骨折

⚡ 即時アクション
  1. 10〜14歳の足関節外傷では靱帯損傷より 骨端線骨折を疑う
  2. AP + lateral + mortise で 脛骨遠位骨端線の縦骨折線を確認
  3. 関節内骨折なら CT 追加・即時整形コンサルト
  4. シーネ固定 + 非荷重で手術待機

概要：10〜14歳、脛骨遠位骨端線の縦骨折線が決め手
Tillaux骨折 = 思春期の脛骨遠位 Salter-Harris III
  • 10〜14 歳の足関節外傷で、靱帯より骨端線が弱いため発生
  • Tillaux 骨折＝Salter-Harris Type III（脛骨遠位骨端の関節内骨折）
  • triplane 骨折＝Salter-Harris Type IV（矢状面・前額面・水平面の3面に骨折線）
  • レントゲンで 脛骨遠位骨端線の縦骨折線を確認 → CT で精査
  • 関節内骨折のため 手術になりやすい、即時整形コンサルト
📞 この疾患が該当する主訴
  • 足関節・足部の外傷 — 思春期の足関節外傷で疑う

📝 患者層・受傷機転

患者層

  • 小学校高学年〜中学1年生（おおむね 10〜14 歳）
  • 脛骨遠位骨端線は 14〜15 歳で閉鎖、それ以前は骨端線が損傷部位になりやすい
  • 骨端線閉鎖前は 靱帯 > 骨端線の強度 → 同じ外力でも成人の捻挫が小児では骨折に

受傷機転

  • 足を 捻る（内反・外反・回旋）
  • 受傷状況は成人の足関節捻挫と同じ（スポーツ・転倒・段差）
  • 同じ機転でも年齢によって骨折部位が変わる

🩺 身体所見

  • 足関節前外側の圧痛・腫脹（脛骨遠位外側骨端部）
  • 荷重困難
  • 10〜14 歳で「足関節捻挫」と紹介されたら骨端線骨折を一度疑う
  • 外果・内果の圧痛では成人型と区別できないため、画像で判断

🧪 画像

レントゲン 3 方向

  • AP + lateral + mortise で撮影
  • 見るべきポイント：脛骨遠位骨端線に縦骨折線があるか
  • 骨端線の縦に走る骨折線が確認できれば Tillaux/triplane を強く疑う
  • 判別しにくければ健側との比較撮影

CT

  • 縦骨折線を確認したら CT 追加
  • Type III（Tillaux）と Type IV（triplane）の区別、転位量、手術計画
  • 分類の最終判断は CT で整形外科に委ねる

📊 Salter-Harris 分類との関係

固有名Salter-Harris骨折線特徴
Tillaux 骨折Type III骨端線 + 骨端（関節内）脛骨遠位外側の関節内骨折
triplane 骨折Type IV骨幹端 + 骨端線 + 骨端矢状面・前額面・水平面の 3 面に骨折線
ポイント
  • Tillaux も triplane も 脛骨遠位の 関節内骨折
  • 骨端線が部分的に閉鎖し始める 移行期（外側がまだ開いている）に発生
  • 分類詳細は 小児骨端線骨折 を参照

💉 マネジメント

非整形外科医の役割

  1. 10〜14 歳の足関節外傷では 骨端線骨折を疑う
  2. AP + lateral + mortise で 脛骨遠位骨端線の縦骨折線を確認
  3. 縦骨折線あれば CT 追加・即時整形コンサルト
  4. 関節内骨折のため手術になりやすい（解剖学的整復が必要）

固定（手術待機中）

  • U-slab（後方シーネ + U 字シーネ）
  • 90° 中間位
  • 松葉杖で 非荷重

手術

  • 転位 ≥ 2 mm の関節内骨折は手術適応
  • 整形外科で CT 評価後、観血的整復固定

🏠 帰宅時の説明

  • 松葉杖で 非荷重、荷重開始は整形外科医の指示まで
  • 手術待機中は患肢挙上・冷却で腫脹軽減
  • 腫脹・疼痛・しびれの増強で再受診
  • 関節内骨折のため 長期 follow（成長障害評価）が必要と説明
