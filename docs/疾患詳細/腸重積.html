腸重積（Intussusception）

⚡ 即時アクション
  1. 小児 3 ヶ月〜3 歳が典型：間欠的啼泣 + 嘔吐 + 血便（イチゴゼリー状）の三徴。成人は「腸閉塞の etiology」として鑑別、間欠期にも痛みあり・局在圧痛・腫瘤触知が手がかり
  2. US で target sign（短軸）/ pseudo-kidney sign / doughnut sign。着目すべきは二重リングではなく腸間膜（血管と脂肪織）が腸管内に引き込まれている像。成人では先進部腫瘤の確認も
  3. 高圧浣腸（空気・液体）で整復：小児型・先進部なし・腹膜炎所見なしが適応。穿孔リスクに備え外科スタンバイ
  4. 整復不能 / 穿孔疑い / 成人で先進部腫瘍ありは外科手術（重積部の切除）。Damage control 視点で残存腸管を温存

📞 この疾患が該当する主訴
  • 腹痛 — 間欠痛＋持続痛、局在性のある圧痛と腫瘤
  • 嘔気・嘔吐 — 約 80%、小腸（特に空腸）重積で初期から
  • 血便・下血 — 約 30%（大腸重積で出やすい）、小児ではイチゴゼリー状
  • 発熱 — 腸管虚血・壊死進行例

🔬 分類・先進部

本項では成人の腸重積を扱う（小児は別項）。腸閉塞の原因の 1% と稀（年間 100 例の腸閉塞に 1 例くらい）。小児と異なり多くは重積の原因となる疾患（先進部）が存在する — 腸が内腔の腫瘍を異物と誤認して蠕動で押し出そうとした結果、口側腸管が肛門側に嵌り込む（逆蠕動パターンもあり）。

先進部となる病変

  • 小腸：良性は脂肪腫、悪性は転移性腫瘍
  • 大腸：良性は脂肪腫、悪性は腺癌
  • 稀：メッケル憩室、ポリポーシス（Peutz-Jeghers 症候群など）

分類

  • ① Entero-enteric：小腸の重積
  • ② Colo-colic：大腸の重積
  • ③ Ileo-colic：先進部は回腸、上行結腸内に重積
  • ④ Ileo-cecal：バウヒン弁が先進部となって回腸が結腸内に重積

方向：順行性（口側 → 肛門側内）と逆行性がある。

📖 病歴

  • 発症は sudden ではなく acute onset。先行症状は伴わない
  • 重積腸管は虚血状態で、病変部に一致した痛み（約 80%）
  • 大腸重積：腸閉塞症状が出るまで時間がかかる → 初期は痛みで来院、胆汁混じり大量嘔吐は出にくい
  • 小腸（特に空腸）重積：初期から嘔吐
  • 嘔吐自体は重積全般に約 80% で見られるが、食物残渣のみや嘔気の割に内容少なら腸閉塞らしくない

痛みの性状

  • 小腸重積：持続痛＋蠕動時の間欠痛
  • 大腸重積：多くは持続痛のみ

その他

  • 排便排ガスは典型例で消失
  • 血便（重積部の虚血からの出血）：約 30%、大腸重積で出やすい
  • 来院前に軽微な症状を繰り返していることがあり（重積と解除を反復）

🩺 身体所見・初期診断と鑑別

身体所見

  • 小腸重積は基本的に「小腸閉塞」が臨床像。腸重積の 50% は初期診断「腸閉塞」（間違っていないが etiology の言及が必要）
  • 腹部膨満は著明でない、腸雑音は通常亢進
  • 重積した腸管はそこそこ大きな腫瘤として触知でき、この部位に一致した圧痛
  • 反跳痛は初期はない
  • 大腸重積では大腸閉塞症状が出る前（数日かかる）に重積の痛みで来院する → 重積の局所所見のみのことが多い

初期診断と鑑別

  • 小腸重積は腸閉塞の etiology の鑑別
  • 癒着性腸閉塞との違い：手術歴がない／間欠期にも痛みがある／局在した圧痛部位／同部位に腫瘤を触れる
  • NG ドレナージ後、間欠痛は消失するが局在性のある持続痛が続くのも特徴
  • 腫瘤を触知できるのは 5% 程度との報告もあり、体型・部位による
  • 大腸重積は大腸癌の穿孔（穿通）に伴う膿瘍形成などが鑑別
  • 範囲が広くなる腸炎は鑑別になりにくい

🖥 検査と確定診断

単純 X 線

  • 非特異的所見にしかならない
  • 典型的な single obstruction では大腸ガスが消失するが、腸重積では重積部の痛みで早期来院するので大腸ガスが残っていることが多い

精密画像（CT・US）

  • 重積部位の直接観察で診断可能。有名な「target sign」
Target sign より「腸間膜の引き込み」を見る
着目すべきは二重リングではなく、腸間膜（血管と脂肪織）が腸管内に引き込まれている像。腸間膜の厚みがあるからこそ重積が「嵌頓」となり虚血を生じる。
「腸間膜が引き込まれている」所見を示せなければ本物の腸重積とは言えない。
さらに、成人腸重積は先進部に腫瘤性病変を有することが多いので、これも確認する。

偽性腸重積（Pseudo intussusception）の落とし穴

「target sign」という用語が独り歩きし、CT で「ターゲットサイン様」を見ただけで腸重積と相談されることがある。主に 2 パターン：

  • パターン 1：全層性壁肥厚した腸管（腸アニサキスなど）— 粘膜と筋層が血流豊富で二重リングに造影される。ホンモノよりサイズが小さい／きれいな同心円すぎる。多くは短時間で自然軽快。腸炎像で閉塞症状なし
  • パターン 2：腫瘍浸潤による壁変形（pseudo intussusception）— 漿膜面に達した腫瘍が周囲を引き込み「重積様」に変形。無症状か軽微な症状で偶然見つかる。CT だけ見て慌てて緊急手術しない

💊 治療

  • 成人は先進部（leading point）に腫瘍性病変を有することが多い → 手術（重積部の切除）が原則
  • 術前に悪性でないと判明、または残存小腸が少ないなどの理由があれば、重積解除のみを施行することも
  • とくに若年成人で小児型（先進部なし）の回盲部重積（ileo-colic／ileo-cecal）に遭遇することがあり、内視鏡的整復も試みられる

🧭 胃切除術後の腸重積

  • 特殊なタイプ。胃手術の何年後にも発症しうる
  • 胃空腸吻合部あるいはやや遠位の空腸で重積、先進部（腫瘍など）を有さないのが特徴。順行性も逆行性もあり
  • Billroth I 法での発症報告はほぼなく、空腸再建（胃全摘・Roux-Y・Billroth II・Brown 吻合）に限られる
  • 頻度：胃手術数千に 1 例
  • 逆行性の場合、虚血粘膜からの出血が吐血（暗褐色）として現れる（小児腸重積の下血と同様）
  • 治療：胃手術部に近い逆行性なら内視鏡的空気整復可能なことも。順行性 or 距離が遠ければ手術
