Maisonneuve骨折

同時並行で評価する 3 軸
典型像
  • 足関節骨折の 約 5%（1840 年 Maisonneuve 報告）
  • 外反 + 外旋ストレス（スキー・サッカー・ランニング、中高年女性の転倒）
  • 遠位脛腓靱帯断裂 ＋ 骨間膜損傷 ＋ 腓骨近位端骨折。内果骨折を伴うことが多い
  • 所見：内果圧痛 or 三角靱帯部圧痛、脛腓間圧痛
  • 膝外側直下・腓骨頭部の圧痛（最重要）
  • 足関節 X 線では腓骨近位は写らない（フィルムアウト）→ 見逃し最大の落とし穴
  • Weber 分類では Type C、Lauge-Hansen の SE-IV / PE-III/IV 相当
血管

通常は問題ないが、不安定骨折なので確認の流儀。

  • 足背動脈・後脛骨動脈を必ず触知（左右比較）
  • 腓骨近位端の腫脹大では局所循環評価
  • 触れない／弱い → 整復・固定後に再評価、改善なければ CTA
神経

腓骨神経麻痺の合併に注意（腓骨頭近位の骨折で牽引・圧迫）。

  • 足関節背屈・第一趾背屈（深腓骨神経・L5）
  • 足背の感覚（浅腓骨神経）
  • 第 1・2 趾間の感覚（深腓骨神経）
  • 麻痺あればシーネ固定肢位を見直す（圧迫回避）
  • 記録を残す（後の責任分界点）
画像の見方

足関節 X 線だけで帰さない

  • 足関節痛で来院 → Ottawa で足関節だけ撮影 → 腓骨近位端骨折を見逃す、が最大の落とし穴
  • 腓骨近位端圧痛あり → 下腿全長 or 膝レントゲン追加

レントゲン

  • 足関節 3 方向（正面・側面・mortise）
  • 下腿 or 膝レントゲン（腓骨近位端を含む）
  • medial clear space 拡大 → 三角靱帯損傷
  • tibiofibular clear space 拡大 → 脛腓靱帯結合損傷

CT

  • 後果骨片の評価
  • 関節面の段差評価
治療方針 — 帰宅手術が基本

病態の本質

  • 脛腓靱帯結合 + 骨間膜の広範な損傷 = 不安定性大
  • 手術で靱帯結合スクリュー or プレート固定

マネジメント 5 択

状況5 択マネジメントER の動き
典型 Maisonneuve（皮膚・神経血管 OK）日中 即時整形 / 夜間 帰宅手術後方シーネ ＋ 整形 cons.
腓骨神経麻痺合併即時整形（早期手術）固定肢位見直し・整形緊急 cons.
開放骨折・脱臼骨折即時整形洗浄・抗菌薬・破傷風

脛骨と腓骨が離断するため、脛腓間スクリュー or suture button での手術が必要。保存加療できそうな単独内果骨折と勘違いしないこと。

ER で打つ手

  1. 腓骨近位端圧痛 を疑って下腿・膝レントゲン追加
  2. 後方シーネを 90° 中間位で固定。腓骨近位端より中枢側まで延長（骨折点を越える）
  3. 松葉杖で絶対非荷重
  4. 腓骨神経麻痺をチェックし記録
  5. 整形外科紹介（日中なら即時整形、夜間・休日なら帰宅手術）
  6. 挙上＋アイシング、絶飲食、抗凝固薬の有無確認

帰宅指示

  • 絶対荷重禁止、松葉杖使用
  • 数日以内に整形外科で手術日調整
  • しびれ・蒼白・強い疼痛増強・足背屈低下 → 即受診
