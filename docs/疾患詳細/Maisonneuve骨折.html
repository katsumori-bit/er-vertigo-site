#骨折 #Maisonneuve #腓骨近位端 #ER #見逃し

Maisonneuve 骨折 = 足関節捻挫と思ったら 腓骨近位端 で骨折
  • 外反 + 外旋ストレスで、力が 骨間膜〜腓骨近位端まで伝達
  • 足関節痛で来院、腓骨近位端に圧痛を触ったら疑う
  • 足関節レントゲンでは 腓骨は写らない → 膝直下まで撮影を追加
  • 脛腓靱帯結合損傷 + 腓骨近位端骨折 = 不安定 → 手術

定義

  • 足関節外反 + 外旋ストレスで発生
  • 力が下腿骨間膜を伝わって 腓骨近位端を骨折
  • 足関節レントゲンでは写らない → 見逃しやすい

受傷機転

  • 足関節の外反捻挫（内側が開く）
  • スキー・サッカー・ランニング中の受傷が多い
  • 中高年女性の転倒でも発生

身体所見

足関節の所見

  • 内果圧痛 or 三角靱帯部圧痛
  • 脛腓間（足関節前面）の圧痛
  • 足関節の腫脹・荷重不能

腓骨近位端の圧痛チェック（最重要）

  • 膝外側 直下、腓骨頭部の圧痛
  • 足関節痛患者は必ず腓骨近位端を触る
  • 陽性なら足関節レントゲン + 下腿 or 膝レントゲン追加
足関節レントゲンだけで帰さない

足関節痛で来院 → Ottawa で足関節だけ撮影 → 腓骨近位端骨折を見逃す、が最大の落とし穴。

腓骨近位端圧痛あり → 下腿全長 or 膝レントゲン追加。

画像

  • 足関節 3 方向（正面・側面・mortise）
  • 下腿 or 膝レントゲン（腓骨近位端を含む）
  • 足関節の medial clear space 拡大 や tibiofibular clear space 拡大 があれば脛腓靱帯結合損傷

マネジメント — 帰宅手術が基本

病態の本質

  • 脛腓靱帯結合 + 骨間膜の広範な損傷 = 不安定性大
  • 手術で靱帯結合スクリュー or プレート固定

ER での対応

  1. 腓骨近位端圧痛 を疑って撮影追加
  2. U-slab（足関節シーネ）+ 松葉杖非荷重
  3. 整形外科紹介（帰宅手術 or 入院整形）

整形外科医が知りたいこと

  • 受傷機転（外反 or 内反か）
  • 腓骨近位端の圧痛・腫脹の程度
  • 脛腓間の開大（mortise view で評価）
  • 皮膚障害・神経血管障害の有無

鑑別

疾患鑑別点
足関節捻挫（ATFL 損傷）外反ではなく内反、腓骨近位端圧痛なし
足関節果部骨折（単果）腓骨 遠位端 の骨折、近位は健常
腓骨近位端単独骨折直達外力、足関節の所見が乏しい

帰宅時の説明（手術待機例）

  • 絶対荷重禁止、松葉杖使用
  • 数日以内に整形外科で手術日調整
  • しびれ・蒼白・強い疼痛増強 → 即受診
