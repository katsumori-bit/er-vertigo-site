Mallet finger / Jersey finger

同時並行で評価する 3 軸
典型像
  • Mallet finger＝伸筋腱（終止腱）断裂／末節骨基部裂離 → DIP 自動伸展不能
  • Jersey finger＝FDP 腱断裂（環指多） → DIP 自動屈曲不能
  • 受傷機転：球技（ボールが指先に当たる）／ジャージを掴んだ
  • DIP 関節周囲の腫脹・圧痛
  • 初療で見落とされやすい（軽症に見える）
血管

血管損傷は通常合併しない。固有指動脈の触知/拍動感・CRT を必ず確認。

  • 固有指動脈の拍動・色調・CRT（< 2 秒）
  • 左右指で比較
  • 強い腫脹は挙上・冷却
  • 創あれば洗浄・破傷風
神経

神経損傷は通常 OK。固有指神経の感覚を独立して確認。

  • 各指の固有指神経の感覚（橈尺両側）
  • 正中・尺骨・橈骨神経領域 [問題なし]
  • 麻痺所見は記録
  • 腱損傷と神経損傷を分けて評価
画像の見方
  • 患指 AP・lateral・oblique（3 方向）
  • Mallet：
    • 末節骨基部背側の裂離骨片の有無・大きさ
    • 関節面の1/3 以上を占める骨片は要手術
    • DIP 関節の掌側亜脱臼の有無（lateral）
  • Jersey：
    • 末節骨基部掌側の裂離骨片を探す
    • 骨片が掌側で近位に retraction していることがある
    • Xp で骨片が見えない型もあり、身体所見が決め手
  • 骨片陰性の腱断裂は MRI / 超音波で評価
治療方針（タイプ × 骨片）
タイプ所見方針ER の動き
Mallet・腱性 骨片なし、自動伸展不能 保存：DIP 伸展位固定 6 週 Stack 副子／アルミ副子（ペーパークリップ可） ＋ 翌日整形
Mallet・骨性（小骨片） 関節面 < 1/3、亜脱臼なし 保存：DIP 伸展位固定 Stack 副子 ＋ 翌日整形
Mallet・骨性（大骨片／亜脱臼） 関節面 ≥ 1/3 ／DIP 掌側亜脱臼 手術（経皮ピンニング等） 整形相談
Jersey finger DIP 自動屈曲不能（FDP） 早期手術（受傷後 1〜2 週以内） 同日コンサル（腱断端 retraction 進行する）
開放創合併 創あり 洗浄・抗菌薬・整形相談 洗浄・破傷風・即コンサル

ER で打つ手

  • Mallet：DIP 関節を伸展位（または軽度過伸展）で固定。PIP 関節は動かせるように
  • 固定具：Stack 副子 / アルミ副子 / カスタム熱可塑性
  • 固定中の DIP 屈曲を絶対禁忌として本人に説明（「外す時も伸展のまま」）
  • Jersey：シーネ固定 ＋ 同日コンサル（早期手術が予後を決める）
  • 鎮痛は NSAIDs／アセトアミノフェン

関連ハブ・隣接疾患
  • 主訴入口：手関節・手の外傷
  • 隣接疾患：PIP 関節脱臼 ／ 中手骨頸部骨折 ／ ゲームキーパー母指

  • 合併症対応：Jersey は retraction 進行で再建困難、早期手術が必須
