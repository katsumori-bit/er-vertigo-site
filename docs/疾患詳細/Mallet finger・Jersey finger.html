#腱損傷 #mallet #jersey #DIP #ER #指

指の「自動で曲げ / 伸ばし ができない」= 腱損傷を疑う
  • Mallet finger（槌指）：DIP 伸展ができない → 伸筋腱断裂
  • Jersey finger：DIP 屈曲ができない → 深指屈筋腱断裂（環指に多い）
  • 骨片を伴う場合あり（撕脱骨折）
  • Mallet は シーネ固定で保存可、Jersey は 帰宅手術が原則

Mallet finger（槌指）

受傷機転

  • DIP 関節に強制屈曲（ボール・布団シーツ・ドアで指先を突く）
  • 伸筋腱終末部の断裂 ± 末節骨背側の撕脱骨折

身体所見

  • DIP 関節の自動伸展不可
  • 他動伸展は可能（痛みあり）
  • DIP が屈曲位で下垂（槌のよう）

画像

  • 末節骨 側面像で 背側の撕脱骨片 を確認
  • 関節面 1/3 以上の骨片 → 手術考慮

マネジメント

保存（ほぼ全例）

  • DIP 伸展位固定（過伸展ぎみ）、PIP は自由
  • stack splint or アルミシーネ
  • 6–8 週間完全固定、1 日も外さない（外すと再断裂）
  • その後さらに 2–4 週夜間固定

手術

  • 関節面 1/3 以上 の骨片
  • 掌側亜脱臼を伴う
  • 保存で改善しない chronic mallet

帰宅時の説明

  • 1 度でも外すと振り出しに戻る → 入浴・着替えも注意
  • 皮膚トラブルで固定外れる場合は整形外科で代替法検討

Jersey finger

受傷機転

  • ラグビー・アメフトで 相手のジャージをつかんで指が引っ張られる
  • 深指屈筋腱の終末部断裂 ± 撕脱骨折
  • 環指に多い（指のつかみ方で環指に負荷がかかる）

身体所見

  • DIP 関節の自動屈曲不可
  • 他の指を伸展位で保持させ、評価対象の DIP だけ屈曲できるかチェック
  • 他動屈曲は可能

画像

  • 末節骨 側面像で 掌側の撕脱骨片 を確認
  • 腱の退縮位置（レベル）を MRI で評価することもある

マネジメント — ほぼ全例手術

  • 深指屈筋腱を早期に（7–10 日以内）再固定しないと予後不良
  • 紹介は 帰宅手術として 3–5 日以内

ER での対応

  1. DIP 屈曲位のシーネ固定
  2. 即日〜数日以内に整形外科（手外科）へ紹介

鑑別のコツ

MalletJersey
障害自動伸展不可自動屈曲不可
腱伸筋腱深指屈筋腱
受傷機転突き指（屈曲強制）引っ張られる（伸展強制）
