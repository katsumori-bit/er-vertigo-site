総指伸筋腱中央索断裂・ボタン指変形

本疾患を疑う 3 軸
典型像
  • 突指（球技・ボール・道着掴み・転倒）
  • PIP 過屈曲外力
  • PIP 背側・中節骨基部背側の圧痛
  • 受傷直後は PIP 伸展可で正常に見える
  • 1 ヶ月後にボタン指変形で戻ってくる
  • 「突指」「打撲」と誤診されやすい最頻ピットフォール
解剖

総指伸筋腱は PIP 直前で中央索と側索に分かれる。中央索 → 中節骨基部背側付着（PIP 伸展）／側索 → 末節骨基部背側付着（DIP 伸展）。

  • 中央索断裂 → 本来は PIP 伸展不能
  • しかし側索が温存される限り PIP は自動伸展できる（落とし穴）
  • 時間経過で側索が掌側へ滑落
  • → 屈曲力 + DIP 過伸展力に転じる
  • → boutonniere（ボタン指）変形：PIP 屈曲・DIP 過伸展
  • 骨性：中節骨基部背側の avulsion 骨片を伴う型
Elson テスト

受傷直後で PIP 伸展可の症例を拾うための最も鋭敏な身体所見。必ず実施し記録する。

  • PIP をテーブル縁にひっかけ 90° 屈曲位にする
  • 患者に PIP を伸展するよう抵抗下で指示
  • 正常：DIP は柔らかく屈曲したまま（中央索が PIP 伸展に使われ、側索は緩む）
  • 異常（陽性）：DIP が強直性に伸展する（側索が PIP 伸展に動員されている）
  • 左右差で判定する
  • 補助：PIP 背側中央索付着部の圧痛
診察のコツ（身体所見＋画像）

身体所見の優先順

  • PIP 背側中央索付着部の圧痛（中節骨基部背側）
  • PIP の自動伸展：受傷直後は可のことが多い（側索温存のため）
  • Elson 試験（必須）
    • PIP 90° 屈曲位 → 抵抗下で PIP 伸展指示
    • 陽性：DIP が強直伸展（側索が引っ張られる）
    • 陰性：DIP は柔らかく屈曲位を保つ
  • PIP 腫脹・自発痛も補助所見
  • 慢性例（1 ヶ月以上）：完成した boutonniere 変形（PIP 屈曲・DIP 過伸展）

画像

  • 指 Xp AP / lateral（患指中心の撮影が必須）
  • lateral で中節骨基部背側の avulsion 骨片を確認
    • 骨片なし → 腱性断裂
    • 骨片あり → 骨性断裂、転位量を計測
  • PIP 関節面の段差・亜脱臼の有無
  • 「画像が陰性 = 否定」ではない。Elson 陽性なら腱性断裂として治療開始

絶対に外せない落とし穴

  • PIP が伸展できる → 正常と判断してしまう（最頻誤診）
  • Elson 試験を行わない
  • 「突指」「打撲」と片付け、固定せず帰宅
  • 1 ヶ月後の boutonniere で初診となる症例が後を絶たない
治療方針（腱性／骨性／開放性）
パターン所見方針ER の動き
腱性断裂（骨片なし） Elson 陽性、Xp で骨片なし 保存：PIP 伸展位 0° で 6 週間 aluminum splint／Capener splint ＋ 1 週以内整形外来
骨性・転位小 中節骨基部背側 avulsion、転位わずか 保存：PIP 伸展位 0° で 6 週間 シーネ ＋ 1 週以内整形外来
骨性・転位大／関節面不整 骨片の明らかな転位、関節面段差 手術適応 シーネ ＋ 即時整形コール
開放性 背側皮膚裂創 ＋ 伸展不能 緊急整形 ＋ 一次修復 洗浄・破傷風確認 ＋ 即時整形
慢性 boutonniere 受傷 1 ヶ月以上、変形完成 整形外科で評価（保存 or 再建術） シーネ ＋ 外来紹介

ER で打つ手

  • aluminum splint（または Capener splint）で PIP 伸展位 0° 固定
    • DIP は free（積極的に動かしてよい：側索の掌側滑落を防ぐ）
    • MP も free
  • 6 週間 絶対屈曲禁止を本人と家族に明確に説明
    • 入浴・着替えでも 1 度も PIP を曲げない
    • 1 度でも曲げると治癒が遅れ、ボタン指変形に進む
    • DIP は動かしてよい（むしろ動かす）
  • 鎮痛：NSAIDs／アセトアミノフェン
  • 1 週以内に整形外科外来受診を指示
  • disposition：原則帰宅保存。骨性転位大・開放性は帰宅手術 or 即時整形

関連ハブ・隣接疾患
  • 主訴入口：手関節・手の外傷
  • 隣接疾患：Mallet finger・Jersey finger ／ PIP 関節脱臼
  • 処置：aluminum splint（PIP 伸展位 0°、DIP free）／ Capener splint
  • 合併症対応：1 ヶ月後の boutonniere 変形は受傷時に拾えなかった症例が大半。Elson 試験の徹底が予防策
