めまい vs 前失神の見極め

⚡ 即時アクション
  1. 「めまい」の主訴は 4 カテゴリに分かれる：真のめまい（vertigo）／ 失調感（disequilibrium）／ 前失神（presyncope）／ 浮動感（lightheadedness）
  2. 「どんな感じ？」だけでは分類できない — 状況・誘因・持続時間・随伴症状で判定（TiTrATE 思想）
  3. 前失神を見落とすと心血管疾患（不整脈・大動脈解離・出血等）の見逃しに直結
  4. 前失神疑い → 意識消失フローへ。真のめまい → 持続性判定へ

📚 「めまい」4カテゴリの整理

カテゴリ感じ方典型誘因 / 状況代表疾患
真のめまい
(vertigo)		 ぐるぐる回る回転感 頭位変換（BPPV）／ 自発性持続（前庭神経炎・脳血管） BPPV、前庭神経炎、後方循環梗塞、Ménière 病
失調感
(disequilibrium)		 ふらつき・歩行不安定 歩行時に増悪、坐位では軽減 小脳病変、末梢神経障害、薬剤性、加齢性
前失神
(presyncope)		 眼前暗黒感・気が遠くなる 立位で悪化・労作・動悸・胸痛 不整脈、起立性低血圧、出血、心原性失神、大動脈解離
浮動感
(lightheadedness)		 ふわふわ・地に足つかず 慢性的、持続性、過換気、不安 過換気症候群、不安・うつ、薬剤性
最優先で除外すべきは「前失神」

真のめまい・失調感・浮動感は急性致死性が比較的低い。一方、前失神は不整脈・大動脈解離・消化管出血など致死的疾患の前駆症状であることがある。「めまい」と訴える患者でまず前失神を否定するのが安全策。

🗣 問診のコツ

「どんな感じ？」は鵜呑みにしない

患者の感覚言語は曖昧で、「めまいがする」と訴えても実際は前失神や失調感のことが多い。感覚の質を聞くより、状況と誘因を聞くほうが正確。

必ず聞く 6 項目

  1. 発症様式：突然（秒〜分）／ 亜急性（時間〜日）／ 慢性
  2. 持続時間：今も続いているか／ 何分・何時間・何日
  3. 誘因 / 状況：体位変換／ 立位／ 労作／ 自発／ 食事後／ 排尿後
  4. 随伴症状：眼前暗黒感・動悸・胸痛・呼吸苦・嘔気・耳鳴・難聴・神経症状
  5. 反復性：初発か繰り返しか／ 過去の発作の様子
  6. 背景：薬剤（降圧・利尿・抗凝固・抗うつ）・心血管リスク・脱水・出血

前失神を強く疑う特徴

  • 立位で悪化、臥位で改善
  • 眼前暗黒感（目の前が暗くなる、視野狭窄）
  • 動悸・胸痛・呼吸苦合併
  • 労作誘発（労作性失神は心原性を強く示唆）
  • 食事後・排尿後・咳嗽後（迷走神経反射）
  • 過去に失神・前失神の既往

真のめまい（vertigo）を強く疑う特徴

  • 明確な回転感（自分または周囲が回る）
  • 頭位変換で誘発（BPPV）
  • 強い嘔気・嘔吐
  • 耳症状（難聴・耳鳴）合併
  • 固視できない（眼振あり）

🔍 身体診察

必ずとる項目

  • バイタル（BP 左右差含む・HR・RR・SpO2・体温）
  • 起立性血圧変化：臥位 → 立位（または坐位）で SBP >20mmHg or DBP >10mmHg 低下／ 症状再現で陽性
  • 歩行・起立観察：歩けない／ 立てない＝中枢徴候
  • 自発眼振：注視方向性・方向交代性・固視抑制
  • 全般神経診察（脳神経・運動感覚・小脳・構音）
  • 聴力（指こすり・ささやき）
  • 指尖血糖（低血糖は前失神・失調感の鑑別）

前失神を示唆する所見

  • 起立性低血圧（症状再現を伴う SBP 20mmHg 以上低下）
  • 頻脈・徐脈・不整脈
  • 顔面蒼白・冷汗・末梢冷感
  • 心雑音（大動脈弁狭窄症等）

真のめまい（vertigo）を示唆する所見

  • 自発眼振あり（特に水平回旋性）
  • Dix-Hallpike 陽性（BPPV）
  • 歩行不安定だが起立は可能
  • 聴力異常（Ménière, AICA 梗塞）

⚡ 12誘導 ECG の適応

めまいでルーチン全例に ECG は不要。下記いずれかなら必須：

  • 前失神型を 1 つでも疑う（眼前暗黒感・動悸・胸痛・労作誘発）
  • 意識消失歴あり
  • 高齢 + 心血管リスク（HT / DM / DL / AF / 喫煙 / 既往）
  • 不整脈・心疾患の既往
  • 抗凝固薬・抗不整脈薬使用
気になるパターン
  • QT 延長（薬剤性 Torsade de Pointes）
  • 新規 AF（心原性塞栓 → 後方循環梗塞）
  • Brugada 型・WPW・洞不全
  • ST-T 変化（虚血、特に下壁誘導は迷走神経刺激も）

🔀 振り分け（次のステップ）

前失神疑い

  • → 意識消失フロー へ
  • ECG・モニター装着、起立性血圧、循環器評価
  • 大動脈解離・消化管出血・心タンポナーデ等の鑑別を広げる

真のめまい（vertigo）

  • → めまい持続性判定 へ
  • 持続性（AVS）か反復性（EVS）かで workflow が分かれる

失調感・浮動感

  • 失調感：神経診察を丁寧に（小脳・末梢神経・薬剤性）
  • 浮動感：器質性除外後、過換気・不安・うつ等の機能性も視野に

⚠️ ピットフォール

  • 「めまいがする」を真のめまいと決めつける — 前失神を見逃して心血管疾患を逃す
  • 「ふらつき」を末梢前庭で片付ける — 失調感は中枢病変（小脳）も
  • 立位で悪化を軽視 — 起立性低血圧・心血管由来の典型サイン
  • 労作誘発を見落とす — 労作性失神は心原性を強く示唆
  • ECG を撮らない — 高齢・心血管リスクでは閾値を下げて
  • 薬剤性めまい / 前失神を見落とす — 降圧薬・利尿薬・α遮断薬・抗うつ薬
  • 過換気症候群と決めつけ — 過換気の影に塞栓症や代謝性アシドーシス
