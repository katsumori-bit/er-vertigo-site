第1中手骨基部骨折

⚡ 即時アクション
  1. Bennett vs Rolando vs 関節外を区別 — 母指 CMC 関節周囲圧痛・短縮・回内変形
  2. AP + lat + oblique X 線で関節内骨折線を確認、必要なら CT
  3. Bennett・Rolando は手術（関節内骨折、ORIF または pinning）
  4. 関節外骨折は保存（thumb spica）、転位 30° 以内なら保存可
📝受傷機転 📊分類
Bennett / Rolando 🩺身体所見 🧪画像 💉マネジメント 👨‍⚕️非整形外科医の役割 🏠帰宅時の説明 📚出典
📞 この疾患が該当する主訴
  • 手関節・手の外傷 — 母指軸圧外傷後の CMC 関節圧痛
  • 転倒 — 転倒で母指を突いて受傷

📝 受傷機転

  • 母指に軸圧（殴打・転倒で母指を突く）
  • 若年男性に多い
  • スポーツ外傷（スキー・ボクシング）

📊 分類

分類所見治療
Bennett 骨折関節内、2 骨片（内側小骨片が靱帯で残る + 中手骨本体が外側へ転位）帰宅手術（ORIF または pinning）
Rolando 骨折関節内、粉砕（Y/T 字型）帰宅手術（整復困難例は人工関節）
関節外骨折基部〜骨幹部の関節外保存（許容度大）

🩺 身体所見

  • 母指 CMC 関節周囲の腫脹・圧痛
  • 母指の外転・把握痛
  • 変形：母指が回内位・短縮

🧪 画像

  • 手 3 方向 + 母指特殊撮影
  • 正面像で関節内骨折線を確認
  • CT で骨折線の詳細を評価（手術計画）

💉 マネジメント

Bennett / Rolando 骨折（関節内）

  • 帰宅手術または即時整形
  • 母指スパイカシーネで固定して紹介

関節外骨折

  • 転位角度 30° 以内なら保存可
  • 母指スパイカシーネ固定
  • 3–4 週でリハビリ開始

👨‍⚕️ 非整形外科医の役割

  1. 母指 CMC 関節の圧痛を評価
  2. 画像で関節内進展を確認
  3. 関節内骨折は母指スパイカ固定 + 帰宅手術
  4. 関節外でも転位が大きければ整形外科紹介

🏠 帰宅時の説明

  • 母指は親指の機能要 → 早期診断・治療が機能予後に直結
  • 「母指を使う仕事・趣味」を紹介状に記載
  • 1 週以内に整形外科外来
