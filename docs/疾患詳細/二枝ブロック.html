⚡ これを見たら 二枝ブロック
  1. RBBB + LAFB（左軸偏位）or LPFB（右軸偏位）
  2. 完全房室ブロックへ進行リスクあり
  3. 症状（失神）あればペースメーカ
  4. CT/MRI で器質的疾患（虚血・心筋症・サルコイドーシス）を探索

📞 この疾患が該当する主訴

📖 心電図の読み方

  • 二枝ブロック（bifascicular block）は以下のいずれか
    • RBBB + 左脚前枝ブロック（電軸左偏位、Ⅰ・aVL で小さな q、Ⅱ・Ⅲ・aVF で rS など）
    • RBBB + 左脚後枝ブロック（電軸右偏位など、まれ）
  • これに PR ≥200 ms（Ⅰ度房室ブロック） が重なることがある
「三枝ブロック」の呼称について

二枝 + Ⅰ度房室ブロックを「三枝ブロック」と俗称することがあるが、現行ガイドラインでは推奨されない。PR 延長の多くは 房室結節側の遅延で、脚の第3経路が障害されている確証にはならない。用語にこだわらず、どの枝 + どの房室伝導状態かを分解して扱う。

⚖️ 交代性脚ブロック との違い

パターン意味合い緊急度
二枝ブロック + Ⅰ度脚系に障害 + 房室伝導時間の延長（結節性の成分が混ざりうる）同日相談 が基本
交代性脚ブロック左右脚が時系列で入れ替わる → より広範な His-Purkinje 病変即相談

🚨 アクション

  • 同日循環器コンサルトが基本（入院時心電図 の位置づけ）
  • 次があると 相談の必要性が上がる
    • 失神・前失神・転倒
    • 旧ECG なし／比較不能
    • 多薬併用（房室結節作用薬、抗不整脈薬）
    • 腎障害・透析
    • 急性虚血の疑い
  • 無症状・旧ECG と同一・血行安定なら、主治医と循環器で外来フォロー調整もありうる（初見入院なら循環器に一度は乗せる運用が多い）

失神・前失神があるとき

失神の原因がほかで説明できないときは、循環器で Holter・電気生理の要否を相談。 参考：2017 ACC/AHA/HRS Syncope Guideline <https://www.jacc.org/doi/10.1016/j.jacc.2017.03.003>

📝 同時に整理する

  • 旧心電図と内服確認（旧ECG、内服、既往）
  • 急性虚血の有無 → STEMI
  • K / Mg / Ca、TSH

⚠️ ピットフォール

  • 自動診断の「三枝ブロック」をそのままコンサルト文に書く（RBBB+LAFB+PR延長 と分解して伝える）
  • 無症状だからとフォロー先を決めずに終わる
  • 交代性脚ブロック と混同する
