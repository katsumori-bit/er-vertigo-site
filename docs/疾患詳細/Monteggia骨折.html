Monteggia 骨折

「尺骨骨折を見たら全例で橈骨頭脱臼を探す」 — 尺骨単独骨折にしか見えなくても、radiocapitellar line で小頭中央通過を確認するクセを。必ず正面・側面 2 方向で（片方しか異常が出ないことがある）。PIN 損傷 10〜20%。
同時並行で評価する 3 軸
典型像
  • 尺骨近位部骨折＋橈骨頭脱臼を呈する脱臼骨折
  • 肘外傷の 1% 未満と稀だが、見逃しやすく早期介入しないと機能予後に影響
  • 受傷機転は FOOSH（肘脱臼の代わりに尺骨が折れて橈骨頭だけ脱臼）
  • 好発年齢：4〜10 歳（小学生肘痛では顆上・外顆と並び鑑別）
  • 所見：肘〜前腕近位部の腫脹・変形・激痛、橈骨頭部の圧痛
血管

血管損傷は稀だが、変形強い例では橈骨動脈・尺骨動脈の触知を必ず。

  • 橈骨動脈・尺骨動脈を触知（左右比較）
  • CRT・色調・温度・SpO₂ の確認
  • 6P（Pain・Pallor・Paralysis・Paresthesia・Pressure・Pulselessness）
  • 開放創なら破傷風・抗菌薬・即整形＋血管外科
神経

PIN（後骨間神経／橈骨神経の枝）損傷が 10〜20%。下垂指がサイン、感覚障害なし。

  • PIN 損傷の徴候 = 下垂指（drop finger）：MP 関節の伸展障害。手関節伸展は可能
  • 感覚障害はなし（純粋運動枝）
  • 正中神経・尺骨神経も評価（橈骨頭脱臼で正中も時に圧迫）
  • 整復前後で必ず記録（医原性の責任分界）
画像の見方（Bado 分類 × radiocapitellar line）
  • 肘＋前腕の 2 方向（正面・側面）— 必ず両方撮る
  • 尺骨近位部骨折を見たら橈骨頭脱臼を能動的に探す
  • Radiocapitellar line（橈骨中心線の延長線）：
    • 正常：小頭の中央を通過
    • 異常：小頭の上 or 外側を通過 → 橈骨頭脱臼あり = Monteggia 骨折
    • 正常小児でも 15.6% で中央を外れる → 小児尺骨骨折を見た時に限って引くのがコツ
  • Bado 分類（橈骨頭脱臼の方向）：
    • Type I（70%、最多）：側面像で radiocapitellar line が小頭の上を通過（橈骨頭の前方脱臼）
    • Type III（25%）：正面像で radiocapitellar line が小頭の外側を通過（橈骨頭の外側脱臼）
    • Type II / IV は稀でラインも結局中央を通る → 非整形外科医は I と III の 2 タイプを覚えれば十分
  • 片方しか異常が出ないことがある → 正面・側面の 2 方向で必ず確認
  • 整復前後でレントゲンを 2 枚撮影し、紹介状に添付
治療方針（小児 Monteggia は原則 即時整形）
所見マネジメントER の動き
小児 Monteggia 骨折（Bado I/III） 即時整形コンサルト（脱臼整復＋尺骨整復・固定） 肘 90° 屈曲シーネで仮固定、絶飲食、ライン確保、PIN 評価記録
PIN 麻痺合併（下垂指） 即時整形＋神経評価 即コンサルト、整復前後の所見を必ず記録
open / 神経血管症状 緊急手術 抗菌薬・破傷風・絶飲食・即コンサルト
成人 Monteggia 骨折（稀） 原則手術（観血的整復＋プレート固定） シーネ仮固定、整形外科コンサルト

ER で打つ手

  • 肘 90° 屈曲位の後方シーネ＋三角巾で仮固定（応急処置）
  • 固定だけでの帰宅は不可 — 神経損傷リスクが高く、橈骨頭脱臼の整復が必要
  • 絶飲食、ライン確保、採血・胸部 X 線・心電図（術前準備）
  • 鎮痛はアセトアミノフェン中心、神経ブロックは整形医到着前は避ける（神経評価困難になる）
  • PIN 麻痺（下垂指）の有無を必ず記録（整復前後で）
  • 家族へ：神経合併症の説明、術後経過の説明

関連ハブ・隣接疾患
  • 主訴入口：肘周囲の外傷 ／ 転倒

  • 合併症：開放骨折 ／ コンパートメント症候群
