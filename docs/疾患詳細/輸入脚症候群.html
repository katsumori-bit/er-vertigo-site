輸入脚症候群（Afferent Loop Syndrome）

⚡ 即時アクション
  1. Billroth-II / Roux-en-Y 術後の合併症
  2. 腹部膨満 + 胆汁性嘔吐 + 上腹部痛（食後数時間）
  3. CT・上部消化管造影で診断
  4. 保存的（NPO + IVH）or 内視鏡 / 手術

📞 この疾患が該当する主訴
  • 腹痛 — 持続する強い上腹部痛（胃手術歴必須）
  • 嘔気・嘔吐 — 慢性型では胆汁性嘔吐、急性型では出にくい

🧠 概要（疫学・病態）

特殊型の腸閉塞。Billroth II 法（B-II 再建）後の 1% に発症（米国データ）とされるが現代ではその頻度はかなり低い印象。B-II 再建が減ったことと、H2 ブロッカーなど制酸薬の出現で潰瘍治療としての胃切除自体が減ったことが背景。

🫀 解剖と生理

  • 胃手術後の再建小腸は輸入脚（afferent loop）と輸出脚（efferent loop）の 2 系統
  • 輸入脚が吻合部付近で狭窄／閉塞して十二指腸液（胆汁＋膵液）の流れが滞ることで症状発現
  • 慢性型：思い出したように噴出する胆汁性嘔吐／急性型：吻合部完全閉塞で輸入脚が closed loop。本項は急性型を扱う
十二指腸の口側が断端となっている再建ならいずれも輸入脚を持つ。幽門側胃切除後 B-I 法 or 胃全摘後の小腸置換以外のすべての胃手術（B-II・B-II + Brown・Roux-Y・gastric bypass）で発症しうる。
輸入脚は空腸だが食物は通らず、胆汁＋膵液（1,000〜2,000 mL/日）の導管として機能している。

輸入脚が閉塞すると行き場を失った胆汁・膵液で十二指腸〜輸入脚がパンパンに膨れる（closed loop の完成）。持続すれば二次性に膵管拡張・急性膵炎を併発する。胆管の拡張・閉塞性黄疸はあまり経験しない。

📝 病歴

  • 胃の手術歴がなければ発症しない（30 年前の十二指腸潰瘍手術＝幽門側胃切除のことも、近年は減量手術（gastric bypass）後にも発症報告あり）
  • 慢性輸入脚症候群（繰り返す腹部違和感＋胆汁性嘔吐）が先行することもあるが、多くは acute onset の持続する強い上腹部痛として発症
  • 持続すれば膵炎併発で背部痛も
「腸閉塞」だが「腸閉塞症状」を呈さない
腸管（輸入脚）が閉塞して起こる疾患だが、食べ物が通る道としての消化管には閉塞がないため：
  • 吐き気・嘔吐は出にくく、出ても水様の大量嘔吐ではない
  • 排便排ガスは消失しない
これが診断を難しくする。

🩺 身体所見

  • 主に十二指腸が緊満するため右上腹部に限局性に乏しい圧痛
  • 初期は反跳痛・筋性防御なし。出ていれば輸入脚が破裂寸前かもしれない
  • 拡張しているのは輸入脚だけなので腹部膨満は見られない
  • 腸雑音は通常異常なし（パツンパツンの輸入脚はガスがなく音がしない、それ以外の正常腸管音が聞こえる）
  • 上腹部に手術創を認めたら手術歴を執拗に確認

🔍 初期診断と鑑別

  • 稀疾患のため初期診断の筆頭にあがるのは難しい
  • 強い上腹部痛として① 上部消化管穿孔、② 急性膵炎を念頭に置きつつ、これらを否定／併存していないかを考える
  • 急性膵炎は本疾患の二次性合併症として出現するため、胃術後例で急性膵炎を見つけた場合に輸入脚症候群を見過ごさないことが重要 — 膵炎と診断して保存治療を続けたら治療が遅れたという報告がある

🧪 検査と確定診断

血液

  • 特異所見なし。膵炎合併で膵酵素（アミラーゼ・リパーゼ）上昇

単純 X 線

  • 胸部立位で穿孔のフリーエアをチェック
  • 輸入脚 closed loop 内は腸液充満でガスなし。狭義の消化管にも閉塞がないので小腸ガスも溜まりにくい → 全体的にガスレス、上腹部（特に右側）の X 線透過性低下

腹部 CT — gold standard

  • 十二指腸断端から輸入脚吻合部までの腸管のみが拡張している像
  • 急性膵炎を併発していると膵炎に目を奪われがちだが、麻痺性イレウスはガスを含むのに対して輸入脚 closed loop はガスレスなので区別は容易
  • 「拡張した小腸（断面が正円形）を見つけたら必ず端まで追う」のが鉄則

腹部超音波

  • 拡張腸管はガスがないので超音波観察に適している
  • 輸入脚 closed loop は完全ガスレスなので走査が容易、慣れた人なら十分診断可能

上部消化管造影や内視鏡は一般的検査ではない（輸入脚は閉塞がなくても造影されないことがあり、内視鏡では Brown 吻合・Roux-Y では輸入脚に到達しない）。

💉 治療

  • 緊急手術が基本。閉塞部解除＋必要なら輸入脚再建（吻合）
  • 頻度の高い B-II 法後 → 胃空腸吻合部分を切除して Roux-Y 再建に変更するのがよく行われる
  • 一般的でないが内視鏡的ドレナージ／経皮的輸入脚ドレナージ／PTCD 経由のドレナージが報告あり。一時しのぎや腫瘍性閉塞の緩和目的

⚠️ 予後とピットフォール

早期診断が治療のすべて。治療開始が遅れると致死的（死亡率 57% との報告）。死因は輸入脚の循環不全 → 壊死か破裂 → 腹膜炎。

最大のピットフォール — 「二次性膵炎」を「原発性膵炎」と誤診
強い上腹部痛＋膵酵素上昇＋CT で膵周囲液貯留 → 「急性膵炎」と判断して保存治療を続けると詰む。
「いつもの膵炎と違う！」と感じる手がかり：
  • 膵炎の etiology がはっきりしない（胆石でもアルコールでもない）
  • 慢性膵炎の既往がないのに膵管が拡張している
  • 十二指腸水平脚が拡張している（通常の腸閉塞では見られない）
胃手術歴の聴取と CT での「拡張した小腸を端まで追う」習慣がすべて。
