大腸閉塞

⚡ 即時アクション
  1. 腹部 X 線 ± CT で diameter 評価：大腸短軸 > 5 cm が拡張、盲腸径 > 8 cm で破裂リスク高（盲腸壁にラプラスの定理で最強張力）
  2. 閉塞部位を特定：右側（横行結腸まで）vs 左側（下行結腸〜）。小腸ガスの有無で回盲弁機能（competent / incompetent）を評価 → competent + 右側は破裂リスク
  3. NPO + NG チューブ + 補液：incompetent valve なら口側からドレナージ可、症状落ち着けば翌日処置でも可
  4. 緊急手術（穿孔・絞扼・盲腸 > 8 cm）vs 内視鏡的減圧（コロレクタールチューブ／ステント）vs 待機手術を判断。competent valve + 切迫破裂は肛門側アプローチか手術

📞 この疾患が該当する主訴
  • 腹痛 — 右下腹部痛が典型（盲腸壁に最強張力）。閉塞部位と一致しないことに注意
  • 嘔気・嘔吐 — incompetent valve では小腸閉塞様の嘔吐、competent valve では出にくい
  • 発熱 — 穿孔・腹膜炎合併で出現（重症化サイン）

📂 分類

要点：成因 × 形態で分類するのは小腸と同じ。ただし形態は「回盲弁の機能」と「閉塞位置（右側／左側）」の 2 軸で考える。

① 成因

  • 内因性：ほとんどが大腸癌。良性疾患では大腸憩室（炎症後狭窄）など
  • 外因性：癒着性は起こらない（大腸は間膜が乏しく壁が厚い）。捻転（S 状結腸 ≥ 盲腸 > 横行結腸）／腸重積（多くは大腸癌が先進部）／壁外圧迫・浸潤（胃癌・膵癌・骨盤内腫瘍など）
  • 管腔内異物：糞便嵌頓（fecal impaction）など

② 形態

大腸閉塞は基本的に closed loop obstruction。回盲弁が機能している限り大腸 → 小腸への逆流が起きないため、閉塞 1 カ所だけで closed loop が成立する。
ただし高齢者などで回盲弁機能が低下している場合は遠位の小腸閉塞と同じ形態になる。
  • 回盲弁機能：competent（正常）か incompetent（機能不全）か
  • 閉塞位置：「右側」（横行結腸まで）か「左側」（下行結腸から）か。境目は体の中心ではなく横行結腸

📝 病歴

要点：回盲弁が competent な間は症状が乏しい。閉塞位置（右／左）と弁機能で訴え方が変わる。

  • 左側閉塞 + competent valve：来院数週間前から血便・便通異常などの自覚症状をもつことが多い
  • 右側閉塞 + competent valve：前日まで無症状・快便だったというパターンも珍しくない
  • incompetent valve：遠位の小腸閉塞と同じ症状（軽度腹痛・腹部膨満が徐々に進行）。ただし最初から incompetent のケースは少なく、大腸拡張が進むにつれて機能不全になるため初期症状は乏しい
  • 閉塞が顕著化すると、閉塞部位に関係なく右下腹部痛を自覚 — ラプラスの定理で内径最大の盲腸壁に最強の張力がかかるため。「虫垂炎」と早期診断されることもあるし、常に盲腸穿孔の危険を意味する

🩺 身体所見

  • 遠位大腸閉塞では腹部膨満が著明になり、視診で明らかに膨隆していることが多い
  • 触診の圧痛も上記理由で盲腸部にあり、必ずしも病変部と一致しない
  • 大腸の走行に沿って腫瘤を探る。膨満が強ければ腹側からはわかりにくい → X 線で部位の見当がつけば、側腹部から（腹膜のない側面を触れるイメージ）触診するのも一法。痩せた人を除き相当深く触らないと触れない
  • 直腸診を忘れない。直腸診で容易に触れる下部直腸癌が内視鏡や CT で初めて指摘されたら、診察態度を改めるべき

🔍 初期診断と鑑別疾患

要点：身体所見だけでの診断は困難。とくに incompetent valve の大腸閉塞と遠位小腸閉塞は臨床的に区別がつかない。

大腸閉塞（incompetent valve）を疑う手がかり

  • 小腸閉塞にしては嘔吐量が少ない
  • 小腸閉塞にしては腹部膨満が強い
  • 結腸（と思われる部位）あるいは直腸に腫瘤を触れる

competent valve では

  • 悪心嘔吐が出にくい
  • 左側閉塞 → 腹部膨満＋右下腹部痛
  • 右側閉塞 → 右下腹部痛のみ

鑑別疾患

  • 急性虫垂炎（盲腸部の右下腹部痛で誤診されうる）
  • Toxic megacolon（C. difficile 関連症／潰瘍性大腸炎など）
  • 腹痛が顕著でない場合：Ogilvie 症候群（偽性大腸閉塞）／慢性大腸拡張（精神科薬の常用・高齢者）

🧪 検査

血液

  • 疾患特異的検査はなし。多くが高齢者なので全身評価目的（造影 CT 前の腎機能、大腸癌疑いでの貧血チェックなど）

腹部単純 X 線（最重要）

多くのケースで単純 X 線で部位診断と緊急性が判断できる。見るべきは 2 点：

  • 大腸ガスの分布：左側閉塞では大腸ガスが目立つ、右側閉塞は内部に泥状便が充満してガスは目立たない（よく見ると X 線透過性低下部位に小ガスを含む）。大腸の拡張は短軸径 > 5 cmが目安
  • 小腸ガスの有無 ＝ 回盲弁機能の評価：小腸ガスがほぼなし → competent、ガスで拡張した小腸あり → incompetent
閉塞部位＼回盲弁CompetentIncompetent
右側ガスレスフィルム（右側結腸の透過性高、小ガス）大腸ガス目立たず、小腸ガスあり → 遠位小腸閉塞と区別困難
左側大腸ガス目立つ、小腸ガス認めず大腸ガス・小腸ガスともに目立つ

腹部超音波

  • 左側閉塞はガスが多く向かない
  • 右側閉塞では泥状便が充満して拡張大腸がよく見える。さらに肛門側に追えば閉塞起点となる腫瘍性病変まで指摘できることが多い → X 線で右側閉塞が確認できたら積極的に施行（CT 前に大腸癌を指摘できる）

腹部 CT（造影）

  • 大腸閉塞そのものの評価のほか、大腸癌の診断（造影効果のある全周性壁肥厚像）が可能

⚠️ 確定診断と緊急性の評価

要点：大腸閉塞の存在は画像で確定する。問題は「すぐ減圧しないと破裂するのか、翌日まで待てるのか」の判断。

破裂を危惧すべき状態（Emergency）
  • 嘔吐がない
  • 間欠的な腹痛がない（＝小腸閉塞症状なし＝competent valve）
  • 右下腹部に反跳痛がある（破裂は盲腸に起こり、まず漿膜が裂ける → 軽い反跳痛は漿膜が裂けている可能性を示唆）
  • 盲腸径 > 8 cm
  • 画像上小腸の拡張がない（＝competent valve）
  • 右側閉塞（左側より閉塞腸管が短く圧の逃げ場が少ない）
切迫破裂はなく少し待てる状態（Urgency）
  • 水様性吐瀉物の嘔吐がある
  • 間欠的な腹痛がある（＝小腸閉塞症状＝incompetent valve）
  • 軽い圧痛以上の腹部所見はない
  • 画像上小腸が拡張している（＝incompetent valve）

💉 治療

① Incompetent valve の場合

  • 多くは切迫破裂の状態にない。小腸閉塞同様口側からドレナージ可能 → NG チューブで症状が落ち着けば、処置は翌日以降でも可

② Competent valve で切迫破裂の場合

  • 早急な閉塞部ドレナージが必要。口側からのドレナージは不能（弁が逆流を防ぐ）→ 肛門側からのアプローチか手術

ドレナージの方法

  • 内視鏡的減圧：
    • コロレクタールチューブ挿入(一時的減圧 → 手術への橋渡し)
    • ステント挿入（一時的 or 永久的に手術不能の場合）
  • 手術による減圧：
    • 右側閉塞：ほとんどのケースで右半結腸切除術（一期的吻合）が安全
    • 左側閉塞：1 期的（subtotal colectomy／on-table irrigation + segmental resection）か、2 期的（loop colostomy → 切除吻合 + ストーマ閉鎖／Hartmann → ストーマ閉鎖／切除吻合 + loop ileostomy → ileostomy 閉鎖）
    • 不安定／免疫抑制／経験の少ない施設では2 期的を選択。1 期的の安全性報告は大腸閉塞を多数手がける施設発が多く、たまたま遭遇した症例で 1 期的を選ぶのは慎重さに欠ける
    • 状態の悪い直腸閉塞では 3 期的（横行結腸ストーマ → 低位前方切除 → ストーマ閉鎖）

内視鏡的減圧で待機手術にできれば最も安全かつ低侵襲。

➕ その他の大腸閉塞

結腸捻転（Volvulus）

  • 頻度：S 状結腸 ≥ 盲腸 > 横行結腸
  • S 状結腸軸捻転は自力排便に問題があるレベルの高齢者に多い。本人の訴えより周囲の発見（痛がっている／膨満／嘔吐）が来院契機
  • 身体所見の高度膨満＋典型的 X 線（coffee bean sign）で診断は難しくない。X 線では 1/3 で見逃しがあり CT が必要との意見
  • S 状結腸で発症早期なら大腸内視鏡による整復が一般的。それ以外は結腸切除
  • 反跳痛がある場合は内視鏡禁忌（穿孔・敗血症誘発のリスク）
  • 緊急手術はハルトマン手術が伝統的だが、壊死・腹膜炎なしなら切除吻合でよい意見もある
  • 再発率が高いため、内視鏡整復成功例も待機的切除を推奨

腸重積（Intussusception）

  • 成人の腸重積は先進部が腫瘍であることが多く、切除手術が原則
  • 先進部となるのは小さく可動性のある腫瘍。大きな進行癌では起こりにくい（漿膜浸潤で周囲を引き込んで重積様に見えるが、実際は閉塞しておらず無症状＝pseudo intussusception）
  • 「ターゲットサイン」だけで腸重積と判断しない。診断には腸間膜が腸管内に入り込んでいる所見が必要。「無痛で閉塞のない腸重積」は説明に無理がある

糞便による閉塞（Fecal impaction）

  • 高齢者・基礎疾患・腸蠕動低下薬の服用などの素因がないと起こらない
  • 大腸に便があるからといって痛みの原因と直結しない。広範結腸壊死や下部消化管穿孔を「糞便イレウス」と誤診する例が現にある
  • 診断には先進部の硬便＋口側大腸の拡張（> 5 cm）が必要
  • 直腸に充満する便塊は多くの場合「自力排便ができていないレベルの ADL／リスクが高い患者」という意味しか持たない
