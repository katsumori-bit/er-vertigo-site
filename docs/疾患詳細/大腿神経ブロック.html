#神経ブロック #大腿神経 #鎮痛 #ER #エコーガイド

大腿神経ブロック = 大腿骨近位部・骨幹部骨折の超強力な鎮痛
  • 大腿骨近位部骨折・骨幹部骨折 の疼痛を劇的に軽減
  • エコーガイド下 で 10–20 mL の局麻薬
  • オピオイド依存を減らし、高齢者のせん妄を予防
  • ER でできる基本手技、整形外科オーダー前に実施

適応

  • 大腿骨近位部骨折（頸部・転子部）
  • 大腿骨骨幹部骨折
  • 膝関節前面・大腿前面の疼痛コントロール
  • 中〜高度の鎮痛が必要な下肢外傷

なぜやるべきか

メリット内容
強力な鎮痛オピオイドと同等以上、持続 6–12 時間
オピオイド節約高齢者のせん妄・呼吸抑制を避ける
安全エコーガイドで合併症少ない
血行動態への影響が少ない高齢者でも安全に実施可
整復・牽引時の鎮痛シーネ固定・体位変換時に有用

解剖

  • 大腿動脈の外側に大腿神経が走行
  • 鼠径靱帯の下方を走る
  • 腸腰筋・大腿神経・大腿動脈・大腿静脈の位置関係（NAVEL: Nerve, Artery, Vein, Empty space, Lymphatics の内側から）

手技（エコーガイド下）

準備

  • 18–20G 針、エコーリニアプローブ
  • 1% リドカイン 10–20 mL または 0.25% レボブピバカイン 10–20 mL
  • 消毒・ドレーピング

手順

  1. 仰臥位、患側下肢を軽度外旋
  2. 鼠径靱帯直下にプローブを横断面で置く
  3. 大腿動脈を同定 → その外側の大腿神経を確認
  4. out-of-plane or in-planeで穿刺
  5. 大腿神経の周囲（神経外）に 10–20 mL 注入
  6. 注入前に 吸引で血液混入がないか確認（大腿動脈穿刺に注意）

筋膜下法（Fascia iliaca block）

  • 大腿神経ブロックの簡便版
  • 腸腰筋上の筋膜下に 20–30 mL 注入
  • エコーなくても landmark で可能
  • 大腿神経 + 外側大腿皮神経 + 閉鎖神経 を一度にブロック

合併症

合併症対応
大腿動脈穿刺圧迫止血、経過観察
局麻中毒上限量を守る、脂肪乳剤（Intralipid）準備
神経損傷エコーで神経内穿刺を避ける
感染清潔操作

効果判定

  • 10–15 分後に大腿前面の感覚低下
  • 疼痛 NRS 8 → 2–3 程度まで減少
  • 持続時間：リドカイン 2–3 時間、ブピバカイン 6–12 時間

禁忌

  • 穿刺部位の感染
  • 抗凝固薬使用（相対的、INR > 1.5 など）
  • 局麻薬アレルギー
  • 患者が強く拒否

ER でのタイミング

  1. 受傷評価 → 大腿骨骨折確定 → 即実施
  2. レントゲン撮影のための体位変換前
  3. シーネ固定・牽引前
  4. 整形外科コンサルト前でも OK
