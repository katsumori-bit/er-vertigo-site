QT延長

⚡ これを見たら QT 延長
  1. QTc > 500 ms で TdP リスク高
  2. TdP（torsade de pointes）= 多形性 VT、R-on-T で発症
  3. 薬剤（QT 延長薬）・電解質（K, Mg, Ca）を確認
  4. Mg 静注（2 g IV）、ペーシング、isoproterenol が治療

📞 この疾患が該当する主訴
  • 意識消失（失神） — TdP による失神は突然死リスク、即モニター
  • 入院時心電図 — QTc ≥ 500 ms / +60 ms 延長の発見契機

📏 測定と閾値

  • QT は 最も長く見える誘導（多くは Ⅱ、V5）で測る
  • 補正式：Bazett（QT / √RR）が最一般。心拍数極端なら Fridericia を併用
  • U 波が大きいときは QT と混同しない
QTc（ms）解釈
男性 <450、女性 <460正常
男性 ≥450、女性 ≥460境界〜延長

ベースラインから +60 ms新規延長として警戒

自動QTcを鵜呑みにしない。U 波を拾ったり、T 波終末を誤検出することがある。

🚨 即相談のトリガー

循環器相談
  • QTc ≥500 ms + 誘発因子（薬剤 / 電解質）
  • 失神・前失神・動悸を伴う
  • モニターで多形性VT（Torsades）
  • 先天性QT延長症候群疑い（若年 / 家族歴）

💊 誘発薬・誘発状態

薬剤（代表）

  • 抗不整脈薬：Ⅰa（キニジン、ジソピラミド）、Ⅲ（アミオダロン、ソタロール、ニフェカラント）
  • マクロライド、フルオロキノロン、アゾール系抗真菌薬
  • 抗精神病薬（ハロペリドール、クエチアピン等）、一部抗うつ薬
  • メサドン、オンダンセトロン、ドロペリドール
  • 抗ヒスタミン薬（一部）

全リストは CredibleMeds を参照：<https://crediblemeds.org/>

電解質・状態

  • 低K、低Mg、低Ca
  • 徐脈（pause-dependent TdP）
  • 低体温、甲状腺機能低下
  • 急性脳血管障害（特にくも膜下出血）
  • 急性冠症候群

🏥 救急での対応

  • モニター継続、除細動器をベッドサイドへ
  • K 4.0 以上、Mg 2.0 mg/dL 以上 を目標に補正
  • 誘発薬の中止 / 変更（可能なもの）
  • TdP 発作：
    • 硫酸マグネシウム 1–2 g 静注（発作再燃予防に持続追加可）
    • 徐脈依存なら心拍を上げる：イソプロテレノール、経静脈ペーシング
    • 血行動態不安定なら直流除細動

🔍 鑑別

  • 大きな U 波を QT に含めて計測ミス
  • 心室内伝導障害で QRS が広くなっていることによる見かけ上の延長
  • Brugada心電図（V1–V2 を別途評価）

⚠️ ピットフォール

  • 自動計測のみで判断する
  • QTc 470 ms で「正常寄り」と流す（誘発薬が複数なら警戒）
  • 低K・低Mg を「軽度」と見なす
  • 多科処方の薬剤相互作用を確認しない
