腹痛をきたす胸部疾患

⚡ 即時アクション
  1. 心筋梗塞（特に下壁）、急性肺塞栓、肺炎下葉、気胸 などを心電図・CXR で除外
  2. 心電図 + トロポニン + D-dimer + CXR をルーチンに（食後心窩部痛・嘔吐後上腹部痛・腹痛+呼吸器症状）
  3. 腹痛を訴える胸部疾患は ER で見逃しやすい（食事誘発の AMI、痛みのない解離、Boerhaave、背側膿胸）
  4. 右心室梗塞による腹痛・肺塞栓による腹痛・MI による上腹部痛を念頭。ハイリスクは「胸部 + 腹部」両方検索
❤️心筋梗塞
不安定狭心症 🩸大動脈解離 💨Boerhaave 症候群
食道破裂 🫁肺炎・胸膜炎・膿胸
📞 この疾患が該当する主訴
  • 腹痛 — 腹腔内臓器が原因とは限らない。胸部疾患でも腹痛を主訴とすることあり
  • 胸痛 — 同時に胸痛を伴うことも
  • 呼吸困難 — 肺炎・膿胸・PE による腹痛 + 呼吸器症状
  • 嘔気・嘔吐 — 下壁 MI で吐き気主体／Boerhaave は嘔吐後の上腹部痛

❤️ 心筋梗塞／不安定狭心症

  • 労作性は典型的だが、食後に症状が顕著化する冠血管疾患は相当数ある
  • 機序：腹腔内臓器は空腹時に血流少、食事で急激に血流増 → このシフトで冠血流が低下
  • リスクファクターを有する例で食事中・食後に始まった心窩部痛、急な吐き気には心筋梗塞の否定から始める
ハイリスク患者では「心筋梗塞もある／腸管虚血もある」ということもある。ST 上昇しているから腹部疾患ではないとは限らない。

🩸 大動脈解離

  • Stanford A 型は発症 1 週間以内 mortality > 50%（AMI の 30% より高い）
  • 腹部大動脈まで解離が及んでいても症状の多くは背部痛。腹痛を訴えるときは解離そのものの痛みではなく解離に伴う臓器虚血を念頭に
「胸背部痛がない解離」の落とし穴
  • 一過性意識消失で来院し、現時点では何も症状がない高齢者は要注意。解離による頸動脈一過性低灌流で意識消失発作の症状となる。安易に帰宅させると次は CPA で来るかも
  • 痛みのない解離もある
  • 早期血栓閉塞型：偽腔が早期に血栓で閉塞してフラップが見えず「解離ではない」と判断される。石灰化の位置に注目（壁内血栓 → 石灰化は外側／解離 → 内腔側）。単純 CT で早期血栓は CT 値上昇で鑑別できる → 血管病変 CT は「単純＋造影」が原則

「A 型は手術／B 型は手術不要」の紋切り型は危険。B 型でも腹腔内臓器への主要動脈閉塞があれば緊急手術。とくに上腸間膜動脈閉塞が問題（→ 急性腸管虚血参照）。

💨 特発性食道破裂（Boerhaave 症候群）

  • 稀な疾患だが初期診断の遅れは即死亡率上昇
  • 罹患部位は通常胸部だが、上腹部痛・左季肋部痛として表現されることがある
  • 強い嘔吐の後の急激な上腹部痛では念頭に

🫁 肺炎／胸膜炎／膿胸

  • 膿胸は知っておいて損がない疾患。背側病変では支配領域の肋間神経が腹部に達するため、強い腹痛として自覚することがある
  • 肺炎による腹痛は小児に多い
  • 上腹部外側・側腹部痛で腹部臓器に異常がはっきりせず高熱がある場合は鑑別に
  • 原著筆者は虫垂炎と診断されて実は膿胸であった経験を複数回している
