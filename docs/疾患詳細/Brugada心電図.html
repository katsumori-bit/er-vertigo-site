📞 この疾患が該当する主訴
  • 意識消失（失神） — Brugada 失神は突然死の前兆、循環器即相談
  • 入院時心電図 — V1-3 の coved 型 ST 上昇で発見されることが多い

📈 Type 1（診断的）

  • V1 および / または V2 で、J 点から ≥2 mm の ST上昇
  • coved 型（上に凸、下行性）、続いて 陰性T
  • 高位（第2肋間）記録でより出やすいことがある

📉 Type 2（saddle-back）

  • V1–V2 で 鞍型、J 点 ≥2 mm、T 波は陽性または二相性
  • 単独では診断にならない（type 1 へ誘発されるか評価が必要）
Type 2 の扱い

Type 2 心電図 + 強い臨床（失神・家族歴）なら循環器相談。薬剤誘発試験は専門科判断。

🔥 顕在化させる因子

  • 発熱（最頻）
  • Na チャネル作用薬：抗不整脈薬（Ⅰa / Ⅰc）、三環系抗うつ薬、精神科薬（例：リチウム、一部抗精神病薬）、局所麻酔薬
  • 電解質異常（高K、低K、高Ca）
  • 迷走神経緊張、アルコール多飲、コカインなど交感神経作動薬

🚨 アクション

無症状で初めて見つけた Type 1

  • モニター継続
  • 発熱があれば積極的に解熱（アセトアミノフェン）
  • 誘発薬の中止・変更を検討（精神科処方などは単独中止せず相談）
  • 循環器 / 不整脈 コンサルト（同日〜翌日）

失神 / 心室性不整脈 / 家族突然死歴あり + Type 1

即相談
  • 循環器オンコール
  • 入院・モニター
  • ICD の議論は循環器

🔍 鑑別（Brugada 様に見えるもの）

  • 右脚ブロック（V1 の形）
  • 右室肥大、右室負荷
  • 急性肺塞栓
  • 早期再分極
  • 低体温（J / Osborn 波）
  • 心膜炎 / 心筋炎
  • 高K血症 → 高K血症心電図

⚠️ ピットフォール

  • 発熱を下げずに「様子を見る」
  • Type 2 を「Brugada ではない」として流す（高リスク背景なら循環器へ）
  • 処方薬を全部止めて精神科との調整を壊す
