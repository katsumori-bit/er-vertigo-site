旧心電図と内服確認

⚡ これを見たら○○
  1. 過去の心電図と必ず比較（変化あれば異常を疑う／旧同一なら陳旧の可能性）
  2. 内服薬の影響（β-blocker・Ca-blocker・ジゴキシン・抗不整脈薬・QT 延長薬）を確認
  3. ペースメーカ・ICD の確認（spike の有無、デバイス下心電図の解釈）
  4. 電解質補正薬（K 製剤・利尿薬）の使用歴 → 高K / 低K / QT 延長を疑う

🌙 夜間・休日で現実的な情報源

本人が話せなくても、順に当たる。

  1. 家族・同伴者（普段の心電図・ペースメーカ・飲んでいる薬の名前）
  2. お薬手帳・写真・調剤薬局（夜間でも写真があれば十分なことが多い）
  3. 当院 EMR（過去入院・外来・健診）
  4. 紹介状・他院サマリ・画像添付
  5. 救急隊の記録（搬送時バイタル・現場12誘導があれば宝）
電話前に1枚

循環器にかける前に、一番古い旧ECG が1枚でも手元にあると会話が早い。

📄 旧心電図（優先順）

  1. 当院 EMR（過去入院 / 外来）
  2. 紹介状添付（PDF / 画像 / CD）
  3. 健診・人間ドック
  4. 救急搬送時・別日の記録

判定の書き方：

  • 完全一致 → 陳旧所見の可能性。緊急度は下がりやすい（臨床悪化があれば別）
  • 微妙に違う / 画質が悪い → 新規断定を避ける。同日フォロー
  • 旧なし → 初見として扱う（最多）

📋 既往・背景（心電図とセットで聞く）

「内服」と同列で、短いYes/Noで回す。

項目なぜ聞くか
虚血性心疾患・PCI・CABG 既往ACS・再狭窄の文脈
心不全・心筋症・肥大型心筋症ST/T・QRS の解釈、血行の余裕
不整脈・カテ・アブレーション・ICD/ペースデバイス下の心電図、薬剤
失神・転倒の反復伝導障害・QT・Brugada の解釈を変える
家族の突然死（若年）チャネル病・遺伝性不整脈
透析・CKD 4–5 期高K、薬剤、造影の話
最近の発熱・感染心筋炎、Brugada 顕在化
甲状腺疾患AF・徐脈の可逆原因
妊娠の有無（該当すれば）ACS・循環動態の扱いが変わる

紹介状に書いてあれば 本人に聞き直さなくてよい ところは省く。

💊 内服・直近の変更

房室結節・徐脈に関わる

  • β 遮断薬
  • 非DHP Ca 拮抗薬（ベラパミル、ジルチアゼム）
  • ジゴキシン
  • アミオダロン
  • イバブラジン
  • クロニジン
  • オピオイド（大量）

QT を延ばす

  • 抗不整脈薬 Ⅰa / Ⅲ
  • マクロライド、フルオロキノロン、アゾール系
  • 抗精神病薬（ハロペリドール、クエチアピン等）
  • オンダンセトロン、ドロペリドール、メサドン
  • 一部の抗うつ薬、抗ヒスタミン薬

詳細リストは QT延長。

Na チャネル作用薬（Brugada 顕在化）

  • 抗不整脈薬 Ⅰa / Ⅰc
  • 三環系抗うつ薬
  • 一部の精神科薬、局所麻酔薬

詳細は Brugada心電図。

電解質・腎に効く

  • 利尿薬（ループ / サイアザイド / スピロノラクトン）
  • RAS 阻害薬
  • 瀉下薬
  • K補充、Mg補充

🧪 最小ラボ

迷ったら揃える：

  • K / Mg / Ca
  • トロポニン（ACS 灰ゾーン）
  • Cr、eGFR
  • TSH（新規 AF、徐脈）
  • 必要に応じて 血糖、CK、BNP

🎯 アクションの型（コピペ用）

  • 旧同一 + 無症状 + 血行安定 + 既往も変わらず → 主治医判断で経過。伝導障害は翌日までに循環器OPD予約
  • 旧なし or 比較不能 + 即相談所見（入院時心電図）→ 循環器。夜間はオンコール
  • 旧なし + 灰ゾーン → トロポニン動態 + 翌朝再12誘導 + 同日循環器
  • 内服が濃い or 透析・CKD → 高K / QT を先に疑い、循環器と並行で腎内科・ICUも選択肢
  • 非内科医で判断に自信がない → 相談してよい（電話用メモが揃っていれば相手も助かる）

📞 循環器への電話用・一行メモ（例）

80M / 転倒入院予定 / 旧ECGなし / DM・CKD3 / ベラパミル内服 / K未 / ECGで完全房室ブロック・心室率38・狭いQRS

⚠️ ピットフォール

  • 旧ECGを探さずに「新規」と電話する
  • お薬手帳が手元にないからスキップする（家族・調剤薬局・EMR）
  • 既往（特に 透析・デバイス・心筋症）を聞かない
  • 自動計測の QTc、自動診断の「AF」を鵜呑みにする
  • 夜間相談で「旧ECG未確認・内服不明」と言わずに依頼する（不明と正直に言う方が安全）
