BPPV（良性発作性頭位めまい症）

⚡ 即時アクション
  1. 頭位変換で誘発される短時間（数秒〜1分）回転性めまい＝ t-EVS の代表
  2. Dix-Hallpike 陽性（眼振方向と潜時を確認） → Epley 法で治療
  3. 画像検査は原則不要。前庭抑制薬を漫然と使わない
  4. 非典型例（眼振パターン非典型 / 反応乏しい / 持続性）は中枢性も鑑別

📞 この疾患が該当する主訴
  • めまい — 三半規管 BPPV ノード（t-EVS / 誘因あり）から到達

🎯 典型像（診断の目安）

BPPV を疑うパターン

  • 体位変換で誘発される回転性めまい（寝返り、起き上がり、上を向く）
  • 持続時間 数秒〜1分（最大でも数十秒）。発作間は無症状
  • 悪心・嘔吐を伴うことが多いが、聴力低下や耳鳴は伴わない
  • 神経脱落症状なし、歩行は通常可能

関与する半規管

  • 後半規管 BPPV：圧倒的多数（>80%）。Dix-Hallpike で誘発
  • 水平（外側）半規管 BPPV：Supine roll test で誘発
  • 前半規管 BPPV：稀
「回転性 = 末梢」の罠

後方循環梗塞も回転性めまいで来ることがある。**持続時間と誘発性**で BPPV を疑うこと。質ではなく文脈で分類する（TiTrATE）。

🩺 Dix-Hallpike 法（後半規管 BPPV の誘発）

手技

  1. 患者を診察台の長軸上で座位にし、頭を検査側に 45°回旋させる
  2. 素早く仰臥位にし、頭部を20-30° 後屈（診察台端から下げる）させる
  3. 30-60 秒間、眼振と症状を観察
  4. 反対側でも同様に行う

陽性所見（典型）

  • 潜時 1-30 秒後に出現する眼振
  • 上向き + 患側回旋成分（後半規管）
  • 持続 1分以内に消失
  • 反復で減衰（疲労現象）
  • 強い回転性めまい・悪心を伴う
中枢性を疑うパターン
  • 潜時なし・持続性の眼振（疲労せず）
  • 純粋な垂直または回旋のみの眼振
  • めまいなしで眼振のみ

これらは中枢性病変（脳幹・小脳）を示唆。後方循環梗塞の鑑別へ。

🔄 Epley 法（耳石置換法・後半規管）

手技

  1. STEP 1：座位、頭を患側に 45°回旋（Dix-Hallpike 開始位置）
  2. STEP 2：素早く仰臥位＋頭部を後屈（眼振が出るまで or 30秒保持）
  3. STEP 3：頭を反対側に 90°回旋（合計 90° 回転）して 30秒保持
  4. STEP 4：体ごと反対側に転がり、頭をさらに 90°下方へ向け 30秒保持
  5. STEP 5：ゆっくり起き上がり座位に戻す

効果と注意

  • 1 回で 70-80% 改善。改善不十分なら再施行可
  • 施行直後 24 時間は強い動作を避けるよう指導
  • 頸椎症や椎骨脳底動脈不全症が疑われる場合は慎重に
改善確認

施行後に再度 Dix-Hallpike を行い、眼振陰性化を確認すると説得力ある説明ができる。

↔️ 水平半規管 BPPV：Supine roll test

手技

  1. 仰臥位で頭を 30° 持ち上げる
  2. 頭を左右に 90° 素早く回旋させ、それぞれ眼振を観察

陽性パターンと治療

  • Geotropic（地向性）型：強い側が患側 → Gufoni 法またはLempert 360° roll
  • Apogeotropic（反地向性）型：弱い側が患側 → 同様に Gufoni / Lempert

手技の詳細は耳鼻咽喉科コンサルト or 専門ガイドラインを参照。

📤 disposition

帰宅可能な条件

  • 典型的 BPPV で Epley 後に明らかな改善
  • 歩行可、経口可、ふらつき軽微
  • 独居でなく、フォローアップ可能
  • 中枢性所見・赤旗なし

帰宅指導

  • 当日は急な頭位変換を避ける
  • 再発時の対応（自己 Epley も可能、ただし習得困難）
  • 症状が改善しない / 持続する / 神経症状が出た場合は再受診
  • 耳鼻咽喉科外来へ紹介（再発予防・難治例）

入院 / 再評価を検討

  • Epley で改善しない
  • 持続性めまい・神経所見出現
  • 独居・フォロー困難・嘔吐強い

⚠️ ピットフォール

  • 「回転性だから末梢」と決めつける — 後方循環梗塞も回転性で来る
  • 持続性めまいに BPPV と診断する — BPPV は必ず短時間・誘発性
  • 非典型な眼振（垂直のみ・潜時なし・疲労せず）を BPPV と扱う — 中枢性疑い
  • Epley で改善しないのに BPPV を続行 — 分類を疑い、AVS への再評価を
  • 前庭抑制薬を漫然投与 — 中枢性代償を遅らせる。短期のみ
  • 画像検査をルーチンで撮る — 典型例なら不要。非典型・赤旗時のみ
