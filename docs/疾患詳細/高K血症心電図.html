高K血症心電図

⚡ これを見たら○○
  1. テント T → 幅広 QRS → sine wave → 心停止 の進行を見極める
  2. まずグルコン酸 Ca（心筋膜安定化、心電図異常あれば即時、採血を待たない）
  3. ブドウ糖 + インスリン + β2 刺激薬（K の細胞内シフト）で時間を稼ぐ
  4. 透析考慮（K > 6.5 / 急性腎障害 / 心電図異常）— 腎内科・ICU 早期相談

📈 心電図の進行段階

段階心電図目安 K（参考値、個人差大）
1尖鋭・対称・狭いT波（テント状T）5.5–6.5 mEq/L
2PR 延長、P 波低平化〜消失6.5–7.5
3QRS 幅広化、伝導遅延7.0–8.0
4サインウェーブ、心室細動・停止≥8–9
数値依存しない

心電図は 数値より早く悪化することがある。進行所見を見たら 血液ガスや生化学を待たない。

💉 即座の治療

1. 心筋保護（1–3分）

  • グルコン酸カルシウム 10% 10 mL（1アンプル）を 2–5分で静注
  • 効果 10–20分、改善なければ 5–10分後に再投与可
  • 中心静脈が使えるなら塩化カルシウムでも可
  • ジギタリス中毒疑いではカルシウムは慎重投与（ゆっくり）

2. K を細胞内に移す（30–60分）

  • インスリン 10単位 + 50% ブドウ糖 50 mL 静注
  • 血糖 <250 mg/dL なら必ずブドウ糖併用、以降の低血糖を監視
  • サルブタモール吸入 10–20 mg（大量、効くまで15分）
  • アシドーシスがあれば 重炭酸ナトリウム

3. K を体外に出す

  • ループ利尿薬（腎機能が残っていれば）
  • 陽イオン交換樹脂（新規製剤 patiromer / ZS-9 を含む。急性期の主役ではない）
  • 透析：無尿 / 乏尿、薬剤で下がらない、心電図が重症
救急外来での流れ

ECG で所見 → モニター → Ca 静注 → インスリン＋糖 → 採血確認 → β2吸入 → 腎内科 / ICU 相談

🔍 同時にやる

  • 原因検索：AKI / CKD、RAAS阻害薬、カリウム保持性利尿薬、K補充、横紋筋融解、溶血、腫瘍崩壊、副腎不全
  • 採血：K / Mg / Ca、Cr、CK、LDH、血糖、血液ガス
  • ECG 再検（治療後の改善確認）

🆚 鑑別（高K血症に「見える」心電図）

  • de Winterパターン：T 波の形が違う（上行性STが主役）
  • 急性心膜炎
  • 早期再分極

⚠️ ピットフォール

  • 採血結果を待ってから動く
  • 偽性高K（溶血・採血困難・白血球 /血小板 著増）で焦る／見落とす
  • インスリン＋糖のあと血糖監視を忘れる（遅発性低血糖）
  • 「心電図変化なし」を理由に介入しない（透析例・急速上昇例は危険）
