Le Fort 骨折

🚨 Le Fort II / III は高エネ機転。顔面浮腫・後咽頭血腫・舌位低下で気道閉塞リスク。
頭蓋底骨折合併時は経鼻挿管・経鼻胃管は禁。麻酔科・形成外科に早期コール。
典型像（疑うきっかけ）
  • 機転：高エネルギー外傷（交通／高所転落／顔面への直達外力）
  • 顔面長軸の延長、顔面浮腫、皮下血腫、鼻出血
  • 咬合異常（open bite）、上顎の動揺（鼻骨〜上顎を把持して揺らすと顔面中部が動く）
  • Le Fort III では dish-face deformity（皿状陥凹）
  • 頭蓋顔面分離（craniofacial disjunction）の臨床所見
  • 頭蓋底骨折合併が多い（Battle 徴候・Raccoon eyes・髄液漏）

→ 詳細所見は 同時並行 2 軸 と コンサル前確認 へ

同時並行で評価する 2 軸（治療方針を分ける所見）
気道（最優先・早期挿管判断）

II / III では早期挿管準備。経鼻はリスクを評価してから。

  • 顔面浮腫・後咽頭血腫・舌位低下で気道閉塞
  • 分泌物・血液・歯片の誤嚥
  • 頭蓋底骨折合併 → 経鼻挿管・経鼻胃管は原則禁（特に篩板骨折で脳内へ誤挿入のリスク。系統的レビューでは実際の報告は稀だが、ER の安全側ルールとして経口挿管を第一選択）
  • 困難気道予測 → 経口挿管 ± 外科的気道（輪状甲状靱帯切開）の準備
  • SBP ≥ 110、SpO₂ ≥ 94% を維持
合併損傷（治療ルートが分かれる）

頭蓋内・視神経・頸椎を同時に評価。それぞれ別ルートで動く。

  • 頭蓋内出血（→ 頭部外傷評価）：脳外即時
  • 頭蓋底骨折（前頭蓋底）：髄液漏・嗅覚低下・経鼻挿管禁
  • 視神経管骨折（→ 視神経管骨折）：急性視力低下＋RAPD → 眼科＋脳外即時
  • 頸椎損傷（→ 頸部外傷・頸部痛）：高エネ機転で必ず外す
  • ZMC ／眼窩底／鼻骨／下顎との複合骨折がほぼ常
ER で打つ手（初期処置）
  1. 気道確保：II / III では早期に判断。経口挿管が原則、頭蓋底骨折で経鼻禁、困難気道予測時は外科的気道準備
  2. 頸椎カラー保持（高エネ＝合併率高）
  3. 循環：顔面出血コントロール、輸液
  4. 頭部・頸椎・顔面 CT を同時オーダー（3D 再構成）
  5. 抗生剤投与（副鼻腔と交通 → 開放骨折扱い）
  6. 絶飲食、鎮痛、頭部挙上 30°
  7. 視機能（視力・対光反射・RAPD）を記録
  8. 形成外科コンサルト（気道安定後に整復計画）
  9. 脳外科コンサルト（頭蓋内出血・頭蓋底骨折合併時）
形成外科コンサルト前に確認すべきこと

Le Fort 骨折と診断したら、コンサル前に以下を整理しておく（電話・紹介状で必ず聞かれる）。気道安定が前提。

  • 気道状況：自発呼吸維持／挿管管理中（経口）／検討中
  • 循環：バイタル、ショックの有無
  • 意識・GCS：頭蓋内損傷の合併評価
  • 顔面所見：顔面長軸延長、dish-face deformity の有無、上顎の動揺
  • 咬合異常：open bite、本人申告
  • Le Fort 線の分類（CT 所見）：I／II／III／混合型、左右対称性
  • 頭蓋底徴候：Battle／Raccoon／髄液漏／鼓室血腫の有無
  • 視機能（治療方針は変えないが記録必須）：視力、対光反射、RAPD、眼球運動
  • 頬部しびれ（V2 領域）の有無
  • 合併骨折：ZMC、眼窩底、鼻骨、下顎、視神経管
  • 頭部 CT：頭蓋内病変、頸椎 CT：頸椎損傷
  • 背景情報：抗凝固薬・抗血小板薬・既往歴・受傷前 ADL
  • ER 処置の記録：気道確保（時刻・方法）、抗生剤投与時刻、絶飲食、頭部挙上
  • 術前検査：採血・心電図・胸部 X 線の済／未
画像所見と治療方針

① Le Fort 分類（I / II / III）

1901 年 René Le Fort（仏）が屍体実験で分類。臨床的に純粋型はまれで混合型が多い。顔面 CT（軸位 + 冠状位 + 3D-CT）で骨折線の全体像と複合型を把握する。

  • Le Fort I（Guérin 骨折）：水平骨折。上顎骨を歯槽部とともに鼻底〜翼状突起底部で離断。上顎の浮動。咬合異常 + 上口唇腫脹。気道リスク低〜中
  • Le Fort II（pyramidal）：鼻骨〜上顎骨〜頬骨上顎縫合〜翼状突起にかけてピラミッド型に離断。顔面中央部の浮動。眼窩下神経麻痺。髄液鼻漏は篩板・前頭蓋底損傷を合併した時に生じる（II 単独では通常起こらない）
  • Le Fort III（craniofacial disjunction）：頭蓋顔面分離。鼻骨〜眼窩内側壁〜外側壁〜翼状突起。顔面全体が頭蓋から離れて浮動。dish-face deformity、視神経管・頭蓋底損傷を伴う
  • 混合型（例：左 Le Fort II + 右 Le Fort I+II + ZMC）は珍しくない、左右非対称評価が必要

② 治療方針

状態必要な処置タイミング
気道閉塞リスク（II/III、顔面浮腫、後咽頭血腫） 気道確保（経口挿管 or 外科的気道）。経鼻禁の確認 直ちに
頭蓋内出血／頭蓋底骨折合併 脳外コンサルト、ICP 対応、髄液漏管理 直ちに
視神経管骨折合併 眼科＋脳外、ステロイド／減圧術判断 受傷 8h 以内が望ましい
気道・頭蓋内安定後 骨接合術（下行的アプローチ：前頭骨〜頬骨〜上顎骨〜歯槽部、ミニプレート／マイクロプレート） 受傷後 10〜14 日の予定手術（浮腫軽快後）

③ 手術の原則

  • 受傷後 10〜14 日（浮腫軽快後）の予定手術が原則。緊急性はまず気道・出血・頭蓋内病変
  • 下行的アプローチ：上から下へ（前頭骨〜頬骨〜上顎骨〜歯槽部）固定
  • 標準切開は 冠状切開＋口腔前庭切開（sublabial）＋下眼瞼切開／経結膜切開 の組合せ。ミニプレート／マイクロプレート固定
  • Le Fort II/III は顎間固定を併用することがある
  • 頭蓋底骨折合併・髄液漏は脳外と協議（保存的経過観察 vs 修復）
