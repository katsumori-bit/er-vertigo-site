高度房室ブロック

⚡ これを見たら○○
  1. Mobitz II / 完全房室ブロック（3 度 AVB）= 危険。原則、即循環器コール
  2. 症状（失神・徐脈による低灌流）あれば緊急ペーシング（経皮 → 経静脈）
  3. 原因検索（虚血・薬剤・電解質・心筋症・Lyme）を同時並行で実施
  4. 恒久的ペースメーカが原則（可逆原因が除外できない／取れない場合）

分類心電図の特徴現場対応
Ⅰ度PR ≥200 ms、脱落なし無症状なら経過観察
Ⅱ度 Mobitz I（Wenckebach）PR が徐々に延長→脱落、狭いQRS無症状・迷走神経性を考える。薬剤・虚血を整理

🚨 即相談のトリガー

迷わず循環器
  • Mobitz II / 2:1 / 高度 / Ⅲ度
  • 逸搏QRSが 広い（心室性逸搏 = 予備能が低い）
  • 心拍数が 遅い（40未満など）
  • 血行動態が不安定（低血圧、前失神、意識障害、心不全所見）
  • 急性虚血所見を伴う
  • 可逆原因が取り切れない

🔍 同時に検索する可逆原因

  • 薬剤：β遮断薬、非DHP Ca拮抗薬（ベラパミル / ジルチアゼム）、ジゴキシン、アミオダロン、イバブラジン、クロニジン、オピオイド
  • 電解質：高K血症（高K血症心電図）、高Mg、甲状腺機能低下
  • 急性虚血：特に下壁MI（右冠動脈支配）→ STEMI
  • 感染 / 炎症：心筋炎、Lyme、心内膜炎による弁周囲膿瘍
  • 迷走神経性：嘔吐・排尿直後など

💉 救急でのアクション

  • モニター、ルート、12誘導の保存（あとで時系列比較）
  • 採血：K / Mg / Ca、TSH、ジゴキシン濃度（内服中）、トロポニン
  • 血行動態が不安定なら：
    • アトロピン 0.5–1 mg 静注（狭いQRSの徐脈で有効性が期待できる）
    • 経皮ペーシング準備
    • ドパミン / アドレナリン持続投与
    • 循環器に経静脈ペーシングを依頼
  • 広いQRSの逸搏・完全房室ブロックでは アトロピンは効きにくいことが多い

🟢 安定・無症状での分岐

  • 明らかな可逆原因（薬剤・高K）→ 是正して再評価、循環器相談は同日
  • 可逆原因が不明 → 入院継続 + 循環器相談

⚠️ ピットフォール

  • 2:1 房室ブロックを Mobitz I と決めつける
  • 心拍数 50台でも無症状だから放置（広いQRS逸搏は危険）
  • 薬剤リストを見ないまま循環器に丸投げする
  • 下壁STEMIに伴うAVBで、虚血自体を忘れる
