#骨折 #脊椎 #破裂骨折 #高エネルギー #ER #即時整形

破裂骨折 = 椎体の前後が潰れる高エネルギー外傷
  • 高エネルギー外傷（転落・交通事故）による椎体骨折
  • 椎体後方成分が脊柱管に突出 → 脊髄損傷のリスク
  • 全例 CT 必須、神経症状の有無に関わらず 即時整形
  • 圧迫骨折と違い「陳旧性」という概念はない — 所見があれば新鮮骨折

圧迫骨折との違い（要）

圧迫骨折破裂骨折
受傷機転低エネルギー（しりもち・軽微）高エネルギー（転落・交通）
変形の範囲前方成分のみ（楔状）前方〜後方成分まで
不安定性なし強い
神経症状稀リスク高い（脊髄・馬尾）
緊急度低い高い
CT不要（MRI を優先）必須
  • 高齢者でも高エネルギーなら破裂骨折、低エネルギーなら圧迫骨折と割り切ってよい。
  • 破裂骨折では 楔形 ≠ 陳旧性。所見があれば新鮮骨折として扱う。

受傷機転

  • 高所からの転落（梯子・屋根・工事現場）
  • 交通事故（車の正面衝突・バイク転倒）
  • ダッシュボード損傷（膝 → 大腿骨 → 骨盤 → 脊椎への軸圧）
  • 重量物の落下

身体所見

  • 背部〜腰部の強い圧痛（棘突起正中部）
  • CFPS（closed-fist percussion sign）陽性
  • 神経所見：下肢の運動・感覚・腱反射、鞍状麻酔・膀胱直腸障害（馬尾症候群）
  • 四肢麻痺があれば救命救急センターへ搬送
見逃せないレッドフラッグ
  • 下肢の運動麻痺 / 感覚障害
  • 膀胱直腸障害
  • 鞍状麻酔
  • 受傷時の一過性の麻痺（現在改善していても聴取する）

画像

CT は必須

  • 椎体後方成分の脊柱管内突出（retropulsion）を評価
  • 椎体後壁の連続性が失われていれば破裂骨折
  • Denis 3 column 分類（中柱の破綻が破裂骨折の定義）
椎体の柱構成破綻の意味
前柱椎体前方 2/3 + 前縦靱帯単独なら圧迫骨折
中柱椎体後方 1/3 + 後縦靱帯破綻 = 破裂骨折
後柱椎弓・棘突起・後方靱帯単独破綻は稀

MRI

  • 脊髄損傷の評価
  • 後縦靱帯・黄色靱帯の損傷評価
  • 骨挫傷の範囲（新鮮骨折の裏付け）

マネジメント — 原則 即時整形

  • 神経症状の有無に関わらず、椎体後壁破綻を見たら整形外科 / 脊椎外科へ即コンサルト
  • 不安定性が強ければ緊急固定術（後方固定術）
  • 神経症状があれば減圧術を併用
  • 安定型で神経症状なければ TLSO などの硬性コルセット + 安静で保存可能だが、判断は専門医
非整形外科医の役割
  1. CT で椎体後壁の連続性を評価
  2. 神経所見を丁寧にとる（下肢・肛門括約筋・膀胱直腸）
  3. 即日整形 / 脊椎外科へコンサルト
  4. コンサルトまで 体位変換は logroll、不必要に動かさない

横突起骨折（併発する場合）

  • 受傷機転は共通（高エネルギー）だが、単独では 脊椎の安定性に影響しない
  • 傍脊柱筋圧痛（椎体から 1–2 cm 外側）が身体所見のポイント
  • レントゲンでは腸管と重なって見えない → CT で診断
  • 原則帰宅保存、疼痛強ければ入院整形で数日の鎮痛加療
  • 高エネルギー外傷に伴う横突起骨折は、全身 CT の初回読影で見逃されることがある → 腰痛が続く例は再読影を

帰宅時の説明（稀に保存例）

  • 体幹のねじれ・屈曲は絶対禁止
  • 硬性コルセット装着
  • 下肢のしびれ・脱力・排尿排便障害が出たら即受診
  • 数日後の整形外科外来で再評価
