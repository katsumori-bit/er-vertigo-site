脊椎破裂骨折

同時並行で評価する 3 軸
典型像
  • 若年〜中年：高所転落・交通外傷（高エネ）
  • 高齢者：軽微外傷でも DISH ではなり得る
  • 主訴：強い背部痛・腰痛、立てない、下肢麻痺
  • 圧痛・棘突起間ギャップ・後彎変形
  • 胸腰椎移行部（Th11-L2）に好発
血管

高エネ外傷では 多発外傷・大動脈損傷を必ず除外。

  • ABCD の評価、ショック傾向の有無
  • 胸腹部 CT で大動脈損傷・実質臓器損傷
  • 抗凝固薬例では硬膜外血腫リスク
  • 神経原性ショックの鑑別（後述）
神経

完全／不完全脊髄損傷の評価が最重要。ASIAを必ず記録。

  • ASIA impairment scale（A-E）
  • MRC スコア：腸腰筋 / 大腿四頭筋 / 前脛骨筋 / 長母趾伸筋 / 下腿三頭筋
  • 感覚レベル（デルマトーム）、Light touch / Pin prick
  • 会陰部知覚・肛門括約筋トーヌス・Sacral sparing
  • 残尿・尿閉、Priapism
画像の見方
  • 胸腰椎 Xp（正面・側面）から開始
  • Xp で必ず追うポイント：
    • 椎体高低下、椎体後壁の骨片後方転位（後壁連続性の途絶）
    • 椎弓根間距離の開大（正面像）
    • 後彎変形・椎間離開
    • 多椎体障害（高エネは複数椎体評価）
    • 横突起骨折の併発（高エネ外傷で典型、椎体から 1–2 cm 離れた傍脊柱筋に圧痛）
  • 高エネ外傷では 脊椎破裂骨折・横突起骨折を鑑別に挙げる
  • 破裂骨折を疑えば必ず CT（後壁骨片の脊柱管内突出、AO 分類確定）
  • 神経症状あり → MRI（脊髄損傷・浮腫・椎間板損傷の評価）
  • Denis 3 column theory：前柱・中柱・後柱のうち 中柱（後壁）損傷で不安定型
  • AO Spine 分類：A（圧迫型）／B（離開型）／C（回旋型）— B・C は不安定
  • Load-sharing classification で手術アプローチ選択
  • 移送は log rollで脊柱中立を保持
治療方針（緊急性高）
所見方針ER の動き
神経症状なし、安定型（AO A） 入院、保存 vs 手術判断 整形コンサルト、体幹装具
不安定型（AO B/C） すみやかにコンサルト、手術ほぼ必須 絶飲食、ライン、整形即時
脊髄損傷（不完全） 緊急手術（除圧）検討 ICU 管理、整形即時
脊髄損傷（完全） すみやかにコンサルト、ICU 管理 呼吸循環管理、整形即時
呼吸障害／神経原性ショック ICU、各科協力 挿管、昇圧（α作動）、ステロイドは推奨せず

ER で打つ手

  • 体位は log roll、移送は backboard（脊柱中立、二次損傷予防）
  • 頸椎カラーは頚椎損傷合併を疑う場合（多発外傷では併存頻度高い）
  • 絶飲食、ライン確保、術前検査（採血・心電図・胸部 X 線）
  • 鎮痛は循環動態を見ながら（オピオイド少量）
  • ステロイド大量療法は推奨されない（有効性乏しく感染合併症増加）
  • 呼吸障害（高位胸髄）・神経原性ショックは ICU 管理
  • DVT 予防（弾性ストッキング、必要時抗凝固）、便秘・尿閉対策
