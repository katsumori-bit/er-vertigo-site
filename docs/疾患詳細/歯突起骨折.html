歯突起骨折（Odontoid / Dens 骨折）

同時並行で評価する 3 軸
典型像
  • 上位頸椎損傷で最多
  • 高齢者：転倒など軽微外傷でも骨折
  • 若年：高エネルギー外傷、過屈曲・過伸展
  • 後頸部正中圧痛、頸部可動制限
  • 「自分で頭を支えていないと不安」という訴え
  • 高齢者の軽微頸部痛で見落としやすい
血管

歯突起骨折単独で急性血管損傷は通常ないが、横突孔骨折合併で BCVI（椎骨動脈損傷）を考慮。

  • 両側橈骨動脈触知
  • 頸部血管雑音、Horner 徴候の有無
  • 横突孔骨折合併なら頸部 CTA を低い閾値で
  • 抗凝固薬例は遅発性出血・固定後の出血リスク
神経

転位が大きいと延髄・上位頸髄圧迫で致命的。ASIA 評価を必ず記録。

  • ASIA grade（E が正常）、四肢 MMT・感覚
  • 項部硬直、頭部支持感覚（自分で頭を支えていると訴える）
  • 呼吸状態（高位脊髄損傷で横隔膜麻痺）
  • 麻痺所見は時系列で記録
画像の見方
  • 頸椎 CT が第一選択（推奨）。Open mouth view のみで除外しない
  • 高齢者は読影困難（重なり・骨密度低下）でCT を低い閾値で
  • CT で必ず追うポイント：
    • 歯突起の骨折線（先端／基部／椎体内進展）
    • 転位（mm）・前方／後方 angulation
    • 環椎（C1）・軸椎（C2）他の合併骨折
    • 横突孔骨折（BCVI スクリーニング）
  • 横靱帯損傷の評価が必要なら MRI
  • 神経症状・脊髄圧迫評価にも MRI を考慮
  • Anderson-D'Alonzo 分類で部位を記述
治療方針（Anderson-D'Alonzo 分類）
Type所見方針ER の動き
Type I 歯突起先端の oblique 剥離（稀） 保存（硬性カラー 6-8 週）、安定 硬性カラー固定 ＋ 脊椎外科コンサル
Type II 歯突起基部の骨折（不安定） 偽関節リスク高（高齢者 30-40%）。手術（odontoid screw／posterior C1-C2 fusion）vs Halo。高齢者は合併症から方針を共同決定 硬性カラー固定 ＋ 脊椎外科即コンサル
Type III 椎体内に骨折線が進展 通常安定、保存で癒合良好（硬性カラー or Halo 8-12 週） 硬性カラー固定 ＋ 脊椎外科コンサル
横靱帯損傷合併 MRI で横靱帯断裂 不安定、手術考慮 in-line stabilization 維持 ＋ 即コンサル
神経症状あり ASIA 低下、脊髄圧迫 緊急手術考慮 気道確保（in-line）＋ 即コンサル

ER で打つ手

  • 硬性カラー固定、頭頸部の in-line stabilization
  • 気道操作時は in-line 維持、video laryngoscope を考慮
  • 高齢者は骨密度・抗凝固薬を必ず確認、誤嚥対策
  • 転倒の機序評価（失神／不整脈／脱水など）
  • 合併損傷（頭蓋底骨折・他頸椎骨折・BCVI）を見落とさない
  • Type II 高齢例は偽関節リスクを家族に説明、治療方針共同決定

ピットフォール
  • 高齢者の軽微頸部痛・転倒で歯突起骨折を見逃す（最頻）
  • Open mouth view のみで除外（重なり・骨密度低下で読影困難）
  • 高齢者 Type II の偽関節リスク説明不足
  • 抗凝固薬例で固定後の遅発性出血を想定しない
  • 横突孔骨折合併時の BCVI（椎骨動脈損傷）スクリーニング漏れ
  • 気道確保時の不適切な頸部操作で神経損傷を増悪
関連ハブ・隣接疾患
  • 主訴入口：頸部外傷・頸部痛 ／ 転倒
  • 処置：硬性カラー ／ in-line stabilization
  • 合併評価：BCVI（頸部 CTA）、横靱帯損傷（MRI）、頭蓋底骨折
