急性腸管虚血（Acute Mesenteric Ischemia）

#ER #救急 #腹痛 #SMA #SMAE #SMAT #NOMI #hospitaldelay

致死率 60% 以上（心筋梗塞の 3 倍、脳梗塞の 4 倍）。20 年間救命率がほとんど改善していない。救命の鍵は精密検査でも高度医療でもなく「いかに早期診断できるか」に尽きる。「初期診断で軽症と判断 → 時間経過 → 顕在化したときには手遅れ」の hospital delay の典型疾患。

原著の失敗例（要約）
79 歳女性、心房細動・胃切除後・脳梗塞後で常用薬中断中。突然の腹痛と嘔吐で来院 → 腹部所見ほぼ陰性、検査も大きな異常なし → 経過観察 → 痛みが楽になり排便あり → 帰宅 → 翌朝意識障害・呼吸不全で再搬送。 典型 5 段階：① 基礎疾患あり ② sudden onset ③ 強い自発痛だが所見ほぼなし（介入できれば救命可） ④ いったんよくなる ⑤ 意識障害・呼吸不全（もう遅い）。

分類

SMA 領域の虚血で 4 タイプ。

分類背景
SMAE（上腸間膜動脈塞栓症）最多。心原性血栓（心房細動・弁疾患・IE の vegetation・PFO 経由の paradoxical embolus）。SMA 起始部から 3〜10 cm 末梢で閉塞
SMAT（上腸間膜動脈血栓症）動脈硬化・石灰化が背景。SMA 起始部で閉塞。食後の腹痛（abdominal angina）が先行することあり
NOMI（非閉塞性腸管虚血）透析・心臓血管術後・ICU・血管収縮薬使用中などで発症。健康者には起こらない
SMVT（上腸間膜静脈血栓症）症状・重症度が他 3 者と異なるため別項で記載

同じ最終病態をきたす疾患群：

  • 大動脈解離に伴う SMA 閉塞（B 型解離に多い）。「B 解離だから保存治療」は危険。臓器虚血を伴う解離は緊急手術
  • SMA 単独解離（閉塞は稀）
  • 小腸軸捻転：先天性腸回転異常／腸間膜根部が短い（痩せ身に多い）

病歴（3 相の自然経過）

要点：sudden onset か否かが最大のキー。「有意な身体所見がないことが有意な所見」（NOMI は必ずしも sudden ではない）。

相所見
① Hyperactive phase突然の強い腹痛（局在性なく腹部全般）。痛みの割に身体所見が乏しい。ここで診断できれば救命可能
② Paralytic phase痛みが軽減してくる（虚血遷延 → 壊死で痛みを感じなくなる）。食事できることもある。腹部は膨満
③ Shock phase細菌性腹膜炎・敗血症進行。代謝性アシドーシス＋頻呼吸、皮膚（特に下肢）網状斑（mottling）、意識レベル低下

危険因子

  • 不整脈（特に心房細動）／弁疾患・弁置換後／ワーファリン服用／動脈血栓塞栓症の既往／透析／心臓血管手術後／血管収縮薬／（他の原因の）ショック
  • SMAT では abdominal angina（食後の腹痛・腹部違和感）が先行することあり — 過去に同様の食後痛を聞き出す

身体所見

疾患特異的な所見は存在しない。強い腹痛の割に腸雑音は正常、押しても圧痛点はない。虚血の段階では腹部所見は出現しない。

  • 下行〜S 状結腸は SMA 灌流外なのでその領域の虚血は回避される
  • 唯一拾える所見は腹部の冷感：「胸や側腹部は温かいのに下腹部が冷たい」
  • 塞栓症が疑わしい例では四肢の動脈触知・左右差・神経学的所見を必ず取る（他塞栓の有無＋以後の比較のため）
  • 腸管壊死で汎発性腹膜炎になれば反跳痛・筋性防御 → この時点で診断したのでは遅い

初期診断と鑑別疾患

初期診断で「疑えるかどうか」がすべて。

  • 虚血段階の鑑別：sudden onset の血管系疾患 — 破裂性腹部大動脈瘤／大動脈解離
  • 腹膜炎期の鑑別：下部消化管穿孔／絞扼性腸閉塞／小腸軸捻転（病態的にほぼ同じ）
  • 鑑別はすべて重症疾患のみ → 「非常にヤバイ」か「まったく大丈夫（杞憂）」かの二者択一、中間はない

診断後の注意

  • SMAEでは血栓源（心房細動・IE・PFO）が残存 → 次は脳梗塞などほかの塞栓が来る可能性を意識する
  • NOMIでは「腸管だけ」なのか「全身虚血の一部」なのかを判断 — 全身虚血の一部なら腸管切除しても病状改善は期待できない

検査

血液

  • 発症早期に有用な特異検査は乏しい
  • 乳酸：感度 77〜100%、特異度 42%。正常で否定できない
  • アミラーゼ：半数で上昇
  • D ダイマー > 0.9 mg/L：感度 60% / 特異度 82%
  • 代謝性アシドーシスは早期には出にくく、ないことで否定はできない
  • 「ICU でわけのわからない代謝性アシドーシス → 腸管虚血を疑え」は名言

単純 X 線

  • 特異所見なし。早期は小腸ガスが一部見える程度。腸管壊死で麻痺性イレウス（びまん性拡張）

腹部超音波

  • SMAE はカラードプラで SMA フローなしを確認すれば診断可能
  • SMAT は元々狭窄なので前後比較がないと判断困難。NOMI はフロー有無では診断不能
  • 副所見：通常は SMV > SMA だが、SMA 閉塞では SMA 近位拡張・SMV 細小化で径の逆転現象
  • 暗赤色血性腹水を穿刺できればすでに壊死

腹部造影 CT — ゴールドスタンダード

  • 可能なら動脈相のダイナミック CT。SMAE と SMAT は診断可能
  • 造影効果があっても虚血の否定にはならない
  • NOMI は CT で確定困難。CT アンギオで空腸・回腸動脈の狭窄・攣縮を見るとの報告もあるが一般的には困難
  • 壊死の副所見：腸管壁内ガス・門脈ガス。ただし 10% は壊死以外でも出る
  • 腎機能低下例では造影 CT を躊躇するが、単純 CT だけ撮る意義はさらに低い → こんなときはエコーで SMA 近位部を見るのが有用。透析中なら腎機能はもう廃絶しているので造影に躊躇不要
門脈ガス vs 胆道ガスの見分け方
肝末梢までガスが広がる（血流に沿う）→ 門脈ガス（壊死示唆）／中枢側に集簇（胆汁の流れに逆行）→ pneumobilia（胆道再建後・EST 後など）。肝縁 2 cm 以内のガスは門脈ガスと考えてよい。

血管造影

  • かつては診断のゴールドスタンダードだったが現在は CT に置換
  • 診断より血管内治療の意義（腹膜炎前なら血栓溶解／吸引）
  • NOMI で典型的なはずの「腸間膜動脈の狭窄・攣縮（sausage sign）」はショック一般でも見られ、診断は施行者の感覚に依存

確定診断

SMAE と SMAT はCT で診断可能。NOMI は CT ではわからないが、健康人には起きない疾患なので「起こし得る対象か」で判断する。心疾患＋透析の両方該当なら強く疑ってよい。

「CT は何ともないけど高齢で気になる、本人はちょっとよくなったというけど帰して大丈夫だろうか？」と思ったときの判断法：
「明日この人が救急車で運ばれて、手術した外科医から『あの人 NOMI だったよ』と知らされたら自分は驚くか？」 「やっぱり」と思うなら診断は NOMI でよい。

治療方針

① SMAE

  • 診断時点で抗凝固（ヘパリン）開始（手術を理由に躊躇しない、大きな脳梗塞のリスクのほうが怖い）
  • 基本は手術（血栓除去＋壊死腸管切除）。残存腸管はできるだけ長く残し、不安があれば24〜48h 後に 2nd look surgery を予定。Damage control として初回は吻合・ストーマを作らず 2 回目に行うのがよい
  • カテーテル血栓溶解療法の適応：発症 8 時間以内／腹膜刺激症状なし／溶解禁忌なし。4 時間で溶けない or 壊死症状出現で手術へ
  • 救命後はワーファリンで再発抑制

② SMAT

  • 原則は塞栓症と同じ手術。違いは虚血のみで壊死前なら血行再建術（バイパス）を行う点
  • 実際には壊死前の段階で診断できることは少なく、腸管切除を要することが多い
  • 血管内治療（ステント）も選択肢。術後はアスピリン

③ NOMI

  • 腹膜刺激症状なしなら保存治療：脱水補正＋絶飲食（腸管血流需要を増やさない）。抗菌薬や腸管減圧のエビデンスはない
  • パパベリン持続動注療法はケースレポート程度の根拠
  • 腹膜刺激症状あれば手術 → 重度基礎疾患（透析）の上での腸管壊死は予後きわめて悪い
  • そもそも「手術するかどうか」自体が悩ましいのが NOMI の特徴 — 全身状態が極めて不良な場合「手術してもしなくても助からない」のほうが現実に近い

予後

  • どのタイプも死亡率は高い。NOMI の死亡率 70〜90%
  • 虚血段階で介入し腸管切除を要さなければハンディなく社会復帰可能
  • 壊死すれば残存小腸で予後が決まる。SMA 本幹閉塞では空腸 40〜50 cm のみ残存することがあり短腸症候群（SBS）必発
  • 救命できても侵襲的集中治療と SBS の長期管理が待つ。ACP（事前ケア計画）と家族への正確な情報提供が重要：「手術を乗り切れるか」だけが家族の関心になりがちだが、多くの死亡は手術中ではなく術後 ICU で起こること、救命後も SBS で長期生存が難しいことを伝える
  • 周術期予測は APACHE II・SOFA・MODS・ACS NSQIPS Surgical Risk Calculator など具体的な数字を使うと話し手と受け手の温度差が縮まる

結腸虚血（Colonic Ischemia）

「虚血性大腸炎」と同義だが、ここでは最重症型である結腸広範壊死を扱う（SMA 領域の急性腸管虚血と臨床像はほぼ一致）。

  • 病態：脾湾曲部中心の大腸広範壊死／非閉塞性大腸虚血。下腸間膜動脈閉塞の既往＋全身低血流が背景
  • 他に大動脈瘤手術での下腸間膜動脈結紮、急性大動脈分岐部閉塞（saddle block）などで起こる
  • 3 相は SMA 虚血と同じだが早期に血便がある点が違う（粘膜虚血による出血）
  • 多くは shock phase（敗血症）で来院。意識レベルが悪く腹痛の病歴が取れないこともあり、「正体不明の敗血症」として鑑別が始まることも
  • 診断は基本的に開腹所見（小腸広範壊死と異なり腹水はあまり出ない）
  • 治療は急性腸管壊死に準じ、初回は壊死腸管切除＋一時閉鎖、循環改善後に二期手術

上腸間膜静脈血栓症（SMVT）

4 型の 1 つだが他 3 者より明らかに軽症で、別扱いがよい。

  • 下肢 DVT と同じ局所病態。75% で先天性凝固異常（プロテイン C／S 欠損など）。他に後天性凝固異常・門脈圧亢進・鈍的腹部外傷・急性膵炎後

3 つの発症形態

  • Acute：突然発症で動脈閉塞同様の虚血を起こしうる。直ちに壊死しないが側副路がないと最終的に壊死、手術を要すかもしれない病態
  • Subacute：突然発症ではなく、側副路が一部あり腸管浮腫で腹痛・下痢。重症急性膵炎の二次的現象として多い。手術にはなりにくい
  • Chronic：基本的に無症状。偶然見つかる。肝硬変など門脈圧亢進では消化管出血の原因に

Acute SMVT の診療

  • SMA 閉塞より突然発症ではなく症状軽微。腹膜炎までの時間も 3〜5 日と長い（SMA 閉塞は 12〜24 h）
  • 悪心嘔吐は約半数（他の腸管虚血より多い）。下痢・下血は通常ない
  • 身体所見：壁うっ血肥厚で圧痛は存在（虚血と異なる）。腹水は高頻度
  • 血液：抗凝固を始める前にPT/APTT・AT-III・プロテイン C／S を提出。静脈閉塞は虚血が進んでもアシドーシスになりにくい
  • 家族歴・DVT 既往を聞く
