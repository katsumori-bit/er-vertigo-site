上腕骨顆上骨折

⚡ 即時アクション
  1. 小児の伸展型がほとんど（4–8 歳、FOOSH、95% 伸展型／成人は稀）
  2. 血管神経評価必須：橈骨動脈触知＋正中神経 / 前骨間神経（AIN・OK サイン）/ 橈骨神経 / 尺骨神経 を全例で記載
  3. Gartland 分類（Ⅰ-Ⅲ）：側面像の anterior humeral line と後方皮質の連続性で判定
  4. Gartland Ⅱ・Ⅲ は小児整形に即コンサルト。後方シーネ（90° 屈曲、過屈曲は禁忌）+ 三角巾 + 挙上、血管障害があれば緊急

📞 この疾患が該当する主訴
  • 肩の外傷・痛み — 上肢外傷の鑑別、特に小児の肘腫脹で本症を想起
  • 手関節・手の外傷 — FOOSH の小児では手関節と肘の両方を必ず評価
  • 転倒 — 4–8 歳の FOOSH が典型受傷機転

📚 疫学

上腕骨顆上骨折 = 小児肘外傷で最多、神経血管障害に注意
  • 4–8 歳の小児、FOOSH で受傷
  • 小児肘関節骨折の 60% を占める
  • 正中神経・前骨間神経・橈骨神経麻痺 の合併が多い
  • Volkmann 拘縮（上腕動脈損傷 → 虚血）は緊急対応
  • Gartland 分類で II 以上は 即時整形
  • 小児肘関節骨折の 60% を占める（最多）
  • 好発年齢 4–8 歳
  • 伸展型 95%（FOOSH）、屈曲型 5%（肘を直接打つ）

🩺 身体所見

  • 肘の腫脹・変形（骨折線の高さで段差）
  • 肘窩のしわの消失（腫脹で埋まる）
  • 上腕遠位部の圧痛
  • 神経所見を必ずとる（受傷直後の評価が重要）

神経所見（全例で実施）

神経評価麻痺時の所見
正中神経OK サイン・母指示指感覚OK サイン不能
前骨間神経母指 IP 関節屈曲（最多）丸を作れない（Pinch sign）
橈骨神経手関節背屈下垂手
尺骨神経小指外転鉤爪変形

血管所見

  • 橈骨動脈触知：触れなければ上腕動脈損傷を疑う
  • 前腕コンパートメント緊満 → Volkmann 拘縮の前駆症状
  • 5P（pain, pallor, pulselessness, paresthesia, paralysis）
Volkmann 拘縮の 5P + 過度な疼痛
  • 受動伸展での激痛はコンパートメント症候群を示唆
  • 上腕動脈損傷を放置すると 不可逆性の前腕虚血 → 拘縮
  • 血管障害があれば即時整形 + 血管外科

🖼️ 画像

  • 肘 2 方向（正面・側面）
  • 側面像で骨折線を探す
  • Posterior fat pad sign 陽性なら骨折あり
  • 前方線（anterior humeral line） が橈骨頭中央を通らなければ転位あり

📊 Gartland 分類

Type所見治療
I転位なしギプス固定で保存
II後方皮質のみ連続整復 + ピンニング（即時整形）
III全周転位整復 + ピンニング（即時整形）
IV多方向不安定整復困難、即時整形

🩹 マネジメント

非整形外科医の役割

  1. 神経所見・血管所見の評価と記載
  2. 後方シーネ + 三角巾固定（90° 屈曲位、過屈曲は禁忌 → 血流悪化）
  3. アイシング + 挙上
  4. 整形外科コンサルト
    • Type I → 整形外来で保存
    • Type II–IV → 即時整形
  5. 血管障害・コンパートメント兆候があれば 緊急コンサルト

小児への配慮

  • 疼痛管理を十分に（アセトアミノフェン・イブプロフェン）
  • 親への説明：神経麻痺の多くは自然回復するが、3 ヶ月は経過観察が必要

🏠 帰宅時（保存例のみ）の注意

  • 過度な疼痛・しびれ・指先の蒼白 → 即受診
  • ギプス後 24–48 時間は腫脹ピークのため要注意
