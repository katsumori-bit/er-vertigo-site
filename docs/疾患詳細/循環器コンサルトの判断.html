循環器コンサルトの判断

⚡ 即時アクション
  1. STEMI / 致死性不整脈（VT, VF, 完全 AVB）は即コール — 外来に逃がさない
  2. 急性心不全（Killip Ⅲ-Ⅳ）：肺水腫・心原性ショック領域は同日コール
  3. 心原性ショック：低血圧 + 末梢冷感 + 意識障害 + 乏尿 → ICU と並走で循環器コール
  4. 弁膜症急性増悪 / 大動脈解離 / 肺塞栓：循環器・心臓外科・血管外科の橋渡しを即時に

📞 この疾患が該当する主訴
  • 胸痛 — STEMI / NSTE-ACS / 大動脈解離 / 肺塞栓の振り分け
  • 呼吸困難 — 急性心不全 Killip Ⅲ-Ⅳ・肺塞栓・肺水腫
  • 失神（意識消失） — 心原性失神（VT/VF・完全 AVB・大動脈狭窄など）
  • 動悸 — 新規 AF・VT・心房粗動など

🧭 基本方針

一行で

入院中の初回評価責任は原則 院内循環器に寄せる。退院後のフォロー分岐は評価が終わってから。

戻る: 入院時心電図

  • 入院中に拾った異常心電図 → 初回評価は院内（循環器）
  • 退院後に「診療所で十分 / 外来循環器 / 侵襲を含む精査」のどれかに振り分ける

🔀 分岐表

状況アクション
即相談圏（STEMI / de Winterパターン / 後壁梗塞 / 高度房室ブロック（完全房室ブロック・Mobitz II） / 交代性脚ブロック / 高K血症心電図 / Brugada心電図 Type 1 + 背景 / 重度 QT延長 / VT/VF）院内急性経路。外来に逃がさない
同日圏（Wellens症候群 / NSTE-ACS 疑い / 新規 AF / 新規 LVH + 心不全 / 二枝ブロック + Ⅰ度房室ブロック）同日循環器コンサルト
灰ゾーン（軽微 ST 変化、境界例）旧ECG + 薬 + 最小ラボ（旧心電図と内服確認） → 48時間以内に循環器
旧ECG同一で安定、急性リスクなし退院後は かかりつけ / 診療所 フォローでよいことが多い
生活習慣病管理が主、心電図は軽微かかりつけ主、循環器はオプション
ペーシング / ICD / アブレーション / 冠動脈介入の議論循環器が主役（診療所では決めない）

🏥 「診療所に振る」ための最低条件

以下を すべて満たす ときに現実的：

  1. 急性症候なし（胸痛・失神・意識障害・喘鳴・血圧不安定なし）
  2. トロポニン陰性（必要なら）、電解質正常
  3. 旧ECGと同一、または説明可能な安定所見
  4. 危険薬の変更なし（QT延長薬・房室結節作用薬・Naチャネル作用薬）
  5. 帰宅後の再受診指示が明確（症状トリガーを文書化）

📞 電話前に揃える情報（1分テンプレ）

  • 年齢、性別、主訴、バイタル、意識
  • 入院（予定）理由
  • 心電図所見の要約（即相談所見／伝導／QTc／電解質所見）
  • 旧ECG との比較（ありなし、違いの有無）
  • 内服（房室結節・QT・Na チャネル・利尿）
  • 最小ラボ（K / Mg / Ca、トロポニン、Cr）
  • これまでにやった介入（モニター、ルート、投薬）
  • 相談したい具体的な質問（ペーシングの要否 / カテの要否 / 薬剤中止の可否）

📝 退院時の書面で必ず書くこと

  • 現在の心電図所見と 旧との比較結果
  • 内服変更の有無、QT / 徐脈に関する注意
  • 次回フォロー先：循環器外来 or かかりつけ
  • 症状トリガー：失神・動悸・胸痛・呼吸困難 → 即受診

⚠️ ピットフォール

  • 「かかりつけで見てもらってください」と、評価を終わらせずに投げる
  • 退院時サマリで心電図所見を省く
  • 夜間の当直が「循環器には明日相談」として朝を待つ（即相談圏の見落とし）
  • 逆に、旧同一・無症状・安定の心電図で夜間オンコールを叩く（過剰）
