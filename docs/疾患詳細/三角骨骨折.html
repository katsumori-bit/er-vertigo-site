三角骨骨折（背側剥離）

同時並行で評価する 3 軸
典型像
  • FOOSH（手をついて転倒・背屈強制）／手関節背屈位の直達外力
  • 手根骨骨折の 18.3%（舟状骨骨折の次に多い）
  • 尺骨延長線・小指球の真下（三角骨背側）の圧痛
  • 掌側・背側両方から圧痛確認（舟状骨骨折と同様）
  • 手関節背屈で疼痛増強
  • 多くは背側剥離骨折（dorsal chip / dorsal avulsion）
血管

背側剥離骨折で血管損傷を合併することはほぼない。橈骨動脈・尺骨動脈を念のため確認。

  • 橈骨動脈触知（手首掌側橈側）
  • 尺骨動脈触知（手首掌側尺側）
  • 指動脈・毛細血管再充満（CRT）、左右差なし
  • 急性虚血を疑う所見があれば高エネルギー外傷を疑い再評価
神経

三角骨は Guyon 管に近接。尺骨神経領域の感覚・運動を確認。

  • 尺骨神経領域（小指掌側／環指尺側半分）の感覚
  • 指外転（骨間筋）／小指対立（小指対立筋）
  • 正中神経・橈骨神経領域は通常問題なし（記録）
  • 麻痺所見があれば必ずカルテに残す
画像の見方
  • 手関節 Xp PA／lateral ＋ 斜位像（oblique） の 3 方向が原則
  • PA 単独では見落としやすい。三角骨は手根骨が重なって写るため、PA では背側チップが識別困難
  • 斜位像・側面像で背側チップを確認（骨折ハンター p.2026 引用）
    • 側面像：背側に小さな剥離骨片（チップ）が浮き出る
    • 斜位像（回内 45°）：三角骨背側面が描出されチップが明瞭になる
  • 身体所見で疑ったら必ず斜位像を追加オーダー
  • Xp で疑診も確診もできない場合：CT で精査（骨片サイズ・転位の正確な評価）
  • MRI：TFCC 損傷併発を疑う場合（背側剥離は TFCC 牽引機序）
  • 必ず合併骨折チェック：舟状骨骨折／月状骨周囲脱臼／橈骨遠位端骨折
治療方針（転位 × 骨片サイズ）
パターン所見方針ER の動き
背側剥離・転位なし／小骨片 三角骨背側にチップ、転位 (−) 保存（sugar tong splint 4〜6 週） シーネ ＋ 整形外来 1〜2 週以内
転位あり／大骨片 骨片サイズ大、関節面巻き込み 整形コンサル（手術検討、稀） シーネ ＋ 整形相談
TFCC 損傷併発疑い 尺側手関節痛持続、TFCC 圧痛 (+) MRI 追加／整形コンサル シーネ ＋ MRI 予約／整形外来
合併損傷 舟状骨骨折／月状骨周囲脱臼／橈骨遠位端骨折 該当疾患に準じる 同日整形コンサル

ER で打つ手

  • sugar tong splint（前腕回旋制限を含む）が基本
    • cock-up splint で済ませない — 前腕回内外を制限できず、TFCC・三角骨への牽引が続くため
  • 三角巾装着、患肢挙上指導
  • 鎮痛は NSAIDs／アセトアミノフェン
  • 整形外科外来 1〜2 週以内に紹介
  • 「尺側痛が長引く・回内外で痛む」場合は TFCC 損傷を疑い再診を指示

disposition

  • 原則帰宅保存（手術適応は稀）
  • 整形外来 1〜2 週以内、再診時に転位・癒合を再評価

関連ハブ・隣接疾患
  • 主訴入口：手関節・手の外傷 ／ 転倒
  • 隣接疾患：舟状骨骨折 ／ 月状骨周囲脱臼 ／ 橈骨遠位端骨折
  • 処置：sugar tong splint（前腕回旋制限）／三角巾
  • ピットフォール：
    • PA 単独で「異常なし」と判定 — 斜位・側面で見える
    • TFCC 損傷の併発を見逃す（背側剥離は TFCC 牽引機序）
    • cock-up splint で済ませる — 前腕回旋を制限できない
