📞 この疾患が該当する主訴
  • めまい — AVS（持続性めまい）の中枢性 vs 末梢性鑑別に使用

🎯 適応：AVS 限定

HINTS が使えるのは AVS だけ

  • AVS（Acute Vestibular Syndrome）：持続性めまい + 眼振 + 歩行不安定（数時間〜数日続く）
  • 持続性めまいであることが必須（発作性ではない）
  • AVS の原因は 前庭神経炎（末梢性）vs 小脳・脳幹梗塞（中枢性）の鑑別が中心

使ってはいけない場面

  • t-EVS（triggered Episodic Vestibular Syndrome）：体位変換で誘発される BPPV など → Dix-Hallpike を使う
  • s-EVS（spontaneous EVS）：自発性の発作性めまい（メニエール病、片頭痛性など）
  • 診察時に眼振がない例
適応を間違えると無意味

BPPV に HINTS を行っても偽陽性・偽陰性が大量発生する。「持続性 + 眼振あり」を確認してから使うこと。

👁️ Head Impulse Test (HI)

方法

  • 患者に検者の鼻を見続けるよう指示
  • 検者は患者の頭部を両手で支え、素早く小さく（10-20°）左右どちらかに振る
  • 頭を振った直後の眼球の動きを観察

解釈

  • 陽性（abnormal）：頭を振った側でcatch-up saccade（眼が一旦ずれて、戻ってくる動き）が出る → 末梢性（前庭神経炎）を示唆
  • 陰性（normal）：頭を振っても眼が常に鼻を捉え続ける → 中枢性を示唆
直感に反する

「異常所見（catch-up saccade）が出た方が末梢性」という逆説的な解釈が HINTS のポイント。前庭機能が低下している側では VOR が破綻するため saccade が出る。

〰️ Nystagmus type（眼振の型）

観察

  • 正面視・左方視・右方視で眼振の方向を確認
  • 急速相の方向で命名（左眼振 = 急速相が左向き）

解釈

  • 一方向性水平眼振（どの方向を見ても急速相が同じ向き）→ 末梢性を示唆
  • 方向交代性眼振（右を見ると右、左を見ると左へ向く）→ 中枢性を示唆
  • 純粋垂直性・回旋性眼振も中枢性を強く示唆

⚖️ Test of Skew（斜偏視）

方法（cover-uncover test）

  • 患者に検者の鼻を見続けるよう指示
  • 片方の眼を手で覆い、素早く外す（左右交互に繰り返す）
  • 覆いを外した瞬間の眼の垂直方向の動きを観察

解釈

  • 陽性（skew あり）：覆いを外した眼が垂直方向に動いて修正される → 中枢性を示唆（脳幹病変）
  • 陰性（skew なし）：眼の垂直偏位なし → 末梢性に矛盾しない
Skew は感度低・特異度高

Skew 陽性は中枢性に強く偏るが、陰性でも中枢性を否定はできない。3 つのうち 1 つでも中枢性パターンなら中枢性として扱う。

🚨 INFARCT pattern と解釈

INFARCT（中枢性を示唆する所見）

  • Impulse Normal — Head Impulse 陰性
  • Fast-phase Alternating — 方向交代性眼振
  • Refixation on Cover Test — Test of Skew 陽性

判定ルール

所見末梢性中枢性（INFARCT）
Head Impulse陽性（catch-up saccade あり）陰性（normal）
Nystagmus一方向性水平方向交代性 / 垂直 / 回旋
Test of Skew陰性陽性
  • 3 項目全てが末梢性パターン → 末梢性（前庭神経炎）の診断に有用
  • 1 つでも中枢性パターン（INFARCT）→ 中枢性として扱う
  • 判断困難 = 中枢性として扱う（safer side に倒す）
  • 感度は早期 MRI DWI を上回る（発症 24-48h 以内では DWI の感度 80% 程度）

⚠️ 落とし穴

適応外で使ってしまう

  • BPPV など t-EVS には適応外。Dix-Hallpike を使う
  • 持続性めまい + 眼振 + 歩行不安定の 3 つが揃った AVS でのみ有効

術者依存性

  • HINTS は習熟が必要。経験の浅い検者では感度が大幅に低下する
  • 不安なら 中枢性として扱い MRI（DWI） を考慮

MRI 陰性でも安心しない

  • 発症 24-48h 以内の DWI 感度は 80% 程度。陰性でも小脳・脳幹梗塞は否定できない
  • HINTS で中枢性パターンが出た例は、DWI 陰性でも入院 + 再評価を考慮

聴力検査の併用：HINTS plus

  • HINTS に 聴力低下（hearing loss）を加えた HINTS plus がある
  • 急性発症の一側性難聴は AICA 領域梗塞を示唆 → 中枢性として扱う

判断困難 = 中枢性

  • 所見が曖昧な場合は中枢性として扱うのが安全
  • 「末梢性に見えるが INFARCT が 1 つでもあれば中枢性」が原則
