ATFL損傷

⚡ 即時アクション
  1. Ottawa Ankle Rules（骨折除外）
  2. 前方引き出しテスト + 内反ストレステスト
  3. MRI（grade 3 / 重症例）
  4. RICE → 早期可動・荷重（長期固定は推奨されない）

📞 この疾患が該当する主訴
  • 足関節・足部の外傷 — 内反捻挫の最多原因
  • 転倒 — 階段踏み外し・歩行中の段差

🦴 解剖

  • 足関節外側靱帯は 3 本
    • 前距腓靱帯（ATFL）：最弱、最も損傷多い
    • 踵腓靱帯（CFL）：次に多い
    • 後距腓靱帯（PTFL）：最強、稀にしか損傷しない

⚡ 受傷機転

  • 内反捻挫（足を内側にひねる）
  • スポーツ・階段での踏み外し・歩行中の段差

🩺 身体所見

  • 外果前下方の圧痛（ATFL の走行部）
  • 腫脹（受傷後数時間で増大）
  • 前方引き出しテスト：距骨を前方に引く、健側と比較して開大
  • 内反ストレステスト：不安定性

🖼️ 画像

  • Ottawa Ankle Rulesで撮影適応判断
  • 陰性なら骨折の可能性極低 → レントゲン不要
  • 持続疼痛が 1 週以上続けば再評価（occult fracture の除外）

📊 重症度分類

Grade靱帯の状態所見
I伸展 / 微細断裂軽度圧痛、不安定性なし、荷重可
II部分断裂中等度圧痛・腫脹、軽度不安定、荷重困難
III完全断裂強い圧痛・腫脹、明らかな不安定、荷重不能

💉 マネジメント

急性期（RICE）

  • Rest：活動制限、必要なら松葉杖
  • Ice：20 分 x 数回/日
  • Compression：弾性包帯
  • Elevation：挙上

固定

  • Grade I–II → 装具（ankle brace）、3–5 日後から可動域訓練
  • Grade III → U-slab シーネ 1–2 週 → その後装具へ

早期可動域訓練

  • 完全固定よりも早期可動域訓練の方が予後良好（Grade I–II）
  • 疼痛許容範囲で底背屈・円運動

薬物

  • アセトアミノフェン 第一選択
  • NSAIDs は短期間なら可（長期は骨・靱帯治癒を遅らせる可能性）

🔁 慢性足関節不安定症の予防

  • 初回捻挫後に 再発 30%
  • バランストレーニング・腓骨筋強化 が再発予防に有効
  • 反復する捻挫 → 整形外科で靱帯再建術を検討

⚠️ 鑑別・見落とし要注意

疾患鑑別点
足関節果部骨折外果の圧痛、Ottawa 陽性
Maisonneuve骨折腓骨近位端の圧痛
第5中足骨近位端骨折第 5 中足骨基部圧痛
距骨骨折距骨頸部の圧痛、持続痛
腓骨筋腱脱臼外果後方のポキっとした感覚、歩行時の引っかかり

🏠 帰宅時の説明

  • 最初の 48 時間は RICE 徹底
  • 3–5 日で疼痛軽減 → 装具装着で日常生活
  • 1 週後も疼痛・腫脹・歩行困難が続く → 整形外科再評価
  • Grade II–III は 2–6 週での完治、再発予防のバランス運動を推奨
