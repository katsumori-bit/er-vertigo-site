頬骨上顎複合体骨折（ZMC fracture）

典型像（疑うきっかけ）
  • 機転：殴打・スポーツ・交通外傷・転倒（側面打撲）
  • 頬部の陥凹・平坦化（浮腫が引いてから明瞭化）、結膜下出血、皮下血腫
  • 触診：眼窩外側縁・下縁・頬骨弓の段差・圧痛
  • 開口障害（陥凹した頬骨弓が下顎筋突起と impingement）
  • 頬部しびれ（眼窩下神経 V2）が高頻度
  • 頬骨弓孤立骨折は他面骨折を合併することがある（頻度は文献により幅広く一概に言えない）

→ 詳細所見は 同時並行 2 軸 と コンサル前確認 へ

同時並行で評価する 2 軸（治療方針を分ける所見）
視神経管骨折の合併（眼科＋脳外即時）

ZMC は眼窩外側壁・眼窩底を含み、視神経管骨折を伴うことがある。急性視力低下＋RAPDは即時対応。

  • 視力（Snellen / 指数弁 / 光覚弁）を必ず記録
  • 対光反射・RAPD：陽性なら視神経管骨折を疑う
  • 進行性顔面浮腫による遅発性視神経損傷あり、経時で再評価
  • 陽性 → 視神経管骨折 ルート（眼科＋脳外即時、薄スライス CT）
眼窩底骨折の合併（外眼筋嵌頓）

ZMC 骨折線が眼窩底に及ぶことが多い。小児 trap door は 6〜24h 以内の緊急手術、成人嵌頓は数日〜1〜2 週で許容。

  • 下方視・上方視で複視・眼球運動制限
  • oculocardiac reflex（徐脈・嘔気）＝嵌頓の強い示唆
  • 小児の trap door type は外見正常でも嵌頓あり（white-eyed）
  • 陽性 → 眼窩吹き抜け骨折 ルート（眼科・形成外科即コール）
ER で打つ手（初期処置）
  • 視力・対光反射・RAPD・眼球運動を必ず記録（合併の有無で動きが変わる）
  • 頬部しびれ（V2）・開口障害の程度を記録
  • 鼻かみ禁止（眼窩気腫・上顎洞交通）を患者・家族に説明
  • 抗生剤投与（副鼻腔と交通する開放骨折扱い）
  • 鎮痛・冷却・頭部挙上 30°
  • 頭部 CT を同時オーダー（合併損傷確認 → 頭部外傷評価）
  • 視神経管合併 → 眼科＋脳外即時
  • 眼窩底嵌頓合併 → 眼科・形成外科即時
  • 合併なし → 形成外科コンサルト（コンサル前確認 → コンサル文 の順で）
形成外科コンサルト前に確認すべきこと

ZMC 骨折と診断したら、コンサル前に以下を整理しておく（電話・紹介状で必ず聞かれる）。

  • 視機能：視力、対光反射、RAPD、眼球運動（複視の方向）
  • 外眼筋嵌頓徴候：oculocardiac reflex の有無
  • 頬部陥凹：あり／浮腫で判定困難、健側との比較
  • 下眼瞼下垂の有無
  • 開口障害：開口距離（cm）、頬骨弓陥凹由来か
  • 顔面感覚（治療方針は変えないが記録必須）：眼窩下神経 V2 領域（頬部・上口唇）しびれ
  • 触診所見：眼窩外側縁・下縁・頬骨弓の段差の部位
  • 画像所見：顔面 CT・3D-CT で骨折の型（tripod／tetrapod／頬骨弓孤立）、4 縫合の評価、眼窩底・内側壁合併
  • Le Fort 進展の評価：複合型の有無
  • 頭部 CT：頭蓋内病変、視神経管
  • 背景情報：抗凝固薬・抗血小板薬・既往歴・受傷前 ADL
  • ER 処置の記録：抗生剤投与時刻、鼻かみ禁の指示、鎮痛
画像所見と治療方針

① 画像で見る項目

顔面 CT（軸位 + 冠状位 + 3D-CT）が基本。骨折線・転位・合併損傷を一度に評価。

頬骨は 4 つの付着部で構成され、これらの離開を評価する：

  • 前頭頬骨縫合（zygomaticofrontal）
  • 頬骨上顎縫合（zygomaticomaxillary、眼窩下縁周辺）
  • 頬骨側頭縫合（頬骨弓部）
  • 蝶形骨頬骨縫合（眼窩外側壁奥）

"clear sinus sign"（上顎洞内液体貯留なし）は ZMC 単独・少転位の指標。

② 骨折のタイプ

  • 三脚骨折（tripod / trimalar fracture）＝ZMC の主要 3 縫合骨折として広く使われる名称
  • tetrapod fracture（完全 ZMC）：実際は 4 縫合すべてが骨折
  • 頬骨弓孤立骨折（in-fracture）：頬骨弓が V 字状に内方陥凹。3D-CT で明瞭
  • Le Fort I/II との複合型もある（→ Le Fort 骨折）

③ 治療方針

所見マネジメントタイミング
視神経管合併（急性視力低下＋RAPD） 視神経管骨折として対応（ステロイド・減圧術はエビデンス限定、専門医判断） 早期介入が望ましいが「8h」を絶対視せず
眼窩底 trap door 合併（嵌頓・oculocardiac reflex） 緊急手術（嵌頓解除） 小児 6〜24h／成人は数日〜1〜2 週
頬骨弓孤立骨折／軽度陥凹のみ Gillies approach（側頭部小切開 → エレベーターで頬骨弓を挙上） 浮腫軽快後（5〜10 日）
転位のある ZMC 骨折 観血的整復＋ミニプレート固定。前頭頬骨縫合・頬骨上顎縫合に 2 点固定が基本 浮腫軽快後（5〜10 日）
眼窩底欠損合併 眼窩底再建（人工骨・自家骨）併用 同時手術
非転位・症状軽微 保存（経過観察、軟食、開口訓練） 外来フォロー
