TFCC 損傷（三角線維軟骨複合体損傷）

同時並行で押さえる 3 軸
典型像
  • 急性外傷：FOOSH／回外回旋外力／DRUJ 脱臼に併発
  • 慢性／変性：反復回旋動作（ゴルフ・テニス・バット）
  • 尺側手関節痛（特に回旋・尺屈時）
  • クリック感、握力低下
  • X 線陰性で「捻挫」と誤診されやすい
誘発テスト

TFCC は disk + 背側/掌側橈尺靱帯 + meniscus homologue + ECU 腱鞘 + lunotriquetral 靱帯で構成。DRUJ 安定化と尺側荷重伝達を担う。

  • fovea sign：尺骨頭と TFCC 間（fovea）の直接圧痛
  • grind test：尺屈位 ＋ 軸圧 ＋ 回旋で疼痛誘発
  • piano key sign：ulnar styloid の異常可動（DRUJ 不安定）
  • DRUJ ballottement：尺骨遠位端の前後動揺
Palmer 分類

Class 1 = 外傷性、Class 2 = 変性。1B（尺骨付着部断裂）は DRUJ 不安定を伴い手術適応になりやすい。

  • 1A：disk 中央穿孔
  • 1B：尺骨付着部断裂（DRUJ 不安定）
  • 1C：遠位（ulnocarpal）付着部断裂
  • 1D：橈骨付着部断裂
  • 2A〜2E：変性（thinning → 穿孔 → 軟骨損傷 → LT 損傷 → 関節炎）
誘発テスト・画像の見方

身体診察（4 つ全部やる）

  • fovea sign：尺骨頭遠位 ・尺側手根屈筋（FCU）腱と ECU 腱の間の窩を圧迫 → 圧痛で陽性（感度・特異度高い）
  • grind test（TFCC stress test）：手関節尺屈 ＋ 軸圧 ＋ 回内外で疼痛誘発
  • piano key sign：前腕回内位で ulnar styloid を背側から押すと容易に沈み込む → DRUJ 不安定
  • DRUJ ballottement test：橈骨遠位端を固定し尺骨遠位を背掌側に動かす → 左右差で不安定性評価
  • 握力低下、回旋時クリック感

画像

  • 手関節 Xp PA／lateral：
    • ulnar variance の評価（plus variance は 1B／Class 2 のリスク因子）
    • DRUJ alignment、尺骨茎状突起骨折の有無
    • 三角骨・月状骨周囲脱臼の合併確認
  • MRI 関節造影（MR arthrography）が標準：disk 損傷の感度・特異度が最も高い
  • 単純 MRI は感度劣るが外来で第一選択になることが多い
  • 関節鏡が最終的な diagnostic gold standard
  • ER では X 線陰性でも臨床所見陽性ならTFCC 損傷疑いとして固定 ＋ 専門外来送り
治療方針（Palmer 分類 × 不安定性）
パターン所見方針ER の動き
急性外傷・DRUJ 安定 fovea sign ＋、piano key −、X 線陰性 保存（sugar tong ／ TFCC brace 4-6 週） シーネ ＋ 手の外科外来（1-2 週以内）＋ MRI 予約
急性外傷・DRUJ 不安定 piano key ＋、ballottement で動揺 手術適応（1B repair） 整形即コール、長腕シーネで回外位固定
慢性・反復動作歴 尺側痛、握力低下、ulnar plus variance 保存 → 無効なら debridement ＋ 尺骨短縮術 装具 ＋ 手の外科紹介
X 線で骨折合併 尺骨茎状突起 ・三角骨骨折等 骨折に準じて固定、整形コンサル 同日コンサル ／ 翌日外来
若年運動選手 競技復帰を急ぐ 早期 arthroscopic repair 検討 整形相談、競技休止指示

Palmer 分類別の標準治療

  • 1A（中央穿孔）：arthroscopic debridement
  • 1B（尺骨付着部）：arthroscopic／open repair（DRUJ 不安定なら必須）
  • 1C／1D：repair または reconstruction
  • Class 2 系：debridement ＋ ulnar shortening osteotomy（ulnar plus variance ある時）

ER で打つ手

  • sugar tong シーネまたはTFCC brace（前腕回旋制限）で固定
  • 三角巾装着、患肢挙上指導
  • 鎮痛は NSAIDs／アセトアミノフェン
  • DRUJ 不安定例は回外位で長腕シーネ固定（橈尺靱帯を緩めない）
  • 「X 線陰性でも捻挫ではなく TFCC 損傷の可能性」を本人に明確に説明
  • 1-2 週以内の手の外科外来受診を必ず指示
