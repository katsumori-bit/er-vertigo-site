⚡ これを見たら 交代性脚ブロック
  1. RBBB と LBBB が交互に出現
  2. 高度房室ブロックの前駆であり危険
  3. 即ペースメーカ植込み（一時的 → 恒久的）
  4. 失神既往あれば緊急対応

📞 この疾患が該当する主訴
  • 意識消失（失神） — 高度房室ブロック前駆としての失神、即ペーシング検討
  • 入院時心電図 — RBBB と LBBB の入れ替わりで気付くことが多い

🖼️ 図解リンク（心電図.com）

  • 交代性脚ブロック（alternating bundle branch block）

🔍 何を見て判断するか

  • 同一患者の経時的記録で、右脚ブロック様と 左脚ブロック様（LBBB） が 交互に出現する
  • 1枚のECG内で一拍ごとに切り替わることもある
  • 単発の「今日RBBB、昔の紙はLBBB」だけでは診断にしない。短期間の切り替わりが要点

🧠 臨床的意味

  • His 束以下の伝導系が 左右にまたがって広く障害されている
  • 完全房室ブロックへ進行しやすい背景
  • 2018 ACC/AHA/HRS 徐脈・伝導遅延ガイドライン：Class I で永久ペーシング適応になりやすい

🚨 アクション

原則、即相談
  • 循環器オンコール
  • モニター、ルート、採血（K / Mg / Ca、トロポニン）
  • 旧ECG・内服・既往は 旧心電図と内服確認
  • 血行不安定・意識障害・広い逸搏 → 高度房室ブロック と同じ救急対症を並行

🔎 鑑別・注意

  • 誘導の取り違えで形が変わっただけのことを除外する
  • ペースメーカ下の心電図は解釈が変わる

⚠️ ピットフォール

  • 「形の違う心電図が続いた」として流す
  • 自動診断ラベルだけで会話する（脚のパターンが交代していると言語化する）
