肘内障

同時並行で評価する 3 軸
典型像
  • 1–4 歳の幼児（5 歳以上はまれ）
  • 機転：手を引っ張られた後に泣き出した
  • 所見：患肢を 軽度屈曲・回内位で動かさない（pseudoparalysis）
  • 腫脹・変形なし、皮下出血なし
  • 整復一発ですぐに泣き止んで使い始める
血管

血管障害は通常なし。腫脹・色調変化があれば骨折を疑う。

  • 橈骨動脈触知 OK が基本
  • 腫脹・冷感・変色 → 骨折／脱臼を再考
  • 左右差なし、CRT 正常
神経

神経障害は通常なし。麻痺所見があれば骨折・脱臼を再考。

  • 整復後は患肢を自発的に使い始める
  • 感覚・運動の評価は幼児ゆえ困難 → 観察で判断
  • 整復後も使わない → Xp 再撮影、整形相談
画像の見方
  • 典型例では Xp 不要（病歴・所見＋整復成功で診断）
  • Xp を撮る適応：
    • 受傷機転が不明 or 引っ張られていない
    • 腫脹・皮下出血・変形あり
    • 整復後も患肢を使わない
    • 5 歳以上、または再発を繰り返す
  • Xp を撮る場合：肘 2 方向＋健側比較（骨端線評価）
  • 鑑別：上腕骨顆上骨折 ／ 上腕骨外顆骨折 ／ 鎖骨骨折
整復方法・治療方針
場面所見マネジメントER の動き
典型的肘内障 引っ張り＋pseudoparalysis、腫脹なし 徒手整復（ER で完結） 整復→使い始め確認→帰宅
整復成功 整復音 ＋ すぐに使い始める 固定不要、帰宅 家族へ再発予防説明
整復不能 / 使用しない 整復試行で改善せず Xp 撮影＋整形コンサルト 三角巾固定、整形外来へ
非典型例 腫脹／変形／皮下出血 骨折・脱臼を疑う Xp 必須、整形コンサル
反復例 過去に複数回 家族指導＋整形外来 整復後に経過観察

整復手技

  • 回外屈曲法：肘を 90° 屈曲、前腕を回外しつつ完全屈曲。橈骨頭にクリック感を触れる
  • 過回内法（hyperpronation）：前腕を完全回内する。回外屈曲法より成功率が高いとの報告あり
  • 整復成功 → 数分以内に泣き止んで 自発的に使い始める
  • 使い始めなければ 5–10 分後に再試行 or 反対手技
  • 2–3 回試行しても改善なし → Xp ＋ 整形コンサルに切替

ER で打つ手

  • 典型例は整形コンサル不要で完結する
  • 整復後は固定不要、帰宅可
  • 家族説明：再発予防（手を引っ張らない、抱き上げ方の指導）、5 歳頃には自然軽快
  • 使用しない／非典型例 → Xp ＋ 整形相談
整復後の説明・整形フォロー文（必要時）

典型例は整形コンサル不要。整復後の家族説明と、整復不能・非典型時の整形フォロー文の 2 種を用意。後者は『他科コンサルト実践マニュアル』のフォーマットに準拠。

家族への説明（カルテ・帰宅指示書用）

【病名】肘内障（橈骨頭亜脱臼） 【整復】[回外屈曲法／過回内法]で整復、整復音 [あり／なし] 【整復後経過】[○分後に自発的に使い始め／泣き止んで普段通り] 【家族説明】 ・幼児に多い肘の亜脱臼で、骨折ではありません。 ・手を引っ張ったり、急に持ち上げたりすると再発します。 ・5 歳前後には自然に起きにくくなります。 ・整復後は固定不要、入浴・食事・遊びは普通通りで構いません。 ・万が一、再び腕を使わない・腫れてきた場合は再受診してください。

整形外来フォロー文（整復不能・反復例・非典型例）

【目的】肘内障 [整復不能／反復例／非典型例]の精査・ご加療依頼。 【疾患】#1 [右/左]肘内障（[整復不能／○回目の反復／腫脹あり]） 【既往歴】[特記事項なし／○○] 【内服薬】[なし／○○] 【ADL（受傷前）】[年齢相当] 【現病歴】[年齢]歳[男/女]児。[受傷機転：手を引っ張られた／不明]で発症し、[来院経緯]で来院。 ・身体所見：患肢を軽度屈曲・回内位で使用せず（pseudoparalysis）、腫脹・変形 [なし／あり] ・血管：橈骨動脈触知 [良好]、CRT [正常] ・神経：麻痺 [なし／○○] ・画像：Xp [未／施行（骨折・脱臼なし／○○）] ・ER 処置：[回外屈曲法／過回内法]で整復試行 [○回]、[整復後使用開始／使用しない] よろしくお願いいたします。
関連ハブ・隣接疾患
  • 主訴入口：肘の外傷・痛み ／ 小児外傷
  • 隣接疾患：上腕骨顆上骨折 ／ 肘関節脱臼 ／ 鎖骨骨折
  • 関連処置：徒手整復（鎮痛・鎮静不要が原則）
  • 家族指導：手を引っ張らない、抱き上げ方の工夫
