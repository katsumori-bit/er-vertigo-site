de Winterパターン

⚡ これを見たら：LAD 完全閉塞（STEMI 同等）
  1. LAD 完全閉塞（STEMI と同等）→ 「ST 上昇がないから」と止めない
  2. V1-6 で J 点 ST 低下（1-3 mm）+ tall, prominent T 波 + aVR 軽度 ST 上昇
  3. STEMI 基準を満たさないが緊急カテ適応 → 循環器オンコール、施設の STEMI / ACS 経路を起動
  4. 若年男性に多い、見逃すと心筋ダメージ大。経過で典型 ST 上昇に化けうる → 短い間隔で 12 誘導再検

📞 この疾患が該当する主訴
  • 胸痛 — 若年男性の急性胸痛で見落とすと心筋ダメージ大
  • 入院時心電図 — 「ST 低下 = 虚血」だけで止めない
  • 失神（意識消失） — LAD 完全閉塞での致死性不整脈

🔗 図解リンク（心電図.com）

  • de Winter ST/T wave complex

📐 定義（典型像）

  • V1–V6 の J 点で 上行性（upsloping）ST低下 ≥1 mm
  • これに続く 高く対称性で尖ったT波
  • しばしば aVR で軽度ST上昇（0.5–1 mm）
  • QRS幅は通常 延長しない
注意

「ST上昇がない」からSTEMIではない、とは言えない。 分布・形・T波の高さをセットで見る。

🩺 臨床的意味

  • 急性期 LAD近位部閉塞相当
  • 前壁STEMIの約 2% で初期に出現するとされる
  • 経過中に典型的なST上昇へ移行することがある

⚡ アクション

即動く
  • 循環器オンコール、施設のSTEMI / ACS急性経路を起動
  • モニター、ルート、採血(トロポニン、K / Mg / Ca、Cr)
  • 12誘導を 短い間隔で再検（経過で典型的ST上昇に化けうる）
  • 症状がなくてもトリアージは STEMI と同じ扱いで開始する

🔍 鑑別

  • 高K血症の尖鋭T（PR延長・QRS幅広がりを合わせて 高K血症心電図）
  • 早期再分極（形が違う、J点上昇、ノッチ）
  • Wellens症候群（無痛期・T波陰性化が主役）

⚠️ ピットフォール

  • 「ST低下 = 虚血」だけで止めて循環器に上げない
  • 自動診断で「正常」「非特異的変化」と出るのを信じる
  • 無症状を理由に経過観察してしまう
