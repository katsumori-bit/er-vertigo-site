踵骨骨折

同時並行で評価する 3 軸
典型像
  • 足根骨骨折で最多（60% 超）。多くは高所からの踵着地（高エネルギー）
  • 必ず両側裸足にして診察（両側合併 10%、Mondor sign を見逃さない）
  • 所見：踵部腫脹＋Mondor sign（足底血腫）、踵を挟み込んで圧痛
  • 後足部の外反変形・幅広化、荷重不能
  • 合併：脊椎骨折（Th12〜L2）10%、他下肢外傷 30% — 腰背部・反対側踵を必ず確認
血管

足底コンパートメント症候群に最大注意（踵骨骨折は最高リスク群）。

  • 足背・後脛骨動脈を必ず触知（左右比較）
  • 後脛骨動脈は踵骨内側を走行 → 内側転位で損傷可
  • 足底の異常な緊満・激痛・他動的趾屈曲時痛 → コンパートメント症候群を疑う
  • 疑い → 内圧測定・緊急筋膜切開（鉤趾変形を防ぐ）
神経

踵骨内側面の脛骨神経・外側の腓腹神経を確認。

  • 足底感覚（脛骨神経 → 内側／外側足底神経）
  • 足部外側感覚（腓腹神経）
  • 足趾屈曲（内側／外側足底神経）
  • 整復前後で再評価し記録
画像の見方
  • 正面 + 側面 + 軸位（Harris-Beath view）の 3 方向。正面像は踵骨自体は写らないが足関節合併骨折の評価に必要
  • 踵骨は海綿骨成分が多く皮質が薄いため、卵の殻のように粉砕するのが明確骨折タイプ — 診断は容易
  • 微妙骨折タイプ（粉砕でないパターン）は骨折線イメージを持たないと拾えない — 側面像 + 軸位像で確認
  • Xp で必ず追うポイント：
    • Böhler 角（正常 20–40°、< 20° で感度・特異度ともに 99%）
    • Gissane 角（正常 100-130°、増大＝後関節面陥没）
    • 後関節面の段差（関節内骨折＝手術適応）
    • 踵骨幅の増大、後足部外反
    • 結節部裂離骨折（avulsion、皮膚壊死リスク）
  • 関節内骨折は CT 必須（Sanders 分類は冠状断 CT で確定）
  • 胸腰椎 Xp も同時に（圧迫骨折合併 10%、Th12〜L2 が多い）
  • 急性期 MRI は不要

※ 非整形外科医的には明確骨折タイプならマネジメントは Böhler 角測定で変わらない。微妙骨折タイプで迷ったときに測定するのが実践的。

治療方針（Sanders 分類 × マネジメント）
分類所見マネジメントER の動き
関節外骨折 結節部・前方突起など 帰宅保存（avulsion 大は手術） シーネ ＋ 翌日整形
Sanders Ⅰ：転位 < 2 mm 後関節面に骨折線あるが転位なし 帰宅保存 シーネ ＋ 整形 cons.

ER で打つ手 / 入院 vs 帰宅の判断（骨折ハンター準拠）

状況方針
単独・片側 踵骨骨折帰宅手術 or 帰宅保存（後方シーネ＋非荷重で送り後日整形）
両側 踵骨骨折歩行不能のため 入院整形
脊椎・他下肢の合併、転位高度合併外傷の重症度に応じ 入院整形
足底コンパートメント徴候緊急筋膜切開を整形と
  • 下腿シーネ固定（足関節中間位、足趾露出して循環確認）
  • 挙上＋クーリングを徹底（腫脹軽減が手術タイミングを決める）
  • 非荷重を厳守
  • 足底コンパートメント徴候を毎時観察（疼痛増悪・他動趾屈曲時痛）
  • 胸腰椎 Xp ／ CT で圧迫骨折精査＋反対側踵もチェック
  • 結節部 avulsion で皮膚緊張あれば緊急整復（壊死回避）
  • 絶飲食、ライン確保、抗凝固薬の有無を確認
  • 帰宅時は腫脹増悪・コンパートメント症候群の説明と疼痛時再診を指示
