踵骨骨折

⚡ 即時アクション
  1. 墜落・交通外傷で起こる
  2. AP+lat+axial（Harris）+ CT 必須
  3. Sanders 分類で評価
  4. 脊椎圧迫骨折・骨盤骨折の合併に注意（高エネルギー外傷の連鎖）
💥受傷機転 🚨高エネルギー
外傷の連鎖 🩺身体所見 📷画像
Böhler / CT 📊分類
Sanders 💉マネジメント ⚠️合併症 🕳️落とし穴 🏠帰宅時の説明
＋出典
📞 この疾患が該当する主訴
  • 足関節・足部の外傷 — 踵の強い圧痛・腫脹・荷重不能
  • 転倒 — 高所からの飛び降り・墜落で発生
  • 脊椎圧迫骨折 — 高エネルギー外傷の連鎖で 10–15% に合併

💥 受傷機転

  • 高所からの転落（屋根・梯子、2 階以上）
  • 垂直軸圧が踵骨に伝達
  • 交通事故（ダッシュボード損傷）

🚨 高エネルギー外傷の連鎖

踵骨骨折を見たら必ず探すもの
  1. 反対側の踵骨骨折（両側 10%）
  2. 脊椎圧迫骨折・破裂骨折（10–15%）
  3. 距骨骨折
  4. 下腿 / 大腿骨骨折

→ 全脊椎 + 両下肢のレントゲン or 全身 CT を追加

🩺 身体所見

  • 踵の強い圧痛 + 腫脹 + 血腫
  • Mondor sign：足底・踵の皮下血腫（踵骨骨折に特異的）
  • 荷重不能
  • 皮膚損傷、コンパートメント症候群（足の内側）

📷 画像

レントゲン

  • 踵骨 側面像・軸位像
  • Böhler 角 < 20°（正常 20–40°）→ 骨折
  • Gissane 角 > 130° → 骨折

CT

  • 全例 CT（関節内進展評価、手術計画）
  • Sanders 分類（関節内骨片数・位置で I–IV）

📊 分類

関節内 vs 関節外

  • 関節内 75%（距踵関節面を含む）
  • 関節外 25%

Sanders 分類（関節内骨折、CT で判定）

Type所見治療
I転位なし保存
II2 分割帰宅手術
III3 分割帰宅手術
IV粉砕帰宅手術（距踵関節固定術の可能性）

💉 マネジメント

ER での対応

  1. 全脊椎・両下肢・骨盤の評価（必要なら全身 CT）
  2. 踵シーネ固定 + 挙上 + アイシング
  3. 皮膚損傷・コンパートメント症候群の確認
  4. 整形外科コンサルト

手術タイミング

  • 軟部組織の回復を待つ（血疱・腫脹が落ち着くまで 1–2 週）
  • Pilon 骨折同様 staged treatment

保存（転位なし）

  • 非荷重 10–12 週
  • 早期 ROM 訓練

⚠️ 合併症

合併症背景
コンパートメント症候群（足）踵骨骨折の 10%
外傷性関節症（距踵関節）関節内骨折で必発
腓骨筋腱炎（踵骨外側の変形）長期の慢性症状
皮膚壊死早期手術のリスク

🕳️ 落とし穴

  • 「足首を挫いた」と受診して踵骨骨折（高所からの飛び降り自殺企図など）
  • 側面像で Böhler 角を測定しないと見逃す
  • 脊椎圧迫・破裂骨折の合併を忘れる

🏠 帰宅時の説明（保存例）

  • 完全非荷重 10–12 週
  • 松葉杖使用、足関節 ROM 訓練を早期開始
  • 定期的な整形外科受診
