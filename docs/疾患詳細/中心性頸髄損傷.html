中心性頸髄損傷（Central Cord Syndrome）

同時並行で評価する 3 軸
典型像「cape-like 上肢優位麻痺」
  • 高齢者・頸椎症あり＋顔面打撲（過伸展機転）
  • 上肢 > 下肢の運動麻痺（特に手内筋）
  • cape-like distribution（肩〜上肢遠位）
  • sacral sparing：肛門周囲感覚・直腸緊張保たれる
  • 最頻の incomplete spinal cord injury
  • 軽微外傷でも頸椎症性脊柱管狭窄で発症
機序

過伸展機転で脊柱管狭窄部位の脊髄中心部が圧挫される。脊髄中心部・側索内側の損傷が優位となり、結果として上肢優位麻痺を呈する（古典的な皮質脊髄路の層構造説は近年再評価あり）。

  • 高齢者の軽微な顔面打撲・転倒（古典的）
  • 頸椎症性脊柱管狭窄＋過伸展で挟み撃ち
  • 若年では高エネルギー外傷（交通事故・転落）
  • FOOSH 後の頸部過伸展でも発症
  • CT で骨傷なしが多い（pure soft tissue injury）
ASIA 評価

ASIA Impairment Scaleで incomplete/complete を分類。中心性頸髄損傷は通常 ASIA B/C/D（sacral sparing あり）。

  • 上肢 MMT：肩〜手指まで筋ごとに記録（特に手内筋）
  • 下肢 MMT：相対的に保たれる
  • 感覚レベルと pinprick / light touch を分けて評価
  • 肛門周囲感覚・直腸緊張（sacral sparing の確認）
  • 球海綿体反射、Hoffmann、Babinski
  • 神経原性ショック（徐脈＋低血圧）の有無
画像の見方
  • 頸椎 CT：まず骨傷の有無を評価
    • 骨折・脱臼・棘突起骨折・椎体終板損傷
    • 頸椎症性変化、骨棘、OPLL（後縦靱帯骨化症）
    • 骨傷なしでも中心性頸髄損傷は否定できない
  • 頸椎 MRI（必須）：神経症状があれば必ず撮像
    • T2 高信号（脊髄内信号変化）：損傷の決定的所見
    • 脊柱管狭窄の程度（軽度／中等度／重度）
    • 椎間板ヘルニア、靱帯損傷（前縦靱帯・黄色靱帯）
    • STIR で椎前血腫・棘間靱帯損傷
    • 脊髄浮腫の範囲（予後因子）
  • ピットフォール：高齢者の顔面打撲で「CT 異常なし」と帰宅させない。神経所見があれば MRI を追加
  • 動的撮影（屈曲・伸展側面像）は急性期には行わない（症状増悪リスク）
治療方針（急性期管理 + 手術適応）
パターン所見方針ER の動き
典型例（高齢・過伸展） 上肢優位麻痺＋ MRI T2 高信号 入院・脊椎外科即コンサルト カラー固定、MAP 85-90、MRI 手配
持続性脊柱管狭窄 重度の狭窄＋神経症状進行 早期減圧手術（24h 以内） 脊椎外科に手術タイミング相談
骨折・脱臼合併 頸椎不安定性あり 手術（固定＋減圧） 厳重カラー固定、手術準備
軽症（ASIA D） 上肢軽度麻痺のみ 入院・保存 → 改善次第手術判断 入院、リハ科コンサルト
神経原性ショック合併 徐脈＋低血圧 輸液＋昇圧（NA / DOA） A-line、CVC、ICU 入室考慮

ER で打つ手

  • 頸椎硬性カラー固定、in-line stabilization
  • ベッド上 30° head-up（脊髄浮腫対策）
  • MAP 85-90 mmHg 維持（7 日間目標、AOSpine guideline）：輸液＋昇圧
  • 神経原性ショック対策：徐脈にはアトロピン、低血圧にはノルアドレナリン
  • 導尿（尿閉対策）、DVT 予防（圧迫法から）
  • ステロイドはルーチン非推奨（NASCIS II/III 以降、AOSpine guideline 2017 推奨せず）
  • 呼吸管理：高位頸髄損傷（C3-5 障害）では呼吸不全に注意
  • 体温管理（低体温に注意、神経原性ショックでは保温）
  • 脊椎外科コール（早期減圧手術 24h 以内の評価）
  • 家族説明：約 75% が再歩行可能だが、手内筋・巧緻運動の改善は遅い

関連ハブ・隣接疾患
  • 主訴入口：頸部外傷・頸部痛 ／ 転倒
  • 鑑別：前脊髄症候群／Brown-Séquard 症候群／後脊髄症候群／complete SCI
  • 合併評価：頸椎骨折・脱臼、頭部外傷、顔面外傷
  • 処置：硬性カラー固定 ／ in-line stabilization ／ MAP 維持
  • ピットフォール：CT 陰性で帰宅させない／顔面打撲で頸髄症状を確認しない／ステロイドルーチン投与
