このページは「診断がつくまで」＋「制吐薬のハブ」
  1. 危険徴候を拾う
  2. どの系統の問題か（腹部 / 中枢前庭 / 全身代謝 / 妊娠 / 薬剤）を決める
  3. 初期対応（補液・血糖・電解質・制吐）を走らせる
  4. 診断がついたら疾患ページへ → 腹痛 / 頭痛 / めまい / DKA・HHS / 急性胃腸炎 / 異所性妊娠
  5. このページは制吐薬の用量・選択の参照ハブも兼ねる（§7）

最初に決めること

  • 今すぐ緊急対応が必要か
  • 腹部症状が主か
  • 中枢・前庭症状が主か
  • 全身性・代謝性か
  • 妊娠関連か
  • 薬剤・中毒関連か

0. 危険徴候

1つでもあれば、原因検索を急ぐ
  • ショック / 重度脱水 / ぐったりしている
  • 頻回嘔吐で水分が全く入らない
  • 強い腹痛, 腹膜刺激, 腹部膨満
  • 胸痛 / 背部痛
  • 激しい頭痛, 意識障害, けいれん, 局所神経症状
  • 頭部外傷後
  • 吐血 / コーヒー残渣様嘔吐
  • 妊娠の可能性 + 腹痛 or 性器出血
  • 糖尿病患者の嘔吐
  • 術後 / ヘルニア / 排ガス排便停止
  • 高齢者・免疫不全・担癌患者で原因不明
最初の実務
  • バイタル確認
  • 脱水評価
  • 血糖測定
  • 妊娠可能年齢なら hCG確認
  • 腹部所見 / 神経所見
  • 内服・中毒・大麻使用歴の確認

1. どの系統の問題か

A. 腹部が主

  • 腹痛が前景
  • 腹部膨満
  • 排ガス排便低下
  • 食後悪化
  • 限局した圧痛がある

→ 腹痛 / 腸閉塞 / 急性膵炎 / 胆道感染・胆嚢炎 / 急性胃腸炎

B. 中枢・前庭が主

  • 頭痛
  • 意識変容
  • 神経脱落症状
  • めまい
  • 頭部外傷後

→ 頭痛 / 神経診察 / めまい

C. 全身性・代謝性が主

  • 発熱
  • 口渇・多尿
  • 強い倦怠感
  • 呼吸異常
  • 感染症や敗血症を疑う

→ DKA/HHS / 敗血症 / 尿路感染症 / 電解質異常

D. 妊娠関連

  • 妊娠可能年齢
  • 月経遅れ
  • 妊娠判明済み
  • 腹痛 / 性器出血の有無を必ず確認

→ 妊娠初期診療 / 妊娠悪阻・つわり / 異所性妊娠

E. 薬剤・中毒関連

  • 新規内服
  • 薬剤増量
  • アルコール
  • 大麻
  • 毒物曝露

→ 薬剤性嘔気・嘔吐 / 中毒総論 / CHS

2. 5秒フロー

迷ったらこの順で考える
  1. 危険徴候があるか
  2. 腹部症状が主か
  3. 頭痛・神経症状・めまいがあるか
  4. 発熱・代謝異常・糖尿病があるか
  5. 妊娠可能性があるか
  6. 薬剤・大麻・中毒が関係しそうか
  7. どれにもはっきり乗らなければ、支持療法しつつ再評価する

3. 初期対応

原因未確定でも先にやること
  • 補液が必要か判断する
  • 血糖を確認する
  • 脱水と電解質異常を評価する
  • 制吐薬で再評価しやすい状態を作る
  • 効かなければ、想定している原因が違う前提で見直す
よく出す検査
  • 血糖
  • hCG
  • CBC
  • 電解質
  • 腎機能
  • 肝胆道系酵素
  • 必要時 CRP
  • 必要時 VBG / ABG
  • 必要時 Lipase
  • 尿検査
  • ECG
  • 必要時 画像検査

4. 病歴で拾うこと

病歴
  • いつからか
  • 急性か、反復か、慢性か
  • 食事との関係
  • 嘔吐内容：食物残渣 / 胆汁 / 血液 / 便臭
  • 腹痛、下痢、便秘、排ガス低下
  • 発熱、頭痛、めまい、胸痛、背部痛
  • 同居人の同症状、旅行、食事歴
  • 新規内服、増量、飲み忘れ
  • アルコール、大麻、その他薬物
  • 糖尿病、腎不全、妊娠、術後、癌、ヘルニア歴

5. 身体所見で外しにいくもの

腹部
  • 膨満
  • 限局圧痛
  • 腹膜刺激
  • ヘルニア
  • 腸雑音異常
  • CVA tenderness
神経
  • 意識レベル
  • 項部硬直
  • 眼球運動
  • 小脳失調
  • 局所神経症状
全身
  • 口腔乾燥
  • 頻脈
  • 起立性低血圧
  • 発熱
  • Kussmaul様呼吸
  • 中毒を疑う所見

6. すぐ使う鑑別

まず多いもの

  • 急性胃腸炎
  • 薬剤性嘔気・嘔吐
  • 妊娠悪阻・つわり
  • めまい
  • 片頭痛

見逃したくないもの

  • 腸閉塞
  • 急性膵炎
  • 胆道感染・胆嚢炎
  • DKA/HHS
  • 頭痛
  • 急性冠症候群
  • 異所性妊娠
  • 敗血症

忘れやすいもの

  • CHS
  • 胃排出遅延・胃不全麻痺
  • 尿路感染症
  • 副腎不全
  • 尿毒症
  • 有害事象・薬物中毒

7. 制吐薬の実務メモ（ハブ）

制吐薬は原因治療ではない

補液と原因検索が本体。制吐薬は再評価しやすくするため。 効かない嘔吐では危険疾患や CHS を考え直す。

ED 内（静注）

薬剤用量向く場面注意
オンダンセトロン4 mg IV（必要なら 8 mg まで）幅広く使える第一選択QT 延長、妊婦は第一選択にしにくい（アポカル点は弱い）が中等症以上で使用可
メトクロプラミド（プリンペラン）10 mg IV、腎機能低下時は減量（CCr <40 で半量）胃排出遅延、片頭痛、妊婦錐体外路症状、アカシジア、QT 延長
プロクロルペラジン（ノバミン）10 mg IM / 経口（本邦は IV 製剤限定的）片頭痛、反復嘔吐錐体外路症状、QT 延長、鎮静
ハロペリドール1–2 mg IV/IM難治性、中毒、CHS錐体外路症状、QT 延長
ジフェンヒドラミン25–50 mg IV前庭性嘔吐、メトクロプラミド併用で EPS 予防鎮静、抗コリン
デキサメタゾン4–8 mg IV化学療法後、片頭痛再燃予防、妊娠悪阻（治療抵抗）血糖上昇

帰宅時処方（成人・腎肝正常）

・ドンペリドン錠 10 mg       1回1錠 1日3回 食前              （QT 延長・肝障害注意）
・メトクロプラミド錠 5 mg    1回1錠 1日3回 食前              （EPS・アカシジア・QT 注意）
・OS-1 等の経口補水液        少量頻回（5–10 mL を 5–10 分ごと）
  • オンダンセトロン OD 錠 4 mg は本邦で制吐の保険適用が限定（化学療法・放射線）のため、ED 帰宅処方では使いにくい
  • 妊婦 → メトクロプラミド or ジフェンヒドラミン優先。オンダンセトロンは第 1 三半期は相対的に避ける（胎児心奇形・口裂の報告は一貫しないが安全側）
CHS を疑うとき

大麻常用 / 周期性嘔吐 / 熱いシャワーで軽くなる / 通常の制吐薬が効きにくい → ハロペリドール 1–2 mg IV、カプサイシンクリーム腹部塗布、大麻の中止指導 → CHS

8. 帰宅か観察か

帰宅を考えやすい
  • 危険徴候なし
  • 重篤な器質疾患が低そう
  • 水分摂取が可能
  • バイタル安定
  • 再診目安を説明できる
入院 / 観察を考える
  • 脱水が強い
  • 経口摂取不能
  • 電解質異常 / AKI
  • 腸閉塞, 膵炎, 胆道感染, DKA, 中枢疾患などを疑う
  • 妊娠関連で重症
  • 診断が不確実で再評価が必要

9. 再診・悪化時の赤旗

  • 腹痛が強くなる
  • 胸痛 / 背部痛が出る
  • 頭痛 / ふらつき / 意識障害が出る
  • 血を吐く
  • 水分が全く入らない
  • 尿が極端に減る
  • 妊娠中に腹痛 / 出血が出る
