肩の外傷・痛み

⚡ 即時アクション
  1. 血管神経評価（橈骨動脈・末梢冷感 / 腋窩神経 = deltoid 外側知覚 + 三角筋収縮 / 開放外傷）→ 異常あれば整形即コール
  2. 必須 X-line：肩 AP + Y view + 腋窩位（脱臼方向と関節適合の評価に必須）
  3. 脱臼は早期整復：Stimson 法 / Spaso 法を第一選択、鎮痛・筋弛緩を準備
  4. 初回脱臼 vs 反復性脱臼で固定期間と紹介を分ける、整復後は必ずX-line + 神経再評価

🚨 Red flags（血管・神経・開放）

血管

  • 上肢の冷感・蒼白・橈骨動脈触知低下 / 左右差
  • 毛細血管再充満時間延長、末梢 SpO2 低下
  • 高エネルギー外傷後の脱臼・骨折で腋窩動脈損傷を疑う

神経

  • 腋窩神経：deltoid 外側皮膚の知覚 + 三角筋収縮（最重要・脱臼で最多障害）
  • 筋皮神経：前腕外側知覚、肘屈曲
  • 正中・橈骨・尺骨：手指の運動 + 知覚
  • 高齢・高エネルギー外傷では腕神経叢損傷も

その他

  • 開放外傷（皮膚損傷 + 骨折・脱臼）
  • 整復不能な脱臼、著明な変形
1 つでもあれば即整形コンサルト

主な緊急鑑別：

  • 腋窩動脈損傷 / 腋窩神経損傷 / 腕神経叢損傷
  • 肩関節脱臼（整復不能・後方脱臼）
  • 開放骨折 / 上腕骨近位部骨折（4-part / 脱臼骨折）

🩺 最小診察（腋窩神経が最重要）

その場で行う最小セット

  • 視診：肩の輪郭（squared off shoulder = 前方脱臼の典型）、変形、皮下出血、開放創
  • 触診：鎖骨、肩鎖関節、肩峰、上腕骨大結節、骨頭の位置
  • 可動域：自動・他動の挙上 / 外旋 / 内旋（脱臼疑いでは無理に動かさない）
  • 腋窩神経：整復前後で必ず deltoid 外側皮膚の知覚 + 三角筋収縮を確認
  • 橈骨・尺骨・正中神経：手指の運動 + 知覚
  • 血管：橈骨動脈触知、毛細血管再充満、末梢 SpO2、皮膚色温

腱板の機能評価（脱臼除外後）

  • 棘上筋：empty can test、外転筋力
  • 棘下筋・小円筋：external rotation lag sign
  • 肩甲下筋：lift-off / belly press test
後方脱臼を見逃さない

てんかん発作 / 電撃傷 / 高エネルギー外傷後の内旋固定 + 外旋不能は後方脱臼を疑う。AP 単独では light bulb sign のみで見落とされる → 腋窩位 or Y view 必須。

📷 X-line の置き方（AP / Y / 腋窩）

撮影方向分かること注意
肩 AP（true AP） 骨頭の位置、骨折線、関節裂隙 後方脱臼は light bulb sign のみで分かりにくい
Y view（scapular Y） 骨頭の前後位置（前方 vs 後方脱臼の判定） Y の交点に骨頭がない = 脱臼
腋窩位 関節適合、Hill-Sachs / Bankart 病変 後方脱臼の確定に最有用。疼痛で困難なら Velpeau view 代用
追加 鎖骨 AP / Zanca view（肩鎖関節） 肩鎖関節脱臼の grade 判定に有用
1 方向のみで判断しない
  • AP のみで「整復済み」と判断しない → Y view / 腋窩位で確認
  • 整復後も必ず X-line で位置と Hill-Sachs / Bankart / 大結節骨折を再評価
  • 4-part 骨折・骨折脱臼は CT 追加

🦴 主要鑑別（脱臼・骨折・腱板）

病態手がかり進む先
肩関節脱臼（前方） 外傷後に自動運動不可、squared off shoulder、外転外旋位で固定 肩関節脱臼
肩関節脱臼（後方） てんかん・電撃傷後、内旋位固定、外旋不能 肩関節脱臼
上腕骨近位部骨折 高齢者転倒、肩周囲の腫脹と圧痛、皮下出血 上腕骨近位部骨折
鎖骨骨折 肩峰近傍の圧痛、皮下出血、肩を抱える肢位 鎖骨骨折
肩鎖関節脱臼 肩峰部の限局痛、段差変形、ピアノキー徴候 肩鎖関節脱臼
腱板損傷 外傷後の挙上困難、X 線で骨折なし、外転筋力低下 腱板損傷

🔄 脱臼の整復法

1. Stimson 法（うつ伏せ重錘牽引）
  • うつ伏せ、患肢をベッド端から下垂、5〜10 kg を 15〜30 分間重錘牽引
  • 鎮痛・筋弛緩を併用、術者の負担少なく成功率高い
  • 第一選択になりやすい
2. Spaso 法
  • 仰臥位で患肢を 90° 前方挙上 + 軽度外旋しながら牽引
  • 1 人で行え、鎮痛があれば成功率高い
3. Kocher 法 / 外旋法
  • 外旋法：肘 90° 屈曲位でゆっくり外旋、自然に整復
  • Kocher 法は古典的だが上腕骨骨折のリスクがあり推奨度低下
4. 整復前後のチェック
  • 整復前：腋窩神経・末梢神経 + 血管、X-line（脱臼方向 + 大結節骨折の有無）
  • 整復後：腋窩神経・末梢神経 + 血管 を再評価、X-line で整復確認（AP + Y）
  • 大結節骨折を伴う場合や高齢者は整形コンサルト
鎮痛・筋弛緩の選択肢
  • 静注鎮痛（フェンタニル / モルヒネ）+ ベンゾジアゼピン少量
  • 関節内リドカイン注入（IAL block）：呼吸抑制が少なく外来でも安全
  • 処置鎮静（procedural sedation）：モニター下で実施

🩹 初期固定・鎮痛

固定の選び方

  • 肩関節脱臼整復後：sling（三角巾）+ swathe
  • 上腕骨近位部骨折：sling + swathe、転位大なら U 字副木
  • 鎖骨骨折：8 字包帯 / sling（成人は sling 単独で十分との報告）
  • 肩鎖関節脱臼：sling、grade III 以上は整形相談

鎮痛

  • 第一選択：アセトアミノフェン、必要時 NSAIDs
  • 整復前：オピオイド静注 ± ベンゾジアゼピン or IAL block
  • 冷却を併用
固定後は必ず再評価

固定後に神経血管所見を再確認。指先の色温・知覚・運動を本人にも教え、悪化時受診を説明する。

📤 disposition（初回 vs 反復で違う）

初回脱臼

  • 整復後 sling 固定 2〜3 週間（年齢・骨片で調整）
  • 整形外科外来へ紹介（Bankart / Hill-Sachs 評価、若年者は手術検討）
  • 20 歳代の若年者は再脱臼率が高く、早期手術の選択肢を説明

反復性脱臼

  • 整復後 sling は短期間（数日〜1 週間）
  • 外来で手術適応を相談（鏡視下 Bankart 修復など）
  • 受診歴がある整形主治医がいれば優先連携

入院 / 整形コンサルト

  • 整復不能な脱臼、後方脱臼、骨折脱臼
  • 血管神経損傷、開放骨折
  • 4-part 上腕骨近位部骨折、転位大の鎖骨骨折
  • 高齢者で疼痛コントロール困難・独居で安全性低い
  • 夜間で整形対応不能 + 再評価必要例

⚠️ ピットフォール

  • 後方脱臼を AP のみで見落とす — Y view / 腋窩位を必ず追加
  • 整復前後で腋窩神経をチェックしない — 整復による神経障害の見落とし
  • 整復後 X-line を撮らない — 大結節骨折・再脱臼の確認漏れ
  • 高齢者の挙上困難を「腱板」と決めつける — 上腕骨近位部骨折を rule out
  • 初回と反復性で同じ管理 — 固定期間と紹介プロトコルが違う
  • 鎖骨骨折で 8 字包帯にこだわる — 成人は sling 単独でアウトカム同等
  • てんかん発作後の肩痛を軽視 — 後方脱臼の典型シーン
