肩の外傷・痛み

更新元: HTML 直編集

このページの使い方（30 秒）
  1. 神経血管障害と開放外傷がないかを最初に確認する
  2. 脱臼・骨折・軟部損傷の順で鑑別する
  3. 診断がついたら下の疾患ページへ進む

まず除外する危険所見

  • 上肢の冷感・蒼白・脈拍低下（血管損傷）
  • しびれ・運動麻痺（神経損傷）
  • 開放外傷（骨折・感染リスク）
  • 整復不能な脱臼、著明な変形
危険所見があれば即コンサルト

整形外科へ緊急連絡し、必要に応じて鎮痛・固定・画像を並行します。

最初に考える鑑別

病態手がかり進む先
肩関節脱臼 外傷後に自動運動不可、明らかな変形 肩関節脱臼
上腕骨近位部骨折 高齢者転倒、肩周囲の腫脹と圧痛 上腕骨近位部骨折
鎖骨骨折 肩峰近傍の圧痛、皮下出血 鎖骨骨折
肩鎖関節脱臼 肩峰部の限局痛、段差変形 肩鎖関節脱臼
腱板損傷 外傷後の挙上困難、X線で骨折なし 腱板損傷

初期対応の要点

  • 鎮痛先行（アセトアミノフェンを第一選択）
  • 画像は 2 方向を基本にし、必要なら追加撮影
  • 固定後に神経血管を再評価する
  • 転倒全体の評価は 転倒 へ戻る
