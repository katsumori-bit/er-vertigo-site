  2. なければ #同日相談（今日中に循環器）
  3. 迷えば 旧心電図と内服確認 で電話用の情報を詰める
  4. 振り分けは 循環器コンサルトの判断
コラム：「入院だから全例12誘導」はエビデンスで絶対か

多くの病院では 運用・安全網（無症候性虚血・伝導障害・AFの拾い、院内急変時のベースライン）として 全例取得が慣習化している。一方、「全入院患者にルーチン12誘導を取れば転帰が改善する」と示した大規模RCTのような強いエビデンスは乏しい。無症状一般成人のスクリーニング心電図については、米国予防サービス作業部会（USPSTF）も 利益が不明確としている。 → 「病院ルールだから取る」は正しいが、「ルールだから必ず臨床的に意味がある」とは限らない。 取ったあとの判断は別問題。

即相談（今すぐ電話）

無症状でも 以下があれば、夜間・休日でも循環器オンコールを検討。

虚血・電解質・チャネル病

  • VT / VF（モニター上で明らかなため詳細リンクは省略）
  • STEMI 基準のST上昇
  • de Winterパターン
  • 後壁梗塞（V1–V4のST低下 + 高いR波 / 直立T）
  • 高K血症心電図 の進行所見
  • Brugada心電図 1型様（新規・強疑）
  • QT延長 で著明（QTc ≥500 ms）＋ 誘発因子

伝導障害で即相談に入るもの

  • 完全房室ブロック（Ⅲ度） → 高度房室ブロック
  • Mobitz II 型・2:1 房室ブロック・連続脱落の高度房室ブロック → 高度房室ブロック
  • 交代性脚ブロック（RBBB と LBBB が時系列で入れ替わる等）
  • 上記に関わらず 血行不安定・心室率が著しく遅い・逸搏QRSが寛い のいずれかを伴う徐脈
最短の行動
  1. 循環器オンコール
  2. 同時に：モニター、末梢ルート、採血（K / Mg / Ca / トロポニン）
  3. 原因の手掛かり（旧心電図と内服確認）

同日相談（今日中に循環器）

今夜カテ即ではない が、外来に逃がさず日中に方針を取る。

虚血・AF・心不全

  • Wellens症候群（V2–V3の二相性T / 深いT陰性）
  • 広範なST低下で NSTE-ACS を疑う所見 + トロポニン動態
  • 新規心房細動
  • 新規の著明な左室肥大所見 + 心不全 / 高血圧急症 / 腎障害

伝導障害で同日相談に入るもの

判断に困うときは相談してよい。入院・転倒・失神の文脈では閾値を下げる。

  • 二枝ブロック（右脚ブロック + 左脚前枝または後枝）+ Ⅰ度房室ブロック で、旧ECGなし／比較不能、または 転倒・失神・多薬併用・腎障害 のいずれか → 同日循環器（夜間は施設ルール：オンコールに出すか、記録＋翌朝フォローか）
  • Mobitz I（Wenckebach） で狭いQRS・血行安定でも、初見・原因不明・内服が濃い → 同日相談または翌朝までの明確なフォロー指示

即に寄せる／同日でよいの境界が迷ったら、血行と逸搏の形で即寄りに倒す。

電話前に詰める（旧ECG・内服・既往）

「新規か？」に固執しない。夜間・休日・非内科医でも回せる順で、循環器に渡す材料を揃える。

詳細は 旧心電図と内服確認。

相談先の決め方

入院中の初回判断は 原則として循環器（院内）。退院後の振り分けは評価後。

循環器コンサルトの判断

ピットフォール

  • ST上昇だけを探す（de Winter・Wellens・後壁を見落とす）
  • 無症状だから安心する（糖尿病高齢者・鎮静下で症状が出ない）
  • 数値上の徐脈だけで循環器に回す／回さないを決める
  • 旧ECGを探さないまま「新規」と報告する
  • 内服・既往（心疾患・透析・ペースメーカ）の確認漏れ
  • 心電図自動診断レポートを鵜呑みにする
