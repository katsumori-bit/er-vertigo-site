🚨 即相談（今すぐ電話）

無症状でも 以下があれば、夜間・休日でも循環器オンコールを検討。

虚血・電解質・チャネル病

  • VT / VF（モニター上で明らかなため詳細リンクは省略）
  • STEMI 基準のST上昇
  • de Winterパターン
  • 後壁梗塞（V1–V4のST低下 + 高いR波 / 直立T）
  • 高K血症心電図 の進行所見
  • Brugada心電図 1型様（新規・強疑）
  • QT延長 で著明（QTc ≥500 ms）＋ 誘発因子

伝導障害で即相談に入るもの

  • 完全房室ブロック（Ⅲ度） → 高度房室ブロック
  • Mobitz II 型・2:1 房室ブロック・連続脱落の高度房室ブロック → 高度房室ブロック
  • 交代性脚ブロック（RBBB と LBBB が時系列で入れ替わる等）
  • 上記に関わらず 血行不安定・心室率が著しく遅い・逸搏QRSが寛い のいずれかを伴う徐脈
最短の行動
  1. 循環器オンコール
  2. 同時に：モニター、末梢ルート、採血（K / Mg / Ca / トロポニン）
  3. 原因の手掛かり（旧心電図と内服確認）

💔 虚血パターン（STEMI / NSTE-ACS）

STEMI 基準（要点）

  • 連続2誘導での ST 上昇 — V2–V3 で男性≥2 mm（≥40歳）/ ≥2.5 mm（<40歳）/ 女性≥1.5 mm、その他誘導≥1 mm
  • 新規 LBBB は虚血と区別困難 → Sgarbossa 基準を当てる
  • 詳細 → STEMI

STEMI 等価所見（見落とし注意）

  • de Winterパターン — V1–V6 で J 点 ST 低下 + 高い T 波 + aVR の ST 上昇
  • 後壁梗塞 — V1–V4 の ST 低下 + 高い R 波 + 直立 T。後壁誘導（V7–V9）追加で確認
  • aVR の ST 上昇 + 広範な ST 低下 — 左主幹部 / 三枝病変の可能性

NSTE-ACS を疑う所見

  • Wellens症候群（V2–V3 の二相性 T / 深い T 陰性）
  • 広範な ST 低下 + トロポニン動態
  • 新規 T 波陰転、症状連動の動的変化
ST 上昇＝STEMI ではない

早期再分極、心膜炎、Brugada、左室肥大、LBBB、ペーシング — いずれも ST 上昇を起こす。分布・形・経時変化・症状・トロポニンで総合判断。迷ったら循環器に相談。

⚡ 致死性不整脈（VT / VF / Wide QRS）

すぐに見るもの

  • VF — 不規則な細動波 → 直ちに除細動 + CPR
  • 持続性 VT（≥30 秒 or 血行不安定）— 同期下カルディオバージョン or 薬物
  • Wide QRS 頻拍 — 原則 VT として対応（特に虚血・心不全既往あり）
  • Torsades de Pointes — Mg 静注、QT 延長薬中止、誘発因子是正

基質を疑う所見

  • Brugada心電図 1 型（V1–V2 で coved 型 ST 上昇 + T 陰転）
  • QT延長（QTc ≥500 ms は TdP リスク）
  • 高K血症心電図 進行（テント T → P 消失 → QRS 拡大 → sine wave）
Wide QRS に迷ったら VT

Wide QRS 頻拍を「SVT + 変行伝導」と決めつけて Ca 拮抗薬を入れると、本物の VT で血行破綻する。分からなければ VT として扱うのが安全。

🧱 伝導障害（房室ブロック / 脚ブロック）

房室ブロックの分類

分類所見対応
Ⅰ度PR >200 ms、脱落なし原則経過観察、原因薬剤確認
Ⅱ度 Mobitz I（Wenckebach）PR が徐々に延長 → 脱落狭 QRS・血行安定なら経過観察。初見・原因不明・内服濃いなら同日相談
Ⅱ度 Mobitz IIPR 一定で突然脱落、しばしば広 QRS即相談。ペーシング検討
2:1 / 高度房室P 波の半数以上が脱落即相談
Ⅲ度（完全）P と QRS が完全解離即相談、ペーシング

脚ブロック・分枝ブロック

  • 新規 LBBB — 虚血と区別困難。症状連動なら STEMI 等価として扱う
  • 二枝ブロック（RBBB + 左脚前枝 or 後枝） + Ⅰ度房室ブロック、旧 ECG なし or 失神・転倒・多薬・腎障害ありなら同日相談
  • 交代性脚ブロック — 即相談（高度房室への進行リスク）
境界例で迷ったら

血行と逸搏 QRS の形で即寄りに倒す。心室率が遅い・逸搏 QRS が寛い・血行不安定 — どれか一つでも即相談。

📅 同日相談（今日中に循環器）

今夜カテ即ではない が、外来に逃がさず日中に方針を取る。

虚血・AF・心不全

  • Wellens症候群（V2–V3 の二相性 T / 深い T 陰性）
  • 広範な ST 低下で NSTE-ACS を疑う所見 + トロポニン動態
  • 新規心房細動
  • 新規の著明な左室肥大所見 + 心不全 / 高血圧急症 / 腎障害

伝導障害で同日相談に入るもの

判断に困うときは相談してよい。入院・転倒・失神の文脈では閾値を下げる。

  • 二枝ブロック（右脚ブロック + 左脚前枝または後枝）+ Ⅰ度房室ブロック で、旧 ECG なし／比較不能、または 転倒・失神・多薬併用・腎障害 のいずれか → 同日循環器（夜間は施設ルール：オンコールに出すか、記録＋翌朝フォローか）
  • Mobitz I（Wenckebach） で狭い QRS・血行安定でも、初見・原因不明・内服が濃い → 同日相談または翌朝までの明確なフォロー指示

即に寄せる／同日でよいの境界が迷ったら、血行と逸搏の形で即寄りに倒す。

📖 読み方の順序（6 ステップ）

「目に入った所見から飛びつく」ではなく、毎回同じ順で見る。

  1. リズム — 洞調律か / P-QRS 関係 / 規則性 / レート
  2. 軸 — 正常 / 左軸偏位 / 右軸偏位（Ⅰ・aVF の向きで判定）
  3. 間隔 — PR（120–200 ms）、QRS（<120 ms）、QT / QTc（男性 <450、女性 <460 ms 目安）
  4. ST-T — 上昇 / 低下 / T 波の向き、分布（前壁 / 下壁 / 側壁）
  5. 房室ブロック — PR 延長、Wenckebach、Mobitz II、完全房室
  6. 心室肥大 — Sokolow-Lyon、Cornell、左房負荷所見

正常変異 vs 病的の区別

  • 早期再分極 — 若年男性、V2–V5 で凹型 ST 上昇 + notch / slur — 通常良性
  • 若年男性の生理的 ST 上昇 — 1–2 mm、症状なし、経時変化なし
  • 運動選手の徐脈・Ⅰ度 AVB・早期再分極 — しばしば生理的
  • ただし失神既往・突然死家族歴があれば「正常変異」と決めつけない
過去の心電図と比較

「新規変化」は強い情報。同じ所見でも過去にあれば慢性、なければ急性病態を疑う。電子カルテで必ず旧 ECG を呼び出す。

📞 電話前に詰める（旧 ECG・内服・既往）

「新規か？」に固執しない。夜間・休日・非内科医でも回せる順で、循環器に渡す材料を揃える。

そろえるもの

  • 旧 ECG（電子カルテ・他院紹介状・お薬手帳の写真）
  • 内服（抗不整脈薬、QT 延長薬、β 遮断薬、Ca 拮抗薬、ジゴキシン、利尿薬、抗うつ薬）
  • 既往（心疾患、透析、ペースメーカ / ICD、失神・転倒、突然死家族歴）
  • 採血（K / Mg / Ca / Cr / トロポニン / TSH）
  • バイタルと症状の有無（無症候か / 失神 / 動悸 / 胸痛）

詳細は 旧心電図と内服確認。

相談先の決め方

入院中の初回判断は 原則として循環器（院内）。退院後の振り分けは評価後。

→ 循環器コンサルトの判断

コラム：「入院だから全例 12 誘導」はエビデンスで絶対か

多くの病院では 運用・安全網（無症候性虚血・伝導障害・AF の拾い、院内急変時のベースライン）として 全例取得が慣習化している。一方、「全入院患者にルーチン 12 誘導を取れば転帰が改善する」と示した大規模 RCT のような強いエビデンスは乏しい。無症状一般成人のスクリーニング心電図については、米国予防サービス作業部会（USPSTF）も 利益が不明確としている。 → 「病院ルールだから取る」は正しいが、「ルールだから必ず臨床的に意味がある」とは限らない。 取ったあとの判断は別問題。

⚠️ ピットフォール

  • ST 上昇だけを探す（de Winter・Wellens・後壁を見落とす）
  • 無症状だから安心する（糖尿病高齢者・鎮静下で症状が出ない）
  • 数値上の徐脈だけで循環器に回す／回さないを決める
  • 旧 ECG を探さないまま「新規」と報告する
  • 内服・既往（心疾患・透析・ペースメーカ）の確認漏れ
  • 心電図自動診断レポートを鵜呑みにする
  • Wide QRS 頻拍を SVT + 変行と決めつけて Ca 拮抗薬を入れる
  • 新規 LBBB を「ただの脚ブロック」と流す（症状あれば STEMI 等価）
