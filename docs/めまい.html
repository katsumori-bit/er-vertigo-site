⚠️ トリガー連動の注意（条件で意識）

特定の状況に出会ったら必ず思い出すべき注意点。フローの判断と並行して意識する。

致死的疾患を疑うトリガー

  • if抗凝固薬内服
    中枢性（出血性卒中・椎骨動脈解離）を強く疑う。CT 閾値を下げる
  • if強い後頭部痛 / 頸部痛
    椎骨動脈解離・SAH を必ず想起。画像評価へ
  • if胸痛 / 動悸 / 失神前合併
    循環器ルートへ分岐。心原性失神・不整脈・大動脈解離。ECG・モニター
  • if持続性 + 難聴 / 耳鳴
    AICA 梗塞 を必ず鑑別。「難聴=末梢」と決めつけない
  • if抗凝固服用 + 軽微外傷歴
    慢性硬膜下血腫・外傷性出血を想起。頭部 CT 閾値を下げる

分類・判断のトリガー

  • if高齢 + 多血管リスク
    AVS で末梢示唆所見でも中枢を優先して評価
  • ifAVS で HINTS 判定困難（中間例）
    中枢扱いとして MRI へ（後悔最小化）
  • if典型的でない BPPV（眼振非典型・反応乏しい）
    t-EVS と決め打ちせず、中枢の可能性を保持
  • if症状改善後の評価
    改善 = 末梢の保証ではない。TIA・Ménière 発作後を考慮、観察継続
  • if回転性主訴
    末梢を直感で選ばずTiTrATE で再分類（後方循環梗塞も回転性で来る）
  • if反復性短時間 + 体位誘発
    Dix-Hallpike 適応判断。後半規管 BPPV なら Epley
  • if高齢 / 脱水 / 利尿薬 / 前失神型
    静脈採血 + 12誘導 ECG を追加
  • if頭部 CT 正常 + 後方循環疑い
    中枢除外にはならない。MRI/MRA・再評価
  • ifMRI 早期陰性 + 強い臨床疑い
    24-48h 後再撮影、入院観察
  • if独居 / フォロー困難
    帰宅閾値を上げる。disposition の重要トリガー

ピットフォール（決め打ち回避）

  • ⚠HINTS の誤適用
    HINTS はAVS 限定。t-EVS（BPPV など）への流用は適応外
  • ⚠「歩けないめまい」を軽視
    体幹失調は中枢性の重要所見。歩行不能・起立不能は中枢扱いの強い根拠
  • ⚠BPPV と決めた後の持続性
    Epley 後も改善せず持続するなら分類を疑う。AVS への再評価も

🚨 中枢性確認（red flags で末梢で片付けない）

これがあれば末梢で片付けない

  • 歩行不能 / 起立不能、強い体幹・四肢失調
  • 構音障害・複視・嚥下障害・片麻痺・感覚障害
  • 垂直眼振 / 方向交代性眼振
  • 強い後頭部痛 / 頸部痛
  • 新規難聴 + 中枢所見
  • 高齢 + 血管危険因子
  • 抗凝固薬内服
  • 失神前状態 / 胸痛 / 動悸
1 つでもあれば必ず鑑別を広げる

除外すべき主な致死的疾患：

  • 後方循環梗塞（小脳梗塞 / 延髄梗塞 / 橋梗塞 / AICA梗塞）
  • 椎骨動脈解離 / くも膜下出血 / 急性脳出血（抗凝固薬例・重度神経症状）
  • 心原性失神 / 致死性不整脈 / 大動脈解離

🎯 めまいの分類（4症候群）

「回る vs ふわふわ」の質より、持続時間・反復性・誘因・随伴症状・ターゲット診察で分類する。

症候群典型代表疾患
失神前状態 眼前暗黒感、立位で悪化、動悸、胸痛 起立性低血圧 / 不整脈 / 低血糖 / 脱水 / 貧血 / 薬剤性めまい
AVS（急性持続性） 急性発症・24 h 以上持続・悪心嘔吐・眼振 or 歩行不安定 前庭神経炎 / 後方循環梗塞 / 小脳梗塞 / 脳幹梗塞 / AICA梗塞
t-EVS（頭位誘発短時間） 頭位変換で数秒〜1 分、発作間は軽快 BPPV、まれに中枢性
s-EVS（誘因なし反復） 誘因なく発作、発作間は軽快 Ménière病 / 前庭性片頭痛 / TIA / 不整脈 / 起立性低血圧
分類で workflow が決まる

分類できたら → §4「症候群別の対応」へ。各症候群で診察・検査・disposition の組み立てが違う。

Newman-Toker DE et al. Neurol Clin 2015（TiTrATE）

🩺 最小診察 + HINTS

その場で行う最小セット

  • バイタル（失神前疑いなら臥位／立位）
  • 座位・立位・歩行（保てる？立てる？歩ける？）
  • 眼振（自発／頭位誘発、方向、固視）→ 眼振の見方
  • 聴力（指こすり・ささやき）
  • 神経診察
  • 採血（血糖・電解質・Hb）
  • 12 誘導 ECG（胸痛・動悸・失神前・不整脈懸念）
HINTS は AVS 限定

頭部衝動・眼振・偏視（test of skew）のベッドサイドセット。AVS で適応と手技の訓練がそろったときの追加。t-EVS（BPPV）には使わない。手技・解釈 → HINTS

HINTS の解釈
  • 末梢パターン：頭部衝動陽性（catch-up saccade あり）+ 一方向性水平眼振 + 偏視なし
  • 中枢パターン（INFARCT）：頭部衝動陰性 / 方向交代性眼振 / 偏視あり — 1 つでもあれば中枢性
  • 判断困難なら中枢性として扱う

🔄 症候群別の対応

1. 失神前状態
  • 採血（血糖含む）、12 誘導 ECG、モニター
  • 起立性変化を確認、胸部症状があれば循環器評価
  • 不整脈疑いなら入院モニター
  • 詳細 → 失神前状態 / 意識消失
2. AVS（急性持続性）
  • HINTS（適応があれば）、聴力、座位・立位・歩行、神経診察
  • 末梢性示唆：一方向性水平眼振、明らかな神経脱落症状なし、難聴なし、HINTS 末梢パターン
  • 中枢性示唆：歩行不能 / 強い体幹失調、垂直眼振、方向交代性眼振、HINTS 中枢パターンまたは判断困難、構音・複視・嚥下・片麻痺、頭痛 / 頸部痛、血管危険因子強い
AVS の原則

CT をルーチンには使わない（後方循環虚血に感度低い）。 ベッドサイドで末梢示唆なら経過観察、中枢示唆 or 判断困難なら MRI / MRA。

3. t-EVS（頭位誘発）
  • Dix-Hallpike、必要に応じて Supine roll test
  • 後半規管 BPPV なら Epley 法で治療
  • 詳細 → BPPV / Dix-Hallpike / Epley法
典型的 BPPV の原則

不要な画像検査や漫然とした前庭抑制薬は避ける。Epley で改善することを確認して帰宅。

4. s-EVS（誘因なし反復）
  • 脳神経診察、視野、眼球運動、四肢協調運動、歩行、聴覚症状、血管危険因子、胸痛 / 動悸 / 失神
  • 詳細 → Ménière病 / 前庭性片頭痛 / TIA
s-EVS での画像

中枢・末梢の区別に CT は使わない。TIA・大血管病変が気になれば血管評価。

🧠 疾患別ポイント（入口）

疾患一言特徴詳細
前庭神経炎 AVS の代表、持続性めまい + 悪心、難聴なし。ステロイドは全例必須ではなく、発症 3 日以内で短期投与を相談 前庭神経炎
BPPV t-EVS の代表、Dix-Hallpike → Epley で治療、画像・前庭抑制薬は漫然と使わない BPPV
Ménière病 自発性発作 20 分〜12 h、低〜中音感音難聴 + 耳鳴 / 耳閉。難聴があっても末梢と決めつけない Ménière病
後方循環梗塞 AVS で見逃してはいけない。めまい単独に見える。歩行不能・体幹失調・眼振パターン・HINTS 異常が重要。早期 MRI でも見逃しあり 後方循環梗塞
前庭性片頭痛 反復するめまい + 片頭痛既往、視覚症状や畏光・畏音を伴うことがある 前庭性片頭痛

🔬 検査の置き方

  • 採血：血糖、電解質、Hb、脱水・感染の疑いで適宜
  • 12 誘導 ECG：失神前・胸痛・動悸・不整脈懸念
  • 頭部 CT：出血疑い / 外傷 / 激しい頭痛 / 局所神経症状（後方循環虚血の除外には弱い）
  • MRI / MRA：AVS で中枢性疑い、HINTS 中枢 or 判断困難、高リスク背景、局所神経症状（早期陰性でも絶対安心ではない）
CT で「異常なし」≠ 中枢性除外

後方循環虚血（小脳梗塞・脳幹梗塞）は 急性期 CT で写らないことが多い。臨床所見が中枢性を示唆するなら CT 正常でも MRI/MRA を回す。

📤 disposition

帰宅を考えやすい

  • 典型的 BPPV で Epley 後に改善、歩行可、経口可
  • 中枢性 / 循環器性の所見が乏しい
  • 独居でなく、フォロー可能

入院 / コンサルト

  • 中枢性を否定できない（HINTS 中枢 / 判断困難 / 神経所見）
  • 歩行不能、持続する嘔吐
  • 局所神経症状あり、高リスク背景（高齢 + 血管危険因子 / 抗凝固薬）
  • 独居などで安全性が低い
  • 不整脈疑いの失神前型

⚠️ ピットフォール

  • 「回転性だから末梢」と決めつける — 後方循環梗塞も回転性めまいで来る
  • CT 正常で安心する — 後方循環虚血に弱い
  • AVS 以外で HINTS を使う — t-EVS には適応なし
  • 「難聴 = 中枢性ではない」と思い込む — AICA 梗塞では難聴を伴う
  • 症状改善 = 安全と考える — TIA や Ménière 発作後
  • 歩けないめまいを軽視する — 体幹失調は中枢性の重要所見
