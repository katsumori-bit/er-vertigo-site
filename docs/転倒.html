🚨 1. 命に関わるか

すぐ動く赤旗
  • 意識レベル低下（GCS ≤ 14）、瞳孔不同、新規の片麻痺・けいれん → 頭蓋内出血
  • ショック、止まらない出血
  • 大腿骨骨幹部の変形（出血 1–1.5 L、直達牽引が必要）
  • 頸部痛 → 頸髄損傷の可能性、頸椎保護を継続
  • 失神が先行している転倒 → 致死性不整脈・PE・出血・SAH・解離
  • 抗凝固薬・抗血小板薬内服中の頭部打撲 → 軽症でも頭蓋内出血を疑う

🦴 2. 後遺症を残さないか

「命は助かっても、転倒前の生活に戻れない」を防ぐ視点。神経・機能・社会の 3 層で考える。

後遺症の 3 層
  • 神経学的後遺症：中心性頸髄損傷、慢性硬膜下血腫（抗凝固薬内服者で 24–72 時間後も含め注意）、びまん性軸索損傷による高次脳機能障害
  • 機能的後遺症：大腿骨近位部骨折・脆弱性骨盤骨折・脊椎圧迫骨折による歩行喪失、入院中のせん妄・廃用、橈骨遠位端骨折後の握力低下、肩の可動域制限
  • 社会的後遺症：独居継続困難、介護度の上昇、再転倒

見逃すと上記に直結する病態と入口：

見逃したくない病態入口
頸髄損傷（高齢者の軽微外傷でも中心性頸髄損傷が起こる）頸部外傷・頸部痛
慢性硬膜下血腫（受傷数週後の性格変化・歩行障害）頭部外傷評価
大腿骨近位部骨折（歩行できても否定できない occult あり）股関節痛 / 大腿骨近位部骨折
脆弱性骨盤骨折・脊椎圧迫骨折（ADL 喪失の主因）脆弱性骨盤骨折 / 脊椎圧迫骨折
上肢骨折（橈骨遠位端・上腕骨近位部・鎖骨）手関節・手の外傷 / 肩の外傷・痛み
下肢骨折（膝・足関節）膝の外傷・痛み / 足関節・足部の外傷

🔍 3. 転倒の原因は何か

「転んだ」のか「倒れた」のか。失神が先行していれば、外傷の治療だけで帰してはいけない。

原因検索の手順
  1. 本人・目撃者の証言から失神の否定（−）を確認
  2. 否定できない → syncope / syncope mimic の検索（ECG・心腹部エコー・起立試験・SpO2）
  3. 何も見つからない → 逆行性健忘の可能性（頭部打撲で記憶が飛び「つまずいただけ」になっている）
原因確認すること
つまずき・滑り環境要因（段差・暗さ・履物）
起立性低血圧臥位→立位で収縮期 20 mmHg 以上低下。降圧薬・脱水
失神・前失神目撃者の証言、前駆症状、ECG
迷走神経反射長時間立位、排尿後、食後、痛み刺激
不整脈ECG：VT/VF、AF、III 度 AVB、Mobitz II、SSS、長い pause

薬剤性睡眠薬・ベンゾ・降圧薬・利尿薬・オピオイド・血糖降下薬
アルコール飲酒歴、臭い、血中アルコール

PE胸痛、SpO2 低下
失神が先行している転倒

「転んだ」のではなく「倒れた」可能性がある。目撃者なし + 受傷機転が不明確 + 外傷のわりに防御反応がないなら失神を疑う。ECG は必ず取る。心疾患の既往があれば循環器へ相談も検討（ESC 2018 失神ガイドライン）。

🛠 外傷のマネジメント方針（骨折ハンター式 5 択）

上肢外傷は原則「帰宅」、下肢外傷は原則「入院」

上肢は歩けるので通院可、手術も待機的。例外は 開放骨折 / 整復不能な脱臼 / 血管・神経合併 の 3 つのみ。下肢は荷重困難なら入院マネジメント。

分類定義典型例
即時整形夜間・休日でも呼ぶ開放骨折、整復不能な脱臼、血管・神経損傷、脊椎破裂骨折、大腿骨骨幹部骨折、脆弱性骨盤骨折で血行動態不安定
入院整形入院 → 後日・翌朝コンサルト大腿骨近位部骨折（夜間）、脆弱性骨盤骨折、脊椎圧迫骨折で生活困難、occult 疑いの経過観察
帰宅手術待機手術、いったん帰宅可転位の強い橈骨遠位端骨折、上腕骨近位部骨折 2 part 以上
帰宅保存保存加療で外来フォロー転位の少ない橈骨遠位端骨折、安定型の上腕骨近位部骨折、鎖骨骨折、横突起骨折
再診不要短期保存で済む打撲、軽度の指骨折、scrape
整形コンサルトの作法（スムーズに通すコツ）
  • 診断名は「大腿骨近位部骨折」ではなく「大腿骨頸部骨折」「大腿骨転子部骨折」と区別して伝える
  • 頸部骨折なら Garden 分類を添える（転子部は不要）
  • 橈骨遠位端骨折なら 活動性（ADL）と関節内／外を紹介状に記載
  • 上腕骨近位部骨折なら ADL + CT（できれば）
  • occult 疑いなら「内外旋で疼痛あり、MRI で STIR 白・T1 黒です」と言えると通る
  • 夜間・週末を避けて overnight で翌朝コンサルトできる病態は多い
骨折ハンター式 TIPS（現場で効く）
整形外傷 5 箇条
  1. 画像より先に鑑別を挙げる（年齢 + 受傷機転）
  2. 骨折線イメージを持ってから読影する
  3. 「Xp 異常なし」で止めない。occult を疑ったら CT / MRI
  4. マネジメントは 5 択（即時 / 入院 / 帰宅手術 / 帰宅保存 / 再診不要）で決める
  5. コンサルトは 分類名 + 骨折線 + ADL を添える
  • 外旋 + 短縮位を見たら身体所見を省略して Xp 直行でもよい（大腿骨近位部骨折）
  • 恥骨・坐骨・仙骨は体表から触れる骨 → 圧痛があれば骨折と診断してよい
  • 股関節・膝・肩のような関節部は触れないので他動運動で代用（内外旋痛 = 大腿骨近位部）
  • Xp はほぼ必ず 2 方向（正面 + 側面、股関節は正面 + 軸位）
  • CT で骨折線を見つけるコツ：3 断面のうち 2 つで見える、2–3 スライス連続、骨皮質の途絶
  • MRI で新鮮骨折 = STIR 白・T1 黒（筋挫傷も同パターン、場所が骨か筋肉かで区別）
  • 高齢者転倒後の骨密度介入漏れ：既往があって BP 未処方なら介入機会

📤 disposition / 帰宅説明

帰宅できる条件

  • 意識清明、新規神経所見なし
  • 頭部 CT が正常、または CT 不要と判断（→ 頭部外傷評価）
  • 骨折がない、または保存的に帰宅管理できる骨折
  • 歩行できる、日常生活が自力でできる
  • 転倒原因が明らかで重篤でない
  • 抗凝固薬内服中でも CT 正常 + 経過良好なら帰宅可（遅発性出血の説明が必須）

入院 or 専門科コンサルト

  • 頭蓋内出血 → 脳神経外科
  • 頸椎・頸髄損傷の疑い → 整形外科 / 脊椎外科
  • 大腿骨近位部骨折 → 整形外科（48 時間以内、可能なら 24 時間以内の手術が目標）
  • 大腿骨骨幹部骨折・脊椎破裂骨折 → 即時整形（直達牽引）
  • 開放骨折 / 整復不能な脱臼 / 血管・神経損傷 → 即時整形
  • 脊椎圧迫骨折 / 脆弱性骨盤骨折で生活困難 → 入院整形
  • 失神の精査が必要 → 入院 or 循環器
  • 歩行不能 / ADL が維持できない → 入院
  • 抗凝固薬内服中 + 頭部打撲 → 経過観察入院を検討
  • 独居高齢者で見守り困難 → 環境調整が済むまで入院

帰宅時の説明

  • 頭部打撲時 → 頭部外傷評価 の再受診基準を必ず伝える（嘔吐・意識変容・強い頭痛・数日後の性格変化や歩行障害）
  • 骨折なしでも痛みが続く場合 → 再受診して再評価（occult / 不全骨折の可能性）
  • 上肢外傷で固定した場合 → 指の屈伸を積極的に、挙上、しびれ・蒼白・強痛で即受診
  • 肩関節脱臼整復後 → Bankart / Hill-Sachs 損傷の可能性を説明して整形フォロー

再転倒予防の視点（退院・帰宅時）

  • 起立性低血圧の誘発薬（α 遮断薬・ニトロ・利尿薬）を減量
  • ベンゾジアゼピン・睡眠薬の整理
  • 白内障・視力・聴力の確認
  • 住環境（段差・照明・浴室手すり）の見直し
  • 骨粗鬆症治療（BP 未処方ならビスホスホネート導入の提案）

#高齢者の転倒 s/o ●●による△△

●転倒原因検索：本人・目撃者の証言から失神の否定（−） → syncope / syncope mimic の検索

  • 心腹部エコー：low EF / ACS / HOCM / 心タンポ / AD / AAA
  • ECG：VT/VF、AF、III 度 AVB、Mobitz II、SSS、pause
  • その他：PE（胸痛・SpO2）、迷走神経反射、起立性低血圧
  • 何もないなら 逆行性健忘 の可能性

●外傷評価 → マネジメント

  • ABCD：気道 / フレイルチェスト / 緊張性気胸 / FAST
  • 頭部：CT 適応、抗凝固薬確認
  • 頸部：正中圧痛、頸部外傷・頸部痛 の MRI 不要判定 ①②③
  • 外傷性頸部症候群（むち打ち様）：高齢者は遅発症状に注意
  • 胸腹部：胸郭圧痛、呼吸音左右差、腹膜刺激、抗凝固薬内服中は腹腔内出血も
  • 腰背部：CFPS、傍脊柱筋圧痛（横突起）
  • 骨盤・股関節：恥坐仙骨圧痛、内外旋痛、大転子部血腫
  • 四肢：部位別の鑑別 → 骨折線イメージ → 画像照合

⚠️ 見落としやすいもの・やりがちなミス

見落としやすいもの

  • 後頭部の打撲・裂創（仰臥位のまま確認していない）
  • 大腿骨近位部の occult 骨折（初回 X 線で見えない → 痛みが続けば CT / MRI）
  • 脆弱性骨盤骨折の仙骨部（Xp 感度 7–44%、恥坐骨が折れていれば仙骨も疑う）
  • 中心性頸髄損傷（高齢者の軽微外傷でも起こる）
  • 脊椎圧迫骨折（背部痛として訴える、初回 Xp で責任椎体を決めない）
  • 失神が原因の転倒（「つまずいた」で片付けている）
  • 抗凝固薬の確認漏れ
  • 慢性硬膜下血腫（数週後の認知低下・歩行障害で気づく）
  • インプラント周囲骨折（術後患肢の転倒疼痛で必ず鑑別に）
  • 舟状骨骨折（嗅ぎたばこ窩圧痛、初回 Xp 陰性率が高い）
  • Maisonneuve 骨折（足関節捻挫と思ったら膝下まで触る）
  • 鎖骨骨折（肩 2 方向だけだと写らない、鎖骨 2 方向を追加）
  • 大腿骨骨幹部骨折（出血 1–1.5 L、大腿神経ブロック + 直達牽引）

やりがちなミス

  • 「頭を打っていない」という本人の言葉だけで CT を省略する（目撃者がいなければ否定できない）
  • 頭部だけ見て四肢・骨盤を診ない
  • 外傷対応だけして転倒原因を考えない
  • 高齢者の「歩ける」を過信して帰宅させる（翌日歩けなくなる occult 骨折あり）
  • 抗凝固薬内服者の頭部外傷で帰宅時説明が不十分
  • Xp で異常なし → 思考停止（鑑別を先に挙げていないから）
  • 脊椎圧迫骨折疑いで楔形変形を見つけて満足（陳旧性の可能性）
  • コンサルト時の診断名が「大腿骨近位部骨折」（頸部 or 転子部で伝えないと整形が嫌がる）
  • NSAIDs を高齢者に出してしまう（腎障害・消化管出血）
  • 歩行 = 骨折否定と考える（occult の大腿骨近位部骨折でも歩けることがある）
