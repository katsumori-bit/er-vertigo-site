頸部外傷・頸部痛

⚡ 即時アクション
  1. 頸椎固定を継続（カラー / 用手 in-line 固定）— NEXUS / Canadian C-spine で除外できなければ解除しない
  2. 気道・呼吸を最優先 — C3–5 横隔神経＝呼吸停止リスク。低換気・腹式呼吸消失に注意
  3. 外傷あり vs 外傷なしで大きく分岐。非外傷の致死的頸部痛（髄膜炎・SAH・椎骨動脈解離）も常に頭に置く
  4. 椎骨動脈損傷を念頭 — 高エネルギー・回旋外傷・頸椎横突孔骨折で CTA を検討
🚨Red flags
命・赤旗 🧭鑑別の枠組み
外傷あり / なし 🦴主要病態詳細 🩺身体所見・画像
NEXUS / CCSR 💊マネジメント 📤disposition 📝カルテテンプレ ⚠️見落とし・ミス 📚出典・関連

🚨 Red flags（命と後遺症を外す）

これがあれば固定を緩めない・即時介入
  • 呼吸不全・腹式呼吸消失 — C3–5 横隔神経。SpO2 低下・低換気は 高位頸髄損傷を示唆
  • 四肢麻痺・対麻痺・感覚レベル（不全麻痺含む）
  • 神経原性ショック（高位頸髄）— 徐脈 + 低血圧 + 末梢温暖
  • 頸部正中圧痛・新規の頸部変形
  • 意識障害・酩酊を伴う頸部外傷（評価困難 → 固定継続）
  • 高エネルギー外傷（高所転落・高速衝突・歩行者と車・車外放出）
  • 高齢者の軽微外傷でも中心性頸髄損傷を起こしうる（過伸展機転）
  • 抗凝固薬内服例 — 微小外傷でも頸部血腫・硬膜外血腫
  • 後頸部の打撲機転＋神経所見・めまい → 椎骨動脈損傷
  • 皮下気腫・嗄声・嚥下時痛・吐血 — 喉頭・気管・食道損傷

非外傷でも忘れない致死的頸部痛

「ただの寝違え・凝り」と片付けない
  • 髄膜炎 — 発熱 + 項部硬直 + 意識変容。Kernig / Brudzinski
  • くも膜下出血 — 突発の頭頸部痛、emesis、意識消失先行
  • 椎骨動脈解離 — 後頭部〜後頸部痛、回旋誘発、めまい・後方循環梗塞
  • 咽後膿瘍・頸部蜂窩織炎 — 嚥下時痛、開口障害、発熱
  • 頸動脈解離 — 一側性頸部痛、Horner、片麻痺
  • 頸髄硬膜外膿瘍 — 発熱 + 進行性の四肢麻痺、糖尿病・透析・薬物使用歴
気道介入は in-line stabilization が原則

頸椎損傷疑いでの挿管は用手 in-lineで行う。過度の頸部進展を避ける。ビデオ喉頭鏡が有用。輪状軟骨圧迫は喉頭損傷例では避ける。

並行して外す

  • 頭部の合併損傷 → 頭部外傷評価
  • 失神先行の転倒 → 意識消失
  • 意識障害下で評価困難 → 意識障害

🧭 鑑別の枠組み

まず外傷の有無で大きく二分する。非外傷の致死的鑑別を逃さないことが第一歩。

外傷あり

カテゴリ典型像主な病態

「外傷なし」と決めつけない

高齢者・酩酊・認知症では受傷歴が拾えないことがある。頸部正中圧痛と神経所見を必ず取り、機転不明でも疑いがあれば画像へ進む。

🦴 主要病態詳細

上位頸椎（C1–2）の骨折

病態機転特徴・注意
Jefferson 骨折（C1 破裂骨折）軸圧（ダイブ・物が頭上から）C1 前後弓の破裂。横靱帯断裂で不安定型。Open mouth 像で側方転位
歯突起骨折（C2）過屈曲・過伸展。高齢者の軽微外傷Type II（基部）が最多で偽関節になりやすい。高齢者で見落としやすい
Hangman 骨折（C2 椎弓根）過伸展 + 軸圧（自動車事故）両側 pars 骨折。多くは神経学的に保たれるが不安定

下位頸椎（C3–7）の損傷

  • 圧迫骨折：屈曲 + 軸圧。前方楔状変形。比較的安定だが破裂で後方転位なら不安定
  • 破裂骨折：軸圧。後方の骨片が脊柱管に侵入 → 脊髄損傷
  • 屈曲涙滴骨折：屈曲 + 圧迫。前方下端の三角骨片。脊髄損傷の合併が多い
  • 伸展涙滴骨折：過伸展。前縦靱帯付着部の小骨片。比較的良好
  • 両側 facet 脱臼：屈曲 - 牽引。50% 以上の前方転位、強い不安定性、脊髄損傷高率
  • 片側 facet 脱臼：屈曲 + 回旋。25% の前方転位、神経根症状

頸髄損傷

完全 vs 不完全（ASIA 分類）
  • ASIA A 完全：S4–5 を含めた感覚・運動全消失
  • ASIA B：感覚は保たれ運動消失
  • ASIA C/D：運動が部分的に保たれる（C：主要筋群の半数以上が grade < 3、D：grade ≥ 3）
  • ASIA E：正常

初期は脊髄ショックでフラットになっている可能性。球海綿体反射の有無で区別を試みる（24–48 h で回復）。

不完全頸髄損傷の症候群

症候群典型像機転
中心性頸髄損傷上肢 > 下肢の運動麻痺、手指の巧緻運動低下、温痛覚障害が斑状高齢の過伸展（顔面から転倒）。CT で骨傷なし → MRI
前脊髄症候群運動完全麻痺 + 温痛覚消失、振動・位置覚保たれる軸圧、前脊髄動脈虚血
Brown–Séquard同側運動・深部感覚消失、対側温痛覚消失穿通性。一側髄損傷
後脊髄症候群後索の振動・位置覚障害、運動保たれる稀。過伸展で後方圧迫

外傷性頸部症候群（むち打ち / Quebec 分類）

Quebec Task Force 分類
  • Grade 0：症状なし、所見なし
  • Grade I：頸部痛・凝りのみ、他覚所見なし
  • Grade II：頸部症状 + 筋骨格所見（圧痛・可動域制限）
  • Grade III：頸部症状 + 神経学的所見（反射低下・筋力低下・感覚障害）
  • Grade IV：頸部症状 + 骨折・脱臼

I–II の大半は数週で軽快。II 以下は早期の頸部運動・通常活動再開が予後良好。

椎骨動脈損傷（BCVI）

Modified Denver / Memphis screening

以下のいずれかで 頸部 CTA を考慮。

  • 後頭部・後頸部の打撲、高エネルギー、回旋・過伸展機転
  • 頸椎骨折、特に 横突孔骨折・C1–3 損傷・facet 脱臼
  • Le Fort II/III 顔面骨折、頭蓋底骨折（特に岩錐部・斜台）
  • GCS < 8 の頭部外傷で原因不明
  • 頸部血管雑音、Horner 症候群、頸部血腫
  • 所見と画像が説明できない局所神経所見・後方循環症状

遅発性脳梗塞のリスクが高く、診断がついたら抗血栓療法（抗血小板 or 抗凝固）を脳外科 / 血管外科と相談。

その他

  • 頸椎症性脊髄症（CSM）— 緩徐な手指巧緻運動障害・歩行障害。Hoffmann 反射陽性、Babinski、痙性歩行
  • 頸椎症性神経根症 — デルマトーム性の上肢痛・しびれ。Spurling テスト
  • SCIWORA（小児）— 画像陰性で脊髄症状あり。MRI で脊髄信号変化を評価
  • 喉頭・気管損傷 — 嗄声、皮下気腫、ハンドルマーク。CT + 耳鼻科
  • 食道損傷 — 縦隔気腫、嚥下時痛、吐血。造影 / 内視鏡

骨傷の全体像は 頸椎損傷 ページへ。

🩺 身体所見・画像

診察の最小セット（固定継続のまま）

  • 受傷機転（高エネ / 軽微 / 失神先行 / 機転不明）
  • 後頸部正中圧痛を一指で触知。広い圧痛＝筋筋膜性、棘突起の限局圧痛＝骨傷示唆
  • 視診・触診：変形、皮下気腫、頸部血腫、ハンドルマーク、絞扼痕
  • 四肢の運動・感覚・腱反射・Babinski、肛門周囲感覚と肛門括約筋緊張
  • 呼吸パターン：腹式呼吸消失・浅速呼吸・SpO2 低下は高位頸髄を示唆
  • 脈・血圧：徐脈 + 低血圧 + 末梢温暖 = 神経原性ショック
  • 頸部前面・側面・後頭部（喉頭・気管・大血管・甲状腺）
  • 巧緻運動：握力左右差、つまみ動作、Hoffmann 反射（CSM スクリーニング）
  • 項部硬直・Kernig（非外傷の致死的鑑別）

NEXUS 基準

5 項目すべて陰性で画像不要・固定解除可
  • 後頸部正中圧痛なし
  • 中毒なし（アルコール・薬物・鎮静）
  • 意識清明（GCS 15、見当識保持）
  • 局所神経所見なし
  • 気が逸れる損傷なし（distracting injury：長管骨骨折、腹部痛など）

感度は高いが特異度は低い。陰性で画像省略・陽性なら CT へ。

Canadian C-Spine Rule（CCSR）

3 ステップで判定（GCS 15 の安定外傷成人）

Step 1：高リスク要素（1 つでもあれば画像）

  • 65 歳以上
  • 危険な受傷機転（高所転落 ≥ 1 m / 5 段、車外放出、転覆、自転車との衝突、高速衝突）
  • 四肢のしびれ・刺すような痛み

Step 2：低リスク要素（少なくとも 1 つあれば Step 3 へ。なければ画像）

  • 単純追突
  • ER で座位を取れた
  • 受傷後いつでも歩行可能
  • 頸部痛が遅発性（受傷直後ではない）
  • 後頸部正中圧痛なし

Step 3：能動回旋

  • 左右 45° 自由に回せれば画像不要・固定解除可
適用が外れる場面

意識障害・中毒・神経所見あり・非外傷・小児・穿通性外傷・既存の脊椎疾患では NEXUS / CCSR の適用外。固定継続のうえ画像で評価する。

画像の置き方

モダリティ得意選ぶとき
頸椎 Xp（3 方向）明確な脱臼・大きな骨折感度が低く除外には弱い。CT が使えない環境の暫定。日常的には推奨されない
頸椎 CT（C0–T1）骨傷・脱臼・椎間板損傷の間接所見第一選択。NEXUS / CCSR で陽性、高エネルギー、意識障害例。頭部 CT 撮るときは頸椎下端まで含める
頸椎 MRI脊髄・椎間板・靱帯・硬膜外血腫・SCIWORA神経所見あり、CT 陰性でも症状が説明できないとき、中心性頸髄損傷疑い、SCIWORA 疑い、CSM 評価
頸部 CTA椎骨動脈・頸動脈損傷BCVI スクリーニング陽性、頸椎横突孔骨折、後頭部痛 + 神経所見
頸部 CT 軟部喉頭・気管・食道・咽後間隙皮下気腫、嗄声、嚥下時痛、絞扼、咽後膿瘍疑い
CT 正常 = 頸髄損傷除外ではない

SCIWORA、中心性頸髄損傷、靱帯損傷、硬膜外血腫は CT では拾えない。神経所見があれば MRI。

💊 マネジメント

頸椎固定の解除条件

以下をすべて満たすときに解除を検討
  • NEXUS 5 項目すべて陰性、または CCSR で「画像不要」と判定
  • 能動的な左右 45° 回旋が可能（CCSR Step 3）
  • 新規神経所見なし
  • 適用除外条件（意識障害・中毒・神経所見）に該当しない

固定を続ける目安

  • 軟性カラー：軽症・むち打ち Grade I–II の鎮痛目的に短期。長期装着は予後を悪化させ得る
  • 硬性カラー：骨傷・脱臼確定、または評価困難で固定継続が必要なとき。整形に引き継ぎ
  • むち打ちでは早期の通常活動再開が遷延予防につながる

不安定骨折・脱臼の初期対応

  • 用手 in-line stabilization を維持、ログロール、硬性カラー + 可能なら背板
  • 整形外科 / 脊椎外科にコンサルト。手術適応・牽引の判断は専門科
  • 気道介入：ビデオ喉頭鏡、頸部の動きを最小化
  • ASIA A の完全麻痺でも初期は脊髄ショックの可能性 → 評価を経時的に再確認

神経原性ショックへの対応

徐脈 + 低血圧 + 末梢温暖
  • 輸液で前負荷を上げる（過剰輸液に注意。脊髄浮腫を悪化させ得る）
  • 輸液不応なら昇圧薬（ノルアドレナリン or フェニレフリン）
  • 徐脈強い場合はアトロピン。重症ではペーシング
  • 出血性ショックを並行して必ず除外（FAST、骨盤・大腿骨）
  • MAP 目標は 85–90 mmHg を脊椎外科と相談（脊髄灌流維持）

ステロイド

急性頸髄損傷のメチルプレドニゾロン大量投与はルーチン推奨されない

NASCIS II/III 以降、利益が不明確で合併症（感染、消化管出血、血糖上昇）のリスクが上回るとされ、現在はルーチン非推奨。投与は脊椎外科と個別に相談。

鎮痛・対症療法

  • NSAIDs / アセトアミノフェン（出血傾向・抗凝固例は注意）
  • 筋弛緩薬は短期に限定
  • むち打ちでは早期の頸部可動・温熱・通常活動を推奨
  • 遷延例にはペインクリニック・リハ介入

📤 disposition

帰宅可

  • NEXUS / CCSR で画像不要、神経所見なし、機能保たれる
  • むち打ち Grade I–II で疼痛コントロール可、独居でない、フォロー可
  • 頸椎症性神経根症で安定、外来フォロー可能

観察入院 / 短期 ICU 不要

  • 骨傷なしだが疼痛強く ADL 困難
  • 抗凝固薬例で受傷から経過短く血腫・遅発性出血を経過観察
  • むち打ち Grade III（軽度神経所見あり）で経過観察方針

入院 / コンサルト

  • 頸椎骨折・脱臼 → 脊椎外科に即コンサルト・入院
  • 頸髄損傷の疑い（神経所見あり）→ MRI 評価後、脊椎外科 / 神経内科
  • 椎骨動脈・頸動脈損傷 → CTA、脳外 / 血管外科、抗血栓療法を相談
  • 意識障害があり頸椎評価困難 → 固定継続のうえ入院
  • 非外傷の致死的鑑別確定 → 各専門科

ICU/HCU

  • 高位頸髄損傷（C5 以上）／神経原性ショック
  • 気道閉塞リスク（喉頭損傷、頸部腫脹進行、絞扼後）
  • 不安定骨折で術前管理が必要
  • 多発外傷の合併

むち打ち帰宅時の説明

伝えること
  • 大半は数日〜数週で自然軽快。早期の通常活動再開が回復を早める
  • 長期のカラー固定はかえって遷延を招くことがある
  • 鎮痛薬の使い方、温熱・ストレッチの目安
  • 再受診のタイミング：手のしびれ・脱力、巧緻運動障害、排尿障害、激しい後頭部痛・めまい、症状が増悪・新規出現
  • 2 週以上遷延する場合は整形外科外来へ

⚠️ 見落とし・ミス

  • 高齢者の中心性頸髄損傷を見逃す — 軽微な顔面打撲・転倒でも過伸展で起こる。CT 骨傷なくても上肢優位の脱力・巧緻運動障害があれば MRI へ
  • 転倒で頭部だけ見て頸椎を触らない — 頭部 CT を撮るときは頸椎下端（C7–T1）まで含める。頭部単独の評価で済ませない
  • 酩酊・意識障害下で NEXUS / CCSR を当てて「異常なし」とする — 適用外。固定継続のうえ画像
  • 椎骨動脈損傷の遅発性発症を考えない — 後頭部痛 + 後方循環症状 + 高エネ・回旋・横突孔骨折で CTA
  • 歯突起骨折を Open mouth 像のみで除外 — 高齢者では撮像困難・読影困難。CT を低い閾値で
  • 外傷性頸部症候群を「ただのむち打ち」で軽視 — Grade III 以上は神経所見あり。Grade を分けて評価し、再受診基準を説明
  • 長期の硬性カラー固定でむち打ちを悪化させる — Grade I–II は早期可動・通常活動が予後良好
  • 頸部血腫・気道閉塞の進行を見逃す — 抗凝固例・絞扼・喉頭損傷例。受診時 OK でも数時間で進行する
  • 非外傷の致死的頸部痛を「凝り・寝違え」で帰す — 髄膜炎・SAH・椎骨動脈解離・咽後膿瘍・硬膜外膿瘍。発熱・突発・神経所見・嚥下時痛が手がかり
  • 抗凝固薬例の遅発性硬膜外血腫を見逃す — 軽微外傷でも経過観察 + 再受診指示
  • SCIWORA を CT 陰性で除外 — 小児・若年で神経症状あれば MRI
  • 気道介入で頸部を動かす — in-line stabilization が原則。事前計画を立てる
最低限やる「セーフティネット」
  • 頸椎クリアランスの判定根拠（NEXUS / CCSR）をカルテに残す
  • 適用外なら理由を書く
  • 帰宅時は具体的な再受診基準を口頭・書面で渡す
  • 抗凝固例・高齢者・むち打ち Grade III は翌日 follow-upを設定
