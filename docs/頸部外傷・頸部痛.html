頸部外傷・頸部痛

⚡ 即時アクション
  1. 頸椎固定を継続（カラー / 用手 inline 固定）— NEXUS / Canadian C-spine で除外可なら解除
  2. 気道・呼吸を最優先 — C5 以上で麻痺リスク、横隔膜麻痺＝呼吸停止を念頭
  3. CT 適応：Canadian C-spine（高リスク要素 / 危険な受傷機転）+ 高エネルギー外傷で閾値を下げる
  4. 椎骨動脈損傷を念頭 — 神経所見・大血管損傷リスク・回旋外傷で CTA を検討
🚨Red flags
致死・後遺症 🛡頸椎固定
NEXUS / Canadian 🩺診察と評価 🔬画像の置き方

🚨 Red flags（致死・後遺症を外す）

これがあれば頸椎固定を緩めない

  • 四肢麻痺・感覚低下（不全麻痺含む）
  • 頸部正中圧痛
  • 意識障害を伴う外傷（評価困難 → 固定継続）
  • 高エネルギー外傷（高所転落・歩行者と車・車外放出）
  • 高齢者の軽微外傷でも中心性頸髄損傷を起こしうる
  • 抗凝固薬内服
  • 頸部・後頭部痛 + 神経所見 → 椎骨動脈損傷を疑う
危険所見あり

頸椎保護を継続し、整形外科 / 脊椎外科へ早期コンサルト。気道管理は in-line stabilization で。

  • 頸髄損傷（中心性頸髄損傷を含む）
  • 頸椎損傷（骨傷）
  • 椎骨動脈損傷（外傷性解離）

🛡 頸椎固定の判断（NEXUS / Canadian C-spine）

「外して良いか」を意識的に判断する。固定は除外できれば解除。漫然と続けない。

NEXUS 基準（5 つすべて陰性で固定解除可）
  • 後頸部正中圧痛なし
  • 中毒なし（アルコール・薬物）
  • 意識清明（GCS 15、見当識保持）
  • 局所神経所見なし
  • 気が逸れる損傷なし（distracting injury）

すべて陰性なら画像不要として固定解除を検討。

Canadian C-spine Rule

意識清明で安定した外傷成人に適用。

  • 高リスク要素（1 つでもあれば画像）：65 歳以上、危険な受傷機転（高所転落・車外放出・高速衝突）、四肢のしびれ
  • 低リスク要素（1 つもなければ可動域評価へ）：単純追突、座位で評価可、歩行可、遅発性発症、正中圧痛なし
  • 能動的に左右 45° 回旋可 → 画像不要で固定解除可
適用が外れる場面に注意

意識障害・中毒・高エネルギー外傷・神経所見ありのときは NEXUS / Canadian の適用外。固定継続のうえ画像で評価する。

🩺 診察と評価

その場で行う最小セット

  • 受傷機転の確認（高エネルギー / 軽微 / 失神先行）
  • 頸椎固定継続のまま視診・触診（後頸部正中圧痛、変形、皮下気腫）
  • 神経学的評価：四肢の運動・感覚・腱反射、肛門括約筋反射、Babinski
  • 気道・呼吸：呼吸パターン、横隔膜運動、SpO2
  • 四肢の左右差（中心性頸髄損傷では上肢 > 下肢の不全麻痺）
  • 頸部前面・側面・後頭部の損傷（喉頭・気管・大血管・甲状腺）
中心性頸髄損傷の早期サイン

高齢者の軽微外傷（特に過伸展）で、上肢優位の不全麻痺・手指の巧緻運動低下・尿閉を見たら疑う。CT では骨傷なく見える → MRI へ。

気道管理時の注意

頸椎損傷疑いでの挿管は 用手 in-line stabilization で行う。輪状軟骨圧迫・過度の頸部進展は避ける。ビデオ喉頭鏡が有用。

🔬 画像の置き方

モダリティ得意選ぶとき
頸椎 Xp（3 方向）明確な脱臼・骨折軽微外傷で CT 不要なとき。感度低いので除外には弱い
頸椎 CT骨傷・脱臼・椎間板損傷の間接所見第一選択。NEXUS / Canadian で陽性、高エネルギー外傷、意識障害例
頸椎 MRI脊髄・椎間板・靱帯・硬膜外血腫神経所見あり、CT 陰性でも症状が説明できないとき、中心性頸髄損傷疑い
頸部 CTA椎骨動脈・頸動脈損傷神経所見 + 大血管損傷リスク（高エネルギー、回旋外傷、頸椎横突孔骨折、Le Fort II/III、Horner 症候群）
CT 正常 = 頸髄損傷除外ではない

SCIWORA / 中心性頸髄損傷・靱帯損傷・硬膜外血腫は CT では拾えない。神経所見があれば MRI。

🧠 主要鑑別

病態手がかり進む先
頸椎損傷（骨傷）正中圧痛、神経症状、高エネルギー頸椎損傷
中心性頸髄損傷高齢・過伸展、上肢優位の不全麻痺、CT 骨傷なし頸髄損傷
椎骨動脈損傷後頭部痛、めまい、構音障害、視野欠損、高エネルギー / 回旋外傷椎骨動脈損傷
外傷性頸部症候群（むち打ち様）頸部痛中心、神経症状乏しい、遅発症状あり転倒 で再診基準を確認
喉頭・気管損傷発声異常、皮下気腫、嗄声、呼吸困難耳鼻科 / 外科
頸部血腫・大血管損傷進行性腫脹、ショック、雑音血管外科 / 救命

📤 disposition

帰宅を考えやすい

  • NEXUS / Canadian で固定解除可、神経所見なし、画像陰性
  • 外傷性頸部症候群で機能保たれる
  • 独居でなく、フォロー可能

入院 / コンサルト

  • 頸椎骨折・脱臼 → 整形 / 脊椎外科に即コンサルト
  • 頸髄損傷の疑い（神経所見あり）→ 入院、MRI 評価、脊椎外科
  • 椎骨動脈損傷の疑い → CTA、脳外 / 血管外科コンサルト、抗血栓療法を相談
  • 気道閉塞リスク（喉頭損傷、頸部腫脹進行）→ ICU、早期気道確保
  • 意識障害があり頸椎評価困難 → 固定継続のうえ入院

⚠️ ピットフォール

  • NEXUS / Canadian の適用外で当てはめる（意識障害・中毒・高エネルギー）
  • CT 正常で安心する — 中心性頸髄損傷・靱帯損傷・SCIWORA は写らない
  • 高齢者の軽微外傷を軽視 — 過伸展で中心性頸髄損傷が起こる
  • 椎骨動脈損傷を考えない — 後頭部痛・めまい + 高エネルギーで CTA を
  • 外傷性頸部症候群で帰すときに遅発症状の説明をしない
  • 気道管理で頸部を動かす — in-line stabilization が原則
  • 抗凝固薬例の頸部血腫を見逃す（進行性腫脹に注意）
