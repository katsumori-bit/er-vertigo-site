構音障害

⚡ 即時アクション
  1. 脳梗塞 → tPA 適応窓を確認（発症 4.5 h 以内？最終健常時刻は？）
  2. NIHSS 評価 + 頭部 CT/MRI（CTA・CTP で LVO とペナンブラ）
  3. 構音 vs 言語 vs 嚥下を区別（dysarthria / aphasia / dysphagia）
  4. バイタル + 血糖（ストロークミミック除外、低血糖は補正で改善）

⏱ 最初に決めること

  • いつから症状が出たか / 最後に普通に喋れていたのはいつか
  • 脳卒中っぽいか（構音障害 + 他の神経症状があるか）
  • 低血糖ではないか（全例で血糖測定）
  • 意識は清明か
Time is Brain

構音障害の約 70% は脳血管障害（脳梗塞 49%、TIA 19%、脳出血 1%）。 「ろれつが回らない」は脳卒中として動く — mimic だったとしても、それは後で分かること。

  • 発症 4.5 時間以内 → tPA の適応がありうる
  • 大血管閉塞 → 血栓回収療法の適応がありうる
  • 専門科を呼びつつ、自分がやるべき初期対応を最短で回す

🩺 まずやること（並行して進める）

1. 発症時刻の確定

  • 「いつから？」「最後に普通だったのはいつ？」
  • 本人が答えられないなら 家族・目撃者・救急隊に確認
  • 起床時発症（寝る前は普通 → 朝起きたらおかしい）→ 最終健常確認は就寝前
発症時刻が不明でも止まらない

「いつからかわからない」は後で専門科が画像で判断する。 初期対応を遅らせる理由にはならない。

2. 血糖測定

  • 全例で血糖を測る
  • 低血糖で構音障害・片麻痺が出ることがある → 補正すれば改善 → 脳卒中ではない
  • 低血糖の除外は stroke mimic で最も重要

3. バイタル・モニター・ルート確保

  • SpO2・血圧・脈拍・体温
  • 心電図モニター装着（AF の有無 → 心原性塞栓の判断に重要）
  • 末梢静脈路確保（造影 CT・tPA 投与に備える。できれば 18–20G）
  • 輸液は生理食塩水をつなぐ（脳卒中疑いではラクテックではなく生食。ブドウ糖含有輸液は脳浮腫を悪化させうるため避ける）

4. NIHSS を取る

専門科が来る前に 自分で NIHSS を取っておく。約 6–7 分で取れる。

NIHSS のスコアシートは救急外来に常備しておく

完璧でなくてよい。専門科が来た時に「NIHSS ○点です」と言えることが重要。 構音障害だけなら NIHSS 1–2 点（軽症）のことが多いが、他の所見を見落としていないかの確認にもなる。

5. 採血 + 迅速クレアチニン

採血（造影 CT と tPA 判断に必要な項目）：

  • 血算・凝固（PT-INR、APTT）・生化（Cr、BUN、電解質）・血糖
  • 抗凝固薬内服中なら凝固は特に重要

迅速クレアチニン：

  • 放射線科に連絡して迅速 Cr 検査を依頼
  • 造影 CT（CTP）の前に腎機能を確認するため
  • eGFR < 30 → 造影は原則回避（造影剤腎症リスク高い）
  • eGFR 30–45 → 造影可能だが補液等の予防策を講じる
  • eGFR ≥ 45 → 通常通り施行可
急性期脳梗塞では造影の利益がリスクを上回ることが多い

腎機能が多少悪くても、大血管閉塞を見逃す方がはるかに危険。 eGFR が低い場合は専門科と相談して判断する。 日本放射線学会の造影剤ガイドライン 2018 では、eGFR < 30 でもリスク・ベネフィットを勘案して施行する場合があるとしている。

6. 12誘導心電図

  • AF（心原性脳塞栓の原因として最多）
  • 急性心筋梗塞の合併
  • 採血やルート確保と並行で施行

7. 脳外科 / 脳神経内科に連絡

NIHSS を取り、迅速 Cr を出した段階で連絡する。

伝えること：

  • 年齢・性別・主訴（構音障害 ± 他の神経症状）
  • 発症時刻 / 最終健常確認時刻
  • NIHSS スコア
  • 血糖値
  • バイタル（特に血圧）
  • 既往歴（AF、脳卒中歴、抗凝固薬・抗血小板薬）
  • 迅速 Cr の結果（出ていれば）
  • ルート確保済み、造影 CT に行ける状態か

🧠 頭部造影 CT（CTP）

迅速 Cr の結果が出たら 頭部造影 CT（CT perfusion） に行く。

CT で見ること

モダリティ見るもの
単純 CT出血の有無（出血あり → tPA 禁忌 → 脳外科対応）
CTA（CT angiography）大血管閉塞（LVO）の有無（ICA、MCA M1/M2、BA）
CTP（CT perfusion）虚血コア vs ペナンブラ（救済可能な脳組織の評価）
  • 出血あり → tPA 禁忌 → 脳外科
  • LVO あり + ペナンブラが大きい → 血栓回収療法の適応
  • LVO なし + 小さい梗塞 → 抗血小板薬 or tPA の判断（専門科）
CT は専門科が来る前に撮っても、来てから撮っても構わない

施設のワークフローに従う。大事なのは CT に行ける準備（ルート・Cr 確認）を済ませておくこと。 専門科がこのタイミングで合流すれば、一緒に CT の画像を見ることになる。

📞 専門科に委ねること

上記の初期対応（NIHSS・採血・迅速Cr・ルート・モニター・心電図・造影CT）は全部自分でやる。 検査を回すことにリスクはない。

専門科の判断が必要なのは治療の部分：

  • tPA（アルテプラーゼ / テネクテプラーゼ）の投与判断 → リスクがある治療なので専門科
  • 血栓回収療法（EVT）の適応判断
  • 降圧の要否と目標値の指示

血圧について

自己判断で血圧を下げない

脳梗塞では虚血部位への灌流を保つために血圧が上がっている。 不用意に降圧すると梗塞が拡大する可能性がある。 降圧するかどうか・目標値は専門科の指示に従う。

🔍 構音障害だが脳卒中ではないかもしれないとき

構音障害の約 30% は脳卒中以外の原因。

原因手がかり

構音障害だけで NIHSS が低くても、TIA（一過性脳虚血発作）の可能性がある。 TIA は「症状が治った = 大丈夫」ではなく、脳梗塞の前触れ。 症状が改善しても精査は必要。
