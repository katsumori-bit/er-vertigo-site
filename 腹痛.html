腹痛

元: 臨床/ER/腹痛.md

#ER #救急 #腹痛 #急性腹症 #穿孔 #腸閉塞 #虚血 #胆道系

このページの使い方
  1. まず 不安定・腹膜刺激・見逃せない腹痛 を拾う
  2. カルテで先に勝てる情報 を取る
  3. 部位 × パターン で枝を決め、画像を切る
  4. 鎮痛・制吐・輸液・抗菌薬 を並行で回す
  5. 詳細はリンク先へ飛ぶ

最初に決めること

  • 不安定（ショック・敗血症・腹膜炎） ではないか
  • 妊娠関連疾患 を除外すべき人ではないか
  • 血管イベント（AAA・腸間膜虚血）を疑うべきか
  • 閉塞・絞扼・穿孔 が切迫していないか
  • 部位 で絞れる腹痛か、びまん性 か
先に拾う
  • ショック（低血圧・頻脈・冷汗・意識変容）
  • 腹膜刺激徴候
  • 持続する激痛
  • 痛みの割に腹部所見が乏しい
  • 吐血 / 下血 / 黒色便
  • 妊娠可能年齢
  • 高齢者
  • 免疫不全 / 担癌
  • 抗凝固薬
  • 心房細動 / 血管リスク
  • 開腹歴
  • 排ガス排便停止 + 腹部膨満
  • 背部痛

急性腹痛は ED 受診の約 10% を占め、150 以上の病因が挙げられるが、上位 20 疾患で 70% をカバーする（AAFP 2023）。「質問の順序」よりも 不安定性の評価 → カルテ情報 → 部位×パターン の流れで鑑別を絞る構造が実務に直結する。

カルテで先に拾う

  • 開腹歴 → 癒着性腸閉塞を上げる
  • 担癌 → 悪性腸閉塞、穿孔、感染、治療関連有害事象
  • オピオイド開始 / 増量 → 便秘、麻痺性イレウス
  • 抗凝固薬 → 出血、腹壁血腫
  • 心房細動 / 血管リスク → 腸間膜虚血
  • 胆石歴 → 胆道系
  • 尿管結石歴 → 尿路結石
  • 婦人科既往 → 異所性妊娠、卵巣捻転、PID
  • 下剤歴 → 便秘と決めつけず閉塞を考える
患者到着前にまず5つ

開腹歴・担癌・オピオイド・抗凝固薬・心房細動。 これだけで CT の閾値と緊急度がかなり変わる。

このページで最優先に外す急性腹症

  • 絞扼性腸閉塞
  • 腸間膜虚血
  • 消化管穿孔
  • AAA（腹部大動脈瘤関連）
  • 重症胆管炎 / 重症胆嚢炎
  • 重症膵炎
  • 異所性妊娠
  • 卵巣捻転
  • 嵌頓 / 絞扼ヘルニア
迷ったらこの4つを先に外す

AAA・腸間膜虚血・消化管穿孔・異所性妊娠

まず打つ手

1. 不安定 / 腹膜刺激あり

  • ルート確保 → 採血（乳酸含む）→ 血液ガス
  • 輸液・鎮痛・制吐
  • 必要なら早期抗菌薬
  • 外科コール
  • 造影 CT を急ぐ（AAA が強ければ POCUS 先行も可）

非局在性急性腹痛では腹部骨盤造影 CT が通常適切（ACR Appropriateness Criteria）。早期鎮痛は診断精度を損なわない（急性腹症診療ガイドライン 2015）。

2. 局在はあるが外科腹症を否定できない

  • 基本は造影 CT を低い閾値で
  • ただし例外あり
    • 右上腹部痛 → まず US
    • 妊娠可能年齢の骨盤痛 → まず US
    • 典型的尿管結石 → 単純 CT

3. 痛みの割に所見が乏しい

  • 腸間膜虚血 を疑う
  • 乳酸が正常でも否定しない
  • CTA を考える

急性腸間膜虚血は見逃すと致死的。画像の遅れが不利益に直結する。

まず取るべき最小情報

  • 発症様式：突然 / 徐々に
  • 経過：持続 / 間欠
  • 部位：心窩部 / 右上腹部 / 右下腹部 / 左下腹部 / 下腹部 / びまん性
  • 性質：疝痛 / 持続痛 / 移動痛
  • 随伴：嘔吐、下痢、便秘、発熱、背部痛、胸痛、失神、血便、排尿症状
  • 女性：最終月経、妊娠可能性、不正出血
  • 背景：高齢、手術歴、血管リスク、免疫不全、抗凝固薬、胆石歴、尿路結石歴

部位別の考え方

右上腹部痛

  • 胆嚢炎 / 胆石疝痛 / 胆管炎 / 肝胆道系疾患
  • 肺炎 / 胸膜炎も忘れない

次の一手

  • Murphy 徴候、発熱、黄疸を確認
  • 採血：CBC、CRP、肝胆道系、ビリルビン
  • 初手画像は超音波（AAFP 2023：RUQ pain の first-line は US）
  • 黄疸や肝胆道系酵素上昇があれば胆管炎を疑い CT へ → 石が見つかれば 胆管結石
胆管炎を疑うとき — Charcot / Reynolds / TG18
  • Charcot の三徴：発熱・黄疸・右上腹部痛（すべて揃わなくても胆管炎はありうる）
  • Reynolds の五徴：上記 ＋ ショック ＋ 意識障害 → 重症胆管炎として即動く

TG18 診断基準（急性胆管炎）

カテゴリ基準
A. 全身性炎症発熱 >38℃ / 悪寒・戦慄、または WBC 異常・CRP ≥1
B. 胆汁うっ滞T-Bil ≥2、または肝胆道系酵素 >1.5×ULN
C. 画像胆管拡張、または原因の描出（結石・狭窄・ステント）

疑診：A 1項目 ＋ B or C 1項目　／　確診：A ＋ B ＋ C 各1項目

→ CT で石を確認したら 胆管結石 へ。Reynolds 五徴が揃えば CT を待たず消化器内科コール＋抗菌薬開始。

心窩部痛

  • 消化性潰瘍 / 穿孔 / 膵炎 / 胆道疾患
  • ACS / 大動脈病変

次の一手

  • 胸部症状がなくても ECG・トロポニン で ACS を外す
  • 背部放散 + 持続痛 + 嘔吐 → 膵炎（リパーゼ）
  • 腹膜刺激 / 急激悪化 → 穿孔
  • 採血：CBC、CRP、肝胆道系、リパーゼ、トロポニン
心窩部痛で見落としたくない

ACS・大動脈病変・穿孔・重症膵炎

右下腹部痛

  • 虫垂炎 / 尿管結石 / 腸炎 / Crohn
  • 女性：卵巣捻転、異所性妊娠、PID

次の一手

  • 食欲低下、発熱、移動痛、反跳痛
  • 尿検査
  • 妊娠反応（妊娠可能年齢なら必須）
  • 画像：成人は CT が有力。女性・若年では US 先行も考える（WSES 2025：虫垂炎は CT・US・臨床スコア併用で精度向上）

左下腹部痛

  • 憩室炎 / 便秘 / 腸炎 / 尿管結石
  • 女性：婦人科疾患

次の一手

  • 発熱、便通変化、局所圧痛
  • 採血：CBC、CRP
  • 画像：診断・重症度判定には CT が有力
  • 反復歴があっても、重症感や腹膜刺激があれば軽く見ない

下腹部痛 / 骨盤痛

  • 膀胱炎 / 腎盂腎炎 / 尿閉 / 婦人科疾患 / PID
  • 卵巣捻転 / 異所性妊娠
  • 精巣捻転による関連痛 / 嵌頓ヘルニア

次の一手

  • 妊娠反応（飛ばさない）
  • 排尿症状、性器出血、月経歴
  • 精巣 / 鼠径部も必要なら診る
  • 画像：婦人科疑い → 骨盤超音波
妊娠可能年齢の下腹部痛

妊娠反応を飛ばさない。hCG 値だけで異所性妊娠は除外できない（ACOG / ACEP）。

びまん性腹痛

  • 胃腸炎 / 腸閉塞 / 便秘 / 腸間膜虚血 / DKA / 敗血症 / 腹膜炎
  • 非腹部疾患（ACS、DKA、副腎不全など）

次の一手

  • 重症感の評価を先にする
  • 膨満、腸雑音、腹膜刺激、脱水
  • 採血：CBC、CMP、CRP、乳酸、血糖、必要に応じて血ガス
  • 画像：原因が広いなら 造影 CT を考える（ACR：非局在性腹痛で造影 CT が通常適切）
  • 「所見に乏しい激痛」は虚血を疑う

検査の考え方

血液

  • CBC、電解質 / 腎機能、肝胆道系、CRP、リパーゼ
  • 乳酸（重症感・虚血疑い）
  • トロポニン（心窩部痛・高齢・冠危険因子あり）

尿

  • 尿定性
  • 妊娠反応（妊娠可能年齢で原則全例）

心電図

  • 心窩部痛、高齢、冠危険因子あり

画像選択の基本

場面初手
非局在性 / びまん性 / 外科腹症が気になる造影 CT
右上腹部痛超音波
妊娠可能年齢の骨盤痛骨盤超音波
尿管結石が強く疑わしい単純 CT
高齢 + 低血圧 + 腹痛 / 背部痛AAA を POCUS or CT で除外
痛みの割に所見が乏しい / 虚血疑いCTA

ACR Appropriateness Criteria は非局在性急性腹痛で造影 CTを「通常適切」、腹部単純 X 線を「管理方針を変えないことが多い」としている。画像は一律ではなく臨床像との対応で選択する。

画像を読むときの順番

急性腹痛の CT は手術・IVR・緊急コンサルトに直結する所見を先に拾い、炎症性疾患はその後に見る。臨床的に血管を疑う場面（高齢・低血圧・背部痛・Af・所見に乏しい激痛）では血管を最初に見る。

  1. 穿孔：free air、腹水、限局炎症
  2. 血管：AAA、解離、SMA / SMV 閉塞・血栓
  3. 閉塞：拡張腸管、transition point、closed loop
  4. 絞扼 / 虚血：腸管壁造影不良、腸間膜浮腫、腹水、pneumatosis、portal venous gas、whirl sign
  5. 炎症：虫垂、憩室、胆嚢、膵周囲
  6. 骨盤：卵巣腫大、骨盤液体貯留

SBO では「閉塞の有無」だけでなく絞扼・虚血のサインを拾う（腸管壁造影不良は虚血予測で特異度 95%：Defined SR）。WSES は閉塞 + 腹膜刺激 / 絞扼徴候があれば非手術的対応に固執しないことを推奨。

薬剤の使い方

鎮痛

  • アセトアミノフェン IV（アセリオ）：1回 1000 mg（15分かけて点滴静注）。体重 50 kg 未満は 15 mg/kg。投与間隔 4–6時間以上、1日上限 4000 mg。急性腹症診療ガイドライン 2015 で診断前の早期投与が推奨（推奨度 A）
  • NSAIDs：尿管結石・胆石疝痛で有用（例：フルルビプロフェン 50 mg IV、ジクロフェナク坐剤 25–50 mg）。腎機能低下・脱水・潰瘍・出血リスクでは慎重
  • ブチルスコポラミン（ブスコパン）：疝痛（胆石・尿管結石）に 20 mg 筋注 or 静注。持続痛には効果が乏しい
痛みを後回しにしない

早期鎮痛は診断精度を損なわない（急性腹症診療ガイドライン 2015）。 まずアセトアミノフェン IV → 疝痛なら NSAIDs or ブスコパン → 効果不十分なら外科コンサルトを早める。

制吐

  • メトクロプラミド（プリンペラン）：10 mg IV or 筋注。腎機能低下時は減量（CCr <40 で半量）
  • オンダンセトロン：4 mg IV（使える環境なら有力）
  • イレウス疑いでは制吐はしても、原因評価と減圧の遅れを作らない

輸液

  • 嘔吐・脱水・敗血症・膵炎・閉塞では早めに補正
  • 膵炎：moderately aggressive が ACG 2024 推奨（乳酸リンゲル推奨、過剰は避ける）。心・腎合併症があれば慎重に

抗菌薬

  • すべての腹痛に入れるわけではない
  • 入れる場面：胆管炎、胆嚢炎、穿孔疑い、腹腔内感染、壊死性腸管疑い、complicated appendicitis 疑い、敗血症
  • 市中腹腔内感染の初手（JAID/JSC 2023 準拠）
    • SBT/ABPC 3 g IV × 3–4回/日
    • or CMZ 1 g IV × 3回/日
    • or CTRX 2 g IV × 1回/日 + MNZ 500 mg IV × 3回/日
  • 重症 / 院内発症：TAZ/PIPC 4.5 g IV × 3回/日、またはカルバペネム
抗菌薬の原則

感染を強く疑ったら開始。ただし source control が本体。 ドレナージや手術が遅れないようにする。 重症胆管炎では TG18 が重症度に応じた抗菌薬と早期ドレナージを重視。

症候群で考える鑑別

突然の激痛

AAA / 腸間膜虚血 / 穿孔 / 卵巣捻転 / 尿管結石 / 胆石疝痛 / 異所性妊娠破裂

嘔吐 + 膨満 + 排ガス排便停止

単純性腸閉塞 / 絞扼性腸閉塞 / 嵌頓ヘルニア / 結腸閉塞 / 麻痺性イレウス

発熱 + 腹痛

胆嚢炎 / 胆管炎 / 虫垂炎 / 憩室炎 / 腎盂腎炎 / PID / 腹腔内膿瘍

痛みの割に所見が弱い

腸間膜虚血 / 初期閉塞 / 早期穿孔前 / 代謝性疾患 / 帯状疱疹前駆期

便秘・オピオイド誘発性便秘

  • ただの便秘と決めつけない
  • 便秘で済ませにくい：排ガスも停止、嘔吐が強い、膨満が強い、持続痛、発熱、腹膜刺激、担癌、開腹歴、炎症反応高い
  • オピオイド誘発性便秘：オピオイド開始後、排便回数低下が明確、腹膜刺激なし、画像で閉塞・虚血が乏しい → 下剤の予防・早期介入が基本。ただし閉塞が疑わしいときは下剤で押さない

disposition

帰しにくい

  • 不安定
  • 腹膜刺激あり
  • 原因不明なのに痛みが強い
  • 高齢者
  • 免疫不全 / 担癌
  • コントロール不能な嘔吐 / 疼痛
  • 腸管虚血、穿孔、胆管炎、AAA、婦人科救急が否定できない
  • 画像で虚血 / 閉塞 / 穿孔 / 膿瘍

帰宅を考えやすい

  • バイタル安定
  • 危険サインなし
  • 経口摂取可能
  • 痛みが改善
  • 重大疾患の除外ができている
  • 再診基準を明確に伝えられる
帰宅時に必ず伝える
  • 痛み増悪 / 発熱 / 嘔吐持続
  • 血便 / 黒色便
  • 失神 / 新たな腹膜刺激感
  • 再受診の目安
折りたたみ用メモ

外してはいけないもの

  • AAA
  • 腸間膜虚血
  • 消化管穿孔
  • 絞扼性腸閉塞
  • 異所性妊娠
  • 卵巣捻転
  • 重症胆管炎（TG18 Grade III）

AAA は腹部拍動性腫瘤がなくても否定できない。腸間膜虚血は初期に腹部所見が乏しく乳酸上昇は遅れうる。異所性妊娠は hCG 値だけで除外しない（ACOG / ACEP）。

やりがちなミス

  • 「便秘で済ませる」（開腹歴・担癌・排ガス停止があるのに）
  • 鎮痛を後回しにする（診断精度は損なわない）
  • 右上腹部痛でいきなり CT（初手は US）
  • 心窩部痛で ECG を取らない（ACS の見逃し）
  • 妊娠可能年齢で妊娠反応を飛ばす
  • 「乳酸が正常だから虚血ではない」と決めつける
  • CT のオーダーに迷って画像が遅れる

帰宅前に最低限

  • 再診基準を具体的に伝える（痛み増悪、発熱、嘔吐持続、血便など）
  • 鎮痛の処方（帰宅後の疼痛管理）
  • 食事・水分の指導
  • フォロー先と時期の確認

  • WSES SBO：Bologna Guidelines for adhesive SBO. World J Emerg Surg. 2013–2024.（絞扼 CT 所見、手術タイミング）
