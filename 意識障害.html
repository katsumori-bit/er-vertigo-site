意識障害

元: 臨床/ER/意識障害.md

#ER #救急 #意識障害 #意識変容 #昏睡 #せん妄

このページの使い方
  1. 血糖を測る（最速で治せる原因）
  2. バイタル・モニター・ルート → 簡易神経診察
  3. バイタルと神経所見で方向性を決める
  4. 採血 + 頭部 CT + 心電図
  5. 原因に応じて専門科コール or 経験的治療

最初に決めること

  • 意識レベルは？（GCS / JCS）
  • 気道は安全か（発声できるか、いびき様呼吸か、嘔吐していないか）
  • いつから？（急性 vs 亜急性 vs 慢性）— 急性発症ほど緊急性が高い
  • 既往・内服薬の情報が取れるか（本人が言えないなら救急隊・家族・施設から）
すぐ動く
  • 気道が保てない（GCS ≤ 8 / いびき様呼吸 / 嘔吐）→ 気道確保・挿管準備
  • ショック（低血圧・頻脈・末梢冷感）
  • 瞳孔不同 + 意識低下 → 脳ヘルニア → 頭部 CT を最優先
  • けいれん持続 → けいれん重積の対応
  • 高度低体温（< 30°C）

まずやること（並行して進める）

血糖測定

意識障害 = まず血糖

最も頻度が高く、最も早く治せる原因。 意識障害を見たら何よりも先に指先血糖を測定する。

  • 低血糖（< 70 mg/dL） → ブドウ糖 50% 40 mL 静注（チアミン先行が望ましい）
  • 高血糖（> 400 + 脱水 + アシドーシス） → DKA / HHS を考える

バイタル・モニター・ルート確保

  • SpO2・心電図・自動血圧 を装着
  • 末梢ルート確保（20G 以上）
  • 体温も必ず測定（低体温・高体温を見逃さない）

簡易神経診察（30秒で）

項目確認すること
GCSE（開眼）V（発語）M（運動）で点数化
瞳孔左右差・サイズ・対光反射
四肢の動き左右差があるか（片麻痺 → 脳血管障害）
項部硬直抵抗があるか（髄膜炎・くも膜下出血）

経験的投与を考える場面

状況投与
低血糖 or 血糖不明チアミン 100 mg 静注 → 続けて 50% ブドウ糖 40 mL 静注
縮瞳 + 呼吸抑制（オピオイド疑い）ナロキソン 0.4–2 mg 静注（反復可）
アルコール多飲・低栄養チアミン 100 mg 静注（Wernicke 脳症予防）
チアミンはブドウ糖の前に

低栄養・アルコール多飲者にブドウ糖を先行すると Wernicke 脳症を誘発しうる。 迷ったらチアミンを先に入れる。

バイタルと神経所見で方向性を決める

バイタルから絞る

異常考える疾患
発熱髄膜炎・脳炎・敗血症・熱中症・悪性症候群
低体温低体温症・甲状腺機能低下・敗血症（高齢者）
高血圧（著明）脳血管障害・高血圧性脳症・頭蓋内圧亢進
低血圧ショック（出血・敗血症・心原性）・副腎不全
低酸素呼吸不全・PE・CO 中毒
徐脈高度房室ブロック・頭蓋内圧亢進（Cushing 反応）・低体温

神経所見から絞る

所見考える疾患
片麻痺 / 瞳孔不同脳血管障害（梗塞・出血）→ 脳卒中疑い
両側縮瞳オピオイド中毒・有機リン中毒・橋出血
両側散瞳抗コリン薬中毒・低酸素脳症・心停止後
項部硬直髄膜炎・くも膜下出血 → 腰椎穿刺を検討
眼球偏位けいれん後（偏位と逆側）・脳幹病変
除脳硬直 / 除皮質硬直重篤な脳幹障害 → 予後不良のサイン

採血・検査

採血

区分項目
血算CBC, Diff-%
生化BUN, Cr, Na, K, Ca, Glu, AST, ALT, NH3, T-Bil
凝固PT, APTT
血ガスpH, PaCO2, PaO2, HCO3, Lac, CO-Hb
その他CRP, TSH（亜急性なら）, 薬物スクリーニング（疑えば）
特に見逃したくない代謝異常
  • 低 Na（< 120 で意識障害）
  • 高 Ca（悪性腫瘍）
  • 高 NH3（肝性脳症）
  • 高 Lac + 代謝性アシドーシス（敗血症・中毒・虚血）
  • CO-Hb 上昇（CO 中毒 — 冬季・閉鎖空間）

頭部 CT

以下のいずれかがあれば撮影する：

  • 新規の巣症状（片麻痺・瞳孔不同・失語）
  • 頭部外傷あり or 疑い
  • 抗凝固薬内服中
  • 急性発症の意識障害で原因不明
  • 項部硬直（くも膜下出血の除外）
  • けいれん後

→ 実質的に、原因不明の意識障害はほぼ全例 CT を撮る

心電図

  • 不整脈（VT, 高度AVB, 新規AF → 脳塞栓の可能性）
  • ST 変化（ACS → 心原性ショック → 低灌流）
  • QT 延長（薬剤性・電解質異常）

それでも原因がわからないとき

非けいれん性てんかん重積（NCSE）を忘れない

明らかなけいれんがなくても、意識障害の原因が NCSE であることがある。 採血・CT・心電図で原因不明 → 脳波検査 を検討。 頻度：原因不明の意識障害の 約 8–20% が NCSE — Leitinger (2019) Neurology

  • 精神疾患は除外診断：器質的原因をすべて否定してから考える
  • 薬物スクリーニング：疑えば尿中・血中の薬物検査を追加

disposition

  • 挿管 / ショック / 脳ヘルニア徴候 → ICU
  • 脳血管障害 → SCU / 脳外科 or 脳内科
  • 髄膜炎・脳炎 → ICU or 感染症科
  • 代謝異常（補正で改善） → 入院 + 原因精査
  • 中毒 → 経過観察入院（重症なら ICU）
  • 低血糖（補正後に意識回復 + 原因明確） → 経過観察後に帰宅も検討
