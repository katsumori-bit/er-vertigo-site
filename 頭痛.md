#ER #救急 #頭痛 #SAH #髄膜炎 #片頭痛

> [!summary] このページは「診断がつくまで」
>
> 1. **危険な二次性頭痛**を 5 秒でふるい分ける（SNNOOP10）
> 2. **パターン別に first に考える疾患**を決める
> 3. **CT / CTA / LP** の適応を判断
> 4. **診断がついたら疾患ページへ** → [[くも膜下出血]] / [[髄膜炎・脳炎]] / [[頭痛の対症療法]] など

> [!danger] 一次性頭痛と決め打ちしない赤旗（SNNOOP10）
>
> - **雷鳴頭痛**（1 時間以内にピーク）
> - **発熱 / 項部硬直 / 意識障害**
> - **局所神経症状 / けいれん**
> - **妊娠中 / 産褥 6 週以内の新規頭痛**
> - **眼痛 + 視力低下 + 充血**
> - **50 歳以降の新規頭痛**
> - **癌 / 免疫不全 / 抗凝固薬**
> - **乳頭浮腫 / 体位で増悪 / 進行性悪化**
> - **外傷後・最近の穿刺操作**

## 最初の 5 秒

- バイタル・意識
- 雷鳴頭痛か
- 発熱・項部硬直
- 局所神経症状（軽い失語・半盲・歩行失調・複視を落とさない）
- 妊娠・産褥、50 歳以上の新規、癌・免疫不全・抗凝固薬
- 眼痛・視力低下・充血
- **いつもと同じ頭痛か、初めて / 最悪 / 違うタイプか**

> [!tip] 頭痛で最初に見るもの
> **発症様式・ピーク到達時間・初回か・随伴症状・背景（妊娠・癌・免疫不全・抗凝固薬・穿刺/外傷）**

## パターン別：first に考える疾患とやること

| パターン | first | ED でやること | 詳細 |
|---|---|---|---|
| **雷鳴頭痛**（1h 以内ピーク） | **くも膜下出血** | 頭部単純 CT → 必要で CTA/LP、血圧、神経所見 | [[くも膜下出血]] / [[脳静脈洞血栓症]] / [[頸動脈・椎骨動脈解離]] |
| **発熱 + 項部硬直 / 意識障害** | **髄膜炎・脳炎** | 血培 → 抗菌薬 **1 h 以内**、LP（CT 先行判断は下記）、デキサメタゾン併用考慮 | [[髄膜炎・脳炎]] |
| **神経脱落症状 / 意識障害** | **脳卒中・脳炎・脳腫瘍・膿瘍・PRES** | 神経診察、頭部画像、必要で血管画像、血圧・血糖 | [[神経診察]] / [[脳卒中]] |
| **妊娠中 / 産褥 6 週以内** | **子癇前症 / CVST / PRES / RCVS** | 血圧、尿蛋白、眼症状、画像をためらわない | [[妊娠・産褥の頭痛]] / [[脳静脈洞血栓症]] |
| **眼痛 + 視力低下 + 充血** | **急性閉塞隅角緑内障** | 視力・瞳孔・眼位・角膜混濁、眼圧、**眼科コンサルト** | [[急性閉塞隅角緑内障]] |
| **50 歳以上の新規頭痛** | **巨細胞性動脈炎** | 側頭部痛・顎跛行・視覚症状・頭皮圧痛、**ESR・CRP**、強く疑えばステロイドを遅らせない | [[巨細胞性動脈炎]] |
| **頭蓋内圧異常**（朝強い / 咳いきみで増悪 or 立位で増悪） | **ICP 亢進 / 低髄液圧** | 眼底、神経診察、画像、LP 適応は慎重 | [[頭蓋内圧亢進]] / [[低髄液圧症候群]] |
| **典型的な一次性頭痛** | **片頭痛・緊張型・群発** | 赤旗がなく以前と同じパターンなら対症療法へ | **[[頭痛の対症療法]]** |

> [!tip] 雷鳴頭痛での CT/LP のタイミング（Ottawa SAH Rule / ACEP）
>
> - **発症 6 時間以内 + 神経所見正常 + 高性能 CT 陰性** → SAH は実質除外可
> - **6 時間超 or なお疑いが残る** → **LP または CTA** を追加

> [!warning] 髄膜炎で CT を先行する条件（WHO）
> GCS 低下 / 局所神経徴候 / 脳神経障害 / 乳頭浮腫 / 新規けいれん / 重度免疫不全
> **該当しなければ CT を待たずに LP → 抗菌薬**

## 最低限の診察

意識 / 血圧 / 体温 / 項部硬直 / 瞳孔・眼位・眼球運動 / 視力 / 顔面左右差 / 構音 / 上下肢麻痺 / 感覚 / 小脳失調 / 歩行 / 可能なら乳頭浮腫
詳細 → [[神経診察]]

## 検査の置き方

| 検査 | 適応 |
|---|---|
| **頭部単純 CT**（急ぐ） | 雷鳴頭痛、神経症状、意識障害、ICP 亢進疑い、外傷後、抗凝固薬、発熱で占拠性病変を否定したい |
| **CTA / CTV** | SAH 疑いで CT 陰性だが未解決、解離疑い、CVST 疑い、RCVS 疑い |
| **LP** | 髄膜炎 / 脳炎疑い、SAH 疑いで CT 陰性、占拠性病変・ヘルニアリスク評価の上で |
| **ESR / CRP** | 50 歳以上の新規頭痛（GCA） |

## 一次性頭痛の初期対応（赤旗がないとき）

- 暗室安静・補液・制吐
- **オピオイド（モルヒネ・ペンタゾシン・トラマドール）は第一選択にしない**
- 具体的な薬剤・用量・帰宅処方は → **[[頭痛の対症療法]]**

## 救急外来での実戦サマリ

> [!summary] 一行ずつ
>
> - **雷鳴** → SAH first
> - **発熱 / 項部硬直** → 髄膜炎 first
> - **神経症状 / 意識障害** → 脳卒中・脳炎 first
> - **妊娠 / 産褥** → 子癇前症 / CVST / PRES first
> - **眼痛 + 視力低下** → 緑内障 first
> - **50 歳以上の新規** → GCA first
> - **赤旗なし + いつもの頭痛** → 一次性頭痛 → [[頭痛の対症療法]]

---

## やりがちなミス

- **「いつもの片頭痛」と決めつけ神経所見を取らない**
- **雷鳴頭痛で CT 陰性 = SAH 除外と早合点**（6h 超なら LP/CTA）
- **髄膜炎疑いで CT 待ちして抗菌薬が遅れる**
- **50 歳以上の側頭部痛で ESR を測らず帰す**（GCA で失明）
- **妊娠・産褥の頭痛を片頭痛で帰す**（子癇前症・CVST）
- **オピオイドで片頭痛を抑える**（乱用性頭痛・慢性化）
