嘔気・嘔吐

元: 臨床/ER/嘔気・嘔吐.md

このページの使い方
  1. まず 危険徴候 を見る
  2. 次に どの系統の問題か を決める
  3. 必要なところだけ開く
  4. 詳細はリンク先で確認する

最初に決めること

  • 今すぐ緊急対応が必要か
  • 腹部症状が主か
  • 中枢・前庭症状が主か
  • 全身性・代謝性か
  • 妊娠関連か
  • 薬剤・中毒関連か

0. 危険徴候

1つでもあれば、原因検索を急ぐ
  • ショック / 重度脱水 / ぐったりしている
  • 頻回嘔吐で水分が全く入らない
  • 強い腹痛, 腹膜刺激, 腹部膨満
  • 胸痛 / 背部痛
  • 激しい頭痛, 意識障害, けいれん, 局所神経症状
  • 頭部外傷後
  • 吐血 / コーヒー残渣様嘔吐
  • 妊娠の可能性 + 腹痛 or 性器出血
  • 糖尿病患者の嘔吐
  • 術後 / ヘルニア / 排ガス排便停止
  • 高齢者・免疫不全・担癌患者で原因不明
最初の実務
  • バイタル確認
  • 脱水評価
  • 血糖測定
  • 妊娠可能年齢なら hCG確認
  • 腹部所見 / 神経所見
  • 内服・中毒・大麻使用歴の確認

1. どの系統の問題か

A. 腹部が主

  • 腹痛が前景
  • 腹部膨満
  • 排ガス排便低下
  • 食後悪化
  • 限局した圧痛がある

→ 腹痛 / 腸閉塞 / 急性膵炎 / 胆道感染・胆嚢炎 / 急性胃腸炎

B. 中枢・前庭が主

  • 頭痛
  • 意識変容
  • 神経脱落症状
  • めまい
  • 頭部外傷後

→ 頭痛 / 神経診察 / めまい

C. 全身性・代謝性が主

  • 発熱
  • 口渇・多尿
  • 強い倦怠感
  • 呼吸異常
  • 感染症や敗血症を疑う

→ DKA/HHS / 敗血症 / 尿路感染症 / 電解質異常

D. 妊娠関連

  • 妊娠可能年齢
  • 月経遅れ
  • 妊娠判明済み
  • 腹痛 / 性器出血の有無を必ず確認

→ 妊娠初期診療 / 妊娠悪阻・つわり / 異所性妊娠

E. 薬剤・中毒関連

  • 新規内服
  • 薬剤増量
  • アルコール
  • 大麻
  • 毒物曝露

→ 薬剤性嘔気・嘔吐 / 中毒総論 / CHS

2. 5秒フロー

迷ったらこの順で考える
  1. 危険徴候があるか
  2. 腹部症状が主か
  3. 頭痛・神経症状・めまいがあるか
  4. 発熱・代謝異常・糖尿病があるか
  5. 妊娠可能性があるか
  6. 薬剤・大麻・中毒が関係しそうか
  7. どれにもはっきり乗らなければ、支持療法しつつ再評価する

3. 初期対応

原因未確定でも先にやること
  • 補液が必要か判断する
  • 血糖を確認する
  • 脱水と電解質異常を評価する
  • 制吐薬で再評価しやすい状態を作る
  • 効かなければ、想定している原因が違う前提で見直す
よく出す検査
  • 血糖
  • hCG
  • CBC
  • 電解質
  • 腎機能
  • 肝胆道系酵素
  • 必要時 CRP
  • 必要時 VBG / ABG
  • 必要時 Lipase
  • 尿検査
  • ECG
  • 必要時 画像検査

4. 病歴で拾うこと

病歴
  • いつからか
  • 急性か、反復か、慢性か
  • 食事との関係
  • 嘔吐内容：食物残渣 / 胆汁 / 血液 / 便臭
  • 腹痛、下痢、便秘、排ガス低下
  • 発熱、頭痛、めまい、胸痛、背部痛
  • 同居人の同症状、旅行、食事歴
  • 新規内服、増量、飲み忘れ
  • アルコール、大麻、その他薬物
  • 糖尿病、腎不全、妊娠、術後、癌、ヘルニア歴

5. 身体所見で外しにいくもの

腹部
  • 膨満
  • 限局圧痛
  • 腹膜刺激
  • ヘルニア
  • 腸雑音異常
  • CVA tenderness
神経
  • 意識レベル
  • 項部硬直
  • 眼球運動
  • 小脳失調
  • 局所神経症状
全身
  • 口腔乾燥
  • 頻脈
  • 起立性低血圧
  • 発熱
  • Kussmaul様呼吸
  • 中毒を疑う所見

6. すぐ使う鑑別

まず多いもの

  • 急性胃腸炎
  • 薬剤性嘔気・嘔吐
  • 妊娠悪阻・つわり
  • めまい
  • 片頭痛

見逃したくないもの

  • 腸閉塞
  • 急性膵炎
  • 胆道感染・胆嚢炎
  • DKA/HHS
  • 頭痛
  • 急性冠症候群
  • 異所性妊娠
  • 敗血症

忘れやすいもの

  • CHS
  • 胃排出遅延・胃不全麻痺
  • 尿路感染症
  • 副腎不全
  • 尿毒症
  • 有害事象・薬物中毒

7. 制吐薬の実務メモ

制吐薬は原因治療ではない
  • 補液と原因検索を優先する
  • 制吐薬は再評価しやすくするために使う
  • 効かない嘔吐では危険疾患や CHS を考え直す
よく使う選択肢
  • オンダンセトロン
    • 幅広く使いやすい
    • QT延長に注意
  • メトクロプラミド
    • 胃排出遅延や片頭痛関連で使いやすい
    • 錐体外路症状、アカシジアに注意
  • プロクロルペラジン / ハロペリドール系
    • 状況によって有効
    • QT延長、錐体外路症状に注意
CHSを疑う時
  • 大麻常用
  • 周期性嘔吐
  • 熱いシャワーで軽くなる
  • 通常の制吐薬が効きにくい

→ CHS

8. 帰宅か観察か

帰宅を考えやすい
  • 危険徴候なし
  • 重篤な器質疾患が低そう
  • 水分摂取が可能
  • バイタル安定
  • 再診目安を説明できる
入院 / 観察を考える
  • 脱水が強い
  • 経口摂取不能
  • 電解質異常 / AKI
  • 腸閉塞, 膵炎, 胆道感染, DKA, 中枢疾患などを疑う
  • 妊娠関連で重症
  • 診断が不確実で再評価が必要

9. 再診・悪化時の赤旗

  • 腹痛が強くなる
  • 胸痛 / 背部痛が出る
  • 頭痛 / ふらつき / 意識障害が出る
  • 血を吐く
  • 水分が全く入らない
  • 尿が極端に減る
  • 妊娠中に腹痛 / 出血が出る
