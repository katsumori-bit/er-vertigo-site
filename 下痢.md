#ER #救急 #下痢 #急性下痢 #脱水 #感染性腸炎

> [!summary] このページは「診断がつくまで」
>
> 1. **赤旗と "胃腸炎で済ませてはいけない疾患"** を外す
> 2. **炎症性 / 非炎症性**で枝分け、**便検査の適応**を判断
> 3. **初期対応（補液・制吐・鎮痛）**まで走る
> 4. **診断がついたら疾患ページで続き**を読む → [[急性胃腸炎]] / [[CDI]] / [[STEC・EHEC]] / [[旅行者下痢]]

> [!tip] 主訴が下痢ではないとき
> - **嘔吐が前景**で下痢は軽い → [[嘔気・嘔吐]]
> - **腹痛が前景** → [[腹痛]]
> - **血便・粘血便が主体** → [[血便・下血]]
> - **発熱が前景**で下痢は軽い → [[発熱]]

## 最初に決めること

- **不安定（ショック・敗血症・重度脱水）** ではないか
- **「胃腸炎」で済ませてはいけない疾患** ではないか
- **炎症性下痢（血便 / 38.5℃以上の発熱 / 強い腹痛）** か、**非炎症性下痢（水様・発熱なし）** か
- **抗菌薬使用歴・入院歴** があるか（→ [[CDI]]）
- **免疫不全・担癌・高齢・小児・妊婦** ではないか
- **渡航歴・集団発生・食歴** はあるか

> [!danger] 先に拾う赤旗
>
> - ショック / 重度脱水 / 起立性低血圧
> - 高熱（>38.5℃）+ 血便
> - **痛みの割に腹部所見が乏しい**（→ 腸間膜虚血）
> - 強い限局性腹痛・腹膜刺激（→ 外科腹症 → [[腹痛]]）
> - 嘔吐主体 + 排ガス排便停止 + 腹部膨満（→ 腸閉塞 → [[腹痛]]）
> - 糖尿病患者で嘔吐 + 下痢 + 多尿 + 過呼吸（→ [[DKA・HHS]]）
> - 高齢 + 心房細動 / 血管リスク + 腹痛（→ 腸間膜虚血）
> - 抗菌薬使用後 / 入院歴あり（→ [[CDI]]）
> - 担癌 / 化学療法中（→ [[好中球減少時発熱]]・腸炎・CDI）
> - 免疫不全（HIV、ステロイド、生物学的製剤）
> - 抗凝固薬内服 + 血便
> - 大量水様便で脱水進行が速い（→ コレラ様、Vibrio、毒素性）

## このページで最優先に外す疾患

下痢は急性胃腸炎が圧倒的多数だが、**胃腸炎で済ませると致命的**になる疾患がある。

- **腸間膜虚血**（高齢・Af・所見に乏しい激痛 + 下痢／血便）
- **絞扼性腸閉塞 / paradoxical diarrhea**（液状便だけが流れ出る）
- **[[DKA・HHS]]**（嘔吐 + 下痢 + 多尿 + Kussmaul）
- **[[副腎不全・副腎クリーゼ]]**（嘔吐 + 下痢 + 低血圧 + 低 Na・高 K）
- **敗血症**（感染巣不明の発熱 + 下痢）
- **重症 / 劇症 [[CDI]]**（高齢・抗菌薬曝露後、白血球著増、AKI）
- **[[STEC・EHEC]]**（血便 + 強い腹痛、抗菌薬と止痢薬は HUS リスク）
- **コレラ・Vibrio・毒素性**（米のとぎ汁様便、急速脱水）
- **異所性妊娠**（妊娠可能年齢で腹痛 + 嘔気は妊娠反応必須）
- **AKI / 低 Na / 低 K** による嘔気（電解質を測らずに帰さない）

> [!danger] 迷ったらまずこの 5 つを外す
> **腸間膜虚血・腸閉塞・DKA・敗血症・CDI**

## カルテで先に拾う

- **抗菌薬使用歴**（直近 8 週以内）→ **CDI**
- **入院歴 / 施設入所**（直近 12 週）→ **CDI**、ノロ集団感染
- **担癌・化学療法**（直近 2–3 週）→ 好中球減少、CDI、免疫関連腸炎（irAE）
- **糖尿病**（特に 1 型）→ DKA
- **副腎不全のリスク**（ステロイド長期内服中の中断、下垂体疾患）→ 副腎クリーゼ
- **抗凝固薬・抗血小板薬** → 血便、消化管出血
- **心房細動 / 血管リスク** → 腸間膜虚血
- **PPI 長期内服** → CDI、サルモネラ等のリスクが上がる
- **NSAIDs / 鉄剤 / メトホルミン / SGLT2 / 化学療法薬 / 新規開始薬**
- **妊娠可能性**
- **渡航歴・生水・生肉・生魚・買い置き食品・集団発生**

## 炎症性 vs 非炎症性で枝分け

| | 炎症性下痢 | 非炎症性下痢 |
|---|---|---|
| 便 | 血性・粘血便、少量頻回 | 水様、大量 |
| 発熱 | あり（しばしば >38.5℃） | なし〜微熱 |
| 腹痛 | 強い、しぶり腹（tenesmus） | 軽い〜中等度、疝痛 |
| 病原体 | サルモネラ、シゲラ、カンピロバクター、**STEC**、赤痢アメーバ、**CDI**、IBD | ノロ、ロタ、ETEC、Vibrio、毒素型食中毒、ウイルス性 |
| 検査 | 便培養 ± CDI ± STEC ± 採血 + 画像考慮 | 通常は不要、補液中心 |
| 抗菌薬 | 重症 / 高リスクで考慮（**STEC は避ける**） | 原則不要 |

詳細は → [[急性胃腸炎]]

## まず取るべき最小情報

- **発症様式 / 経過**：突然 / 徐々、何時間 / 何日
- **下痢**：回数、性状（水様・泥状・血性・粘血便）、量、米のとぎ汁様か
- **嘔吐の有無**（嘔吐前景なら → [[嘔気・嘔吐]]）
- **腹痛**：部位、強さ、持続 / 疝痛、移動の有無
- **発熱**の有無と程度
- **食歴**：生もの、貝類、鶏肉、卵、ハンバーグ、買い置き食品、集団発生
- **渡航歴**：地域、生水、ストリートフード
- **薬歴**：抗菌薬、PPI、化学療法、新規薬
- **入院歴 / 施設入所**：直近 12 週
- **既往**：糖尿病、IBD、HIV、担癌、ステロイド、副腎不全
- **女性**：最終月経、妊娠可能性

## 身体所見

- **バイタル + 起立性血圧変化**（起立で sBP 20 mmHg 以上低下、脈拍 30/分以上上昇は脱水を示唆）
- **意識・倦怠感**：DKA、副腎不全、敗血症のサイン
- **粘膜乾燥・皮膚ツルゴール・毛細血管再充満時間**
- **呼吸様式**：Kussmaul（DKA）
- **腹部**：膨満、腸雑音、限局圧痛、腹膜刺激、ヘルニア門
- **直腸診**：血便・タール便・宿便
- **CVA tenderness**：腎盂腎炎は嘔気・下痢のことも

> [!tip] 「胃腸炎にしては痛い」を見逃さない
> 急性胃腸炎の腹痛は**疝痛 + 排便で軽くなる**のが基本。
> **持続性・限局性・腹膜刺激・所見に乏しい激痛**があれば、外科腹症 / 虚血を考え直し → [[腹痛]]

## 検査の考え方

### 血液（基本セット）

- **CBC、CRP**
- **生化**：Na, K, Cl, BUN, Cr, Glu, AST, ALT, LDH, T-Bil, ALB, Ca, Mg
- **乳酸**（重症感・虚血疑い・ショック）
- **リパーゼ**（心窩部痛・持続嘔吐）
- **静脈血ガス**（重症脱水・代謝性アシドーシス疑い・DKA 疑い）
- **血糖**：必ず測る（DKA / HHS の見落とし回避）

### 尿

- **尿定性 + ケトン**（DKA・脱水評価）
- **妊娠反応**（妊娠可能年齢で原則全例）

### 便検査（適応を絞る）

ACG 2016 / IDSA 2017 では **急性下痢の全例に便培養は推奨しない**。
判断軸はシンプルに **「結果で方針が変わるか」**。健常成人の水様下痢の大多数はウイルス性で自然軽快し、検査しても治療は変わらない。

> [!check] 便検査を考慮する場面（どれかに当てはまるとき）
>
> - **血便 / 粘血便**（→ 便培養 + STEC）
> - **高熱（>38.5℃）+ 強い腹痛**（→ 便培養）
> - **重症脱水・敗血症像・入院を要する**
> - **免疫不全・担癌・妊婦・高齢（特に施設入所）**
> - **14 日以上持続する下痢**（→ 慢性下痢のワークアップ）
> - **直近 8 週の抗菌薬使用 / 入院歴 / PPI 長期 / 施設入所**（→ **CDI 検査**、[[CDI]]）
> - **集団発生・食中毒疑い・公衆衛生上の調査**
> - **渡航歴・MSM・好酸球増多**（→ 寄生虫）

- **便培養**（サルモネラ・シゲラ・カンピロバクター・STEC O157）：炎症性 / 重症 / 免疫不全 / 集団発生
- **STEC（O157）迅速・培養**：血便・小児・HUS 既往 → [[STEC・EHEC]]
- **CDI（GDH + トキシン / NAAT）**：**24 時間以内に 3 回以上の不形便** + リスク因子 → [[CDI]]
- **寄生虫検査**：渡航歴・MSM・慢性化・好酸球増多
- **ノロ・ロタ PCR / 迅速**：**原則ルーチンでは出さない**（下記参照）

> [!note] ノロを「PCR で確定」しに行かない理由
>
> - **健常成人の非重症ノロは対症療法で 24–72 h で軽快** → 結果が出ても方針は変わらない
> - ノロ PCR の**保険適用は限定的**（3 歳未満・65 歳以上・悪性腫瘍・免疫不全・臓器移植後 等）
> - **下痢便の採取は患者・スタッフの曝露リスク**が高い（ノロはアルコール無効、次亜塩素酸が必要）
> - **家族内にノロ罹患者・流行期・潜伏期と症状が一致** なら**臨床診断で十分**（今回の症例のように状況証拠で押さえる）
> - ただし、以下では検査閾値を下げる：
>     - **施設・学校・飲食店の集団発生を疑うとき**（公衆衛生目的で PCR → 保健所と連携）
>     - **小児・高齢者・免疫不全・重症**（鑑別を絞る / 隔離方針が変わる）

> [!warning] 「採便が難しいから検査しない」は許されるか
> 下痢便の採取は現実的に難しい場面が多く、「**結果で方針が変わらない**」ときは省略してよい。
> ただし、**以下のときは採便を諦めず工夫する**：
>
> - 血便、CDI リスク、重症、免疫不全、集団発生
> - 採取法：**直腸スワブ / オムツ・パッドからの採取 / 失禁時に備えて容器をベッドサイドに置いておく**（CDI は "形のない便" の確認が必須）
> - 赤旗のある症例で「採りにくいから検査省略」はしない。検査しない判断を選ぶときはカルテに**「なぜ検査しなかったか（リスク層別・臨床診断根拠）」**を一言残す

## 重症度評価

> [!check] 重症化のサイン
>
> - sBP < 90、脈拍 > 120、起立性低血圧（+）
> - 意識変容
> - 経口摂取不能・尿量減少
> - AKI（Cr 上昇 / BUN/Cr 比 >20）
> - 著明な電解質異常（Na <130 or >150、K <3.0 or >5.5）
> - 著明な代謝性アシドーシス（HCO3 <15、乳酸 >2）
> - 高熱 >39℃ + 血便
> - 腹膜刺激・著明な腹部膨満
> - 妊婦、高齢、免疫不全、担癌、IBD 既往

## 初期対応（診断がつくまで）

> [!tip] このページは「診断がつくまで」
> 具体的な処方・用量・止痢薬・抗菌薬・帰宅指導は、**診断ごとのリンク先**で解説。
> → [[急性胃腸炎]] / [[CDI]] / [[STEC・EHEC]] / [[旅行者下痢]] / [[腹痛]] / [[嘔気・嘔吐]]

- **蘇生**：不安定なら ABC、細胞外液ボーラス（成人 500–1000 mL、小児 20 mL/kg）
- **補液**：経口可なら **ORS 少量頻回**、経口不能・重症脱水は **細胞外液 IV**（詳細・ORS 組成 → [[急性胃腸炎]]）
- **制吐**：**オンダンセトロン 4 mg IV**（QT 注意。選択肢・帰宅処方 → [[嘔気・嘔吐]]）
- **腹痛の鎮痛**：**アセリオ 1000 mg 15 分点滴**を基本（早期鎮痛は診断を邪魔しない）。疝痛に **ブスコパン 20 mg**。**NSAIDs は避ける**（AKI・HUS・出血）。用量詳細・禁忌・帰宅処方 → [[腹痛]]
- **電解質補正**：Na・K・Mg・Ca・HCO3
- **止痢薬 / 抗菌薬 / 食事 / 帰宅処方セット** → 診断がついてから [[急性胃腸炎]] / [[CDI]] / [[STEC・EHEC]] / [[旅行者下痢]] で解説

> [!warning] ED で即決しておくべき 3 つ
>
> - **止痢薬は血便・高熱・STEC 疑い・CDI 疑い・小児では使わない**（詳細 → [[急性胃腸炎]]）
> - **STEC 疑いに経験的抗菌薬は出さない**（HUS → [[STEC・EHEC]]）
> - **NSAIDs で痛みを抑えて帰さない**（AKI・HUS・出血 → [[腹痛]]）

## disposition

### 入院 / 観察を考える

- 重度脱水 / 経口摂取不能
- AKI、著明な電解質異常、代謝性アシドーシス
- 重症 [[CDI]] / 劇症 CDI
- [[STEC・EHEC]] + HUS 兆候（特に小児）
- 敗血症像
- 外科腹症 / 虚血が否定できない
- 免疫不全 / 担癌 / 妊婦 / 高齢で重症
- DKA / HHS / 副腎クリーゼ
- 診断不確実で再評価が必要

### 帰宅を考える

- バイタル安定、起立性変化なし
- 経口補液が可能（ED で 1–2 時間 ORS が入って嘔吐しない）
- 危険徴候・赤旗なし
- 重症化リスク（高齢・免疫不全・担癌・妊娠・DM）が低い
- 再診基準を具体的に説明できる

> [!warning] 帰宅時に必ず伝える
>
> - **再診基準**：高熱が続く、血便、強い腹痛、嘔吐で水分が入らない、尿量減少、ぐったりする、意識がぼんやりする
> - **経口補液の方法**：ORS を少量頻回（5–10 mL を 5–10 分ごとから）
> - **食事**：消化のよい炭水化物中心、無理に絶食しない
> - **手洗い・タオル分け**（家族内感染予防、ノロ・ロタ・CDI）
> - **就労・就学制限**：飲食業・医療・保育・学校の規定を確認

## 折りたたみ用メモ

### 外してはいけないもの

- 腸間膜虚血
- 絞扼性腸閉塞 / 腸閉塞
- [[DKA・HHS]]
- [[副腎不全・副腎クリーゼ]]
- 重症 [[CDI]] / 劇症 CDI
- [[STEC・EHEC]] + HUS
- 異所性妊娠
- 敗血症（菌血症 / 腹腔内感染）
- 急性虫垂炎・急性膵炎・胆管炎

### やりがちなミス

- **「胃腸炎」と決めつけて電解質と血糖を測らない**
- **若年女性で妊娠反応を取らない**
- **抗菌薬使用歴を聞かずに CDI を見落とす**
- **血便にロペラミドを処方する**
- **STEC 疑いで抗菌薬を出して HUS を増やす**
- **高齢 + Af + 「腹痛 + 下痢」を胃腸炎で帰す**（→ 腸間膜虚血）
- **嘔吐主体 + 排ガス停止を「胃腸炎」と判断**（→ 腸閉塞）
- **DKA をブドウ糖含有輸液で進行させる**
- **"採便が難しい" を理由に赤旗症例で検査を省略**（血便・CDI リスク・重症では採便方法を工夫する）
- **NSAIDs で痛みを抑えて帰す**（AKI・HUS・消化管出血）
- **ブスコパンで痛みが取れた＝胃腸炎、と決めつける**（胆管炎・腸閉塞・虚血でも一時的に効く）

### 帰宅前に最低限

- バイタル + 起立性変化 + 体温の再評価
- 経口補液が ED で実際に入ったか
- 採血の結果（電解質・腎機能・血糖）を確認
- 再診基準を口頭 + 書面で伝える
- 制吐薬・ORS の処方とフォロー先
