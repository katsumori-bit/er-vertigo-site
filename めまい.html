#ER #救急 #めまい #dizziness #AVS #BPPV #HINTS #神経診察 #脳梗塞

このページの使い方
  1. まず #Red flags を確認する
  2. 次に、どの症候群に当てはまるかを決める
  3. 該当する枝だけ読む

Red flags

  • 歩行不能 / 起立不能
  • 構音障害
  • 複視
  • 嚥下障害
  • 片麻痺 / 感覚障害
  • 明らかな体幹失調 / 四肢失調
  • 垂直眼振
  • 方向交代性眼振
  • 強い後頭部痛 / 頸部痛
  • 新規難聴 + 中枢所見
  • 高齢 + 血管危険因子
  • 抗凝固薬内服
  • 明らかな失神前状態 / 胸痛 / 動悸
1つでもあれば要注意

末梢性めまいとして安易に片付けない。 このようなRed flag所見があれば、以下の重篤な疾患を十分に疑い除外すること：

  • #後方循環梗塞（椎骨脳底動脈系梗塞：小脳梗塞、延髄梗塞、橋梗塞 AICA梗塞などを含む）
  • 椎骨動脈解離
  • くも膜下出血（急な頭痛や頸部痛があれば）
  • 急性脳出血（抗凝固薬内服例や重度神経症状の場合）
  • 心原性失神・致死性不整脈・大動脈解離 など、中枢神経系疾患以外の致死的疾患も文脈に応じて考慮すること

最初に決めること

  • #1. 失神前状態 / 全身性ふらつき か
  • #2. AVS（急性持続性前庭症候群）か
  • #3. t-EVS（頭位などで誘発される短時間発作性前庭症候群）か
  • #4. s-EVS（誘因なし反復発作性前庭症候群）か
Timing and Triggers

めまいは「回る」「ふわふわする」といった症状の質よりも、 持続時間・反復性・誘因・随伴症状・ターゲット診察で分類する。 詳しくは 「TiTrATE めまい」

その場で行う最小診察

ほぼ全例で下記。HINTS は #2. AVS で適応と手技の訓練がそろったときの追加。短時間・頭位誘発が主（#3. t-EVS）なら原則使わない。

  • バイタル（循環器・失神前が疑われれば臥位／立位も）
  • 座位・立位・歩行（座位を保てるか、立てるか、歩けるか）
  • 眼振（自発／頭位誘発、方向、固視）… 眼振の見方
  • 聴力（指こすり・ささやきなどで可）
  • 神経診察
  • 採血（血糖ほか、電解質・Hbなど）
  • 12誘導ECG（胸痛・動悸・失神前・不整脈懐疑、施設方針）

HINTS

頭部衝動・眼振・偏視による末梢／中枢（後循環）鑑別のベッドサイドセット。手技・判定の細部は HINTS。適応・解釈・次の一手は #2. AVS。

初期評価の要点

上のリストをサクッと終わらせる。

症候群で分ける

1. 失神前状態

こういう訴えならこちらを優先

  • 眼前暗黒感
  • ふっと遠のく感じ
  • 立位で悪化
  • 動悸
  • 胸痛
  • 一過性意識消失に近いエピソード

主な鑑別

  • 起立性低血圧
  • 不整脈
  • 低血糖
  • 脱水
  • 貧血
  • 薬剤性めまい

まず行うこと

  • 採血（血糖含む・適宜各項目）
  • 12誘導ECG
  • モニター
  • 起立性変化
  • 胸部症状があれば循環器評価

2. AVS

定義

  • 急性発症
  • 持続性のめまい / 回転感 / 平衡障害
  • 悪心・嘔吐
  • 眼振または歩行不安定
  • 通常は24時間以上持続

主な鑑別

  • #前庭神経炎
  • #後方循環梗塞
  • 小脳梗塞
  • 脳幹梗塞
  • まれに AICA梗塞

AVSで確認すること

  • #HINTS（適応がある場合）
  • 聴力（指こすり・ささやきなど）
  • 座位・立位・歩行（歩行評価）
  • 神経診察

AVSで末梢性を示唆する所見

  • 一方向性水平眼振
  • 明らかな神経脱落症状なし
  • 難聴なし
  • #HINTS が末梢パターン

AVSで中枢性を示唆する所見

  • 歩行不能 / 強い体幹失調
  • 垂直眼振
  • 方向交代性眼振
  • #HINTS が中枢パターン、または判断困難
  • 構音障害、複視、嚥下障害、片麻痺
  • 頭痛 / 頸部痛
  • 血管危険因子が強い
AVSの原則

CTをルーチンには使わない。 訓練された診療者がいるなら、まずベッドサイド評価を優先する。 #HINTS が中枢パターン、または判断困難なら #MRI / MRA を考える。

3. t-EVS

典型

  • 頭位変換で誘発される
  • 数秒〜1分程度
  • 発作間欠期は比較的落ち着く

主な鑑別

  • #BPPV
  • まれに、位置変化で悪化する中枢性病変

t-EVSで行うこと

  • Dix-Hallpike
  • 必要に応じて Supine roll test

後半規管BPPVなら

  • Epley法
BPPVの原則

典型的な後半規管BPPVでは、 不要な画像検査や漫然とした前庭抑制薬は避ける。

4. s-EVS

典型

  • 明確な誘因なし
  • 発作性に起こる
  • 発作間欠期は比較的落ち着く

主な鑑別

  • #Ménière病
  • 前庭性片頭痛
  • TIA
  • 不整脈
  • 起立性低血圧

s-EVSで重視すること

  • 脳神経診察
  • 視野
  • 眼球運動
  • 四肢協調運動
  • 座位・立位・歩行（歩行評価）
  • 聴覚症状
  • 血管危険因子
  • 胸痛 / 動悸 / 失神
s-EVSでの画像

中枢と末梢の区別にCTは使わない。 TIAや大血管病変が気になるなら、血管評価を考える。

救急でまず押さえる代表疾患

前庭神経炎

  • AVSの代表
  • 強い持続性めまい
  • 悪心・嘔吐を伴う
  • 難聴は通常ない
  • 神経所見があれば再考する
  • ステロイドは「全員に必須」ではなく、発症3日以内であれば短期投与を患者と相談して検討する位置づけ

BPPV

  • t-EVSの代表
  • Dix-Hallpike で後半規管BPPVを診断する
  • Epley法 で治療する
  • 不要な画像検査は減らす
  • 前庭抑制薬を漫然と続けない

Ménière病

  • 自発性めまい発作
  • 発作は20分〜12時間
  • 低〜中音域優位の感音難聴
  • 耳鳴り / 耳閉感を伴うことが多い
  • ただし救急では、難聴があるから末梢性と決めつけない

後方循環梗塞

  • AVSで見逃してはいけない危険診断
  • めまい単独のように見えることがある
  • 歩行不能、体幹失調、眼振パターン、HINTS異常が重要
  • 早期MRIでも見逃しはありうるため、臨床判断を捨てない

検査

基本

  • 採血（血糖＋脱水・感染・貧血・電解質などは適応で）
  • 12誘導ECG

画像

頭部CT

  • 出血疑い
  • 外傷
  • 激しい頭痛
  • 明らかな局所神経症状
  • ただし 後方循環虚血の除外には弱い

MRI / MRA

  • AVSで中枢性が疑わしい
  • #HINTS が中枢パターン、または判断困難
  • 高リスク背景
  • 局所神経症状あり
  • ただし 早期陰性でも絶対安心ではない

disposition

帰宅を考えやすい場合

  • 典型的な #BPPV
  • 歩行可能
  • 経口摂取可能
  • 中枢性 / 循環器性が低そう
  • フォロー可能

入院 / コンサルトを考える場合

  • 中枢性を否定できない
  • 歩行不能
  • 持続する嘔吐
  • 局所神経症状あり
  • 高リスク背景
  • 独居などで安全性が低い

ピットフォール

  • 「回転性だから末梢」と決めつける
  • CTが正常だから安心する
  • AVSに適応がないのに #HINTS を使う
  • 難聴があるから中枢性ではないと思い込む
  • 症状改善 = 安全 と考える
  • 歩けないめまいを軽視する
