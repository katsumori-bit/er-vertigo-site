主訴が失神ではないとき
  • 意識が戻っていない / 持続している意識変容 → 意識障害
  • 転倒の評価が主体（外傷あり）→ 転倒
  • めまい / ふらつき → めまい
  • 動悸 / 胸痛が前景 → 胸痛
  • けいれんの目撃あり → てんかん発作（神経内科コンサル）

0. 失神の定義

TLOC（Transient Loss of Consciousness） のうち、以下を満たすもの：

  • 一過性（通常 < 1 分、せいぜい数分以内）
  • 完全な意識消失
  • 起立保持できない（倒れる）
  • 自然に・完全に回復する
  • 原因は脳全体の灌流低下
失神でないもの
  • てんかん発作：強直・間代運動、舌咬傷、尿失禁、発作後もうろう状態
  • 低血糖：血糖測定で判明
  • 代謝性昏睡 / 中毒：意識が戻らない
  • TIA / 脳卒中：通常は意識消失より神経脱落症状が前景
  • 転倒のみ（意識喪失なし）

1. 危険な失神を外す（5 大致死的）

疾患手がかり
心原性失神（不整脈・構造的心疾患）労作中、臥位での失神、前駆症状なし、心疾患の既往、家族の突然死、ECG 異常
大量出血（消化管・大動脈瘤・異所性妊娠）蒼白、頻脈、立位で悪化、吐血・下血、腹部痛、妊娠可能年齢
肺塞栓（PE）突然の呼吸困難 + 失神、頻脈、片側下肢腫脹、長期臥床・術後
くも膜下出血（SAH）雷鳴頭痛、嘔吐、項部硬直、頭痛が前景
大動脈解離胸背部の引き裂かれる痛み、左右上肢血圧差、新規 AR 雑音
「労作時失神」は心原性

運動中・運動直後の失神は、HCM（肥大型心筋症）・大動脈弁狭窄・冠動脈異常起始・遺伝性不整脈 の可能性が高い → 入院 + 専門科コンサル

2. 病歴

必ず聞くこと

  • いつ・どこで・何をしているとき（労作中？立位？臥位？トイレ？）
  • 前駆症状：眼前暗黒感・冷汗・嘔気（神経調節性の典型）vs 前駆症状なし（心原性）
  • 持続時間：数秒 vs 数分以上
  • 目撃者の証言：けいれん様運動の有無、顔色、回復過程
  • 回復後の状態：完全に元通り vs もうろう状態（てんかんを示唆）
  • 失禁・舌咬傷（てんかん示唆だが失神でも稀に起こる）
  • 随伴症状：頭痛、胸痛、呼吸困難、動悸、腹痛、背部痛、出血
  • 既往歴：心疾患、不整脈、糖尿病、貧血、消化管出血、てんかん
  • 家族歴：突然死、心疾患（特に若年）
  • 内服薬：降圧薬、利尿薬、QT 延長薬、抗凝固薬、インスリン
  • 女性：最終月経、妊娠可能性

失神のタイプ

タイプ特徴
神経調節性（vasovagal）前駆症状あり（冷汗・嘔気）、誘因（採血・痛み・恐怖・長時間立位）、若年に多い
状況性排尿・排便・咳・嚥下中の失神
頸動脈洞症候群高齢、首の動きで誘発、頸動脈洞マッサージで再現
起立性低血圧立位で悪化、降圧薬・利尿薬・自律神経障害（パーキンソン病・糖尿病）
心原性前駆症状なし、労作中・臥位、心疾患既往、家族の突然死

3. 身体所見

  • バイタル（両上肢の血圧、必ず）
  • 起立性血圧変化：仰臥位 → 立位 1 分・3 分後の血圧
    • 収縮期 ≥ 20 mmHg 低下、拡張期 ≥ 10 mmHg 低下、脈拍 ≥ 30/分上昇 → 起立性低血圧
  • 頸：頸静脈怒張、頸動脈雑音
  • 心音：雑音（特に大動脈弁狭窄・HCM）、不整、S3/S4
  • 呼吸音：左右差・wheeze
  • 腹部：圧痛（AAA・異所性妊娠）、拍動性腫瘤
  • 直腸診：黒色便（消化管出血）
  • 下肢：浮腫、片側腫脹（DVT）
  • 神経所見：脱落症状あれば 意識障害 / 脳卒中疑いに切り替え

4. 検査

全例で

  • 12 誘導 ECG（来院後すぐ）
    • 不整脈、心筋梗塞、QT 延長、Brugada、WPW、HCM、PE 示唆所見
    • 詳細 → 入院時心電図 / QT延長 / Brugada心電図
  • 血糖
  • モニター心電図（不整脈の検出）

San Francisco Syncope Rule（CHESS）

以下のいずれかがあれば重大事象高リスク：

  • Congestive heart failure（心不全の既往）
  • Hematocrit < 30%
  • ECG abnormal（新規異常）
  • Shortness of breath（呼吸困難）
  • Systolic BP < 90 mmHg

6. 診断がついたときの入口

治療の詳細は疾患ページで解説

ここは「どの方向に進めるか」だけ。用量・レジメン・適応は各疾患ページへ。

病態ED での方向性詳細
vasovagal誘因回避、補液、再発予防教育、症状時の対応（しゃがむ・寝る）帰宅可
起立性低血圧原因薬剤の見直し、補液、必要でミドドリン / フルドロコルチゾン / 弾性ストッキング—
不整脈 / 心原性循環器コンサル、構造的心疾患の精査、ペースメーカー / ICD 適応評価入院時心電図 / QT延長 / Brugada心電図
出血性輸液 + 輸血、出血源コントロール吐血 / 血便・下血
PE抗凝固、必要に応じて線溶肺塞栓

大動脈解離β 遮断薬で降圧、Stanford A → 緊急手術胸痛 / 大動脈解離

7. disposition

入院

  • CSRS ≥ +1 / SFSR の CHESS 陽性
  • 心原性の疑いが残る（労作時、臥位、ECG 異常、新規雑音）
  • 構造的心疾患の既往
  • 家族歴に若年突然死
  • 失神に伴う重大外傷（転倒 と並行評価）
  • 持続する症状 / バイタル不安定

帰宅検討

  • CSRS 0 点以下 / SFSR 全項目陰性
  • 典型的な vasovagal で誘因明確
  • ECG 正常 + バイタル安定 + 採血で異常なし
  • 6 時間以上の観察で再発なし
  • 再診基準を口頭 + 書面で説明
帰宅前に必ず伝える
  • 再度失神したら 救急外来再診
  • 運転は当面控える（少なくとも数日〜数週間、原因と再発リスクによる）
  • 入浴・水泳は単独で行わない
  • 動悸・胸痛・呼吸困難・頭痛が出たら受診
  • 必要なら 循環器外来 / 神経内科外来 のフォロー予約
